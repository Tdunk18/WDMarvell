Creazione di un backup USB

Attenersi ai passaggi seguenti per eseguire il backup dei dati nel proprio My Cloud™ verso un dispositivo USB o per eseguire il backup dei dati presenti nel dispositivo USB verso il dispositivo My Cloud.

Nota: Questa informazione si applica solo alle porte USB presenti sul retro del My Cloud per ottenere informazioni su come creare una copia di backup via USB usando la porta USB situata nella parte anteriore del dispositivo, vedere Creazione di backup mediante la porta USB frontale con il pulsante Copia USB.

    1. Fare clic su Backup nella barra di navigazione per visualizzare la schermata Backup.
    2. Se necessario, fare clic su Backup USB.
    3. Fare clic su Crea processo.
    4. Inserire le informazioni seguenti per creare un processo di backup via USB:

      Nome processo

      Immettere un nome di processo per il backup.

      Server remoto

      Indicare la direzione del backup selezionandola dal menu a discesa. Le opzioni includono:

      • da USB a NAS: Esegue il backup dei dati nel dispositivo USB verso il dispositivo My Cloud.
      • da NAS a USB: Esegue il backup dei dati nel proprio dispositivo My Cloud verso un dispositivo USB.     

      Cartella di origine

      Fare clic su Sfoglia e selezionare la cartella in cui eseguire il backup.

      Cartella di destinazione

      Fare clic su Sfoglia e selezionare la cartella di destinazione per il backup.

      Tipo di backup

      Indicare il tipo di backup che si desidera eseguire.

      • Copia
      • Sincronizza

      Avvio automatico quando connesso

      Questa opzione avvia automaticamente il processo quando il dispositivo è collegato. Per attivare o disattivare l'opzione, fare clic sul pulsante di attivazione.

       

    1. Fare clic su Crea.
    1. Nell’area Processi di backup USB, fare clic sull'icona Avvio backup per iniziare l'operazione.
      L’avanzamento del processo di backup viene visualizzato nell’area processi backup USB.
