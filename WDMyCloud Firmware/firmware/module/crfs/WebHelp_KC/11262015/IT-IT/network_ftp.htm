Abilitazione dell’accesso FTP

È possibile abilitare o disabilitare il trasferimento di file tra dispositivi di calcolo nella rete tramite FTP (File Transfer Protocol).

Attenersi ai passaggi seguenti per abilitare/disabilitare l'accesso FTP:

    1. Fare clic su Impostazioni.
    2. Nel riquadro sinistro, fare clic su Rete.
    3. Nell'area Servizio di rete>Accesso FTP, fare clic su per abilitare o disabilitare l'accesso FTP.
    4. Leggere il messaggio, quindi fare clic su OK.
    5. Fare clic su Configura.
    6. Nella schermata Impostazioni FTP, inserire le informazioni seguenti:
    • Numero massimo utenti: Dal menu a discesa, selezionare il numero massimo di utenti a cui si desidera consentire l'accesso FTP.
    • Tempo di inattività: Immettere l’intervallo, in minuti, al termine del quale si desidera che l’FTP diventi inattivo.
    • Porta: Immettere la porta da utilizzare per l'accesso FTP.
    • Controllo flusso: Selezionare Illimitato o Personalizza. Se si seleziona Personalizza, immettere un valore di controllo del flusso.
    1. Fare clic su Avanti.
    2. Nel campo Modalità passiva, selezionare Predefinito o Personalizza. Se si seleziona Personalizza, immettere i valori della Modalità passiva.
    3. Selezionare la casella di controllo se si desidera riportare l’IP esterno in modalità PASV (Passiva).
    • Immettere l'indirizzo IP esterno, oppure fare clic su Ottieni IP per lasciare che il sistema immetta l’indirizzo
    1. Fare clic su Avanti.
    2. Inserire le informazioni seguenti:
    3. Lingua client: Selezionare la lingua del client dal menu a discesa.
    4. TLS: Selezionare la casella di controllo Transport Layer Security (TLS) accanto a connessioni TLS implicite o esplicite, se applicabile.
    5. FXP: Fare clic su per abilitare o disabilitare il File Exchange Protocol (FXP).
    1. Fare clic su Avanti.
    2. Immettere gli indirizzi IP a cui si desidera impedire l'accesso FTP, quindi selezionare Permanentemente o Temporaneamente dal menu a discesa. Fare clic su Applica per salvare le voci.
    3. Fare clic su Fine.
