Creazione di un’immagine ISO

Un'immagine ISO è un file che contiene una copia esatta di un file system esistente. Per creare un’immagine ISO, attenersi ai passaggi seguenti.

Per creare un’immagine ISO, attenersi ai passaggi seguenti.

    1. Fare clic su Impostazioni.
    2. Nel riquadro sinistro, fare clic su Utilità.
    3. Nell’area Elenco condivisioni ISO, fare clic su Crea immagine ISO.
    4. Nel campo Dimensione immagine, selezionare la dimensione dell'immagine ISO desiderata dal menu a discesa.
    5. Nel campo Percorso immagine, immettere il percorso dei dati di cui si desidera creare l'immagine oppure fare clic su Sfoglia per selezionare la cartella di cui si desidera creare l’immagine, quindi fare clic su OK.
    6. Nel campo Nome immagine inserire un nome per l’immagine ISO.
    7. Fare clic su Avanti.
    8. Nel campo Seleziona, fare clic su Sovrascrivi o Ignora.
    • Sovrascrivi: Se viene selezionata l'opzione Sovrascrivi, i file con lo stesso nome verranno sovrascritti.
    • Ignora: Se viene selezionata l'opzione Ignora, i file con lo stesso nome verranno ignorati.
    1. Selezionare le cartelle e i file che si desidera aggiungere al file immagine nella colonna di sinistra, quindi fare clic su Aggiungi>>.
    2. Selezionare le cartelle e i file che si desidera rimuovere dall'immagine ISO, quindi fare clic su <<Rimuovi.
    3. Fare clic su Avanti. Viene creata l’immagine ISO.
    1. Fare clic su Fine.
