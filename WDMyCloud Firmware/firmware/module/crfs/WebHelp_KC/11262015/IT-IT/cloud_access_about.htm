Informazioni su Accesso cloud

L'accesso cloud è essenzialmente l'accesso al proprio cloud personale (vale a dire, al dispositivo My Cloud™) da un dispositivo di elaborazione (smartphone, tablet o computer). Per una panoramica delle possibili modalità di accesso i contenuti in remoto, vedere Elementi di base di Accesso remoto.

Come amministratore, è possibile eseguire le seguenti operazioni di accesso cloud.

    • Impostare Accesso cloud  
    • Attivare/disattivare l'accesso remoto
    • Determinare come i dispositivi accedono all'unità (impostazione di connessioni statiche, DHCP o manuali)  
    • Controllare lo stato dell'accesso remoto 
    • Migliorare le prestazioni mediante port forwarding
