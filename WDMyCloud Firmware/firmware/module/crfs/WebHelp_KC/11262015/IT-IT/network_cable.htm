Connessione del dispositivo My Cloud alla rete

La presente procedura illustra come connettere il dispositivo My Cloud™ alla propria rete domestica per mezzo di un cavo Ethernet. Dopo aver effettuato la connessione iniziale, è possibile configurare l'unità per la connessione wireless alla rete.

Per la connessione tramite cavo Ethernet, attenersi ai passaggi seguenti.

    1. Collegare un'estremità del cavo alla porta Ethernet sul retro del dispositivo My Cloud.
    1. Collegare l'altra estremità del cavo direttamente a un router o a un porta di commutazione di rete.
    2. Collegare l'estremità dell'adattatore alla presa di alimentazione sul retro dell'unità.
    3. Inserire l'adattatore di alimentazione in una presa di corrente. Il dispositivo My Cloud si accende automaticamente.

Importante: Prima di configurare il connettore, attendere che il dispositivo My Cloud termini la procedura di accensione (circa tre minuti).

Dopo aver connesso il dispositivo  My Cloud alla rete tramite cavo Ethernet, è possibile utilizzare l'interfaccia Web per configurare una connessione wireless. Vedere Configurazione delle opzioni di connessione.
