Informazioni sugli utenti

Chiunque acceda a proprio cloud personale, deve disporre di un account utente per utilizzare il dispositivo My Cloud I proprietari del dispositivo dispongono di un account utente speciale (admin) che fornisce privilegi di amministratore. Tali privilegi consentono di impostare e configurare il dispositivo in base alle esigenze specifiche, nonché aggiungere altri utenti al proprio cloud personale. Si ha anche la facoltà di stabilire esattamente chi può utilizzare il proprio dispositivo.

Per impostazione predefinita, il nome utente admin non dispone di una password. Tuttavia, è possibile aggiungerne una in qualsiasi momento.

Importante: La prima volta che si configura il dispositivo, mentre è in esecuzione il software di installazione del dispositivo My Cloud o la Procedura guidata introduttiva, si ha la possibilità di modificare il nome amministratore e impostare una password facoltativa. Assicurarsi di tenere traccia di nome e password. Senza tali credenziali, non sarà possibile visualizzare l'interfaccia Web di My Cloud.

Come amministratore del dispositivo My Cloud, è possibile:

    • Aggiungere utenti

    • Aggiungere o modificare le password degli utenti

    • Visualizzare le informazioni degli utenti

    • Modificare l'accesso di un utente a una condivisione  

    • Rimuovere gli utenti.

    • Modificare il nome utente e la password di un utente
