Abilitazione del Client iSCS

Il protocollo Internet Storage Name Service (iSNS) è usato per l’interazione tra server iSNS e client iSNS. I client iSNS sono computer, noti anche come iniziatori, in grado di rilevare dispositivi di archiviazione, noti anche come destinazioni, in una rete Ethernet.

Attenersi ai passaggi seguenti per configurare il client iSCSI iSNS.

Nota: iSNS è utilizzato principalmente per connettersi a un Server Windows.

    1. Fare clic su Storage.
    2. Fare clic su iSCSI.
    3. Fare clic su per abilitare iSCSI.
    4. Nel campo Client iSCSI iSNS, fare clic su ON.
    5. Fare clic su Configura.
    6. Inserire l’indirizzo server del client iSNS (normalmente l’indirizzo IP del proprio Server Windows), quindi fare clic su Applica.
