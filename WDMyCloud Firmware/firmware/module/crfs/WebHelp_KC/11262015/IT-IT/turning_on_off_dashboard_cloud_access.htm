Attivazione/disattivazione dell’Accesso cloud dashboard

Per attivare/disattivare Accesso cloud dashboard, attenersi ai passaggi seguenti.

    1. Fare clic su Impostazioni.
    2. Nel riquadro sinistro, fare clic su Generali.
    3. Nel campo Accesso cloud dashboard, fare clic su per attivare l’accesso remoto al Dashboard dal cloud.
      Nota: Questa opzione consente di eseguire le funzioni di amministratore in remoto.
    4. Leggere la schermata informativa, immettere e inserire nuovamente la password per l’account di amministratore quindi fare clic su OK.
    • Se l’account di amministratore è già associato a una password, esaminare la schermata Accesso cloud dashboard, quindi fare clic su OK.
      Nota: Se le impostazioni del cloud sono configurate su Manuale, è necessario accedere al dispositivo in remoto utilizzando le porte selezionate durante la configurazione manuale del cloud. Esempio: Se la porta HTTP è stata configurata manualmente come 5040, utilizzare l'indirizzo seguente per accedere al dispositivo: http://<Indirizzo IP del dispositivo>:5040.
