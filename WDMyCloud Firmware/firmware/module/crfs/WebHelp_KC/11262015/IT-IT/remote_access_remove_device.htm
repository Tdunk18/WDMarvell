Rimozione di un dispositivo

Attenersi ai passaggi seguenti per rimuovere un dispositivo dal cloud.

    1. Fare clic su Accesso cloud.
    2. Nel riquadro sinistro, fare clic sull'utente di cui si desidera eliminare il dispositivo.
    3. Nella sezione Dispositivi cloud, individuare il dispositivo.
      Le informazioni di stato riflettono una connessione (Connesso) per le app WD Photos e/o WD My Cloud.
    4. Eseguire una delle seguenti operazioni:
    • Per la riga WD Photos corrispondente, fare clic sull'icona Elimina () per rimuovere l'accesso di un dispositivo al cloud tramite l'applicazione mobile WD Photos.
    • Per la riga WD My Cloud corrispondente, fare clic sull'icona Elimina () per rimuovere l'accesso di un dispositivo al cloud tramite l'applicazione WD My Cloud.
    • Per la riga che corrisponde al dispositivo mobile, fare clic sull'icona Elimina () per rimuovere il dispositivo dall'ambiente My Cloud.
    1. Nella finestra di dialogo di conferma, fare clic su OK.
