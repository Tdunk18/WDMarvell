Attivazione/disattivazione di Accesso cloud

Se si dispone dei privilegi di amministratore, è possibile attivare/disattivare l'accesso cloud per tutti gli utenti (vale a dire, è possibile controllare se i dispositivi di elaborazione possono o meno accedere ai contenuti del dispositivo My Cloud™).

Per attivare/disattivare Accesso cloud, attenersi ai passaggi seguenti.

    1. Fare clic su Impostazioni.
    2. Nel riquadro sinistro, fare clic su Generali.
    3. Nel campo Accesso cloud> Servizio cloud, fare clic su per attivare o disattivare il servizio cloud, in base alle esigenze.
    • On: Abilita l'accesso remoto per tutti gli utenti.
    • Off: Disabilita l'accesso remoto per tutti gli utenti.
    1. Fare clic su Configura per modificare il tipo di connessione che si utilizza per l'accesso cloud. La schermata Opzioni di connessione ad Accesso cloud, presenta tre opzioni di accesso.
    2. Modalità automatica: La modalità automatica utilizza il protocollo UPnP (Universal Plug and Play) per tentare di aprire le porte sul router. Se il tentativo va a buon fine, viene stabilita una connessione diretta tra il dispositivo e le app.
    3. Manuale: Stabilisce una connessione tramite le due porte selezionate. Se una porta non è disponibile, viene stabilita una connessione di inoltro. Per questa opzione, è necessaria la configurazione manuale del router. Per informazioni su come impostare il router per la configurazione manuale, consultare nella Knowledge Base la risposta con ID 8526.
    4. Win XP: Stabilisce una connessione tramite le porte 80 e 443. Se tali porte non sono disponibili, viene stabilita una connessione di inoltro. Questa opzione è necessaria se si utilizza Windows XP.
      Nota: Per impostazione predefinita, il dispositivo My Cloud stabilisce automaticamente una connessione diretta tra i dispositivi mobili e il router.
    1. Nel campo Database contenuti, fare clic su Ripristina, per ripristinare il database WDMyCloud.
      Nota: Eseguire questa opzione solo per risolvere problemi se si sospetta un danneggiamento del database.
    2. Fare clic su Applica. Il campo Stato connessione indica che il dispositivo è collegato.

    Per dettagli sulla disattivazione dell'accesso per un singolo dispositivo, vedere Rimozione di un dispositivo.

    Nota: Le informazioni sull'area Opzioni di connessione sono disponibili in Configurazione delle opzioni di connessione, che illustra le diverse opzioni per il firewall e comprende una spiegazione su come migliorare le prestazioni. Per impostazione predefinita, My Cloud stabilisce automaticamente una connessione diretta tra i dispositivi mobili e il router.

    Se l'accesso mobile è abilitato attraverso una connessione di inoltro (vale a dire che Stato di connessione indica "Connesso (trasmissione)"), è possibile migliorare le prestazioni abilitando il port forwarding nel connettore WD Venue (vedere Abilitazione del port forwarding).
