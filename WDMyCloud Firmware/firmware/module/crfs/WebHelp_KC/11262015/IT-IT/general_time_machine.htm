Back up dei dati con Apple Time Machine

È possibile effettuare il backup dei file presenti sul dispositivo My Cloud™ su Apple Time Machine in un computer Mac.

Importante: Per ripristinare i backup di Time Machine, è necessario impostare una password per l’account di amministratore.

Per eseguire il backup dei dati in Apple Time Machine, attenersi ai passaggi seguenti.

    1. Fare clic su Impostazioni.
    2. Nel riquadro sinistro, fare clic su Generali.
    3. Nell'area Backup MAC, fare clic su per attivare l'opzione di backup Time Machine.
    4. Fare clic su Configura.
    5. Nell'elenco a discesa Seleziona una condivisione, selezionare la condivisione in cui risiederanno i file di backup.
    6. Nel campo Dimensione massima, spostare il dispositivo di scorrimento per indicare la dimensione massima desiderata per il backup.
      Nota: L'aumento del valore massimo dopo il backup iniziale non produce effetti sui backup esistenti. WD consiglia di lasciare l'impostazione sul valore massimo consentito per il backup iniziale. Una volta completato il primo backup, è possibile modificare la dimensione del backup trascinando il dispositivo di scorrimento della dimensione massima.
    7. Fare clic su Salva.

    Importante: Quando Time Machine avvia il backup dei file verso una condivisione selezionata, si suggerisce di continuare a eseguire il backup verso la stessa condivisione. Cambiare condivisioni comporta la creazione di un nuovo file di backup che non contiene i dati salvati precedentemente.
