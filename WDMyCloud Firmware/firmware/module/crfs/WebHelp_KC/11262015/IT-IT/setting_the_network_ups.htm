Impostazione dell’UPS rete

Un gruppo di continuità (UPS) è un dispositivo che contiene una serie di accumulatori in grado di mantenere i dispositivi collegati alla porta UPS in funzione almeno per un breve periodo, se la fonte di alimentazione principale subisce un'interruzione. Se un dispositivo My Cloud™ è connesso all'unità di rete via UPS, l'unità si spegne automaticamente quando la carica dell’UPS si riduce a una determinata percentuale.

La comunicazione con il dispositivoMy Cloud avviene mediante un protocollo master/slave che prevede un dispositivo principale che controlla altri dispositivi secondari.

Se l’UPS è collegato al dispositivo My Cloud tramite USB, lo stesso diventa automaticamente dispositivo UPS di rete principale. Eventuali altri dispositivi My Cloud collegati all’UPS possono essere diretti all'indirizzo IP dell’UPS principale. Se la carica dell’UPS si riduce a una determinata percentuale, tutti i dispositivi secondari My Cloud si arrestano correttamente. Per ottenere un elenco di dispositivi UPS, consultare la risposta della Knowledge Base con ID 11852.

Attenersi ai passaggi seguenti per impostare l’UPS rete del dispositivo My Cloud™.

    1. Fare clic su Impostazioni.
    2. Nel riquadro sinistro, fare clic su Rete.
    3. Nell'area Servizi di rete>Secondario, fare clic su per abilitare il DNS dinamico.
    4. Inserire l'indirizzo IP dell’UPS principale in rete.
    5. Fare clic su Avanti. Se l'impostazione è aggiornata, viene visualizzata una schermata “in corso”; la schermata Rete compare con L’UPS rete secondario impostato su OFF.
