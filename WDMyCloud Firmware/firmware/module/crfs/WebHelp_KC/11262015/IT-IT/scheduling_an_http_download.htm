Pianificazione di un download HTTP

L'app per il download HTTP consente di pianificare un download HTTP nel proprio dispositivo My Cloud™. Attenersi ai passaggi seguenti per pianificare un download HTTP.

    1. Fare clic su App.
    2. Fare clic su Download HTTP.
    3. Nel campo dell'URL, immettere l'URL del file di cui si desidera eseguire il download.
      Esempio: http://192.168.0.32/test.txt
    4. Nel campo Salva in immettere la posizione in cui si desidera salvare il download oppure fare clic su Sfoglia per selezionare la posizione in cui salvare il download.
      Esempio: Volume_1/Public/Shared Videos 
    5. Per cambiare il nome file, nel campo Rinomina immettere un nuovo nome per il file.
    6. Fare clic su  in caso di backup ricorrente.
    • Selezionare la frequenza del backup (Ogni giorno, Ogni settimana, Ogni mese)
    • Inserire l'ora, il giorno e la data del backup, in base alla frequenza selezionata.
    1. Immettere la data per l'esecuzione del download (mm/gg/aaaa) oppure fare clic su  per selezionare una data.
    2. Immettere un'ora per il download dai menu a discesa.
    3. Fare clic su Crea. Viene creato il download e viene aggiunto alla Coda download HTTP.
