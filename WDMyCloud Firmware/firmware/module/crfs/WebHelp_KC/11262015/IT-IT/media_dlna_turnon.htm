Abilitazione/disabilitazione DLNA

Attivare l'impostazione Server DLNA per rendere i contenuti del cloud accessibili a un dispositivo DLNA (Digital Living network Alliance).

Per attivare/disattivare il server DLNA, attenersi ai passaggi seguenti:

    1. Fare clic su Impostazioni.
    2. Nel riquadro sinistro, fare clic su Contenuti multimediali.
    3. Nell'area Server multimediale DLNA>Streaming multimediale, fare clic su per attivare o disattivare lo streaming multimediale, in base alle esigenze.
    • On: Consente ai dispositivi DLNA di accedere al cloud.
    • Off: Nega l'accesso ai dispositivi DLNA.
    1. Leggere la nota, quindi fare clic su OK. All'interno dell'area del server multimediale DLNA vengono visualizzate le seguenti informazioni:

Versione

Mostra la versione software corrente del server multimediale DLNA.

Contenuti multimediali

Identifica il numero di brani musicali, immagini e video disponibili sul server DLNA.

Ultimo aggiornamento

Indica il giorno, la data e l'ora dell’ultimo aggiornamento del Server DLNA.

Database DLNA

Questi pulsanti consentono di eseguire una Nuova scansione o di Ripristinare il database del server DLNA.
