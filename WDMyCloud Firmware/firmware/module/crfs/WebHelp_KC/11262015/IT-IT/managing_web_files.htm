Gestione file Web

Visualizzatore file Web è un'app che consente di visualizzare e gestire i file nel proprio dispositivo My Cloud™.

Attenersi ai passaggi seguenti per gestire i file Web.

    1. Fare clic su App.
    2. Fare clic su Visualizzatore file Web. I volumi di file disponibili nel dispositivo vengono visualizzati nel riquadro di destra.

Visualizzazione file Web

I volumi, le cartelle e i file vengono visualizzati nel riquadro destro di Visualizzatore file Web. Per visualizzare i contenuti di qualsiasi volume o cartella, fare doppio clic sulla cartella.

Gestione file Web

Le azioni disponibili che è possibile eseguire in file e cartelle sono visualizzate nella parte superiore destra di Visualizzatore file Web. Selezionare una delle azioni seguenti da eseguire nei file Web:

Icona

Funzione

Descrizione

Copia

Questa azione consente di copiare un file selezionato in un'altra posizione. Attenersi ai passaggi seguenti per copiare un file in una nuova posizione:

    1. Nell'area Visualizzatore file Web, selezionare il file che si desidera copiare.
    2. Fare clic su .
    3. Selezionare la destinazione della copia del file.
    4. Fare clic su OK. Il file viene copiato nella destinazione selezionata.

Sposta

Questa azione consente di spostare un file da una posizione all'altra. Attenersi ai passaggi seguenti per spostare un file in una nuova posizione:

    1. Selezionare il file da spostare.
    2. Fare clic su .
    3. Selezionare la destinazione in cui si desidera spostare il file.
    4. Fare clic su OK. Il file viene spostato nella destinazione selezionata.

Carica

Questa azione consente di caricare un file dal computer o da un altro dispositivo nel server My Cloud . Attenersi ai passaggi seguenti per caricare un file nel dispositivo:

    1. Nel riquadro destro di Visualizzatore file Web, selezionare la destinazione per il file caricato.
    2. Fare clic su .
    3. Nella schermata Carica, fare clic sull'icona Seleziona file
    4. Individuare e selezionare il file da caricare nel dispositivo.
    5. Fare clic su Apri.
    6. Fare clic su Carica.
    • Per caricare più file, eseguire i passaggi 3 e 4 fino al caricamento completo di tutti i file.
    1. Fare clic su Chiudi. Nella cartella di destinazione vengono visualizzati i file caricati.

Download

Questa azione consente di eseguire il download di un file dal dispositivo My Cloud nel computer o in altri dispositivi.
Attenersi ai passaggi seguenti per eseguire il download di un file nel computer o in altri dispositivi:

    1. Nel riquadro destro di Visualizzatore file Web, selezionare il file di cui si desidera eseguire il download.
    2. Fare clic su .
    3. Viene eseguito il download del file selezionato nel computer o in altri dispositivi.

Rinomina

Questa azione consente di rinominare un file selezionato nel dispositivo My Cloud . Per rinominare il file attenersi ai passaggi seguenti.

    1. Nel riquadro destro di Visualizzatore file Web, selezionare il file da rinominare.
    2. Fare clic su .
    3. Nel campo Rinomina, rinominare il file.
    4. Fare clic su Salva. Il file è stato rinominato.

Elimina

Questa azione consente di eliminare un file selezionato dal dispositivo My Cloud. Per eliminare un file, attenersi ai passaggi seguenti.

    1. Nel riquadro destro di Visualizzatore file Web, selezionare il file da eliminare.
    2. Fare clic su .
    3. Fare clic su OK per verificare che si desidera eliminare il file selezionato. Il file selezionato viene eliminato dal dispositivo.

Aggiorna

Questa azione aggiorna l'elenco dei file, visualizzando i nuovi file aggiunti dall'ultimo aggiornamento o dall'ultimo accesso all'elenco dei file.

  • Fare clic su . I file sono stati aggiornati.

Crea

Questa azione consente di creare una cartella all'interno della condivisione selezionata.

    1. Navigare alla posizione in cui si desidera creare la nuova cartella.
    2. Fare clic su
    3. Immettere il nome della nuova cartella.
    4. Fare clic su Salva.La nuova cartella viene creata.
