Eseguire la configurazione una sola volta per l'accesso sicuro

Questa procedura illustra come stabilire una connessione sicura tra il proprio smartphone o tablet e il dispositivo My Cloud™. Attenersi alle istruzioni del proprio dispositivo mobile specifico.

    • Smartphone Android o tablet  

    • iPhone o iPad

    • Telefono o tablet Windows

    • Desktop Windows o Mac

Ottenere accesso sicuro per uno smartphone o un tablet Android

    1. Connettere il proprio smartphone o tablet procedendo come segue:

    • Sul proprio dispositivo mobile, toccare Impostazioni.

    • Toccare Wireless e reti.

    • Toccare Impostazioni Wi-Fi.

    • Seleziona la rete domestica.

    • Inserire la password di rete e toccare Connetti.

    1. Lanciare un'app WD.
    2. Toccare Connetti ora.
    3. Toccare il nome del dispositivo My Cloud.
    4. Tocca la freccia di selezione e seleziona il nome utente.
    5. Esegui una delle azioni seguenti:

    • Se è stata impostata una password per l'utente, inserirla e toccare Login.

    • Se non è stata impostata una password per l'utente, toccare Login.

    1. (Opzionale) Se si desidera accedere a qualunque contenuto, toccare il nome di una condivisione, quindi navigare fino al file desiderato.

Ottenere accesso sicuro per un iPhone o iPad

    1. Connettere il proprio smartphone o tablet procedendo come segue:

    • Sul proprio dispositivo mobile, toccare Impostazioni.

    • Toccare Wi-Fi.

    1. Toccare Impostazioni Wi-Fi.

    2. Seleziona la rete domestica.

    3. Inserire la password di rete e toccare Unisci.

    1. Lanciare un'app WD.
    2. Toccare Connetti ora.
    3. Toccare il segno più (+) accanto al nome del dispositivo My Cloud.
    4. Toccare il nome utente desiderato.
      Se è stata impostata una password per l'utente, inserirla.
    5. (Opzionale) Se si desidera accedere a qualunque contenuto, toccare il nome di una condivisione, quindi navigare fino al file desiderato.

Ottenere accesso sicuro per uno smartphone o un tablet Windows

    • Connettere il proprio smartphone o tablet procedendo come segue:

    • Sul proprio dispositivo di elaborazione, toccare Impostazioni.

    • Toccare Wi-Fi.

    • Seleziona la rete domestica.

    • Inserire la password di rete e toccare Fatto.

    1. Lanciare un'app WD.
    2. Toccare Connetti ora.
    3. Toccare il nome dell’unità My Cloud.
    4. Tocca la freccia di selezione e seleziona un nome utente.
      Se è stata impostata una password per l'utente, inserirla.
    5. (Opzionale) Se si desidera accedere a qualunque contenuto, procedere come segue:

    • Nella schermata Dispositivi, toccare il nome della propria unità My Cloud.

    • Tocca una condivisione, quindi portati sul contenuto desiderato.

Ottenere accesso sicuro per un desktop o laptop Windows o Mac

    1. Dopo aver installato WD My Cloud per l'app desktop, fare doppio clic sul collegamento che viene mostrato nel desktop per visualizzare tutti i dispositivi WD in rete. Per informazioni sull'installazione dell'app, vedere Elementi di base dell'accesso remoto.
    2. Potrebbero essere visualizzate informazioni sugli avvisi di sicurezza. Per evitare di visualizzare di nuovo la schermata, selezionare la casella di controllo Comprendo.
    3. Fare clic su Comprendo per visualizzare i dispositivi nella rete.
    4. Fare clic sul dispositivo a cui si desidera connettersi, in questo caso il dispositivo My Cloud. Viene visualizzata la schermata di accesso.
    5. Se è stata impostata in precedenza, immettere la password nel dispositivo My Cloud , quindi fare clic su Accedi. Viene visualizzata la schermata della guida introduttiva.
    6. Per ottenere ulteriori informazioni, fare clic sul collegamento del WD Learning Center.
    7. Fare clic su Start per visualizzare le condivisioni di My Cloud e i file di backup di SmartWare o Time Machine.
