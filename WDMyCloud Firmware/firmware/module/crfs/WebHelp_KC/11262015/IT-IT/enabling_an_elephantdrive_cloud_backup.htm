Abilitazione di un cloud backup ElephantDrive

ElephantDrive è un servizio di cloud backup che fornisce backup sicuri e automatici in remoto ai file. Una volta abilitato, ElephantDrive esegue automaticamente la sincronizzazione con il dispositivo.

Per abilitare un cloud backup utilizzando ElephantDrive, attenersi ai passaggi seguenti.

    1. Fare clic su Backup.
    2. Fare clic su cloud backup.
    3. Fare clic sul pulsante ElephantDrive nella parte alta della schermata.
    4. Fare clic su . Il cloud backup ElephantDrive è stato attivato.
    5. Fare clic su Registra.
    6. Immettere le informazioni seguenti, quindi fare clic su Registra:
    • Indirizzo e-mail: per ricevere informazioni su ElephantDrive, immettere l'indirizzo e-mail.
    • Password: immettere la password del nuovo account
    • Verifica password: immettere di nuovo la password.

Il cloud backup ElephantDrive è stato abilitato.
