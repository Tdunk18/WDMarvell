移除裝置

使用下列步驟，可從雲端移除裝置。

    1. 按一下 [雲端存取]。
    2. 在左窗格中，選擇您要刪除其裝置的使用者。
    3. 在 [雲端裝置] 區段中找到該裝置。
      狀態資訊會反映 WD Photos 和/或 WD My Cloud 的連線狀態 (「已連線」)。
    4. 執行以下其中之一或多個操作：
    • 對於對應的 WD Photos 列，按一下 [刪除] 圖示 () 可移除裝置透過 WD Photos 行動應用程式存取雲端的功能。
    • 對於對應的 WD My Cloud 列，按一下 [刪除] 圖示 () 可移除裝置透過 WD My Cloud 行動應用程式存取雲端的功能。
    • 對於對應至行動裝置的列，按一下 [刪除] 圖示 () 可將裝置從您的 My Cloud 環境中移除。
    1. 按一下確認對話方塊中的 [確定]。
