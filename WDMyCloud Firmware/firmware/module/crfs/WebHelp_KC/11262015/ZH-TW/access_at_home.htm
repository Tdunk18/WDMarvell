在家存取內容

當您在家裡且電腦裝置連線到您的家用網路時，您存取磁碟機內容的方式與遠端存取的方式不同。當您連線至您的家用網路時，您可以：

    • 從智慧型手機或平板電腦存取內容
    • 從電腦存取內容

重要：開始之前，請先確定您已完成設定雲端存取中所述的步驟。

從連線至您家用網路的行動裝置存取內容

    1. 如果您尚未這麼做，請下載 WD 行動應用程式。
    2. 將您的智慧型手機或平板電腦連線至您的家用網路。
      如果您不知道怎麼做，請參閱您智慧型裝置的文件。
    3. 啟動 WD 行動應用程式 (WD My Cloud 或 WD Photos)。
    4. 如果出現提示，請點選 [立即連線]。
    5. 選擇您的 My Cloud 磁碟機。
    6. 選取您要存取的共用，然後瀏覽至所要的內容。

從連接至您家用網路的電腦存取內容

    1. 確定您的電腦已連接至您的家用網路，且您具備網際網路存取權。
    2. 如果您尚未這麼做，請從「WD 學習中心」(http:.wd.com/setup/) 下載 WD My Cloud 桌面應用程式，並安裝在您的 Mac 或 PC 上。
    3. 在您的電腦上啟動 WD My Cloud 桌面應用程式。
    4. 輸入您的登入認證。
    5. 按兩下 My Cloud 磁碟機。
    6. 視需要展開左窗格中的資料夾，瀏覽至所需的內容。
