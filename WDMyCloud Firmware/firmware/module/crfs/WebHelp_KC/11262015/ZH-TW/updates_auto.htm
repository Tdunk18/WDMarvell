自動更新 My Cloud 裝置

自動更新可將 My Cloud™ 装置設定為自動檢查並安裝可用的軟體更新，從而為您帶來方便。這是確定 My Cloud 韌體為最新版的最簡單方法。

注意： 安裝韌體更新後，My Cloud 装置可能重新開機。 重新啟動會影響到使用者存取雲端的能力，應排程更新在使用者存取雲端的可能性最小的時間進行。

注意： 如果您啟用自動更新，更新完成後您需要手動重啟系統。

使用下列步驟，可自動更新 My Cloud 裝置。

    1. 按一下 [設定]。
    2. 按一下左窗格中的 [韌體更新]。
    3. 在 [自動更新] 區域中按一下 ，啟用自動更新。
    4. 在 [更新排程] 欄位中，排程更新的日期和時間：

    • 按一下第一個下拉箭頭，選擇週中的日。

    • 按一下第二個下拉箭頭，選擇時間。

    • 按一下第三個下拉箭頭，選取上午或下午。

    1. 按一下 [儲存]。
