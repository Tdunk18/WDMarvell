使用者說明

任何人存取個人雲端都必須具備使用者帳戶，才能使用 My Cloud 裝置。身為裝置擁有者，您具備特殊的使用者帳戶 (admin)，讓您擁有系統管理員權限。有了這些權限，您便可設定裝置以符合您特定的需求，並將其他人員新增到您的個人雲端。您也可確切決定哪些人可以使用您的裝置。

根據預設，admin 使用者名稱沒有密碼。您可以隨時新增密碼。

重要：一開始設定裝置時，在執行 My Cloud 裝置設定軟體或快速人門精靈的時候，有機會變更系統管理員名稱並設定可選擇的密碼。請務必記下此名稱與密碼。若沒有這些認證，您將無法顯示 My Cloud 網頁介面。

作為 My Cloud 裝置的系統管理員，您可以：

    • 新增使用者

    • 新增或編輯使用者密碼

    • 檢視使用者資訊

    • 變更使用者的共享區存取權  

    • 移除使用者。

    • 變更某人的使用者名稱與密碼
