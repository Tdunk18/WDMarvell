將 My Cloud 裝置連線至您的網路

此程序說明如何使用乙太網路纜線，將 My Cloud™ 装置磁碟機連線至您的家用網路。進行初次連線後，您就能設定磁碟機以無線方式連線至網路。

使用下列步驟，可使用乙太網路纜線連線。

    1. 將纜線的一端連接到 My Cloud 裝置背面的乙太網路連接埠。
    1. 將纜線的另一端直接連接至您路由器或網路交換器連接埠。
    2. 將配接器端連接至磁碟機背面的電源供應插座。
    3. 將電源配接器插入到電源插座。My Cloud 裝置電源會自動開啟。

重要：先等待 My Cloud 裝置完成電源開啟 (約三分鐘)，再設定接頭。

透過乙太網路纜線將 My Cloud 裝置連線至網路後，即可使用網頁介面設定無線連線。請參閱設定連線選項。
