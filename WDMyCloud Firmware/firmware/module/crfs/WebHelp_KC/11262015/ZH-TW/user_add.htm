新增使用者帳戶

遵循下列步驟，將使用者新增至 My Cloud™ 裝置：

    1. 按一下 [使用者]。
    2. 按一下 [新增使用者] 圖示。
    3. 輸入使用者名稱。這是用來登入系統的名稱。
    4. 輸入名字。這是使用者的名字。
    5. 輸入姓氏。這是使用者的姓氏。
    6. 輸入電子郵件地址。這是使用者的電子郵件地址。此地址也將用來建立 WD My Cloud 帳戶。
    7. 按一下套用。您的新使用者會顯示在左窗格中。
