啟用/停用 SSH

您可以設定 My Cloud™ 裝置，讓您可以安全地存取您的雲端，並透過 Secured Shell (SSH) 通訊協定執行命令列操作。

使用下列步驟，可啟用/停用 SSH：

    1. 按一下 [設定]。
    2. 按一下左窗格中的 [網路]。
    3. 在 [網路服務] 區域中，按一下 以啟用或停用 SSH。

      警告：請注意，以正常操作之外的方式修改或嘗試修改此裝置將會喪失您的 WD 保固。

    注意：SSH 登入使用者名稱是 sshd。
