使用智慧型手機或平板電腦存取內容

您可以使用智慧型手機或平板電腦存取已儲存在您的 My Cloud 裝置上的內容。在您這麼做之前，您必須已完成下列工作：設定雲端存取、下載 WD 行動應用程式，以及執行安全存取的一次性設定。

使用下列步驟，可使用智慧型手機或平板電腦存取內容。

    1. 確定您的行動裝置具備 Wi-Fi 或 3G 或更高級別的連線功能。
    2. 啟動 WD 行動應用程式 (WD My Cloud 或 WD Photos)。
    3. 如果出現提示，請點選 [立即連線]。
    4. 選擇您的 My Cloud 裝置。
    5. 選取您要存取的共用，然後瀏覽至所要的內容。
