執行安全存取的一次性設定

此程序說明如何在您的智慧型手機或平板電腦與 My Cloud™ 裝置之間建立安全的連線。使用您特有行動裝置適用的指示：

    • Android 智慧型手機或平板電腦

    • iPhone 或 iPad

    • Windows 手機或平板電腦

    • Windows 或 Mac 桌上型電腦

取得 Android 智慧型手機或平板電腦的安全存取

    1. 執行以下步驟，將您的智慧型手機或平板電腦連線：

    • 點選您行動裝置上的 [設定]。

    • 點選 [無線和網路]。

    • 點選 [Wi-Fi 設定]。

    • 選取您的家用網路。

    • 輸入您的網路密碼，並點選 [連線]。

    1. 啟動 WD 應用程式。
    2. 點選 [現在連線]。
    3. 點選 My Cloud 裝置名稱。
    4. 點選下拉箭頭，然後選擇您的使用者名稱。
    5. 執行下列其中一項：

    • 如果已為使用者設定密碼，請輸入該密碼，再點選 [登入]。

    • 如果尚未為使用者設定密碼，請點選 [登入]。

    1. (選擇性) 如果您要存取您的任何內容，請點選共享區名稱，然後瀏覽至所需的檔案。

取得 iPhone 或 iPad 的安全存取

    1. 執行以下步驟，將您的智慧型手機或平板電腦連線：

    • 點選您行動裝置上的 [設定]。

    • 點選 [Wi-Fi]。

    1. 點選 [Wi-Fi 設定]。

    2. 選取您的家用網路。

    3. 輸入您的網路密碼並點選 [加入]。

    1. 啟動 WD 應用程式。
    2. 點選 [現在連線]。
    3. 點選 My Cloud 裝置名稱旁的加號 (+)。
    4. 點選所需的使用者名稱。
      如果已為使用者設定密碼，請輸入該密碼。
    5. (選擇性) 如果您要存取您的任何內容，請點選共享區名稱，然後瀏覽至所需的檔案。

取得 Windows 智慧型手機或平板電腦的安全存取

    • 執行以下步驟，將您的智慧型手機或平板電腦連線：

    • 點選您電腦裝置上的 [設定]。

    • 點選 [Wi-Fi]。

    • 選取您的家用網路。

    • 輸入您的網路密碼並點選 [完成]。

    1. 啟動 WD 應用程式。
    2. 點選 [現在連線]。
    3. 點選 My Cloud 磁碟機名稱。
    4. 點選下拉箭頭，然後選取使用者名稱。
      如果已為使用者設定密碼，請輸入該密碼。
    5. (選擇性) 如果您要存取您的任何內容，請執行以下步驟：

    • 在 [裝置] 畫面上點選 My Cloud 磁碟機名稱。

    • 觸碰一個共用，然後瀏覽到所要的內容。

取得 Windows 或 Mac 桌上型電腦或筆記型電腦的安全存取

    1. 安裝桌面應用程式適用的 WD My Cloud 後，按兩下現在出現在桌面上的捷徑 ，以顯示網路上的所有 WD 裝置。 如需安裝應用程式的詳細資訊，請參閱遠端存取基本介紹。
    2. 系統可能會顯示安全警示的相關資訊。若要避免再次顯示此畫面，請選取 [我了解] 核取方塊。
    3. 按一下 [我了解]，顯示網路上的裝置。
    4. 按一下您要連線的裝置，在此處為 My Cloud 裝置。接著會顯示 [登入] 畫面。
    5. 輸入 My Cloud 裝置的密碼 (如果先前有設定)，然後按一下 [登入]。接著會顯示 [開始使用] 畫面。
    6. 若想獲得更多資訊，請按一下 [WD 學習中心] 連結。
    7. 按一下 [開始] 顯示 My Cloud 裝置的共享以及 SmartWare 或 Time Machine 備份檔案。
    8. 若想了解如何遠端存取和管理 My Cloud 裝置上儲存的內容，請按一下線上學習中心 My Cloud 下載面板中的 [使用您的電腦存取] 連結。
