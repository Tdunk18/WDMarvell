遠端存取基本介紹

有三個方法可讓您從任何地方存取 My Cloud 上的儲存內容：：

    • 使用如智慧型手機或平板電腦等行動裝置。
    • 使用您的電腦 (最有可能為筆記型電腦)。

行動裝置存取概述

使用下列步驟，可從智慧型手機或平板電腦存取雲端：

1

取得 WD My Cloud 帳號。此步驟通常在您新增使用者至裝置時就已完成。請參閱設定雲端存取。

2

下載免費的 WD Photos 與 WD My Cloud 應用程式並安裝在行動裝置上。
這些免費應用程式可從 Apple App Store、Google Play store 和適用於 Android 的 Amazon Appstore 取得。

3

執行安全存取的一次性設定。請參閱執行安全存取的一次性設定

4

使用行動裝置存取您的內容。請參閱使用智慧型手機或平板電腦存取內容

Windows 或 Mac 桌上型電腦存取概述

當您外出不在家時 (例如在咖啡店)，您可以輕鬆地從筆記型電腦存取您的裝置。進行遠端存取需要有 WD My Cloud 桌面應用程式、啟用 Wi-Fi 的筆記型電腦以及 Wi-Fi 連線。

使用下列步驟，可使用 WD My Cloud 桌面應用程式存取雲端：

1

取得 My Cloud 帳號。此步驟通常在您新增使用者至裝置時就已完成。請參閱設定雲端存取。

2

從「WD 學習中心」http://wd.com/setup 下載 WD My Cloud 桌面應用程式，並安裝在您的電腦上。

3

執行安全存取的一次性設定。請參閱執行安全存取的一次性設定

4

使用筆記型電腦存取內容。請參閱使用筆記型電腦存取內容
