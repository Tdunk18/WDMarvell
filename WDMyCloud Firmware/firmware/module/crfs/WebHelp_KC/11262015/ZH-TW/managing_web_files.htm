管理網頁檔案

網頁檔案檢視是一個應用程式，可讓您檢視和管理 My Cloud™ 裝置上的檔案。

使用下列步驟，可管理網頁檔案。

    1. 按一下 [應用程式]。
    2. 按一下 [網頁檔案檢視器]。您裝置上可用的檔案磁碟區即會顯示在右窗格中。

檢視網頁檔案

您的磁碟區、資料夾和檔案會顯示在 [網頁檔案檢視器] 的右窗格中。若要檢視任何磁碟區或資料夾的內容，請按兩下該資料夾。

複製

此動作可將選定檔案複製至其他位置。使用下列步驟，可將檔案複製至新位置：

    1. 在 [網頁檔案檢視器] 區域中，選取您要複製的檔案。
    2. 按一下 。
    3. 選取檔案複製的目的地。
    4. 按一下 [確定]。您的檔案隨即複製至選定目的地。

移動

此動作可將檔案從一個位置移動至另一個位置。使用下列步驟，可將檔案移動至新位置：

    1. 選取您要移動的檔案。
    2. 按一下 。
    3. 選取您要移動檔案的目的地。
    4. 按一下 [確定]。您的檔案隨即移動至選定目的地。

上傳

此操作可從電腦或其他裝置上載檔案至 My Cloud 伺服器。使用以下步驟上載檔案至裝置：

    1. 在 [網頁檔案檢視器] 的右窗格中，選取上傳檔案的目的地。
    2. 按一下 。
    3. 在 [上傳] 畫面中，按一下 [選取檔案] 圖示
    4. 瀏覽並選取您要上傳至裝置的檔案。
    5. 按一下 [開啟]。
    6. 按一下 [上傳]。
    • 如果要上傳多個檔案，請重複執行步驟 3 和 4，直到所有檔案都已上傳。
    1. 按一下 [關閉]。已上傳檔案會顯示在目的地資料夾中。

下載

此動作可讓您從 My Cloud 裝置下載檔案至您的電腦或其他裝置。
使用下列步驟，可將檔案下載至您的電腦或其他裝置：

    1. 在 [網頁檔案檢視器] 的右窗格中，選取要下載的檔案。
    2. 按一下 。
    3. 選取的檔案就會下載至您的電腦或其他裝置。

重新命名

此動作可讓您重新命名 My Cloud 裝置上的選定檔案。使用下列步驟，可重新命名檔案。

    1. 在 [網頁檔案檢視器] 的右窗格中，選取要重新命名的檔案。
    2. 按一下 。
    3. 在 [重新命名] 欄位中，重新命名您的檔案。
    4. 按一下 [儲存]。您的檔案隨即就會重新命名。

刪除

此動作可讓您從 My Cloud 裝置上刪除選定檔案。使用下列步驟，可刪除檔案。

    1. 在 [網頁檔案檢視器] 的右窗格中，選取要刪除的檔案。
    2. 按一下 。
    3. 按一下 [確定]，確認您要刪除選取的檔案。選取的檔案就會從裝置上刪除。

重新整理

此動作可以重新整理檔案清單，顯示上次重新整理後或上次存取檔案清單後所新增的新檔案。

  • 按一下 。您的檔案就會重新整理。

建立

此操作可在選定共享內建立資料夾。

    1. 導覽至您要建立新資料夾的位置。
    2. 按一下
    3. 輸入新資料夾的名稱。
    4. 按一下 [儲存]。新資料夾已建立。
