使用 Apple Time Machine 備份資料

您可以將 My Cloud™ 裝置上的檔案備份至 Mac 電腦上的 Apple Time Machine。

重要：您必須為管理員帳戶設定密碼以恢復 Time Machine 備份

使用下列步驟，可將資料備份至 Apple Time Machine。

    1. 按一下 [設定]。
    2. 按一下左窗格中的 [一般]。
    3. 在 [MAC 備份] 區域中，按一下 開啟 Time Machine 備份選項。
    4. 按一下 [設定]。
    5. 在「選擇共享」清單中選取要存放您備份檔的共享區。
    6. 在 [最大容量] 欄位，移動滑桿指明您要用於備份的最大值。
      注意：在起始備份之後增加備份值上限並不會影響現有的備份。WD 建議在起始備份中將設定設為最大值。在第一次備份成功之後，您可以藉由拖曳上限值滑桿來變更備份大小。
    7. 按一下 [儲存]。

    重要：Time Machine 開始備份檔案至選取的共享後，我們建議您以後繼續備份至該共享。更換共享會產生新的備份檔案，其中不包含您之前保存的資訊。
