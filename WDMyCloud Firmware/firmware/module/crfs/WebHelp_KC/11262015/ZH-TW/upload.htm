上傳內容至 My Cloud 裝置

若要將相片、影片、音樂與檔案上傳至您的 My Cloud 裝置，您必須使用以下其中一種程序：

    • Mac
    • Windows PC
    • USB

從 Mac 上傳內容

若要將 Mac 中的檔案上傳至 My Cloud 裝置：

    1. 在 Mac 的功能表列上，選擇 [Go] > [Network]。
    2. 執行下列其中一項：
    • 在 Finder 的左窗格中，按兩下 My Cloud 裝置名稱。
    • 在右窗格中找到 My Cloud 裝置名稱並按兩下。
    1. 按兩下特定共享項目的名稱以開啟資料夾，並檢視任何現有的子資料夾(「公用」是系統自動建立的共享項目)。
    2. 將檔案從您的電腦拖放到所需的資料夾。
      請勿將檔案放置在 Time Machine 與 WD SmartWare Backups 資料夾。
      注意：如果您將檔案從電腦複製到 My Cloud 裝置，則這些檔案將位於兩個位置：原始位置與您的裝置上。

從 PC 上傳內容

若要將 Windows PC 中的檔案上傳至 My Cloud：

    1. 在您的電腦上同時按下 Windows 與 R 鍵，開啟 [執行] 對話方塊。
    2. 在 [開啟] 欄位中輸入 \\，後面加上 My Cloud 裝置名稱。(預設名稱為 wdmycloud，不含空格)。範例：\\wdmycloud
    3. 按一下 [確定]。
      Windows 檔案總管隨即開啟，在右窗格中列出可用的共享項目。
    4. 按兩下特定共享項目的名稱以開啟資料夾，並檢視任何子資料夾。(「公用」是系統自動建立的共享項目)。
    5. 將檔案從您的電腦拖放到此共享項目。
      注意：如果您將檔案從電腦複製到 My Cloud 裝置，則這些檔案將位於兩個位置：原始位置與您的裝置上。
