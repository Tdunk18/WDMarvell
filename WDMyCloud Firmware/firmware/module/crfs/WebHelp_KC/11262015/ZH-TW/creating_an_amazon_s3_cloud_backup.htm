建立 Amazon S3 雲端備份

注意：要建立雲端備份，請先至 http://aws.amazon.com/ 申請 Amazon S3 服務。

設定好您的 Amazon S3 帳戶後，使用下列步驟建立雲端備份。

    1. 按一下 [備份]。
    2. 按一下 [雲端備份]。
    3. 按一下畫面頂端的 [Amazon S3] 按鈕。
    4. 按一下 [建立]。
    5. 輸入雲端備份的工作名稱。
    6. 按一下 [下一頁]。
    7. 輸入 Amazon S3 提供的下列資訊：
    • 區域：從下拉式功能表中選取您的雲端所使用區域。
    • 存取金鑰：輸入 Amazon S3 提供給您的存取金鑰。
    • 私人金鑰：輸入 Amazon S3 提供給您的私人金鑰。
    • 遠端路徑：輸入雲端的遠端路徑。這通常就是貯體名稱。
    1. 按一下 [下一頁]。
    2. 在 [類型] 欄位中選取下列其中一個選項：
    • 上傳：將 My Cloud 資料備份至 Amazon S3 貯體。
    • 下載：將 Amazon S3 貯體備份至 My Cloud 伺服器。
    1. 在 [備份類型] 欄位中，從下拉式功能表中選取您要執行的備份類型。選項包括：
    • 覆寫現有檔案：覆寫目標資料夾中，名稱與來源檔案相同的檔案。
    • 完整備份：每次執行備份時，都建立不同的資料夾以包含所有的備份資料。
    • 增量備份：使用比較新的來源檔案，覆寫目標檔案。
    1. 按一下 [下一頁]。
    2. 在 [本機路徑] 欄位中，輸入 My Cloud 設備上的備份路徑。
    • 按一下 [瀏覽]，瀏覽至設備上的備份位置。
      (範例：Volume_1/backup)
    1. 按一下 啟動自動更新功能。如此會根據您建立的排程，自動更新備份。
    • 如果啟動自動更新功能：

      • 選取自動更新排程：每日、每週或每月

      • 從下拉式功能表中選取 [自動更新時間] (每日選項)。

      • 從下拉式功能表中選取 [自動更新日期和時間] (每週或每月選項)。

    1. 如果您不選取自動更新，在 [立即備份] 欄位中指明您是否要立即開始備份。
    • 選取 [是] 立即開始備份。
    1. 按一下 [下一頁]。
    2. 檢閱設定，然後按一下 [完成]。您的 Amazon S3 雲端備份隨即創建。新工作在 Amazon S3 備份頁面的 Amazon S3 佇列區段顯示。

    注意：存取 Amazon S3 雲端備份貯體可以檢視設備備份。
