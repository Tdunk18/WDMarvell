從安全模式中恢復

使用下方描述的步驟從安全模式中恢復您的設備。

    1. 在執行這些步驟前，請從 WD 支援網站 (http://support.wd.com) 下載 My Cloud 韌體，並把它保存到您電腦可以瀏覽的位置。請記住解壓韌體檔案。
    2. 請登錄您路由器的 DHCP LAN/Device Client Table（裝置客戶端表）以獲取您裝置的 IP 地址。
    3. 在瀏覽窗口中輸入裝置的 IP 地址。然後安全模式界面會出現。
    4. 瀏覽到您保存下載韌體的位置。
    5. 按一下 [套用] 加載韌體。
    6. 按一下對話方塊中的 [確定]。裝置會重新啟動。
    7. 重新啟動後，請記住要清除瀏覽器的緩存。查看瀏覽器的 [說明]是找出清除緩存的最佳辦法。
    8. 您的裝置重啟完成後，我們建議您做一次系統還原。
