開啟/關閉雲端存取

如果您是系統管理員，您可以開啟/關閉所有使用者的雲端存取 (亦即，您可以控制電腦裝置是否可存取 My Cloud™ 裝置的內容)。

使用下列步驟，可啟動/停用雲端存取。

    1. 按一下 [設定]。
    2. 按一下左窗格中的 [一般]。
    3. 在 [雲端存取>雲端服務] 欄位中，按一下，適當地開啟或關閉雲端存取：
    • 開啟：啟用所有使用者的遠端存取功能。
    • 關閉：停用所有使用者的遠端存取功能。
    1. 按一下 [設定] 更改用於雲端存取的連線類型。在 [雲端存取連線選項] 螢幕中有三個存取選項。
    2. 自動： 自動使用 UPnP（通用隨插即用）嘗試打開路由器埠。 如果成功，装置和應用程式之間建立直接連線.
    3. 手動：經由兩個所選連接埠建立連線。若無法使用其中一個連接埠，則建立轉送連線。此選項需要手動設定路由器。請參閱知識庫 Answer ID 8526，瞭解有關如何手動設定路由器的資訊。
    4. Win XP：經由連接埠 80 以及 443 建立連線。若無法使用這些連接埠，則建立轉送連線。如果您使用 Windows XP，需要此選項。
      注意：根據預設，My Cloud 装置會自動在您的行動裝置與路由器之間建立直接連線。
    1. 在 [內容資料庫] 欄位，按一下 [重建] 以重建 WDMyCloud 資料庫。
      注意：僅在您懷疑資料庫損壞的情況下執行此選項來排除故障。
    2. 按一下套用。[連線狀態] 欄位指明装置已連接。

    如需關閉單一裝置存取功能的詳細資訊，請參閱移除裝置。

    注意：「連線選項」區域的相關資訊可在設定連線選項中取得，其中說明其他不同的防火牆選項，並解釋如何改善效能。根據預設，My Cloud 會自動在您的行動裝置與路由器之間建立直接連線。

    如果行動裝置是透過轉送連線啟用的 (亦即 [連線狀態] 為 [連線 (轉送)]，您可以啟用 WD Venue 連接器的連接埠轉送來改善效能 (請參閱啟用連接埠轉送)。
