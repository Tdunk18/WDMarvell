啟用 FTP 存取

您可以啟用或停用透過「檔案傳輸通訊協定」(FTP) 在您網路內的電腦裝置之間傳輸檔案的功能。

使用下列步驟，可啟用/停用 FTP 存取：

    1. 按一下 [設定]。
    2. 按一下左窗格中的 [網路]。
    3. 在 [網路服務>FTP 存取] 區域中，按一下 以啟用或停用 FTP 存取。
    4. 閱讀訊息，然後按一下 [確定]。
    5. 按一下 [設定]。
    6. 在 [FTP 設定] 畫面中，輸入以下資訊：
    • 使用者數目上限：從下拉式功能表選擇您希望其擁有 FTP 存取權的使用者數目上限。
    • 閒置時間:輸入您希望 FTP 閒置多長時間（以分鐘為單位）後確定為逾時。
    • 連接埠:輸入用於 FTP 存取的連接埠。
    • [流量控制]:選取 [無限] 或 [自訂]。如果您選取 [自訂]，請輸入 [流量控制] 值。

    2. 在 [被動模式] 欄位選取 [預設] 或 [自訂]。如果您選取 [自訂]，請輸入 [被動模式] 值。
    3. 如果您想以 PASV（被動）模式報告外部 IP，請選取核取方塊。
    • 輸入外部 IP 位址，或按一下 [取得 IP] 讓系統輸入位址

    4. TLS:選取 [隱式 TLS] 或 [顯式 TLS] 旁邊的 [傳輸層安全性] (TLS) 核取方塊（若適用）。
    5. FXP:按一下 啟用或禁用 [文件切換式通訊協定] (FXP)。

    3. 按一下 [完成]。
