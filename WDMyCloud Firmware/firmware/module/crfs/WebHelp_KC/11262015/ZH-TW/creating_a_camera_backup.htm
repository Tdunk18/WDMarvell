建立相機備份

相機備份允許您將相機中的資料備份到 My Cloud™ 設備中。當下載完成後，使用者可導航至相機資料的儲存位置以便存取資料（如果使用者能夠存取該共享資料）。

使用以下步驟，將您的相機資料備份到 My Cloud 設備。

    1. 確保您的相機已連接到您的 My Cloud 設備。
    2. 在導航條上按一下 [備份]，即可顯示備份操作螢幕。
    3. 如果尚未選取相機備份，請按一下相機備份 。
    4. 確保「已連接的相機」區域識別出了您的相機。
    5. 在「設置」區域，輸入以下資訊以便備份相機中的資料。

      自動傳輸

      按一下觸發按鈕 ，即可將相機中的資料自動傳輸到 My Cloud 設備。

      傳輸模式

      選取您喜歡的相機資料傳輸方式。選項包括：

      • 複製：將相機中的資料複製到 My Cloud 設備。這樣可在相機中保留原始資料。

      • 移動：將相機中的文件移動到 My Cloud 設備，這樣會從相機中移除文件。

      資料夾選項

      選取該選項以便為所傳輸的相機文件確定目標路徑。

      1. 在相機資料夾選項螢幕上：

        • 傳輸資料夾：按一下「瀏覽」，進入設備上您想要傳輸資料的位置，然後按一下「OK」。

        • 資料夾名稱：從下拉式功能表中選取資料夾名稱類型。

          • 如果您選取「自訂資料夾名稱」，請在「輸入資料夾名稱」欄位中輸入資料夾名稱。

      1. 按一下 [儲存]。

    1. 在「傳輸文件」欄位中，按一下現在複製/移動即可開始傳輸文件（如果「自動傳輸」已關閉）。
    2. 當傳輸完成後，狀態列會顯示針對已連接設備的備份已完成，並且顯示完成下載的日期和時間。
