設定網路 UPS

不斷電供應系統 (UPS) 是一種装置，在主電源無法供電的情況下，至少可以在短時間內代替電池並維持連接 UPS 埠的装置正常運行。如果 My Cloud™ 装置透過 UPS 連接網路磁碟機，磁碟機在 UPS 電量消耗到指定百分比時會自動關閉。

與 My Cloud 装置的通訊使用主/從通訊協定，主要装置控制從屬装置。

如果 UPS 透過 USB 連接 My Cloud 装置，该装置自動成為 UPS 網路主要装置。其他連接 UPS 的 My Cloud 装置可以指定為 UPS 主要装置的 IP 位址。如果 UPS 電量消耗到指定百分比，所有 My Cloud 從屬装置將適當地自動關閉。請參見知識庫第 Answer ID 11852，查看 UPS 装置清單。

使用下列步驟，可設定 My Cloud™ 裝置的網路 UPS。

    1. 按一下 [設定]。
    2. 按一下左窗格中的 [網路]。
    3. 在 [網路 UPS>從屬] 區域中，按一下 以啟用動態 DNS。
    4. 輸入連接網路之主要 UPS 的 IP 位址。
    5. 按一下 [下一頁]。顯示「正在進行」螢幕，當設定更新完成後，顯示 [網路] 螢幕，[網路 UPS 從屬] 設定為 [關閉]。
