設定雲端存取

必須有 My Cloud™ 帳號，才能從智慧型手機、平板電腦或電腦遠端存取您的雲端(基本上從電腦裝置進行遠端存取是透過 My Cloud 中央伺服器進行的，它可讓您存取與您家用網路連線的 NAS 裝置)。然而，您從行動裝置 (智慧型手機與平板電腦) 遠端存取雲端的方式與從電腦的存取方式略有不同。無論使用何種裝置，每位使用者都需要有 WD My Cloud 帳號。

使用下列步驟，可設定遠端存取。

    1. 確認您的裝置已啟用雲端存取。 我要怎麼做？
    2. 按一下 [雲端存取]。
    3. 在左窗格中，按一下您要允許雲端存取的使用者。
    4. 在 [WDMyCloud.com <用戶名> 登錄] 區段中，按一下 [申請]。此操作會向用戶發送一封關於如何註冊 WD My Cloud 指南的電子郵件。
    1. 執行下列其中一項：
    • 繼續為您自己進行申請程序。 我要怎麼做？  
    • 通知使用者您已申請網路帳號，並讓使用者繼續申請程序。
