啟用 Active Directory

啟用該功能可讓您的 My Cloud™ 設備加入現有 Windows 網域。預設停用 Active Directory。

使用下列步驟啟用 Active Directory。

    1. 按一下 [設定]。
    2. 按一下左窗格中的 [網路]。
    3. 在 Windows 服務>Active Directory 欄位，按一下 啟用 Active Directory。
    4. 在 Active Directory 設定螢幕輸入下列資訊：

    2. 使用者名稱

    輸入 Active Directory 伺服器帳戶名稱。

    密碼

    輸入 Active Directory 伺服器密碼。

    網域名稱

    輸入 Active Directory 伺服器的網域名稱。

    DNS 伺服器

    輸入網域名稱系統 (DNS) 伺服器 IP 位址。
    1. 檢閱畫面資訊，然後按一下 [套用]。
