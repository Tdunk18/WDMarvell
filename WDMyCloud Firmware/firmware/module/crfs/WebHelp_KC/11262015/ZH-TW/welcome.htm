歡迎使用

您最終擁有屬於自己的雲端。把所有東西保存到一個地方，使用電腦、平板和智慧型手機從任何地方存取。My Cloud™ 儲存與目前多個操作系統兼容，如 Mac OS、Windows OS、iOS 和 Android，可與移動應用程式配合使用。

您必須知道

為了能充分運用您的個人雲端，您必須知道一些有關 My Cloud 裝置的重要事項：

    • 使用者角色

此外您應檢閱這些工作，讓您可以利用裝置的核心功能：

    • 正在準備開始

    • 智慧型手機或平板電腦存取

    • 筆記型電腦存取

    • 顯示 My Cloud 網頁介面

    • 與在您網路外部的人共享內容

    • 上傳檔案

個人雲端中的使用者角色

管理員（通常是裝置的擁有者）負責設置裝置。身為 My Cloud 裝置的擁有者，您具備特殊的使用者帳戶 (admin)，讓您擁有系統管理員權限。有了這些權限，您便可設定裝置以符合您特定的需求，並將其他使用者新增到您的個人雲端。您也有能力可確切決定哪些使用者可存取裝置。

根據預設，管理員 (admin) 使用者名稱沒有密碼。您可以遵照新增/改變使用者密碼所述步驟在任何時間建立密碼。

使用智慧型手機或平板電腦存取內容

有兩個 WD 行動應用程式可讓您從智慧型手機或平板電腦存取 My Cloud 裝置的內容：

    • WD Photos：檢視與管理相片。

    • WD My Cloud：檢視與播放相片、影片、音樂與檔案。

可從 WD 學習中心 (http://www.wd.com/setup) 或其他應用程式網站（如 Apple App Store、Google Play Store 或適用於 Android 的 Amazon Appstore）下載這些免費應用程式。

身為 My Cloud 擁有者/系統管理員的您所適用的基本工作流程：

1

設定 My Cloud 裝置可供遠端存取並申請 WDMCloud.com 帳號。做法

2

下載 WD 行動應用程式。做法

3

啟動 WD 應用程式並執行安全存取的一次性設定。做法

4

這樣就完成了！播放並檢視您的內容。

其他人的行動裝置存取工作流程

注意：當您新增使用者至 My Cloud 装置時，等於同時提供使用者的電子郵件地址給 My Cloud，以便建立 WDMyCloud.com 帳號。建立好後，會寄送一封電子郵件至新使用者的電子郵件地址，其中包含 WDMyCloud.com 的連結，讓他們繼續進行註冊程序。

非系統管理使用者的基本工作流程：

啟動 WD 應用程式並執行安全存取的一次性設定。做法

4

播放並檢視內容。

從您的網路外存取和共享內容

您可以使用 WD My Cloud 桌面應用程式，從 Mac 或 PC 存取 My Cloud 裝置的內容。My Cloud 可讓您遠端存取 WD NAS 裝置上儲存的相片、影片、音樂與檔案，WD NAS 裝置包括與您家用網路連線的 My Cloud 與 My Book Live 磁碟機。如同 WD 行動應用程式，您需要極重要的 My Cloud 帳號才能存取您的內容。

可從 http://www.wd.com/setup 下載 My Cloud 桌面應用程式。

如需分解步驟指示，請參閱從您的網路外存取和共享內容。

頂端

顯示 My Cloud 介面

My Cloud 裝置使用者介面可讓您設定與管理您的個人雲端。介面會在您 Mac 或 PC 上執行的網頁瀏覽器中顯示。 

若要顯示網頁介面：

    1. 開啟瀏覽器視窗。
    2. 在網址列中輸入 http://<裝置名稱>；
      範例：http://wdmycloud4100
      注意：如果 URL 無法顯示網頁介面，請嘗試輸入此 URL：My Cloud DL2100：http://<裝置名稱>.local
