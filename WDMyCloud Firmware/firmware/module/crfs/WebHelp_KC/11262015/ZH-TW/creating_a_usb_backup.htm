建立背面 USB 備份

使用以下步驟將 My Cloud™ 裝置上的資料備份到 USB 裝置或者將 USB 裝置資料備份到 My Cloud 裝置。

注意：此資訊只適用於 My Cloud 背面的 USB 埠。請參見使用 USB 複製按鈕建立前端 USB 埠備份瞭解使用設備前端的 USB 埠建立 USB 備份副本的資訊。

    1. 在導航條上按一下 [備份]，即可顯示備份操作螢幕。
    2. 如果尚未選取 USB 備份，請按一下 [USB 備份] 。
    3. 按一下 [建立工作]。
    4. 輸入以下資訊以建立 USB 備份工作：

      工作名稱

      輸入備份的工作名稱。

      遠端伺服器

      在下拉式功能表中指定備份方向。選項包括：

      • USB 至 NAS：將 USB 裝置上的資料備份至 My Cloud 裝置。
      • NAS 至 USB：將 My Cloud 裝置上的資料備份至 USB 裝置。     

      來源資料夾

      按一下 [瀏覽] 並選取您要備份的資料夾。

      目的地資料夾

      按一下 [瀏覽] 並選取備份的目的地資料夾。

      備份類型

      指出要執行的備份類型。

      • 複製
      • 同步

      連線時自動啟動

      連線裝置後，此選項自動開始工作。按一下開關按鈕打開或關閉選項。

       

    1. 按一下 [建立]。
    1. 在 [USB 備份工作] 區域中，按一下 [開始備份] 圖示開始備份。
      備份進度會在 [USB 備份作業] 區域顯示。
