開啟/關閉儀表板雲端存取

使用下列步驟，可啟動/停用儀表板雲端存取。

    1. 按一下 [設定]。
    2. 按一下左窗格中的 [一般]。
    3. 在 [儀表板雲端存取] 欄位，按一下 打開遠端儀表板雲端存取。
      注意：選取此選項允許您在遠端執行管理員功能。
    4. 檢閱螢幕上的資訊，輸入管理員帳戶密碼，再重新輸入一次，然後按一下 [確定]。
    • 如果您的管理員帳戶已經有關聯密碼，檢閱 [儀表板雲端存取] 螢幕，然後按一下[確定]。
      注意：如果您將 [雲端設定] 設定為 [手動]，您必須使用手動進行雲端設定時選取的埠遠端存取装置。範例：如果您將手動 HTTP 埠設定為 5040，您使用以下位址存取装置：http://<装置 IP 位址>:5040。
