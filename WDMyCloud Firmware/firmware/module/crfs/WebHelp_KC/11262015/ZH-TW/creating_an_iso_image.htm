建立 ISO 映像

ISO 映像是現有檔案系統的精確副本檔案。使用下列步驟建立 ISO 映像。

使用下列步驟建立 ISO 映像

    1. 按一下 [設定]。
    2. 按一下左窗格中的 [公用程式]。
    3. 在 [ISO 共享清單] 區域，按一下 [建立 ISO 映像]。
    4. 在 [映像大小] 欄位，從下拉式功能表選擇您要用於 ISO 映像的映像大小。
    5. 在 [映像路徑] 欄位，輸入您要映像的資料路徑或按一下 [瀏覽] 選擇您要映像的資料夾，然後按一下 [確定]。
    6. 在 [映像名稱] 欄位中輸入 ISO 映像的名稱。
    7. 按一下 [下一頁]。
    8. 在 [選取] 欄位中，按一下 [覆寫] 或 [略過]。
    • 覆寫： 如果選取 [覆寫]，同名檔案將被覆寫。
    • 略過： 如果選取 [略過]，同名檔案將被略過。
    1. 在左欄中選取您要添加到映像檔案中的資料夾和檔案，然後按一下新增>>。
    2. 選取您要從 ISO 映像中移除的資料夾和檔案，然後按一下<<移除。
    3. 按一下 [下一頁]。ISO 映像已建立。
    1. 按一下 [完成]。
