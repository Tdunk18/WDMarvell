啟用連接埠轉送

如果您雲端的遠端存取是透過轉送連線啟用的，您可以啟用連接埠轉送來改善網路效能。

注意：若要確定行動存取連線是否為轉送，請參見檢查遠端存取狀態。

重要：在您開始此程序前，您必須知道可用於您路由器的連接埠號碼有效範圍，並為 HTTP 與 HTTPS 選擇適當的連接埠。有效的連接埠號碼範圍是 1024 - 66535。一般會將連接埠 80 指派給 HTTP，將連接埠 443 指派給 HTTPS。

使用下列步驟，可啟用連接埠轉送。

    1. 按一下 [設定]。
    2. 按一下左窗格中的 [一般]。
    3. 在 [雲端存取] 區段中按一下 以開啟雲端存取 (若適用)。
    4. 按一下 [設定]。
    5. 在 [連線能力] 欄位中按一下 [手動]。
    6. 在「外部連接埠 1」欄位中輸入 HTTP 工作階段將採用的連接埠編號。
    7. 在「外部連接埠 2」欄位中輸入 HTTPS 工作階段將採用的連接埠編號。
    8. 按一下套用。
    9. 設定路由器使用您在步驟 6 和 7 中指定的連接埠。
