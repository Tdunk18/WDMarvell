建立遠端備份

遠端備份允許您把 My Cloud™ 裝置備份到另一台 My Cloud 裝置。使用下列步驟，可為您的 My Cloud 裝置建立遠端備份。

    1. 按一下 [備份]。
    2. 按一下 [遠端備份]。
    3. 按一下 [建立工作]。
    4. 輸入以下資訊以建立一個遠端備份工作：

工作名稱

輸入備份的工作名稱。

遠端伺服器

選取遠端伺服器的類型。

  • NAS 伺服器
  • My Cloud       

遠端 IP 位址

輸入遠端位置的 IP 位址。（例如：192.168.1.16）

密碼

輸入遠端備份伺服器的密碼。

SSH 使用者名稱

輸入遠端裝置的安全殼層協定 (SSH) 使用者名稱。

SSH 密碼

輸入遠端裝置的 SSH 密碼。

來源資料夾

按一下 [瀏覽] 並選取您要備份的資料夾。

目的地資料夾

按一下 [瀏覽] 並選取備份的目的地資料夾。

備份類型

指出要執行的備份類型。

  • 複製
  • 同步  

循環

如果您要使用循環功能：

  1. 按一下開關銨鈕啟用 [循環備份] 功能。
  2. 選取備份頻率：每日、每週、每月。
  3. 從下拉式功能表中選取時間 (小時，上午/下午）。

    1. 按一下 [建立]。

    2. 在 [USB 備份工作] 區域中，按一下 [開始備份] 圖示開始備份。備份進度會在 [USB 備份作業] 區域顯示。
