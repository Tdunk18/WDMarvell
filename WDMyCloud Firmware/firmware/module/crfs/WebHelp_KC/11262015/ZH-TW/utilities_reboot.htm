重新啟動 My Cloud 裝置

重要：重新啟動會中斷對目前存取 My Cloud 裝置的使用者所提供的服務。重新啟動正在進行時，請勿拔出裝置。

使用下列步驟，可重新啟動 My Cloud 裝置：

從 [儀表板]：

    1. 在頁面頂部按一下 。
    2. 選取 [重新啟動]。
    3. 檢閱確認訊息，然後按一下 [確定]。My Cloud 安全地重新啟動。

 

從 [設定]：

    1. 按一下 [設定]。
    2. 按一下左窗格中的 [公用程式]。
    3. 在 [裝置維護] 區域中按一下 [重新啟動]。
    4. 檢閱確認訊息，然後按一下 [確定]。My Cloud 安全地重新啟動。
