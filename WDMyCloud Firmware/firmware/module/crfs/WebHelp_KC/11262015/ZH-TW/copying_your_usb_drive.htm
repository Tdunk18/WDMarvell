使用 USB 複製按鈕建立前端 USB 埠備份

使用以下步驟來創建備份作業並將 USB 驅動器中的內容拷貝至 My Cloud 設備中。

    1. 將 USB 設備插入設備的前端 USB 埠（埠 1）。
    2. 點按 USB 埠上方的 [USB 拷貝] 按鈕大約 5 秒鐘。。這將為 USB 設備創建備份作業。
    3. 在導航條上按一下 [備份]，即可顯示備份操作螢幕。
    4. 如果尚未選取 USB 備份，請按一下 [USB 備份] 。
    5. [USB 備份作業] 區域會顯示您的設備備份作業（USB_埠_1）。

    6. 按一下 [開始備份] 按鈕將您的 USB 設備內容拷貝至 My Cloud 設備。備份進度會在 [USB 備份作業] 區域顯示。
      複製後，您可以在公用>USB 匯入資料夾中存取備份內容。
