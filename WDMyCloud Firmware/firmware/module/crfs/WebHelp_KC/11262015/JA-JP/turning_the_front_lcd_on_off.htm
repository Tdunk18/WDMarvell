正面 LCD のオン/オフ

電力消費を最小限にするために、My Cloud™ 筐体の正面にある LCD をオフにできます。
次の手順に従い、正面の LCD をオンまたはオフにします。

注： 正面の LCDは、My Cloud EX2、My Cloud EX4、My Cloud EX4100、および My Cloud DL4100 の機種にのみ装備されています。

    1. [設定] をクリックします。
    2. 左側のペインで、[全般] をクリックします。
    3. [省電力] セクションで、をクリックし、[LCD] オプションをオンまたはオフにします。
