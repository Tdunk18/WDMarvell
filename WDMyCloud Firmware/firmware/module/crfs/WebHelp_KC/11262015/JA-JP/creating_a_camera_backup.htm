カメラ バックアップの作成

カメラ バックアップでは、カメラにあるデータを My Cloud™ デバイスにバックアップすることができます。ダウンロードすると、共有にアクセスできるユーザーは、カメラのデータを保存した場所にナビゲートして、データにアクセスすることができます。

次の手順に従い、My Cloud デバイスにカメラをバックアップします。

    1. カメラが My Cloud デバイスに接続されていることを確認してください。
    2. ナビゲーション バーで [バックアップ] をクリックして、[バックアップ] 画面を表示します。
    3. [カメラ バックアップ] がまだ選択されていない場合は、クリックして選択します。
    4. [接続されているカメラ] のエリアで、ご使用のカメラが識別できることを確認します。
    5. [設定] エリアで次の情報を入力して、カメラのデータをバックアップします。

      自動転送

      トグル ボタン をクリックして、カメラのデータを My Cloud デバイスに自動的に転送します。

      転送モード

      カメラのデータをどのように転送したいかを選択します。次のオプションがあります：

      • [コピー]：カメラの情報を My Cloud デバイスにコピーします。このオプションでは、オリジナル データはカメラに残ります。

      • [移動]：カメラのファイルを My Cloud デバイスに移動し、カメラのファイルを削除します。

      フォルダ オプション

      このオプションを選択すると、カメラ ファイルの転送先を指定できます。

      1. [カメラ フォルダ オプション] 画面には、次の項目があります。

        • [転送フォルダ]：[参照] をクリックして、デバイスでのデータ転送先を入力し、[OK] をクリックします。

        • [フォルダ名]：プルダウン メニューから [フォルダ名] タイプを選択してください。

          • [カスタム フォルダ名] を選択した場合は、[フォルダ名入力] フィールドにフォルダ名を入力してください。

      1. [保存] をクリックします。

    1. [自動転送] がオフになっている場合は、[ファイル転送] フィールドで、[今すぐコピー/移動] をクリックしてファイル転送を開始します。
    2. 完了すると、[ステータス]フィールドに、接続されているデバイスのバックアップが完了したこと、および完了したダウンロードの日付、曜日、時刻が表示されます。
