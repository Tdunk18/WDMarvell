ISO 共有の作成

次の手順に従い、ISO 共有を作成します。

    1. [設定] をクリックします。
    2. 左側のペインで、[ユーティリティ] をクリックします。
    3. [ISO 共有リスト] 領域で、[ISO 共有の作成] をクリックします。
    4. ISO 共有に含めたい ISO ファイルを参照し、ファイルの隣にチェック印を付けます。
    5. [次へ] をクリックします。
    6. ISO 共有の説明を入力し、[次へ] をクリックします。
    7. [次へ] をクリックします。
    8. 以下のフィールドに記入します：
    • メディア サービスを有効にする： ISO 共有のメディア サービスを有効にするには、トグル ボタンをクリックします。
    • FTP アクセス： ISO 共有の FTP アクセスを有効にするには、トグル ボタンをクリックします。
    • WebDAV アクセス：ISO 共有の WebDAV アクセスを有効にするには、トグル ボタンをクリックします。
      注：デバイスの FTP アクセスまたは WebDAV アクセスが無効になっている場合は、[FTP アクセス] および [WebDAV アクセス] フィールドが灰色で表示されます。
    1. [次へ] をクリックします。
    2. トグル ボタンをクリックして、ISO 共有で NFS アクセスを有効にします。
      注：このデバイスで NFS アクセスが無効になっている場合は、[NFS アクセス] のフィールドが灰色で表示されます。
    3. [適用] をクリックします。ISO 共有が作成されます。
