FTP アクセスの有効化

ファイル転送プロトコル (FTP) を使用したネットワーク内のコンピュータ デバイス間のファイル転送を有効または無効にできます。

次の手順に従い、FTP アクセスを有効/無効にします。

    1. [設定] をクリックします。
    2. 左側のペインで、[ネットワーク] をクリックします。
    3. [ネットワーク サービス] > [FTP アクセス] 領域で をクリックし、FTP アクセスを有効または無効にします。
    4. メッセージを確認し、[OK] をクリックします。
    5. [設定] をクリックします。
    6. [FTP 設定] 画面で、次の情報を入力します。
    • 最大ユーザー：ドロップダウン メニューで、FTP アクセスを可能にしたいユーザーの最大数を選択します。
    • アイドル タイム：FTP がタイムアウトするまでのアイドル時間を分単位で入力します。
    • ポート：FTP アクセスに使用するポートを入力します。
    • フロー制御：[無限] または [カスタマイズ] を選択します。[カスタマイズ] を選択した場合は、[フロー制御] の値を入力します。
    1. [次へ] をクリックします。
    2. [パッシブ モード] フィールドで、[デフォルト] または [カスタマイズ] を選択します。[カスタマイズ] を選択すると、[パッシブ モード] の値を入力します。
    3. PASV (パッシブ) モードで外部 IP を報告したい場合は、チェックボックスを選択します。
    • 外部の IP アドレスを入力するか、[IP の取得] をクリックして、システムがアドレスを入力するようにします。
    1. [次へ] をクリックします。
    2. 次の情報を入力します。
    3. クライアント言語]： ドロップダウン メニューでクライアント言語を選択します。
    4. TLS：必要ぬ応じて、[Implicit TLS] または [Explicit TLS] 接続の横にあるトランスポート レイヤー セキュリティ (TLS) チェックボックスを選択します。
    5. FXP：をクリックして、 ファイル エクスチェンジ プロトコル (FXP) を有効または無効にします。
    1. [次へ] をクリックします。
    2. FTP アクセスをブロックしたい IP アドレスを入力し、ドロップダウン メニューで [恒久] または [臨時] を選択します。[適用] をクリックして入力した情報を保存します。
    3. [完了] をクリックします。
