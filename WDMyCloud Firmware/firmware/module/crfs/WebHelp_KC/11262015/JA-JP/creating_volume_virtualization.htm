ボリューム仮想化の作成

ボリューム仮想化では、他のネットワーク ストレージ デバイスの iSCSI ターゲットをマッピングし、My Cloud™ デバイスのボリュームとして表示できるため、拡張性が簡素化されます。

次の手順に従い、デバイスのボリューム仮想化を作成します。

    1. [ストレージ] をクリックします。
    2. [ボリューム仮想化] をクリックします。
    3. 右側のペインで、[作成] をクリックします。
    4. [デバイス IP] フィールドで、iSCSI ターゲットが存在するデバイスの IP アドレスを入力し、[次へ] をクリックします。
    5. 取得したリストから iSCSI ターゲットを選択し、[次へ] をクリックします。
    6. ターゲットに認証を追加するには、 をクリックし、[ユーザー名] と [パスワード] を入力して、[次へ] をクリックします。
    • デフォルトでは、認証は無効です。デフォルト設定を保持するには、[認証] フィールドが [オフ] になっていることを確認し、[次へ] をクリックします。
    1. リストから LUN (ローカル ユニット番号) を選択し、[次へ] をクリックします。
    2. 共有フォルダの名前を入力し、[次へ] をクリックします。
    3. ボリューム仮想化サマリーを見直し、設定が正しいことを確認して、[適用] をクリックします。仮想ボリュームが作成されます。
