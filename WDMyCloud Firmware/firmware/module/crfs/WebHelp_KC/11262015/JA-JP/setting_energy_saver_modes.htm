省電力モードの設定

次の手順に従い、My Cloud™ デバイスの電力消費量を削減します。

    1. [設定] をクリックします。
    2. [全般] をクリックします。
    3. [省電力] 領域で、次のオプションを選択し、デバイスの電力消費量を設定します。

モード

説明

ドライブ スリープ

この機能を有効にすると、ハード ドライブが一定期間にわたり動作がないとスタンバイ モードに入れるようになるため、節電することができます。ドライブ スリープは初期設定で有効になっており、約 10 分後にドライブがスリープ状態になります。

  • をクリックすると、ドライブ スリープが有効または無効になります。

LCD

この機能を有効にすると、一定期間にわたり動作がないとデバイスの LCD がスタンバイ モードに入れるようになるため、節電することができます。LCD は、初期設定で有効になっており、10 分後にスタンバイ モードになるよう設定されています。

  • をクリックすると、LCD が有効または無効になります。

このオプションは、My Cloud EX2 のみで使用できます。My Cloud EX4、My Cloud EX4100、および My Cloud DL4100 機種。

電源回復

停電のために予期しないシャットダウンが発生した後に、デバイスを自動的に再起動します。電源回復は初期設定で有効になっています。

  • をクリックすると、電源回復が有効または無効になります。

電源スケジュール

電源スケジュールを使用すると、My Cloud デバイスのシャットダウンのスケジュールを設定できます。

  • をクリックすると、電源スケジュールが有効または無効になります。
  • [設定] をクリックして電源スケジュールをカスタマイズします。
  • 各曜日にデバイスをアクティブ状態にしたい時刻を選択します。
  • [保存] をクリックします。

 

Web アクセス タイムアウト

デバイスから自動的にタイムアウトするまでのデフォルトの時間を設定します。

  • Web アクセス タイムアウトのドロップダウン メニューで、システムのタイムアウトに使用したい時間を選択します。システムが自動的に更新されます。
