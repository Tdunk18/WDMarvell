FTP ダウンロードのスケジュール

FTP ダウンロード アプリでは、My Cloud™ への FTP ダウンロードをスケジュールできます。次の手順に従い、FTP ダウンロードをスケジュールします。

    1. [アプリ] をクリックします。
    2. [FTP ダウンロード] をクリックします。
    3. [ジョブを作成] をクリックします。
    4. [ログイン方法] フィールドで、FTP ダウンロードで使用する方法を選択します。
    • アカウント:アカウントのユーザー名とパスワードを入力します。
    • 匿名
    1. [タイプ] フィールドで、ダウンロードする情報のタイプを選択します。
    • ファイル
    • フォルダ 
    1. [URL] フィールドで、ダウンロードするファイルの URL を選択します。
      例: ftp://192.168.0.32/test.txt
    2. [保存先] フィールドでダウンロードを保存する場所を入力するか、[参照] をクリックしてダウンロードを保存する場所を選択します。
      例：Volume_1/Public/Shared Videos 
    3. ファイル名を変更する場合は、[名前の変更] フィールドで新しいファイル名を入力します。
    4. [言語] フィールドのプルダウン メニューから、FTP クライアントでサポートされる言語を選択します。
    5. 繰り返しバックアップの場合は、 をクリックします。
    • バックアップの頻度を選択します (毎日、毎週、毎月)。
    • 選択した頻度に応じて、バックアップの時刻、曜日、日付を入力します。
    1. ダウンロードを実行する日付 (mm/dd/yyyy) を入力するか、 をクリックして日付を選択します。
    2. プルダウン メニューからダウンロード時刻を選択します。
    3. [作成] をクリックします。ダウンロードが作成され、[FTP ダウンロード キュー] に追加されます。
