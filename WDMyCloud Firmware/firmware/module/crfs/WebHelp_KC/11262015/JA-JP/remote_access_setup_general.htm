クラウド アクセスの設定

スマートフォン、タブレット、またはコンピュータからクラウドにリモート アクセスするには、My Cloud™ ログインが必要です(基本的に、コンピュータ デバイスからのリモート アクセスは、ホーム ネットワークに接続された NAS デバイスにアクセスできるようにする My Cloud 中央サーバーによって有効になります)。ただし、モバイル デバイス (スマートフォンやタブレット) とコンピュータでは、クラウドにリモート アクセスする方法が若干異なります。いずれにしても、各ユーザーには WD My Cloud ログインが必要です

次の手順に従い、リモート アクセスを設定します。

    1. クラウド アクセスがデバイスで有効なことを確認します。 手順についてはこちらをご覧ください。
    2. [クラウド アクセス] をクリックします。
    3. 左側のウィンドウで、クラウド アクセスを許可するユーザーをクリックします。
    4. WDMyCloud.com ログインの <ユーザー名> セクションで、[登録] をクリックします。この作業により、WD My Cloud に登録する方法が記載されたメールがユーザーに送信されます。
    1. 次のいずれかを行います。
    • 自分で登録処理を続けます。 手順についてはこちらをご覧ください。  
    • セットアップした Web アカウントのユーザーに通知し、続きの登録処理はユーザーに実行してもらいます。
