ファイルからの My Cloud デバイスの更新

My Cloud™ デバイスを更新するために、まず Western Digital サポート サイトからファイルをダウンロードしてインストールすることが必要な場合があります。

注：この手順ではインターネット アクセスが必要です。

次の手順に従い、ファイルから手動で My Cloud デバイスを更新します。

    1. 新しいブラウザ タブまたはウィンドウを開きます。
    2. ブラウザの新しいタブ/ウィンドウで、Western Digital テクニカル サポート サイトにアクセスします。
      http://support.wdc.com
    • 新しいファームウェア更新が使用できる場合、それをコンピュータ デバイス上の場所に保存します。
    1. My Cloud の Web インターフェイスに戻ります。
    2. [設定] をクリックします。
    3. 左側のペインで、[ファームウェア更新] をクリックします。
    4. [手動更新] 領域で、[ファイルから更新] をクリックします。
    5. 手順 3 で更新を保存した場所に移動します。
    6. 更新ファイルを選択し、[開く] をクリックします。
    7. [OK] をクリックしてインストールを確認します。更新がインストールされた後、My Cloud デバイスが再起動します。
      注：My Cloud が再起動すると、現在クラウドにアクセスしているユーザーに対するサービスが中断されます。
