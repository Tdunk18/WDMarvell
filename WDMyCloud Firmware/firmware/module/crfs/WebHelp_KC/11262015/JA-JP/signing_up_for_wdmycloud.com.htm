WDMyCloud.com の登録

次の手順に従い、WDMyCloud.com で登録します。

    1. クラウド アクセスがデバイスで有効なことを確認します。手順についてはこちらをご覧ください。
    2. [クラウド アクセス] をクリックします。
    3. 左側のウィンドウで、クラウド アクセスを許可するユーザーをクリックします。
    4. WDMyCloud.com ログインの <ユーザー名> セクションで、[登録] をクリックします。この作業により、My Cloud に登録する方法が記載されたメールがユーザーに送信されます。
    5. 手順 4 で指定した電子メール アカウントを調べて、電子メール通知を確認します。
    6. 電子メールに記載されているリンクをクリックし、指示に従って登録を完了します。
