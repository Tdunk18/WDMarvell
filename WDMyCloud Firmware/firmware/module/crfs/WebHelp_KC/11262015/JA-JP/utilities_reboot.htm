My Cloud デバイスの再起動

重要：再起動すると、現在 My Cloud デバイスにアクセスしているユーザーに対するサービスが中断されます。再起動中はデバイスを切断しないでください。

次の手順に従い、My Cloud デバイスを復元します。

ダッシュボードで以下の作業を実行します。

    1. ページの一番上で をクリックします。
    2. [再起動] を選択します。
    3. 確認のメッセージを確かめた後、[OK] をクリックします。My Cloud が安全に再起動します。

 

[設定] で以下の作業を行います。

    1. [設定] をクリックします。
    2. 左側のペインで、[ユーティリティ] をクリックします。
    3. [デバイス メンテナンス] 領域で、[再起動] をクリックします。
    4. 確認のメッセージを確かめた後、[OK] をクリックします。My Cloud が安全に再起動します。
