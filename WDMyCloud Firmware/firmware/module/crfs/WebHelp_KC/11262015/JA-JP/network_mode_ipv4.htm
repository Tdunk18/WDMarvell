IPv4 ネットワーク モードの変更

デフォルトでは、ネットワーク モードは DHCP です。そのため、My Cloud™ デバイスが自動的に IP アドレスとその他の設定をネットワークから取得します。IP アドレスを手動で設定する場合は、この手順を実行してモードを変更します。

次の手順に従い、IPv4 ネットワーク モードを変更します。

    1. [設定] をクリックします。
    2. 左側のペインで、[ネットワーク] をクリックします。
    3. [ネットワーク サービス] > [IPv4 ネットワーク モード] 領域で、デバイスの一意の IPv4 アドレスを割り当てる方法を選択します。
      • 固定：固定 IP アドレスでは、接続する度に同一の IP アドレスになります。IP アドレス、サブネット マスク、ゲートウェイ IP アドレス、DNS サーバーを入力するように指示が出ます。(この情報がわからない場合は、ルーターの設定を確認してください)。
      • DHCP：DHCP クライアントでは、My Cloud デバイスがローカル DHCP サーバーから自動的に IP アドレスおよび関連した設定を取得するようになります。
        注：固定または DHCP を設定する時は、今後の参考にするために、サブネット マスク、ゲートウェイ IP アドレス、DNS サーバーを安全な場所に記録しておいてください。この情報は、My Cloud サーバーに入力した後には表示されなくなります。

    1. LAN セットアップ ウィザードを実行して、ネットワーク モードをセットアップします。
