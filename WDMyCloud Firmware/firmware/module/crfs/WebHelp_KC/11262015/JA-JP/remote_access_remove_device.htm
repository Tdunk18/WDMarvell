デバイスの削除

次の手順に従い、デバイスをクラウドから削除します。

    1. [クラウド アクセス] をクリックします。
    2. 左側のペインで、デバイスを削除するユーザーをクリックします。
    3. [クラウド デバイス] セクションで、デバイスを探します。
      ステータス情報には、WD Photos および WD My Cloud の接続が反映されています ([接続])。
    4. 次のうち必要なものを行います。
    • 対応する WD Photos の行で、削除アイコン () をクリックして、WD Photos モバイル アプリを使用したクラウドへのデバイスのアクセスを削除します。
    • 対応する WD My Cloud の行で、削除アイコン () をクリックして、WD My Cloud アプリケーションを使用したクラウドへのデバイスのアクセスを削除します。
    • モバイル デバイスに対応する行で、削除アイコン () をクリックして、My Cloud 環境からデバイスを削除します。
    1. 確認ダイアログ ボックスで、[OK] をクリックします。
