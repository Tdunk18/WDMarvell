他のユーザーのセットアップ

所有者アカウントをセットアップしてリモート アクセスを設定したら、他のユーザーをデバイスに追加できます。

主要なタスクは、新しいユーザーを追加してリモート アクセスをセットアップすることです。「ユーザー アカウントの追加」を参照してください。

クラウド管理者は次のタスクも実行できます。

    • ユーザー アカウントにパスワードを追加します。「パスワードの追加/変更」を参照してください。

    • ユーザーによる共有へのアクセスを変更する。「共有へのアクセスの変更」を参照してください。

    • ユーザー アカウントを削除します。「ユーザー アカウントの削除」を参照してください。

    • ユーザー名とパスワードを変更する。「パスワードの追加/変更」を参照してください。
