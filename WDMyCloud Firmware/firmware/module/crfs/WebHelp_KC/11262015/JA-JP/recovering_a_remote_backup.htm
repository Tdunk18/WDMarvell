リモート バックアップの復元

以下の手順でリモート サーバーに保存したデータを復元します。この手順では、リモート サーバーに保存したデータをローカル サーバーに復元します。

    1. [バックアップ] をクリックします。
    2. [リモート バックアップ] をクリックします。
    3. [リモート バックアップ ジョブ] で、表示したいジョブの隣にある [詳細] アイコンをクリックします。

    4. [詳細] 画面で [バックアップの復元] をクリックします。データの復元が開始します。
