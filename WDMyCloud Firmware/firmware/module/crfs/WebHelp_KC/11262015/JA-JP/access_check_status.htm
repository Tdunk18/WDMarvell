リモート アクセス ステータスの確認

いつでも以下を確認できます。

    • My Cloud™ デバイスがリモート アクセス向けに設定されているかどうか
    • My Cloud デバイスへのリモート アクセスが直接または中継接続で有効になっているかどうか
    • 特定のユーザーが使用する特定のモバイル デバイスによるクラウドへのリモート アクセスが有効になっているか

次の手順に従い、モバイル アクセス ステータスを確認します。

    1. [クラウド アクセス] をクリックします。
    2. 左側のペインで、ユーザー名をクリックします。
      以下の表にはモバイル アクセス フィールドおよび設定可能な値を示します。

セクション

フィールド/オプション名

値と説明

クラウド アクセス ステータス

接続ステータス

  • 無効：クラウド アクセスが無効です。クラウド アクセスを有効にするには、[設定] > [全般] > [クラウド アクセス] に移動します。

  • 接続済み (直接):モバイル アクセスは直接接続で有効です。

  • 接続済み (中継):モバイル アクセスは中継接続で有効です。

  • 失敗:My Cloud はリモート アクセス接続を確立できません。

ユーザー名のクラウド デバイス

デバイスまたはアプリ名

  • 接続済み:デバイスでは、あなたのデバイスへのリモート アクセスを行えます。

  • 待機中:デバイスのアクティブ化を待っています。

  • コードの期限切れ:ライセンス認証コードが期限切れです。

  • 無効:クラウド アクセス サービスがオフになっています。「クラウド アクセスのオン/オフ」を参照してください。

  • 接続失敗:リモート接続に失敗しました。ネットワーク接続のトラブルシューティングおよび最新のファームウェアがインストールされていることを確認して、問題の解決を試みることができます。

 

注:異なるユーザーのモバイル アクセス設定を表示する場合、左の列でその人のユーザー名をクリックします。表示はそのユーザーのクラウド デバイス情報で更新されます。
