ユーザーについて

パーソナル クラウドにアクセスするユーザーが My Cloud デバイスを使用するには、ユーザー アカウントが必要です。デバイス所有者には、管理者特権を持つ特別なユーザー アカウント (admin) が与えられます。管理者特権があると、特定のニーズに合わせてデバイスをセットアップおよび構成し、他のユーザーをパーソナル クラウドに追加できます。また、デバイスを使用するユーザーを厳密に指定できます。

デフォルトでは、admin ユーザー名にはパスワードはありません。いつでもパスワードを追加できます。

重要：最初にデバイスを設定するときには、My Cloud デバイスのセットアップ ソフトウェアまたは開始ウィザードの実行中に、管理者名を変更し、オプションのパスワードを設定できます。この名前とパスワードは、忘れないようにご注意ください。これらの資格情報を入力しないと、My Cloud インターフェイスを表示できません。

My Cloud デバイスの管理者は、以下を行うことができます。

    • ユーザーを追加する。

    • ユーザー パスワードを追加または編集する。

    • ユーザー情報を表示する。

    • ユーザーによる共有へのアクセスを変更する。  

    • ユーザーを削除する。

    • 他のユーザーのユーザー名とパスワードを変更する。
