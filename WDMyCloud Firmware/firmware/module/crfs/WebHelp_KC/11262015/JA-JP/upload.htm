My Cloud デバイスへのコンテンツのアップロード

写真、ビデオ、音楽、ファイルなどを My Cloud デバイスにアップロードするには、次のいずれかの手順を使用します。

    • Mac
    • Windows PC
    • USB

Mac からのコンテンツのアップロード

次の手順に従い、Mac から My Cloud デバイスにファイルをアップロードします。

    1. Mac のメニュー バーから [移動] > [ネットワーク] を選択します。
    2. 次のいずれかを行います。
    • Finder の左側のペインで、My Cloud デバイスの名前をダブルクリックします。
    • 右側のペインで、My Cloud デバイスの名前を探してダブルクリックします。
    1. 特定の共有の名前をダブルクリックしてフォルダを開き、既存のサブフォルダを表示します(Public は自動的に作成される共有です)。
    2. コンピュータからファイルをドラッグして目的のフォルダにドロップします。
      Time Machine および WD SmartWare Backups フォルダにはファイルをドロップしないでください。
      注：コンピュータから My Cloud デバイスにファイルをコピーした場合、これらのファイルは元の場所とドライブの 2 箇所に存在することになります。

PC からのコンテンツのアップロード

次の手順に従い、Windows PC から My Cloud デバイスにファイルをアップロードします。

    1. コンピュータで、Windows キーと R キーを同時に押して [ファイル名を指定して実行] ダイアログ ボックスを開きます。
    2. [名前] フィールドに、\\ に続けて My Cloud デバイスの名前を入力します(デフォルトの名前は、スペースを含まない wdmycloud です)。例：\\wdmycloud
    3. [OK] をクリックします。
      Windows エクスプローラまたはファイル エクスプローラが開き、右側のペインに利用可能な共有が一覧表示されます。
    4. 特定の共有の名前をダブルクリックしてフォルダを開き、サブフォルダを表示します(Public は自動的に作成される共有です)。
    5. コンピュータからファイルをドラッグしてこの共有にドロップします。
      注：コンピュータから My Cloud デバイスにファイルをコピーした場合、これらのファイルは元の場所とドライブの 2 箇所に存在することになります。

     

    USB からのコンテンツのアップロード

    USB ドライブのコンテンツを簡単にアップロードする手順については、「USB コピー ボタンで前面の USB ポートのバックアップを作成する」を参照してください。
