ユーザー情報の表示

次の手順に従い、選択したユーザー アカウントの情報を表示します。詳細には、付与されたアクセス権限が表示されます。

    1. [ユーザー] をクリックします。
    2. 左側のペインで、ユーザーを選択します。右側のペインには、次の情報が表示されます。

セクション

値と説明

ユーザー プロファイル

この領域には、次のような選択したユーザーの基本情報が表示されます。

  • ユーザー名：デバイスにログインするために使用された名前。

  • 名前：ユーザーの名前 (名)。

  • 姓：ユーザーの名前 (姓)。

  • 電子メール アドレス: ユーザーの電子メール アドレス。My Cloud にアクセスするためにも使用されます。

  • パスワード:ユーザーのパスワードを有効または無効にできます。「ユーザー パスワードの追加/変更」を参照してください。

  • グループ メンバーシップ:ユーザーが属するグループを指定します。ユーザーのグループ メンバーシップを変更できます。.

  • 割り当て:ユーザーの割り当てを表示および設定します。「ユーザー割り当ての設定」を参照してください。

共有アクセス

使用可能な共有を表示し、ユーザーが共有に対して持っているアクセスのタイプを示します。ユーザーの共有へのアクセス権を変更するには、次のオプションのいずれかをクリックします。

  •  読み取り/書き込みアクセス:共有を表示して更新できます。

  •  読み取り専用アクセス：共有を表示できますが、更新はできません。

  •  アクセスを禁止。この共有にはアクセスできません。
