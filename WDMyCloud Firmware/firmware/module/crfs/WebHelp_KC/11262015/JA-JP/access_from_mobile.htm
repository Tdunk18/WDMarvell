スマートフォンまたはタブレットからのコンテンツへのアクセス

スマートフォンまたはタブレットを使用して、My Cloud デバイスに保存されたコンテンツにアクセスできます。これを行うには、以下のタスクを完了している必要があります。「クラウド アクセスの設定」、「WD モバイル アプリのダウンロード」、および「安全なアクセス用に 1 回限りのセットアップの実行」。

次の手順に従い、スマートフォンまたはタブレットからコンテンツにアクセスします。

    1. モバイル デバイスに Wi-Fi または 3G 以上の接続があることを確認します。
    2. WD モバイル アプリ (WD My Cloud または WD Photos) を起動します。
    3. 確認メッセージが表示されたら、[今すぐ接続] をタップします。
    4. ご使用の My Cloud デバイスを選択します。
    5. アクセスする共有を選択し、目的のコンテンツに移動します。
