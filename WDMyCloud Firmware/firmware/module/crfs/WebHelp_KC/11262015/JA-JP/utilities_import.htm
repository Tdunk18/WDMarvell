設定ファイルをインポート

管理者は、以前に保存した設定ファイルをインポートできます。基本的に、このタスクは現在の設定を以前保存したバージョンに置き換えます。

重要：デフォルトのシステム設定を復元した後で設定ファイルをインポートしても、ユーザーおよびモバイル デバイスのデータは復元されません。この情報は再度追加する必要があります。インポートが完了すると、My Cloud™ デバイスが再起動するため、現在クラウドにアクセス中のユーザーに対するサービスが中断します。

次の手順に従い、システム設定をインポートします。

    1. [設定] をクリックします。
    2. 左側のペインで、[ユーティリティ] をクリックします。
    3. [システム設定] 領域で、[ファイルのインポート] をクリックします。
    4. 設定ファイルの場所を選択し、[開く] をクリックします。設定が復元されます。
