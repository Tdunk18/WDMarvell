My Cloud デバイスの自動更新

[自動更新] では、利用可能なソフトウェア更新を便利なときにチェックするようにスケジュールして、自動的にインストールするように、My Cloud™ デバイスを設定できます。これは、My Cloud ファームウェアを最も簡単に最新の状態に維持できる方法です。

注： ファームウェアの更新がインストールされた後、My Cloud デバイスが再起動する場合があります。 再起動はユーザーのクラウド アクセスに影響するため、クラウドへのユーザー アクセスが最も少なくなるときに更新が行われるようにスケジュールしてください。

注： [自動更新] を有効にすると、更新が完了したときに手動でシステムを再起動するよう指示が出ます。

次の手順に従い、My Cloud デバイスを自動的に更新します。

    1. [設定] をクリックします。
    2. 左側のペインで、[ファームウェア更新] をクリックします。
    3. [自動更新] 領域で をクリックし、自動更新を有効にします。
    4. [更新スケジュール] フィールドで、更新の日時を指定します。

    • 1 番目のドロップダウン矢印をクリックし、曜日を選択します。

    • 2 番目のドロップダウン矢印をクリックし、時刻を選択します。

    • 3 番目のドロップダウン矢印をクリックし、[午前] または [午後] を選択します。

    1. [保存] をクリックします。
