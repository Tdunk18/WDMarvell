セーフ モードからの回復

以下に記載されている手順に従って、セーフ モードからデバイスを回復します。

    1. これらの手順を実行する前に、WD サポート Web サイト (http://support.wd.com) から My Cloud ファームウェアを ダウンロードし、ご使用のコンピュータから参照できる場所に保存してください。ファームウェア ファイルは必ず解凍しておいてください。
    2. ご使用のルータの DHCP LAN/デバイス クライアント テーブルにログオンし、ご使用のデバイスの IP アドレスを入手します。
    3. デバイスの IP アドレスをブラウザのウィンドウにタイプします。セーフ モードの UI が表示されます。
    4. ダウンロードしたファームウェアを保存しておいた場所を参照します。
    5. [適用] をクリックしてファームウェアをロードします。
    6. ダイアログ ボックスで、[OK] をクリックします。デバイスが再起動します。
    7. 再起動が完了したら、必ずブラウザのキャッシュをクリアしてください。キャッシュの最適なクリア方法については、ブラウザのヘルプを参照してください。
    8. デバイスの再起動が完了したら、システムの復元を行うことをお勧めします。
