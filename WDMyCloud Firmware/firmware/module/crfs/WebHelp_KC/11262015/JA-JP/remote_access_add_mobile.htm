モバイル デバイスの追加

WD Photos アプリと My Cloud™ アプリを使用すると、スマートフォンやタブレットと同様に、モバイル デバイスからもクラウド コンテンツにアクセスできます。この手順を開始する前に、この無料アプリをモバイル デバイスにダウンロードしてください。

モバイル デバイスを追加するには:

  1. まだ行っていない場合は、クラウドですべてのユーザーのリモート アクセスを有効にします。
  2. デバイスの [ナビゲーション] バーで、[クラウド アクセス] をクリックします。
  1. 左側のペインで、クラウドへのアクセスを許可するモバイル デバイスのユーザー名をクリックします。
  1. クラウド デバイスの <ユーザー名> 領域で、[コード取得] をクリックします。My Cloud に、24 時間以内にモバイル デバイスに入力しなければならないコードが表示されます。コードを忘れないように、書き留めます。
  1. [OK] をクリックします。
  2. モバイル デバイスで WD Photos および/または WD My Cloud アプリをインストールします。
  3. ローカル ネットワークに接続していない場合は、要求があったらアクティベーション コードを入力します。
    デバイスが認証されたら、WD Photos アプリと WD My Cloud アプリを使用して、クラウド コンテンツにアクセスします。
