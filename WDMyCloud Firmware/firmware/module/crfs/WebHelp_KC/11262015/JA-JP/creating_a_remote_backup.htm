リモート バックアップの作成

リモート バックアップでは、My Cloud™ デバイスを別の My Cloud デバイスにバックアップすることができます。次の手順に従い、My Cloud デバイスのリモート バックアップを作成します。

    1. [バックアップ] をクリックします。
    2. [リモート バックアップ] をクリックします。
    3. [ジョブを作成] をクリックします。
    4. 次の情報を入力してリモート バックアップ ジョブを作成します。

ジョブ名

バックアップのジョブ名を入力します。

リモート サーバー

リモート サーバーのタイプを選択します。

  • NAS サーバー
  • My Cloud       

リモート IP アドレス

リモート ロケーションの IP アドレスを入力します (例：192.168.1.16)

パスワード

リモート バックアップ サーバーのパスワードを入力します。

SSH ユーザー名

リモート デバイスの Secure Shell (SSH) ユーザー名を入力します。

SSH パスワード

リモート デバイスの SSH パスワードを入力します。

バックアップ元フォルダ

[参照] をクリックし、バックアップしたいフォルダを選択します。

バックアップ先フォルダ

[参照] をクリックし、バックアップ先のフォルダを選択します。

バックアップ タイプ

実行したいバックアップのタイプを示します。

  • コピー
  • 同期  

繰り返し

[繰り返し] 機能を使用したい場合は、次の手順を行います。

  1. トグル ボタンをクリックして、[繰り返しバックアップ] 機能を有効にします。
  2. バックアップの頻度を次の中から選択します。毎日、毎週、毎月。
  3. ドロップダウン メニューから時刻 (時間、午前午後) を選択します。

    1. [作成] をクリックします。

    2. [USB バックアップ ジョブ] 領域で、[バックアップ開始] アイコンをクリックしてバックアップを開始します。バックアップの進捗状況が [USB バックアップ ジョブ] 領域に表示されます。
