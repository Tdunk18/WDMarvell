自宅でのコンテンツへのアクセス

自宅でコンピュータ デバイスがホーム ネットワークに接続されている場合、リモート アクセスを使用する時とは異なる方法でドライブに保存されたコンテンツにアクセスします。ホーム ネットワークに接続されている場合、以下を行うことができます。

    • スマートフォンまたはタブレットからコンテンツにアクセス
    • コンピュータからコンテンツにアクセス

重要：始める前に、「クラウド アクセスの設定」に記述された手順が完了していることを確認します。

ホーム ネットワークに接続されたモバイル デバイスからコンテンツにアクセス

    1. まだダウンロードしていない場合は、WD モバイル アプリをダウンロードします。
    2. スマートフォンまたはタブレットで、ホーム ネットワークに接続します。
      この方法がわからない場合は、お使いのスマート デバイスの説明書を参照してください。
    3. WD モバイル アプリ (WD My Cloud または WD Photos) を起動します。
    4. 確認メッセージが表示されたら、[今すぐ接続] をタップします。
    5. ご使用の My Cloud ドライブを選択します。
    6. アクセスする共有を選択し、目的のコンテンツに移動します。

ホーム ネットワークに接続されたコンピュータからコンテンツにアクセス

    1. コンピュータがインターネット接続のあるホーム ネットワークに接続されていることを確認します。
    2. まだ行っていない場合は、WD My Cloud デスクトップ アプリケーションを WD ラーニング センター (http:.wd.com/setup からダウンロードし、Mac または PC にインストールします。
    3. コンピュータで WD My Cloud デスクトップ アプリケーションを起動します。
    4. ログイン資格情報を入力します。
    5. My Cloud ドライブをダブルクリックします。
    6. 左側のペインで必要なフォルダを展開し、目的のコンテンツに移動します。
