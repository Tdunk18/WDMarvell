クラウド バックアップについて

クラウド バックアップでは、インターネットやさまざまなモバイル デバイス経由でアクセスできるリモート バックアップを作成できます。My Cloud™ デバイスでは、次のクラウド サービスを使用し、リモート バックアップを作成します。

    • ElephantDrive:ElephantDrive は、安全なリモート自動ファイル バックアップを行うクラウド バックアップ サービスです。「ElephantDrive クラウド バックアップの有効化」を参照してください。
    • Amazon S3:Amazon Simple Storage Service (S3) はオンライン ファイル ストレージ Web サービスであり、いつでも Web 上のあらゆる場所から、無制限の量のデータを格納および取得できます。「Amazon S3 クラウド バックアップの作成」を参照してください。
