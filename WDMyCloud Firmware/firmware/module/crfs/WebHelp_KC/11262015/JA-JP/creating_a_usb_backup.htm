USB バックアップの作成

以下の手順に従い、My Cloud™ デバイスのデータを USB デバイスにバックアップしたり、USB デバイスのデータを My Cloud デバイスにバックアップしたりします。

注：この情報は、My Cloud の背面にある USB ポートのみに適用します。デバイスの前面の USB ポートを使用して USB バックアップ コピーを作成する方法については、「USB コピー ボタンで前面の USB ポートのバックアップを作成する」を参照してください。

    1. ナビゲーション バーで [バックアップ] をクリックして、[バックアップ] 画面を表示します。
    2. [USB バックアップ] がまだ選択されていない場合は、それをクリックします。
    3. [ジョブを作成] をクリックします。
    4. 次の情報を入力して USB バックアップ ジョブを作成します。

      ジョブ名

      バックアップのジョブ名を入力します。

      リモート サーバー

      プルダウン メニューからバックアップの方向を指定します。以下のオプションがあります。

      • USB から NAS：USB デバイスのデータを My Cloud デバイスにバックアップします。
      • NAS から USB： My Cloud デバイスのデータを USB デバイスにバックアップします。     

      バックアップ元フォルダ

      [参照] をクリックし、バックアップしたいフォルダを選択します。

      バックアップ先フォルダ

      [参照] をクリックし、バックアップ先のフォルダを選択します。

      バックアップ タイプ

      実行したいバックアップのタイプを示します。

      • コピー
      • 同期

      接続時に自動起動する

      このオプションでは、デバイスを接続したときにジョブが自動的に開始します。トグル ボタンをクリックして、このオプションのオン/オフを切り換えます。

       

    1. [作成] をクリックします。
    1. [USB バックアップ ジョブ] 領域で、[バックアップ開始] アイコンをクリックしてバックアップを開始します。
      バックアップの進捗状況が [USB バックアップ ジョブ] 領域に表示されます。
