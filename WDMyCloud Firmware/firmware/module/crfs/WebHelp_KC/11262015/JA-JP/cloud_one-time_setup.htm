安全なアクセス用に 1 回限りのセットアップを実行する

この手順ではスマートフォンまたはタブレットと My Cloud™ デバイスの間で安全な接続を確立する方法を説明します。以下の特定のモバイル デバイスに応じた手順を使用してください。

    • Android スマートフォンまたはタブレット

    • iPhone または iPad

    • Windows Phone またはタブレット

    • Windows または Mac デスクトップ

Android スマートフォンまたはタブレット用に安全なアクセスを取得

    1. 以下を実行してスマートフォンまたはタブレットを接続します。

    • モバイル デバイスで [設定] をタップします。

    • [ワイヤレスとネットワーク] をタップします。

    • [Wi-Fi 設定] をタップします。

    • ホームネットワークを選択します。

    • ネットワーク パスワードを入力して [接続] をタップします。

    1. WD アプリを起動します。
    2. [今すぐ接続] をタップします。
    3. My Cloud デバイスの名前をタップします。
    4. ドロップダウンの矢印をタップして、ユーザー名を選択します。
    5. 次のいずれかを行います。

    • パスワードがユーザー用に設定されている場合、そのパスワードを入力して [ログイン] をタップします。

    • パスワードがユーザー用に設定されていない場合、[ログイン] をタップします。

    1. (任意) ご自分のコンテンツにアクセスする場合、共有名をタップして目的のファイルに移動します。

iPhone または iPad 用に安全なアクセスを取得

    1. 以下を実行してスマートフォンまたはタブレットを接続します。

    • モバイル デバイスで [設定] をタップします。

    • [Wi-Fi] をタップします。

    1. [Wi-Fi 設定] をタップします。

    2. ホームネットワークを選択します。

    3. ネットワーク パスワードを入力して [参加] をタップします。

    1. WD アプリを起動します。
    2. [今すぐ接続] をタップします。
    3. My Cloud デバイスの名前の隣にあるプラス記号 (+) をタップします。
    4. 目的のユーザー名をタップします。
      パスワードがユーザー用に設定されている場合、そのパスワードを入力します。
    5. (任意) ご自分のコンテンツにアクセスする場合、共有名をタップして目的のファイルに移動します。

Windows スマートフォンまたはタブレット用に安全なアクセスを取得

    • 以下を実行してスマートフォンまたはタブレットを接続します。

    • コンピュータ デバイスで [設定] をタップします。

    • [Wi-Fi] をタップします。

    • ホームネットワークを選択します。

    • ネットワーク パスワードを入力して [完了] をタップします。

    1. WD アプリを起動します。
    2. [今すぐ接続] をタップします。
    3. My Cloud ドライブの名前をタップします。
    4. ドロップダウンの矢印をタップして、ユーザー名を選択します。
      パスワードがユーザー用に設定されている場合、そのパスワードを入力します。
    5. (オプション) ご自分のコンテンツにアクセスする場合、以下を実行します。

    • デバイス画面で My Cloud ドライブの名前をタップします。

    • 共有をタップし、目的のコンテンツに移動します。

Windows または Mac デスクトップまたはノートブック用に安全なアクセスを取得

    1. デスクトップ アプリ用の WD My Cloud をインストールした後で、デスクトップに表示されるショートカットをダブルクリックし、ネットワークのすべての WD デバイスを表示します。 アプリのインストールについては、「リモート アクセスの基本」を参照してください。
    2. セキュリティ アラート情報が表示される場合があります。この画面が再表示されないようにするには、[理解します] チェック ボックスをオンにします。
    3. [理解します] をクリックすると、ネットワークのデバイスが表示されます。
    4. 接続先のデバイスをクリックします。この場合は、My Cloud デバイスです。[サインイン] 画面が表示されます。
    5. My Cloud デバイスのパスワードを入力 (前の手順で設定した場合) し、[サインイン] をクリックします。[開始] 画面が表示されます。
    6. 詳細情報を表示するには、[WD ラーニング センター] リンクをクリックします。
    7. [開始] をクリックすると、My Cloud デバイスの共有および SmartWare または Time Machine ファイルが表示されます。
    8. My Cloud デバイスに保存されたコンテンツにリモート アクセスして管理する方法については、オンライン ラーニング センターの My Cloud ダウンロード パネルにある [コンピュータを使用したアクセス] リンクをクリックします。
