ElephantDrive クラウド バックアップの有効化

ElephantDrive は、安全なリモート自動ファイル バックアップを行うクラウド バックアップ サービスです。有効にすると、ElephantDrive が自動的にデバイスと同期されます。

次の手順に従い、ElephantDrive を使用したクラウド バックアップをを有効にします。

    1. [バックアップ] をクリックします。
    2. [クラウド バックアップ] をクリックします。
    3. 画面の上部にある [ElephantDrive] ボタンをクリックします。
    4. をクリックします。ElephantDrive クラウド バックアップがオンになります。
    5. [登録] をクリックします。
    6. 次の情報を入力し、[登録] をクリックします。
    • 電子メール アドレス:ElephantDrive から情報を受け取るための電子メール アドレスを入力します。
    • パスワード：新しいアカウントのパスワードを入力します。
    • パスワードの確認：パスワードを再入力します。

ElephantDrive クラウド バックアップが有効になりました。
