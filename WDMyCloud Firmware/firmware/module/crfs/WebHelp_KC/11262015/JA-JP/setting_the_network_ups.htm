ネットワーク UPS の設定

無停電電源供給装置 (UPS) は、主電源が停電になっても少なくとも短時間にわたり UPS ポートに接続されているデバイスの動作を可能にする、電池で動作するデバイスです。My Cloud™ デバイスを UPS 経由でネットワーク ドライブに接続すると、UPS が特定の割合まで消耗すると、ドライブが自動的にシャットダウンします。

My Cloud デバイスとの通信は、マスター デバイスがスレーブ デバイスをコントロールする「マスター/スレーブ プロトコル」で行われます。

UPS が My Cloud デバイスに USB で接続されている場合は、デバイスが自動的に UPS ネットワーク マスターになります。同じ UPS に接続されているその他の My Cloud デバイスは、UPS マスターの IP アドレスにポイントできます。UPS が特定の割合まで消耗すると、My Cloud のすべてのスレーブ ドライブが自動的に正しくシャットダウンされます。UPS デバイスのリストについては、ナレッジ ベースの回答 ID 11852 を参照してください。

次の手順に従い、My Cloud™ デバイスのネットワーク UPS をセットアップします。

    1. [設定] をクリックします。
    2. 左側のペインで、[ネットワーク] をクリックします。
    3. [ネットワーク UPS] > [スレーブ] 領域で をクリックし、ダイナミック DNS を有効にします。
    4. ネットワークに接続された UPS マスターの IP アドレスを入力します。
    5. [次へ] をクリックします。「進行中」画面が表示され、設定が更新されると [ネットワーク] 画面が表示され、[ネットワーク UPS スレーブ] がオフに設定されていることが表示されます。
