通知設定

My Cloud™ デバイスは、ファームウェア更新、ファームウェアのインストール成功、システム シャットダウンなどの様々なイベントについての通知を提供します。通知はページの一番上にある [アラート] 領域に表示され、デバイスの設定によってはメールでも提供されます。

[通知] ページでは、通知を受信するユーザーのメール アドレスを最大 5 人までセットアップできます。また、モバイル デバイスの SMS アラート メッセージのセットアップ、通知するアラート レベルの設定も行えます。

電子メール通知

  1. ナビゲーション バーで [設定] アイコンをクリックし、左ペインの [通知] をクリックします。
  2. [アラート電子メール] フィールドで、トグル ボタンをクリックして [オン] にします。
  3. [設定] をクリックします。
  4. [アラート通知] 画面で、スライダ バーを使用してメールで受信したいアラート レベルを選択します。
  • [重大のみ]：重大なアラートのみを、指定されたメール アドレスに送信します。
  • [重大および警告]：重大なアラートと警告を、指定されたメール アドレスに送信します。
  • [すべて]：すべてのアラート (情報、重大、警告) を、指定されたメール アドレスに送信します。
  1. [新しい電子メール アドレス] をクリックします。
  2. アラート メールを受信したいメール アドレスを入力して、[保存] をクリックします。
  3. 手順 5～6 を繰り返して、最大 5 つのメール アドレスを入力します。
  4. [テスト電子メールの送信] をクリックして、入力したメール アドレスを検証します。
  5. [OK] をクリックして、検証メールが届いたかどうか確認します。

 

SMS 通知を有効にする

注：SMS メッセージの送信要件について、SMS サービス プロバイダーに確認します。一部の電話会社では、SMS/テキスト メッセージを電子メールで送る必要があります。

  1. ナビゲーション バーで [設定] アイコンをクリックし、左ペインの [通知] をクリックします。
  2. SMS を有効にするには、トグル ボタンを [オン] にします。
  3. [設定] をクリックします。
  4. [SMS 設定] 画面で、SMS プロバイダの名前を入力します。
  5. 「Hello world」という内容のメッセージで、SMS URL を入力します。
  • この SMS URL は、セットアップのみに使用されます。設定の完了後は、SMS メッセージは送信されません。SMS サービス プロバイダから URL フォーマットの条件についての情報を入手します。以下のパラメータが必要です：ユーザー名、パスワード、送信先の電話番号、メッセージの内容。
  1. [次へ] をクリックします。
  2. ドロップダウン メニューから、各 SMS パラメータのカテゴリーを選択します。
  3. [完了] をクリックします。これで、SMS 通知のセットアップが完了しました。
