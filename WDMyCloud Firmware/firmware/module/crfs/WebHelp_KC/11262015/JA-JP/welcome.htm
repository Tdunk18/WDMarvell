ご自分のクラウドを使用できるようになりました。ファイルなどをすべて 1 ヶ所に保存し、コンピュータ、タブレット、スマートフォンなど、どこからでもアクセスできます。My Cloud™ クラウド ストレージは、Mac OS、Windows OS、iOS、Android などの複数のオペレーティング システムに、モバイル アプリ経由で対応しています。

知っておく必要のあること

パーソナル クラウドを最大限に活用するには、My Cloud デバイスについていくつかの重要なことを理解しておくことが必要です。

    • ユーザーの役割

さらに、デバイスの中核機能を活用できるように、以下のタスクを確認しておく必要があります。

    • 使用開始

    • スマートフォンまたはタブレット アクセス

    • ラップトップ アクセス

    • My Cloud の Web インターフェイスの表示

    • ネットワーク外部の他のユーザーとのコンテンツの共有

    • ファイルのアップロード

パーソナル クラウドのユーザーの役割

管理者 (典型的にはデバイスの所有者) は、デバイスのセットアップの責任者です。My Cloud デバイスの所有者には、管理者特権を持つ特別なユーザー アカウント (admin) が与えられます。管理者特権があると、特定のニーズに合わせてデバイスをセットアップおよび構成し、他のユーザーをパーソナル クラウドに追加できます。また、ユーザーがドライブでアクセスできる対象を厳密に指定できます。

デフォルトでは、管理者 (admin) ユーザー名にはパスワードはありませんが、「ユーザー パスワードの追加/変更」に記載されている手順で、いつでも作成できます。

My Cloudの使用開始

デバイスの接続とセットアップを行う方法については、次のいくつかの情報源を使用できます。

    • クイック インストレーション ガイド：クイック インストレーション ガイドは、デバイスに搭載されており、デバイスの接続方法、使用開始方法についての情報を提供します。

    • WD Learning Center (http://www.wd.com/setup)：WD のオンライン ラーニング センターには、パーソナル クラウド ストレージを最大限に活用していただくために役立つ、最新情報、使用手順、ソフトウェアなどが掲載されています。.

    • My Cloud ユーザー マニュアル (support.wdc.com)：My Cloud ユーザー マニュアルでは、デバイスの接続、セットアップ、使用方法について詳しく解説しています。

スマートフォンまたはタブレットからのコンテンツへのアクセス

次の 2 つの WD モバイル アプリでは、スマートフォンまたはタブレットから My Cloud デバイスのコンテンツにアクセスできます。

    • WD Photos:写真を表示して管理します。

    • WD My Cloud:写真、ビデオ、音楽、およびファイルを表示および再生します。

これらの無料アプリは WD ラーニング センター (http://www.wd.com/setup)、Apple App Store、Google Play、Amazon Appstore for Android などの他のアプリ サイトから入手することができます。

モバイル デバイス アクセスのワークフロー (管理者向け)

My Cloud の所有者/管理者のための基本的なワークフローは次のとおりです。

1

リモート アクセス用にMy Cloud デバイスを設定し、wdmycloud.com ログインに登録します。操作手順

2

WD モバイル アプリをダウンロードします。操作手順

3

WD アプリを起動し、安全なアクセス用に 1 回限りのセットアップを実行します。操作手順

4

Voila!コンテンツを再生および表示します。

モバイル デバイス アクセスのワークフロー (他のユーザー向け)

注:ユーザーを My Cloud に追加するときには、電子メール アドレスを My Cloud に登録し、WDMyCloud.com ログインを作成します。ログイン情報が作成されると、新しいユーザーの電子メール アドレスに電子メールが送信されます。この電子メールには、登録処理を続行するための WDMyCloud.com へのリンクが記載されています。

管理者以外のユーザーの基本的なワークフローは次のとおりです。

1

WDMyCloud.com からのメッセージに含まれるリンクをクリックし、登録処理を続けます。

2

WD モバイル アプリをダウンロードします。操作手順

3

WD アプリを起動し、安全なアクセス用に 1 回限りのセットアップを実行します。操作手順

4

コンテンツを再生および表示します。

ネットワーク外からコンテンツのアクセスおよび共有を行う

Mac または PC から My Cloud デバイスのコンテンツにアクセスするには、 WD My Cloud デスクトップ アプリケーションを使用します。WD My Cloud を使用すると、ホーム ネットワークに接続された My Cloud や My Book Live ドライブなどの WD NAS デバイスに保存されている写真、ビデオ、音楽、ファイルにリモート アクセスできます。WD モバイル アプリと同様に、コンテンツにアクセスするにはすべての重要な My Cloud ログインが必要です。

WD My Cloud デスクトップ アプリケーションは、http://www.wd.com/setup からダウンロードできます。

詳細な手順については、「ネットワーク外からコンテンツのアクセスおよび共有を行う」を参照してください。

トップ

My Cloud のインターフェイスの表示

My Cloud デバイスのユーザー インターフェイスでは、個人向けクラウドをセットアップして管理できます。このインターフェイスは、Mac または PC で実行する Web ブラウザに表示されます。 

Web インターフェイスを表示するには:

    1. ブラウザ ウィンドウを開きます。
    2. アドレス バーに、http://<デバイス名> と入力します。
      例:http://wdmycloud4100
      注：この URL で Web インターフェイスが表示されない場合は、次の URL を入力してみてください。My Cloud DL2100:http://<デバイス名>.local
