Aktivere FTP-tilgang

Du kan aktivere eller deaktivere overføring av filer mellom databehandlingsenhetene i nettverket ditt via File Transfer Protocol (FTP).

Bruk følgende steg for å aktivere/deaktivere FTP-tilgang:

    1. Klikk på Innstillinger.
    2. Klikk på Nettverk i venstre rute.
    3. I området Nettverkstjeneste>FTP-tilgang, klikk  for å aktivere eller deaktivere FTP-tilgang.
    4. Les meldingen, og klikk deretter på OK.
    5. Klikk på Konfigurer.
    6. Angi følgende informasjon på skjermen for FTP-innstillinger:
    • Maksimalt antall brukere: Velg det maksimale antallet brukere du ønsker du skal ha FTP-tilgang fra nedtrekksmenyen.
    • Inaktiv: Angi mengden tid, i minutter, du ønsker FTP skal stå inaktiv før det skjer en tidsutkobling.
    • Port: Angi porten som skal brukes for FTP-tilgang.
    • Flytkontroll: Velg enten Ubegrenset eller Tilpass. Dersom du velger Tilpass, angi en verdi for Hastighetsregulering.
    1. Klikk på Neste.
    2. I feltet Passivmodus, velg enten Standard eller Tilpass. Dersom du velger Tilpass, angi verdier for passivmodus.
    3. Velg avmerkingsboksen dersom du ønsker å rapportere ekstern IP i PASV (Passiv)-modus.
    • Angi den eksterne IP-adressen, eller klikk på Få IP for å få systemet til å angi adressen.
    1. Klikk på Neste.
    2. Angi følgende informasjon:
    3. Klientspråk: Velg klientspråket fra nedtrekksmenyen.
    4. TLS: Velg Transport Layer Security (TLS) ved siden av enten implisitte TLS eller eksplisitte TLS-tilkoblinger, hvis det gjelder.
    5. FXP: Klikk  for å aktivere eller deaktivere File Exchange Protocol (FXP).
    1. Klikk på Neste.
    2. Angi alle IP-adresser du ønsker å nekte FTP-tilgang. og velg deretter enten Permanent eller Midlertidig fra nedtrekksmenyen. Klikk Bruk for å lagre dine innganger.
    3. Klikk på Fullfør.
