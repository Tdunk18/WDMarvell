Aktivere portviderekobling

Hvis ekstern tilgang til nettskyen din er aktivert gjennom en reléforbindelse, kan du forbedre nettverksytelsen ved å aktivere portviderekobling.

Merk: For å avgjøre om din mobile tilgangsforbindelse er gjennom relè, se  Kontrollere ekstern tilgangsstatus.

Viktig: Før du starter på denne prosedyren må du vite gyldig rekkevidde av portnumrene du kan bruke for ruteren din, og velge passende porter for HTTP og HTTPS. Det gyldige portnummerområdet er 1024–66535. Porter som vanligvis tildeles, er 80 for HTTP og 443 for HTTPS.

Bruk følgende trinn for å aktivere portviderekobling.

    1. Klikk på Innstillinger.
    2. Klikk på Generelt i venstre rute.
    3. I Cloud Access-delen, klikk på for å slå på Cloud Access, om anvendbar.
    4. Klikk på Konfigurer.
    5. Gå til feltet Tilkoblingsmuligheter, og klikk på Manuelt.
    6. I ekstern port 1-feltet,skriv inn portnummeret hvor HTTP-øktene vil inntreffe.+
    7. I ekstern port 2-feltet,skriv inn portnummeret hvor HTTP-øktene vil inntreffe.
    8. Klikk på Bruk.
    9. Konfigurer ruteren slik at den bruker de portene du spesifiserte i trinn 6 og 7.
