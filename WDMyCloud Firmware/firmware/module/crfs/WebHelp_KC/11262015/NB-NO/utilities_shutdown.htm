Slå av My Cloud

Viktig: Slå av My Cloud™ -enheten forstyrrer tjenestene for brukere som for øyeblikket bruker nettskyen din.

Bruk følgende trinn for å slå av  My Cloud -enheten din:

    1. Klikk på Innstillinger.
    2. I den venstre ruten klikker du på Verktøy.
    3. I området Enhetsvedlikehold>Enhetsbryter klikker du på Slå av.
    4. Se over bekreftelsesmeldingen, klikk så på OK. Din My Cloud -enhet slår seg trygt av.
