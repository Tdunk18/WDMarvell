Se systemlogger

Systemlogger gir en liste av hendelser som inntreffer på din enhet. Bruk følgende trinn for å se systemloggene dine.

    1. Klikk på Innstillinger.
    2. I den venstre ruten klikker du på Verktøy.
    3. I systemlogg-området, klikk på se logger.
    4. På se logger-dialogen, se gjennom logginngangene for enheten. For å tilpasse loggen:
    • Velg loggnivå fra nedtrekksmenyen.
    • Velg filtrer etter-alternativet fra nedtrekksmenyen.
    • Få tømme loggen, klikk på Tøm.
    1. Når du har mottatt loggen, klikk på Lukk.
