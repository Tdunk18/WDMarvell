Opprette en ekstern sikkerhetskopi

Ekstern sikkerhetskopiering lar deg sikkerhetskopiere din My Cloud™ -enhet til en annen My Cloud -enhet. Bruk følgende steg for å opprette en ekstern sikkerhetskopiering for din My Cloud -enhet.

    1. Klikk på Sikkerhetskopier.
    2. Klikk på Ekstern sikkerhetskopiering.
    3. Klikk på Opprett jobb.
    4. Angi følgende informasjon for å opprette en ekstern sikkerhetskopieringsjobb::

Jobbnavn

Skriv inn et jobbnavn på sikkerhetskopien.

Ekstern server

Velg type ekstern server

  • NAS-server
  • My Cloud       

Ekstern IP-adresse

Angi IP-adressen til den eksterne plasseringen. (Eksempel: 192.168.1.16)

Passord

Angi passordet for server for ekstern sikkerhetskopi.

SSH-brukernavn

Angi Secured Shell protocol (SSH)-brukernavn for den eksterne enheten.

SSH-passord

Angi SSH-passord for den eksterne enheten.

Kildemappe

Klikk på Bla gjennom og velg mappen du ønsker å sikkerhetskopiere.

Målmappe

Klikk på Bla gjennom og velg målmappen for din sikkerhetskopiering.

Type sikkerhetskopi

Indiker type sikkerhetskopiering du ønsker å utføre.

  • Kopier
  • Synkroniser  

Gjentakelse

Dersom du vil bruke Gjentakelse-funksjonen:

  1. Klikk på vippe-knappen for å aktivere funksjonen Gjentakelse av sikkerhetskopiering.
  2. Velg hvor ofte det skal sikkerhetskopieres: Daglig, ukentlig, månedlig.
  3. Velg en tid (time, AM/PM) fra rullegardinmenyen.

    1. Klikk på Opprett.

    2. Klikk på Start sikkerhetskopiering-ikonet for å starte din sikkerhetskopiering i området for USB Sikkerhetskopieringsjobber. Fremgangen til sikkerhetskopieringen vises i området for USB-sikkerhetskopijobber.
