Opprette en USB sikkerhetskopi

Bruk følgende steg for å sikkerhetskopiere dine data på din My Cloud™ -enhet til en USB-enhet eller for å sikkerhetskopiere dine USB enhetsdata til din My Cloud -enhet.

Merk: Denne informasjonen gjelder kun for USB-porter på baksiden av din My Cloud Se Opprette sikkerhetskopier foran på USB-port med USB kopieringsknapp for informasjon om å opprette en USB sikkerhetskopi ved å bruke USB-porten foran på enheten din.

    1. I navigasjonsfeltet, klikk på Sikkerhetskopieringer for å vise skjermen Sikkerhetskopiering.
    2. Klikk på USB Sikkerhetskopieringer, hvis ikke allerede valgt.
    3. Klikk Opprett jobb.
    4. Angi følgende informasjon for å opprette en USB sikkerhetskopieringsjobb:

      Jobbnavn

      Skriv inn et jobbnavn på sikkerhetskopien.

      Ekstern server

      Indiker retningen for din sikkerhetskopiering fra rullegardinmenyen. Alternativer inkluderer:

      • USB til NAS: Sikkerhetskopierer dataene på din USB-enhet til My Cloud -enheten.
      • NAS til USB: Sikkerhetskopierer dataene på din My Cloud -enhet til en USB-enhet.     

      Kildemappe

      Klikk på Bla og velg mappen du ønsker å sikkerhetskopiere.

      Målmappe

      Klikk på Bla gjennom og velg målmappen for din sikkerhetskopiering.

      Type sikkerhetskopi

      Indiker typen sikkerhetskopiering du ønsker å utføre.

      • Kopier
      • Synkroniser

      Automatisk start når den er tilkoblet

      Dette alternativet starter automatisk jobben når enheten er tilkoblet. Klikk på vippeknappen for å slå alternativet av eller på.

       

    1. Klikk på Opprett.
    1. I området for USB Sikkerhetskopieringsjobber, klikk på Start sikkerhetskopiering -ikonet for å starte din sikkerhetskopiering.
      Fremgangen til sikkerhetskopieringen vises i området for USB-sikkerhetskopijobber.
