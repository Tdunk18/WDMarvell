Utføre engangsoppsett for sikker tilgang

Denne prosedyren forklarer hvordan du etablerer en sikker tilkobling mellom din smarttelefon eller nettbrett og din My Cloud™ -enhet. Følg instruksjonene som gjelder for din mobile enhet:

    • Android Smartphone eller Tablet  

    • iPhone eller iPad

    • Windows Phone eller Tablet

    • Windows eller Mac Desktop

Få sikker tilgang for Android-smarttelefon eller -nettbrett

    1. Koble til smarttelefonen eller nettbrettet på følgende måte:

    • Trykk på Innstillinger på den mobile enheten.

    • Trykk på Trådløst og nettverk.

    • Trykk på Wi-Fi-innstillinger.

    • Velg hjemmenettverket ditt.

    • Angi nettverkspassordet, og trykk på Koble til.

    1. Start en WD-app.
    2. Trykk på Koble til.
    3. Trykk på navnet til din My Cloud -enhet.
    4. Trykk på rullegardinpilen og velg brukernavnet ditt.
    5. Gjør ett av følgende:

    • Hvis det er definert et passord for denne brukeren, angir du det og trykker på Innlogging.

    • Dersom et passord ikke ble satt for brukeren, trykk Innlogging.

    1. (Valgfritt) Hvis du vil ha tilgang til noe av innholdet ditt, trykker du på navnet på en delt ressurs og navigerer til ønsket fil.

Få sikker tilgang for iPhone eller iPad

    1. Koble til smarttelefonen eller nettbrettet på følgende måte:

    • Trykk på Innstillinger på den mobile enheten.

    • Trykk på Wi-Fi.

    1. Trykk på Wi-Fi-innstillinger.

    2. Velg hjemmenettverket ditt.

    3. Skriv inn nettverkspassordet ditt og trykk Bli med.

    1. Start en WD-app.
    2. Trykk på Koble til.
    3. Trykk på plusstegnet (+) ved siden av navnet til My Cloud -enheten.
    4. Trykk på ønsket brukernavn.
      Hvis det er definert et passord for brukeren, angir du det.
    5. (Valgfritt) Hvis du vil ha tilgang til noe av innholdet ditt, trykker du på navnet på en delt ressurs og navigerer til ønsket fil.

Få sikker tilgang for Windows smarttelefon eller nettbrett

    • Koble til smarttelefonen eller nettbrettet på følgende måte:

    • Trykk på Innstillinger på databehandlingsenheten.

    • Trykk på Wi-Fi.

    • Velg hjemmenettverket ditt.

    • Angi nettverkspassord og trykk på Fullfør.

    1. Start en WD-app.
    2. Trykk på Koble til.
    3. Trykk på navnet til My Cloud -stasjonen.
    4. Trykk på rullegardinpilen og velg et brukernavn.
      Hvis det er definert et passord for brukeren, angir du det.
    5. (Valgfritt) Hvis du vil ha tilgang til noe av innholdet ditt, gjør du følgende:

    • På skjermen for Enheter, trykk på navnet på din My Cloud -stasjon.

    • Trykk på en delt ressurs og naviger til ønsket innhold.

Få sikker tilgang for Windows eller Mac skrivebord eller bærbar datamaskin

    1. Etter du installerer appen WD My Cloud for skrivebord, dobbeltklikk på snarveien som nå vises på skrivebordet ditt for å vise alle WD-enhetene i nettverket. For mer informasjon om hvordan du installerer appen, se Grunnleggende om ekstern tilgang.
    2. Systemet kan vise informasjon om sikkerhetsvarsler. For å unngå å vise denne skjermen igjen, velg avmerkingsboksen Jeg forstår.
    3. Klikk jeg forstår for å vise enhetene på nettverket.
    4. Klikk på enheten du ønsker å koble til, i dette tilfellet My Cloud -enheten. En innloggingsskjerm vises.
    5. Angi passordet til My Cloud -enheten, dersom du satte opp en tidligere, og klikk på Logg på. Skjermen for Komme i gang vises.
    6. Klikk på WD Learning Center-koblingen for mer informasjon.
    7. Klikk på Start for å vise My Cloud -enhetens delte ressurser og alle SmartWare eller Time Machine sikkerhetskopifiler
    8. For å finne ut hvordan du får tilgang til og administrerer innholdet lagret på din My Cloud -enhet eksternt, klikk på Få tilgang ved å bruke din i online Learning Centers My Cloud nedlastingspanel:
