Adding a Mobile Device

You can access cloud content from a mobile device (like a smart phone or tablet) using the My Albums and My Cloud apps. Before you begin the procedure described here, download these free apps to your mobile device.

To add a mobile device:

  1. If you haven't already done so, enable remote access to your cloud for all users.
  2. Click the Cloud Access icon.
  1. In the left pane, click the name of the user whose mobile device you want to grant cloud  access
  1. Click Get Code. My Cloud displays a code you must enter in to your mobile device within 24 hours. Write down this code to ensure that you remember it.
  1. Click Finish.
  2. On your mobile device, enter the activation code.
    Once your device is activated, access your cloud content through the My Albums and My Cloud apps.
