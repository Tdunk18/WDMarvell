Enabling Active Directory

Enabling Active Directory allows your My Cloud to join an existing Windows Domain.

Use the following steps to enable Active Directory.

    1. Click Settings.
    2. On the General screen, scroll down to the Services area.
    3. Click .
    4. Click Configure.
    5. Review the Enable Active Directory, then click Enable.
