About Cloud Access

Cloud access is essentially access to your personal cloud (that is, the My Cloud device) from a computing device (e.g., smartphone, tablet, or computer). For an overview of the ways you can access your content remotely, see Remote Access Basics.

As the administrator, you can complete the following cloud access tasks:

    • Set up cloud  access  
    • Turn on/off remote access
    • Determine how devices will access your drive (setting Static, DHCP, or Manual connections)  
    • Check remote access status 
    • Improve performance through port forwarding
