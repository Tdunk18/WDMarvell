Enabling Port Forwarding

If remote access to your cloud is enabled through a relay connection, you can improve network performance by enabling port forwarding.
How can I tell if my mobile access connection is relayed?

Click Settings.

Click General.

Under Cloud Access, ensure that Connection Status reads, "Connected (Relay connection established)".

Important: Before you begin this procedure, you must know the valid range of port numbers you can use for your router and select appropriate ports for HTTP and HTTPS. The valid port number range is 1024 - 66535. Commonly assigned ports are 80 for HTTP and 443 for HTTPS.

Use the following stops to enable port forwarding.

    1. Click Settings.
    2. In the left pane, click General.
    3. In the Cloud Access section, click to turn on Cloud Access, if applicable.
    4. Click Configure.
    5. In the Connectivity field, click Manual.
    6. In the External Port 1 field, enter the number of the port through which HTTP sessions will occur.
    7. In the External Port 2 field, enter the number of the port through which HTTPS sessions will occur.
    8. Click Apply.
    9. Configure your router to use the ports you specified in steps 6 and 7.
