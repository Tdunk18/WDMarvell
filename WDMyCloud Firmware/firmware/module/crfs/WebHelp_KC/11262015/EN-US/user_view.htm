Viewing User Information

Use the following steps to view information about the selected user account. Details include access privileges that have been granted.

    1. Click Users.
    2. In the left pane, select a user. The right pane displays the following information.:

Section

Values and Descriptions

User Profile

This area displays basic user information about the selected user including:

  • User Name: The name used to log into the device.

  • First Name: The user's first name.

  • Last Name: The user's last name.

  • Email Address:The email address for the user. This is also used for access to My Cloud.

  • Password: Allows you to enable/disable the user password. See Adding/Changing a User Password

  • Group Membership: Identifies which groups the user belongs to. You can modify the user's group membership here. See Modifying a Group Membership.

  • Quotas: View and set user quotas here. See Setting User Quotas.

Share Access

Displays the available shares and indicates the type of access the user has to those shares. Change a user's access to a share by clicking one of the following options:

  •  Read/Write access: Can view and update the share.

  •  Read-only access: Can view the share but can't update it.

  •  Deny access. User has no access to this share.
