Updating the My Cloud Device Automatically

You can set up the  My Cloud device to conveniently check for available firmwaren updates and install them automatically. This is the easiest way of ensuring the My Cloud firmware is current.

NOTE: Once a software update is automatically installed, the My Cloud device reboots. Since a reboot will affect users' ability to access the cloud, you should schedule the updates to occur at times when the likelihood of users accessing the cloud are minimal.

Use the following steps to update the  My Cloud device automatically:

    1. Click Settings.
    2. In the left pane, click Firmware Update.
    3. In the Auto Update area, click  to enable Auto Updates.
    4. In the Update Schedule field, schedule the day and time of the update:

    • Click the first drop-down arrow and select the day of the week.

    • Click the second drop-down arrow and select a time.

    • Click the third drop-down arrow and select AM or PM.
