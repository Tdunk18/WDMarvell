About Media Settings

From the Media view, you can accomplish tasks regarding the DLNA media server and the iTunes media player.

What is DLNA?

Enable content sharing to various devices.

DLNA Media Server

  • Turn on/off the DLNA media server  
  • Block/unblock access for a DLNA device  
  • Rescan/rebuild the DLNA server database  

iTunes Server

  • Turn on/off the iTunes server  
  • Rescan the iTunes database
