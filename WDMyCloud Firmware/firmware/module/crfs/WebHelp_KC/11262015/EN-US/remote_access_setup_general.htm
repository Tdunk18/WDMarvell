Setting Up Cloud Access

Accessing your cloud remotely from a smartphone, tablet, or computer requires a MyCloud.com login. (Essentially, remote access from a computing device is enabled through the MyCloud.com central server that gives you access to the NAS devices connected to your home network.) However, the manner in which you access the cloud remotely from mobile devices (smartphones and tablets) and computers differ slightly. Regardless, each user must have a MyCloud.com login.

Use the following steps to set up remote access.

    1. Ensure that Cloud access is enabled on your device.  How do I do that?

      Click Settings.

      Click General.

      Under Cloud Access, ensure that Cloud Service is set to On.

       

       

    2. Click Cloud Access.
    3. In the left pane, click the user to whom you'd like to allow cloud access.
    4. In the MyCloud.com Account for <user name> section, click Sign Up. This sends an email to the user with instructions on how to sign up for MyCloud.com
    1. Do one of the following:
    • Continue the sign-up process for yourself. How do I do that?

      Click Sign Up.

      In the prompt, enter <new email> address.

      Check email to confirm activation. If no email, click the Resend icon.

       

       

    • Notify the user that you have set up a web account and have them continue with the sign-up process.
      NOTE: There is another way of giving a mobile device remote access to the My Cloud drive. See Granting Remote Access to a Mobile Device (Alternate Method).
