Managing  Web Files

The Web File View is an app that allows you to view and manage the files on your My Cloud device.

Use the following steps to manage your web files.

    1. Click Apps.
    2. Click Web File Viewer. The file volumes available on your device display in the right pane.

Viewing Web Files

Your volumes, folders and files display in the right pane of the Web File Viewer. To view the contents of any volume or folder, double-click that folder.

Managing Web Files

The available actions you can perform on your files and folders display in the upper right of the  Web File Viewer. Select one of the following actions to perform on your web files:

Icon

Function

Description

Copy

This action allows you to copy a selected file to another location. Use the following steps to copy a file to a new location:

    1. Select the file you’d like to copy.
    2. Click .
    3. Select the destination of your file copy.
    4. Click OK. Your file is copied to the selected destination.

Move

This action allows you to move a file from one location to another. Use the following steps to move a file to a new location:

    1. Select the file you’d like to move.
    2. Click .
    3. Select the destination where you’d like to move your file.
    4. Click OK. Your file is moved to the selected destination.

Upload

This action allows you to upload a file from your computer or other device to your My Cloud server.
Use the following steps to upload a file to your device:

    1. In the right pane of the Web File Viewer, select the destination for your uploaded file.
    2. Click .
    3. On the Upload screen, click the Select File icon
    4. Navigate to and select the file you’d like to upload to your device.
    5. Click Open.
    6. Click Upload.
    • If you’d like to upload more files, perform steps 3 and 4 until all of your files are uploaded.
    1. Click Close. Your uploaded files display in the destination folder.

Download

This action allows you to download  a file from your My Cloud device to your computer or other devices.
Use the following steps to download a file to your computer or other devices:

    1. In the right pane of the Web File Viewer, select the file you’d like to download.
    2. Click .
    3. The selected file is downloaded to your computer or other devices.

Rename

This action allows you to rename a selected file on your My Cloud device Use the following steps to rename a file.

    1. In the right pane of the Web File Viewer, select the file you’d like to rename.
    2. Click .
    3. In the Rename field, rename your file.
    4. Click Save. Your file is renamed.

Delete

This action allows you to delete a selected file from your My Cloud device. Use the following steps to delete a file.

    1. In the right pane of the Web File Viewer, select the file you’d like to delete.
    2. Click .
    3. Click OK to verify that you want to delete the selected file. The selected file is deleted from your device.

Refresh

This action refreshed the file list, displaying any new files that were added since the last refresh or since you accessed the file list.

  • Click  . Your files are refreshed.
