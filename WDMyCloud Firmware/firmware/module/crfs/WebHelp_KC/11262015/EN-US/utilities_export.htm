Exporting System Settings

As an administrator, you can save your current My Cloud settings to a configuration file. You would commonly do this prior to restoring the My Cloud device to its original default setting (How to). Once the  My Cloud device is reset to its factory defaults, you can import your saved settings (How to).

Use the following steps to export your system settings:

    1. Click Settings.
    2. In the left pane, click Utilities.
    3. In the System Configuration area, click Save Config File.
