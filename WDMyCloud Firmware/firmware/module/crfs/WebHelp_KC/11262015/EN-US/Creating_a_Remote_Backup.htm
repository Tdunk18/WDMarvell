Creating a Remote Backup

Remote backup allows you to back up your My Cloud device to another server on or outside of your network.  Use the following steps to create a remote backup for your My Cloud server.

    1. Click Backups.
    2. Click Remote Backup.
    3. Click the Create Remote Backup button.
    4. Place a check mark in the box next to the data youâ€™d like to backup, then click Next.
    5.  In the Remote IP field, enter the IP address of the remote backup device.
      (Example 192.168.1.16)
    6. In the Password field, enter the password for the remote backup device.
    7. Click Next.
