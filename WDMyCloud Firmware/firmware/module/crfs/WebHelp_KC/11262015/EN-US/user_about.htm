About Users

Anyone accessing your personal cloud must have a user account to use the My Cloud device. As the My Cloud owner, you have a special user account (admin) that gives you administrator privileges. With these privileges, you can set up and configure the device to your specific needs and add other people to your personal cloud. You have the power to determine exactly who uses your device.

As a default, the admin user name does not have a password. You can add one at anytime.

Important: When initially setting up your device, while running the My Cloud device setup software or Getting Started wizard, you have the opportunity to change the administrator name and set an optional password  Make sure you keep track of this name and password.
