Changing Access to a Share

You can make changes to a user's access permissions from either the Users screen or the Shares screen. Use the following steps to change a user's access permissions for a selected share.

Change share access from the User Screen

    1. Click Users.
    2. In the left pane, select a user.
    3. In the Shares Access area, change the user's access by clicking the appropriate icon that corresponds to a share:
    • Read/write access
    • Read-only access
    • Deny access

 

Change share access from the Shares Screen

    1. Click Shares.
    2. In the left pane, select a share.
    3. Ensure that the Public Access option is off. Public access allows everyone to view that share.
