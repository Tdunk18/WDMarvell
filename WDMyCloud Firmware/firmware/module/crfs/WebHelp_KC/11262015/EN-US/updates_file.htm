Updating the My Cloud Device from a File

In some cases, you might be required to update the My Cloud device by first downloading a file from the Western Digital support site and then installing it.

NOTE: This procedure requires Internet access.

Use the following steps to update the My Cloud device manually from a file:

    1. Open a new browser tab or window.
    2. In the new tab/window, point your browser to the Western Digital technical support site:
      http://products.wd.com/?id=MyCloud&type=support
    • If a new firmware update is available, save it to a location on your computing device.
    1. Return to the My Cloud web interface.
    2. Click Settings.
    3. In the left pane, click Firmware Update.
    4. In the Manual Update area, click Update From File.
    5. Navigate to the location in which you saved the update in Step 3.
    6. Select the update file, and then click Open.
    7. Click OK to confirm the installation. Following installation of the update,the  My Cloud device reboots.
      NOTE: When My Cloud reboots, service for any users currently accessing the cloud will be interrupted.
