랩톱에서 내용 액세스

WD My Cloud 데스크톱 응용프로그램은 랩톱에서 My Cloud 장치에 저장된 내용에 액세스할 수 있는 방법을 제공합니다. 이 방법을 사용하려면 인터넷에 연결되어 있어야 하며 wdmycloud.com에 로그인해야 합니다(원격 액세스 설정 참조).

다음 단계를 사용하여 랩톱에서 콘텐츠에 액세스합니다.

    1. WD 교육 센터(http:/www./wd.com/setup/)에서 WD My Cloud 데스크톱 응용 프로그램을 다운로드하여 Mac 또는 PC에 설치합니다.
    2. 네트워크에 연결합니다(유선 또는 무선 연결).
    3. 컴퓨터에서 WD My Cloud 데스크톱 응용프로그램을 실행합니다.
    4. 로그인 자격 증명을 입력합니다.
    5. My Cloud 장치를 두 번 클릭합니다.
    6. 왼쪽 창에서 필요에 따라 폴더를 확장하여 공유할 내용을 표시합니다.
    7. 오른쪽 창에서 보거나 재생할 항목을 두 번 클릭합니다.
      중요: 컴퓨터와 브라우저의 구성 방식과 필요한 미디어 보기/재생 프로그램의 설치 여부에 따라 파일을 보거나 재생하지 못할 수도 있습니다.
