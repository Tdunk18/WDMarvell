IPv4 네트워크 모드 변경

기본적으로 네트워크 모드는 DHCP입니다. 즉, My Cloud™ 장치가 네트워크에서 IP 주소와 기타 설정을 자동으로 가져옵니다. IP 주소를 수동으로 설정하려면 다음 절차를 완료하여 모드를 변경하십시오.

다음 단계를 사용하여 IPv4 네트워크 모드를 변경합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 네트워크를 클릭합니다.
    3. 네트워크 서비스>IPv4 네트워크 모드 영역에서 장치의 고유한 IPv4 주소 할당 방법을 선택합니다.
      • 정적: 정적 IP 주소는 연결할 때마다 동일한 IP 주소를 갖도록 허용합니다. IP 주소, 서브넷 마스크, 게이트웨이 IP 주소 및 DNS 서버 관련 메시지가 표시됩니다. (이러한 정보를 모르는 경우, 라우터 설정을 확인하십시오.).
      • DHCP: DHCP 클라이언트를 통해 My Cloud 장치는 IP 주소 및 관련 설정을 로컬 DHCP 서버에서 자동으로 가져옵니다.
        참고: 정적 또는 DHCP를 구성하는 경우 나중에 참조할 수 있도록 서브넷 마스크, 게이트웨이 IP 주소, DNS 서버를 안전한 곳에 기록하십시오. My Cloud 서버에 입력된 후 이 정보는 표시되지 않습니다.

    1. LAN 설치 마법사로 네트워크 모드를 설정합니다.
