iSCSI 클라이언트 활성화

iSNS(Internet Storage Name Service) 프로토콜은 iSNS 서버와 iSNS 클라이언트가 상호작용하기 위해 사용됩니다. iSNS 클라이언트(초기자라고도 함)는 이더넷 네트워크에서 스토리지 장치(대상이라고도 함)를 검색하는 컴퓨터입니다.

다음 단계를 사용하여 iSCSI iSNS 클라이언트를 구성합니다.

참고: iSNS는 Windows 서버를 연결하는 데 주로 사용됩니다.

    1. 스토리지를 클릭합니다.
    2. iSCSI를 클릭합니다.
    3. (을)를 클릭하여 iSCSI를 활성화합니다.
    4. iSCSI iSNS 클라이언트 필드에서 (을)를 클릭하여 켭니다.
    5. 구성을 클릭합니다.
    6. iSNS 클라이언트 서버 주소(일반적으로 Windows 서버의 IP 주소)를 입력한 후 적용을 클릭합니다.
