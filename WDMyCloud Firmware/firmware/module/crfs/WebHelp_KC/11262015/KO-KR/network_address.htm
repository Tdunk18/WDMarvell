장치의 MAC 및 IP 주소 결정

다음 단계를 사용하여 My Cloud™ 장치의 MAC 주소와 IP 주소를 결정합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 네트워크를 클릭합니다. 다음의 정보가 네트워크 프로파일 섹션에 표시됩니다.

    상태

    현재 인터넷 연결 상태입니다.

    MAC 주소

    이 장치의 MAC(Media Access Control) 주소입니다.

    IPv4 IP 주소

    이 장치의 IP 버전 4 주소입니다.

    IPv4 DNS 서버

    이 장치의 IP 버전 4 DNS 서버 주소입니다.

    IPv6 DNS IP 주소

    장치의 IP 버전 6 주소입니다.

    IPv6 DNS 서버

    장치의 IP 버전 6 DNS 서버 주소입니다.
