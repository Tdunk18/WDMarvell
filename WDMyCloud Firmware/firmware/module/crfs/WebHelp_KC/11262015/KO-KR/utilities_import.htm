구성 파일 가져오기

관리자는 이전에 저장한 구성 파일을 가져올 수 있습니다. 기본적으로 이 작업은 현재 구성을 이전에 저장한 버전으로 바꿉니다.

중요: 기본 시스템 설정을 복원한 후에 구성 파일을 가져오면 사용자 및 모바일 장치 데이터가 복원되지 않습니다. 따라서 이 정보를 다시 추가해야 합니다. 가져오기가 완료되면 My Cloud™ 장치가 다시 부팅됩니다. 그 동안 클라우드에 현재 액세스 중인 사용자에 대한 서비스가 중단됩니다.

다음 단계를 사용하여 시스템 설정을 가져옵니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 유틸리티를 클릭합니다.
    3. 시스템 구성 영역에서 파일 가져오기를 클릭합니다.
    4. 구성 파일의 위치를 선택한 후 열기를 클릭합니다. 구성이 복원됩니다.
