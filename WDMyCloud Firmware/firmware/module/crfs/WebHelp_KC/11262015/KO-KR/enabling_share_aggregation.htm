공유 집계 활성화

공유 집계는 다른 My Cloud™ 시스템이나 SMB 프로토콜을 지원하는 네트워크 장치에서 공유를 통합하는 분산 파일 시스템(DFS)과 비슷합니다. 다음 단계를 사용하여 My Cloud 장치의 공유 집계를 활성화합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 네트워크를 클릭합니다.
    3. 네트워크 서비스>공유 집계 영역에서, 을(를) 클릭하여 공유 집계를 활성화합니다.
    4. 루트 폴더 이름을 입력합니다. 루트 폴더는 연결된 원격 공유의 저장소 역할을 하며 공유 집계 공유 아래에 생성됩니다.
    5. 적용을 클릭합니다.
    6. 링크 추가를 클릭합니다.
    7. 다음 정보를 입력합니다.

    로컬 폴더 이름

    루트 폴더 아래에 표시될 폴더 이름을 입력합니다.

    원격 호스트

    대상 장치의 호스트 이름 또는 IP 주소를 입력합니다.

    원격 공유

    원격 공유 이름을 입력하거나 원격 공유 폴더 가져오기를 클릭한 다음, 이 필드의 오른쪽에 있는 풀다운 메뉴에서 공유를 선택합니다.

    1. 적용을 클릭합니다.
    2. 취소를 클릭해 공유 집계 설정을 닫습니다.
