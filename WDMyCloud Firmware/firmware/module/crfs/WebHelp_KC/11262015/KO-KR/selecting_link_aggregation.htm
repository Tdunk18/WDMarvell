연결 집계 선택

이는 Bonding(접합)이라고도 하며 NAS에 있는 두 LAN 카드와 라우터에 연결되어 있는 두 케이블을 동시에 사용하는 것을 가리킵니다. 이렇게 하면 성능과 안정성이 개선됩니다. 다음 단계를 사용하여 My Cloud™ 장치에 사용할 연결 집계 유형을 선택합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 네트워크를 클릭합니다.
    3. 네트워크 서비스>연결 집계 영역에서 풀다운 메뉴의 연결 집계 유형을 선택합니다.
    4. 적용을 클릭합니다.
