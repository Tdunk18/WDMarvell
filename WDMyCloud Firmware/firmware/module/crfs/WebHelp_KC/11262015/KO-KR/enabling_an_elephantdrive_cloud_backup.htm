ElephantDrive 클라우드 백업 활성화

ElephantDrive는 파일을 원격으로 안전하게 자동으로 백업해주는 클라우드 백업 서비스입니다. 활성화되면 ElephantDrive는 자동으로 장치와 동기화합니다.

다음 단계를 사용하여 ElephantDrive를 통해 클라우드 백업을 활성화합니다.

    1. 백업을 클릭합니다.
    2. 클라우드 백업을 클릭합니다.
    3. 화면 상단에 있는 Elephant Drive 버튼을 클릭합니다.
    4. 을(를) 클릭합니다. ElephantDrive 클라우드 백업이 켜집니다.
    5. 등록을 클릭합니다.
    6. 다음 정보를 입력한 후 등록을 클릭합니다.
    • 이메일 주소: ElephantDrive에서 정보를 수신할 때 사용할 이메일 주소를 입력합니다.
    • 암호: 새 계정의 암호를 입력합니다.
    • 암호 확인: 암호를 다시 입력합니다.

이제 ElephantDrive 클라우드 백업이 활성화되었습니다.
