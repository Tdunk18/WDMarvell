하드 드라이브 정보 보기

다음 단계를 사용하여 My Cloud™ 장치의 디스크 상태를 살펴봅니다.

    1. 스토리지를 클릭합니다.
    2. 디스크 상태를 클릭합니다.
    3. 시스템 디스크 영역에서, 정보를 확인하려는 디스크 옆의 드라이브 <드라이브 #>를 선택합니다.
    4. 하드 드라이브 정보를 검토한 후 닫기를 클릭합니다. 하드 드라이브 화면에 다음 데이터가 표시됩니다.

공급업체

하드 드라이브를 제공한 공급업체.

모델

선택한 하드 드라이브의 모델 번호.

일련 번호

선택한 하드 드라이브의 일련 번호.

용량

선택한 하드 드라이브의 용량.

펌웨어 버전

선택한 하드 드라이브에서 사용 중인 현재 펌웨어 버전.
