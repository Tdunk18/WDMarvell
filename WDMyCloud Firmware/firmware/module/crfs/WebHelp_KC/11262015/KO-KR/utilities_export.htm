구성 파일 저장

관리자는 현재 My Cloud™ 장치의 설정을 구성 파일에 저장할 수 있습니다. 이 작업은 주로 My Cloud 를 원래 기본 설정으로 복원(방법)하기 전에 수행합니다. My Cloud 장치가 초기 기본값으로 다시 설정된 후에는 저장된 설정을 가져올 수 있습니다(방법).

다음 단계를 사용하여 시스템 설정을 내보냅니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 유틸리티를 클릭합니다.
    3. 시스템 구성 영역에서 구성 파일 저장을 클릭합니다. 구성 파일이 데스크탑에 저장됩니다.
