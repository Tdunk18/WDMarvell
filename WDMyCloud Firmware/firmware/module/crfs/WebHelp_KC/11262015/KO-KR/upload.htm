My Cloud 장치에 내용 업로드

My Cloud 장치에 사진, 동영상, 음악, 파일을 업로드하려면 다음 절차 중 하나를 사용합니다.

    • Mac
    • Windows PC
    • USB

Mac에서 내용 업로드

다음 단계를 사용하여 Mac의 파일을 My Cloud 장치에 업로드합니다.

    1. Mac의 메뉴 표시줄에서 이동 > 네트워크를 선택합니다.
    2. 다음 작업 중 하나를 수행합니다.
    • Finder의 왼쪽 창에서 My Cloud 장치의 이름을 두 번 클릭합니다.
    • 오른쪽 창에서 My Cloud 장치의 이름을 찾아 두 번 클릭합니다.
    1. 특정 공유의 이름을 두 번 클릭하여 폴더를 열고 모든 기존 하위 폴더를 확인합니다. 공용은 자동으로 만들어진 공유입니다.
    2. 컴퓨터에서 원하는 폴더로 파일을 끌어 놓습니다.
      Time Machine 및 WD SmartWare 백업 폴더에 파일에 끌어 놓지 마십시오.
      참고: 컴퓨터의 파일을 My Cloud 장치에 복사한 경우 이들 파일이 두 위치(원래 위치와 장치)에 상주하게 됩니다.

PC에서 내용 업로드

다음 단계를 사용하여 Windows PC의 파일을 My Cloud 장치에 업로드합니다.

    1. 컴퓨터에서 Windows 키와 R 키를 동시에 눌러 실행 대화 상자를 엽니다.
    2. 열기 필드에서 \\와 My Cloud 장치의 이름을 차례로 입력합니다. (기본 이름은 공백 없이 wdmycloud입니다.) 예: \\wdmycloud와 같이 입력하면 됩니다.
    3. 확인을 클릭합니다.
      오른쪽 창에 사용 가능한 공유가 나열된 Windows 탐색기 또는 파일 탐색기가 열립니다.
    4. 특정 공유의 이름을 두 번 클릭하여 폴더를 열고 모든 하위 폴더를 확인합니다. 공용은 자동으로 만들어진 공유입니다.
    5. 컴퓨터에서 이 공유로 파일을 끌어 놓습니다.
      참고: 컴퓨터의 파일을 My Cloud 장치에 복사한 경우 파일이 두 곳(원래 위치와 장치)에 있게 됩니다.

     

    USB에서 내용 업로드

    USB 드라이브 컨텐츠를 신속하게 업로드하는 단계는 USB 복사 버튼으로 전면 USB 포트 백업 생성을 참조하십시오.
