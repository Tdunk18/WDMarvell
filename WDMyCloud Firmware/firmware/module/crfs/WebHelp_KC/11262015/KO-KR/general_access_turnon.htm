클라우드 액세스 켜기/끄기

관리자는 모든 사용자에 대한 클라우드 액세스를 켜거나 끌 수 있습니다. (즉, 컴퓨팅 장치에서 My Cloud™ 장치의 내용에 액세스할 수 있는지 여부를 제어할 수 있습니다.)

다음 단계를 사용하여 클라우드 액세스를 활성화/비활성화합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 일반을 클릭합니다.
    3. 클라우드 액세스> 클라우드 서비스 필드에서 을 클릭하여 필요에 따라 클라우드 서비스를 켜거나 끕니다.
    • 켜기: 모든 사용자에 대해 원격 액세스를 활성화합니다.
    • 끄기: 모든 사용자에 대해 원격 액세스를 비활성화합니다.
    1. 구성을 클릭하여 클라우드 액세스에 사용하는 연결 유형을 변경합니다. 클라우드 액세스 연결 옵션 화면에는 세 가지 액세스 옵션이 있습니다.
    2. 자동: uPnP(범용 플러그 앤 플레이)를 자동으로 사용하여 라우터에서 포트 열기를 시도합니다. 포트가 열리면 장치와 앱 간에 직접 연결이 설정됩니다.
    3. 수동: 선택된 두 포트를 통해서 연결을 설정합니다. 둘 중 어느 한 포트를 사용할 수 없는 경우 릴레이 연결이 설정됩니다. 이 옵션에 대해서는 수동으로 라우터를 구성해야 합니다. 라우터를 수동으로 구성하는 방법에 대한 내용은 기술 자료 답변 ID 8526을 참조하십시오.
    4. Win XP: 포트 80~443을 통해 연결을 설정합니다. 이러한 포트를 사용할 수 없는 경우 릴레이 연결이 설정됩니다. Windows XP를 사용하는 경우 이 옵션이 필요합니다.
      참고: 기본적으로 My Cloud 장치에서는 모바일 장치와 라우터 간의 직접 연결을 자동으로 설정합니다.
    1. 콘텐츠 데이터베이스 필드에서 재구성을 클릭하여 WDMyCloud 데이터베이스를 재구성합니다.
      참고: 이 옵션은 데이터베이스 손상이 의심되는 경우에 문제 해결을 위해서만 사용하십시오.
    2. 적용을 클릭합니다. 연결 상태 필드는 사용자 장치가 연결되었음을 나타냅니다.

    단일 장치에 대한 액세스를 끄는 방법에 대한 자세한 내용은 장치 제거를 참조하십시오.

    참고: 연결 옵션 영역에 대한 정보는 연결 옵션 구성에서 확인할 수 있습니다. 여기에는 여러 방화벽 옵션 및 성능을 향상시키는 방법이 설명되어 있습니다. 기본적으로 My Cloud 에서는 모바일 장치와 라우터 간의 직접 연결을 자동으로 설정합니다.

    모바일 액세스가 릴레이 연결을 통해 활성화된 경우(즉, 연결 상태가 '연결됨(릴레이)'으로 표시된 경우) WD Venue 커넥터에서 포트 전달을 활성화하여 성능을 향상시킬 수 있습니다(포트 전달 사용 참조).
