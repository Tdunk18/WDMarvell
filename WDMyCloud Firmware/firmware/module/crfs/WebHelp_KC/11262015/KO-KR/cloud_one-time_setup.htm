안전하게 액세스하기 위한 1회 설정 수행

다음 절차에서는 스마트폰 또는 태블릿과 My Cloud™ 장치 간에 보안 연결을 설정하는 방법에 대해 설명합니다. 특정 모바일 장치에 대한 지침을 사용하십시오.

    • Android 스마트폰 또는 태블릿  

    • iPhone 또는 iPad

    • Windows Phone 또는 Tablet

    • Windows 또는 Mac 데스크톱

Android 스마트폰 또는 태블릿에 대한 보안 액세스 설정

    1. 다음 작업을 수행하여 스마트폰 또는 태블릿을 연결합니다.

    • 모바일 장치에서 설정을 누릅니다.

    • 무선 및 네트워크를 누릅니다.

    • Wi-Fi 설정을 누릅니다.

    • 홈 네트워크를 선택합니다.

    • 네트워크 암호를 입력하고 연결을 누릅니다.

    1. WD 앱을 실행합니다.
    2. 지금 연결을 누릅니다.
    3. My Cloud 장치의 이름을 누릅니다.
    4. 드롭다운 화살표를 누르고 사용자 이름을 선택합니다.
    5. 다음 작업 중 하나를 수행합니다.

    • 해당 사용자에 대해 암호가 설정된 경우 암호를 입력하고 로그인을 누릅니다.

    • 해당 사용자에 대해 암호가 설정되지 않은 경우 로그인을 누릅니다.

    1. (선택 사항) 내용에 액세스하려면 공유 이름을 누른 다음 원하는 파일로 이동합니다.

iPhone 또는 iPad에 대한 보안 액세스 설정

    1. 다음 작업을 수행하여 스마트폰 또는 태블릿을 연결합니다.

    • 모바일 장치에서 설정을 누릅니다.

    • Wi-Fi를 누릅니다.

    1. Wi-Fi 설정을 누릅니다.

    2. 홈 네트워크를 선택합니다.

    3. 네트워크 암호를 입력하고 연결을 누릅니다.

    1. WD 앱을 실행합니다.
    2. 지금 연결을 누릅니다.
    3. My Cloud 장치 이름 옆의 더하기 기호(+)를 누릅니다.
    4. 원하는 사용자 이름을 누릅니다.
      해당 사용자에 대해 암호가 설정된 경우 암호를 입력합니다.
    5. (선택 사항) 내용에 액세스하려면 공유 이름을 누른 다음 원하는 파일로 이동합니다.

Windows 스마트폰 또는 태블릿에 대한 보안 액세스 설정

    • 다음 작업을 수행하여 스마트폰 또는 태블릿을 연결합니다.

    • 컴퓨팅 장치에서 설정을 누릅니다.

    • Wi-Fi를 누릅니다.

    • 홈 네트워크를 선택합니다.

    • 네트워크 암호를 입력하고 완료를 누릅니다.

    1. WD 앱을 실행합니다.
    2. 지금 연결을 누릅니다.
    3. My Cloud 드라이브의 이름을 누릅니다.
    4. 드롭다운 화살표를 누르고 사용자 이름을 선택합니다.
      해당 사용자에 대해 암호가 설정된 경우 암호를 입력합니다.
    5. (선택 사항) 내용에 액세스하려면 다음을 수행합니다.

    • 장치 화면에서 My Cloud 드라이브의 이름을 누릅니다.

    • 공유를 누른 다음 원하는 콘텐츠로 이동합니다.

Windows/Mac 데스크톱 또는 랩톱에 대한 보안 액세스 설정

    1. 데스크톱 앱용 WD My Cloud를 설치한 후 데스크톱에 나타나 네트워크의 모든 WD 장치를 표시하는 바로 가기 를 두 번 클릭합니다. 앱 설치에 대한 정보는 원격 액세스 기본 사항을 참조하십시오.
    2. 시스템이 보안 알림에 관한 정보를 표시할 수 있습니다. 이 화면을 다시 표시하지 않으려면 이해함 확인란을 선택합니다.
    3. 이해함을 클릭하여 네트워크의 장치를 표시합니다.
    4. 연결할 대상 장치를 클릭합니다. 이 경우 My Cloud 장치를 클릭합니다. 로그인 화면이 표시됩니다.
    5. 이전에 암호를 설정한 경우 My Cloud 장치에 대한 암호를 입력하고, 로그인을 클릭합니다. 시작 화면이 표시됩니다.
    6. 자세한 정보를 얻으려면 WD 교육 센터 링크를 클릭합니다.
    7. 시작을 클릭하여 My Cloud 장치의 공유 파일과 모든 SmartWare 또는 Time Machine 백업 파일을 표시합니다.
    8. My Cloud 장치에 저장된 콘텐츠를 원격으로 액세스하고 관리하는 방법을 알아보려면 온라인 교육 센터의 My Cloud 다운로드 패널에 있는 컴퓨터 액세스 링크를 클릭합니다.
