대시보드 액세스 켜기/끄기

다음 단계를 사용하여 대시보드 클라우드 액세스를 활성화/비활성화합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 일반을 클릭합니다.
    3. 대시보드 클라우드 액세스 필드에서 을(를) 클릭하여 대시보드 액세스를 클라우드에서 원격으로 켭니다.
      참고: 이 옵션을 선택하면 관리자 기능을 원격으로 수행할 수 있습니다.
    4. 화면에서 정보를 검토하고 관리자 계정 암호를 입력 후 다시 입력한 다음, 확인을 클릭합니다.
    • 관리자 계정에 연결된 암호가 이미 존재하는 경우 대시보드 클라우드 액세스 화면을 검토한 다음 확인을 클릭합니다.
      참고: 클라우드 설정을 수동으로 설정한 경우, 수동 클라우드 설정 시 선택한 포트를 사용해 장치에 원격으로 액세스해야 합니다. 예: 수동 HTTP 포트를 5040으로 설정한 경우 장치에 액세스하기 위해 사용하는 주소는 http://<장치 IP 주소>:5040입니다.
