Apple Time Machine을 사용하여 데이터 백업

My Cloud™ 장치의 파일을 Mac 컴퓨터의 Apple Time Machine에 백업할 수 있습니다.

중요: Time Machine 백업을 복구하려면 관리자 계정에 암호를 설정해야 합니다.

다음 단계를 사용하여 Apple Time Machine에 데이터를 백업합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 일반을 클릭합니다.
    3. MAC 백업 영역에서 을 클릭하여 Time Machine 백업 옵션을 켭니다.
    4. 구성을 클릭합니다.
    5. 공유 선택 드롭다운 목록에서 백업한 파일을 둘 공유를 선택합니다.
    6. 최대 크기 필드에서 슬라이드를 이동시켜 백업에 사용할 최대 값을 표시합니다.
      참고: 초기 백업 후에 최대 백업 값을 늘리면 기존 백업에는 이 값이 아무런 영향을 미치지 않습니다. WD에서는 이 설정을 초기 백업에 대해 최대값으로 유지할 것을 권장합니다. 최초 백업이 완료된 후 최대 크기 슬라이더를 끌어서 백업 크기를 변경할 수 있습니다.
    7. 저장을 클릭합니다.

    중요: Time Machine이 파일들을 선택한 공유로 백업하기 시작하면 해당 공유에 계속해서 백업을 하는 것이 좋습니다. 공유를 변경하면 이전에 저장된 정보를 포함하지 않은 새로운 백업 파일이 생성됩니다.
