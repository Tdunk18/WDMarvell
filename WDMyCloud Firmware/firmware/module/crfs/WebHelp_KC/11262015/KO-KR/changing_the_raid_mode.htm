RAID 모드 변경

다음 단계를 사용하여 My Cloud™ 장치에서 현재 RAID 모드를 변경합니다.

경고: RAID 모드를 변경하면 모든 데이터 및 사용자 설정이 삭제됩니다. 사용자 설정 저장에 대한 내용은 시스템 설정 내보내기를 참조하십시오.

    1. 스토리지를 클릭합니다.
    2. 아직 선택하지 않은 경우 RAID를 클릭합니다.
    3. 화면 하단에 있는 RAID 모드 변경을 클릭합니다.
    4. 경고 메시지를 검토한 후 확인을 클릭합니다.
    5. My Cloud 장치에 사용할 RAID 모드를 선택합니다. 옵션은 다음과 같습니다.

모드

설명

JBOD

RAID 구성에 없지만 별도의 논리 볼륨으로 관리되는 하나 이상의 드라이브 사용.

스패닝

큰 논리 볼륨 하나를 만들기 위한 선형의 드라이브 조합.

RAID 0

성능이 개선되도록 드라이브 둘 이상의 동작을 결합하여, 데이터가 여러 하드 드라이브로 스트라이프되고, 데이터 읽기와 기록이 가속화됩니다. 하지만, 한 드라이브에서 장애가 발생할 경우 모든 데이터를 잃을 수 있습니다.

RAID 1

두 동시 데이터 사본이 별도의 하드 드라이브에 기록되어, 이 중 하나가 실패할 경우 결함 있는 하드 드라이브를 교환할 때까지 다른 하나가 데이터 읽기/쓰기를 계속할 수 있습니다.

RAID 5

3개 이상의 드라이브에 데이터를 스트라이프하고 각 드라이브의 1/4을 결함 허용 전용으로 사용하여, 우수한 성능과 보호를 제공합니다. 한 드라이브에서 장애가 발생할 경우, 장애가 있는 드라이브를 교체할 때까지 다른 드라이브가 데이터 읽기/쓰기를 계속합니다.

이 옵션은 4베이 My Cloud 장치 전용입니다.

RAID 10

데이터가 보조 디스크에 미러링된 여러 개의 주 디스크에 스트라이프로 기록되는 RAID 프로토콜. 한 드라이브에서 장애가 발생할 경우, 장애가 있는 드라이브를 교체할 때까지 다른 드라이브가 데이터 읽기/쓰기를 계속합니다.

이 옵션은 4베이 My Cloud 장치 전용입니다.

    1. 이용하려는 스토리지 옵션을 검토 및 선택하고, <스토리지 모드>로 전환 확인란을 선택한 후 다음을 클릭합니다. 드라이브 셀프 테스트가 수행됩니다.
    2. 화면 상단에 있는 경고를 검토합니다.
    3. 셀프 테스트가 완료되면, 하드 드라이브 상태를 검토하고 다음을 클릭합니다.
    4. 볼륨의 상태를 검토한 후 다음을 클릭합니다.
    5. 볼륨 슬라이더가 표시되면 선택된 RAID 모드에 전용으로 사용할 공간 용량을 선택합니다.
    6.  RAID 선택에 전체 볼륨을 사용하지 않도록 선택하려는 경우 스패닝으로 남은 디스크 공간을 구성합니다. 확인란을 선택하여 남은 디스크 공간을 스패닝으로 구성할 수 있습니다.

    1. 볼륨 슬라이더를 사용하여 선택된 RAID 모드에 전용으로 사용할 공간 용량을 선택합니다.

      참고:RAID 선택에 전체 볼륨을 사용하지 않도록 선택하려는 경우 스패닝으로 남은 디스크 공간을 구성합니다. 확인란을 선택하여 남은 디스크 공간을 스패닝으로 구성할 수 있습니다.

    1. RAID 5의 경우, "디스크4" 가 핫 스패어가 될 수 있는 RAID 5+spare를 구성할 수 있습니다. 예비 디스크 만들기 확인란을 선택해 RAID 5+Spare를 구성합니다. RAID 5+spare를 만들지 않으려면 예비 디스크 만들기의 선택을 취소하십시오.
    2. 다음을 클릭하여 계속 진행하십시오.
    3. 디스크를 인식한 다음 RAID를 자동으로 재구성하려면, 를 클릭한 후 다음을 클릭합니다.

      참고: JBOD, 스패닝, RAID 0 모드의 경우에는 이 화면이 표시되지 않습니다.
    1. 각 드라이브의 암호화 옵션을 선택합니다. 옵션은 다음과 같습니다.
    • 잠김
    • 잠금 해제됨
    1. RAID의 변경 내용을 검토하여 변경 내용이 올바른지 확인한 후 다음을 클릭합니다.
    2. 경고 화면을 검토합니다.
      경고: 변경 모드에 있는 경우 모든 데이터가 지워집니다.
    1. 마침을 클릭합니다. 요청된 하드 드라이브 변경이 시작됩니다. 이러한 변경이 진행 중일 때는 My Cloud 장치를 끄지 마십시오.
    2. 프로세스가 완료되면 마침을 다시 클릭합니다.
