네트워크 UPS 설정

UPS(Uninterruptible Power Supply)는 배터리에서 작동하며 기본 전원 장치의 전원이 중단되는 경우 UPS 포트에 연결된 장치의 실행을 잠시 동안이라도 유지하는 장치입니다. My Cloud™ 장치가 UPS를 통해 네트워크 드라이브에 연결되면, UPS 전원이 특정 퍼센트까지 소모되었을 때 드라이브가 자동으로 종료됩니다.

My Cloud 장치와의 통신은 마스터 장치가 종속 장치를 제어하는 마스터/종속 프로토콜을 사용해 이루어집니다.

UPS가 USB를 통해 My Cloud 장치에 연결된 경우 이 장치가 자동으로 UPS 네트워크 마스터가 됩니다. 이 UPS에 연결된 다른 My Cloud 장치는 UPS 마스터의 IP 주소를 나타낼 수 있습니다. UPS가 특정 퍼센트까지 방전되면 모든 My Cloud 종속 드라이브가 자동으로 정상 종료됩니다. UPS 장치 목록은 기술 자료 답변 ID 11852를 참조하십시오.

다음 단계를 사용하여 My Cloud™ 장치의 네트워크 UPS를 설정합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 네트워크를 클릭합니다.
    3. 네트워크 UPS>종속 영역에서 을(를) 클릭하여 동적 DNS를 활성화합니다.
    4. 네트워크로 연결된 UPS 마스터의 IP 주소를 입력합니다.
    5. 다음을 클릭합니다. "진행 중" 화면이 표시됩니다. 설정이 업데이트되면, 네트워크 UPS 종속이 꺼짐으로 설정된 상태로 네트워크 화면이 표시됩니다.
