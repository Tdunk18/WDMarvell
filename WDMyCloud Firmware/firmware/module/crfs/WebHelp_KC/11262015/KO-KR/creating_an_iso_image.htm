ISO 이미지 만들기

ISO 이미지는 기존 파일 시스템의 정확한 사본인 파일입니다. 다음 단계를 사용하여 ISO 이미지를 만듭니다.

다음 단계를 사용하여 ISO 이미지를 만듭니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 유틸리티를 클릭합니다.
    3. ISO 공유 목록 영역에서 ISO 이미지 만들기를 클릭합니다.
    4. 이미지 크기 필드의 풀다운 메뉴에서 ISO 이미지에 사용할 이미지 크기를 선택합니다.
    5. 이미지 경로 필드에서 이미지로 표시하려는 데이터의 경로를 입력하거나 찾아보기를 클릭해 이미지로 표시하려는 폴더를 선택한 다음 확인을 클릭합니다.
    6. 이미지 이름 필드에 ISO 이미지 이름을 입력합니다.
    7. 다음을 클릭합니다.
    8. 선택 필드에서 덮어쓰기 또는 건너뛰기를 클릭합니다.
    • 덮어쓰기: 덮어쓰기가 선택되면, 이름이 같은 파일이 덮어쓰기됩니다.
    • 건너뛰기: 건너뛰기가 선택되면, 이름이 같은 파일이 건너뛰기됩니다.
    1. 왼쪽 열의 이미지 파일에 추가하려는 폴더와 파일을 선택한 다음 추가>>를 클릭합니다.
    2. ISO 이미지에서 삭제하려는 폴더와 파일을 선택한 다음 <<제거<를 클릭합니다.
    3. 다음을 클릭합니다. ISO 이미지가 만들어집니다.
    1. 마침을 클릭합니다.
