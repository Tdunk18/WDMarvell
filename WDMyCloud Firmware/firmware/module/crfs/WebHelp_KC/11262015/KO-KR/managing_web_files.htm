웹 파일 관리

웹 파일 보기는 My Cloud™ 장치에서 파일을 보고 관리할 수 있는 앱입니다.

다음 단계를 사용하여 웹 파일을 관리합니다.

    1. 앱을 클릭합니다.
    2. 웹 파일 뷰어를 클릭합니다. 장치에서 사용할 수 있는 파일 볼륨이 오른쪽 창에 표시됩니다.

웹 파일 보기

볼륨, 폴더, 파일이 웹 파일 뷰어의 오른쪽 창에 표시됩니다. 볼륨이나 폴더의 내용을 보려면 해당 폴더를 두 번 클릭합니다.

웹 파일 관리

파일과 폴더에 수행할 수 있는 작업이 웹 파일 뷰어의 오른쪽 상단에 표시됩니다. 웹 파일에 수행할 다음 작업 중 하나를 선택합니다.

아이콘

기능

설명

복사

이 작업을 통해 선택된 파일을 다른 위치에 복사할 수 있습니다. 다음 단계를 사용하여 파일을 새 위치에 복사합니다.

    1. 웹 파일 뷰어 영역에서 복사할 파일을 선택합니다.
    2. 을(를) 클릭합니다.
    3. 파일을 복사할 대상 위치를 선택합니다.
    4. 확인을 클릭합니다. 파일이 선택된 대상 위치에 복사됩니다.

이동

이 작업을 통해 파일을 한 위치에서 다른 위치로 이동할 수 있습니다. 다음 단계를 사용하여 파일을 새 위치로 이동합니다.

    1. 이동할 파일을 선택합니다.
    2. 을 클릭합니다.
    3. 파일을 이동할 대상 위치를 선택합니다.
    4. 확인을 클릭합니다. 파일이 선택된 대상 위치로 이동됩니다.

업로드

이 작업을 통해 컴퓨터나 다른 장치의 파일을 My Cloud 서버에 업로드할 수 있습니다. 다음 단계를 사용하여 파일을 장치에 업로드합니다.

    1. 웹 파일 뷰어의 오른쪽 창에서 업로드한 파일의 대상 위치를 선택합니다.
    2. 을 클릭합니다.
    3. 업로드 화면에서 파일 선택 아이콘을 클릭합니다.
    4. 장치에 업로드할 파일을 찾아서 선택합니다.
    5. 열기를 클릭합니다.
    6. 업로드를 클릭합니다.
    • 파일을 추가로 업로드하려면 모든 파일이 업로드될 때까지 3단계와 4단계를 반복합니다.
    1. 닫기를 클릭합니다. 업로드된 파일이 대상 폴더에 표시됩니다.

다운로드

이 작업을 통해 My Cloud 장치의 파일을 컴퓨터나 다른 장치로 다운로드할 수 있습니다.
다음 단계를 사용하여 컴퓨터나 다른 장치로 파일을 다운로드합니다.

    1. 웹 파일 뷰어의 오른쪽 창에서 다운로드할 파일을 선택합니다.
    2. 을 클릭합니다.
    3. 선택된 파일이 컴퓨터나 장치로 다운로드됩니다.

이름 바꾸기

이 작업을 통해 My Cloud 장치에서 선택한 파일의 이름을 바꿀 수 있습니다. 다음 단계를 사용하여 파일 이름을 바꿉니다.

    1. 웹 파일 뷰어의 오른쪽 창에서 이름을 바꿀 파일을 선택합니다.
    2. 을 클릭합니다.
    3. 아름 바꾸기 필드에서 파일의 이름을 바꿉니다.
    4. 저장을 클릭합니다. 파일의 이름이 바뀝니다.

삭제

이 작업을 통해 My Cloud 장치에서 선택한 파일을 삭제할 수 있습니다. 다음 단계를 사용하여 파일을 삭제합니다.

    1. 웹 파일 뷰어의 오른쪽 창에서 삭제할 파일을 선택합니다.
    2. 을 클릭합니다.
    3. 확인을 클릭하여 선택된 파일을 삭제하도록 확인합니다. 선택된 파일이 장치에서 삭제됩니다.

새로 고침

이 작업은 파일 목록을 새로 고쳐서, 마지막으로 새로 고치거나 마지막으로 파일 목록에 액세스한 이후 추가된 모든 새 파일을 표시합니다.

  • 을 클릭합니다. 파일이 새로 고쳐집니다.

만들기

이 작업을 통해 선택한 공유 내에서 폴더를 만들 수 있습니다.

    1. 새 폴더를 만들려는 위치를 찾습니다.
    2. "축하합니다" 화면에서
    3. 새 폴더의 이름을 입력합니다.
    4. 저장을 클릭합니다.새 폴더가 생성됩니다..
