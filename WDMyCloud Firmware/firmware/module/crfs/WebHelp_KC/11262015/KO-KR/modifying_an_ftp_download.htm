FTP 다운로드 수정

다음 단계를 사용하여 예정된 FTP 다운로드를 수정합니다.

    1. 앱을 클릭합니다.
    2. FTP 다운로드를 클릭합니다.
    3. FTP 다운로드 대기열에서 수정할 다운로드를 선택합니다.
    4. 수정을 클릭합니다. 예정된 다운로드가 FTP 다운로드 영역에 표시됩니다.
    5. 필요한 내용을 변경한 후 저장을 클릭합니다. 업데이트된 예정된 다운로드가 FTP 다운로드 대기열에 표시됩니다.
