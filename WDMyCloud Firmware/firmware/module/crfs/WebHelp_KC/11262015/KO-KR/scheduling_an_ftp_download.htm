FTP 다운로드 일정 예약

FTP 다운로드 앱을 사용하면 My Cloud™ 로 FTP 다운로드 일정을 예약할 수 있습니다. 다음 단계를 사용하여 FTP 다운로드 일정을 예약합니다.

    1. 앱을 클릭합니다.
    2. FTP 다운로드를 클릭합니다.
    3. 작업 만들기를 클릭합니다.
    4. 로그인 방법 필드에서 FTP를 다운로드할 때 사용할 방법을 선택합니다.
    • 계정: 계정의 사용자 이름과 암호를 입력합니다.
    • 익명
    1. 유형 필드에서 다운로드할 정보의 유형을 선택합니다.
    • 파일
    • 폴더 
    1. URL 필드에 다운로드할 파일의 URL을 입력합니다.
      예: ftp://192.168.0.32/test.txt
    2. 저장 위치 필드에 다운로드를 저장할 위치를 입력하거나 찾아보기를 클릭하여 다운로드를 저장할 위치를 선택합니다.
      예: Volume_1/공개/공유된 비디오 
    3. (파일 이름을 변경하려는 경우) 이름 바꾸기 필드에 파일의 새 이름을 입력합니다.
    4. 언어 필드의 풀다운 메뉴에서 FTP 클라이언트가 지원하는 언어를 선택합니다.
    5. 반복 백업인 경우 을 클릭합니다.
    • 백업 빈도(매일, 매주, 매월)를 선택합니다.
    • 선택한 빈도에 따라 백업의 시간, 요일, 날짜를 입력합니다.
    1. 다운로드를 수행할 날짜(mm/dd/yyyy)를 입력하거나 을 클릭하여 날짜를 선택합니다.
    2. 풀다운 메뉴에서 다운로드 시간을 입력합니다.
    3. 만들기를 클릭합니다. 다운로드가 만들어져서 FTP 다운로드 대기열에 추가됩니다.
