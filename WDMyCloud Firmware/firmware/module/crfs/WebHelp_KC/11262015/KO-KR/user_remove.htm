사용자 계정 삭제

관리자는 My Cloud™ 시스템에서 사용자 계정을 제거할 수 있습니다. 사용자의 액세스를 연결된 특정 공유 또는 장치로 제한하기만 하려는 경우에는 공유 액세스 변경을 참조하십시오. 다음 단계를 사용하여 사용자 계정을 제거합니다.

    1. 사용자를 클릭합니다.
    2. 왼쪽 창에서 제거할 사용자의 이름을 클릭합니다.
    3. 사용자 제거 아이콘을 클릭합니다.
    4. 확인을 클릭합니다. 선택된 사용자 계정이 삭제되고 왼쪽 창에서 제거됩니다.
