사용자 정보 보기

다음 단계를 사용하여 선택된 사용자 계정에 대한 정보를 봅니다. 세부 정보에는 부여된 액세스 권한이 포함됩니다.

    1. 사용자를 클릭합니다.
    2. 왼쪽 창에서 사용자를 선택합니다. 오른쪽 창에 다음 정보가 표시됩니다.

섹션

값 및 설명

사용자 프로파일

이 영역에 다음 사항을 포함하여 선택된 사용자에 대한 기본 사용자 정보가 표시됩니다.

  • 사용자 이름: 장치에 로그인할 때 사용된 이름.

  • 이름: 사용자의 이름.

  • 성: 사용자의 성.

  • 이메일 주소: 사용자의 이메일 주소. 이메일 주소는 My Cloud 에 액세스할 때에도 사용됩니다.

  • 암호: 사용자 암호를 활성화/비활성화할 수 있습니다. 사용자 암호 추가/변경을 참조하십시오.

  • 그룹 멤버십: 사용자가 속한 그룹을 식별합니다. 여기서 사용자의 그룹 멤버십을 수정할 수 있습니다. .

  • 할당량: 여기서 사용자 할당량을 보고 설정합니다. 사용자 할당량 설정을 참조하십시오.

공유 액세스

사용 가능한 공유를 표시하며, 사용자가 이러한 공유에 대해 가진 액세스의 유형을 나타냅니다. 다음 옵션 중 하나를 클릭하여 사용자의 공유 액세스를 변경합니다.

  •  읽기/쓰기 액세스: 공유를 보고 업데이트할 수 있습니다.

  •  읽기 전용 액세스: 공유를 볼 수 있지만 업데이트할 수는 없습니다.

  •  액세스 거부. 사용자에게 이 공유에 대한 액세스가 없습니다.
