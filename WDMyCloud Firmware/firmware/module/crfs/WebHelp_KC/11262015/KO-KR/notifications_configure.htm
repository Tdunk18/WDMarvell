알림 구성

My Cloud™ 장치는 펌웨어 업데이트, 펌웨어 설치 성공, 시스템 종료와 같은 여러 이벤트에 대한 알림을 제공합니다. 알림은 페이지 상단의 경고 영역에 표시되며 사용자 장치의 구성 방식에 따라 이메일을 통해 수신할 수도 있습니다.

알림 페이지를 이용하면 알림을 수신할 최대 5명의 사용자 이메일 주소를 설정하고 모바일 장치용 SMS 경고 메시지를 설정하며 통지될 경고의 수준을 설정할 수 있습니다.

이메일 알림

  1. 탐색 표시줄에서 설정 아이콘을 클릭하고 왼쪽 창의 알림을 클릭합니다.
  2. 알림 이메일 필드에서 토글 버튼 켜기를 클릭합니다.
  3. 구성을 클릭합니다.
  4. 경고 알림 화면에서 슬라이더 막대를 사용해 이메일로 수신하고자 하는 경고의 수준을 선택합니다.
  • 중요만: 중요한 알림만 지정된 이메일 주소로 발송합니다.
  • 중요 및 경고: 중요한 알림과 경고 알림 모두를 지정된 이메일 주소로 발송합니다.
  • 모두: 모든 알림(정보, 중요,경고)을 지정된 이메일 주소로 발송합니다.
  1. 새 이메일을 클릭합니다.
  2. 경고 이메일을 수신할 이메일 주소를 입력하고 저장을 클릭합니다.
  3. 5 - 6단계를 반복해 최대 5개의 이메일 주소를 입력합니다.
  4. 테스트 이메일 보내기를 클릭해 입력한 이메일 주소를 확인합니다.
  5. 확인을 클릭한 다음 확인 이메일이 수신되었는지 확인합니다.

 

SMS 알림 활성화

참고: SMS 서비스 공급자의 SMS 메시지 발송 요구사항을 확인하십시오. 일부 통신사의 경우 SMS/문자 메시지를 이메일을 통해 발송하도록 요구할 수 있습니다.

  1. 탐색 표시줄에서 설정 아이콘을 클릭하고 왼쪽 창의 알림을 클릭합니다.
  2. SMS를 활성화하려면 토클 버튼 켜기를 클릭합니다.
  3. 구성을 클릭합니다.
  4. SMS 설정 화면에서 해당 SMS 공급자 이름을 입력합니다.
  5. "Hello world"로 메시지 내용이 설정된 SMS URL을 입력합니다.
  • 이 SMS URL은 설정에만 이용합니다. 설정이 완료된 후에는 SMS 메시지가 전송되지 않습니다. SMS 서비스 공급자의 URL 형식 요구 사항을 확인하십시오. 이러한 URL에는 사용자 이름, 암호, 대상 전화, 메시지 내용과 같은 매개 변수가 포함되어야 합니다.
  1. 다음을 클릭합니다.
  2. 드롭다운 메뉴에서 각 SMS 매개 변수에 해당하는 범주를 선택합니다.
  3. 마침을 클릭합니다. SMS 알림이 설정되었습니다.
