IPv6 네트워크 모드 변경

IPv6 형식은 여러 IP 네트워크의 패킷 형식과 주소 지정 구성표를 지정하는 새 IP 표준입니다. 이 옵션의 기본 설정은 끄기입니다.

다음 단계를 사용하여 IPv6 네트워크 모드를 변경합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 네트워크를 클릭합니다.
    3. 네트워크 서비스>IPv6 네트워크 모드 영역에서 장치의 IPv6 주소 할당 방법을 선택합니다.
      • 자동
      • DHCP
      • 정적
      • 끄기

    1. 선택한 후 관련 화면에서 다음 필드를 입력하고 적용을 클릭합니다.

      • IP 주소
      • 접두어 길이
      • 기본 게이트웨이
      • DNS 서버

      • 참고: 정적 또는 DHCP를 구성하는 경우 나중에 참조할 수 있도록 서브넷 마스크, 게이트웨이 IP 주소, DNS 서버를 안전한 곳에 기록하십시오. My Cloud 서버에 입력된 후 이 정보는 표시되지 않습니다.
