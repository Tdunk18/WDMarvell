LLTD 프로토콜 설정

다음 단계를 사용하여 My Cloud™ 장치에서 LLTD(링크 계층 토폴로지 검색) 프로토콜을 설정합니다. LLTD는 Microsoft Windows 시스템에서 네트워크 검색 및 진단 기능을 개선합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 일반을 클릭합니다.
    3. Windows 서비스>LLTD 영역에서 을(를) 클릭하여 LLTD 프로토콜을 활성화/비활성화합니다.
