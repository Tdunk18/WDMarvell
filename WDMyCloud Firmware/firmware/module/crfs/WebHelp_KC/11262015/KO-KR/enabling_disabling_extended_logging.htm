확장 로그 활성화

이 옵션을 이용하면 진단에서 확장 로그를 캡처할 수 있습니다.

참고: 로그 기능을 확장하기 전에 백업, 파일 작업 또는 파일 전송이 이루어지지 않도록 해야 합니다.

다음 단계를 사용하여 확장 로그를 활성화합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 유틸리티를 클릭합니다.
    3. 확장 로그 영역에서 을(를) 클릭하여 확장 로그를 활성화합니다.
    4. 확장 로그 메시지를 검토한 후 확인을 클릭합니다.
    1. 로그를 검토한 후, 닫기를 클릭합니다. 시스템이 재부팅됩니다. 완료되면 확장 로그가 활성화됩니다.
