사용자 정보

개인 클라우드에 액세스하는 모든 사용자는 사용자 계정이 있어야 My Cloud 장치를 사용할 수 있습니다. 소유자는 관리자 권한이 있는 특별한 사용자 계정(admin)을 소유합니다. 이러한 권한이 있으면 특정 필요에 맞게 장치를 설정 및 구성하고 개인 클라우드에 다른 사용자를 추가할 수 있습니다. 소유자는 또한 장치를 사용할 사용자를 정확하게 지정할 수도 있습니다.

기본적으로, admin 사용자 이름에는 암호가 없습니다. 그러나 언제든지 암호를 추가할 수 있습니다.

중요: 장치를 처음 설치할 때 My Cloud 장치 설치 소프트웨어 또는 시작 마법사를 실행하는 동안 관리자 이름을 변경하고 선택적 암호를 설정할 수 있습니다. 이 이름과 암호는 잘 보관해 두어야 합니다. 이러한 자격 증명 없이는 My Cloud 웹 인터페이스를 표시할 수 없습니다.

My Cloud 장치의 관리자는 다음 작업을 수행할 수 있습니다.

    • 사용자 추가

    • 사용자 암호 추가 또는 편집

    • 사용자 정보 보기

    • 공유에 대한 사용자 액세스 변경  

    • 사용자 제거.

    • 다른 사용자의 사용자 이름 및 암호 변경
