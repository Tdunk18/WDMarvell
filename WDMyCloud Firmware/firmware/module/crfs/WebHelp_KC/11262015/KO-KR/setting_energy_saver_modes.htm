절전 모드 설정

다음 단계를 사용하여 My Cloud™ 장치의 에너지 소비를 줄일 수 있습니다.

    1. 설정을 클릭합니다.
    2. 일반을 클릭합니다.
    3. 절전 영역에서 다음 옵션을 선택하여 장치의 에너지 소비를 설정합니다.

모드

설명

드라이브 대기

이 기능을 활성화하면 하드 드라이브를 대기 모드로 전환하여 지정된 시간 동안 작업이 없을 경우 전력을 절약할 수 있습니다. 기본적으로 약 10분 후에 대기 모드로 전환되도록 드라이브 절전이 활성화됩니다.

  • 을 클릭하여 드라이브 절전을 활성화/비활성화합니다.

LCD

이 기능을 활성화하면 장치의 LCD를 대기 모드로 전환하여 지정된 시간 동안 작업이 없을 경우 전력을 절약할 수 있습니다. 기본적으로 약 10분 후에 대기 모드로 전환되도록 LCD가 활성화됩니니다.

  • 을 클릭하여 LCD를 활성화/비활성화합니다.

이 옵션은 My Cloud EX2 장치 전용입니다. My Cloud EX4, My Cloud EX4100 및 My Cloud DL4100 모델

전원 복구

정전 때문에 갑자기 종료된 경우 장치를 자동으로 다시 시작합니다. 전원 복구는 기본적으로 활성화되어있습니다.

  • 을 클릭하여 전원 복구를 활성화/비활성화합니다.

전원 일정

전원 일정을 사용하여 My Cloud 장치 종료를 예약할 수 있습니다.

  • 을 클릭하여 전원 일정을 활성화/비활성화합니다.
  • 구성을 클릭하여 전원 일정을 사용자 지정합니다.
  • 각 요일별로 장치를 활성화할 시간을 선택합니다.
  • 저장을 클릭합니다.

 

웹 액세스 제한 시간

장치에서 자동으로 로그아웃할 기본 시간을 설정합니다.

  • 웹 액세스 제한 시간 드롭다운 메뉴에서 시스템 제한 시간에 사용할 시간을 선택합니다. 시스템이 자동으로 업데이트됩니다.
