네트워크에 My Cloud 장치 연결

다음 절차에서는 이더넷 케이블을 사용하여 My Cloud™ 장치를 홈 네트워크에 연결하는 방법에 대해 설명합니다. 이 초기 연결을 마친 후 네트워크에 무선으로 연결하도록 드라이브를 구성할 수 있습니다.

다음 단계에 따라 이더넷 케이블을 사용하여 연결합니다.

    1. My Cloud 장치 후면의 이더넷 포트에 케이블의 한쪽 끝을 연결합니다.
    1. 케이블의 다른 쪽 끝을 라우터 또는 네트워크 스위치 포트에 직접 연결합니다.
    2. 드라이브의 후면에 있는 전원 공급 장치 소켓에 어댑터의 끝을 연결합니다.
    3. 전원 어댑터를 전원 콘센트에 꽂습니다. My Cloud 장치의 전원이 자동으로 켜집니다.

중요: 커넥터를 구성하기 전에 My Cloud 장치의 전원이 완전히 켜질 때까지(약 3분 정도) 기다리십시오.

이더넷 케이블을 사용하여 My Cloud 장치를 네트워크에 연결한 후 웹 인터페이스를 사용하여 무선 연결을 설정할 수 있습니다. 연결 옵션 구성을 참조하십시오.
