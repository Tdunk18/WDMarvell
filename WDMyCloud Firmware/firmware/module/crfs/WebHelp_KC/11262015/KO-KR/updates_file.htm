파일에서 My Cloud 장치 업데이트

경우에 따라 먼저 Western Digital 지원 사이트에서 파일을 다운로드한 다음 설치하여 My Cloud™ 장치를 업데이트해야 할 수도 있습니다.

참고: 다음 절차를 수행하려면 인터넷에 연결되어 있어야 합니다.

다음 단계를 사용하여 파일에서 수동으로 My Cloud 장치를 업데이트합니다.

    1. 브라우저 탭 또는 창을 엽니다.
    2. 브라우저의 새 탭/창에서 다음 Western Digital 기술 지원 사이트로 이동합니다.
      http://support.wdc.com
    • 펌웨어 업데이트를 사용할 수 있는 경우 컴퓨팅 장치의 원하는 위치에 저장합니다.
    1. My Cloud 웹 인터페이스로 돌아갑니다.
    2. 설정을 클릭합니다.
    3. 왼쪽 창에서 펌웨어 업데이트를 클릭합니다.
    4. 수동 업데이트 영역에서 파일에서 업데이트를 클릭합니다.
    5. 3단계에서 업데이트를 저장한 위치로 이동합니다.
    6. 업데이트 파일을 선택하고 열기를 클릭합니다.
    7. 확인을 클릭하여 설치를 확인합니다. 업데이트 설치가 완료되면 My Cloud 장치가 다시 부팅됩니다.
      참고: My Cloud 가 다시 부팅되는 동안 클라우드에 현재 액세스 중인 모든 사용자에 대한 서비스가 중단됩니다.
