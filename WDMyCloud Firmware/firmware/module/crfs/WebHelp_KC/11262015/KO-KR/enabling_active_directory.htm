Active Directory 활성화

Active Directory를 활성화하면 My Cloud™ 를 기존 Windows 도메인에 연결할 수 있습니다. 기본 설정은 Active Directory 비활성화입니다.

다음 단계를 사용하여 Active Directory를 활성화합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 네트워크를 클릭합니다.
    3. Windows 서비스>Active Directory 필드에서 (을)를 클릭하여 Active Directory를 활성화합니다.
    4. Active Directory 설정 화면에 다음 정보를 입력합니다.

    2. 사용자 이름

    Active Directory 서버 계정 이름을 입력합니다.

    암호

    Active Directory 서버 암호를 입력합니다.

    도메인 이름

    Active Directory 서버 도메인 이름의 이름을 입력합니다.

    DNS 서버

    DNS(Domain Name System) 서버 IP 주소를 입력합니다.
    1. 화면에 입력한 정보를 검토한 후 적용을 클릭합니다.
