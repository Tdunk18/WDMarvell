포트 전달 사용

릴레이 연결을 통해 클라우드에 대한 원격 액세스를 사용하는 경우 포트 전달을 사용하여 네트워크 성능을 향상시킬 수 있습니다.

참고: 모바일 액세스 연결 중계 여부를 확인하려면 원격 액세스 상태 확인을 참조하십시오.

중요: 다음 절차를 시작하기 전에 라우터에 사용할 수 있는 유효한 포트 번호 범위를 알아야 하며 HTTP 및 HTTPS에 적절한 포트를 선택해야 합니다. 유효한 포트 번호 범위는 1024~66535입니다. 일반적으로 할당되는 포트는 HTTP의 경우 80, HTTPS의 경우 443입니다.

다음 단계를 사용하여 포트 전달을 활성화합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 일반을 클릭합니다.
    3. 클라우드 액세스 섹션에서 을 클릭하여 클라우드 액세스를 켭니다(해당하는 경우).
    4. 구성을 클릭합니다.
    5. 연결 필드에서 수동을 클릭합니다.
    6. 외부 포트 1 필드에 HTTP 세션이 발생하는 포트 번호를 입력합니다.
    7. 외부 포트 2 필드에 HTTPS 세션이 발생하는 포트 번호를 입력합니다.
    8. 적용을 클릭합니다.
    9. 6단계와 7단계에서 지정한 포트를 사용하도록 라우터를 구성합니다.
