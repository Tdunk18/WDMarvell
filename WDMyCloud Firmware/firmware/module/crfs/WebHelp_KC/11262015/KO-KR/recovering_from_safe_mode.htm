안전 모드에서 복구

다음 단계를 사용하여 안전 모드에서 장치를 복구합니다.

    1. 이 단계를 수행하기 전, WD 지원 웹사이트(http://support.wd.com)에서 My Cloud 펌웨어를 다운로드한 후 컴퓨터에서 검색할 수 있는 위치에 저장하십시오. 펌웨어 파일의 압축을 풉니다.
    2. 라우터의 DHCP LAN/장치 클라이언트 테이블에 로그인하여 장치의 IP 주소를 가져옵니다.
    3. 브라우저 창에 장치 IP 주소를 입력합니다. 안전 모드 UI가 나타납니다.
    4. 펌웨어를 다운로드하여 저장한 위치를 찾습니다.
    5. 적용을 클릭하여 펌웨어를 로드합니다.
    6. 대화 상자에서 확인을 클릭합니다. 장치가 다시 부팅됩니다.
    7. 다시 부팅이 완료되면 브라우저의 캐시를 삭제합니다. 브라우저의 도움말을 확인하여 캐시를 삭제하기 위한 최적의 방법을 결정합니다.
    8. 장치 다시 부팅이 완료되면 시스템을 복구할 것을 권장합니다.
