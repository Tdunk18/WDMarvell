DLNA 활성화/비활성화

DLNA(Digital Living Network Alliance) 장치에서 클라우드 내용에 액세스할 수 있도록 DLNA 서버 설정을 켜십시오.

다음 단계를 사용하여 DLNA 서버 설정을 켜거나 끕니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 미디어를 클릭합니다.
    3. DLNA 미디어 서버>미디어 스트리밍 영역에서 을 클릭하여 필요에 따라 미디어 스트리밍을 켜거나 끕니다.
    • 켜기: DLNA 장치가 클라우드에 액세스하도록 허용합니다.
    • 끄기: DLNA 장치의 액세스를 거부합니다.
    1. 참고를 검토한 후 확인을 클릭합니다. DLNA 미디어 서버 영역에 다음의 정보가 표시됩니다.

버전

DLNA 미디어 서버의 현재 소프트웨어 버전을 표시합니다.

미디어

DLNA 서버에서 현재 사용할 수 있는 음악 트랙, 사진, 비디오의 수를 알려 줍니다.

마지막 업데이트

마지막 DLNA 서버 업데이트의 요일, 날짜 및 시간을 알려 줍니다.

DLNA 데이터베이스

이들 버튼을 사용해 DLNA 서버 데이터베이스를 다시 검색하거나 재구성할 수 있습니다.
