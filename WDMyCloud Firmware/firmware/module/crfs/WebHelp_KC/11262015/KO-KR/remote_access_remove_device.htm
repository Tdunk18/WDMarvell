장치 제거

다음 단계를 사용하여 클라우드에서 장치를 제거합니다.

    1. 클라우드 액세스를 클릭합니다.
    2. 왼쪽 창에서 삭제할 장치의 사용자를 클릭합니다.
    3. 클라우드 장치 섹션에서 장치를 찾습니다.
      상태 정보에 WD Photos 또는 WD My Cloud에 대한 연결이 표시됩니다(연결됨).
    4. 다음 작업 중 하나를 수행합니다.
    • 해당 WD Photos 행에서 삭제 아이콘(()을 클릭하여 WD Photos 모바일 응용프로그램을 통한 장치의 클라우드 액세스를 제거합니다.
    • 해당 WD My Cloud 행에서 삭제 아이콘()을 클릭하여 WD My Cloud 응용프로그램을 통한 장치의 클라우드 액세스를 제거합니다.
    • 모바일 장치에 해당하는 행에서 삭제 아이콘()을 클릭하여 My Cloud 환경에서 장치를 제거합니다.
    1. 확인 대화 상자에서 확인을 클릭합니다.
