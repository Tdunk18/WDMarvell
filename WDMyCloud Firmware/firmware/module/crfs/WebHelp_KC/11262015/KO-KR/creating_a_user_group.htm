그룹 만들기

그룹은 관리 가능한 단위로 모은 사용자 계정들의 컬렉션입니다. 다음 단계를 사용하여 그룹을 만듭니다.

    1. 사용자를 클릭합니다.
    2. 그룹 버튼을 클릭합니다.
    3. 추가 아이콘을 클릭합니다.
    4. 새 그룹의 그룹 이름을 입력합니다. 이름의 길이는 최대 16자이며, 공백은 사용할 수 없습니다.
    5. 새 그룹에 추가할 사용자를 선택합니다.
    6. 적용을 클릭합니다. 새 그룹이 만들어져서 화면의 왼쪽 열에 표시됩니다.
