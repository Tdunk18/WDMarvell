사용자 계정 추가

다음 단계에 따라 My Cloud™ 장치에 사용자를 추가합니다.

    1. 사용자를 클릭합니다.
    2. 사용자 추가 아이콘을 클릭합니다.
    3. 사용자 이름을 입력합니다. 이 이름이 시스템에 로그인할 때 사용됩니다.
    4. 이름을 입력합니다. 이것은 사용자의 이름입니다.
    5. 성을 입력합니다. 이것은 사용자의 성입니다.
    6. 이메일 주소를 입력합니다. 이것은 사용자의 이메일 주소입니다. 이 주소는 WD My Cloud 계정을 만드는 데에도 사용됩니다.
    7. 적용을 클릭합니다. 새 사용자가 왼쪽 창에 표시됩니다.
