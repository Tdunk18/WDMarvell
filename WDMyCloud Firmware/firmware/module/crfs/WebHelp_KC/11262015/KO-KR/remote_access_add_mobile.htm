모바일 장치 추가

WD Photos와 My Cloud™ 앱을 사용하여 모바일 장치(스마트폰 또는 태블릿)에서 클라우드 콘텐츠에 액세스할 수 있습니다. 여기에 설명된 절차를 시작하기 전에 무료 앱을 모바일 장치에 다운로드합니다.

모바일 장치를 추가하려면:

  1. 모든 사용자에 대해 클라우드에 대한 원격 액세스를 활성화하지 않은 경우 지금 활성화합니다.
  2. 장치 탐색 표시줄에서 클라우드 액세스를 클릭합니다.
  1. 왼쪽 창에서 클라우드에 대한 액세스 권한을 부여할 모바일 장치 사용자의 이름을 클릭합니다.
  1. <user name> 영역에 대한 클라우드 장치에서 코드 받기를 클릭합니다. My Cloud 에 코드가 표시됩니다. 이 코드를 24시간 이내에 모바일 장치에 입력해야 합니다. 이 코드를 잊어버리지 않게 적어 두십시오.
  1. 확인을 클릭합니다.
  2. 모바일 장치에서 WD Photos 및/또는 WD My Cloud 앱을 설치합니다.
  3. 로컬 네트워크에 연결되지 않은 경우 요청에 따라 활성화 코드를 입력합니다.
    장치가 활성화되면 WD Photos와 WD My Cloud 앱을 통해 클라우드 콘텐츠에 액세스합니다.
