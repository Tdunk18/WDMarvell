Změna síťového režimu IPv4

Ve výchozím nastavení se používá síťový režim DHCP. Znamená to, že zařízení My Cloud™ získává IP adresy a další nastavení ze sítě automaticky. Pokud chcete nastavovat IP adresy ručně, změňte režim pomocí následujícího postupu.

Chcete-li změnit síťový režim IPv4, postupujte následovně.

    1. Klikněte na položku Nastavení.
    2. V levém podokně klikněte na položku Síť.
    3. V oblasti Síťová služba > Síťový režim IPv4 zvolte metodu přiřazování jedinečné IPv4 adresy k zařízení:
      • Staticky: Statická IP adresa umožňuje mít stejnou IP adresu při každém připojení. Budete vyzváni k zadání IP adresy, masky podsítě, IP adresy brány a serveru DNS. (Pokud tyto informace neznáte, zkontrolujte nastavení svého směrovače.).
      • DHCP: Klient DHCP způsobuje, že zařízení My Cloud získá IP adresu a související nastavení automaticky z místního serveru DHCP.
        Poznámka: Při konfiguraci staticky nebo pomocí DHCP si pro budoucí potřeby na bezpečné místo zaznamenejte masku podsítě, IP adresu brány a server DNS. Jakmile jsou tyto informace zadány do serveru My Cloud , již nebudou zobrazeny:

    1. Dokončením průvodce nastavením sítě LAN nastavíte síťový režim.
