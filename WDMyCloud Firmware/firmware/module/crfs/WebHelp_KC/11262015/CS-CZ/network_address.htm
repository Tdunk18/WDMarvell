Určení adresy MAC a IP zařízení

Chcete-li určit adresy MAC a IP zařízení My Cloud™, postupujte následovně:

    1. Klikněte na položku Nastavení.
    2. V levém podokně klikněte na položku Síť. Následující informace se zobrazí v sekci Profil sítě.

    Stav

    Aktuální stav připojení k internetu.

    Adresa MAC

    Adresa MAC (Media Access Control) tohoto zařízení.

    IP adresa IPv4

    IP adresa verze 4 tohoto zařízení.

    Server DNS IPv4

    IP adresa verze 4 serveru DNS pro toto zařízení.

    IP adresa IPv6 DNS

    IP Adresa verze 6 tohoto zařízení.

    IPv6 serveru DNS

    IP adresa verze 6 serveru DNS pro toto zařízení.
