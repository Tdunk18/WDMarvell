Vytvoření vzdálené zálohy

Vzdálená záloha vám umožňuje zálohovat obsah zařízení My Cloud™ do jiného zařízení My Cloud. Chcete-li vytvořit vzdálenou zálohu svého zařízení My Cloud, postupujte následovně.

    1. Klikněte na položku Zálohy.
    2. Klikněte na položku Vzdálená záloha.
    3. Klikněte na možnost Vytvořit úlohu.
    4. Zadáním následujících informací vytvořte úlohu vzdáleného zálohování:

Název úlohy

Zadejte Název úlohy pro svou zálohu.

Vzdálený server

Vyberte typ vzdáleného serveru.

  • Server NAS
  • My Cloud       

Vzdálená IP adresa

Zadejte IP adresu vzdáleného umístění. (Například: 192.168.1.16)

Heslo

Zadejte heslo serveru vzdáleného zálohování.

Uživatelské jméno SSH

Zadejte uživatelské jméno protokolu SSH (Secured Shell) vzdáleného zařízení.

Heslo SSH

Zadejte heslo SSH pro vzdálené zařízení.

Zdrojová složka

Klikněte na tlačítko Procházet a vyberte složku, kterou chcete zálohovat.

Cílová složka

Klikněte na tlačítko Procházet a vyberte cílovou složku zálohování.

Typ zálohování

Uveďte typ zálohování, který chcete provést.

  • Kopírovat
  • Synchronizovat  

Opakování

Chcete-li využít funkci Opakování:

  1. Kliknutím na přepínač povolte funkci Opakované zálohování.
  2. Vyberte frekvenci zálohování: Denně, týdně, měsíčně.
  3. Vyberte čas (hodinu, dopoledne/odpoledne) v rozevírací nabídce.

    1. Klikněte na Vytvořit.

    2. V oblasti Úlohy zálohování USB kliknutím na ikonu Spustit zálohování spustíte zálohování. Proces zálohování se zobrazí v oblasti Úlohy zálohování USB.
