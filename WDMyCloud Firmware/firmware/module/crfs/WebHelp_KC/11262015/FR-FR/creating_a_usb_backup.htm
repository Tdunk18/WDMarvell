Création d'une sauvegarde USB du port arrière

Suivez ces étapes pour sauvegarder les données de votre dispositif My Cloud™ vers votre dispositif USB ou pour sauvegarder les données de votre dispositif USB vers votre dispositif My Cloud.

Remarque : ces informations concernent uniquement les ports USB situés à l'arrière de votre dispositif My Cloud Consultez Création de sauvegardes sur un port USB avant à l'aide du bouton Copie USB pour en savoir plus sur la création d'une copie de sauvegarde USB à l'aide du port USB avant de votre dispositif.

    1. Sur la barre de navigation, cliquez sur Sauvegardes pour afficher l'écran Sauvegardes.
    2. Cliquez sur Sauvegardes USB, si cette option n'est pas déjà sélectionnée.
    3. Cliquez sur Créer une tâche.
    4. Saisissez les informations suivantes pour créer une tâche de sauvegarde USB :

      Nom de tâche

      Saisissez un nom de tâche pour votre sauvegarde.

      Serveur distant

      Indiquez la direction de votre sauvegarde dans le menu déroulant. Les options incluent :

      • USB vers NAS : sauvegarde les données de votre dispositif USB sur le dispositif My Cloud.
      • NAS vers USB : sauvegarde les données de votre dispositif My Cloud sur un dispositif USB.     

      Dossier source

      Cliquez sur Parcourir et sélectionnez le dossier que vous souhaitez sauvegarder.

      Dossier de destination

      Cliquez sur Parcourir et sélectionnez le dossier de destination de votre sauvegarde.

      Type de sauvegarde

      Indiquez le type de sauvegarde que vous souhaitez effectuer.

      • Copier
      • Synchroniser

      Démarrage auto une fois connecté

      Cette option démarre automatiquement la tâche lorsque le dispositif est connecté. Cliquez sur le bouton pour activer ou désactiver l'option.

       

    1. Cliquez sur Créer.
    1. Dans la zone Tâches de sauvegarde USB, cliquez sur l'icône Démarrer la sauvegarde pour démarrer la sauvegarde.
      La progression de la sauvegarde apparaît dans la zone Tâches de sauvegarde USB.
