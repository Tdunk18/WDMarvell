Création d'une sauvegarde sur le cloud Amazon S3

Remarque : avant de créer votre sauvegarde sur le cloud, inscrivez-vous au service Amazon S3 sur le site http://aws.amazon.com/.

Une fois votre compte Amazon S3 configuré, procédez comme suit pour créer une sauvegarde sur le cloud.

    1. Cliquez sur Sauvegarde.
    2. Cliquez sur Sauvegardes sur le cloud.
    3. Cliquez sur le bouton Amazon S3 en haut de l'écran.
    4. Cliquez sur Créer.
    5. Saisissez un nom de tâche pour votre sauvegarde sur le cloud.
    6. Cliquez sur Suivant.
    7. Saisissez les informations suivantes fournies par Amazon S3 :
    • Région : sélectionnez la région utilisée pour votre cloud dans le menu déroulant.
    • Clé d'accès : saisissez la clé d'accès fournie par Amazon S3.
    • Clé privée : saisissez la clé privée fournie par Amazon S3.
    • Chemin d'accès distant : saisissez le chemin d'accès distant de votre cloud. Il s'agit normalement de votre nom de paquet.
    1. Cliquez sur Suivant.
    2. Dans le champ Type, sélectionnez l'une des options suivantes :
    • Chargement : sauvegarde vos données My Cloud dans votre compartiment Amazon S3.
    • Téléchargement : Sauvegarde votre compartiment Amazon S3 sur votre serveur My Cloud.
    1. Dans le champ Type de sauvegarde, sélectionnez le type de sauvegarde que vous souhaitez réaliser dans le menu déroulant. Les options incluent :
    • Remplacer les fichiers existants : remplace les fichiers du dossier cible dont les noms sont identiques à ceux des fichiers sources.
    • Sauvegarde complète : à chaque sauvegarde, crée un dossier séparé contenant toutes les données sauvegardées.
    • Sauvegarde incrémentielle : remplace les fichiers dont les fichiers sources sont plus récents que les fichiers cibles.
    1. Cliquez sur Suivant.
    2. Dans le champ Chemin d'accès local, saisissez le chemin de la sauvegarde sur votre dispositif My Cloud.
    • Cliquez sur Parcourir pour indiquer l'emplacement de votre sauvegarde sur le dispositif.
      (Volume_1/sauvegarde, par exemple)
    1. Cliquez sur pour activer la fonctionnalité Mise à jour auto. Cela permet de mettre automatiquement votre sauvegarde à jour selon votre calendrier.
    • Si vous activez la fonctionnalité Mise à jour auto. :

      • Sélectionnez la fréquence de mise à jour auto. : Quotidienne, Hebdomadaire ou Mensuelle.

      • Sélectionnez l'heure de la mise à jour auto. dans le menu déroulant (option Quotidienne).

      • Sélectionnez la date et l'heure de la mise à jour auto. dans le menu déroulant (option Hebdomadaire ou Mensuelle).

    1. Si vous ne sélectionnez pas la mise à jour auto., indiquez si vous souhaitez commencer la sauvegarde maintenant dans le champ Sauvegarder maintenant.
    • Sélectionnez Oui pour commencer votre sauvegarde maintenant.
    1. Cliquez sur Suivant.
    2. Révisez vos paramètres, puis cliquez sur Terminer. Votre sauvegarde sur le cloud Amazon S3 est créée. La nouvelle tâche s'affiche dans la section de la file d'attente sur la page de sauvegarde Amazon S3.

    Remarque : accédez à votre compartiment de sauvegarde sur le cloud Amazon S3 pour visualiser les sauvegardes de votre dispositif.
