Planification des arrêts du dispositif

Le calendrier d'arrêt, situé à la page Paramètres > Général, vous permet de planifier des arrêts pour votre dispositif My Cloud™.
Suivez ces étapes pour planifier les arrêts de votre appareil.

    1. Cliquez sur Paramètres.
    2. Cliquez sur Général, si cette option n'est pas déjà sélectionnée.
    3. Dans la zone Économiseur d'énergie > Calendrier d'arrêt, cliquez sur le bouton du calendrier d'arrêt.
    1. Cliquez sur Configurer pour personnaliser votre calendrier d'arrêt.
    2. Sélectionnez les heures auxquelles vous souhaitez que votre appareil reste actif pour chaque jour de la semaine.
    1. Cliquez sur Enregistrer.
      Remarque : vous pouvez utiliser WD Quick View pour remettre le système en marche.
