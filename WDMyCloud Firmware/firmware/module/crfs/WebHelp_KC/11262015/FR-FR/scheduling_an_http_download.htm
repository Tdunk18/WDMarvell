Planification d'un téléchargement HTTP

L'application de téléchargement HTTP vous permet de planifier un téléchargement HTTP sur votre dispositif My Cloud™. Suivez ces étapes pour planifier un téléchargement HTTP.

    1. Cliquez sur Applications.
    2. Cliquez sur Téléchargements HTTP.
    3. Dans le champ URL, saisissez l'URL du fichier que vous souhaitez télécharger.
      Exemple : http://192.168.0.32/test.txt
    4. Dans le champ Enregistrer vers, saisissez l'emplacement où vous souhaitez enregistrer le fichier téléchargé ou cliquez sur Parcourir pour sélectionner son emplacement.
      Exemple : Volume_1/Public/Shared Videos 
    5. Dans le champ Renommer, saisissez un nouveau nom de fichier si vous souhaitez modifier le nom du fichier.
    6. Cliquez sur s'il s'agit d'une sauvegarde périodique.
    • Sélectionnez la fréquence de sauvegarde (Quotidienne, Hebdomadaire, Mensuelle).
    • Saisissez l'heure, le jour et la date de la sauvegarde, selon la fréquence sélectionnée.
    1. Saisissez la date à laquelle vous souhaitez effectuer le téléchargement (mm/jj/aaaa), ou cliquez sur pour sélectionner une date.
    2. Saisissez l'heure du téléchargement dans les menus déroulants.
    3. Cliquez sur Créer. Votre téléchargement est créé et ajouté à la file d'attente de téléchargements HTTP.
