Création d'une cible iSCSI

Suivez ces étapes pour créer une cible iSCSI.

    1. Cliquez sur Stockage.
    2. Cliquez sur iSCSI.
    3. Cliquez sur pour activer le protocole iSCSI.
    4. Cliquez sur Créer une cible iSCSI en bas de l'écran.
    5. Dans l'écran Créer une cible iSCSI, saisissez les informations suivantes :
    • Alias : l'alias doit être un nom descriptif de votre cible.
    • Créé sur : sélectionné le volume de résidence de la cible iSCSI.
    • Taille : la taille de la cible, ainsi que l'unité de mesure correspondante (ex. : Go, To).
    1. Cliquez sur Suivant.
    • Si vous souhaitez attribuer un niveau de sécurité à votre cible, cliquez sur CHAP.
    • Saisissez un nom d'utilisateur et un mot de passe existants.
    1. Cliquez sur Appliquer. Votre nouvelle cible iSCSI est créée.
