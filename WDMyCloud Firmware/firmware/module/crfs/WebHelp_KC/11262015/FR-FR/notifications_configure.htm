Configuration des notifications

Le dispositif My Cloud™ transmet des notifications concernant divers événements tels que les mises à jour du micrologiciel, les installations réussies de micrologiciels et les arrêts du système. Les notifications s'affichent dans la zone Alerte située en haut de la page et sont également disponibles par e-mail, selon la configuration de votre dispositif.

La page de notification vous permet de configurer des adresses e-mail pour l'envoi de notifications à un maximum de cinq utilisateurs, de configurer des messages d'alerte pour les dispositifs mobiles et de configurer le niveau d'alerte pour lequel vous serez notifié.

Notifications par e-mail

  1. Dans la barre de navigation, cliquez sur l'icône Paramètres puis sur Notifications dans le volet gauche.
  2. Dans le champ E-mails d'alerte, cliquez sur le bouton ACTIVÉ.
  3. Cliquez sur Configurer.
  4. Sur l'écran Notification d'alerte, sélectionnez le niveau d'alerte auquel vous souhaitez être averti par e-mail à l'aide du curseur :
  • Critique uniquement : envoyez uniquement des alertes critiques aux adresses e-mail indiquées.
  • Critique et Avertissement : envoyez des alertes d'avertissement aux adresses e-mail indiquées.
  • Toutes : envoyez toutes les alertes (informatives, critiques ou d'avertissement) aux adresses e-mail indiquées.
  1. Cliquez sur Nouvelle adresse électronique.
  2. Saisissez l'adresse e-mail à laquelle vous souhaitez recevoir des e-mails d'alerte et cliquez sur Enregistrer.
  3. Répétez les étapes 5 et 6 pour saisir jusqu'à cinq adresses e-mail.
  4. Cliquez sur Envoyer un e-mail de test pour valider les adresses e-mail saisies.
  5. Cliquez sur OK, puis consultez l'e-mail de validation dans votre boîte de réception.

 

Activation des notifications par SMS

Remarque : consultez votre fournisseur de services SMS pour connaître les exigences relatives à l'envoi de messages SMS. Certains opérateurs peuvent vous demander d'envoyer des messages SMS/textos par e-mail.

  1. Dans la barre de navigation, cliquez sur l'icône Paramètres puis sur Notifications dans le volet gauche.
  2. Pour activer l'option SMS, cliquez sur le bouton ACTIVÉ.
  3. Cliquez sur Configurer.
  4. Sur l'écran Paramètres SMS, saisissez le nom de votre fournisseur de SMS.
  5. Saisissez l'URL SMS dont le message défini est « Hello world ».
  • Cette URL SMS ne sert qu'à des fins de configuration. Aucun message SMS ne sera envoyé une fois la configuration terminée. Obtenez les exigences relatives au format d'URL de votre fournisseur de services SMS. Elles comprennent notamment les paramètres suivants : nom d'utilisateur, mot de passe, numéro de téléphone cible et contenu du message.
  1. Cliquez sur Suivant.
  2. Sélectionnez la catégorie correspondante pour chaque paramètre SMS à partir du menu déroulant.
  3. Cliquez sur Terminer. Votre notification par SMS est maintenant configurée.
