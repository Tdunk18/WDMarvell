Création d'une sauvegarde à distance

La sauvegarde à distance permet d'effectuer une sauvegarde de votre dispositif My Cloud™ sur un autre dispositif My Cloud. Suivez ces étapes pour créer une sauvegarde à distance de votre dispositif My Cloud.

    1. Cliquez sur Sauvegarde.
    2. Cliquez sur Sauvegarde à distance.
    3. Cliquez sur Créer une tâche.
    4. Saisissez les informations suivantes pour créer une tâche de sauvegarde à distance :

Nom de tâche

Saisissez un nom de tâche pour votre sauvegarde.

Serveur distant

Sélectionnez le type de serveur à distance.

  • Serveur NAS
  • My Cloud       

IP à distance

Saisissez l'adresse IP de l'emplacement à distance. (Exemple : 192.168.1.16)

Mot de passe

Saisissez le mot de passe du serveur de sauvegarde à distance.

Nom d'utilisateur SSH

Saisissez le nom d'utilisateur du protocole Secured Shell (SSH) du dispositif à distance.

Mot de passe SSH

Saisissez le mot de passe SSH du dispositif à distance.

Dossier source

Cliquez sur Parcourir et sélectionnez le dossier sur lequel vous souhaitez effectuer la sauvegarde.

Dossier de destination

Cliquez sur Parcourir et sélectionnez le dossier que vous souhaitez sauvegarder.

Type de sauvegarde

Indiquez le type de sauvegarde que vous souhaitez effectuer.

  • Copier
  • Synchroniser  

Périodicité

Si vous souhaitez utiliser la fonction Sauvegarde périodique :

  1. Cliquez sur le bouton pour activer la fonction de sauvegarde périodique.
  2. Sélectionnez la fréquence de sauvegarde : Quotidienne, Hebdomadaire, Mensuelle.
  3. Dans le menu déroulant, sélectionnez une heure (au format AM/PM).

    1. Cliquez sur Créer.

    2. Dans la zone Tâches de sauvegarde USB, cliquez sur l'icône Démarrer la sauvegarde pour démarrer la sauvegarde. La progression de la sauvegarde apparaît dans la zone Tâches de sauvegarde USB.
