Configuration des modes d'économie d'énergie

Réduisez la consommation d'énergie de votre dispositif My Cloud™ en suivant ces étapes.

    1. Cliquez sur Paramètres.
    2. Cliquez sur Général.
    3. Dans la zone Économiseur d'énergie, sélectionnez les options suivantes pour définir la consommation d'énergie de votre dispositif.

Mode

Description

Mise en veille du lecteur

L'activation de cette fonction permet aux disques durs de passer en mode veille pour économiser de l'énergie après la période d'inactivité choisie. La mise en veille du lecteur est activée par défaut après environ 10 minutes d'inactivité.

  • Cliquez sur pour activer/désactiver la mise en veille du lecteur.

LCD

L'activation de cette fonction permet de mettre l'écran LCD du dispositif en mode veille afin d'économiser de l'énergie après lune période d'inactivité définie. Par défaut, le dispositif LCD passe en mode veille au bout de 10 minutes.

  • Cliquez sur pour activer/désactiver l'écran LCD.

Cette option est uniquement disponible sur les dispositifs My Cloud EX2. My Cloud EX4, My Cloud EX4100 et My Cloud DL4100.

Rétablissement de l'alimentation

Redémarre automatiquement votre dispositif après un arrêt inattendu dû à une panne de courant. Le rétablissement de l'alimentation est activé par défaut.

  • Cliquez sur pour activer/désactiver le rétablissement de l'alimentation.

Calendrier d'arrêt

Le calendrier d'arrêt vous permet de programmer des arrêts pour votre dispositif My Cloud.

  • Cliquez sur pour activer/désactiver le calendrier d'arrêt.
  • Cliquez sur Configurer pour personnaliser votre calendrier d'arrêt.
  • Sélectionnez les heures auxquelles vous souhaitez que votre appareil reste actif pour chaque jour de la semaine.
  • Cliquez sur Enregistrer.

 

Délai d'expiration de l'accès Web

Choisissez l'heure par défaut à laquelle le dispositif se déconnecte automatiquement.

  • Dans le menu déroulant Délai d'expiration de l'accès Web, sélectionnez le délai d'expiration que vous souhaitez utiliser pour votre système. Le système est mis à jour automatiquement.
