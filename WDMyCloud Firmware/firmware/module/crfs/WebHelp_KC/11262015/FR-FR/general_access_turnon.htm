Activation/désactivation de l'accès au cloud

Si vous êtes administrateur, vous pouvez activer ou désactiver l'accès au cloud de tous les utilisateurs (autrement dit, vous pouvez déterminer les dispositifs informatiques autorisés à accéder au contenu présent sur le dispositif My Cloud™).

Suivez ces étapes pour activer/désactiver l'accès au cloud.

    1. Cliquez sur Paramètres.
    2. Dans le volet de gauche, cliquez sur Général.
    3. Dans le champ Accès au cloud > Service cloud, cliquez sur pour activer ou désactiver le service cloud :
    • Activé : active l'accès distant pour tous les utilisateurs.
    • Désactivé : désactive l'accès distant pour tous les utilisateurs.
    1. Cliquez sur Configurer pour modifier le type de connexion utilisé pour accéder au cloud. L'écran Options de connexion pour l'accès au cloud propose trois options d'accès.
    2. Auto : l'option Auto utilise UPnP (Universal Plug and Play) pour essayer d'ouvrir des ports sur votre routeur. En cas de succès, une connexion directe est établie entre votre dispositif et vos applications.
    3. Manuel : établit une connexion par l'intermédiaire des deux ports sélectionnés. Si les deux ports ne sont pas disponibles, une connexion relais est établie. Pour utiliser cette option, le routeur doit être configuré manuellement. Consultez la réponse nº 8526 de la base de connaissances pour savoir comment configurer votre routeur en mode manuel.
    4. Win XP : établit une connexion par les ports 80 et 443. Si ces ports ne sont pas disponibles, une connexion relais est établie. Cette option est requise si vous utilisez Windows XP.
      Remarque : par défaut, le dispositif My Cloud établit automatiquement une connexion directe entre vos dispositifs mobiles et le routeur.
    1. Dans le champ Base de données de contenu, cliquez sur Restaurer pour restaurer votre base de données WDMyCloud.
      Remarque : n'utilisez cette option de dépannage que si vous pensez que la base de données est corrompue.
    2. Cliquez sur Appliquer. Le champ État de la connexion indique que votre dispositif est connecté.

    Pour en savoir plus sur la désactivation de l'accès d'un dispositif unique, consultez Suppression d'un dispositif.

    Remarque : pour en savoir plus sur la zone Options de connexion, consultez la rubrique Configuration des options de connexion, qui décrit les différentes options de pare-feu et explique comment améliorer les performances. Par défaut, My Cloud établit automatiquement une connexion directe entre vos dispositifs mobiles et le routeur.

    Si l'accès mobile est activé via une connexion relais, (c'est-à-dire, si le champ État de la connexion affiche « Connecté (Relais) »), vous pouvez améliorer les performances en activant le réacheminement de port sur votre connecteur WD Venue (reportez-vous à la rubrique Activation du réacheminement de port).
