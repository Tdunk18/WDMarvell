Déterminer les adresses MAC et IP du dispositif

Suivez les étapes ci-dessous pour définir les adresses MAC et IP du dispositif My Cloud™ :

    1. Cliquez sur Paramètres.
    2. Dans le volet de gauche, cliquez sur Réseau. Les informations suivantes sont affichées dans la section Profil réseau.

    État

    L'état actuel de la connexion Internet.

    Adresse MAC

    L'adresse MAC (Media Access Control) du dispositif.

    Adresse IPv4

    L'adresse IP version 4 du dispositif.

    Serveur DNS IPv4

    L'adresse IP version 4 du serveur DNS du dispositif.

    Adresse IPv6 du serveur DNS

    L'adresse IP version 6 du dispositif.

    Serveur DNS IPv6

    L'adresse IP version 6 du serveur DNS du dispositif.
