Modification du mode RAID

Suivez les étapes suivantes pour modifier le mode RAID de votre dispositif My Cloud™.

Avertissement : toute modification du mode RAID supprimera vos données et vos paramètres utilisateur. Consultez la section Exportation des paramètres système pour en savoir plus sur l'enregistrement de vos paramètres utilisateur.

    1. Cliquez sur Stockage.
    2. Cliquez sur RAID si cette option n'est pas déjà sélectionnée.
    3. Cliquez sur Modifier le mode RAID en bas de l'écran.
    4. Révisez le message d'avertissement, puis cliquez sur OK.
    5. Sélectionnez le mode RAID que vous souhaitez appliquer à votre dispositif My Cloud. Les options incluent :

Mode

Description

JBOD

L'utilisation d'un ou de plusieurs lecteurs n'étant pas en configuration RAID, mais gérés en tant que volumes logiques séparés.

Fractionnement

Combinaison de lecteurs en ligne pour créer un grand volume logique.

RAID 0

Les données sont agrégées par bande sur plusieurs disques durs, permettant une lecture et un enregistrement accélérés des données en combinant le fonctionnement d'au moins deux lecteurs pour augmenter les performances. Toutefois, en cas de défaillance de l'un des lecteurs, toutes vos données seront perdues.

RAID 1

Deux copies instantanées des données sont enregistrées sur des disques durs séparés. En cas de défaillance de l'un d'eux, les autres continuent à lire et enregistrer les données jusqu'à ce que le disque dur défectueux soit remplacé.

RAID 5

Offre des performances et une protection supérieures en agrégeant les données par bande sur au moins trois lecteurs, et en dédiant le quart de chaque lecteur à la tolérance de panne. En cas de défaillance de l'un des lecteurs, les autres continuent à lire et à enregistrer les données jusqu'à ce que le disque dur défectueux soit remplacé.

Cette option est uniquement disponible pour les dispositifs My Cloud à quatre emplacements.

RAID 10

Protocole RAID selon lequel les données sont enregistrées par bande sur les disques principaux mis en miroir sur les disques secondaires. En cas de défaillance de l'un des lecteurs, les autres continuent à lire et à enregistrer les données jusqu'à ce que le disque dur défectueux soit remplacé.

Cette option est uniquement disponible pour les dispositifs My Cloud à quatre emplacements.

    1. Vérifiez et sélectionnez l'option de stockage que vous souhaitez utiliser, cochez la case Basculer vers <Storage Mode>, puis cliquez sur Suivant. Un test automatique du lecteur est effectué.
    2. Révisez les avertissements en haut de l'écran.
    3. À la fin du test automatique, vérifiez l'état de vos disques durs et cliquez sur Suivant.
    4. Vérifiez l'état de vos volumes, puis cliquez sur Suivant.
    5. Si le curseur du volume s'affiche, sélectionnez l'espace que vous souhaitez dédier au mode RAID sélectionné.
    6.  Si vous choisissez de ne pas utiliser la totalité du volume pour votre sélection RAID, vous pouvez fractionner l'espace restant du disque en cochant la case Fractionner l'espace disque restant.

      REMARQUE : si vous choisissez de ne pas utiliser la totalité du volume pour votre sélection RAID, vous pouvez fractionner l'espace restant du disque en cochant la case Fractionner l'espace disque restant.

    1. En mode RAID 5, vous pouvez opter pour la configuration RAID 5+spare pour faire de « Disk4 » le disque de rechange. Cochez la case Créer un disque de rechange pour configurer l'option RAID 5+spare. Décochez Créer un disque de rechange si vous ne souhaitez pas configurer l'option RAID 5+spare.
    2. Cliquez sur Suivant pour continuer.
    3. Si vous souhaitez restaurer automatiquement la configuration RAID une fois le disque reconnu, cliquez sur , puis sur Suivant.
