Gestion des fichiers Web

La visionneuse de fichiers Web est une application permettant de visualiser et de gérer les fichiers de votre dispositif My Cloud™.

Suivez ces étapes pour gérer vos fichiers Web.

    1. Cliquez sur Applications.
    2. Cliquez sur Visionneuse de fichiers Web. Les volumes de fichiers disponibles sur votre dispositif s'affichent dans le volet droit.

Visualisation des fichiers Web

Les volumes, les dossiers et les fichiers s'affichent dans le volet droit de la visionneuse de fichiers Web. Pour visualiser les contenus de tout volume ou dossier, double-cliquez sur le volume ou le dossier concerné.

Gestion des fichiers Web

Les actions disponibles applicables à vos fichiers et dossiers s'affichent en haut à droite de la visionneuse de fichiers Web. Sélectionnez l'une des actions suivantes à appliquer à vos fichiers Web :

Icône

Fonction

Description

Copier

Cette action vous permet de copier un fichier sélectionné vers un autre emplacement. Suivez ces étapes pour copier un fichier vers un nouvel emplacement :

    1. Dans la zone Visionneuse de fichiers Web, sélectionnez le fichier que vous souhaitez copier.
    2. Cliquez sur .
    3. Sélectionnez la destination du fichier copié.
    4. Cliquez sur OK. Votre fichier a été copié vers la destination choisie.

Déplacer

Cette action vous permet de déplacer un fichier d'un emplacement à l'autre. Suivez ces étapes pour déplacer un fichier vers un nouvel emplacement :

    1. Sélectionnez le fichier que vous souhaitez déplacer.
    2. Cliquez sur .
    3. Sélectionnez la destination vers laquelle vous souhaitez déplacer votre fichier.
    4. Cliquez sur OK. Votre fichier a été déplacé vers la destination choisie.

Chargement

Cette action vous permet de charger un fichier depuis votre ordinateur ou un autre dispositif vers votre serveur My Cloud . Suivez ces étapes pour charger un fichier vers votre dispositif :

    1. Dans le volet droit de la visionneuse de fichiers Web, sélectionnez la destination du fichier que vous souhaitez charger.
    2. Cliquez sur .
    3. Sur l'écran Chargement, cliquez sur l'icône Sélectionner un fichier.
    4. Recherchez et sélectionnez le fichier que vous souhaitez charger vers votre dispositif.
    5. Cliquez sur Ouvrir.
    6. Cliquez sur Chargement.
    • Si vous souhaitez charger d'autres fichiers, suivez les étapes 3 et 4 jusqu'à ce que tous les fichiers aient été chargés.
    1. Cliquez sur Fermer. Vos fichiers chargés s'affichent dans le dossier de destination.

Télécharger

Cette action vous permet de télécharger un fichier depuis votre dispositif My Cloud vers votre ordinateur ou d'autres dispositifs.
Suivez ces étapes pour télécharger un fichier vers votre ordinateur ou d'autres dispositifs :

    1. Dans le volet droit de la visionneuse de fichiers Web, sélectionnez le fichier que vous souhaitez télécharger.
    2. Cliquez sur .
    3. Le fichier sélectionné a été téléchargé vers votre ordinateur ou d'autres dispositifs.

Renommer

Cette action vous permet de renommer un fichier sélectionné sur votre dispositif My Cloud . Suivez ces étapes pour renommer un fichier.

    1. Dans le volet droit de la visionneuse de fichiers Web, sélectionnez le fichier que vous souhaitez renommer.
    2. Cliquez sur .
    3. Dans le champ Renommer, renommez le fichier.
    4. Cliquez sur Enregistrer. Votre fichier a été renommé.

Supprimer

Cette action vous permet de supprimer un fichier sélectionné de votre dispositif My Cloud . Suivez ces étapes pour supprimer un fichier.

    1. Dans le volet droit de la visionneuse de fichiers Web, sélectionnez le fichier que vous souhaitez supprimer.
    2. Cliquez sur .
    3. Cliquez sur OK pour confirmer que vous souhaitez supprimer le fichier sélectionné. Le fichier sélectionné a été supprimé du dispositif.

Actualiser

Cette action permet d'actualiser la liste de fichiers et affiche tout nouveau fichier ayant été ajouté depuis la dernière actualisation ou depuis la dernière fois que vous y avez accédé.

  • Cliquez sur . La liste de fichiers a été actualisée.

Créer

Cette action vous permet de créer un dossier dans le partage sélectionné.

    1. Accédez à l'emplacement où vous souhaitez créer votre nouveau dossier.
    2. Cliquez sur
    3. Saisissez le nom de votre nouveau dossier.
    4. Cliquez sur Enregistrer.Le nouveau dossier est créé.
