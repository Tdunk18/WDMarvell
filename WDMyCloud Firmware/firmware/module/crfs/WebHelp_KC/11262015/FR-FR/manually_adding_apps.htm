Ajout manuel d'applications

Si vous souhaitez ajouter une application non répertoriée sur l'écran Ajouter une application, suivez ces étapes pour l'ajouter manuellement sur votre dispositif My Cloud™.

Remarque : WD recommande de comprendre dans sa totalité la nature de toute application avant de l'installer sur votre dispositif.

    1. Cliquez sur Applications.
    2. Cliquez sur le bouton Ajouter une application dans le coin inférieur gauche de l'écran.
      La première fois que vous ajoutez une application, l'écran Conditions générales s'affiche. Lisez et acceptez le contrat.
    3. Cliquez sur le lien Pour installer une application manuellement, cliquez ici en bas de la boîte de dialogue.
    4. Accédez à l'application que vous souhaitez installer sur votre dispositif.
    5. Sélectionnez l'application que vous souhaitez installer, puis cliquez sur Ouvrir.
      L'application sélectionnée est installée et ajoutée sur votre dispositif.
