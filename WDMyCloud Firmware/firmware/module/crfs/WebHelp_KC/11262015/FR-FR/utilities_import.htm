Importation d'un fichier de configuration

En tant qu'administrateur, vous pouvez importer un fichier de configuration précédemment enregistré. En bref, cette opération remplace la configuration actuelle par la version précédemment enregistrée.

Important : l'importation d'un fichier de configuration après avoir restauré les paramètres par défaut du système ne restaure pas les données de l'utilisateur et du dispositif mobile. Vous devrez rajouter ces informations. Une fois l'importation terminée, le dispositif My Cloud™ se réinitialise, ce qui interrompt le service pour tout utilisateur accédant à votre cloud à ce moment-là.

Suivez ces étapes pour importer les paramètres système :

    1. Cliquez sur Paramètres.
    2. Dans le volet gauche, cliquez sur Outils.
    3. Dans la zone Configuration système, cliquez sur Importer un fichier.
    4. Sélectionnez l'emplacement du fichier de configuration, puis cliquez sur Ouvrir. Votre configuration a été restaurée.
