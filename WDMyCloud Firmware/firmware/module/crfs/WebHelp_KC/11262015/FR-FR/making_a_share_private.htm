Rendre un partage privé

Si vous décidez de limiter l'accès d'un partage public, suivez ces étapes pour le convertir en partage privé. Suivez ces étapes pour rendre un partage existant privé :

    1. Cliquez sur Partages.
    2. Dans le volet gauche, sélectionnez le partage que vous souhaitez rendre privé.
    3. Dans la zone Profil du partage, cliquez sur pour le DÉSACTIVER.
    4. Dans la zone Accès utilisateur, sélectionnez le niveau d'accès de chaque utilisateur en cliquant sur l'icône appropriée pour le partage (par exemple, lecture/écriture, lecture seule ou aucun accès).
