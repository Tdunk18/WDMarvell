Activation de l'accès FTP

Vous pouvez activer ou désactiver le transfert de fichiers entre les dispositifs informatiques de votre réseau via le protocole FTP (File Transfer Protocol).

Suivez ces étapes pour activer/désactiver l'accès FTP :

    1. Cliquez sur Paramètres.
    2. Dans le volet de gauche, cliquez sur Réseau.
    3. Dans la zone Service réseau>Accès FTP, cliquez sur pour activer ou désactiver l'accès FTP.
    4. Lisez le message, puis cliquez sur OK.
    5. Cliquez sur Configurer.
    6. Sur l'écran Paramètres FTP, entrez les informations suivantes :
    • Nombre maximum d'utilisateurs : à partir du menu déroulant, sélectionnez le nombre maximum d'utilisateurs auxquels vous souhaitez donner un accès FTP.
    • Temps d'inactivité : saisissez le temps d'inactivité souhaité, en minutes, avant l'expiration de l'accès FTP.
    • Port : saisissez le port qui sera utilisé pour l'accès FTP.
    • Contrôle de flux : sélectionnez Illimité ou Personnalisé. Si vous sélectionnez Personnalisé, saisissez une valeur dans le champ Contrôle de flux.
    1. Cliquez sur Suivant.
    2. Dans le champ Mode passif, sélectionnez Par défaut ou Personnalisé. Si vous sélectionnez Personnalisé, saisissez des valeurs dans le champ Mode passif.
    3. Cochez la case si vous souhaitez rapporter l'IP en mode PASV (Passif).
    • Saisissez l'adresse IP externe ou cliquez sur Obtenir l'adresse IP pour demander au système de saisir l'adresse
    1. Cliquez sur Suivant.
    2. Saisissez les informations suivantes :
    3. Langue client : sélectionnez la langue du client à partir du menu déroulant.
    4. TLS : cochez la case Protocole TLS (sécurité de la couche de transport) à côté des connexions TLS implicite ou TLS explicite, le cas échéant.
    5. FXP : cliquez sur pour activer ou désactiver le protocole FXP (protocole d'échange de fichiers).
    1. Cliquez sur Suivant.
    2. Saisissez l'adresse IP dont vous souhaitez bloquer l'accès FTP, puis sélectionnez Permanent ou Temporaire à partir du menu déroulant. Cliquez sur Appliquer pour enregistrer vos informations.
    3. Cliquez sur Terminer.
