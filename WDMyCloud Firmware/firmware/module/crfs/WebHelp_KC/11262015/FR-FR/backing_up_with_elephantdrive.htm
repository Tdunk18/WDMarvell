Sauvegardes avec ElephantDrive

Une fois ElephantDrive activé, suivez ces étapes pour créer une sauvegarde sur le cloud.

    1. Cliquez sur Sauvegarde.
    2. Cliquez sur Sauvegarde sur le cloud.
    3. Cliquez sur le bouton Elephant Drive en haut de l'écran.
    4. Dans le champ Accéder aux sauvegardes, cliquez sur Ouverture de session.
    5. Saisissez l'e-mail et le mot de passe que vous avez utilisés pour vous inscrire sur ElephantDrive.

    6. Cliquez sur Ouverture de session.

    7. Saisissez l'e-mail et le mot de passe que vous avez utilisés pour vous inscrire sur ElephantDrive à l'étape 6 de la section Activation d'une sauvegarde sur le cloud ElephantDrive. Cliquez sur Ouverture de session.

    8. Dans le champ Elephant Drive, cliquez sur le lien Assistant de sauvegarde du nouveau dispositif.

    9. Suivez les étapes de l'assistant de sauvegarde ElephantDrive pour créer votre sauvegarde sur le cloud.
