Activation du protocole SNMP

Suivez ces étapes pour activer/désactiver le protocole Simple Network Management Protocol (SNMP) sur votre dispositif My Cloud™.

    1. Cliquez sur Paramètres.
    2. Dans le volet de gauche, cliquez sur Réseau.
    3. Dans la zone Service réseau > SNMP, cliquez sur pour activer ou désactiver le protocole SNMP.
    4. Cliquez sur Configurer.
    5. Sur l'écran SNMP, saisissez les informations suivantes :
    • Communauté SNMP : saisissez le nom de votre communauté SNMP.
    • Notification : cliquez sur pour activer ou désactiver la notification SNMP.
    • Adresse IP de notification : saisissez l'adresse IP pour votre notification SNMP.
    1. Cliquez sur Enregistrer.
