Création d'une sauvegarde interne

Les sauvegardes internes vous permettent de sauvegarder le contenu existant d'un emplacement de stockage à un autre de votre dispositif My Cloud™.

Suivez ces étapes pour créer une sauvegarde interne.

    1. Cliquez sur Sauvegarde.
    2. Cliquez sur Sauvegardes internes.
    3. Cliquez sur Créer une tâche.
    4. Saisissez les informations suivantes pour créer une tâche de sauvegarde interne :

      Nom de tâche

      Saisissez un nom de tâche pour votre sauvegarde interne.

      Dossier source

      Cliquez sur Parcourir et sélectionnez le dossier que vous souhaitez sauvegarder.

      Dossier de destination

      Cliquez sur Parcourir et sélectionnez le dossier de destination de votre sauvegarde.

      Type de sauvegarde

      Indiquez le type de sauvegarde que vous souhaitez effectuer.

      • Copier
      • Synchroniser
      • Incrémentiel

      Périodicité

      Si vous souhaitez utiliser la fonction Sauvegarde périodique :

      1. Cliquez sur le bouton Activer/Désactiver pour activer la fonction de périodicité.
      2. Sélectionnez la fréquence de sauvegarde : Quotidienne, Hebdomadaire, Mensuelle.
      3. Dans le menu déroulant, sélectionnez une heure (au format AM/PM).
    5. Cliquez sur Créer. Votre tâche apparaît dans la file d'attente de sauvegarde interne et démarrera à l'heure indiquée.
