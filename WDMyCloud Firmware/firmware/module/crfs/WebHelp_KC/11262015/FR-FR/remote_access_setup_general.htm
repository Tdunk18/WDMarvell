Configuration de l'accès au cloud

Vous devez disposer d'un identifiant My Cloud™ pour accéder à votre cloud à distance depuis un smartphone, une tablette ou un ordinateur. (L'accès distant depuis un dispositif informatique est activé via le serveur central de My Cloud qui vous accorde l'accès aux dispositifs NAS connectés à votre réseau domestique.) Toutefois, la manière dont vous accédez au cloud à distance depuis vos dispositifs mobiles (smartphones et tablettes) et depuis vos ordinateurs est légèrement différente. Dans tous les cas, chaque utilisateur doit disposer d'un identifiant WD My Cloud.

Suivez ces étapes pour configurer l'accès distant :

    1. Assurez-vous que l'accès au cloud est activé sur votre dispositif. Comment faire ?
    2. Cliquez sur Accès au cloud.
    3. Dans le volet gauche, cliquez sur l'utilisateur auquel vous souhaitez permettre l'accès au cloud.
    4. Dans la section de connexion correspondant à <user name> sur WDMyCloud.com, cliquez sur Inscription. L'utilisateur recevra un e-mail avec les modalités d'inscription sur WD My Cloud.
    1. Effectuez l'une des actions suivantes :
    • Poursuivez le processus d'inscription vous-même. Comment faire ?  
    • Avertissez l'utilisateur que vous avez configuré un compte Web et invitez-le à poursuivre le processus d'inscription.
