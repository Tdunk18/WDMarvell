Activation/désactivation du mode veille

Pour limiter la consommation d'énergie, vous pouvez déterminer à quel moment le dispositif My Cloud™ doit passer en mode veille après une période d'inactivité.

Suivez ces étapes pour activer ou désactiver le mode veille.

    1. Cliquez sur Paramètres.
    2. Dans le volet de gauche, cliquez sur Général.
    3. Dans le champ Économiseur d'énergie > Veille profonde, cliquez sur pour activer ou désactiver la mise en veille du lecteur :
    • Activé : le dispositif se met en veille pendant les périodes d'inactivité.
    • Désactivé : le dispositif ne se met jamais en veille.
