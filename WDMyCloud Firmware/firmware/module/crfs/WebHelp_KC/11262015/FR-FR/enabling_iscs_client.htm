Activation du client iSCS

Le protocole iSNS (Internet Storage Name Service) sert à gérer les interactions entre les serveurs iSNS et les clients iSNS. Les clients iSNS sont les ordinateurs, ou initiateurs, qui essaient de découvrir des appareils de stockage, ou cibles, sur un réseau Ethernet.

Suivez ces étapes pour configurer le client iSCSI iSNS.

Remarque : le service iSNS est principalement utilisé pour la connexion à un serveur Windows.

    1. Cliquez sur Stockage.
    2. Cliquez sur iSCSI.
    3. Cliquez sur pour activer le protocole iSCSI.
    4. Dans le champ Client iSCSI iSNS, cliquez sur et sélectionnez ACTIVÉ.
    5. Cliquez sur Configurer.
    6. Saisissez l'adresse de serveur du client iSNS (il s'agit normalement de l'adresse IP de votre serveur Windows), puis cliquez sur Appliquer.
