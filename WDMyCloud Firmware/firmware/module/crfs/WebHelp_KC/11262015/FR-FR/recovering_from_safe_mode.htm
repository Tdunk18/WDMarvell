Récupération à l'aide du mode sans échec

Suivez les étapes décrites ci-après pour récupérer les données de votre dispositif à l'aide du mode sans échec.

    1. Avant de suivre ces étapes, téléchargez le micrologiciel My Cloud sur le site Web d'assistance WD (http://support.wd.com) et enregistrez-le dans un emplacement auquel vous pouvez accéder à partir de votre ordinateur. N'oubliez pas de décompresser le fichier du micrologiciel.
    2. Connectez-vous au réseau LAN DHCP du routeur ou à la table client du dispositif pour obtenir l'adresse IP de votre dispositif.
    3. Saisissez l'adresse IP du dispositif dans la fenêtre d'un navigateur. L'interface utilisateur du mode sans échec s'affiche.
    4. Accédez à l'emplacement où vous avez enregistré le micrologiciel téléchargé.
    5. Cliquez sur Appliquer pour charger le micrologiciel.
    6. Cliquez sur OK dans la boîte de dialogue. Le dispositif se réinitialise.
    7. Une fois la réinitialisation terminée, assurez-vous d'effacer le cache du navigateur. Consultez la rubrique d'aide du navigateur pour déterminer le meilleur moyen d'effacer le cache.
    8. Une fois la réinitialisation du dispositif terminée, nous vous recommandons d'effectuer une restauration système.
