Exécution d'une configuration unique pour obtenir un accès sécurisé

Cette procédure explique comment établir une connexion sécurisée entre votre smartphone ou votre tablette et votre dispositif My Cloud™. Suivez les instructions relatives à votre dispositif mobile :

    • Tablette ou smartphone Android  

    • iPhone ou iPad

    • Smartphone ou tablette Windows

    • Ordinateur Windows ou Mac

Obtention d'un accès sécurisé pour un smartphone ou une tablette Android

    1. Connectez votre smartphone ou votre tablette comme suit :

    • Sur votre dispositif mobile, appuyez sur Paramètres.

    • Appuyez sur Sans fil et réseaux.

    • Appuyez sur Paramètres Wi-Fi.

    • Sélectionnez votre réseau domestique.

    • Saisissez votre mot de passe réseau et appuyez sur Se connecter.

    1. Lancez une application WD.
    2. Appuyez sur Se connecter.
    3. Appuyez sur le nom du dispositif My Cloud.
    4. Appuyez sur la flèche de déroulement, puis sélectionnez votre nom d'utilisateur.
    5. Effectuez l'une des actions suivantes :

    • Si un mot de passe a été défini pour cet utilisateur, saisissez-le, puis appuyez sur Ouverture de session.

    • Si aucun mot de passe n'a été défini pour cet utilisateur, appuyez sur Ouverture de session.

    1. (Facultatif) Si vous souhaitez accéder à un contenu particulier, appuyez sur le nom d'un dossier partagé, puis naviguez vers le fichier souhaité.

Obtention d'un accès sécurisé pour un iPhone ou un iPad

    1. Connectez votre smartphone ou votre tablette comme suit :

    • Sur votre dispositif mobile, appuyez sur Paramètres.

    • Appuyez sur Wi-Fi.

    1. Appuyez sur Paramètres Wi-Fi.

    2. Sélectionnez votre réseau domestique.

    3. Saisissez votre mot de passe réseau et appuyez sur Rejoindre.

    1. Lancez une application WD.
    2. Appuyez sur Se connecter.
    3. Appuyez sur le signe plus (+) à côté du nom du dispositif My Cloud.
    4. Appuyez sur le nom d'utilisateur souhaité.
      Si un mot de passe a été défini pour cet utilisateur, saisissez-le.
    5. (Facultatif) Si vous souhaitez accéder à un contenu particulier, appuyez sur le nom d'un dossier partagé, puis naviguez vers le fichier souhaité.

Obtention d'un accès sécurisé pour un smartphone ou une tablette Windows

    • Connectez votre smartphone ou votre tablette comme suit :

    • Sur votre dispositif informatique, appuyez sur Paramètres.

    • Appuyez sur Wi-Fi.

    • Sélectionnez votre réseau domestique.

    • Saisissez votre mot de passe réseau et appuyez sur Terminé.

    1. Lancez une application WD.
    2. Appuyez sur Se connecter.
    3. Appuyez sur le nom du lecteur My Cloud.
    4. Appuyez sur la flèche de déroulement, puis sélectionnez un nom d'utilisateur.
      Si un mot de passe a été défini pour cet utilisateur, saisissez-le.
    5. (Facultatif) Si vous souhaitez accéder à un contenu particulier, procédez comme suit :

    • Sur l'écran répertoriant les dispositifs, appuyez sur le nom de votre lecteur My Cloud.

    • Appuyez sur une ressource partagée, puis accédez au contenu de votre choix.

Obtention d'un accès sécurisé pour un ordinateur Windows ou Mac

    1. Après avoir installé l'application bureautique WD My Cloud, double-cliquez sur le raccourci qui apparaît désormais sur votre poste de travail pour afficher tous les dispositifs WD du réseau. Pour en savoir plus sur l'installation de l'application, consultez Concepts de base de l'accès distant.
    2. Le système peut afficher des informations à propos des alertes de sécurité. Pour éviter l'affichage répété de cet écran, cochez la case Je comprends.
    3. Cliquez sur Je comprends pour afficher les dispositifs du réseau.
    4. Cliquez sur le dispositif auquel vous souhaitez vous connecter, dans le cas présent le dispositif My Cloud. Une fenêtre de connexion s'affiche.
    5. Dans le cas où le mot de passe du dispositif My Cloud a été défini antérieurement, saisissez-le, puis cliquez sur Se connecter. La fenêtre Mise en route s'affiche.
    6. Pour obtenir plus d'informations, cliquez sur le lien Centre d'apprentissage WD.
    7. Cliquez sur Démarrer pour afficher les dossiers partagés du dispositif My Cloud ainsi que tout fichier de sauvegarde SmartWare ou Time Machine.
    8. Pour découvrir comment accéder aux contenus stockés sur votre dispositif My Cloud et les gérer à distance, cliquez sur le lien Accès via l'ordinateur dans le panneau de téléchargement My Cloud du centre d'apprentissage en ligne :
