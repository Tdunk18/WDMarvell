Mise à jour du dispositif My Cloud depuis un fichier

Dans certains cas, pour mettre à jour le dispositif My Cloud™, vous pouvez être contraint(e) de télécharger au préalable un fichier sur le site d'assistance Western Digital et de l'installer.

Remarque : pour appliquer cette procédure, un accès Internet est nécessaire.

Suivez ces étapes pour mettre manuellement à jour le dispositif My Cloud depuis un fichier :

    1. Ouvrez un nouvel onglet ou une nouvelle fenêtre dans votre navigateur.
    2. Dans le nouvel onglet/la nouvelle fenêtre, accédez au site d'assistance technique Western Digital :
      http://support.wdc.com
    • Si une nouvelle mise à jour du micrologiciel est disponible, enregistrez-la sur un emplacement de votre dispositif informatique.
    1. Retournez sur l'interface Web My Cloud.
    2. Cliquez sur Paramètres.
    3. Dans le volet gauche, cliquez sur Mise à jour du micrologiciel.
    4. Dans la zone Mise à jour manuelle, cliquez sur Mettre à jour depuis le fichier.
    5. Accédez à l'emplacement d'enregistrement de la mise à jour à l'étape 3.
    6. Choisissez le fichier de mise à jour, puis cliquez sur Ouvrir.
    7. Cliquez sur OK pour confirmer l'installation. Une fois la mise à jour installée, le dispositif My Cloud se réinitialise.
      Remarque : lors de la réinitialisation de My Cloud, le service sera interrompu pour tout utilisateur accédant au cloud à ce moment-là.
