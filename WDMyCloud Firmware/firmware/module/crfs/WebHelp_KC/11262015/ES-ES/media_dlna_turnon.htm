Habilitar/deshabilitar DLNA

Active la configuración Servidor DLNA para que el contenido de su nube esté disponible para un dispositivo DLNA (Alianza de Redes Digitales Vivas).

Siga estos pasos para activar/desactivar la configuración Servidor DLNA:

    1. Haga clic en Configuración.
    2. En el panel izquierdo, haga clic Multimedia.
    3. En el área Servidor multimedia DLNA>Transmisión multimedia, haga clic en para activar o desactivar la transmisión multimedia, como proceda.
    • Activado: permite que los dispositivos DLNA accedan a la nube.
    • Desactivado: deniega el acceso a los dispositivos DLNA.
    1. Revise la nota y, a continuación, haga clic en Aceptar. El área Servidor multimedia DLNA muestra la siguiente información:

Versión

Muestra la versión de software actual de su servidor multimedia DLNA.

Multimedia

Identifica el número de pistas de música, imágenes y vídeos disponibles en la actualidad en su servidor DLNA.

Última actualización

Indica el día, fecha y hora de la última actualización del servidor DLNA.

Base de datos DLNA

Estos botones le permiten Volver a analizar o Reconstruir su base de datos de servidores DLNA.
