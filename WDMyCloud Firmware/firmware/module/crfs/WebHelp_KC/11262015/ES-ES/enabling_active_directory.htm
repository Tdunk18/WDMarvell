Habilitar Active Directory

Al habilitar esta función se permite que My Cloud™ se una a un dominio de Windows existente. Active Directory está deshabilitado por defecto.

Siga estos pasos para habilitar Active Directory.

    1. Haga clic en Configuración.
    2. En el panel izquierdo, haga clic en Red.
    3. En el campo Servicios de Windows>Active Directory, haga clic en activar Active Directory.
    4. Introduzca la siguiente información en la pantalla de Configuración de Active Directory:

    2. Nombre de usuario

    Introduzca el nombre de cuenta del servidor de Active Directory.

    Contraseña

    Introduzca la clave de Active Directory.

    Nombre de dominio

    Introduzca el nombre del servidor de nombres de dominio de Active Directory

    Servidor DNS

    Introduzca la dirección IP del Servidor del Sistema de Nombres de Dominio (DNS).
    1. Revise la información en la pantalla y, a continuación, haga clic en Aplicar.
