Conectar el dispositivo My Cloud a la red

Este procedimiento explica cómo conectar el dispositivo My Cloud™ a su red doméstica mediante un cable Ethernet. Tras realizar esta conexión inicial, puede configurar la unidad para conectarla de forma inalámbrica a su red.

Siga estos pasos para conectarla mediante un cable Ethernet.

    1. Conecte un extremo del cable al puerto Ethernet de la parte trasera del dispositivo My Cloud.
    1. Conecte el otro extremo del cable directamente a un enrutador o a un puerto de conmutador de red.
    2. Conecte el extremo del adaptador al enchufe de alimentación de la parte trasera de la unidad.
    3. Enchufe el adaptador de corriente a una toma de corriente. El dispositivo My Cloud se encenderá automáticamente.

Importante: Espere a que el dispositivo My Cloud termine de encenderse (unos tres minutos) antes de configurar el conector.

Después de conectar el dispositivo My Cloud a la red con un cable Ethernet, puede utilizar la interfaz web para configurar una conexión inalámbrica. Consulte Configuración de las opciones de conexión.
