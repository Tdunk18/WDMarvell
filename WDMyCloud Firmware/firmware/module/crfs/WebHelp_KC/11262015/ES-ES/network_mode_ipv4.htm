Cambio del modo de red IPv4

Por defecto, el modo de red es DHCP, lo que significa que el dispositivo My Cloud™ detecta automáticamente la dirección IP y otros ajustes de su red. Si desea definir una dirección IP manualmente, cambie el modo completando este procedimiento.

Siga estos pasos para cambiar el modo de red IPv4.

    1. Haga clic en Configuración.
    2. En el panel izquierdo, haga clic en Red.
    3. En el área de Servicios de la red>Modo de red IPv4, seleccione el método para asignar la dirección IPv4 del dispositivo:
      • Estática: la dirección IP estática le permite tener la misma dirección IP cada vez que se conecta. Se le solicitará una dirección IP, máscara de subred, dirección IP de puerta de enlace y servidor DNS. (Si no conoce esta información, compruebe la configuración del enrutador.).
      • DHCP: el cliente DHCP hace que el dispositivo My Cloud obtenga una dirección IP y la configuración asociada automáticamente del servidor DHCP local.
        Nota: Al configurar Estática o DHCP, registre su máscara de Subred, dirección IP de puerta de enlace y Servidor DNS en un sitio seguro como referencia futura. Dicha información no se volverá a mostrar una vez que haya sido introducida en el servidor de My Cloud.

    1. Complete el asistente de configuración LAN para configurar su modo de red.
