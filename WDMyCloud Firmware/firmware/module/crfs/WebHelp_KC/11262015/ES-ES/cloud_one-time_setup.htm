Realización de una configuración única para acceso remoto

Este procedimiento explica cómo establecer una conexión segura entre el smartphone o la tableta y el dispositivo My Cloud™. Siga las instrucciones para su dispositivo móvil específico:

    • Smartphone o tablet Android  

    • iPhone o iPad

    • Teléfono o tableta Windows

    • Escritorio de Windows o Mac

Obtención de acceso seguro para un smartphone o una tableta Android

    1. Conecte su smartphone o su tableta siguiendo uno de estos procedimientos:

    • En el dispositivo móvil, toque Configuración.

    • Toque Wi-Fi y Redes.

    • Toque Configuración Wi-Fi.

    • Seleccione la red doméstica.

    • Introduzca la contraseña de la red y toque Conectar.

    1. Abra una aplicación de WD.
    2. Toque Conectar ahora.
    3. Toque el nombre del dispositivo My Cloud.
    4. Toque la flecha hacia abajo y seleccione su nombre de usuario.
    5. Realice alguna de las siguientes acciones:

    • Si hay una contraseña establecida para ese usuario, introdúzcala y toque Iniciar sesión.

    • Si no hay ninguna contraseña establecida, toque Iniciar sesión.

    1. (Opcional) Si desea acceder a cualquier parte del contenido, toque el nombre de un archivo compartido y navegue hasta el archivo deseado.

Obtención de acceso seguro para un iPhone o un iPad

    1. Conecte su smartphone o su tableta siguiendo uno de estos procedimientos:

    • En el dispositivo móvil, toque Configuración.

    • Toque Wi-Fi.

    1. Toque Configuración Wi-Fi.

    2. Seleccione la red doméstica.

    3. Introduzca la contraseña de la red y toque Conectarse.

    1. Abra una aplicación de WD.
    2. Toque Conectar ahora.
    3. Toque el signo más (+) junto al nombre del dispositivo My Cloud.
    4. Toque el nombre de usuario deseado.
      Si hay una contraseña establecida para el usuario, introdúzcala.
    5. (Opcional) Si desea acceder a cualquier parte del contenido, toque el nombre de un archivo compartido y navegue hasta el archivo deseado.

Obtención de acceso seguro para un smartphone o una tableta Windows

    • Conecte su smartphone o su tableta siguiendo uno de estos procedimientos:

    • En el dispositivo informático, toque Ajustes.

    • Toque Wi-Fi.

    • Seleccione la red doméstica.

    • Introduzca la contraseña de la red y toque Listo.

    1. Abra una aplicación de WD.
    2. Toque Conectar ahora.
    3. Toque el nombre de la unidad My Cloud.
    4. Toque la flecha hacia abajo y seleccione un nombre de usuario.
      Si hay una contraseña establecida para el usuario, introdúzcala.
    5. (Opcional) Si desea acceder a cualquier parte del contenido, haga lo siguiente:

    • En la pantalla Dispositivos, toque el nombre de su unidad My Cloud.

    • Haga clic en un recurso compartido y, a continuación, desplácese hasta el contenido deseado.

Obtención de acceso seguro para escritorio o portátil de Windows o Mac

    1. Después de instalar WD My Cloud para aplicación de escritorio, haga doble clic en el acceso directo del escritorio para mostrar todos los dispositivos WD de la red. Para obtener información sobre cómo instalar la aplicación, consulte Introducción al Acceso remoto.
    2. Es posible que el sistema muestre información acerca de las alertas de seguridad. Para impedir que se muestre esta pantalla, seleccione la casilla de verificación Acepto.
    3. Haga clic en Acepto para mostrar los dispositivos de la red.
    4. Haga clic en el dispositivo al que desea conectarse, en este caso el dispositivo My Cloud. Una pantalla de inicio de sesión aparece.
    5. Introduzca la contraseña del dispositivo My Cloud, si ya la estableció anteriormente, y haga clic en Iniciar sesión. La pantalla de introducción aparece.
    6. Para obtener más información, haga clic en el vínculo Centro de aprendizaje de WD.
    7. Haga clic en Inicio para mostrar los archivos compartidos del dispositivo My Cloud, así como los archivos de copia de seguridad de SmartWare o Time Machine.
