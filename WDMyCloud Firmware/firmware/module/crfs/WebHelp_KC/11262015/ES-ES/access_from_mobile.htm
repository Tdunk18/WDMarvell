Acceso al contenido desde un smartphone o una tableta

Puede usar su smartphone o su tableta para acceder al contenido almacenado en el dispositivo My Cloud. Antes de poder hacerlo, debe haber completado previamente las siguientes tareas: Configuración del acceso a la nube, Descargar aplicaciones móviles WD y Realización de una configuración única para acceso remoto.

Siga estos pasos para acceder al contenido desde un smartphone o una tableta.

    1. Asegúrese de que su dispositivo móvil tiene conexión Wi-Fi o 3G o superior.
    2. Inicie una aplicación móvil de WD (WD My Cloud o WD Photos).
    3. Si se le solicita, toque Conectar ahora.
    4. Seleccione su dispositivo My Cloud.
    5. Seleccione el recurso compartido al que desea acceder y, a continuación, desplácese hasta el contenido deseado.
