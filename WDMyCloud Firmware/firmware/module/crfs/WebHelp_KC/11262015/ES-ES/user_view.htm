Ver información de usuario

Siga estos pasos para ver la información de la cuenta de usuario seleccionada. En los detalles se incluyen los privilegios de acceso que se han concedido.

    1. Haga clic en Usuarios.
    2. En el panel izquierdo, seleccione un usuario. En el panel derecho se muestra la siguiente información:

Sección

Valores y descripciones

Perfil del usuario

En esta área se muestra la información básica del usuario seleccionado, que incluye:

  • Nombre de usuario: el nombre utilizado para iniciar sesión en el dispositivo.

  • Nombre: el nombre del usuario.

  • Apellidos: los apellidos del usuario.

  • Dirección de correo electrónico: la dirección de correo electrónico del usuario. Esto también se utiliza para acceder a My Cloud.

  • Contraseña: le permite activar/desactivar la contraseña de usuario. Consulte Agregar/Cambiar contraseña de usuario

  • Pertenencia al grupo: identifica a qué grupos pertenece el usuario. Aquí puede modificar la pertenencia a un grupo de un usuario.

  • Cuotas: aquí puede ver y establecer cuotas de usuario. Consulte Establecer cuotas de usuario.

Compartir acceso

Muestra los archivos compartidos disponibles e indica el tipo de acceso que el usuario tiene a esos archivos. Cambie el acceso de un usuario a un archivo compartido haciendo clic en una de las siguientes opciones:

  •  Acceso de lectura/escritura: puede ver y actualizar el archivo compartido.

  •  Acceso de solo lectura: puede ver el archivo compartido pero no puede actualizarlo.

  •  Denegar acceso. El usuario no tiene acceso a este archivo compartido.
