Activar/desactivar el acceso a la nube

Si es administrador, puede activar/desactivar el acceso a la nube para todos los usuarios (es decir, puede controlar si los dispositivos informáticos pueden acceder al contenido del dispositivo My Cloud™).

Siga estos pasos para activar/desactivar el acceso a la nube.

    1. Haga clic en Configuración.
    2. En el panel izquierdo, haga clic en General.
    3. En el campo Acceso a la nube>Servicio de nube, haga clic en para activar o desactivar el servicio de nube, como proceda.
    • Activado: Habilita el acceso remoto para todos los usuarios.
    • Desactivado: Deshabilita el acceso remoto para todos los usuarios.
    1. Haga clic en Configurar para cambiar el tipo de conexión que utiliza para su acceso a la nube. En la pantalla opciones de conexión para acceso a la nube, hay tres opciones de acceso.
    2. Automática: Automática utiliza UPnP (Plug and Play Universal) para intentar abrir los puertos en el enrutador. Si tiene éxito, se establece una conexión directa entre su dispositivo y sus aplicaciones.
    3. Manual: Establece una conexión a través de los dos puertos seleccionados. Si no hay ningún puerto disponible, se establece una conexión de retransmisión. Se requiere una configuración manual del enrutador para esta opción. Consulte Knowledge Base Answer ID 8526 para obtener información sobre cómo configurar el enrutador manualmente.
    4. Win XP: Establece una conexión a través de los puertos 80 y 443. Si estos puertos no están disponibles, se establece una conexión de retransmisión. Esta opción es obligatoria si utiliza Windows XP.
      Nota: De forma predeterminada, el dispositivo My Cloud establece automáticamente una conexión directa entre los dispositivos móviles y el enrutador.
    1. En el campo Base datos contenidos, haga clic en Reconstruir para reconstruir la base de datos WDMyCloud.
      Nota: realice esta opción solo si sospecha que la base de datos está dañada.
    2. Haga clic en Aplicar. El campo Estado de conexión indica que su dispositivo está conectado.

    Para ver detalles sobre la desactivación del acceso para un dispositivo en particular, consulte Eliminación de un dispositivo.

    Nota: Hay disponible información sobre el área Opciones de conexión en Configuración de opciones de conexión, que explica las diferentes opciones de cortafuegos e incluye una explicación sobre cómo mejorar el rendimiento. De forma predeterminada, la unidad My Cloud establece automáticamente una conexión directa entre los dispositivos móviles y el enrutador.

    Si el acceso móvil se realiza a través de una conexión de retransmisión (es decir, el estado de conexión dice “Conectado (retransmitir)”, puede mejorar el rendimiento habilitando el enrutamiento de puertos en el conector WD Venue (consulte Habilitar el enrutamiento de puertos).
