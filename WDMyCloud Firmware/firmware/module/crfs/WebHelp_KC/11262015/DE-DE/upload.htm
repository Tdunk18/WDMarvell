Hochladen von Inhalten auf das My Cloud -Gerät

Zum Hochladen von Fotos, Videos, Musik und Dateien auf Ihr My Cloud -Gerät verwenden Sie eine der folgenden Vorgehensweisen:

    • Mac
    • Windows PC
    • USB

Hochladen von Inhalt von einem Mac-Computer

So laden Sie Dateien von Ihrem Mac-Computer auf Ihr My Cloud -Gerät hoch:

    1. Wählen Sie auf Ihrem Mac in der Menüleiste Go > Netzwerk.
    2. Führen Sie einen der folgenden Schritte aus:
    • Doppelklicken Sie im linken Fenster des Finders auf den Namen des My Cloud -Geräts.
    • Suchen Sie im rechten Fenster den Namen des My Cloud -Geräts und doppelklicken Sie darauf.
    1. Doppelklicken Sie auf den Namen einer bestimmten Freigabe, um den Ordner zu öffnen und alle vorhandenen Unterordner anzuzeigen. (Öffentlich ist eine automatisch erstellte Freigabe.)
    2. Ziehen Sie Dateien aus Ihrem Computer in den gewünschten Ordner, und legen Sie sie ab.
      Legen Sie keine Dateien in den Time Machine- und WD SmartWare Backups-Ordnern ab.
      Hinweis: Wenn Sie Dateien vom Computer auf das My Cloud -Gerät kopieren, befinden sich diese Dateien an zwei Speicherorten: am ursprünglichen Speicherort und auf Ihrem Gerät.

Hochladen von Inhalt von einem PC

So laden Sie Dateien von Ihrem Windows-PC auf das My Cloud -Gerät hoch:

    1. Öffnen Sie auf Ihrem Computer das Dialogfeld „Ausführen“, indem Sie gleichzeitig die Windows- und die R-Taste drücken.
    2. Geben Sie in das Feld „Öffnen“ \\ gefolgt vom Namen des My Cloud -Geräts ein. (Der Standardnamen lautet wdmycloud ohne Leerzeichen). Beispiel: \\wdmycloud
    3. Klicken Sie auf OK.
      Windows Explorer oder Datei-Explorer wird aufgerufen. Die verfügbaren Freigaben werden im rechten Fenster aufgeführt.
    4. Doppelklicken Sie auf den Namen einer bestimmten Freigabe, um den Ordner zu öffnen und alle Unterordner anzuzeigen. (Öffentlich ist eine automatisch erstellte Freigabe.)
    5. Ziehen Sie Dateien aus Ihrem Computer in diese Freigabe, und legen Sie sie ab.
      Hinweis: Wenn Sie Dateien vom Computer auf das My Cloud -Gerät kopieren, befinden sich diese Dateien an zwei Speicherorten: am ursprünglichen Speicherort und auf Ihrem Gerät.

     

    Hochladen von Inhalt von einem USB-Gerät

    Informationen zum schnellen Hochladen von USB-Laufwerksinhalten erhalten Sie unter Erstellen von Datensicherungen des vorderen USB-Anschlusses über die Taste „USB-Kopie“.
