Einrichten des Cloud-Zugriffs

Für den Remotezugriff auf Ihre Cloud von einem Smartphone, Tablet oder Computer aus ist ein My Cloud™ -Anmeldename erforderlich. (Grundsätzlich wird der Remotezugriff von einem Gerät aus über den My Cloud -Zentralserver ermöglicht, der Zugriff auf die mit Ihrem Heimnetzwerk verbundenen NAS-Geräte gewährt.) Die Art und Weise des Remotezugriffs von Mobilgeräten (Smartphones und Tablets) und Computern unterscheidet sich jedoch geringfügig. Jeder Benutzer muss über einen WD My Cloud-Anmeldenamen verfügen.

Gehen Sie folgendermaßen vor, um den Remotezugriff einzurichten:

    1. Stellen Sie sicher, dass der Cloud-Zugriff auf dem Gerät aktiviert ist. Wie geht das?
    2. Klicken Sie auf Cloud-Zugriff.
    3. Klicken Sie im linken Bereich auf den Benutzer, dem Sie Cloud-Zugriff gewähren möchten.
    4. Klicken Sie in der WDMyCloud.com-Anmeldung für <Benutzername> auf Anmelden. Dadurch wird eine E-Mail an den Benutzer gesendet, die Anweisungen zur Registrierung bei WD My Cloud enthält.
    1. Führen Sie einen der folgenden Schritte aus:
    • Setzen Sie den Registrierungsprozess selbst fort. Wie geht das?  
    • Benachrichtigen Sie den Benutzer, dass Sie ein Webkonto eingerichtet haben und lassen Sie ihn mit dem Registrierungsprozess fortfahren.
