Konfigurieren von Verbindungsoptionen

Sie können eine Verbindungsoption angeben um festzulegen, wie Mobilgeräte und Computer remote auf das My Cloud™ -Gerät zugreifen.

Gehen Sie folgendermaßen vor, um die Verbindungsoptionen zu konfigurieren:

    1. Klicken Sie auf Einstellungen.
    2. Klicken Sie im linken Fenster auf Allgemein.
    3. Klicken Sie im Bereich „Cloud-Zugriff“ im Feld „Cloud-Dienst“ auf Konfigurieren.
    4. Klicken Sie im Dialogfeld auf eine der drei Einstellungen:
    • Automatisch: Es wird eine Direktverbindung zwischen Ihrem Mobilgerät und dem Router hergestellt. Kann die Verbindung keine Ports über uPnP (Universal Plug and Play) herstellen, wird eine Relaisverbindung aufgebaut.
    • Manuell: Es wird eine Verbindung zwischen den beiden gewählten Ports hergestellt. Mit dieser Option wird mit Portweiterleitung eine Direktverbindung eingerichtet. Eine schrittweise Anleitung zur Verwendung dieser Option finden Sie unter Aktivieren der Portweiterleitung. Im Rahmen dieser Option müssen Sie Ihren Router manuell entsprechend konfigurieren.
    • Win XP: Eine Verbindung wird über die Ports 80 und 443 hergestellt. Wenn diese Ports nicht verfügbar sind, wird eine Relaisverbindung hergestellt. Diese Option muss ausgewählt werden, wenn auf mindestens einem Ihrer Remotegeräte das Betriebssystem Windows XP läuft.
    1. Klicken Sie auf Übernehmen.
