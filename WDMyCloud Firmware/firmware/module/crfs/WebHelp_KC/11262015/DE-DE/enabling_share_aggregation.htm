Aktivieren der Aggregationsfreigabe

Aggregationsfreigabe ist vergleichbar mit dem Distributed File System (DFS), das Freigaben von anderen My Cloud™ -Systemen oder Netzwerkgeräten, die das SMB-Protokoll unterstützen, konsolidiert. Führen Sie die nachfolgend genannten Schritte aus, um die Aggregationsfreigabe für das My Cloud -Gerät zu aktivieren.

    1. Klicken Sie auf Einstellungen.
    2. Klicken Sie im linken Fenster auf Netzwerk.
    3. Klicken Sie im Bereich „Netzwerkdienste>Aggregationsfreigabe" auf , um Aggregationsfreigae zu aktivieren.
    4. Geben Sie den Stammordnernamen ein. Der Stammordner funktioniert als Container für verknüpfte Remotefreigaben und wird unter der Aggregationsfreigabe erstellt.
    5. Klicken Sie auf Übernehmen.
    6. Klicken Sie auf Link hinzufügen.
    7. Geben Sie folgende Informationen ein:

    Name des lokalen Ordners

    Geben Sie einen Namen für den Ordner ein, der unter dem Stammordner angezeigt wird.

    Remotehost

    Geben Sie den Hostnamen oder die IP-Adresse des Zielgeräts ein.

    Remotefreigabe

    Geben Sie den Remote-Freigabenamen ein, oder klicken Sie auf „Remotefreigabeordner abrufen", und wählen Sie dann rechts in diesem Feld die Freigabe aus dem Dropdown-Menü aus.
    1. Klicken Sie auf Übernehmen.
    2. Klicken Sie auf Abbrechen, um das Fenster „Einstellungen der Aggregationsfreigabe" zu schließen.
