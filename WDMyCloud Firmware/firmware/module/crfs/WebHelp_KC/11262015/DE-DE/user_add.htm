Hinzufügen eines Benutzerkontos

Führen Sie die nachfolgend genannten Schritte aus, um einen Benutzer zum My Cloud™ -Gerät hinzufügen:

    1. Klicken Sie auf Benutzer.
    2. Klicken Sie auf das Symbol Benutzer hinzufügen.
    3. Geben Sie einen Benutzernamen ein. Dieser Name wird zur Anmeldung am System verwendet.
    4. Geben Sie einen Vornamen ein. Dies ist der Vorname des Benutzers.
    5. Geben Sie einen Nachnamen ein. Dies ist der Nachname des Benutzers.
    6. Geben Sie eine E-Mail-Adresse ein. Dies ist die E-Mail-Adresse des Benutzers. Diese Adresse wird auch zur Erstellung des WD My Cloud-Kontos verwendet.
    7. Klicken Sie auf Übernehmen. Der neue Benutzer wird im linken Bereich angezeigt.
