Hinzufügen eines Mobilgeräts

Mit den Apps WD Photos und My Cloud™ können Sie von einem Mobilgerät (beispielsweise einem Smartphone oder Tablet) aus auf die Inhalte in der Cloud zugreifen. Laden Sie die kostenlosen Apps auf Ihr Mobilgerät herunter, bevor Sie den hier beschriebenen Vorgang beginnen.

So fügen Sie ein Mobilgerät hinzu:

  1. Sofern noch nicht erfolgt, aktivieren Sie Remotezugriff auf Ihre Cloud für alle Benutzer.
  2. Klicken Sie in der Navigationsleiste Ihres Geräts auf Cloud-Zugriff.
  1. Klicken Sie im linken Fenster auf den Namen des Benutzers, dessen Mobilgerät Sie Zugriff auf die Cloud gewähren möchten.
  1. Klicken Sie im Bereich „Cloud-Geräte für <Benutzername>" auf Code abrufen. My Cloud zeigt einen Code an, den Sie innerhalb von 24 Stunden in Ihr Mobilgerät eingeben müssen. Notieren Sie diesen Code, um ihn jederzeit präsent zu haben.
  1. Klicken Sie auf OK.
  2. Installieren Sie auf Ihrem Mobilgerät die Apps WD Photoes und/oder WD My Cloud.
  3. Geben Sie bei Aufforderung den Aktivierungscode ein, wenn Sie nicht mit Ihrem lokalen Netzwerk verbunden sind.
    Nach der Aktivierung Ihres Geräts können Sie über die Apps WD Photos und WD My Cloud auf die Cloud-Inhalte zugreifen.
