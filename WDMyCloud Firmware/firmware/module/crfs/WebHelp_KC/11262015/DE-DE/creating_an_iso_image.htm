Erstellen eines ISO-Abbilds

Ein ISO-Abbild ist eine Datei, die eine exakte Kopie eines vorhandenen Dateisystems darstellt. Gehen Sie folgendermaßen vor, um ein ISO-Abbild zu erstellen:

Gehen Sie folgendermaßen vor, um ein ISO-Abbild zu erstellen

    1. Klicken Sie auf Einstellungen.
    2. Klicken Sie im linken Fenster auf Hilfsprogramme.
    3. Klicken Sie im Bereich „ISO-Freigabe" auf „ISO-Abbild erstellen".
    4. Wählen Sie im Feld „Abbildgröße" im Dropdownmenü die Bildgröße aus, die Sie für das ISO-Abbild verwenden möchten.
    5. Geben Sie im Feld „Abbildpfad" den Pfad für die Daten ein, von denen Sie ein Abbild erstellen möchten, oder klicken Sie auf Durchsuchen, um den Ordner auszuwählen, von dem Sie ein Abbild erstellen möchten, und klicken Sie dann auf OK.
    6. Geben Sie im Feld „Auftragsname“ einen Namen für das ISO-Abbild ein.
    7. Klicken Sie auf Weiter.
    8. Klicken Sie im Feld „Auswahl" auf Überschreiben oder Überspringen.
    • Überschreiben: Bei Auswahl von „Überschreiben" werden Dateien, deren Namen bereits vorhanden ist, überschrieben.
    • Überspringen: Bei Auswahl von „Überspringen" werden Dateien, deren Namen bereits vorhanden ist, übersprungen.
    1. Wählen Sie in der linken Spalte die Ordner und Dateien aus, die Sie in Ihre Abbilddatei einschließen möchten, und klicken Sie dann auf Hinuzfügen>>.
    2. Wählen Sie die Ordner und Dateien aus, die Sie aus dem ISO-Abbild entfernen möchten, und klicken Sie dann auf <<entfernen.
    3. Klicken Sie auf Weiter. Das neue ISO-Abbild wird erstellt.
    1. Klicken Sie auf Fertig stellen.
