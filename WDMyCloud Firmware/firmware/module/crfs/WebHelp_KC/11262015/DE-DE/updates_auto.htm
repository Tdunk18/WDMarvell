Automatisches Aktualisieren des My Cloud -Geräts

Durch die Automatische Aktualisierung können Sie das My Cloud™ -Gerät so einrichten, dass bequem nach verfügbaren Software-Aktualisierungen gesucht wird und diese automatisch installiert werden. Das ist die einfachste Möglichkeit um sicherzustellen, dass die My Cloud -Firmware aktuell ist.

Hinweis: Nach der Installation einer Firmware-Aktualisierung führt das My Cloud -Gerät möglicherweise einen Neustart durch. Da sich ein Neustart auf die Verfügbarkeit der Cloud für die Benutzer auswirkt, sollten Sie die Aktualisierungen für bestimmte Zeiten planen, wenn möglichst wenige Benutzer auf die Cloud zugreifen.

Hinweis: Wenn Sie Automatische Aktualisierung aktivieren, werden Sie aufgefordert, Ihr System erneut zu starten, wenn die Aktualisierung abgeschlossen ist.

Führen Sie die nachfolgend genannten Schritte aus, um das My Cloud -Gerät automatisch zu aktualisieren:

    1. Klicken Sie auf Einstellungen.
    2. Klicken Sie im linken Fenster auf Firmware-Aktualisierung.
    3. Klicken Sie im Bereich „automatische Aktualisierung“ auf , um die automatischen Aktualisierungen zu aktivieren.
    4. Legen Sie im Feld „Aktualisierungszeitplan“ den Tag und die Uhrzeit für die Aktualisierung fest:

    • Klicken Sie auf den ersten Dropdownpfeil und wählen Sie den Wochentag aus.

    • Klicken Sie auf den zweiten Dropdownpfeil und wählen Sie eine Uhrzeit aus.

    • Klicken Sie auf den dritten Dropdownpfeil und wählen Sie AM oder PM aus.

    1. Klicken Sie auf Speichern.
