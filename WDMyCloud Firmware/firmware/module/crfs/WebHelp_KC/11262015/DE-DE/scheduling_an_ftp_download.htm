Planen eines FTP-Downloads

Die FTP-Download-App ermöglicht Ihnen, einen FTP-Download zu Ihrem My Cloud™ zu planen. Führen Sie die folgenden Schritte aus, um einen FTP-Download zu planen.

    1. Klicken Sie auf Apps.
    2. Klicken Sie auf FTP-Downloads.
    3. Klicken Sie auf „Auftrag erstellen".
    4. Wählen Sie im Feld „Anmeldemethode“ die Methode aus, die zum Herunterladen über FTP verwendet werden soll:
    • Konto: Geben Sie den Benutzernamen und das Passwort für das Konto ein.
    • Anonym
    1. Geben Sie im Feld „Typ“ die Art der Informationen ein, die heruntergeladen werden soll:
    • Datei
    • Ordner 
    1. Geben Sie im URL-Feld die URL der Datei ein, die heruntergeladen werden soll.
      Beispiel: ftp://192.168.0.32/test.txt
    2. Geben Sie im Feld „Speichern unter“ den gewünschten Speicherort für die heruntergeladene Datei ein oder klicken Sie auf Durchsuchen, um den gewünschten Speicherort auszuwählen.
      Beispiel: Volume_1/Öffentlich/Freigegebene Videos 
    3. Geben Sie je nach Wunsch im Feld „Umbenennen“ einen neuen Namen für die Datei ein.
    4. Wählen Sie im Feld „Sprache“ aus dem Pulldownmenü die vom FTP-Client unterstützte Sprache aus.
    5. Klicken Sie auf , wenn es sich um eine wiederholte Sicherung handelt.
    • Wählen Sie die Sicherungsfrequenz aus (täglich, wöchentlich, monatlich).
    • Geben Sie je nach ausgewählter Frequenz die Uhrzeit, den Tag bzw. das Datum für die Sicherung ein.
    1. Geben Sie das Datum ein, an dem der Download ausgeführt werden soll (MM/TT/JJJJ), oder klicken Sie auf , um ein Datum auszuwählen.
    2. Wählen Sie aus den Pulldownmenüs eine Uhrzeit für den Download aus.
    3. Klicken Sie auf Erstellen. Der Download wird erstellt und zur Warteschlange der FTP-Downloads hinzugefügt.
