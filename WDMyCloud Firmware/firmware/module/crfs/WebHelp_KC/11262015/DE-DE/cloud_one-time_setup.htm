Durchführen eines einmaligen Setups für Secure Access

Mit dieser Vorgehensweise können Sie eine sichere Verbindung zwischen Ihrem Smartphone oder Tablet und Ihrem My Cloud™ -Gerät herstellen. Gehen Sie nach den Anweisungen für Ihr Mobilgerät vor:

    • Smartphone oder Tablet mit Android  

    • iPhone oder iPad

    • Windows Phone oder Tablet

    • Windows- oder Mac-Desktop-PC

Einrichten von Secure Access für ein Android-Smartphone oder Tablet

    1. Stellen Sie mit den folgenden Schritten eine Verbindung zwischen Ihrem Smartphone oder Tablet her:

    • Tippen Sie an Ihrem Mobilgerät auf „Einstellungen“.

    • Tippen Sie auf „WLAN und Netzwerke“.

    • Tippen Sie auf „WLAN-Einstellungen“.

    • Wählen Sie Ihr Heimnetzwerk aus.

    • Geben Sie Ihr Netzwerkpasswort ein, und tippen Sie auf Verbinden.

    1. Rufen Sie eine WD-App auf.
    2. Tippen Sie auf „Jetzt verbinden“.
    3. Tippen Sie auf den Namen des My Cloud -Geräts.
    4. Tippen Sie auf den Dropdownpfeil, und wählen Sie Ihren Benutzernamen aus.
    5. Führen Sie einen der folgenden Schritte aus:

    • Wurde ein Passwort für diesen Benutzer eingerichtet, geben Sie es ein, und tippen Sie dann auf Anmelden.

    • Wurde kein Passwort für den Benutzer eingerichtet, tippen Sie auf Anmelden.

    1. (Optional) Möchten Sie auf Ihren Inhalt zugreifen, tippen Sie auf einen Freigabenamen, und navigieren Sie dann zur gewünschten Datei.

Einrichten von Secure Access für ein iPhone oder iPad

    1. Stellen Sie mit den folgenden Schritten eine Verbindung zwischen Ihrem Smartphone oder Tablet her:

    • Tippen Sie an Ihrem Mobilgerät auf „Einstellungen“.

    • Tippen Sie auf „WLAN“.

    1. Tippen Sie auf „WLAN-Einstellungen“.

    2. Wählen Sie Ihr Heimnetzwerk aus.

    3. Geben Sie Ihr Netzwerkpasswort ein, und tippen Sie auf Verbinden.

    1. Rufen Sie eine WD-App auf.
    2. Tippen Sie auf „Jetzt verbinden“.
    3. Tippen Sie auf das Pluszeichen (+) neben dem Namen des My Cloud -Geräts.
    4. Tippen Sie auf den gewünschten Benutzernamen.
      Wurde ein Passwort für den Benutzer eingerichtet, geben Sie es ein.
    5. (Optional) Möchten Sie auf Ihren Inhalt zugreifen, tippen Sie auf einen Freigabenamen, und navigieren Sie dann zur gewünschten Datei.

Einrichten von Secure Access für ein Windows-Smartphone oder Tablet

    • Stellen Sie mit den folgenden Schritten eine Verbindung zwischen Ihrem Smartphone oder Tablet her:

    • Tippen Sie an Ihrem Gerät auf Einstellungen.

    • Tippen Sie auf „WLAN“.

    • Wählen Sie Ihr Heimnetzwerk aus.

    • Geben Sie Ihr Netzwerkpasswort ein, und tippen Sie auf Fertig.

    1. Rufen Sie eine WD-App auf.
    2. Tippen Sie auf „Jetzt verbinden“.
    3. Tippen Sie auf den Namen des My Cloud -Laufwerks.
    4. Tippen Sie auf den Dropdownpfeil, und wählen Sie Ihren Benutzernamen aus.
      Wurde ein Passwort für den Benutzer eingerichtet, geben Sie es ein.
    5. (Optional) Möchten Sie auf Ihren Inhalt zugreifen, gehen Sie folgendermaßen vor:

    • Tippen Sie auf dem Bildschirm „Geräte“ auf den Namen Ihres My Cloud -Laufwerks.

    • Tippen Sie auf eine Freigabe und navigieren Sie dann zu dem gesuchten Inhalt.

Einrichten von Secure Access auf einem Desktop oder Laptop unter Windows oder Mac

    1. Doppelklicken Sie nach der Installation der Desktop-App „WD My Cloud“ auf die Verknüpfung , die auf dem Desktop angezeigt wird, um alle WD-Geräte im Netzwerk anzuzeigen. Weitere Informationen zur Installation der App finden Sie unter Grundlagen zum Remotezugriff.
    2. Das System kann Informationen zu Sicherheitswarnmeldungen anzeigen. Um das erneute Anzeigen dieses Bildschirms zu verhindern, aktivieren Sie das Kontrollkästchen „Ich verstehe“.
    3. Klicken Sie auf „Ich verstehe“, um die Geräte in Ihrem Netzwerk anzuzeigen.
    4. Klicken Sie auf das Gerät, mit dem Sie eine Verbindung herstellen möchten, in diesem Fall das My Cloud -Gerät. Ein Anmeldebildschirm wird angezeigt.
    5. Geben Sie das Passwort des My Cloud -Geräts ein, falls Sie zuvor eines eingerichtet haben, und klicken Sie auf „Anmelden“. Der Bildschirm „Erste Schritte“ wird angezeigt.
    6. Um weitere Informationen zu erhalten, klicken Sie auf den Link zum WD-Schulungszentrum.
    7. Klicken Sie auf „Starten“, um die Freigaben des My Cloud -Geräts und die SmartWare- und Time Machine-Sicherungsdateien anzuzeigen.
    8. Klicken Sie im Download-Bereich My Cloud des Schulungszentrums auf den Link „Mit Ihrem Computer zugreifen“, um zu erfahren, wie Sie die auf Ihrem My Cloud -Gerät gespeicherten Inhalte remote verwalten und nutzen können:
