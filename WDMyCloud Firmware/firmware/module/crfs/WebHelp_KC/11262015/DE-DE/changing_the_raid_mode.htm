Ändern des RAID-Modus

Führen Sie die nachfolgend genannten Schritte aus, um den aktuellen RAID-Modus des My Cloud™ -Geräts zu ändern.

Warnung: Bei einer Änderung des RAID-Modus werden alle auf dem Gerät gespeicherten Daten und Ihre Benutzereinstellungen gelöscht. Informationen zur Speicherung Ihrer Benutzereinstellungen finden Sie unter Exportieren von Systemeinstellungen.

    1. Klicken Sie auf Speicher.
    2. Klicken Sie auf RAID, sofern dies nicht bereits ausgewählt ist.
    3. Klicken Sie unten am Bildschirm auf RAID-Modus ändern.
    4. Überprüfen Sie die Warnmeldung und klicken Sie auf OK.
    5. Wählen Sie den RAID-Modus aus, den Sie für das My Cloud -Gerät verwenden möchten. Folgende Optionen stehen zur Verfügung:

Modus

Beschreibung

JBOD

Hierbei werden ein oder mehrere Laufwerke ohne RAID-Konfiguration verwendet, die als getrennte logische Volumes verwaltet werden.

Aufteilung

Lineare Kombination aus Laufwerken zum Erstellen eines großen logischen Laufwerks.

RAID 0

Die Daten werden über mehrere Festplatten verteilt. Durch die Verwendung mehrerer Laufwerke und die damit verbundene Leistungssteigerung wird die Lese- und Aufzeichnungsgeschwindigkeit erhöht. Fällt jedoch ein Laufwerk aus, gehen alle Ihre Daten verloren.

RAID 1

Zwei Datenkopien werden gleichzeitig auf getrennten Festplatten aufgezeichnet. Wenn eine der Festplatten ausfällt, können die Daten weiterhin auf der/den anderen Festplatte(n) gelesen und geschrieben werden, bis das fehlerhafte Laufwerk ersetzt wurde.

RAID 5

Bietet eine höhere Leistung und besseren Schutz, indem Daten über 3 oder mehr Laufwerke verteilt werden und ein Viertel jedes Laufwerks für die Fehlertoleranz reserviert wird. Wenn eine der Festplatten ausfällt, können die Daten weiterhin auf der/den anderen Festplatte(n) gelesen und geschrieben werden, bis das fehlerhafte Laufwerk ersetzt wurde.

Diese Option ist nur für My Cloud -Geräte mit vier Schächten verfügbar.

RAID 10

RAID-Protokoll, in dem die Daten über die primären Datenträger verteilt werden. Von den primären Datenträgern wird ein Mirror auf den sekundären Datenträgern erstellt. Wenn eine der Festplatten ausfällt, können die Daten weiterhin auf der/den anderen Festplatte(n) gelesen und geschrieben werden, bis das fehlerhafte Laufwerk ersetzt wurde.

Diese Option ist nur für My Cloud -Geräte mit vier Schächten verfügbar.
