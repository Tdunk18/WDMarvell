Anzeigen von Benutzerinformationen

Gehen Sie folgendermaßen vor, um Informationen zu einem ausgewählten Benutzerkonto anzuzeigen: Die Details umfassen auch die gewährten Zugriffsberechtigungen.

    1. Klicken Sie auf Benutzer.
    2. Wählen Sie im linken Fenster einen Benutzer aus. Rechts werden folgende Informationen angezeigt:

Abschnitt

Werte und Beschreibungen

Benutzerprofil

In diesem Bereich werden grundlegende Informationen zum ausgewählten Benutzer angezeigt:

  • Benutzername: Dieser Benutzername wird zur Anmeldung am System verwendet.

  • Vorname: Der Vorname des Benutzers.

  • Nachname: Der Nachname des Benutzers.

  • E-Mail-Adresse: Die E-Mail-Adresse des Benutzers. Diese E-Mail-Adresse wird auch für den Zugriff auf My Cloud verwendet.

  • Passwort: Mit dieser Option kann die Verwendung eines Passworts aktiviert/deaktiviert werden. Siehe Hinzufügen/Ändern eines Benutzerpassworts.

  • Gruppenmitgliedschaft: Zeigt an, zu welcher Gruppe der Benutzer gehört. Sie können die Gruppenmitgliedschaft des Benutzers hier ändern. .

  • Kontingente: Hier können Sie Kontingente anzeigen und festlegen. Siehe Festlegen von Benutzerkontingenten.

Freigabezugriff

Zeigt die verfügbaren Freigaben und die Art des Zugriffs an, über die der Benutzer für diese Freigaben verfügt. Durch Klicken auf eine der folgenden Optionen können Sie den Zugriff eines Benutzers auf eine Freigabe ändern:

  •  Lese-/Schreibzugriff: Der Benutzer kann die Freigabe anzeigen und aktualisieren.

  •  Nur Lesezugriff: Der Benutzer kann die Freigabe anzeigen, jedoch nicht aktualisieren.

  •  Zugriff verweigern. Der Benutzer hat keinen Zugriff auf die Freigabe.
