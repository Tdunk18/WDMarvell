Zugreifen auf Inhalt von zuhause aus

Wenn Sie sich zuhause befinden und Ihre Geräte mit dem Heimnetzwerk verbunden sind, greifen Sie anders auf den auf Ihrem Laufwerk gespeicherten Inhalt zu als per Remotezugriff. Wenn Sie mit dem Heimnetzwerk verbunden sind, haben Sie folgende Möglichkeiten:

    • Zugreifen auf Ihren Inhalt von einem Smartphone oder Tablet aus
    • Zugreifen auf Ihren Inhalt von einem Computer aus

Wichtig: Vergewissern Sie sich zunächst, dass Sie die unter Einrichten von Cloud-Zugriff aufgeführten Schritte ausgeführt haben.

Zugreifen auf Inhalt von einem mit Ihrem Heimnetzwerk verbundenen Mobilgerät aus

    1. Laden Sie die WD-Mobil-Apps herunter, sofern noch nicht geschehen.
    2. Stellen Sie über Ihr Smartphone oder Tablet eine Verbindung mit Ihrem Heimnetzwerk her.
      Sollten Sie dabei Unterstützung benötigen, finden Sie diese in der entsprechenden Dokumentation zu Ihrem Smart-Gerät.
    3. Starten Sie eine WD Mobile App (WD My Cloud oder WD Photos).
    4. Wenn Sie dazu aufgefordert werden, tippen Sie auf "Jetzt verbinden".
    5. Wählen Sie Ihr My Cloud -Laufwerk aus.
    6. Wählen Sie die Freigabe aus, auf die Sie zugreifen möchten, und navigieren Sie dann zum gewünschten Inhalt.

ANFANG

Zugreifen auf Inhalt von einem mit Ihrem Heimnetzwerk verbundenen Computer aus

    1. Vergewissern Sei sich, dass Ihr Computer mit Ihrem Heimnetzwerk verbunden ist und dass Sie über Internetzugang verfügen.
    2. Laden Sie gegebenenfalls die Desktop-Anwendung „WD My Cloud“ aus dem WD-Schulungszentrum (http:.wd.com/setup herunter, und installieren Sie diese auf Ihrem Mac oder PC.
    3. Rufen Sie auf Ihrem Computer die WD My Cloud-Desktopanwendung auf.
    4. Geben Sie Ihre Anmeldedaten ein.
    5. Doppelklicken Sie auf das My Cloud -Laufwerk.
    6. Erweitern Sie im linken Bereich Ordner nach Bedarf, um zum gewünschten Inhalt zu navigieren.
