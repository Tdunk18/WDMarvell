Anfordern von Support

Sollte ein Problem mit dem My Cloud™ -Gerät auftreten, benötigt der WD Technical Support Informationen über das My Cloud -Gerät, um eine Fehlersuche durchzuführen und die beste Lösung zu ermitteln. Es gibt zwei Möglichkeiten, diese Geräteinformationen abzurufen und Support anzufordern:

    • Führen Sie einen Systembericht aus, und senden Sie ihn automatisch an WD.

    • Erstellen Sie manuell einen Systembericht aus, und senden Sie ihn an WD.

So führen Sie einen Systembericht aus und senden ihn automatisch an WD:

    1. Klicken Sie oben rechts auf .
    2. Klicken Sie auf Support.
    3. Klicken Sie im Bereich „Automatisierten Support anfordern“ auf den Link „Datenschutzrichtlinie“, um die Datenschutzrichtlinie von WD zu lesen.
    4. Aktivieren Sie das Kontrollkästchen Diagnosebericht meines Geräts beifügen und Support anfordern.
    5. Klicken Sie auf Support anfordern.
    6. Registrieren Sie sich für die WD Knowledge Base (http://support.wdc.com), sofern nicht bereits geschehen.
    7. Füllen Sie Ihre Anforderung aus, und senden Sie sie ab.

So erstellen Sie manuell einen Systembericht und senden ihn an WD:

    1. Klicken Sie oben rechts auf .
    2. Klicken Sie auf Support.
    3. Klicken Sie im Bereich „Systembericht erstellen und speichern“ auf Erstellen und speichern.
    4. Speichern Sie den Bericht auf Ihrem Computer.
    5. Versenden Sie den Bericht per E-Mail an den WD Technical Support.
