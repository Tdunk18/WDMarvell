Verwalten von Webdateien

Der Webdateibetrachter ist eine App zum Anzeigen und Verwalten der Dateien auf Ihrem My Cloudâ„¢ -Gerät.

Gehen Sie folgendermaßen vor, um Ihre Webdateien zu verwalten:

    1. Klicken Sie auf Apps.
    2. Klicken Sie auf Webdateibetrachter. Die auf dem Gerät verfügbaren Dateivolumes werden rechts angezeigt.

Anzeigen von Webdateien

Ihre Volumes, Ordner und Dateien werden rechts im Webdateibetrachter angezeigt. Doppelklicken Sie auf einen Ordner, um den Inhalt des entsprechenden Ordners bzw. Volumes anzuzeigen.

Verwalten von Webdateien

Die verfügbaren Aktionen, die Sie für Dateien und Ordner ausführen können, werden oben rechts im Webdateibetrachter angezeigt. Wählen Sie eine der folgenden Aktionen zum Ausführen auf Ihre Webdateien aus:

Symbol

Funktion

Kopieren

Mit dieser Aktion können Sie eine ausgewählte Datei an einen anderen Speicherort kopieren. Gehen Sie folgendermaßen vor, um eine Datei an einen neuen Speicherort zu kopieren:

    1. Wählen Sie im Bereich „Webdateibetrachter" die Datei aus, die Sie kopieren möchten.
    2. Klicken Sie auf .
    3. Wählen Sie den Zielort zum Kopieren der Datei aus.
    4. Klicken Sie auf OK. Die Datei wird an den ausgewählten Ort kopiert.

Verschieben

Mit dieser Aktion können Sie eine Datei von einem Speicherort zu einem anderen Speicherort verschieben. Gehen Sie folgendermaßen vor, um eine Datei an einen neuen Speicherort zu verschieben:

    1. Wählen Sie die zu verschiebende Datei aus.
    2. Klicken Sie auf .
    3. Wählen Sie den Zielort zum Verschieben der Datei aus.
    4. Klicken Sie auf OK. Die Datei wird an den ausgewählten Ort verschoben.

Heraufladen

Mit dieser Aktion können Sie eine Datei von Ihrem Computer oder einem anderen Gerät auf Ihren My Cloud -Server hochladen. Mit folgenden Schritten können Sie eine Datei auf Ihr Gerät hochladen.

    1. Wählen Sie rechts im Webdateibetrachter den Zielort für die heraufzuladende Datei aus.
    2. Klicken Sie auf .
    3. Klicken Sie im Bildschirm „Heraufladen“ auf das Symbol „Datei auswählen“.
    4. Navigieren Sie zur Datei, die Sie auf das Gerät heraufladen möchten, und wählen Sie sie aus.
    5. Klicken Sie auf Öffnen.
    6. Klicken Sie auf Heraufladen.
    • Wenn Sie mehrere Dateien heraufladen möchten, wiederholen Sie die Schritte 3 und 4, bis alle gewünschten Dateien heraufgeladen sind.
    1. Klicken Sie auf Schließen. Die heraufgeladenen Dateien werden im Zielordner angezeigt.

Herunterladen

Mit dieser Aktion können Sie eine Datei vom My Cloud -Gerät auf Ihren Computer oder andere Geräte herunterladen.
Gehen Sie folgendermaßen vor, um eine Datei zu Ihrem Computer bzw. einem anderen Gerät herunterzuladen:

    1. Wählen Sie rechts im Webdateibetrachter die herunterzuladende Datei aus.
    2. Klicken Sie auf .
    3. Die ausgewählte Datei wird auf Ihren Computer bzw. Ihr Gerät heruntergeladen.

Umbenennen

Mit dieser Aktion können Sie eine ausgewählte Datei auf dem WD My Cloud -Gerät umbenennen. Gehen Sie folgendermaßen vor, um eine Datei umzubenennen:

    1. Wählen Sie rechts im Webdateibetrachter die Datei aus, die Sie umbenennen möchten.
    2. Klicken Sie auf .
    3. Geben Sie im Feld „Umbenennen“ einen neuen Namen für die Datei ein.
    4. Klicken Sie auf Speichern. Die Datei wird umbenannt.

Löschen

Mit dieser Aktion können Sie eine ausgewählte Datei auf dem My Cloud -Gerät löschen. Gehen Sie folgendermaßen vor, um eine Datei zu löschen:

    1. Wählen Sie rechts im Webdateibetrachter die zu löschende Datei aus.
    2. Klicken Sie auf .
    3. Klicken Sie auf OK, um das Löschen der ausgewählten Datei zu bestätigen. Die ausgewählte Datei wird vom Gerät gelöscht.

Aktualisieren

Mit dieser Aktion wird die Dateiliste aktualisiert, sodass alle Dateien angezeigt werden, die seit dem letzten Aktualisieren bzw. dem letzten Zugriff auf die Dateiliste hinzugefügt wurden.

  • Klicken Sie auf . Die Dateiliste wird aktualisiert.

Erstellen

Mit dieser Aktion können Sie einen Ordner in der ausgewählten Freigabe erstellen.

    1. Navigieren Sie zur Stelle, an der Sie den neuen Ordner erstellen möchten.
    2. Klicken Sie auf
    3. Geben Sie den Namen des neuen Ordners ein.
    4. Klicken Sie auf Speichern.Der neue Ordner wird erstellt.
