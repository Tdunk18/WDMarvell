Willkommen

Endlich bekommen Sie Ihre eigene Cloud. Speichern Sie alle Daten an einem Ort und greifen Sie weltweit von Computern, Tablets oder Smartphones darauf zu. My Cloud™ -Speicher funktioniert parallel zu mehreren Betriebssystemen wie Mac OS, Windows, iOS und Android mit mobilen Apps.

Was Sie wissen sollten:

Damit Sie Ihre persönliche Cloud optimal nutzen können, sollten Sie einige wichtige Dinge zu Ihrem My Cloud -Gerät wissen:

    • Benutzerrollen

Zusätzlich sollten Sie diese Aufgaben kennen, um die Kernfunktionen des Geräts nutzen zu können:

    • Erste Schritte

    • Zugriff mit Smartphone oder Tablet

    • Zugriff mit Laptop-Computer  

    • Anzeige der My Cloud -Weboberfläche

    • Freigabe von Inhalt für andere außerhalb Ihres Netzwerks

    • Hochladen von Dateien

Benutzerrollen in Ihrer persönlichen Cloud

Der Administrator – üblicherweise der Eigentümer des Geräts – ist für die Einrichtung des Geräts verantwortlich. Als My Cloud -Geräteeigentümer verfügen Sie über ein spezielles Benutzerkonto (admin), über das Sie Administratorrechte erhalten. Mit diesen Rechten können Sie das Gerät für Ihre speziellen Anforderungen einrichten und konfigurieren sowie andere Benutzer zu Ihrer persönlichen Cloud hinzufügen. Sie können außerdem genau festlegen, auf was Benutzer auf dem Gerät zugreifen dürfen.

Standardmäßig ist dem Administrator-Benutzernamen (admin) kein Passwort zugeordnet. Sie können aber jederzeit mithilfe der in Hinzufügen/Ändern eines Benutzerpassworts erläuterten Schritte ein Passwort erstellen.

Erste Schritte mit My Cloud                                      

Sie können aus mehreren Quellen Informationen zur Herstellung einer Verbindung und zur Einrichtung Ihres Geräts erhalten:

    • Schnellinstallationsanleitung: Die Schnellinstallationsanleitung liegt Ihrem Gerät bei und erläutert den Aufbau einer Verbindung sowie die ersten Schritte bei der Nutzung des Geräts. 

    • WD Learning Center (http://www.wd.com/setup): Das WD Learning Center liefert Ihnen online die aktuellen Informationen, Anweisungen und Softwareversionen, damit Sie Ihren persönlichen Cloud-Speicher optimal nutzen können. .

    • My Cloud -Benutzerhandbuch (support.wdc.com): Im My Cloud -Benutzerhandbuch finden Sie detaillierte Informationen dazu, wie Sie Ihr Gerät verbinden, einrichten und nutzen können.

Zugreifen auf Inhalt von einem Smartphone oder Tablet aus

Zwei WD-Mobil-Apps ermöglichen Ihnen, von einem Smartphone oder Tablet auf die Inhalte Ihres My Cloud -Geräts zuzugreifen:

    • WD Photos: Hier können Sie Ihre Fotos anzeigen und verwalten.

    • WD My Cloud: Mit dieser App können Sie Fotos, Videos, Musik und Dateien anzeigen bzw. wiedergeben.

Sie erhalten diese kostenlosen Apps im WD-Schulungszentrum (http://www.wd.com/setup) und auf App-Websites wie Apple App Store, Google Play, Windows Store oder Amazon.com.

Workflow für den Mobilgerätezugriff für Sie (als Administrator)

Der Basis-Workflow für Sie als Eigentümer/Administrator von My Cloud:

1

Konfigurieren Sie das My Cloud -Gerät für den Remotezugriff, und registrieren Sie sich für wdmycloud-Anmeldedaten. Vorgehensweise

2

Laden Sie die WD-Mobil-Apps herunter. Vorgehensweise

3

Starten Sie eine WD-App und führen Sie ein einmaliges Setup für den sicheren Zugriff durch. Vorgehensweise

4

Fertig! Geben Sie Ihren Inhalt wieder, und zeigen Sie ihn an.

Workflow für den Mobilgerätezugriff für andere

HINWEIS: Beim Hinzufügen eines Benutzers zum My Cloud -Gerät geben Sie bei My Cloud die E-Mail-Adresse des Benutzers an, wodurch ein WD My Cloud-Anmeldename erstellt wird. Nach dem Erstellen der Anmeldedaten wird eine E-Mail an die Adresse des neuen Benutzers gesendet. Die E-Mail enthält eine Verknüpfung zu WDMyCloud.com, über die der Benutzer den Registrierungsvorgang fortführen kann.

Basis-Workflow für Nicht-Administratoren:

1

Klicken Sie auf den Link in der Nachricht von WDMyCloud.com, und setzen Sie die Registrierung fort.

2

Laden Sie die WD-Mobil-Apps herunter. Vorgehensweise

3

Starten Sie eine WD-App und führen Sie ein einmaliges Setup für den sicheren Zugriff durch. Vorgehensweise

4

Geben Sie Ihren Inhalt wieder, und zeigen Sie ihn an.

Zugreifen auf und Freigeben von Inhalt von außerhalb Ihres Netzwerks

Mit der WD My Cloud-Desktop-Anwendung können Sie von Ihrem Mac oder PC auf die auf dem My Cloud -Gerät gespeicherten Inhalte zugreifen. Mit My Cloud können Sie remote auf Fotos, Videos, Musik und Dateien zugreifen, die Sie auf WD-NAS-Geräten gespeichert haben. Zu diesen Geräten zählen auch My Cloud - und „My Book Live“-Laufwerke, die mit Ihrem Heimnetzwerk verbunden sind. Wie bei der WD-Mobil-App benötigen Sie auch hier die überaus wichtigen My Cloud -Anmeldedaten für den Zugriff auf Ihren Inhalt.

Laden Sie die WD My Cloud-Desktop-Anwendung unter http://www.wd.com/setup herunter.

Schrittweise Anweisungen finden Sie unter Zugreifen auf und Freigeben von Inhalt von außerhalb Ihres Netzwerks.

ANFANG

Anzeige der My Cloud -Oberfläche

Mit der Benutzeroberfläche des My Cloud -Geräts können Sie Ihre persönliche Cloud einrichten und verwalten. Die Oberfläche wird in einem auf einem Mac-Computer oder PC ausgeführten Webbrowser angezeigt. 

So lassen Sie die Weboberfläche anzeigen:

    1. Öffnen Sie ein Browserfenster.
    2. Geben Sie in der Adresszeile http://<Gerätename> ;
      Beispiel: http://wdmycloud4100
      HINWEIS: Wird mit der URL die Weboberfläche nicht angezeigt, geben Sie diese URL ein: My Cloud DL2100 http://<Gerätename>.local

Hochladen von Dateien

Laden Sie Dateien mit Ihrem Mac-Computer oder PC auf das My Cloud -Gerät hoch. Ihr Computer muss mit Ihrem Netzwerk verbunden sein. Eine schrittweise Anleitung finden Sie unter Hochladen von Inhalt auf das My Cloud -Gerät.
