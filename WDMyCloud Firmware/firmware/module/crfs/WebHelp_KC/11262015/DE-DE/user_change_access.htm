Ändern des Zugriffs auf eine Freigabe

Sie können die Zugriffsberechtigungen eines Benutzers über den Benutzerbildschirm oder den Freigabenbildschirm ändern. Führen Sie die folgenden Schritte aus, um die Zugriffsberechtigungen eines Benutzers auf eine ausgewählte Freigabe zu ändern:

Hinweis: Eine Freigabe muss auf dem Bildschirm „Freigaben“ zunächst privatisiert werden. Alle öffentlichen Freigaben sind ausgegraut. Weitere Informationen hierzu finden Sie unter Info zu Freigaben.

Freigabezugriff über den Benutzerbildschirm ändern

    1. Klicken Sie auf Benutzer.
    2. Wählen Sie im linken Fenster einen Benutzer aus.
    3. Ändern Sie den Zugriff eines Benutzers, indem Sie im Bereich „Freigabezugriff“ auf das entsprechende Symbol für eine Freigabe klicken:
    • Lese-/Schreibzugriff
    • Nur-Lese-Zugriff
    • Zugriff verweigern

 

Freigabezugriff über den Freigabenbildschirm ändern

    1. Klicken Sie auf Freigaben.
    2. Wählen Sie im linken Fenster eine Freigabe aus.
    3. Vergewissern Sie sich, dass die Option „Öffentlicher Zugriff“ deaktiviert ist. Wenn der öffentliche Zugriff aktiviert ist, können alle Benutzer die Freigabe anzeigen.

    • Zugriff verweigern
