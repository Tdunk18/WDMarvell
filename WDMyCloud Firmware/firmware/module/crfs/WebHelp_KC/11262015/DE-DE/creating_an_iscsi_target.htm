Erstellen eines iSCSI-Ziels

Gehen Sie folgendermaßen vor, um ein iSCSI-Ziel zu erstellen:

    1. Klicken Sie auf Speicher.
    2. Klicken Sie auf iSCSI.
    3. Klicken Sie auf , um iSCSI zu aktivieren.
    4. Klicken Sie unten am Bildschirm auf iSCSI-Ziel erstellen.
    5. Geben Sie im Bildschirm „iSCSI-Ziel erstellen" folgende Angaben ein:
    • Alias: Geben Sie hier einen beschreibenden Namen für das Ziel ein.
    • Erstellt am: Wählen Sie das gewünschte Volume für das iSCSI-Ziel aus.
    • Größe: Geben Sie einen Wert und eine Maßeinheit (z. B. GB, TB) für die Größe des Ziels ein.
    1. Klicken Sie auf Weiter.
    • Wenn Sie dem Ziel Sicherheit zuweisen möchten, klicken Sie auf CHAP.
    • Geben Sie einen vorhandenen Benutzernamen und das entsprechende Passwort ein.
    1. Klicken Sie auf Übernehmen. Das neue iSCSI-Ziel wird erstellt.
