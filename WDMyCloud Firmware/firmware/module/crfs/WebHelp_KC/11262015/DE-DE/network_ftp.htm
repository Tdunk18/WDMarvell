Aktivieren von FTP-Zugriff

Sie können die Dateiübertragung zwischen den Geräten in Ihrem Netzwerk über FTP (File Transfer Protocol) aktivieren beziehungsweise deaktivieren.

Gehen Sie folgendermaßen vor, um den FTP-Zugriff zu aktivieren/deaktivieren:

    1. Klicken Sie auf Einstellungen.
    2. Klicken Sie im linken Fenster auf Netzwerk.
    3. Klicken Sie im Bereich „Netzwerkdienst>FTP-Zugriff“ auf , um den FTP-Zugriff zu aktivieren bzw. zu deaktivieren.
    4. Lesen Sie die Meldung, und klicken Sie auf OK.
    5. Klicken Sie auf Konfigurieren.
    6. Geben Sie im Bildschirm „FTP-Einstellungen" folgende Angaben ein:
    • Maximale Benutzerzahl: Wählen Sie im Dropdownmenü die gewünschte maximale Benutzerzahl mit FTP-Zugriff aus.
    • Leerlaufzeit:: Geben Sie die gewünschte Leerlaufzeit in Minuten ein, die sich der FTP bis zur Unterbrechung der Verbindung in Ruhe befindet.
    • Port: Geben Sie den für den FTP-Zugriff zu verwendenden Anschluss ein.
    • Flusssteuerung: Wählen Sie Unbegrenzt oder Anpassen. geben Sie bei Auswahl von „Anpassen" einen Flusssteuerungswert ein.
    1. Klicken Sie auf Weiter.
    2. Wählen Sie im Feld „Passiver Modus" entweder Standard oder Anpassen. Geben Sie bei Auswahl von „Anpassen" einen Wert für „Passiver Modus" ein.
    3. Aktivieren Sie das Kontrollkästchen, wenn Sie externe IP im PASV-Modus (passiven) berichten möchten.
    • Geben Sie die externe IP-Adresse ein, oder klicken Sie auf IP abrufen, damit das System die Adresse eingibt.
    1. Klicken Sie auf Weiter.
    2. Geben Sie folgende Informationen ein:
    3. Client-Sprache: Wählen Sie im Dropdownmenü die Client-Sprache aus.
    4. TLS: Aktivieren Sie das Kontrollkästchen zu Transport Layer Security (TLS) für implizite bzw. explizite TLS-Verbindungen, falls zutreffen.
    5. FXP: Klicken Sie auf , um File Exchange Protocol (FXP) zu aktivieren bzw. deaktivieren.
    1. Klicken Sie auf Weiter.
    2. Geben Sie beliebige IP-Adressen ein, die Sie für den FTP-Zugriff blockieren möchten. Wählen Sie dann im Dropdownmenü entweder Dauerhaft oder Vorübergehend aus. Klicken Sie auf Übernehmen, um Ihre Eingaben zu speichern.
    3. Klicken Sie auf Fertig stellen.
