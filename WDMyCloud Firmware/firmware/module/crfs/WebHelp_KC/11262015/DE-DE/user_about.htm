Info zu Benutzern

Alle Personen, die auf Ihre persönliche Cloud zugreifen, müssen über ein Benutzerkonto für das My Cloud Gerät verfügen. Als Geräte-Eigentümer verfügen Sie über ein spezielles Benutzerkonto (admin), über das Sie Administratorrechte erhalten. Mit diesen Rechten können Sie das Gerät für Ihre speziellen Anforderungen einrichten und konfigurieren und andere Benutzer zu Ihrer persönlichen Cloud hinzufügen. Sie können genau festlegen, wer Ihr Laufwerk nutzt.

Standardmäßig ist dem Benutzernamen admin kein Passwort zugeordnet. Sie können jederzeit ein Passwort hinzufügen.

Wichtig: Wenn Sie zur Ersteinrichtung des Geräts die My Cloud -Geräteeinrichtungssoftware oder den Assistenten „Erste Schritte“ ausführen, haben Sie die Möglichkeit, den Administratornamen zu ändern und ein optionales Passwort festzulegen. Notieren Sie diesen Namen und das zugehörige Passwort für den späteren Gebrauch. Ohne diese Anmeldedaten können Sie die My Cloud -Weboberfläche nicht anzeigen.

Als Administrator für das My Cloud -Gerät haben Sie die folgenden Möglichkeiten:

    • Benutzer hinzufügen

    • Benutzerpasswörter hinzufügen oder ändern

    • Benutzerinformationen anzeigen

    • Den Zugriff eines Benutzers auf eine Freigabe ändern  

    • Benutzer entfernen

    • Benutzernamen und Passwort eines Benutzers ändern
