Ein-/Ausschalten des Cloud-Zugriffs

Als Administrator können Sie den Cloud-Zugriff für alle Benutzer ein-/ausschalten (d. h., Sie können steuern, ob Geräte auf den Inhalt des My Cloud™ -Geräts zugreifen können).

Gehen Sie folgendermaßen vor, um den Cloud-Zugriff zu aktivieren/deaktivieren:

    1. Klicken Sie auf Einstellungen.
    2. Klicken Sie im linken Fenster auf Allgemein.
    3. Klicken Sie im Feld „Cloud-Zugriff>Cloud-Dienst“ auf , um den Cloud-Dienst je nach Bedarf ein- bzw. auszuschalten.
    • Ein: Der Remotezugriff für alle Benutzer wird aktiviert.
    • Aus: Der Remotezugriff für alle Benutzer wird deaktiviert.
    1. Klicken Sie auf „Konfigurieren", um die Verbindungsart für den Cloud-Zugriff zu ändern. Am Bildschirm „Verbindungsoptionen für den Cloud-Zugriff" werden drei Zugriffsoptionen angezeigt.
    2. Automatisch: Mit „Automatisch" wird UPnP (Universal Plug and Play) für den Versuch, Anschlüsse auf Ihrem Router zu öffnen, verwendet. Bei Erfolg wird eine direkte verbindung zwischen Ihrem Gerät und Ihren Apps hergestellt.
    3. Manuell: Eine Verbindung wird über die zwei ausgewählten Ports hergestellt. Wenn einer der beiden Ports nicht verfügbar ist, wird eine Relaisverbindung hergestellt. Für diese Option ist eine manuelle Router-Konfiguration erforderlich. Weitere Informationen zur manuellen Einrichtung Ihres Routers finden Sie unter Knowledge Base-Antwort-ID 8526.
    4. Win XP: Eine Verbindung wird über die Ports 80 und 443 hergestellt. Wenn diese Ports nicht verfügbar sind, wird eine Relaisverbindung hergestellt. Diese Option ist erforderlich, wenn Sie Windows XP verwenden.
      Hinweis: My Cloud stellt standardmäßig eine direkte Verbindung zwischen Ihren Mobilgeräten und dem Router her.
    1. Klicken Sie im Feld „Inhaltsdatenbank" auf Wiederherstellen, um Ihre WDMyCloud-Datenbank wiederherzustellen.
      Hinweis: Führen Sie diese Option nur zur Fehlerbehandlung aus, wenn Sie von einer beschädigten Datenbank ausgehen.
    2. Klicken Sie auf Übernehmen. Das Feld „Verbindungsstatus" zeigt an, dass Ihr Gerät verbunden ist.

    Einzelheiten für die Deaktivierung des Zugriffs für ein einzelnes Gerät finden Sie unter Entfernen eines Geräts.

    Hinweis: Informationen zu den Verbindungsoptionen finden Sie unter Konfigurieren von Verbindungsoptionen. Hier werden die verschiedenen Firewall-Optionen erläutert und erklärt, wie die Leistung gesteigert werden kann. My Cloud stellt standardmäßig eine direkte Verbindung zwischen Ihren Mobilgeräten und dem Router her.

    Erfolgt der mobile Zugriff über eine Relaisverbindung (als Verbindungsstatus wird „Verbunden (Relais)“ angegeben), können Sie die Leistung steigern, indem Sie die Portweiterleitung am WD-Venue-Connector aktivieren (siehe Aktivieren der Portweiterleitung).
