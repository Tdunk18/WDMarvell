Criando um backup interno

Os backups internos permitem fazer cópias de segurança do conteúdo existente de um local de armazenamento para outro no seu dispositivo My Cloud™.

Use as etapas a seguir para criar um backup interno.

    1. Clique em Backups.
    2. Clique em Backups internos.
    3. Clique em Criar trabalho.
    4. Digite a seguinte informação para criar um trabalho de backup interno:

      Nome do trabalho

      Digite um nome de trabalho para seu backup interno.

      Pasta de origem

      Clique em Buscar e selecione a pasta que deseja fazer o backup.

      Pasta de destino

      Clique em Buscar e selecione a pasta de destino para seu backup.

      Tipo de backup

      Indica o tipo de backup que deseja realizar.

      • Copiar
      • Sincronizar
      • Adicional

      Repetição

      Se quiser usar o recurso Recorrência:

      1. Clique no botão alternar para habilitar o recurso de Recorrência.
      2. Selecione a frequência do backup: Diária, semanal, mensal.
      3. Selecione o horário (hora, AM/PM) no menu suspenso.
    5. Clique em Criar. Seu trabalho aparece na Fila de Backup Interno e começará no horário indicado.
