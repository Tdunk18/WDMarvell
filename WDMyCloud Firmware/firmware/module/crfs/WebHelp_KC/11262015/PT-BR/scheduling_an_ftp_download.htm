Programando um download via FTP

O aplicativo FTP Download permite programar o download de um FTP para seu My Cloud™ Use as etapas a seguir para programar.

    1. Clique em Aplicativos.
    2. Clique em Downloads via FTP.
    3. Clique em Criar trabalho.
    4. No campo Método de login, selecione o método que gostaria de usar para o download do seu FTP:
    • Conta: Informe o nome de usuário e senha para a conta.
    • Anônimo
    1. No campo Tipo, selecione o tipo de informação que você gostaria de baixar:
    • Arquivo
    • Pasta 
    1. No campo de URL, informe o URL do arquivo que você gostaria de baixar.
      Exemplo: ftp://192.168.0.32/test.txt
    2. No campo Salvar Em, digite o local onde deseja salvar o download ou clique em Buscar para selecionar o local.
      Exemplo: Volume_1/Public/Shared Videos 
    3. No campo Renomear, digite um novo nome para o arquivo, se quiser alterá-lo.
    4. No campo Idioma, selecione o idioma aceito pelo cliente de FTP para o menu suspenso.
    5. Clique em se este for um backup recorrente.
    • Selecione a frequência do backup (diariamente, semanalmente ou mensalmente)
    • Insira o horário, dia e data do backup, dependendo da frequência selecionada.
    1. Digite a data em que deseja realizar o download (mm/dd/aaaa) ou selecione  para selecionar uma data.
    2. Insira um horário para seu download dos menus suspensos.
    3. Clique em Criar. Seu download é criado e adicionado à fila de downloads via FTP.
