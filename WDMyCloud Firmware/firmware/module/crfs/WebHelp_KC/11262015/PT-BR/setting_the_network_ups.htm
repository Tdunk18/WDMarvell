Configuração do UPS de rede

Um Uninterruptible Power Supply (UPS - Alimentação Contínua) é um dispositivo que funciona sem bateria e que mantém os dispositivos como se estivessem conectados à porta UPS em funcionamento, por, pelo menos, um curto período, caso haja queda da energia principal. Quando o dispositivo My Cloud™ estiver conectado por UPS à unidade da rede, a unidade irá desligar automaticamente quando a UPS for drenada para uma porcentagem específica.

A comunicação com o dispositivo My Cloud é feita usando um protocolo mestre/escravo em que o dispositivo mestre controla os escravos.

Quando a UPS estiver conectada ao dispositivo My Cloud pelo USB, o dispositivo automaticamente se torna um UPS Network Master. Outros dispositivos My Cloud que estão conectados a UPS podem ser indicados ao endereço de IP do mestre UPS. Se o UPS for descarregado a uma porcentagem específica, todas as unidades escravas do My Cloud irão fechar automaticamente de modo correto. Consulte a Resposta Base de Conhecimento ID 11852 para ver uma lista de dispositivos UPS.

Use as etapas a seguir para configurar a UPS da rede para seu dispositivo My Cloud™.

    1. Clique em Configurações.
    2. No painel na esquerda, clique em Rede.
    3. Na área UPS de rede > Escravo, clique em  para habilitar o DNS dinâmico.
    4. Digite o endereço de IP para o mestre UPS em rede.
    5. Clique em Seguinte. Uma tela "em progresso" aparece e, quando a configuração for atualizada, a tela de rede aparece com o escravo UPS da rede desligado.
