Criar um ISO Share

Use as etapas abaixo para criar um compartilhamento ISO.

    1. Clique em Configurações.
    2. No painel esquerdo, clique em Utilitários.
    3. Na área da Lista de ISO Share, clique em Criar ISO share
    4. Busque o arquivo ISO que deseja incluir no seu ISO share e coloque uma marca ao lado.
    5. Clique em Seguinte.
    6. Digite uma descrição para o ISO share e clique em Seguinte.
    7. Clique em Seguinte.
    8. Complete os campos abaixo:
    • Habilitar servidor de mídia: Para habilitar o servidor de mídia para o ISO Share, clique no botão para alternar
    • Acesso FTP: Para habilitar o acesso FTP para o ISO Share, clique no botão para alternar.
    • Acesso WebDAV: Para habilitar o acesso WebDAV para o ISO Share, clique no botão para alternar.
      Observação: Se o acesso FTP ou WebDAV estiver desabilitado para o dispositivo, os respectivos campos estarão em cinza.
    1. Clique em Seguinte.
    2. Clique no botão alternar para habilitar o acesso NFS para o ISO Share.
      Observação: Se o acesso NFS estiver desabilitado para o dispositivo, os respectivos campos estarão em cinza.
    3. Clique em Aplicar. Seu ISO Share foi criado.
