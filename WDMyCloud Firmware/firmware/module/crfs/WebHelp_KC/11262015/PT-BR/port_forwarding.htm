Ativando o encaminhamento de porta

Se o acesso remoto à sua nuvem estiver ativado por meio de uma conexão com retransmissão, você pode melhorar o desempenho da rede ativando o encaminhamento de porta.

Observação: Para determinar se a sua conexão de acesso móvel está restabelecida, consulte Verificar status do acesso remoto.

Importante: Antes de começar este procedimento, é preciso saber o intervalo válido nos números de porta que podem ser usados para o roteador e selecionar as portas apropriadas para HTTP e HTTPS. O intervalo de números de porta válidos é 1024 - 66535. As portas normalmente atribuídas são 80 para HTTP e 443 para HTTPS.

Use as etapas a seguir para ativar o encaminhamento de porta.

    1. Clique em Configurações.
    2. No painel na esquerda, clique em Geral.
    3. Na seção Acesso à nuvem, clique em para ativar o Acesso à nuvem, se aplicável.
    4. Clique em Configurar.
    5. No campo Conectividade, clique em Manual.
    6. No campo Porta externa 1, insira o número da porta pela qual as sessões HTTP ocorrerão.
    7. No campo Porta externa 2, insira o número da porta pela qual as sessões HTTPS ocorrerão.
    8. Clique em Aplicar.
    9. Configure o seu roteador usando as portas especificadas nas etapas 6 e 7.
