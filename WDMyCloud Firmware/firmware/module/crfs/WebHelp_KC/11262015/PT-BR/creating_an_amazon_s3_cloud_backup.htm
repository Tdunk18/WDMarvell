Criando um backup de nuvem do Amazon S3

Observação: Antes de criar seu backup de nuvem, registre-se no serviço Amazon S3 em http://aws.amazon.com/.

Quando a conta Amazon S3 estiver configurada, use as seguintes etapas para criar um backup de nuvem.

    1. Clique em Backups.
    2. Clique em Backups de Nuvem.
    3. Clique no botão na parte superior da tela Amazon S3.
    4. Clique em Criar.
    5. Insira um nome de trabalho para seu backup de nuvem.
    6. Clique em Seguinte.
    7. Insira as seguintes informações fornecidas pelo Amazon S3:
    • Região: Selecione a Região usada para sua nuvem a partir do menu suspenso.
    • Chave de acesso: Insira a chave de acesso fornecida para você pelo Amazon S3.
    • Chave privada: Insira a chave privada fornecida para você pelo Amazon S3.
    • Atalho remoto: Inserira o caminho remoto para sua nuvem. Este normalmente é seu nome de Bucket.
    1. Clique em Seguinte.
    2. No campo Tipo selecione uma das seguintes opções:
    • Upload: Faz o backup dos seus dados do My Cloud para seu conteúdo do Amazon S3.
    • Download: Faz o backup do seu conteúdo do Amazon S3 ao seu servidor My Cloud.
    1. No campo Tipo de Backup, selecione o tipo de backup que você deseja executar a partir do menu suspenso. As opções incluem:
    • Substituir arquivos existentes: Substitua arquivos na pasta de destino que têm nome idêntico ao seu arquivo de origem.
    • Backup completo: Crie uma pasta separada contendo todos os dados de backup cada vez que o backup for realizado.
    • Backup adicional: Substitua arquivos por arquivos de origem que são mais recentes do que os arquivos de destino.
    1. Clique em Seguinte.
    2. No campo Atalho Local, digite um atalho para seu backup no dispositivo My Cloud.
    • Clique em Procurar para procurar um local para o backup de seu dispositivo.
      (Exemplo Volume_1/backup)
    1. Clique em para ativar o recurso de atualização automática. Isso atualiza automaticamente o backup com base em um agendamento criado por você.
    • Se você ativar o recurso de atualização automática:

      • Selecione o agendamento de atualização automática: Diariamente, semanalmente, ou mensalmente

      • Selecione a hora da atualização automática a partir do menu suspenso (opção Diariamente).

      • Selecione a data e hora da atualização automática a partir dos menus suspensos (opção Semanalmente ou Mensalmente).

    1. Se você não selecionar a atualização automática, no campo Fazer backup agora, indique se deseja começar o backup agora.
    • Selecione Sim para iniciar o backup agora.
    1. Clique em Seguinte.
    2. Revise suas configurações, depois clique em Finalizar. Seu backup de nuvem do Amazon S3 foi criado. A nova tarefa exibe na seção da Fila do Amazon S3 da página de backup da Amazon S3.

    Observação: Acesse seu backup de nuvem do Amazon S3 bucket para ver seus dispositivos de backup.
