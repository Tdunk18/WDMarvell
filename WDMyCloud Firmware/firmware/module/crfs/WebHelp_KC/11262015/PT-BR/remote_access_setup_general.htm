Configuração do acesso à nuvem

Acessar sua nuvem remotamente por um smartphone, tablet ou computador requer um login do My Cloud™. (O acesso essencialmente remoto por um dispositivo é habilitado pelo servidor central do My Cloud que dá o acesso aos dispositivos NAS conectados à sua rede doméstica). No entanto, a maneira pela qual você acessa a nuvem remotamente a partir de dispositivos móveis (smartphones e tablets) e computadores é um pouco diferente. Independente disso, cada usuário deve ter um login do WD My Cloud.

Use as etapas a seguir para configurar o acesso remoto.

    1. Garanta que o acesso à nuvem esteja ativado para seu dispositivo. Como faço isso?
    2. Clique em Acesso à nuvem.
    3. No painel à esquerda, clique no nome do usuário ao qual você deseja permitir acesso à nuvem.
    4. No login WDMyCloud.com para a seção <nome do usuário>, clique em Registrar. Isto enviará um e-mail para o usuário com instruções sobre como inscrever-se para o WD My Cloud.
    1. Faça uma das seguintes opções:
    • Continue o processo de inscrição para você mesmo. Como faço isso?  
    • Notifique o usuário de que uma conta web foi configurada e faça-o continuar o processo de registro.
