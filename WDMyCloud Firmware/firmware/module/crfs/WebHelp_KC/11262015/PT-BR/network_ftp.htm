Ativando o acesso FTP

Você pode ativar ou desativar a transferência de arquivos entre dispositivos de computação na sua rede por meio do File Transfer Protocol (FTP).

Use as etapas a seguir a seguir para ativar/desativar o acesso de FTP:

    1. Clique em Configurações.
    2. No painel na esquerda, clique em Rede.
    3. Na área Serviço de Rede > Acesso FTP, clique em para habilitar ou desabilitar o acesso FTP.
    4. Leia a mensagem e clique em OK.
    5. Clique em Configurar.
    6. Na tela de Configurações FTP, digite a seguinte informação:
    • Usuários máximos: No menu suspenso, selecione o número máximo de usuários que deseja que tenha acesso ao FTP.
    • Tempo de espera: Digite o valor do tempo, em minutos, que deseja que o FTP entre em espera antes de desligar.
    • Porta: Digite a porta a ser usada para o acesso FTP.
    • Controle de fluxo: Selecione Ilimitado ou Personalizado. Se selecionar Personalizado, digite o valor do Controle de Fluxo.
    1. Clique em Seguinte.
    2. No campo de Modo Passivo, selecione Padrão ou Personalizado. Se selecionar Personalizado, digite os valores do Modo Passivo.
    3. Marque a caixa de seleção, se desejar relatar um IP externo no modo PASV (Passivo).
    • Digite o endereço de IP externo ou clique em Adquirir IP para fazer o sistema entrar no endereço
    1. Clique em Seguinte.
    2. Informe o seguinte:
    3. Idioma do cliente: Selecione o idioma do cliente no menu suspenso.
    4. TLS: Marque a caixa de seleção Transport Layer Security (TLS - Segurança da Camada de Transporte) para as conexões TLS Implícita ou TLS Explícita, se for aplicável.
    5. FXP: Clique em para habilitar ou desabilitar o File Exchange Protocol (FXP - Protocolo de Troca de Arquivo).
    1. Clique em Seguinte.
    2. Digite qualquer endereço de IP que gostaria de bloquear o acesso ao FTP, depois selecione Permanente ou Temporário no menu suspenso. Clique em Aplicar para salvar suas entradas.
    3. Clique em Finalizar.
