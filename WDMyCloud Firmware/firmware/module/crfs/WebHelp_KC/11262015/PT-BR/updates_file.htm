Atualizar o dispositivo My Cloud a partir de um arquivo

Em alguns casos, pode ser necessário atualizar o dispositivo My Cloud™ ao descarregar, primeiro, um arquivo do site de suporte da Western Digital e, depois, instalar.

Observação: Este procedimento exige acesso à Internet.

Use as seguintes etapas para atualizar o dispositivo My Cloud manualmente a partir de um arquivo:

    1. Abra uma nova guia ou janela do navegador.
    2. Na nova guia/janela, direcione seu navegador no site de suporte técnico da Western Digital:
      http://support.wdc.com
    • Se uma nova atualização de firmware estiver disponível, salve-a em um local no seu dispositivo.
    1. Retorne para a interface Web do My Cloud.
    2. Clique em Configurações.
    3. No painel esquerdo, clique em Atualização de firmware.
    4. Na área Atualização manual, clique em Atualizar a partir de arquivo.
    5. Navegue para o local em que você salvou a atualização na Etapa 3.
    6. Selecione o arquivo de atualização e clique em Abrir.
    7. Clique em OK para confirmar a instalação. Depois de instalar a atualização, o dispositivo My Cloud reinicia.
      Observação: Quando o My Cloud reinicia, o serviço para todos os usuários que estão acessando a nuvem neste momento é interrompido.
