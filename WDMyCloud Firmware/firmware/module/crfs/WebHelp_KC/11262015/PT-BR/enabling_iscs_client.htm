Habilitando um cliente iSCS

O protocolo Serviço do Nome de Armazenamento da Internet (iSNS) é usado para interação entre os servidores iSNS e os clientes iSNS. Clientes iSNS são computadores, também conhecidos como iniciadores, que estão tentando descobrir dispositivos de armazenamento, também conhecidos como alvos, na rede Ethernet.

Use as etapas a seguir para configurar o cliente iSNS do iSCSI.

Observação: o iSNS é usado principalmente para se conectar a um servidor do Windows.

    1. Clique em Armazenar.
    2. Clique em iSCSI.
    3. Clique em para habilitar o iSCSI.
    4. No campo Cliente iSNS do iSCSI, clique em para ATIVAR.
    5. Clique em Configurar.
    6. Digite o endereço do servidor do cliente iSNS (normalmente o endereço de IP do seu servidor Windows) e clique em Aplicar.
