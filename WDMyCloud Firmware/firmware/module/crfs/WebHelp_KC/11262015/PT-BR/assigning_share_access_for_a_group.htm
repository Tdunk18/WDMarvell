Atribuição de acesso de compartilhamento para um grupo

Depois de criar um grupo, você pode determinar o nível de acesso para os membros desse grupo.

Use as etapas a seguir para atribuir um compartilhamento de acesso para um grupo.

Observação: Primeiro, um compartilhamento deve ser privado na tela Compartilhamentos. Todos os compartilhamentos públicos estão em cinza. Consulte Sobre compartilhamentos para informação adicional.

    1. Clique em Usuários.
    2. Clique no botão Grupos.   
    3. No painel esquerdo, selecione o grupo ao qual você deseja atribuir acesso. Exibe informações sobre o grupo na área de Perfil do Grupo.
    4. Na área de acesso do compartilhamento, selecione o nível de acesso para cada divisão em seu dispositivo.
    • Acesso de leitura/gravação. Selecione esta opção para permitir acesso total à informação selecionada.
    • Acesso somente para leitura. Permite que um membro do grupo possa ler mas não alterar as informações selecionadas.
    • Negar acesso. O membro do grupo não tem acesso a esta informação.

Compartilhamentos agora estão atribuídos ao grupo selecionado.
