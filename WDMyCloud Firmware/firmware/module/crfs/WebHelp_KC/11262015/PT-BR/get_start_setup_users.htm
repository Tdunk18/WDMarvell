Configuração de outros usuários

Após configurar sua conta de proprietário e o acesso remoto, você estará pronto para adicionar outros usuários ao seu dispositivo.

A tarefa principal é adicionar novos usuários e configurá-los com acesso remoto. Consulte Adicionando uma conta de usuário.

Como administrador da nuvem, você também pode realizar os seguintes trabalhos:

    • Adicione uma senha a uma conta de usuário. Consulte Adicionando/alterando uma senha

    • Alterar o acesso de um usuário a um compartilhamento. Consulte Alteração de acesso a um compartilhamento

    • Excluir uma conta de usuário. Consulte Excluindo uma conta de usuário.

    • Alterar nome e senha de um usuário. Consulte Adicionando/alterando uma senha
