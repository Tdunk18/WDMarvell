Alterando o modo de RAID

Use as seguintes etapas para alterar o modo RAID atual do seu dispositivo My Cloud™.

Advertência: As alterações feitas a seu modo RAID irão excluir todos os seus dados e suas configurações de usuário. Consulte Exportar configurações do sistema para informações sobre como salvar suas configurações do usuário.

    1. Clique em Armazenar.
    2. Clique em RAID, caso ainda não esteja selecionado.
    3. Clique em Alterar o modo de RAID na parte inferior da tela.
    4. Revise a mensagem de aviso, em seguida clique em OK.
    5. Selecione o modo RAID que gostaria de usar para seu dispositivo My Cloud. As opções incluem:

Modo

Descrição

JBOD

O uso de uma ou mais unidades que não deve ser feito em uma configuração de RAID; ele deve ser gerenciado como volumes lógicos individuais.

Expansão

Combinação de unidades de forma linear para criar um grande volume lógico.

RAID 0

Os dados são distribuídos em vários discos rígidos, permitindo a leitura acelerada e gravação de dados pela combinação do trabalho de duas ou mais unidades para aumentar o desempenho. Porém, se uma unidade falhar, todos os dados serão perdidos.

RAID 1

Duas cópias instantâneas dos dados são registradas em discos rígidos separados, caso um deles falhe, o(s) outro(s) continua(m) a ler/gravar os dados até que o disco rígido com problema seja trocado.

RAID 5

Oferece desempenho e proteção superiores ao registrar dados por 3 ou mais unidades e dedicando um quarto de cada unidade para tolerância de erros. Se uma das unidades falhar, a(s) outra(s) continua(m) a ler/gravar os dados até que o disco rígido com problema seja trocado.

Esta opção está disponível apenas para dispositivos My Cloud com 4 baías.

RAID 10

O protocolo de RAID no qual os dados são gravados em tiras em discos primários que foram espelhados em discos secundários. Se uma das unidades falhar, a(s) outra(s) continua(m) a ler/gravar os dados até que o disco rígido com problema seja trocado.

Esta opção está disponível apenas para dispositivos My Cloud com 4 baías.

    1. Revise e selecione a opção de Armazenamento que gostaria de usar, marque a caixa de seleção Mudar para <Modo de Armazenamento> e clique em Seguinte. É realizado um auto-teste de funcionamento.
    2. Revise os Avisos na parte superior da tela.
    3. Quando o auto-teste estiver completo, revise o status dos seus discos rígidos e clique em Seguinte.
    4. Revise o status dos seus volumes e clique em Seguinte.
    5. Se o slider de volume aparecer, selecione o espaço que gostaria de dedicar ao modo RAID selecionado.
    6.  Se escolher não usar todo o volume para sua seleção RAID, é possível configurar o resto do espaço do disco como Transposição ao selecionar Configurar o espaço de disco remanescente como a caixa de seleção Transposição.

    1. Usando o slider de Volume, selecione a quantidade de espaço que você gostaria de atribuir ao modo de RAID selecionado.

      OBSERVAÇÃO: Se você optar por não usar todo o volume para sua seleção de RAID, você pode configurar o espaço de disco restante como Expansão ao marcar a caixa de seleção Configurar o espaço em disco restante como de expansão.

    1. Para RAID 5, é possível configurar RAID 5+spare, que permite que o "Disk4" se torne a peça extra principal. Marque a caixa de seleção Criar disco extra para configurar o RAID 5+Spare. Desmarque Criar disco extra, se não quiser criar um RAID 5+spare.
    2. Clique em Próximo para continuar.
    3. Se quiser recriar automaticamente o RAID depois que o disco for reconhecido, clique em , e clique em Seguinte.

      OBSERVAÇÃO: Esta tela não exibe os modos JBOD, Expansão e RAID 0
    1. Selecione a opção de criptografia para cada unidade. As opções incluem:
    • Bloqueado
    • Desbloqueado
    1. Analise as mudanças do RAID e verifique se estão corretas. Em seguida, clique em Seguinte.
    2. Analise a tela de aviso.
      Advertência: Quando você estiver em modos de alteração, todos os dados serão apagados.
    1. Clique em Finalizar. A mudança de disco rígido solicitada inicia. Não desligue o seu dispositivo My Cloud enquanto essas alterações estiverem em andamento.
    2. Quando o processo estiver completo, clique em Finalizar novamente.
