Ligar/desligar o acesso à nuvem

Se você for um administrador, é possível ligar o acesso à nuvem ou desligar para todos os usuários (ou seja, é possível controlar se os dispositivos de computação podem acessar o conteúdo do dispositivo My Cloud™).

Use as etapas a seguir para ativar/desativar o Acesso à nuvem.

    1. Clique em Configurações.
    2. No painel na esquerda, clique em Geral.
    3. No campo Acesso à nuvem > Serviço da nuvem, clique em para ligar ou desligar o serviço da nuvem, conforme apropriado.
    • On: Ativa o acesso remoto para todos os usuários.
    • Off: Desativa o acesso remoto para todos os usuários.
    1. Clique em Configurar para alterar o tipo de conexão utilizada para seu acesso à nuvem. Na tela de Opções de conexão de acesso à nuvem, há três opções de acesso.
    2. Automático: O automático usa UPnP (Universal Plug and Play - conectar e ligar universal) para tentar abrir as portas do seu roteador. Se tiver sucesso, é estabelecida uma conexão direta entre seu dispositivo e seus aplicativos.
    3. Manual: Estabelece uma conexão por meio da duas portas selecionadas. Se qualquer uma das portas não estiver disponível, será estabelecida uma conexão de retransmissão. Para esta opção, é necessária uma configuração manual do roteador. Consute a Resposta de base de conhecimento ID 8526 para informações sobre como configurar seu roteador para configuração manual.
    4. Win XP: Estabelece uma conexão usando as portas 80 e 443. Se essas portas não estiverem disponíveis, uma conexão de retransmissão será estabelecida. Esta opção é necessária, se estiver usando Windows XP.
      Observação: Por padrão, o dispositivo My Cloud estabelece automaticamente uma conexão direta entre seus dispositivos móveis e o roteador.
    1. No campo de Banco de Dados de Conteúdo, clique em Reconstruir, para recriar seu banco de dados do WDMyCloud.
      Observação: Realize esta opção somente para solucionar problemas, caso haja suspeita de banco de dados corrompido.
    2. Clique em Aplicar. O campo de Status de Conexão indica que seu dispositivo está conectado.

    Para obter detalhes sobre a desativação do acesso para um único dispositivo, consulte Removendo um dispositivo.

    Observação: As informações sobre a área das Opções de conexão estão disponíveis em Configurando opções de conexão, que explica as diferentes opções de firewall e inclui uma explicação de como melhorar o desempenho. Por padrão, o My Cloud estabelece automaticamente uma conexão direta entre seus dispositivos móveis e o roteador.

    Se o acesso móvel estiver ativado por meio de uma conexão com retransmissão, isto é, o Status da conexão for "Conectado (retransmitido)", você pode melhorar o desempenho ativando o encaminhamento de porta no seu conector local da WD (consulte Ativando o encaminhamento de porta).
