Determinando os endereços MAC e IP do dispositivo

Use as seguintes etapas para determinar os endereços de IP e MAC do seu dispositivo My Cloud™.

    1. Clique em Configurações.
    2. No painel na esquerda, clique em Rede. As seguintes informações são exibidas na seção de Perfil de Rede.

    Status

    O status atual da conectividade da internet.

    Endereço MAC

    O endereço MAC (Media Access Control - Controle de Acesso de Mídia) para este dispositivo.

    Endereço de IP IPv4

    O endereço de IP versão 4 para este dispositivo.

    Servidor DNS de IPv4

    O endereço do servidor de DNS de IP versão 4 para este dispositivo.

    Endereço de IP de DNS de IPv6.

    O endereço de IP versão 6 para este dispositivo.

    Servidor DNS de IPv6

    O endereço do servidor de DNS de IP versão 6 para este dispositivo.
