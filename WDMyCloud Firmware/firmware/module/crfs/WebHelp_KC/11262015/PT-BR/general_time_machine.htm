Fazendo backup dos seus dados com o Time Machine da Apple

É possível fazer o backup dos arquivos no dispositivo My Cloud™ para o Apple Time Machine em um computador Mac.

Importante: Defina uma senha para sua conta de Admin para restaurar seus backups no Time Machine.

Use as etapas a seguir para fazer backup dos seus dados para o Time Machine da Apple.

    1. Clique em Configurações.
    2. No painel na esquerda, clique em Geral.
    3. Na área de Backups do Mac, clique em para ativar a opção de backup do Time Machine.
    4. Clique em Configurar.
    5. Na lista suspensa Selecionar um compartilhamento, selecione o compartilhamento que armazenará os seus arquivos de backup.
    6. No campo Tamanho Máximo, mova o slider para indicar o valor máximo que deseja usar para seu backup.
      Observação: Aumentar o valor máximo de backup após o backup inicial não tem efeito nos backups existentes. A WD recomenda que essa configuração seja mantida no valor máximo para o seu backup inicial. Quando o backup inicial for concluído, você poderá alterar o tamanho dele ao arrastar o controle deslizante de Tamanho máximo.
    7. Clique em Salvar.

    Importante: Quando o Time Machine começar a fazer o backup dos seus arquivos para um compartilhamento selecionado, recomendamos continuar a fazer nesse mesmo. Mudar o compartilhamento gera um novo arquivo de backup que não irá conter as informações que foram salvas anteriormente.
