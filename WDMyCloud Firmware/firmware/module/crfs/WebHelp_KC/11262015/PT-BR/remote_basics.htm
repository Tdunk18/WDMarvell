Fundamentos do acesso remoto

Há três maneiras de acessar o conteúdo armazenado no My Cloud de qualquer lugar::

    • De um dispositivo móvel, como um smartphone ou tablet  
    • Do seu computador (mais provável que seja o seu laptop).  

Visão geral do acesso ao dispositivo móvel

Use as etapas a seguir para acessar sua nuvem a partir de smartphone ou tablet:

1

Obtenha um login do WD My Cloud. Isso é normalmente feito no momento que você adicionar um usuário ao seu dispositivo. Consulte Configuração do acesso à nuvem.

2

Faça download e instale os aplicativos gratuitos WD Photos e WD My Cloud no seu dispositivo móvel.
Esses aplicativos estão disponíveis na Apple App Store, Google Play store , Amazon Appstore for Android.

3

Execute uma configuração única para acesso seguro. Consulte Executando uma configuração única para acesso seguro

4

Acesse seu conteúdo a partir de um dispositivo móvel. Consulte Acesso ao conteúdo de um smartphone ou tablet

Visão geral de acesso a um computador Windows ou Mac

Quando você estiver fora de casa (como em uma cafeteria), é possível acessar facilmente seu dispositivo a partir do seu laptop. O aplicativo para computador do WD My Cloud, o laptop ativado com Wi-Fi e uma conexão Wi-fi são necessários para acesso remoto.

Use as etapas a seguir para acessar sua nuvem com o aplicativo para computador do WD My Cloud:

1

Faça um login no My Cloud . Isso é normalmente feito no momento que você adicionar um usuário ao seu dispositivo. Consulte Configuração do acesso à nuvem.

2

Faça o download do aplicativo para desktop WD My Cloud do Centro de Aprendizagem WD (http:/www./wd.com/setup) e instale no seu computador.

3

Execute uma configuração única para acesso seguro. Consulte Executando uma configuração única para acesso seguro

4

Acesse seu conteúdo a partir de um laptop. Consute Acessar conteúdo a partir do laptop.
