Criando um backup remoto

O backup remoto permite fazer cópias de segurança do seu dispositivo My Cloud™ para outro My Cloud.  Use as seguintes etapas para criar um backup remoto do seu dispositivo My Cloud.

    1. Clique em Backups.
    2. Clique em Backup Remoto.
    3. Clique em Criar trabalho.
    4. Digite a seguinte informação para criar um trabalho de backup remoto:

Nome do trabalho

Insira um nome de trabalho para seu backup.

Servidor remoto

Selecione o tipo de servidor remoto.

  • Servidor NAS
  • My Cloud       

Endereço de IP remoto

Digiteo endereço de IP do local remoto (exemplo: 192.168.1.16)

Senha

Digite a senha para o servidor de backup remoto.

Nome de usuário SSH

Digite o nome de usuário SSH (Secured Shell Protocol) para o dispositivo remoto.

Senha SSH

Digite a senha SSH para dispositivo remoto.

Pasta de origem

Clique em Buscar e selecione a pasta que deseja fazer o backup.

Pasta de destino

Clique em Buscar e selecione a pasta de destino para seu backup.

Tipo de backup

Indica o tipo de backup que deseja realizar.

  • Copiar
  • Sincronizar  

Repetição

Se quiser usar o recurso Recorrência:

  1. Clique no botão alternar para habilitar o recurso de Backup Recorrente.
  2. Selecione a frequência do backup: Diária, semanal, mensal.
  3. Selecione o horário (hora, AM/PM) no menu suspenso.

    1. Clique em Criar.

    2. Na área de Trabalhos de backup em USB, clique no ícone Iniciar Backup para começar. O progresso do backup aparece na área Trabalhos de Backup de USB.
