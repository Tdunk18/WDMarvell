Adicionando/alterando uma senha

Use as seguintes etapas para adicionar ou alterar a senha para uma conta de usuário.

    1. Clique em Usuários.
    2. No painel esquerdo, selecione um nome de usuário.
    3. No campo Senha , clique em para ativar a opção de Senha, se não estiver ativada.
    4. Insira uma senha no campo Senha.
    5. Insira a mesma senha no campo Confirmar senha.
    6. Clique em Aplicar.
