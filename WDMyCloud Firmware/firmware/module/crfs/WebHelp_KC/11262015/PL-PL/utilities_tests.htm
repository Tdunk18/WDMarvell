Wykonywanie testów diagnostycznych

Za pomocą jednego z poniższych testów można określić sprawność urządzenia:

    1. Szybki test: szybki test służy do sprawdzania obecności poważnych problemów wpływających na wydajność dysków twardych. Wynikiem szybkiego testu jest pozytywna lub negatywna ocena stanu dysku twardego. Szybki test może trwać kilka minut.

    • Pełny test: pełny test jest najbardziej wszechstronnym narzędziem diagnostycznym dysku. W tym teście sprawdzany jest każdy sektor dysków twardych. Po zakończeniu testu wyświetlana jest informacja o stanie dysku twardego. Pełny test może trwać kilka godzin, w zależności od wielkości i konfiguracji danych na dyskach twardych.

    • Test systemu: test systemu stosowany jest do sprawdzania stanu sprzętu, tj. dysków twardych, wentylatora, zegara systemowego i temperatury urządzenia.

Aby wykonać test diagnostyczny, zrób tak:

    1. Kliknij przycisk Ustawienia.
    2. W lewym okienku kliknij pozycję Narzędzia.
    3. W obszarze Diagnostyka systemu > Test dysku kliknij przycisk Szybki test lub Pełny test bądź w obszarze Diagnostyka systemu > Test systemu kliknij przycisk Test systemu.
      Wskaźnik oznacza stan testu, a wyniki są wyświetlane w oknie dialogowym.
    4. Po zakończeniu testu przejrzyj wyniki, a następnie kliknij przycisk Zamknij.
