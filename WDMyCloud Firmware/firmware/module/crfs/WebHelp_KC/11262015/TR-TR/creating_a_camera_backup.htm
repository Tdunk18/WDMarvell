Kamera Yedekleme Oluşturma

Kamera yedeklemeleri, kameranızdaki verileri My Cloud™ aygıtına yedeklemenizi sağlar. Kullanıcılar, eğer bu paylaşıma erişimleri varsa, indirmenin ardından kamera verilerinin kaydedildiği konuma gidebilirler.

Kameranızı My Cloud aygıtına yedeklemek için aşağıdaki adımları takip edin.

    1. Kameranızın My Cloud aygıtınıza bağlı olduğundan emin olun.
    2. Backups (Yedeklemeler) ekranını görmek için Navigation (Gezinti) çubuğunda Backups (Yedeklemeler) butonuna tıklayın.
    3. Henüz seçilmemişse Kamera Yedeklemeleri butonuna tıklayın.
    4. Bağlı Kameralar bölümünde kameranızın tanımlandığına emin olun.
    5. Kameranızdaki verileri yedeklemek için Settings (Ayarlar) alanına aşağıdaki bilgileri girin.

      Otomatik Aktarım

      Kameranızdan My Cloud aygıtınıza verileri otomatik olarak aktarmak için değiştirme butonuna tıklayın.

      Aktarım Modu

      Kamera verilerinizi nasıl aktarmak istediğinizi seçin. Seçenekler:

      • Kopyala: Verileri kameranızdan My Cloud aygıtına kopyalar. Böylelikle orijinal veriler kameranızda kalır.

      • Taşı: Kameranızdaki dosyaları kaldırarak My Cloud aygıtına taşır.

      Klasör Seçenekleri

      Aktarılan kamera dosyaları için bir konum belirlemek istiyorsanız bu seçeneği seçin.

      1. Camera Folder Options (Kamera Klasörü Seçenekleri) ekranında:

        • Aktarım Klasörü: Aygıtınızda verilerinizi aktarmak istediğiniz konumu girmek için Browse (Gözat) butonuna, sonra OK (Tamam)'a tıklayın.

        • Klasör Adı: Aşağı açılan menüden bir Folder Name (Klasör Adı) seçin.

          • Eğer Custom Folder Name (Özel Klasör Adı) seçeneğini seçerseniz, Enter Folder Name (Klasör Adı Gir) alanına bir klasör adı girin.

      1. Kaydet butonuna tıklayın.

    1. Eğer otomatik aktarım aktif değilse, klasör aktarımını başlatmak için Dosyaları Aktar alanında Hemen Kopyala/Taşı butonuna tıklayın.
    2. Tamamlandıktan sonra Status (Durum) kısmı bağlanan aygıt için yedeklenmenin tamamlandığını ve tamamlanan indirmenin gün, tarih ve zamanını gösterir.
