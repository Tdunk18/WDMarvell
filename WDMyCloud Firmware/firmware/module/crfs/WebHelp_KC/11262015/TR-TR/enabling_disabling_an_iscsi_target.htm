iSCSI Hedefini Etkinleştirme/Devre Dışı Bırakma

Bir iSCSI hedefini etkinleştirmek veya devre dışı bırakmak için aşağıdaki adımları uygulayın.

iSCSI Hedefini Etkinleştirme

    1. Depolama butonuna tıklayın.
    2. iSCSI butonuna tıklayın.
    3. iSCSI Hedefi listesi altında etkinleştirmek istediğiniz hedefin yanındaki Ayrıntılar butonuna tıklayın.
    4. Etkinleştir butonuna tıklayın. Seçilen hedef etkinleştirilir.

 

iSCSI Hedefini Devre Dışı Bırakma

    1. Depolama butonuna tıklayın.
    2. iSCSI butonuna tıklayın.
    3. iSCSI Hedefi listesi altında devre dışı bırakmak istediğiniz hedefin yanındaki Ayrıntılar butonuna tıklayın.
    4. Devre Dışı Bırak butonuna tıklayın. Seçilen hedef devre dışı bırakılır.
