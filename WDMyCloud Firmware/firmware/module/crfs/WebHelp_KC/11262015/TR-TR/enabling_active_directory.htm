Active Directory'yi Etkinleştirme

Bu işlevin etkinleştirilmesi My Cloud™ aygıtınızın mevcut bir Windows Etki Alanına katılmasına imkan tanır. Active Directory varsayılan olarak devre dışıdır.

Active Directory'yi etkinleştirmek için aşağıdaki adımları uygulayın.

    1. Ayarlar butonuna tıklayın.
    2. Soldaki bölmeden Ağ butonuna tıklayın.
    3. Active Directory'yi etkinleştirmek için Windows Hizmetleri>Active Directory alanında butonuna tıklayın.
    4. Active Directory Ayarları ekranında aşağıdaki bilgileri girin:

    2. Kullanıcı Adı

    Active Directory sunucu hesap adını girin.

    Parola

    Active Directory sunucu parolasını girin.

    Etki Alanı Adı

    Active Directory sunucu etki alanı adını girin.

    DNS Sunucusu

    Alan Adı Sistemi (DNS) Sunucusu IP adresini girin.
    1. Bilgi ekranını gözden geçirin ve ardından Kaydet butonuna tıklayın.
