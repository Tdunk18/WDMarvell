HTTP İndirmesi Planlama

HTTP İndirme uygulaması My Cloud™ aygıtınıza bir http indirmesi planlamanızı sağlar. Bir HTTP indirmesi planlamak için aşağıdaki adımları kullanın.

    1. Uygulamalar butonuna tıklayın.
    2. HTTP İndirmeleri butonuna tıklayın.
    3. İndirmek istediğiniz dosyanın URL'sini URL alanına girin.
      Örnek: http://192.168.0.32/test.txt
    4. Şuraya Kaydet alanına indirmeyi kaydetmek istediğiniz konumu girin veya Gözat butonuna tıklayarak konumu seçin.
      Örnek: Volume_1/Ortak/Paylaşılan Videolar 
    5. Dosyanın adını değiştirmek istiyorsanız Yeniden Adlandır alanına dosya için yeni bir ad girin.
    6. Bu yinelenen bir yedeklemeyse  butonuna tıklayın.
    • Yedeklemenin sıklığını seçin (Günlük, Haftalık, Aylık)
    • Seçilen sıklığa bağlı olarak yedeklemenin Saat, Gün ve Tarihini girin.
    1. İndirmeyi gerçekleştirmek istediğini tarihi (aa/gg/yyyy) girin veya  butonuna tıklayarak bir tarih seçin.
    2. Aşağı açılan menülerden indirme işleminiz için bir saat girin.
    3. Oluştur butonuna tıklayın. İndirme işlemi oluşturulur ve HTTP İndirme Sırasına eklenir.
