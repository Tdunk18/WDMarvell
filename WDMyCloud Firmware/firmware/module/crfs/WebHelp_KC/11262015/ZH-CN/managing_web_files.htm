管理 Web 文件

Web 文件查看器是用来查看和管理 My Cloud™ 设备上的文件的应用。

使用以下步骤管理 Web 文件。

    1. 单击应用。
    2. 单击 Web 文件查看器。设备上的文件卷显示在右窗格。

查看 Web 文件

卷、文件夹和文件显示在 Web 文件查看器的右窗格。若要查看任何卷或文件夹的内容，双击该文件夹。

管理 Web 文件

可对文件和文件夹执行的操作显示在 Web 文件查看器的右上角。选择以下某个要对 Web 文件执行的操作：

复制

此操作用来将选定文件复制到另一个位置。使用以下步骤将文件复制到新位置：

    1. 在“Web 文件查看器”区域，选择要复制的文件。
    2. 单击 。
    3. 选择文件复制的目标。
    4. 单击确定。文件随即复制到所选目标。

移动

此操作用来将文件从一个位置移到另一个位置。使用以下步骤将文件移动到新位置：

    1. 选择要移动的文件。
    2. 单击 。
    3. 选择要将文件移动到哪个目标。
    4. 单击确定。文件随即移至所选目标。

上传

此操作允许您从计算机或其他设备上载文件至 My Cloud 服务器。使用以下步骤上载文件至设备：

    1. 在 Web 文件查看器的右窗格，选择上传文件的目标。
    2. 单击 。
    3. 在“上传”屏幕上，单击“选择文件”图标
    4. 转至要上传到设备的文件，并选中该文件。
    5. 单击打开。
    6. 单击上传。
    • 如果要上传更多文件，请执行步骤 3 到 4，直至所有文件都已上传。
    1. 单击关闭。上传的文件显示在目标文件夹中。

下载

此操作用来将文件从My Cloud 设备下载到计算机或其他设备上。
使用以下步骤将文件下载到计算机或其他设备：

    1. 在 Web 文件查看器的右窗格，选择要下载的文件。
    2. 单击 。
    3. 所选文件随即下载到计算机或其他设备。

重命名

此操作用来重命名 My Cloud 设备上的选定文件。使用以下步骤重命名文件。

    1. 在 Web 文件查看器的右窗格，选择要重命名的文件。
    2. 单击 。
    3. 在“重命名”字段中重命名文件。
    4. 单击保存。文件随即重命名。

删除

此操作用来从 My Cloud 设备上删除所选文件。使用以下步骤删除文件。

    1. 在 Web 文件查看器的右窗格，选择要删除的文件。
    2. 单击 。
    3. 单击确定确认您要删除所选文件。所选文件随即从设备上被删除。

刷新

此操作刷新文件列表，显示自上次刷新或您访问文件列表以来添加的任何新文件。

  • 单击 。您的文件随即刷新。

    1. 导航至您要创建新文件夹的位置。
    2. 单击
    3. 输入新文件夹名称。
