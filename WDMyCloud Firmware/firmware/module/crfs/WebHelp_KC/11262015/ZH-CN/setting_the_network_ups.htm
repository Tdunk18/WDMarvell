设置网络 UPS

不间断电源 (UPS) 是一种设备，在主电源无法供电的情况下，至少可以在短时间内代替电池并维持连接 UPS 端口的设备正常运行。如果 My Cloud™ 设备通过 UPS 连接网络驱动器，驱动器在 UPS 电量消耗到指定百分比时会自动关闭。

与 My Cloud 设备的通信使用主/从协议，主设备控制从设备。

如果 UPS 通过 USB 连接 My Cloud 设备，设备自动成为 UPS 网络主设备。其他连接 UPS 的 My Cloud 设备可以指定为 UPS 主设备的 IP 地址。如果 UPS 电量消耗到指定百分比，所有 My Cloud 从设备将适当地自动关闭。请参见知识库第 11852 号答案查看 UPS 设备列表。

使用以下步骤设置 My Cloud™ 设备的网络 UPS。

    1. 单击设置。
    2. 在左窗格中，单击“网络”。
    3. 在“网络 UPS>从”区域，单击 启用“动态 DNS”。
    4. 输入联网 UPS 主控的 IP 地址。
    5. 单击下一步。显示“正在进行”屏幕，当设置更新完成后，显示“网络”屏幕，“网络 UPS 从”设置为“关”。
