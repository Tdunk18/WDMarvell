执行一次性设置以获得安全访问权限

本过程介绍如何在智能手机或平板电脑与 My Cloud™ 设备之间建立安全连接。请参考以下有关特定移动设备的说明：

    • Android 智能手机或平板电脑

    • iPhone 或 iPad

    • Windows Phone 或平板电脑

    • Windows 或 Mac 桌面计算机

获得 Android 智能手机或平板电脑的安全访问权限

    1. 执行以下步骤连接智能手机或平板电脑：

    • 在移动设备上点击“设置”。

    • 点击“无线和网络”。

    • 点击“Wi-Fi 设置”。

    • 选择您的家庭网络。

    • 输入网络密码，然后点击连接。

    1. 启动 WD 应用。
    2. 点击“立即连接”。
    3. 点击 My Cloud 设备的名称。
    4. 点击下拉箭头，然后选择您的用户名。
    5. 执行以下某一操作：

    • 如果为该用户设置了密码，请输入密码，然后点击登录。

    • 如果没有为该用户设置密码，请点击登录。

    1. （可选）如果您想要访问您的任何内容，请点击某个共享名称，然后导航到所需的文件。

获得 iPhone 或 iPad 的安全访问权限

    1. 执行以下步骤连接智能手机或平板电脑：

    • 在移动设备上点击“设置”。

    • 点击“Wi-Fi”。

    1. 点击“Wi-Fi 设置”。

    2. 选择您的家庭网络。

    3. 输入您的网络密码，然后点击加入。

    1. 启动 WD 应用。
    2. 点击“立即连接”。
    3. 点击 My Cloud 设备名称旁边的加号 (+)。
    4. 点击所需的用户名。
      如果为该用户设置了密码，请输入密码。
    5. （可选）如果您想要访问您的任何内容，请点击某个共享名称，然后导航到所需的文件。

获得 Windows 智能手机或平板电脑的安全访问权限

    • 执行以下步骤连接智能手机或平板电脑：

    • 在您的计算设备上点击设置。

    • 点击“Wi-Fi”。

    • 选择您的家庭网络。

    • 输入您的网络密码，然后点击完成。

    1. 启动 WD 应用。
    2. 点击“立即连接”。
    3. 点击 My Cloud 驱动器的名称。
    4. 点击下拉箭头，然后选择一个用户名。
      如果为该用户设置了密码，请输入密码。
    5. （可选）如果您想要访问您的任何内容，请执行以下操作：

    • 在“设备”屏幕上，点击 My Cloud 驱动器的名称。

    • 点击共享，然后浏览到所需的内容。

获得 Windows 或 Mac 台式机或便携式计算机的安全访问权限

    1. 在安装 WD My Cloud 桌面应用后，双击现在出现在桌面上的快捷方式 ，以显示网络上的所有 WD 设备。 有关安装该应用的信息，请参阅远程访问基础知识。
    2. 系统可能显示有关安全警报的信息。为了避免再次显示此屏幕，请选中“我理解”复选框。
    3. 单击“我理解”显示网络上的设备。
    4. 单击您要连接的设备，本例中为 My Cloud 设备。“登录”屏幕显示。
    5. 如果以前设置过 My Cloud 设备的密码，则输入密码，然后单击“登录”。“开始”屏幕显示。
    6. 若要获得更多信息，请单击“WD 学习中心”链接。
    7. 单击“开始”显示 My Cloud 设备的共享以及任何 SmartWare 或 Time Machine 备份文件。
    8. 若要了解如何远程访问和管理 My Cloud 设备上存储的内容，请单击在线“学习中心”My Cloud 下载面板上的“使用计算机访问”链接：
