创建 ISO 映像

ISO 映像是现有文件系统的精确副本文件。使用以下步骤创建 ISO 映像。

使用以下步骤创建 ISO 映像

    1. 单击“设置”。
    2. 在左窗格中，单击实用工具。
    3. 在“ ISO 共享列表”区域，单击创建 ISO 映像。
    4. 在“映像大小”字段，从下拉菜单选择您要用于 ISO 映像的映像大小。
    5. 在“映像路径”字段，输入您要映像的数据路径或单击浏览选择您要映像的文件夹，然后单击确定。
    6. 在“映像名称”字段中输入 ISO 映像的名称。
    7. 单击“下一步”。
    8. 在“选择”字段单击覆盖或跳过。
    • 覆盖： 如果选择覆盖，同名文件将被覆盖。
    • 跳过： 如果选择跳过，同名文件将被跳过。
    1. 在左栏中选择您要添加到映像文件中的文件夹和文件，然后单击添加>>。
    2. 选择您要从 ISO 映像中移除的文件夹和文件，然后单击<<移除。
    3. 单击“下一步”。ISO 映像随即创建。
    1. 单击“完成”。
