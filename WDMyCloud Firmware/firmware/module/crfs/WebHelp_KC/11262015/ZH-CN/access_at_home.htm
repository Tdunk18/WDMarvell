在家中访问内容

如果您在家中，并且您的计算设备已连接到家庭网络，则您可以通过与远程访问不同的方式访问驱动器中存储的内容。如果您已连接到家庭网络，您可以：

    • 从智能手机或平板电脑访问内容
    • 从计算机访问内容

重要说明：在开始之前，请确保您已完成设置云访问中所述的步骤。

从连接到家庭网络的移动设备访问内容

    1. 如果您尚未下载 WD 移动应用，现在请进行下载。
    2. 在智能手机或平板电脑上连接到家庭网络。
      如果您不知道如何连接，请参考智能设备的文档。
    3. 启动 WD 移动应用（WD My Cloud 或 WD Photos）。
    4. 如果出现提示，请点击“立即连接”。
    5. 选择 My Cloud 驱动器。
    6. 选择要访问的共享，然后浏览到所需的内容。

从连接到家庭网络的计算机访问内容

    1. 确保您的计算机已连接到家庭网络，并且您可以访问 Internet。
    2. 如果您尚未从 WD 学习中心 (http:.wd.com/setup) 下载 WD My Cloud 桌面应用程序，并将其安装在 Mac 或 PC 上，现在请进行下载和安装。
    3. 在计算机上启动 WD My Cloud 桌面应用程序。
    4. 输入您的登录凭据。
    5. 双击 My Cloud 驱动器。
    6. 在左窗格中，根据需要展开文件夹以导航到所需的内容。
