设置节电器模式

使用以下步骤降低 My Cloud™ 设备的能耗。

    1. 单击设置。
    2. 单击常规。
    3. 在“节电器”区域，选择以下选项设置设备的能耗。

模式

描述

驱动器睡眠

启用此功能允许硬盘在处于非活动状态一定时间后进入待机模式以节省电量。驱动器睡眠默认为大约 10 分钟后启用睡眠模式。

  • 单击 启用/禁用驱动器睡眠。

LCD

启用此功能允许设备 LCD 在一定时间后进入待机模式以节省电量。LCD 默认为 10 分钟后进入待机模式。

  • 单击 启用/禁用 LCD。

此选项仅适用于 My Cloud EX2。My Cloud EX4、My Cloud EX4100 和 My Cloud DL4100 型号。

电源恢复

在因为电源故障而发生意外关机后自动重新启动设备。电源恢复自动启用。

  • 单击 启用/禁用电源恢复。

电源计划

电力安排允许您安排关闭您的 My Cloud 设备。

  • 单击 启用/禁用电源计划
  • 单击“配置”自定义您的电力安排。
  • 针对一星期中的每一天，选择您希望保持设备处于活动状态的时间。
  • 单击保存。

 

Web 访问超时

设置设备自动将您从设备上注销的默认时间。

  • 从 Web 访问超时下拉菜单中选择您想用于系统超时的时间。系统自动更新。
