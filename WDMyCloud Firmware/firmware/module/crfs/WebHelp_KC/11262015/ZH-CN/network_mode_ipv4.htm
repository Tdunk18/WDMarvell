更改 IPv4 网络模式

默认情况下，网络模式为 DHCP，这意味着 My Cloud™ 设备将自动从您的网络获取 IP 地址和其他设置。如果您要手动设置 IP 地址，请通过完成此过程来更改模式。

使用以下步骤更改 IPv4 网络模式。

    1. 单击设置。
    2. 在左窗格中，单击“网络”。
    3. 在“网络服务>IPv4 网络模式”区域，选择设备唯一 IPv4 地址的分配方式：
      • 静态：静态 IP 地址允许您每次连接使用相同的 IP 地址。系统将提示您输入 IP 地址、子网掩码、网关 IP 地址和 DNS 服务器。（如果您不知道这些信息，请查看路由器设置。）
      • DHCP：DHCP 客户端使 My Cloud 设备从本地 DHCP 服务器自动获取 IP 地址和相关设置。
        注：当配置“静态”或“DHCP”时，在安全位置记录您的子网掩码、网关 IP地址和 DNS 服务器，供以后参考使用。这些信息在输入到 My Cloud 服务器之后，便不会再显示。

    1. 完成“LAN 设置向导”设置网络模式。
