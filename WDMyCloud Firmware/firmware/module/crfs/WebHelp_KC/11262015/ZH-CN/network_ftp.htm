启用 FTP 访问

您可以允许或禁止通过文件传输协议 (FTP) 在网络中的计算设备之间传输文件。

使用以下步骤启用/禁用 FTP 访问：

    1. 单击“设置”。
    2. 在左窗格中，单击“网络”。
    3. 在“网络服务>FTP访问”区域，单击 启用或禁用 FTP 访问。
    4. 阅读消息，然后单击确定。
    5. 单击配置。
    6. 在 FTP 设置屏幕，输入以下信息：
    • 最大用户数：从下拉菜单选择您希望其拥有 FTP 访问权的最大用户数。
    • 空闲时间：输入您希望 FTP 空闲多长时间（以分钟为单位）后确定为超时。
    • 端口：输入用于 FTP 访问的端口。
    • 流控制：选择无限或自定义。如果您选择“自定义”，请输入“流控制”值。
    1. 单击“下一步”。
    2. 在“被动模式”字段选择默认或自定义。如果您选择“自定义”，请输入“被动模式”值。
    3. 如果您想以 PASV（被动）模式报告外部 IP，请选择复选框。
    • 输入外部 IP 地址，或单击获取 IP 让系统输入地址
    1. 单击“下一步”。
    2. 输入以下信息：
    3. 客户端语言： 从下拉菜单中选择客户端语言。
    4. TLS：根据需要选择“隐式 TLS”或“显式 TLS”旁边的“传输层安全”(TLS) 复选框。
    5. FXP：单击  启用或禁用“文件交换协议”(FXP)。
    1. 单击“下一步”。
    2. 输入您想阻止 FTP 访问的任何 IP地址， 然后选择从下拉菜单选择永久或临时。单击应用保存您的输入。
    3. 单击“完成”。
