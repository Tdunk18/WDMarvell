更改 IPv6 网络模式

IPv6 格式是新的 IP 标准，它规定了跨多个 IP 网络的数据包格式和编址方案。默认情况下，该选项被设置为“关闭”。

使用以下步骤更改 IPv6 网络模式。

    1. 单击设置。
    2. 在左窗格中，单击“网络”。
    3. 在“网络服务”>“IPv6 网络模式”区域，选择设备 IPv6 地址的分配方式：
      • 自动
      • DHCP
      • 静态
      • 关

    1. 选择完成后，请填写相关屏幕的下列字段，然后单击应用:

      • IP 地址
      • 前缀长度
      • 默认网关
      • DNS 服务器

      • 注：当配置“静态”或“DHCP”时，在安全位置记录您的子网掩码、网关 IP地址和 DNS 服务器，供以后参考使用。这些信息在输入到 My Cloud 服务器之后，便不会再显示。
