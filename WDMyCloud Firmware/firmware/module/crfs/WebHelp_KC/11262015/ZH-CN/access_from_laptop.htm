从便携式计算机访问内容

WD My Cloud 桌面应用程序提供用于从便携式计算机访问 My Cloud 设备中存储的内容的方法。此方法需要 Internet 连接，并需要登录到 wdmycloud.com（请参见设置远程访问）。

使用以下步骤从便携式计算机访问内容：

    1. 从 WD 学习中心 (http:/www./wd.com/setup/) 下载 WD My Cloud 桌面应用程序，并将其安装在 Mac 或 PC 上。
    2. 连接到您的网络（有线或无线连接）。
    3. 在计算机上启动 WD My Cloud 桌面应用程序。
    4. 输入您的登录凭据。
    5. 双击 My Cloud 设备。
    6. 在左窗格中，根据需要展开文件夹以显示您要共享的内容。
    7. 在右窗格中，双击某项查看或播放。
      重要说明：能否播放和查看文件取决于计算机和浏览器的配置，以及是否安装了必要的媒体查看和播放程序。
