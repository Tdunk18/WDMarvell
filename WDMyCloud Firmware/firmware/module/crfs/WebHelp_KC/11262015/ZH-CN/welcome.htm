欢迎使用

您最终拥有属于自己的云。把所有东西保存到一个地方，使用计算机、平板和智能手机从任何地方访问。My Cloud™ 存储与目前多个操作系统兼容，如 Mac OS、Windows OS、iOS 和 Android，可与移动应用配合使用。

个人云中的用户角色

管理员（通常是设备的拥有者）负责设置设备。作为 My Cloud 设备的所有者，您有一个特殊的用户账户 (admin)，
该账户具有管理员权限。使用这些权限，您可以根据具体需求设置和配置设备，以及将其他用户添加到您的个人云。您也可以确切地确定用户可以访问设备上的哪些内容。

默认情况下，管理员 (admin) 用户名没有密码。您可以遵照添加/改变用户密码所述步骤在任何时间创建密码。

从智能手机或平板电脑访问内容

有两个 WD 移动应用允许您从智能手机或平板电脑访问 My Cloud 设备上的内容：

    • WD Photos：查看和管理您的照片。

    • WD My Cloud：查看和播放照片、视频、音乐和文件。

可以从 WD 学习中心 (http://www.wd.com/setup) 或其他应用网站（如 Apple App Store、Google Play Store 或适用于 Android 的 Amazon Appstore）获得这些免费应用。

显示 My Cloud 界面

您可以通过 My Cloud 设备用户界面设置和管理个人云。该界面显示在 Mac 或 PC 上运行的 Web 浏览器中。 

要显示 Web 界面，请执行以下操作：

    1. 打开一个浏览器窗口。
    2. 在地址栏中，输入 http://<设备名> ;
      例如：http://wdmycloud4100
      注意：如果输入该 URL 后未能显示 Web 界面，请尝试输入以下 URL：My Cloud DL2100：http://<设备名>.local
