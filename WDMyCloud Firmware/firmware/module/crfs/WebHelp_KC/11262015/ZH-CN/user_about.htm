关于用户

访问个人云的任何人必须有用户账户才能使用 My Cloud 设备。作为设备所有者，您有一个特殊的用户账户 (admin)，
此账户赋予您管理员权限。使用这些权限，您可以根据具体需求设置和配置设备，以及将其他人添加到您的个人云。您还有权确定究竟哪些人可使用您的设备。

默认情况下，admin 用户名没有密码。您可以随时添加密码。

重要说明：初次设置设备时，您可在运行 My Cloud 设备设置软件或“入门”向导时，更改管理员名称，并设置密码（密码可设可不设）。请确保记住此名称和密码。如果没有这些凭据，您将无法显示 My Cloud 网页界面。

作为 My Cloud 设备的管理员，您可以：

    • 添加用户

    • 添加或编辑用户密码

    • 查看用户信息

    • 更改用户的共享访问权限  

    • 删除用户。

    • 更改某人的用户名和密码
