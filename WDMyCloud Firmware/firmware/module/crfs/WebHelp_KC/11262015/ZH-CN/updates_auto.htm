自动更新 My Cloud 设备

“自动更新”允许您将 My Cloud™ 设备设置为自动检查并安装可用的软件更新，从而为您带来方便。这是确保 My Cloud 固件保持最新状态的最简易方法。

注： 安装固件更新后，My Cloud 设备可能重新启动。 由于重新启动会影响到访问云的用户，因此，请将更新安排在用户不太可能会访问云的时间进行。

注： 如果您启用“自动更新”，更新完成后您需要手动重启系统。

使用以下步骤自动更新 My Cloud 设备。

    1. 单击“设置”。
    2. 在左窗格中，单击固件更新。
    3. 在“自动更新”区域，单击 启用自动更新。
    4. 在“更新计划”字段中，安排更新的日期和时间：

    • 单击第一个下拉箭头并选择星期几。

    • 单击第二个下拉箭头并选择时间。

    • 单击第三个下拉箭头并选择 AM 或 PM。

    1. 点击“保存”。
