计划 HTTP 下载

HTTP 下载应用可用来安排 My Cloud™ 设备的 http 下载。使用以下步骤安排 HTTP 下载。

    1. 单击应用。
    2. 单击 HTTP 下载。
    3. 在“URL”字段中，输入您要下载的文件的 URL。
      示例：http://192.168.0.32/test.txt
    4. 在“保存到”字段中，输入您要保存下载文件的位置，或者单击浏览选择要保存下载文件的位置。
      示例：Volume_1/Public/Shared Videos 
    5. 在“重命名”字段中，如果要更改文件名，请输入文件的新名称。
    6. 如果这是定期备份，单击 。
    • 选择备份频率（每天、每周、每月）
    • 根据所选频率，输入备份时间、备份日和备份日期。
    1. 输入要执行下载的日期 (mm/dd/yyyy) 或单击 选择日期。
    2. 从下拉菜单中选择下载时间。
    3. 单击创建。您的下载随即创建并添加到 HTTP 下载队列。
