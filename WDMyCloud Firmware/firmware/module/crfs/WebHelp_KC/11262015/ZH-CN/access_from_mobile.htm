从智能手机或平板电脑访问内容

您可以使用智能手机或平板电脑访问 My Cloud 设备中存储的内容。在执行此操作之前，必须已完成以下任务：设置云访问、下载 WD 移动应用和执行一次性设置以获得安全访问权限。

使用以下步骤从智能手机或平板电脑访问内容。

    1. 确保您的移动设备已建立 Wi-Fi 或 3G 或更高级别的连接。
    2. 启动 WD 移动应用（WD My Cloud 或 WD Photos）。
    3. 如果出现提示，请点击“立即连接”。
    4. 选择 My Cloud 设备。
    5. 选择要访问的共享，然后浏览到所需的内容。
