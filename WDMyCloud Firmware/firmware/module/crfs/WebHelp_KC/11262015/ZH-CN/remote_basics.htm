关于远程访问的基本知识

可以从任何地方，通过三种方式访问存储在 My Cloud 中的内容：

    • 从智能手机或平板电脑等移动设备。
    • 从计算机（最常见的是便携式计算机）。

移动设备访问概述

使用以下步骤从智能手机或平板电脑访问云：

1

获得 WD My Cloud 登录账户。这通常在将用户添加到设备时完成。请参阅设置云访问。

2

将免费的 WD Photos 和 WD My Cloud 应用下载并安装到移动设备上。
这些应用可从 Apple App Store、Google Play store 或适用于 Android 的 Amazon Appstore 获得。

3

执行一次性设置以获得安全访问权限。请参阅执行一次性设置以获得安全访问权限

4

从移动设备访问您的内容。请参阅从智能手机或平板电脑访问内容

Windows 或 Mac 桌面访问概述

当您离开家中时（例如在咖啡屋），您可以使用便携式计算机轻松访问您的设备。远程访问需要 WD My Cloud 桌面应用程序、启用 Wi-Fi 的便携式计算机和 Wi-Fi 连接。

使用以下步骤通过 WD My Cloud 桌面应用程序访问云：

1

获得 My Cloud 登录账户。这通常在将用户添加到设备时完成。请参阅设置云访问。

2

从 WD 学习中心 http://wd.com/setup 下载 WD My Cloud 桌面应用程序，并将其安装在计算机上。

3

执行一次性设置以获得安全访问权限。请参阅执行一次性设置以获得安全访问权限

4

从便携式计算机访问您的内容。请参阅“从便携式计算机访问内容”
