创建背面 USB 备份

使用以下步骤将 My Cloud™ 设备上的数据备份到 USB 设备或者将 USB 设备数据备份到 My Cloud 设备。

注：此信息只适用于 My Cloud 背面的 USB 端口。请参见使用 USB 复制按钮创建前端 USB 端口备份了解使用设备前端的 USB 端口创建 USB 备份副本的信息。

    1. 在导航条上单击“备份”，即可显示备份操作屏幕。
    2. 如果尚未选择，请单击USB备份。
    3. 单击创建作业。
    4. 输入以下信息，创建 USB 备份作业：

      作业名称

      输入备份的作业名称。

      远程服务器

      从下拉菜单中选择备份方向。选项包括：

      • USB 到 NAS：将 USB 设备上的数据备份到 My Cloud 设备。
      • NAS 到 USB：将 My Cloud 设备上的数据备份到 USB 设备。     

      源文件夹

      单击浏览并选择要备份的文件夹。

      目标文件夹

      单击浏览并选择备份的目标文件夹。

      备份类型

      指出要执行的备份类型。

      • 复制
      • 同步

      连接时自动启动

      连接设备后，此选项自动开始作业。单击开关按钮打开或关闭选项。

       

    1. 单击创建。
    1. 在“USB 备份作业”区域，单击开始备份图标以开始备份。
      备份进度会在“USB 备份作业”区域显示。
