重新启动 My Cloud 设备

重要说明：重新启动会中断当前正在访问 My Cloud 设备的用户的服务。重新启动设备时，请不要将它断电。

使用以下步骤重新启动 My Cloud 设备：

在“仪表板”中：

    1. 在页面顶部单击。
    2. 选择重新启动
    3. 查看确认消息，然后单击确定。My Cloud 安全地重新启动。

 

在“设置”中：

    1. 单击设置。
    2. 在左窗格中，单击实用工具。
    3. 在“设备维护”区域中，单击重新启动。
    4. 查看确认消息，然后单击确定。My Cloud 安全地重新启动。
