Slå på/av molnåtkomst

Om du är administratör kan du slå på/av molnåtkomst för alla användare (dvs. du kan styra om datorenheter kan komma åt innehållet påMy Cloud™ -enheten).

Använd följande steg när du vill aktivera/inaktivera molnåtkomst.

    1. Klicka på Inställningar.
    2. Klicka på Allmänt i den vänstra rutan.
    3. I fältet Molnåtkomst>Molntjänst klickar du på för att slå på7av molntjänsten efter vad som är lämpligt.
    • På: Fjärråtkomst aktiveras för alla användare.
    • Av: Fjärråtkomst inaktiveras för alla användare.
    1. Klicka på Konfigurera för att ändra typ av anslutning som du använder för din molnåtkomst. Det finns tre åtkomstalternativ på skärmen Anslutningsalternativ för Molnåtkomst.
    2. Auto: Auto använder UPnP (Universal Plug and Play) för att försöka öppna portarna på routern. Om det lyckas görs en direkt anslutning mellan din enhet och dina appar.
    3. Manuellt: Etablerar en anslutning genom de två valda portarna. Om endera porten inte är tillgänglig etableras en reläanslutning. En manuell routerkonfiguration krävs för detta alternativ. Se svar ID 8526 i kunskapsbasens för information om hur du ställer in routern för manuell inställning.
    4. Win XP: En anslutning upprättas via portarna 80 och 443. Om dessa portar inte är tillgängliga upprättas en reläanslutning. Det här alternativet krävs om du använder Windows XP.
      OBS! Som standard upprättar My Cloud -enheten en direkt anslutning mellan dina mobila enheter och routern automatiskt.
    1. I fältet Innehållsdatabas klickar du på Återskapa för att återskapa din WDMyCloud-databas.
      OBS! Utför endast detta alternativ för att felsöka om du misstänker databasfel.
    2. Klicka på Apply (Verkställ). Fältet Anslutningsstatus indikerar att enheten är ansluten.

    Läs Ta bort en enhet för information om hur du inaktiverar åtkomst för en enda enhet.

    OBS! I Konfigurera anslutningsalternativ finns information om området Anslutningsalternativ. Olika brandväggsalternativ förklaras och innehåller instruktioner om hur man kan förbättra prestanda. Som standard upprättar My Cloud -enheten en direkt anslutning mellan dina mobila enheter och routern automatiskt.

    Om mobil åtkomst aktiveras via en reläanslutning (dvs Anslutningsstatus säger "Ansluten (relä)," kan du förbättra prestandan genom att aktivera vidarebefordran av port på din WD Venue-kontakt (se Aktivera vidarebefordran av port).
