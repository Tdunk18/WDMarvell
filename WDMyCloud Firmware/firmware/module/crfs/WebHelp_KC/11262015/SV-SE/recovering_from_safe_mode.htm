Återställa från felsäkert läge

Så här återställer du enheten från felsäkert läge.

    1. Innan du utför dessa steg hämtar du My Cloud fast programvara från WD:s supportwebbplats (http://support.wd.com) och sparar den på en plats som du kan bläddra från datorn. Se till att du packar upp filen med fast programvara.
    2. Logga in på routerns DHCP LAN/Enhetsklienttabell för att få enhetens IT-adress.
    3. Ange enhetens IP-adress i ett webbläsarfönster. Gränssnittet för felsäkert läge visas.
    4. Bläddra till platsen där du sparade den hämtade fasta programvaran.
    5. Klicka på Apply (Verkställ) för att läsa in den fasta programvaran.
    6. Klicka på OK i dialogrutan. Enheten startas om.
    7. Rensa webbläsarens cacheminne när enheten startats om. Kontrollera webbläsarens hjälp för att avgöra hur du bäst rensar cacheminnet.
    8. När enheten startats om fullständigt rekommenderar vi att du gör en systemåterställning.
