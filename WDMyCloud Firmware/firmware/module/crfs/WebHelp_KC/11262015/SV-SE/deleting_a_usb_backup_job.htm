Ta bort ett USB-säkerhetskopieringsjobb

Så här tar du bort ett USB-säkerhetskopieringsjobb.

OBS! Du kan inte ta bort ett säkerhetskopieringsjobb som skapats med hjälp av knappen USB-kopia på enhetens framsida.

    1. Klicka på Säkerhetskopior.
    2. Klicka på USB-säkerhetskopior.
    3. I avsnittet ett USB-säkerhetskopieringsjobb markerar du det säkerhetskopieringsjobb du vill ta bort.
    4. Klicka på ikonen Ta bort.
    5. Klicka på OK. Säkerhetskopieringsjobbet har nu tagits bort.
