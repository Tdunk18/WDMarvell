Aktivera Resurssammansättning

Resurssammansättning liknar Distributed File System (DFS), som sammanför resurser från andra My Cloud™ -system och nätverksenheter som stöder SMB-protokollet. Använd följande steg för att aktivera Resurssammansättning för din My Cloud -enhet.

    1. Klicka på Inställningar.
    2. Klicka på Nätverk i den vänstra rutan.
    3. I avsnittet Nätverkstjänster>Resurssammansättning klickar du på  för att aktivera Resurssammansättning.
    4. Ange Rotmappnamn. Rotmappen fungerar som behållare för länkade fjärresurser och har skapats under resursen Resurssammansättning.
    5. Klicka på Apply (Verkställ).
    6. Klicka på Lägg till länk.
    7. Ange följande information:

    Lokalt mappnamn

    Ange ett namn för mappen som kommer att visas under rotmappen.

    Fjärrvärd

    Ange värdnamnet eller IP-adressen för målenheten.

    Fjärresurser

    Ange fjärresursensnamn eller klicka på Hämta fjärrresursmapp, välj sedan resursen från rullgardinsmenyn till höger om fältet.
    1. Klicka på Apply (Verkställ).
    2. Klicka på Avbryt för att stänga fönstret Inställning av resurssammansättning.
