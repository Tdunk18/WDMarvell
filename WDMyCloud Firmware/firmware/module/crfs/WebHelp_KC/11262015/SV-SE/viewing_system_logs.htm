Visa systemloggar

Systemloggar ger en förteckning över de händelser som inträffat på enheten. Använd följande steg för att visa dina systemloggar.

    1. Klicka på Inställningar.
    2. Klicka på Funktioner i den vänstra rutan.
    3. I avsnittet Systemloggar klickar du på Visa loggar
    4. Granska enhetens loggposter i dialogrutan Visa loggar. Så här anpassar du loggen:
    • Välj en loggnivå från rullgardinsmenyn.
    • Välj alternativet Filtrera genom från rullgardinsmenyn.
    • För att rensa loggen klickar du på Rensa.
    1. När du har fått loggen klickar du på Stäng.
