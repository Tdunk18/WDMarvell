Visa aktuellt RAID-läge

Använd följande steg för att visa det RAID-läge som används på enheten.

    1. Klicka på Lagring.
    2. Klicka på RAID.
    3. I den högra rutan visas följande information:
    • RAID-hälsa.
    • Automatisk återskapningsstatus (oavsett om Automatisk påslagning är påslagen).
    • RAID-volym som visar antalet volymer för vilka RAID eller JBOD är konfigurerade.
