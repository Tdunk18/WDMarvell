Om användare

Alla som får åtkomst till ditt personliga moln måste ha ett användarkonto för att använda My Cloud -enheten. Som ägare av enheten har du ett särskilt användarkonto (admin) som ger dig administratörsbehörighet. Med hjälp av dessa privilegier kan du installera och konfigurera enheten utifrån dina specifika behov och lägga till andra personer i ditt personliga moln. Du har också befogenhet att avgöra exakt vem som använder din enhet.

Som standard har användarnamnet admin inte något lösenord. Du kan lägga till ett när som helst.

Viktigt! När du initialt ställer in din enhet medan du kör My Cloud -enhetens installationsprogrammet eller guiden Komma igång, har du möjlighet att ändra administratörsnamnet och ange ett valfritt lösenord. Se till att du håller reda på detta namn och lösenord. Du kommer inte att kunna visa My Cloud webbgränssnitt utan dessa inloggningsuppgifter.

Som administratör för My Cloud -enheten kan du:

    • Lägg till användare

    • Lägga till eller redigera användares lösenord

    • Visa användarinformation

    • Ändra en användares åtkomst till en resurs  

    • Ta bort användare.

    • Ändra en användares namn och lösenord
