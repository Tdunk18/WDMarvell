Utföra engångsinstallation för säker åtkomst

Den här proceduren förklarar hur du kan upprätta en säker anslutning mellan din smartphone eller surfplatta och din My Cloud™ -enhet. Använd instruktionerna för din specifika mobila enhet:

    • Smartphone eller surfplatta med Android  

    • iPhone eller iPad

    • Telefon eller surfplatta med Windows

    • Stationär dator med Windows eller Mac

Säker åtkomst för en smartphone eller surfplatta med Android

    1. Anslut din smartphone eller surfplatta genom att göra följande:

    • Tryck på Inställningar på din mobila enhet.

    • Tryck på Trådlöst och nätverk.

    • Tryck på Wi-Fi-inställningar.

    • Välj ditt hemnätverk.

    • Ange ditt nätverkslösenord och tryck på Anslut.

    1. Starta en WD-app.
    2. Tryck på Anslut nu.
    3. Tryck på My Cloud -enhetens namn.
    4. Tryck på pilen för den nedrullningsbara menyn och välj sedan ditt användarnamn.
    5. Utför något av följande:

    • Om ett lösenord har angetts för denna användare anger du det och trycker sedan på Logga in.

    • Om ett lösenord inte har angetts för användaren trycker du på Logga in.

    1. (Valfritt) Om du vill komma åt ditt innehåll trycker du på ett resursnamn och navigerar sedan till önskad fil.

Säker åtkomst för en iPhone eller iPad

    1. Anslut din smartphone eller surfplatta genom att göra följande:

    • Tryck på Inställningar på din mobila enhet.

    • Tryck på Wi-Fi.

    1. Tryck på Wi-Fi-inställningar.

    2. Välj ditt hemnätverk.

    3. Ange ditt nätverkslösenord och tryck på Anslut.

    1. Starta en WD-app.
    2. Tryck på Anslut nu.
    3. Tryck på plustecknet (+) bredvid My Cloud -enhetens namn.
    4. Tryck på det önskade användarnamnet.
      Om ett lösenord har angetts för användaren anger du det.
    5. (Valfritt) Om du vill komma åt ditt innehåll trycker du på ett resursnamn och navigerar sedan till önskad fil.

Säker åtkomst för en smartphone eller surfplatta med Windows

    • Anslut din smartphone eller surfplatta genom att göra följande:

    • Tryck på Inställningar på din enhet.

    • Tryck på Wi-Fi.

    • Välj ditt hemnätverk.

    • Ange ditt nätverkslösenord och tryck på Klar.

    1. Starta en WD-app.
    2. Tryck på Anslut nu.
    3. Tryck på My Cloud -enhetens namn.
    4. Tryck på pilen för den nedrullningsbara menyn och välj ett användarnamn.
      Om ett lösenord har angetts för användaren anger du det.
    5. (Valfritt) Om du vill komma åt något av ditt innehåll, gör följande:

    • På skärmen Enheter trycker du på din My Cloud enhets namn.

    • Tryck på en resurs och navigera sedan till önskat innehåll.

Säker åtkomst för en stationär eller bärbar dator med Windows eller Mac

    1. När du har installerat appen WD My Cloud för skrivbord, dubbelklickar du på genvägen som nu visas på skrivbordet för att visa alla WD-enheter i nätverket. Information om att installera programmet finns i Grunderna för fjärråtkomst.
    2. Information om säkerhetsvarningar kan visas i systemet. Om du inte vill att skärmen ska visas igen markerar du kryssrutan om att du förstår.
    3. Klicka på Jag förstår för att visa enheter i ditt nätverk.
    4. Klicka på enheten som du vill ansluta till, My Cloud -enheten i det här fallet. Ett inloggningsskärm visas.
    5. Ange lösenordet till My Cloud -enheten och du har ställt in ett tidigare och klicka på Logga in. Skärmen Komma igång visas.
    6. Vill du ha mer information klickar du på länken till WD Lärocenter.
    7. Klicka på Start för att visa My Cloud -enhetens resurser och alla SmartWare- eller Time Machine-säkerhetskopieringsfiler.
    8. Om du vill veta hur du kommer åt och hanterar innehåll som lagras på din My Cloud -enhet på distans klickar du på länken Åtkomst med din dator i My Cloud hämtningspanel på online inlärningscentret.
