Aktivera portvidarebefordran

Om fjärråtkomst till ditt moln sker genom en reläanslutning kan du förbättra nätverkets prestanda genom att aktivera portvidarebefordran.

OBS! För att avgöra om din mobila åtkomstanslutning förmedlas, se  Kontrollera status för fjärråtkomst.

Viktigt: Innan du börjar den här proceduren måste du känna till det giltiga intervallet av portnummer som du kan använda för din router och välja lämpliga portar för HTTP och HTTPS. Det giltiga portnummerintervallet är 1024–66535. Ofta tilldelas port 80 för HTTP och port 443 för HTTPS.

Använd följande steg när du vill aktivera portvidarebefordran.

    1. Klicka på Inställningar.
    2. Klicka på Allmänt i den vänstra rutan.
    3. I avsnittet Molnåtkomst klickar du på för att aktivera Molnåtkomst i tillämpliga fall.
    4. Klicka på Konfigurera.
    5. Klicka på Manuell för alternativet Konnektivitet.
    6. I fältet Extern port 1 anger du numret på den port genom vilken HTTP-sessionerna kommer att ske.
    7. I fältet Extern port 2 anger du numret på den port genom vilken HTTPS-sessionerna kommer att ske.
    8. Klicka på Apply (Verkställ).
    9. Konfigurera din router så att den kan använda portarna du angav i steg 6 och 7.
