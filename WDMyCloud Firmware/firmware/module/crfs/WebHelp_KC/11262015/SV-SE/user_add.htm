Lägga till ett användarkonto

Följ stegen nedan för att lägga till en användare till din My Cloud™ -enhet:

    1. Klicka på Användare.
    2. Klicka på ikonen Lägg till användare.
    3. Ange ett Användarnamn. Namnet som används för att logga in på systemet.
    4. Ange ett förnamn. Detta är användarens förnamn.
    5. Ange ett efternamn. Detta är användarens efternamn.
    6. Ange en E-postadress. Detta är användarens e-postadress. Denna adress används också för att skapa ett WD My Cloud-konto.
    7. Klicka på Apply (Verkställ). Den nya användaren visas i den vänstra rutan.
