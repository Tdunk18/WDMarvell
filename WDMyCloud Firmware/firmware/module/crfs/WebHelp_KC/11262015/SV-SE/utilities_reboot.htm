Starta om My Cloud -enheten

Viktigt: Omstart stör servicen för användarna som för närvarande har åtkomst till My Cloud -enheten. Koppla inte ur enheten under omstart.

Använd följande steg för att starta om My Cloud -enheten:

Från instrumentpanelen:

    1. Högst upp på sidan klickar du på .
    2. Välj Starta om.
    3. Granska bekräftelsemeddelandet och klicka sedan på OK. Din My Cloud startar om på ett säkert sätt.

 

Från inställningar:

    1. Klicka på Inställningar.
    2. Klicka på Funktioner i den vänstra rutan.
    3. I avsnittet Enhetsunderhåll klickar du på Starta om.
    4. Granska bekräftelsemeddelandet och klicka sedan på OK. Din My Cloud startar om på ett säkert sätt.
