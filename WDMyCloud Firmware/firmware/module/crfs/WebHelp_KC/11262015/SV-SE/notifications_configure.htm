Konfigurera meddelanden

My Cloud™ -enheten tillhandahåller meddelanden om olika händelser, som t.ex. uppdateringar av fast programvara, lyckade installationer av fast programvara och systemstopp. Meddelanden visas i avsnittet Avisering längst upp på sidan och är också tillgängliga via e-post, beroende på hur enheten är konfigurerad.

Meddelandesidan låter dig ställa in e-postadresser för upp till fem användare som kommer att få meddelanden, ställa in SMS-aviseringsmeddelanden för mobila enheter och ställa in aviseringsnivån som du kommer att meddelas för.

E-postmeddelanden

  1. I Navigeringsfältet klickar du på ikonen Inställningar och klickar på Meddelanden i det vänstra fönstret.
  2. I fältet Aviseringsmeddelanden via e-post klickar du på växlingsknappen till PÅ.
  3. Klicka på Konfigurera.
  4. På skärmen Aviseringsmeddelande använder du skjutreglaget för att välja nivå på aviseringar som du vill ta emot via e-post:
  • Endast kritiska: Skicka bara kritiska aviseringar till den angivna e-postadressen.
  • Kritiska och varning: Skicka både kritiska och varningsaviseringar till den angivna e-postadressen.
  • Alla: Skicka alla aviseringar (informations, kritisk, varning) till den angivna e-postadressen.
  1. Klicka på Nytt e-postmeddelande.
  2. Ange den e-postadress där du vill ta emot aviseringar via e-post och klicka på Spara.
  3. Upprepa steg 5-6 för att ange upp till fem e-postadresser.
  4. Klicka på Skicka testmeddelande via e-post att validera de e-postadresser som du angav.
  5. Klicka på OK och kontrollera sedan din e-post efter en bekräftelse.

 

Aktivera SMS-meddelanden

OBS! Kontrollera med din SMS-serviceleverantör för deras krav för att skicka SMS. Vissa transportörer kan kräva att du skickar SMS/textmeddelanden via e-post.

  1. I Navigeringsfältet klickar du på ikonen Inställningar och klickar på Meddelanden i det vänstra fönstret.
  2. För att aktivera SMS, klickar du på växlingsknappen till PÅ.
  3. Klicka på Konfigurera.
  4. På skärmen SMS-inställningar, ange namnet på din SMS-leverantör.
  5. Ange SMS URL med meddelandeinnehållet inställt på att vara "Hello world."
  • Denna SMS URL används endast för installationen. Inget SMS skickas när installationen är klar. Få kraven för URL-format från din SMS-serviceleverantör. De bör innehålla följande parametrar: användarnamn, lösenord, destinationstelefon och meddelandeinnehåll.
  1. Klicka på Nästa.
  2. Välj motsvarande kategori för varje SMS-parameter från rullgardinsmenyn.
  3. Klicka på Slutför. Din SMS-avisering är nu inställd.
