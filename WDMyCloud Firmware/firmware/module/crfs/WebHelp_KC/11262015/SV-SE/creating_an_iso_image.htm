Skapa en ISO-bild

En ISO-bild är en fil som är en exakt kopia av ett befintligt filsystem. Använd följande steg för att skapa ISO-bild.

Använd följande steg för att skapa ISO-bild:

    1. Klicka på Inställningar.
    2. Klicka på Funktioner i den vänstra rutan.
    3. I avsnittet ISO-resurslista klickar du på Skapa ISO-bild.
    4. I fältet Bildstorlek väljer du bildstorleken som du vill använda för din ISO-bild från rullgardinsmenyn.
    5. I fältet Bildsökväg anger du sökvägen för de data du vill använda till bilden eller klickar på Bläddra för att välja den folder som du vill använda till bilden och klickar sedan på OK.
    6. I fältet Bildnamn anger du ett namn för ISO-bilden.
    7. Klicka på Nästa.
    8. I fältet Välj klickar du på Skriv över eller Hoppa över.
    • Skriv över: När Skriv över väljs, kommer filer med samma namn att skrivas över.
    • Hoppa över: När Hoppa över väljs, kommer filer med samma namn att hoppas över.
    1. Markera de mappar och filer som du vill lägga till i din bildfil i den vänstra kolumnen och klicka på Lägg till>>.
    2. Markera de mappar och filer som du vill ta bort från ISO-bilden och klicka sedan på <<ta bort.
    3. Klicka på Nästa. Din ISO-bild har skapats.
    1. Klicka på Slutför.
