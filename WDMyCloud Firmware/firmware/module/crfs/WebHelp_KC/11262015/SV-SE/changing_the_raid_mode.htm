Ändra RAID-läge

Använd följande steg för att ändra det nuvarande RAID-läget på din My Cloud™ -enhet.

Varning: Ändringar som görs i ditt RAID-läge raderar alla dina data och dina användarinställningar. Se Exportera systeminställningar för information om hur du sparar dina användarinställningar.

    1. Klicka på Lagring.
    2. Klicka på RAID, om det inte redan har valts.
    3. Klicka på Ändra RAID-läge längst ned på skärmen.
    4. Läs varningsmeddelandet och klicka på OK.
    5. Välj det RAID-läge som du vill använda för din My Cloud -enhet. Följande alternativ är tillgängliga:

Läge

Beskrivning

JBOD

En eller flera enheter utan RAID-konfiguration som hanteras som separata logiska volymer.

Disklänkande

Flera enheter kombineras linjärt för att bilda en stor logisk volym.

RAID 0

Data delas över flera hårddiskar vilket gör att data kan läsas och skrivas fortare då två eller fler enheter kan arbeta samtidigt för ökad prestanda. Men om en enhet går sönder, kommer all data att gå förlorad.

RAID 1

Två ögonblickliga kopior av uppgifterna registreras på separata hårddiskar så att om en av dem misslyckas, kommer den/de andra att fortsätta läsa/skriva data tills den felaktiga hårddisken byts ut.

RAID 5

Erbjuder överlägsen prestanda och bättre skydd då data delas över 3 eller fler enheter och en kvart av varje enhet dediceras till feltolerans. Om en av enheterna går sönder, kommer den/de andra att fortsätta läsa/skriva data tills den felaktiga hårddisken byts ut.

Detta alternativ finns endast tillgängligt för My Cloud -enheter med 4 fack.

RAID 10

RAID-protokoll där data skrivs i delsektioner över huvudenheter som speglats till sekundära enheter. Om en av enheterna går sönder, kommer den/de andra att fortsätta läsa/skriva data tills den felaktiga hårddisken byts ut.

Detta alternativ finns endast tillgängligt för My Cloud -enheter med 4 fack.

    1. Granska och välj det lagringsalternativ som du vill använda och välj kryssrutan Ändra till <Storage Mode> och klicka sedan på Nästa. Ett självtest av enheten utförs.
    2. Kontrollera varningarna överst på skärmen.
    3. När självtestet är klart granskar du status på dina hårddiskar och klickar på Nästa.
    4. Granska status på dina volymer och klicka på Nästa.
    5. Om volym-reglaget visas, väljer du hur mycket utrymme du vill avsätta för det valda RAID-läget.
    6.  Om du väljer att inte använda hela volymen för RAID-valet, kan du konfigurera det återstående diskutrymmet som strimling genom att välja kryssrutan Konfigurera återstående diskutrymme som strimling.

    1. Med volymreglaget väljer du hur mycket utrymme du vill avsätta för det valda RAID-läget.

      OBS! Om du väljer att inte använda hela volymen för RAID-urvalet kan du konfigurera det återstående diskutrymmet som Disklänkning genom att markera kryssrutan Konfigurera återstående diskutrymme som Disklänkning.

    1. För RAID 5 kan du konfigurera RAID 5+spare, vilket gör att "Disk4" blir reserv. Markera kryssrutan Skapa reservdisk för att konfigurera RAID 5+Spare. Avmarkera kryssrutan Skapa reservdisk om du inte vill skapa en RAID 5+spare.
    2. Klicka på Nästa för att fortsätta.
    3. Om du vill automatiskt återskapa RAID när disken är erkänd, klickar du på och sedan klickar du på Nästa.

      OBS! Den här skärmen visas inte för lägena JBOD, Disklänkande och RAID 0
