Aktivera FTP-åtkomst

Du kan aktivera eller inaktivera överföring av filer via FTP (File Transfer Protocol) mellan enheter i nätverket.

Använd följande steg för att aktivera/inaktivera FTP-åtkomst:

    1. Klicka på Inställningar.
    2. Klicka på Nätverk i den vänstra rutan.
    3. I avsnittet Nätverkstjänst>FTP-åtkomst klickar du på för att aktivera eller inaktivera FTP-åtkomst.
    4. Läs meddelandet och klicka sedan på OK.
    5. Klicka på Konfigurera.
    6. På skärmen FTP-inställningar, anger du följande information:
    • Maximala användare: Från rullgardinsmenyn väljer du det maximala antalet användare som du vill ha FTP-åtkomst för.
    • Inaktiv tid: Ange den tid i minuter som du väljer FTP att vara inaktiv innan den kopplas bort.
    • Port: Ange porten som ska användas för FTP-åtkomst.
    • Flödeskontroll: Välj antingen Obegränsad eller Anpassa. Om du väljer Anpassa, anger du ett flödeskontrollvärde.
    1. Klicka på Nästa.
    2. I fältet Passivt läge väljer du antingen Standard eller Anpassa. Om du väljer Anpassa anger du värden för passivt.
    3. Markera kryssrutan om du vill rapportera extern IP i PASV (Passivt) läget.
    • Ange den externa IP-adress, eller klicka på Skaffa IP för att systemet ska ange adressen.
    1. Klicka på Nästa.
    2. Ange följande information:
    3. Klientspråk: Välj klientspråk i rullgardinsmenyn.
    4. TLS: Markera kryssrutan Transport Layer Security (TLS) antingen bredvid implicita TLS eller explicita TLS-anslutningar om tillämpligt.
    5. FXP: Klicka på för att aktivera eller inaktivera filöverföringsprotokoll (FXP).
    1. Klicka på Nästa.
    2. Ange eventuella IP-adresser som du vill blockera från FTP-åtkomst, välj sedan antingen Permanent eller Tillfällig från rullgardinsmenyn. Klicka på Verkställ för att spara dina poster.
    3. Klicka på Slutför.
