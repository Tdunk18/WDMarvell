Skapa volymvirtualisering

Volymvirtualisering förenklar skalbarhet genom att låta dig kartlägga iSCSI-mål på andra nätverks lagringsenheter och presentera dem som volymer på din My Cloud™ -enhet.

Använd följande steg för att skapa Volymvirtualisering på enheten.

    1. Klicka på Lagring.
    2. Klicka på Volymvirtualisering.
    3. Klicka på Skapai den högra rutan.
    4. I fältet Enhetens IP anger du IP-adressen för enheten där iSCSI-målet finns och klicka sedan på Nästa.
    5. Välj ett iSCSI-mål från den hämtade listan och klicka på Nästa.
    6. Om du vill lägga till autentisering för målet klickar du på , anger ett Användarnamn och Lösenord och klickar på Nästa.
    • Autentisering är inaktiverat som standard. För att bevara standardalternativet kontrollerar du att autentiseringsfältet är inaktiverat och klickar på Nästa.
    1. Välj en LUN (Local Unit Number) i listan och klicka på Nästa.
    2. Ange ett namn för den delade mappen och klicka sedan på Nästa.
    3. Granska volymvirtualiseringens sammanfattning för att säkerställa att dina inställningar är korrekta och klicka på Verkställ. En virtuell volym skapas.
