Överföra innehåll till My Cloud -enheten

För att överföra bilder, videoklipp, musik och filer till My Cloud -enhet använder du en av följande procedurer:

    • Mac
    • Windows PC
    • USB

Överföra innehåll från en Mac

Använd följande steg för att överföra filer från din Mac till din My Cloud -enhet:

    1. På din Mac väljer du Gå > Nätverk från menyfältet.
    2. Utför något av följande:
    • I det vänstra fönstret i Sökaren dubbelklickar du på My Cloud -enhetens namn.
    • I det högra fönstret lokaliserar du My Cloud -enhetens namn och dubbelklickar på det.
    1. Dubbelklicka på namnet för en viss resurs för att öppna mappen och visa alla befintliga undermappar. (Resursen Offentlig skapas automatiskt.)
    2. Dra och släpp filer från datorn till önskad mapp.
      Lägg inte filer i någon av mapparna Time Machine eller WD SmartWare Backups.
      OBS! Om du kopierar filer från en dator till My Cloud -enheten, kommer dessa filer att befinna sig på två platser: på den ursprungliga platsen och på enheten.

Överföra innehåll från en PC

Använd följande steg för att överföra filer från din Windows PC till din My Cloud -enhet:

    1. På datorn öppnar du dialogrutan Kör genom att trycka på tangenterna Windows och R samtidigt.
    2. I fältet Öppna anger du \\ följt av My Cloud -enhetens namn. (Det förvalda namnet är WDMyCloud, utan mellanslag.) Exempel: \\wdmycloud
    3. Klicka på OK.
      Utforskaren öppnas och tillgängliga resurser visas i den högra rutan.
    4. Dubbelklicka på namnet för en viss resurs för att öppna mappen och visa alla undermappar. (Resursen Offentlig skapas automatiskt.)
    5. Dra och släpp filer från datorn till resursen.
      OBS! Om du kopierar filer från en dator till My Cloud -enheten, kommer dessa filer att befinna sig på två platser: på den ursprungliga platsen och på enheten.

     

    Överföra innehåll från en USB

    Se Skapa säkerhetskopior för främre USB-port med knappen USB-kopiaför steg om hur du snabbt överför USB-enhetens innehåll.
