Создание резервной копии данных камеры

Резервное копирование позволяет создавать резервные копии данных камеры на устройстве My Cloud™. После загрузки пользователи могут перейти в место сохранения данных камеры для доступа к ним, если у них есть доступ к этому ресурсу.

Описанные далее действия позволяют выполнить резервное копирование данных камеры на устройство My Cloud.

    1. Убедитесь, что камера подключена к устройству My Cloud.
    2. На панели навигации нажмите Резервные копии, чтобы открыть экран «Резервные копии».
    3. Нажмите Резервные копии камеры, если этот элемент еще не выбран.
    4. Убедитесь, что в области «Подключенные камеры» указана ваша камера.
    5. В области «Настройки» введите следующую информацию для выполнения резервного копирования данных камеры.

      Автоматический перенос

      Нажмите на переключатель для автоматического переноса данных с камеры на устройство My Cloud.

      Режим перемещения

      Выберите способ переноса данных с камеры. Доступны следующие варианты:

      • Копировать. Копирование информации с камеры на устройство My Cloud. Исходные данные останутся в камере.

      • Переместить. Перемещение файлов с камеры на устройство My Cloud с удалением файлов в камере.

      Параметры папки

      Выберите данный параметр, чтобы указать место для сохранения файлов, перемещенных с камеры.

      1. На экране параметров папки камеры:

        • Папка для передачи: Нажмите «Просмотреть», чтобы указать место на вашем устройстве для сохранения перенесенных данных, затем нажмите «ОК».

        • Название папки: Выберите тип названия папки из раскрывающегося меню.

          • При выборе пользовательского названия папки введите название в поле «Введите название папки».

      1. Нажмите Сохранить.

    1. В поле «Переместить файлы» нажмите Копировать / переместить, чтобы начать перенос файлов, если отключена функция автоматического переноса.
    2. После выполнения переноса в поле «Состояние» будет указано, что резервное копирование для подключенного устройства завершено, а также будут указаны день, дата и время окончания загрузки.
