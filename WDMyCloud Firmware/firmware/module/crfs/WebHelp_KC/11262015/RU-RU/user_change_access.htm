Изменение доступа к ресурсу

Вносить изменения в права доступа для пользователя можно на экране Пользователи или на экране Ресурсы. Описанные далее действия позволяют изменить права доступа пользователя к выбранному ресурсу.

Примечание. Ресурс необходимо сначала сделать частным на экране «Ресурсы». Все общедоступные ресурсы неактивны. Для получения дополнительной информации см. О ресурсах.

Изменение доступа к ресурсу на экране «Пользователи»

    1. Нажмите Пользователи.
    2. На панели слева выберите пользователя.
    3. В разделе «Доступ к ресурсу» измените права доступа для пользователя, нажав на значок, соответствующий ресурсу:
    • доступ для чтения и записи;
    • доступ только для чтения;
    • запретить доступ.

 

Изменение доступа к ресурсу на экране «Ресурсы»

    1. Нажмите Ресурсы.
    2. На панели слева выберите ресурс.
    3. Убедитесь, что не установлен флажок «Открытый доступ». Открытый доступ позволяет любому пользователю видеть данный ресурс.
    • Если открытый доступ активирован, нажмите .
    1. В разделе «Доступ пользователя» измените права доступа для пользователя, нажав на значок, соответствующий ресурсу:
    • доступ для чтения и записи;
    • доступ только для чтения;
    • запретить доступ.
