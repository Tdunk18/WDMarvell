Добавление учетной записи пользователя

Описанные далее действия позволяют добавить пользователя на устройство My Cloud™.

    1. Нажмите Пользователи.
    2. Нажмите на значок Добавить пользователя.
    3. Введите имя пользователя. Это имя используется для входа в систему.
    4. Введите имя. Это имя данного пользователя.
    5. Введите фамилию. Это фамилия данного пользователя.
    6. Введите адрес электронной почты. Это адрес электронной почты данного пользователя. Этот адрес также используется для создания учетной записи WD My Cloud.
    7. Нажмите Применить. Новый пользователь появляется на панели слева.
