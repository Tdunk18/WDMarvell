Настройка режимов экономии энергии

Описанные далее действия позволяют сократить энергопотребление устройства My Cloud™.

    1. Нажмите Настройки.
    2. Нажмите Основные.
    3. В области «Экономия энергии» выберите нужные параметры для настройки энергопотребления устройства.

Режим

Описание

Спящий режим диска

Включение данной функции позволяет жестким дискам переходить в режим ожидания с целью экономии электроэнергии по истечении заданного периода бездействия. По умолчанию переход в спящий режим происходит приблизительно через 10 минут.

  • Нажмите , чтобы включить или отключить спящий режим диска.

Экран

Включение данной функции позволяет ЖК-экрану устройства переходить в режим ожидания с целью экономии электроэнергии по истечении заданного периода времени. По умолчанию отключение ЖК-экрана происходит через 10 минут.

  • Нажмите , чтобы включить или отключить экран.

Эта возможность имеется только у устройств My Cloud EX2. Модели My Cloud EX4, My Cloud EX4100 и My Cloud DL4100.

Восстановление питания

Эта функция автоматически выполняет перезапуск устройства после неожиданного завершения работы из-за перебоя в подаче электропитания. Функция восстановления питания включена по умолчанию.

  • Нажмите , чтобы включить или отключить восстановление питания.

Расписание питания

Расписание подачи питания позволяет вам планировать отключения для вашего устройства My Cloud.

  • Нажмите , чтобы включить или отключить расписание питания.
  • Нажмите Настроить для того, чтобы настроить расписание подачи питания.
  • Выберите время, в течение которого ваше устройство должно быть активным для каждого дня недели.
  • Нажмите Сохранить.

 

Превышение времени ожидания веб-доступа

Настройте время по умолчанию, по истечении которого устройство автоматически выполняет выход из учетной записи.

  • В раскрывающемся меню «Превышение времени ожидания веб-доступа» выберите время, которое вы хотите использовать для превышения времени ожидания для вашей системы. Обновление системы происходит автоматически.
