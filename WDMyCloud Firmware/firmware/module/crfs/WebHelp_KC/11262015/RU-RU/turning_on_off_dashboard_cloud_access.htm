Включение и отключение доступа к панели управления из облака

Описанные далее действия позволяют активировать / деактивировать доступ к панели управления из облака.

    1. Нажмите Настройки.
    2. На панели слева нажмите Основные.
    3. В поле «Доступ к панели управления из облака» нажмите , чтобы включить дистанционный доступ к панели управления из облака.
      Примечание. Выбор этой опции позволяет вам выполнять функции администратора дистанционно.
    4. Ознакомьтесь с информацией на экране, введите и подтвердите пароль для своей учетной записи администратора, затем нажмите OK.
    • Если с вашей учетной записью администратора уже связан пароль, ознакомьтесь с информацией на экране «Доступ к панели управления из облака» и нажмите OK.
      Примечание. Если установить ручную настройку облака, вам придется получать доступ к своему устройству дистанционно, используя порты, которые вы выбрали во время ручной настройки облака. Пример. Если вы установили ручной порт HTTP как 5040, то для доступа к устройству вам придется использовать адрес http://<IP-адрес устройства>:5040.
