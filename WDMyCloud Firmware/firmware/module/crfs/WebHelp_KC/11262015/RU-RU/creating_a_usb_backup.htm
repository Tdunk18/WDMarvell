Создание резервного копирования через задний USB-порт

Описанные далее действия позволяют выполнять резервное копирование данных, хранящихся на устройстве My Cloud™, на USB-устройство либо выполнять резервное копирование данных USB-устройства на устройство My Cloud.

Примечание. Данная информация применима только к USB-портам, расположенным на задней стороне вашего устройства My Cloud. См. раздел Создание резервных копий через передний USB-порт с помощью кнопки «USB Copy» для получения информации о создании резервной копии USB с использованием USB-порта, расположенного на передней стороне вашего устройства.

    1. На панели навигации нажмите Резервные копии, чтобы открыть экран «Резервные копии».
    2. Нажмите Резервные копии USB, если этот элемент еще не выбран.
    3. Нажмите Создать задание.
    4. Введите следующую информацию для создания задания резервного копирования USB:

      Название задания

      Введите название задания для резервного копирования.

      Удаленный сервер

      Выберите направление резервного копирования в раскрывающемся меню. Доступны следующие варианты.

      • С USB в NAS. Выполняется резервное копирование данных, хранящихся на USB-устройстве, на устройство My Cloud.
      • С NAS в USB. Выполняется резервное копирование данных, хранящихся на устройстве My Cloud, на USB-устройство.     

      Исходная папка

      Нажмите Обзор и выберите папку, из которой нужно выполнять резервное копирование.

      Папка назначения

      Нажмите Обзор и выберите папку назначения для резервного копирования.

      Тип резервного копирования

      Выберите нужный тип резервного копирования.

      • Копировать
      • Синхронизация

      Автоматический запуск при подключении

      Данная опция позволяет автоматически начать выполнение задания при подключении устройства. Нажмите на переключатель для включения и отключения опции.

       

    1. Нажмите Создать.

    1. В разделе «Задания резервного копирования по USB» нажмите на значок Начать резервное копирование, соответствующий нужному заданию резервного копирования.
      Ход резервного копирования отображается в разделе «Задания резервного копирования по USB».
