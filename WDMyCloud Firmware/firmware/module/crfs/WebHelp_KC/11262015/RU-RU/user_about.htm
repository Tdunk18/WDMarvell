О пользователях

У каждого пользователя, получающего доступ к вашему персональному облаку, должна быть учетная запись, которая позволяет использовать устройство My Cloud. Как владелец устройства вы используете специальную учетную запись (admin) с правами администратора. Используя эти права, можно настраивать устройство в соответствии со своими потребностями и добавлять других пользователей в свое персональное облако. Вы также имеете возможность определить, кто именно может использовать ваше устройство.

По умолчанию для имени пользователя admin пароль не требуется. Его можно установить в любой момент.

Внимание! При первоначальной настройке устройства My Cloud с помощью программного обеспечения настройки или мастера «Начало работы» можно изменить имя администратора и установить необязательный пароль. Позаботьтесь о том, чтобы не забыть имя и пароль. Без этих учетных данных веб-интерфейс My Cloud отображаться не будет.

На правах администратора устройства My Cloud вы можете выполнять следующие действия.

    • Добавление пользователей.

    • Добавление и изменение паролей пользователей.

    • Просмотр сведений о пользователях.

    • Изменение доступа пользователя к ресурсу.  

    • Удаление пользователей.

    • Изменение имени пользователя и пароля.
