Настройка протокола LLTD

Описанные далее действия позволяют настроить протокол обнаружения топологии связи (LLTD) для устройства My Cloud™. LLTD используется для улучшения обнаружения сетей и диагностики на компьютерах под управлением ОС Microsoft Windows.

    1. Нажмите Настройки.
    2. На панели слева нажмите Основные.
    3. В области «Службы Windows > LLTD» нажмите , чтобы включить / отключить протокол LLTD.
