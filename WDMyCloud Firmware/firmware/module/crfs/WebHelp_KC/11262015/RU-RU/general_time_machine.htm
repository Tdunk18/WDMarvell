Резервное копирование данных с помощью Apple Time Machine

Можно выполнять резервное копирование файлов на устройстве My Cloud™ с помощью функции компьютера Mac Apple Time Machine.

Внимание! Необходимо установить пароль для учетной записи администратора, для того чтобы восстановить резервные копии Time Machine

Описанные далее действия позволяют выполнять резервное копирование данных с помощью Apple Time Machine.

    1. Нажмите Настройки.
    2. На панели слева нажмите Основные.
    3. В области «Резервное копирование Mac» нажмите , чтобы включить резервное копирование в Time Machine.
    4. Нажмите Настроить.
    5. В раскрывающемся списке «Выберите ресурс» необходимо выбрать ресурс, на который будут сохраняться резервные копии ваших файлов.
    6. В поле «Макс. размер» перемещайте ползунок, чтобы указать максимальный объем для резервного копирования.
      Примечание. Увеличение максимального значения размера после первоначального резервного копирования не влияет на существующие резервные копии. WD рекомендует выбрать для этой настройки максимальное значение при выполнении первоначального резервного копирования. После завершения первого резервного копирования можно изменить размер резервной копии с помощью ползунка «Макс. размер».
    7. Нажмите Сохранить.

    Внимание! После того как Time Machine начнет выполнять резервное копирование ваших файлов в выбранный ресурс, рекомендуется продолжить выполнять резервное копирование в этот ресурс. Смена ресурса приведет к созданию нового файла резервной копии, который не будет содержать сохраненную ранее информацию.
