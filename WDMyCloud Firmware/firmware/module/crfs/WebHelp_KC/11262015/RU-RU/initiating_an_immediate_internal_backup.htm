Немедленный запуск внутреннего резервного копирования

С помощью следующих действий можно изменить задание внутреннего резервного копирования.

    1. Нажмите Резервные копии.
    2. Нажмите Внутренние резервные копии.
    3. В области «Очередь внутреннего резервного копирования» выберите задание, которое нужно изменить.
    4. Нажмите на значок Начать резервное копирование. Начнется внутреннее резервное копирование.
