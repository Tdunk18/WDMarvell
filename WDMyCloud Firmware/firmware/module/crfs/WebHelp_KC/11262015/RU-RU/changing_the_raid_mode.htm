Изменение режима RAID-массива

С помощью описанных далее действий можно изменить текущий режим RAID-массива на устройстве My Cloud™.

Предупреждение. При изменении режима RAID-массива будут удалены все данные и настройки пользователя. См. раздел Экспорт системных настроек для получения информации о сохранении настроек пользователя.

    1. Нажмите Хранилище.
    2. Нажмите RAID, если этот элемент еще не выбран.
    3. Нажмите Изменить режим RAID в нижней части экрана.
    4. Ознакомьтесь с предупреждением и нажмите ОК.
    5. Выберите режим RAID-массива, который следует использовать для устройства My Cloud. Доступны следующие варианты.

Режим

Описание

JBOD

Использование одного или нескольких дисков не в конфигурации RAID-массива, а путем управления ими как отдельными логическими томами.

Объединение

Объединение дисков линейно для создания одного большого логического тома.

RAID 0

Данные записываются поочередно на несколько жестких дисков, позволяя быстрее считывать и записывать данные за счет объединения работы двух и большего количества дисков для повышения производительности. Если в работе одного диска произойдет сбой, все ваши данные будут потеряны.

RAID 1

Две одномоментные копии данных записываются на отдельных жестких дисках, чтобы в случае отказа одного из них другие продолжали бы чтение или запись данных до замены неисправного диска.

RAID 5

Обеспечивает более высокие уровни производительности и защиты, распределяя данные по 3 и более дискам и выделяя четверть каждого диска на обеспечение отказоустойчивости. В случае отказа одного из них второй продолжает чтение или запись данных до замены неисправного диска.

Эта возможность имеется только у устройств My Cloud с четырьмя гнездами.

RAID 10

Протокол RAID, подразумевающий запись данных поочередно на основных дисках, для которых создаются зеркальные копии на второстепенных дисках. В случае отказа одного из них второй продолжает чтение или запись данных до замены неисправного диска.

Эта возможность имеется только у устройств My Cloud с четырьмя гнездами.

    1. Изучите и выберите опцию хранения, которую вы хотите использовать, выберите Сохранять по <Режим хранения>, установите флажок и нажмите Далее. Выполняется самотестирование диска.
    2. Ознакомьтесь со всеми предупреждениями в верхней части окна.
    3. Как только будет выполнено самотестирование, просмотрите статус ваших накопителей и нажмите Далее.
    4. Проверьте состояние томов, после чего нажмите Далее.
    5. Если отображается ползунок «Том», выберите объем пространства, который следует выделить выбранному режиму RAID-массива.
    6.  Если вы решите не использовать весь доступный объем для выбранного режима RAID-массива, можно установить флажок Настроить оставшееся дисковое пространство для объединения и воспользоваться соответствующей возможностью.

    1. С помощью ползунка «Том» выберите объем пространства, который следует выделить выбранному режиму RAID-массива.

      ПРИМЕЧАНИЕ. Если вы решите не использовать весь доступный объем для выбранного режима RAID-массива, можно установить флажок Настроить оставшееся дисковое пространство для объединения и воспользоваться соответствующей возможностью.

    1. RAID 5 можно настроить как резервный RAID 5+, после этого «Диск4» станет диском горячего резерва. Установите флажок «Создать резервный диск», чтобы настроить резервный RAID 5+. Снимите флажок «Создать резервный диск», если не хотите создавать резервный RAID 5+.
    2. Нажмите Далее, чтобы продолжить.
    3. Если вы хотите автоматически восстанавливать RAID после распознавания диска, нажмите , после чего нажмите Далее.

      ПРИМЕЧАНИЕ. Этот экран не выводится для режимов JBOD, «Объединение» и RAID 0.
    1. Настройте параметр шифрования для каждого диска. Доступны следующие варианты.
    • Заблокировано
    • Разблокировано
    1. Проверьте внесенные в RAID-массив изменения, убедитесь, что они верны, после чего нажмите Далее.
    2. Ознакомьтесь с экраном предупреждения.
      Предупреждение. При изменении режимов массива стираются все данные.
    1. Нажмите Готово. Начинается внесение запрошенных изменений для жестких дисков. Не выключайте устройство My Cloud до окончания процесса внесения изменений.
    2. После завершения процесса нажмите Готово еще раз.
