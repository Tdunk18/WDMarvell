Настройка параметров подключения

Вы можете указать параметр подключения, который будет определять способ получения мобильными устройствами и компьютерами удаленного доступа к устройству My Cloud™ .

Описанные далее действия позволяют настроить параметры подключения.

    1. Нажмите Настройки.
    2. На панели слева нажмите Основные.
    3. В области «Доступ к облаку» в поле «Служба облака» нажмите Настроить.
    4. В диалоговом окне выберите одну из трех настроек.
    • Автоматически: установка прямого подключения между мобильным устройством и маршрутизатором. Если при подключении не удается открыть порты через uPnP (стандарт Universal Plug and Play), устанавливается подключение с ретрансляцией.
    • Вручную: установка подключения через два порта, выбранных пользователем. Этот параметр позволяет настроить прямое подключение с помощью функции перенаправления портов. Пошаговые инструкции по использованию этого параметра приведены в разделе Включение функции перенаправления портов. Для использования данного параметра следует вручную настроить маршрутизатор соответствующим образом.
    • Win XP: устанавливает подключение через порты 80 и 443. Если эти порты недоступны, устанавливается подключение с ретрансляцией. Этот параметр следует выбирать, если хотя бы на одном из удаленных устройств установлена операционная система Windows XP.
    1. Нажмите Применить.
