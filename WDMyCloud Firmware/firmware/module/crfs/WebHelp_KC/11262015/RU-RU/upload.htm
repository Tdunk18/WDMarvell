Выгрузка содержимого на устройство My Cloud

Для передачи на устройство My Cloud фотографий, видеороликов, музыки и других файлов следует использовать одну из перечисленных ниже последовательностей действий.

    • Mac
    • ПК под управлением ОС Windows
    • USB

Передача содержимого с компьютера Mac

Для того чтобы выгрузить файлы с компьютера Mac на устройство My Cloud, выполните следующие действия.

    1. На компьютере Mac в строке меню последовательно выберите Переход > Сеть.
    2. Выполните одно из следующих действий:
    • На левой панели окна Finder дважды нажмите на название устройства My Cloud.
    • На панели справа найдите название устройства My Cloud и дважды нажмите на него.
    1. Дважды нажмите на название нужного ресурса, чтобы открыть папку и просмотреть существующие вложенные папки. (Public — это ресурс, созданный автоматически.)
    2. Перетащите файлы из своего компьютера в нужную папку.
      Не перетаскивайте файлы в папки резервного копирования Time Machine и WD SmartWare.
      Примечание. При копировании с компьютера на устройство My Cloud файлы будут сохранены в двух местах: в исходной папке и на устройстве.

Передача содержимого с ПК

Описанные далее действия позволяют выгрузить файлы с компьютера под управлением ОС Windows на устройство My Cloud.

    1. На компьютере откройте диалоговое окно «Выполнить», воспользовавшись сочетанием клавиш Windows + R.
    2. В поле «Открыть» введите \\ и название устройства My Cloud. (По умолчанию используется название «wdmycloud» без пробелов.) Пример. \\wdmycloud
    3. Нажмите кнопку ОК.
      Отобразится окно Windows Explorer, в правой области которого будут перечислены доступные ресурсы.
    4. Дважды нажмите на название нужного ресурса, чтобы открыть папку и просмотреть вложенные папки. (Public — это ресурс, созданный автоматически.)
    5. Перетащите файлы из своего компьютера в этот ресурс.
      Примечание. При копировании с компьютера на устройство My Cloud файлы будут сохранены в двух местах: в исходной папке и на устройстве.

     

    Выгрузка содержимого с USB-устройства

    В разделе Создание резервных копий через передний USB-порт с помощью кнопки «USB Copy» описаны действия для быстрой выгрузки содержимого USB-устройства.
