Создание загрузки P2P

Приложение одноранговых (P2P) загрузок позволяет добавить в расписание My Cloud™ загрузку P2P. Описанные далее действия позволяют добавить в расписание загрузку P2P.

    1. Нажмите Приложения.
    2. Нажмите Загрузки P2P.
    3. Нажмите , чтобы включить загрузку P2P.
    • Нажмите Настроить, чтобы внести изменения в конфигурацию загрузки P2P.
    1. Введите URL-адрес или укажите файл торрента для загрузки, выполнив следующие действия.
    • Если у вас есть URL-адрес, сделайте следующее.

      • В поле «Добавить торрент из URL-адреса» введите URL-адрес файла торрента, который нужно загрузить.

      • Нажмите Добавить торрент. Загрузка P2P создается и добавляется в очередь загрузок P2P.

    • Если у вас есть файл торрента, сделайте следующее.

      • Выберите вариант Нажмите для выгрузки файла.

      • Выберите файл торрента для выгрузки и нажмите Открыть. Загрузка P2P создается и добавляется в очередь загрузок P2P.
