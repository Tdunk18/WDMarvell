Включение и отключение доступа к облаку

Будучи администратором, вы можете включать и отключать доступ к облаку для всех пользователей (то есть контролировать возможность получения вычислительными устройствами доступа к содержимому устройства My Cloud™).

Описанные далее действия позволяют активировать и деактивировать доступ к облаку.

    1. Нажмите Настройки.
    2. На панели слева нажмите Основные.
    3. В поле «Доступ к облаку>Облачная служба» нажмите , чтобы включить или отключить облачную службу (при необходимости).
    • ВКЛ: включение удаленного доступа для всех пользователей.
    • ВЫКЛ: отключение удаленного доступа для всех пользователей.
    1. Нажмите «Настроить», чтобы изменить тип соединения, используемого для входа в облако. На экране «Параметры подключения для доступа к облаку» существует три варианта доступа.
    2. Автоматически: автоматическое подключение использует UPnP (стандарт Universal Plug and Play), чтобы попытаться открыть порты на вашем маршрутизаторе. В случае успеха устанавливается непосредственное соединение между вашим устройством и приложениями.
    3. Вручную: устанавливает соединение между двумя выбранными портами. Если один из портов недоступен, устанавливается подключение ретрансляции. Для данной опции требуется конфигурирование маршрутизатора вручную. Информацию о том, как настроить маршрутизатор для выполнения конфигурирования вручную, см. в Базе знаний, ID 8526.
    4. Win XP: устанавливает подключение через порты 80 и 443. Если эти порты недоступны, устанавливается подключение с ретрансляцией. Данная опция требуется, если вы используете Windows XP.
      Примечание. По умолчанию устройство My Cloud автоматически устанавливает прямое подключение между мобильными устройствами и маршрутизатором.
    1. В поле «База данных содержимого» нажмите Восстановить, чтобы восстановить базу данных WDMyCloud.
      Примечание. Используйте эту опцию только для устранения неисправностей ,если вы подозреваете повреждение базы данных.
    2. Нажмите Применить. Поле «Состояние подключения» указывает на то, что ваше устройство подключено.

    Чтобы узнать о том, как отключить доступ для одного устройства, см. раздел Удаление устройства.

    Примечание. Информация о параметрах подключения приведена в разделе Настройка параметров подключения, в котором описываются различные параметры брандмауэра и способы повышения производительности. По умолчанию My Cloud автоматически устанавливает прямое подключение между мобильными устройствами и маршрутизатором.

    Если мобильный доступ осуществляется посредством подключения с ретрансляцией (то есть отображается состояние подключения «Подключено (ретрансляция)»), можно повысить производительность за счет включения перенаправления портов на соединителе WD Venue (см. Включение функции перенаправления портов).
