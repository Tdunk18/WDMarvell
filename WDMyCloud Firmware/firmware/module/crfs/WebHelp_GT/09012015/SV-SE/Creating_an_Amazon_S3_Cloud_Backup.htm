Skapa en Amazon S3-molnsäkerhetskopia

Obs: Innan du skapar en molnsäkerhetskopia registrerar du dig för tjänsten Amazon S3 på http://aws.amazon.com/s3.

När ditt Amazon S3-konto har konfigurerats skapar du en molnsäkerhetskopia med följande steg.

    1. Klicka på Säkerhetskopior.
    2. Klicka på Molnsäkerhetskopior.
    3. Klicka på knappen Amazon S3 överst på skärmen.
    4. Klicka på Skapa.
    5. Ange ett jobbnamn på molnsäkerhetskopian.
    6. Klicka på Nästa.
    7. Ange följande information från Amazon S3:
    • Region: Ange region för molnet genom att välja på rullgardinsmenyn.
    • Åtkomstnyckel: Ange den åtkomstnyckel som du har fått från Amazon S3.
    • Privat nyckel: Ange den privata nyckel som du har fått från Amazon S3.
    • Fjärrsökväg: Ange fjärrsökvägen för molnet. Detta när vanligen behållarnamnet.
    1. Klicka på Nästa.
    2. I fältet Typ väljer du något av följande alternativ:
    • Ladda upp: Säkerhetskopierar My Cloud Mirror-data till Amazon S3-behållaren.
    • Hämta: Säkerhetskopierar Amazon S3-behållaren till My Cloud Mirror-servern.
    1. I fältet Typ av säkerhetskopia väljer du den typ av säkerhetskopia du vill utföra på rullgardinsmenyn. Följande alternativ är tillgängliga:
    • Skriv över befintliga filer: Skriver över filer i målmappen som har samma namn som källfilen.
    • Fullständig säkerhetskopia: Skapar en separat mapp med alla data från säkerhetskopian varje gång säkerhetskopieringen utförs.
    • Stegvis säkerhetskopia: Skriver över filer med källfiler som är nyare än målfilerna.
    1. Klicka på Nästa.
    2. I fältet Lokal sökväg anger du en sökväg för säkerhetskopian på din My Cloud Mirror-enhet.
    • Klicka på Bläddra och sök efter en plats för säkerhetskopiering på enheten.
      (Exempelvolym_1/säkerhetskopia)
    1. Klicka på   för att aktivera Autouppdateringsfunktionen. Detta uppdaterar automatiskt din säkerhetskopia baserat på ett schema som du skapar.
    • Om du har aktiverat funktionen Autouppdatering:

      • Välj schemat Autouppdatering: Dag, vecka eller månad

      • Välj tid för autouppdatering på rullgardinsmenyn (alternativet Dag).

      • Välj datum och tid för autouppdatering på rullgardinsmenyerna (alternativet Vecka eller Månad).

    1. Klicka på Nästa.
    2. Granska inställningen och klicka på Slutför. Amazon S3 Cloud-säkerhetskopian skapas.

OBS! Få åtkomst till din bucket för Amazon S3 molnsäkerhetskopierings för att visa dina säkerhetskopior.
