Ändra en HTTP-hämtning

Använd följande steg när du vill ändra en schemalagd HTTP-hämtning.

    1. Klicka på Appar.
    2. Klicka på HTTP-hämtningar.
    3. I kön för HTTP-hämtningar väljer du den hämtning du vill ändra.
    4. Klicka på Ändra. Din schemalagda hämtning visas i avsnittet HTTP-hämtningar.
    5. Gör ändringarna och klicka sedan på Spara. Den uppdaterade schemalagda hämtningen visas i kön för HTTP-hämtningar.
