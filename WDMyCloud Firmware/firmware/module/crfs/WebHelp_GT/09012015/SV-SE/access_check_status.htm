Kontrollera status för fjärråtkomst

Du kan när som helst kontrollera:

    • My Cloud Mirror-enheten är konfigurerad för fjärråtkomst.
    • Fjärråtkomst till My Cloud Mirror-enheten aktiveras genom en direkt anslutning eller en reläanslutning
    • om en viss användares mobila enhet för närvarande kan få åtkomst till ditt moln på distans.

Kontrollera status för mobilåtkomst så här:

    1. Klicka på Molnåtkomst.
    2. Klicka på ett användarnamn i den vänstra rutan.
      Följande tabell visar fält för mobil åtkomst och deras möjliga värden:

Del

Profil för molnåtkomst

Anslutningsstatus

  • Ansluten (Direkt): Mobil åtkomst aktiverad via en direktanslutning.

  • Ansluten (Relä): Mobil åtkomst aktiverad via en reläanslutning.

  • Misslyckades: My Cloud Mirror kan inte upprätta en fjärranslutning.

Molnenheter för användarnamn

Enhets- eller appnamn

  • Ansluten: Enheten har fjärråtkomst till ditt moln.

  • Väntar: Enhetsaktivering avvaktas.

  • Utgången kod: Aktiveringskoden har gått ut.

  • Inaktiverat: Alternativet för molntjänst är inaktiverat. Se Slå på/av molnåtkomst.

  • Anslutningen misslyckades: Fjärranslutningen misslyckades. Du kan försöka åtgärda problemet genom att felsöka din nätverksanslutning och se till att den senaste versionen av den fasta programvaran är installerad.

 

OBS! Om du vill visa inställningar för mobil åtkomst för en annan användare klickar du på användarnamnet i den vänstra kolumnen. Vyn uppdateras med användarens molnenhetsinformation.
