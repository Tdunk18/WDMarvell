Blockera/avblockera åtkomst för en DLNA-enhet

Som administratör kan du blockera eller avblockera en DLNA-enhets åtkomst till ditt moln. Mer information om DLNA finns i Om My Cloud Mirror och DLNA.

Använd följande steg när du vill blockera/avblockera DLNA-enhetsåtkomst.

    1. Klicka på Inställningar.
    2. Klicka på Media i den vänstra rutan.
    3. Klicka på Visa mediespelare i fältet Medieströmning.
    4. Hitta de enheter vars åtkomst du vill ändra i dialogrutan Alternativ för medieströmning.
    5. Klicka för att ställa in lämpligt alternativ:
    • På: Åtkomst tillåts
    • Av: Åtkomst nekas
    1. Klicka på Stäng.
