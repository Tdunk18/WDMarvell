Aktivera en ElephantDrive-molnsäkerhetskopia

ElephantDrive är en tjänst för molnsäkerhetskopior som tillhandahåller säkra och automatiska säkerhetskopior via fjärrstyrning för dina filer. När ElephantDrive har aktiverats synkroniseras det automatiskt med din enhet.

Så här aktiverar du en molnsäkerhetskopia med ElephantDrive.

    1. Klicka på Säkerhetskopior.
    2. Klicka på Molnsäkerhetskopior.
    3. Klicka på knappen Elephant Drive längst upp på skärmen.
    4. Klicka på . ElephantDrive-molnsäkerhetskopian har aktiverats.
    5. Klicka på Registrera.
    6. Ange följande information och klicka sedan på Registrera:
    • E-postadress: Ange den e-postadress du vill ta emot information på från ElephantDrive.
    • Lösenord: Ange ett lösenord för det nya kontot.
    • Bekräfta lösenord: Upprepa lösenordet.

Du har nu aktiverat din ElephantDrive-molnsäkerhetskopia.
