Dela innehåll med andra utanför ditt nätverk

Använd programmet My Cloud som körs på din dator för att dela innehåll med andra utanför ditt nätverk

Använd följande steg när du vill dela innehåll:

    1. Hämta programmet My Cloud från WD Learning Center (Lärocenter) (http://www.wd.com/setup/wdmycloudmirror) och installera det på din Mac eller PC om du inte redan har gjort det.
    2. Starta programmet My Cloud på datorn.
    3. Dubbelklicka på My Cloud Mirror-enheten.
    4. Öppna mappar i den vänstra rutan för att visa det innehåll du vill dela.
    5. Klicka på ett objekt i den högra rutan (eller tryck på Skift och klicka för att markera flera objekt).
    6. Högerklicka på markeringen och välj sedan E-post.
    7. Fyll i nödvändiga uppgifter, skriv meddelandet och skicka det.
