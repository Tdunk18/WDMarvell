Överföra innehåll till My Cloud Mirror-enheten

För att överföra bilder, videoklipp, musik och filer till My Cloud Mirror-enheten använder du en av följande procedurer:

    • Mac
    • Windows PC
    • USB

Överföra innehåll från en Mac

Använd följande steg för att överföra filer från din Mac till din My Cloud Mirror-enhet:

    1. På din Mac väljer du Gå > Nätverk från menyfältet.
    2. Utför något av följande:
    • I det vänstra fönstret i Sökaren dubbelklickar du på My Cloud Mirror-enhetens namn.
    • I det högra fönstret lokaliserar du My Cloud Mirror-enhetens namn och dubbelklickar på det.
    1. Dubbelklicka på namnet för en viss resurs för att öppna mappen och visa alla befintliga undermappar. (Resursen Offentlig skapas automatiskt.)
    2. Dra och släpp filer från datorn till önskad mapp.
      Lägg inte filer i någon av mapparna Time Machine eller WD SmartWare Backups.
      OBS! Om du kopierar filer från en dator till My Cloud Mirror-enheten, kommer dessa filer att befinna sig på två platser: på den ursprungliga platsen och på enheten.

Överföra innehåll från en PC

Använd följande steg för att överföra filer från din Windows PC till din My Cloud Mirror-enhet:

    1. På datorn öppnar du dialogrutan Kör genom att trycka på tangenterna Windows och R samtidigt.
    2. I fältet Öppna anger du \\ följt av My Cloud Mirror-enhetens namn. (Det förvalda namnet är mycloudmirror, utan mellanslag.) Exempel: \\mycloudmirror
    3. Klicka på OK.
      Utforskaren öppnas och tillgängliga resurser visas i den högra rutan.
    4. Dubbelklicka på namnet för en viss resurs för att öppna mappen och visa alla undermappar. (Resursen Offentlig skapas automatiskt.)
    5. Dra och släpp filer från datorn till resursen.
      OBS! Om du kopierar filer från en dator till My Cloud Mirror-enheten, kommer dessa filer att befinna sig på två platser: på den ursprungliga platsen och på enheten.

     

    Överföra innehåll från en USB

    Så här överför du filer från en USB-enhet till My Cloud Mirror-enheten:

    1. Anslut USB-enheten till dator för bästa prestanda.

    2. Anslut datorn och My Cloud Mirror-enheten direkt till routern med en Ethernet-kabelanslutning.

    3. Kopiera data från din externa USB-enhet till My Cloud Mirror-enheten

OBS! Om du kopierar filer från en dator till My Cloud Mirror-enheten, kommer dessa filer att befinna sig på två platser: på den ursprungliga platsen och på enheten.
