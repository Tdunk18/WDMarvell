Ändra RAID-läge

Använd följande steg för att ändra det nuvarande RAID-läget på din My Cloud Mirror-enhet.

Varning: Ändringar av RAID-läget tar bort alla data.

    1. Klicka på Lagring.
    2. Klicka på RAID.
    3. Klicka på Ändra RAID-läge längst ned på skärmen.
    4. Välj det RAID-läge du vill använda för My Cloud Mirror-servern. Följande alternativ är tillgängliga:

Läge

Beskrivning

JBOD

En eller flera enheter utan RAID-konfiguration som hanteras som separata logiska volymer.

Disklänkande

Flera enheter kombineras linjärt för att bilda en stor logisk volym.

RAID 0

Data delas över flera hårddiskar vilket gör att data kan läsas och skrivas fortare då två eller fler enheter kan arbeta samtidigt för ökad prestanda.

RAID 1

Två omedelbara datakopior görs på separata hårddiskar. Skulle ett fel uppstå på den ena hårddisken kan den andra fortsätta skriva/läsa data tills den felande disken ersätts.

    1. Markera kryssrutan Växla till <RAID-läge>.
    2. Klicka på Nästa.
    3. Kontrollera statusen för dina hårddiskar och klicka sedan på Nästa.
    4. Klicka på Slutför.
    5. Med volymreglaget väljer du hur mycket utrymme du vill avsätta för det valda RAID-läget.

      OBS! Om du väljer att inte använda hela volymen för RAID-urvalet kan du konfigurera det återstående diskutrymmet som Disklänkning genom att markera kryssrutan Konfigurera återstående diskutrymme som Disklänkning.
    1. Klicka på Nästa.
    2. Markera  om du vill aktivera alternativet Automatisk återskapning för RAID-konfigurationen. Klicka sedan på Nästa.

      OBS! Den här skärmen visas inte för lägena JBOD, Disklänkande och RAID 0
    1. Granska ändringarna av RAID och kontrollera att de är korrekta. Klicka sedan på Nästa.
    2. Läs informationen på varningsskärmen.
      Varning: När du ändrar läge raderas alla data.
    1. Klicka på Slutför. De begärda hårddiskändringarna inleds.
    2. Klicka på Slutför. Hårddiskarna har nu konfigurerats om i önskat läge.
