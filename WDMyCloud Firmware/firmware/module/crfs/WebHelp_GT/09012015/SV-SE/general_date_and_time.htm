Ställa in datum och tid

Använd följande steg för att ställa in datum och tid för din My Cloud Mirror-enhet.

Inställning av tidszon

    1. Klicka på Inställningar.
    2. Klicka på Allmänt i den vänstra rutan.
    3. I fältet Tidszon väljer du en tidszon på rullgardinsmenyn.
    4. Klicka på Spara.

Synkronisera datum och tid till NTP

    1. Klicka på Inställningar.
    2. Klicka på Allmänt i den vänstra rutan.
    3. Klicka på för att slå på NTP-tjänsten.
    4. Bestäm vilka servrar som ska användas av NTP-tjänsten. Hur gör jag det?    

       

      Så här konfigurerar du NTP-tjänsten:

      1. Klicka på länken Konfigurera vid fältet Primär server.
      2. Klicka på Lägg till användarens NTP i dialogrutan NTP-inställningar.
      3. Ange serverinformationen och klicka sedan på Spara.
      4. Upprepa steg 1 och 2 vid behov. Du kan ange högst tre NTP-servrar.
