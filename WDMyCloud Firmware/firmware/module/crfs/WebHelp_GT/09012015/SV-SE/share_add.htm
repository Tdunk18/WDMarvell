Lägga till en resurs

Så här skapar du en resurs:

    1. Klicka på Resurser.
    2. Klicka på ikonen Lägg till resurs
    3. Välj den volym som du vill skapa resursen på i rullgardinsmenyn.
    4. Ange ett resursnamn.
    5. Ange en resursbeskrivning. Detta är ett valfritt fält.
    6. Klicka på för att aktivera Papperskorgen. I fältet visas På.
    • Om du inte vill aktivera papperskorgen kontrollerar du att fältet har värdet Av.
    1. Klicka på  för att aktivera media som betjänas. Detta gör det möjligt att strömma media som lagras på denna resurs till andra enheter. Detta är ett valfritt fält.
    • Om du inte vill aktivera media som betjänas kontrollerar du att fältet har värdet Av.
    1. Klicka på och aktivera FTP-åtkomst. Detta tillåter FTP-åtkomst på resursen. Detta är ett valfritt fält.
    • Om du inte vill aktivera FTP-åtkomst kontrollerar du att fältet har värdet Av.
    1. Klicka på  för att aktivera WebDAV-åtkomst. Detta tillåter WebDAV-åtkomst på resursen. Detta är ett valfritt fält.
    • Om du inte vill aktivera WebDAV-åtkomst kontrollerar du att fältet har värdet Av.
    1. Klicka på och aktivera NSF-åtkomst. Det ger NSF (Network File System) åtkomst på denna resurs. Detta är ett valfritt fält.
    • Om du inte vill aktivera NSF-åtkomst kontrollerar du att fältet har värdet Av.
    1. Den nya resursen läggs till och visas i den vänstra rutan.
