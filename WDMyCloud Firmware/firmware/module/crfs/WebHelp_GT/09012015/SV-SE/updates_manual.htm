Söka efter uppdateringar och installera manuellt

Om My Cloud Mirror-enheten inte har konfigurerats för att automatiskt uppdatera fast programvara (se Uppdatera My Cloud Mirror automatiskt), så kan du manuellt söka efter en uppdatering och installera den.

Så här söker du efter en uppdatering och installerar den:

    1. Klicka på Inställningar.
    2. Klicka på Uppdatering av fast programvara i den vänstra rutan.
    3. Klicka på Sök efter uppdatering i avsnittet Tillgängliga uppdateringar.
      När du har gjort det söker My Cloud Mirror-enheten efter tillgängliga uppdateringar. Om statusmeddelandet Inga nya uppdateringar visas är processen klar.
      Om en tillgänglig uppdatering listas går du vidare till steg 4.
    4. Klicka på Utgivningsmeddelanden för att visa information om en tillgänglig uppdatering i ett nytt webbläsarfönster. Stäng fönstret när du är klar med utgivningsmeddelandet.
    5. Klicka på Installera uppdatering.
    6. Klicka på OK för att bekräfta omstarten av My Cloud Mirror-enheten.
      OBS! Kom ihåg att tjänster för alla användare som för närvarande använder molnet kommer att avbrytas när WD My Cloud startas om.
