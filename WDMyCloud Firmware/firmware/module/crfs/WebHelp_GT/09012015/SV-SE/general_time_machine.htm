Säkerhetskopiera dina data med Apple Time Machine

Du kan säkerhetskopiera filerna på din My Cloud Mirror-enhet till Apple Time Machine på en Mac-dator.

Så här säkerhetskopierar du data till Apple Time Machine.

    1. Klicka på Inställningar.
    2. Klicka på Allmänt i den vänstra rutan.
    3. I avsnittet MAC säkerhetskopior klickar du på för att slå på alternativet Time Machine säkerhetskopia.
    4. Klicka på Konfigurera.
    5. Välj resursen där de säkerhetskopierade filerna ska lagras i rullgardinsmenyn Välj en resurs.
    6. Klicka på Spara.
