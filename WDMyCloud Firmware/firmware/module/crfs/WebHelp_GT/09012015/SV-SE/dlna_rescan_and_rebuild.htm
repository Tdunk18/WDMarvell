Söka igenom DLNA-databasen igen/bygga om den

Om du kommer åt mindre My Cloud Mirror-innehåll än du borde via en DLNA-enhet kan du göra något av följande med DLNA-serverns databas:

    • Skanna om: My Cloud Mirror-enheten söker efter ändringar som gjorts i dina mediebibliotek. Prova detta först.
    • Återskapa: My Cloud Mirror-enheten bygger om databasen från grunden.
      OBS! Var försiktig när du gör detta. Beroende på antalet filer och deras storlek kan det ta lång tid att återskapa DLNA-serverns databas.

Så här skannar du om eller återskapar DLNA-databasen:

    1. Klicka på Inställningar.
    2. Klicka på Media i den vänstra rutan.
    3. Klicka på Skanna om eller Återskapa vid alternativet DLNA-databas.
