Uppdatering av My Cloud Mirror-enheten från en fil

I vissa fall kan det krävas att du uppdaterar My Cloud Mirror-enheten genom att först hämta en fil från Western Digital supportwebbplats och sedan installera den.

OBS! Denna process kräver tillgång till internet.

Använd följande steg för att uppdatera My Cloud Mirror-enheten manuellt från en fil:

    1. Öppna en ny flik eller ett nytt fönster.
    2. Öppna Western Digitals tekniska supportwebbplats i din webbläsares nya flik eller nya fönster:
      http://support.wdc.com
    • Om en ny uppdatering av fast programvara är tillgänglig sparar du den på din enhet.
    1. Gå tillbaka till My Cloud Mirror webbgränssnitt.
    2. Klicka på Inställningar.
    3. Klicka på Uppdatering av fast programvara i den vänstra rutan.
    4. Klicka på Uppdatera från fil i delen Manuell uppdatering.
    5. Öppna platsen där du sparade uppdateringen i steg 3.
    6. Markera uppdateringsfilen och klicka sedan på Öppna.
    7. Klicka på OK för att bekräfta installationen. Efter installation av uppdateringen startar My Cloud Mirror-enheten om.
      OBS! När My Cloud Mirror startar om kommer tjänster för alla användare som för närvarande har åtkomst till molnet att avbrytas.
