Ta bort ett användarkonto

Som administratör kan du ta bort ett användarkonto från ditt My Cloud Mirror-system. Om du bara vill begränsa en användares åtkomst till kopplade specifika resurser eller enheter, se Ändra åtkomst till en resurs. Så här tar du bort ett användarkonto

    1. Klicka på Användare.
    2. I den vänstra rutan klickar du på namnet på den användare du vill ta bort.
    3. Klicka på ikonen Ta bort användare.
    4. Klicka på OK. Det markerade användarkontot raderas och tas bort från den vänstra rutan.
