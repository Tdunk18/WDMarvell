Återställa standardinställningar

Med systemåterställning kan administratören utföra en fullständig återställning till fabriksinställningarna på My Cloud Mirror-enheten. Det här alternativet återställer din enhets fabriksinställningar utan att du förlorar några data.

Innan du utför en återställning kan du spara och exportera din nuvarande konfiguration (Så här gör du) och sedan importera den senare (Så här gör du).

Viktigt: En återställning avbryter tjänster för användare som för närvarande använder molnet. Koppla inte ur My Cloud Mirror-enheten under återställningen.

Använd följande steg för att återställa My Cloud Mirror-enheten:

    1. Klicka på Inställningar.
    2. Klicka på Funktioner i den vänstra rutan.
    3. Klicka på Systemåterställning i avsnittet Systemåterställning.
    4. Läs konfigurationsmeddelandet och klicka på OK. Enheten startas om. Koppla inte ur enheten under omstartsprocessen. När omstarten har slutförts öppnar du instrumentpanelen.
