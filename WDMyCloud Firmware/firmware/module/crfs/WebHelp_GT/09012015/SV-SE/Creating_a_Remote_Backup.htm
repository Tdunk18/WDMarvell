Skapa en fjärrsäkerhetskopia

Med en fjärrsäkerhetskopia kan du säkerhetskopiera My Cloud Mirror -enheten till en annan server utanför nätverket.  Så här följer du stegen för att skapa en fjärrsäkerhetskopia för din My Cloud Mirror-server.

    1. Klicka på Säkerhetskopior.
    2. Klicka på Säkerhetskopia via fjärrstyrning.
    3. Klicka på knappen Skapa jobb.
    4. Ange ett jobbnamn.
    5. Välj en server från fjärrserverns rullgardinsmeny.
    6.  I fältet Fjärr-IP anger du IP-adressen för fjärrsäkerhetskopians enhet.
      (Exempel 192.168.1.16)
    7. I fältet Lösenord anger du lösenordet för fjärrsäkerhetskopians enhet.
    8. Ange ett SSH-användarnamn.
    9. I SSH-lösenordsfältet, ange SSH-lösenordet.
    10. Klicka på Bläddra för att källa källmapp.
    11. Från mapplistan över fjärrsäkerhetskopia, markera en eller flera mappar som ska skyddas.
    12. Klicka på OK.
    13. Klicka på Bläddra när du väljer en mapp för fjärrsäkerhetskopior.
    14. Välj en mapp och klicka på OK.
    15. Välj säkerhetskopieringstyp från rullgardinsmenyn.
    16. Om du vill ange ett schema för säkerhetskopiering, klicka på växlingsikonen bredvid Upprepningar .
    17. Klicka på växlingsikonen när du vill ställa in överföring av kryptering.
    18. Klicka på Skapa.
