Lägga till ett användarkonto

Följ stegen nedan för att lägga till en användare till din My Cloud Mirror-enhet:

    1. Klicka på Användare.
    2. Klicka på ikonen Lägg till användare:
    3. Ange ett Användarnamn. Namnet som används för att logga in på systemet.
    4. Ange ett förnamn. Detta är användarens förnamn.
    5. Ange ett efternamn. Detta är användarens efternamn.
    6. Ange en e-postadress. Detta är användarens e-postadress. Denna adress används också för att skapa ett My Cloud-konto.
    7. Ange ett lösenord för användarkontot.
    8. Ange lösenordet på nytt för att bekräfta det.
    9. Klicka på Apply (Verkställ).
    10. Klicka på . Detta ändrar lösenordsfältet till PÅ och aktiverar lösenordsfälten.
    • Om du inte vill ange ett lösenord kontrollerar du att lösenordsfältet är Av. Klicka sedan på Nästa.
