Willkommen

Ihr My Cloud Mirror-Gerätesystem ist ein leistungsfähiges System, mit dem Sie von einem beliebigen, an einen WLAN-Router angeschlossenen Computer, Tablet oder Smartphone aus sicher über eine drahtlose Verbindung auf Ihre Medien und Dateien zugreifen können. Das My Cloud Mirror-Gerät ist die zentrale Komponente Ihrer persönlichen Cloud und bietet eine einfache und stabile Oberfläche für problemlose Konfiguration, Freigabe und Wartung.

Was Sie wissen sollten:

Damit Sie Ihre persönliche Cloud optimal nutzen können, sollten Sie einige wichtige Dinge zu Ihrem My Cloud Mirror-Gerät wissen:

    • Benutzerrollen

Zusätzlich sollten Sie diese Aufgaben kennen, um die Kernfunktionen des Geräts nutzen zu können:

    • Zugriff mit Smartphone oder Tablet

    • Zugriff mit Laptop-Computer  

    • Anzeige der My Cloud Mirror-Weboberfläche

    • Freigabe von Inhalt für andere außerhalb Ihres Netzwerks

    • Hochladen von Dateien

    • Erste Schritte (Basiskonfiguration)

Benutzerrollen in Ihrer persönlichen Cloud

Alle, die auf Ihre persönliche Cloud zugreifen, müssen über ein Benutzerkonto für das My Cloud Mirror-Laufwerk verfügen. Als My Cloud Mirror-Eigentümer verfügen Sie über ein spezielles Benutzerkonto (admin), über das Sie Administratorrechte erhalten. Mit diesen Rechten können Sie das Gerät für Ihre speziellen Anforderungen einrichten und konfigurieren und andere Benutzer zu Ihrer persönlichen Cloud hinzufügen. Sie können genau festlegen, wer auf das Laufwerk zugreifen kann.

HINWEIS: Standardmäßig ist dem Benutzernamen admin kein Passwort zugeordnet. Sie können jederzeit ein Passwort hinzufügen.

Wichtig: Bei der ersten Einrichtung Ihres Geräts können Sie den Administratornamen ändern und ein optionales Passwort festlegen, während Sie die My Cloud Mirror-Einrichtungssoftware oder den Assistenten „Erste Schritte“ ausführen. Halten Sie den Namen und das Passwort fest. Ohne diese Anmeldedaten können Sie die My Cloud Mirror-Weboberfläche nicht anzeigen.

Zugreifen auf Inhalt von einem Smartphone oder Tablet aus

Zwei WD-Mobil-Apps ermöglichen Ihnen, von einem Smartphone oder Tablet auf die Inhalte Ihres My Cloud-Geräts zuzugreifen:

    • My Cloud-Alben: Hier können Sie Ihre Fotos anzeigen und verwalten.

    • My Cloud: Mit dieser App können Sie Fotos, Videos, Musik und Dateien anzeigen bzw. wiedergeben.

Sie erhalten diese kostenlosen Apps im WD-Schulungszentrum (http://www.wd.com/setup/wdmycloudmirror) und auf App-Websites wie Apple App Store, Google Play, Windows Store oder Amazon.com.

Workflow für den Mobilgerätezugriff für Sie (als Administrator)

Der Basis-Workflow für Sie als Eigentümer/Administrator von My Cloud Mirror:

1

Konfigurieren Sie das My Cloud Mirror-Gerät für den Remotezugriff, und registrieren Sie sich für My Cloud-Anmeldedaten. Vorgehensweise

2

Laden Sie die WD-Mobil-Apps herunter. Vorgehensweise

3

Starten Sie eine WD-App und führen Sie ein einmaliges Setup für den sicheren Zugriff durch. Vorgehensweise

4

Fertig! Geben Sie Ihren Inhalt wieder, und zeigen Sie ihn an.

Workflow für den Mobilgerätezugriff für andere

HINWEIS: Beim Hinzufügen eines Benutzers zu My Cloud Mirror geben Sie die E-Mail-Adresse des Benutzers an und es wird automatisch ein My Cloud-Anmeldename bei My Cloud erstellt. Nach dem Erstellen der Anmeldedaten wird eine E-Mail an die Adresse des neuen Benutzers gesendet. Die E-Mail enthält eine Verknüpfung zu My Cloud, über die der Benutzer den Registrierungsvorgang fortführen kann.

Basis-Workflow für Nicht-Administratoren:

1

Klicken Sie auf den Link in der Nachricht von My Cloud, und setzen Sie die Registrierung fort.

2

Laden Sie die WD-Mobil-Apps herunter. Vorgehensweise

3

Starten Sie eine WD-App und führen Sie ein einmaliges Setup für den sicheren Zugriff durch. Vorgehensweise

4

Geben Sie Ihren Inhalt wieder, und zeigen Sie ihn an.

Zugreifen auf Ihren Inhalt von einem Laptop aus

Mit der My Cloud-Desktop-Anwendung können Sie von Ihrem Mac oder PC auf die auf dem My Cloud Mirror-Gerät gespeicherten Inhalte zugreifen. Mit My Cloud können Sie remote auf Fotos, Videos, Musik und Dateien zugreifen, die Sie auf WD-NAS-Geräten gespeichert haben. Zu diesen Geräten zählen auch My Cloud Mirror- und „My Book Live“-Laufwerke, die mit Ihrem Heimnetzwerk verbunden sind. Wie bei der WD-Mobil-App benötigen Sie auch hier die überaus wichtigen My Cloud-Anmeldedaten für den Zugriff auf Ihren Inhalt.

Laden Sie die My Cloud-Desktop-Anwendung unter http://www.wd.com/setup/wdmycloudmirror herunter.

Schrittweise Anweisungen finden Sie unter Zugreifen auf Inhalt von Ihrem Laptop aus.

ANFANG

Anzeige der My Cloud Mirror-Oberfläche

Mit der Benutzeroberfläche des My Cloud Mirror-Geräts können Sie Ihre persönliche Cloud einrichten und verwalten. Die Oberfläche wird in einem auf einem Mac-Computer oder PC ausgeführten Webbrowser angezeigt. 

So lassen Sie die Weboberfläche anzeigen:

    1. Öffnen Sie ein Browserfenster.
    2. Geben Sie in die Adresszeile http:// ein;
      Beispiel: http://wdmycloudmirror
      HINWEIS: Wird mit der URL die Weboberfläche nicht angezeigt, geben Sie diese URL ein: http://wdmycloudmirror.local

Freigeben von Inhalt für andere außerhalb Ihres Netzwerks
