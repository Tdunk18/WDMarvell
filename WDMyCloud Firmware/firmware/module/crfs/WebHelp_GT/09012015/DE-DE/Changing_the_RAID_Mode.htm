Ändern des RAID-Modus

Führen Sie die nachfolgend genannten Schritte aus, um den aktuellen RAID-Modus des My Cloud Mirror-Geräts zu ändern.

Warnung: Bei einer Änderung des RAID-Modus werden alle auf dem Gerät gespeicherten Daten gelöscht.

    1. Klicken Sie auf Speicher.
    2. Klicken Sie auf RAID.
    3. Klicken Sie unten am Bildschirm auf RAID-Modus ändern.
    4. Wählen Sie den RAID-Modus aus, den Sie für den My Cloud Mirror-Server verwenden möchten. Folgende Optionen stehen zur Verfügung:

Modus

Beschreibung

JBOD

Hierbei werden ein oder mehrere Laufwerke ohne RAID-Konfiguration verwendet, die als getrennte logische Volumes verwaltet werden.

Aufteilung

Lineare Kombination aus Laufwerken zum Erstellen eines großen logischen Laufwerks.

RAID 0

Die Daten werden über mehrere Festplatten verteilt. Durch die Verwendung mehrerer Laufwerke und die damit verbundene Leistungssteigerung wird die Lese- und Aufzeichnungsgeschwindigkeit erhöht.

RAID 1

Zwei Datenkopien werden gleichzeitig auf getrennten Festplatten aufgezeichnet. Wenn eine der Festplatten ausfällt, können die Daten weiterhin auf den anderen Festplatten gelesen und geschrieben werden, bis das fehlerhafte Laufwerk ersetzt wurde.

    1. Aktivieren Sie das Kontrollkästchen „Zu <RAID-Modus> wechseln“.
    2. Klicken Sie auf Weiter.
    3. Überprüfen Sie den Status der Festplatten und klicken Sie auf Weiter.
    4. Klicken Sie auf „Fertig“.
    5. Wählen Sie unter Verwendung des Volume-Schiebereglers den Speicherplatz aus, der dem ausgewählten RAID-Modus zugewiesen werden soll.

      HINWEIS: Wenn Sie nicht das gesamte Volume für die RAID-Auswahl verwenden, können Sie den verbleibenden Speicherplatz durch Aktivieren des Kontrollkästchens Verbleibenden Speicherplatz als Aufteilung konfigurieren als Aufteilung konfigurieren.
    1. Klicken Sie auf Weiter.
    2. Wählen Sie aus, um die Option der automatischen Wiederherstellung für die RAID-Konfiguration zu aktivieren. Klicken Sie dann auf Weiter.

      HINWEIS: Dieser Bildschirm wird bei den Modi JBOD, Aufteilung und RAID 0 nicht angezeigt.
    1. Überprüfen Sie die Änderungen des RAID-Modus und klicken Sie auf Weiter.
    2. Überprüfen Sie den Warnbildschirm.
      Warnung: Wenn Sie den Modus ändern, werden alle Daten gelöscht.
    1. Klicken Sie auf „Fertig“. Die angeforderten Änderungen werden an der Festplatte vorgenommen.
    2. Klicken Sie auf „Fertig“. Die Festplatten sind nun mit dem ausgewählten Modus neu konfiguriert.
