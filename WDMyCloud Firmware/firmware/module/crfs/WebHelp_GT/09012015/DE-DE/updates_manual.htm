Aktualisierungen manuell suchen und installieren

Ist das My Cloud Mirror-Gerät nicht zur automatischen Aktualisierung der Firmware eingerichtet (siehe Automatisches Aktualisieren von My Cloud Mirror), können Sie manuell eine Aktualisierung suchen und diese installieren.

Gehen Sie folgendermaßen vor, um nach einer Aktualisierung zu suchen und sie zu installieren:

    1. Klicken Sie auf Einstellungen.
    2. Klicken Sie im linken Fenster auf Firmware-Aktualisierung.
    3. Klicken Sie im Bereich „Verfügbare Aktualisierungen“ auf Nach Aktualisierung suchen.
      Daraufhin sucht das My Cloud Mirror-Gerät nach verfügbaren Aktualisierungen. Lautet die angezeigte Statusmeldung „Keine neuen Aktualisierungen“, haben Sie diesen Prozess abgeschlossen.
      Gibt der angezeigte Status eine verfügbare Aktualisierung an, fahren Sie mit Schritt 4 fort.
    4. Klicken Sie auf „Versionshinweise“, um Einzelheiten zu einer verfügbaren Aktualisierung in einem neuen Browserfenster anzeigen zu lassen. Nachdem Sie die Versionshinweise gelesen haben, schließen Sie das dazugehörige Browserfenster.
    5. Klicken Sie auf Aktualisierung installieren.
    6. Klicken Sie auf OK, um den Neustart des My Cloud Mirror-Geräts zu bestätigen.
      HINWEIS: Bedenken Sie dabei, dass der Dienst für alle Benutzer, die aktuell auf die Cloud zugreifen, unterbrochen wird.
