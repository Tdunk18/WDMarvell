Info zur Verwaltung Ihrer persönlichen Cloud

Die Verwaltung Ihrer persönlichen Cloud umfasst zahlreiche Aufgaben, von denen die meisten nur gelegentlich anfallen. In den meisten Fällen müssen Sie nur wenige Aufgaben zur Cloud-Verwaltung ausführen, nachdem Sie die unter Erste Schritte aufgeführten Aufgaben einmal durchgeführt haben.

Unter diesem Thema werden die Aufgaben zur Cloud-Verwaltung nach Kategorie aufgeführt:

    • Benutzer
    • Freigaben
    • Cloud-Zugriff

Benutzer

Wählen Sie einen der Links unten aus, um Anweisungen für Benutzeraufgaben anzuzeigen.

    • Benutzerkonto hinzufügen. 
    • Benutzerpasswort hinzufügen oder ändern  
    • Benutzerinformationen anzeigen.  
    • Benutzerzugriff auf eine Freigabe ändern  
    • Benutzerkonto entfernen.

Freigaben

Eine Freigabe ist ein Bereich zum Speichern von Dateien, vergleichbar mit einem Ordner. Wählen Sie einen der Links unten aus, um Anweisungen zur Verwaltung von Freigaben anzuzeigen.

    • Freigabe erstellen  
    • Freigabe löschen
    • Benutzerzugriff auf eine Freigabe ändern  

 ANFANG

Cloud-Zugriff

Bevor Sie mit einem Smartphone oder Tablet auf die Inhalte Ihres Geräts zugreifen können, muss das entsprechende Mobilgerät als Bestandteil Ihrer persönlichen Cloud-Umgebung registriert werden. Hierbei wird ein Code generiert, mit dem das Mobilgerät mit dem My Cloud Mirror-Gerät gekoppelt wird. Wählen Sie einen der Links unten aus, um Anweisungen zur Verwaltung des Cloud-Zugriffs anzuzeigen.

    • Cloud-Zugriff einrichten   
    • Einmaliges Setup für sicheren Zugriff von einem Mobilgerät ausführen  
    • Gerät aus Ihrer persönlichen Cloud entfernen  
    • Remotezugriffstatus prüfen  
    • Leistung durch Portweiterleitung steigern
