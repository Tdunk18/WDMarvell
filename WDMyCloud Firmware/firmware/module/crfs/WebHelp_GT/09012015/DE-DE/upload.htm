Hochladen von Inhalten auf das My Cloud Mirror-Gerät

Zum Hochladen von Fotos, Videos, Musik und Dateien auf Ihr My Cloud Mirror-Gerät verwenden Sie eine der folgenden Vorgehensweisen:

    • Mac
    • Windows PC
    • USB

Hochladen von Inhalt von einem Mac-Computer

So laden Sie Dateien von Ihrem Mac-Computer auf Ihr My Cloud Mirror-Gerät hoch:

    1. Wählen Sie auf Ihrem Mac in der Menüleiste Go > Netzwerk.
    2. Führen Sie einen der folgenden Schritte aus:
    • Doppelklicken Sie im linken Fenster des Finders auf den Namen des My Cloud Mirror-Geräts.
    • Suchen Sie im rechten Fenster den Namen des My Cloud Mirror-Geräts und doppelklicken Sie darauf.
    1. Doppelklicken Sie auf den Namen einer bestimmten Freigabe, um den Ordner zu öffnen und alle vorhandenen Unterordner anzuzeigen. (Öffentlich ist eine automatisch erstellte Freigabe.)
    2. Ziehen Sie Dateien aus Ihrem Computer in den gewünschten Ordner, und legen Sie sie ab.
      Legen Sie keine Dateien in den Time Machine- und WD SmartWare Backups-Ordnern ab.
      HINWEIS: Wenn Sie Dateien vom Computer auf das My Cloud Mirror-Gerät kopieren, befinden sich diese Dateien an zwei Speicherorten: am ursprünglichen Speicherort und auf Ihrem Gerät.

Hochladen von Inhalt von einem PC

So laden Sie Dateien von Ihrem Windows-PC auf das My Cloud Mirror-Gerät hoch:

    1. Öffnen Sie auf Ihrem Computer das Dialogfeld „Ausführen“, indem Sie gleichzeitig die Windows- und die R-Taste drücken.
    2. Geben Sie in das Feld „Öffnen“ \\ gefolgt vom Namen des My Cloud Mirror-Geräts ein. (Der Standardname lautet „mycloudmirror“ ohne Leerzeichen.) Beispiel: \\mycloudmirror
    3. Klicken Sie auf OK.
      Windows Explorer oder Datei-Explorer wird aufgerufen. Die verfügbaren Freigaben werden im rechten Fenster aufgeführt.
    4. Doppelklicken Sie auf den Namen einer bestimmten Freigabe, um den Ordner zu öffnen und alle Unterordner anzuzeigen. (Öffentlich ist eine automatisch erstellte Freigabe.)
    5. Ziehen Sie Dateien aus Ihrem Computer in diese Freigabe, und legen Sie sie ab.
      HINWEIS: Wenn Sie Dateien vom Computer auf das My Cloud Mirror-Gerät kopieren, befinden sich diese Dateien an zwei Speicherorten: am ursprünglichen Speicherort und auf Ihrem Gerät.

     

    Hochladen von Inhalt von einem USB-Gerät

    So laden Sie Dateien von einem USB-Gerät zum My Cloud Mirror-Gerät hoch:

    1. Schließen Sie das USB-Gerät an den Computer an, um die Leistung zu optimieren.

    2. Schließen Sie den Computer und das My Cloud Mirror-Gerät über eine kabelgebundene Ethernet-Verbindung an einen Router an.

    3. Kopieren Sie die Daten vom externen USB-Laufwerk auf das My Cloud Mirror-Gerät.

HINWEIS: Wenn Sie Dateien vom Computer auf das My Cloud Mirror-Gerät kopieren, befinden sich diese Dateien an zwei Speicherorten: am ursprünglichen Speicherort und auf Ihrem Gerät.
