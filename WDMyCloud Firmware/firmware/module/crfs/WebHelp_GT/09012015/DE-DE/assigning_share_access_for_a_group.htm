Zuweisen von Freigabezugriffen für eine Gruppe

Nachdem Sie eine Gruppe erstellt haben, können Sie die Zugriffsebene für die Mitglieder der Gruppe festlegen. Gehen Sie folgendermaßen vor, um einer Gruppe Freigabezugriff zuzuweisen:

    1. Klicken Sie auf „Benutzer“.
    2. Klicken Sie auf Gruppen.
    3. Wählen Sie in der Spalte links die Gruppe aus, der Sie Zugriffsrechte zuweisen möchten. Im Gruppenprofilbereich werden die Informationen zur ausgewählten Gruppe angezeigt.
    4. Wählen Sie im Bereich „Freigabezugriff“ für jede Freigabe auf Ihrem Gerät die Zugriffsebene aus.
    • Lese-/Schreibzugriff. Wählen Sie diese Option aus, um vollständigen Zugriff auf die ausgewählten Informationen zu gewähren.
    • Nur Lesezugriff. Mit dieser Zugriffsart kann ein Gruppenmitglied die ausgewählten Informationen lesen, jedoch nicht ändern.
    • Zugriff verweigern. Das Gruppenmitglied hat keinen Zugriff auf die Informationen.

Die Freigaben werden der ausgewählten Gruppe zugewiesen.
