Was sind Benutzergruppen?

Mithilfe von Benutzergruppen können Sie mehrere Ressourcen für eine Gruppe von Benutzern verwalten. Die Berechtigungen und Privilegien, die Sie einem Gruppenkonto zuweisen, legen fest, welche Aktionen die Mitglieder der Gruppe ausführen können.
