Durchführen eines einmaligen Setups für Secure Access

Mit dieser Vorgehensweise können Sie eine sichere Verbindung zwischen Ihrem Smartphone oder Tablet und Ihrem My Cloud Mirror-Gerät herstellen. Gehen Sie nach den Anweisungen für Ihr Mobilgerät vor:

    • Smartphone oder Tablet mit Android  

    • iPhone oder iPad

    • Windows Phone oder Tablet

    • Windows- oder Mac-Desktop-PC

Einrichten von Secure Access für ein Android-Smartphone oder Tablet

    1. Stellen Sie mit den folgenden Schritten eine Verbindung zwischen Ihrem Smartphone oder Tablet her:

      1. Tippen Sie an Ihrem Mobilgerät auf „Einstellungen“.

      2. Tippen Sie auf „WLAN und Netzwerke“.

      3. Tippen Sie auf „WLAN-Einstellungen“.

      4. Wählen Sie Ihr Heimnetzwerk aus.

      5. Geben Sie Ihr Netzwerkpasswort ein, und tippen Sie auf Verbinden.

    1. Rufen Sie eine WD-App auf.
    2. Tippen Sie auf „Jetzt verbinden“.
    3. Tippen Sie auf den Namen des My Cloud Mirror-Geräts.
    4. Tippen Sie auf den Dropdownpfeil, und wählen Sie Ihren Benutzernamen aus.
    5. Führen Sie einen der folgenden Schritte aus:

    • Wurde ein Passwort für diesen Benutzer eingerichtet, geben Sie es ein, und tippen Sie dann auf „Anmelden“.

    • Wurde kein Passwort für den Benutzer eingerichtet, tippen Sie auf Anmelden.

    1. (Optional) Möchten Sie auf Ihren Inhalt zugreifen, tippen Sie auf einen Freigabenamen, und navigieren Sie dann zur gewünschten Datei.

Einrichten von Secure Access für ein iPhone oder iPad

    1. Stellen Sie mit den folgenden Schritten eine Verbindung zwischen Ihrem Smartphone oder Tablet her:

    1. Tippen Sie an Ihrem Mobilgerät auf „Einstellungen“.

    2. Tippen Sie auf „WLAN“.

    3. Tippen Sie auf „WLAN-Einstellungen“.

    4. Wählen Sie Ihr Heimnetzwerk aus.

    5. Geben Sie Ihr Netzwerkpasswort ein, und tippen Sie auf Verbinden.

    1. Rufen Sie eine WD-App auf.
    2. Tippen Sie auf „Jetzt verbinden“.
    3. Tippen Sie auf das Pluszeichen (+) neben dem Namen des My Cloud Mirror-Geräts.
    4. Tippen Sie auf den gewünschten Benutzernamen.
      Wurde ein Passwort für den Benutzer eingerichtet, geben Sie es ein.
    5. (Optional) Möchten Sie auf Ihren Inhalt zugreifen, tippen Sie auf einen Freigabenamen, und navigieren Sie dann zur gewünschten Datei.

Einrichten von Secure Access für ein Windows-Smartphone oder Tablet

    1. Stellen Sie mit den folgenden Schritten eine Verbindung zwischen Ihrem Smartphone oder Tablet her:

    1. Tippen Sie an Ihrem Gerät auf Einstellungen.

    1. Tippen Sie auf „WLAN“.

    2. Wählen Sie Ihr Heimnetzwerk aus.

    3. Geben Sie Ihr Netzwerkpasswort ein, und tippen Sie auf Fertig.

    1. Rufen Sie eine WD-App auf.
    2. Tippen Sie auf „Jetzt verbinden“.
    3. Tippen Sie auf den Namen des My Cloud Mirror-Laufwerks.
    4. Tippen Sie auf den Dropdownpfeil, und wählen Sie Ihren Benutzernamen aus.
      Wurde ein Passwort für den Benutzer eingerichtet, geben Sie es ein.
    5. (Optional) Möchten Sie auf Ihren Inhalt zugreifen, gehen Sie folgendermaßen vor:

    1. Tippen Sie auf dem Bildschirm „Geräte“ auf den Namen Ihres My Cloud Mirror-Laufwerks.

    2. Tippen Sie auf eine Freigabe und navigieren Sie dann zu dem gesuchten Inhalt.
