Durchführen von Diagnosetests

Sie können die Integrität des Geräts mit einem der folgenden Tests prüfen:

    1. Schnell: Ihr Gerät wird auf größere Leistungsprobleme geprüft. Als Ergebnis wird die Bewertung „Bestanden“ oder „Nicht bestanden“ ausgegeben. Die Durchführung eines Schnelltests kann einige Minuten in Anspruch nehmen.

    • Vollständig: Es werden alle Sektoren des Geräts geprüft und Sie erhalten Feedback zum Zustand des My Cloud Mirror-Geräts.  Ein vollständiger Test kann je nach vorhandener Datenmenge mehrere Stunden in Anspruch nehmen.

Gehen Sie folgendermaßen vor, um einen Diagnosetest auszuführen:

    1. Klicken Sie auf Einstellungen.
    2. Klicken Sie im linken Fenster auf Hilfsprogramme.
    3. Klicken Sie im Bereich „Systemdiagnose“ auf Schnelltest oder Vollständiger Test.
      Eine Statusanzeige gibt den Status des Tests an, und die Ergebnisse werden in einem Dialogfeld angezeigt.
