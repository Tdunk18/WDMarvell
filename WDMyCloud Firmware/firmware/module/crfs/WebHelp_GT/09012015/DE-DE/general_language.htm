Festlegen der angezeigten Sprache

Sie können die in der Weboberfläche von My Cloud Mirror angezeigte Sprache wählen.

HINWEIS: Auf Ihrem Gerät müssen die richtigen Schriften installiert und die richtigen Landeseinstellungen konfiguriert sein, damit die Oberfläche einwandfrei angezeigt wird.

Führen Sie die nachfolgend genannten Schritte aus, um die angezeigte Sprache des My Cloud Mirror-Geräts festzulegen.

    1. Klicken Sie auf Einstellungen.
    2. Klicken Sie im linken Fenster auf Allgemein.
    3. Wählen Sie im Feld „Sprache“ eine Sprache aus dem Dropdownmenü aus.
    4. Klicken Sie auf Speichern.
