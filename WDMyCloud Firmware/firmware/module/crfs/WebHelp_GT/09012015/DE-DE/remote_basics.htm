Grundlagen zum Remotezugriff

Ihnen stehen drei Methoden zur Verfügung, um von einem beliebigen Standort aus auf die Inhalte auf My Cloud Mirror zuzugreifen::

    • Von einem Mobilgerät aus, zum Beispiel einem Smartphone oder einem Tablet.  
    • Von Ihrem Computer aus (üblicherweise von einem Laptop).  

Überblick über den Zugriff von einem Mobilgerät aus

Führen Sie folgende Schritte aus, um von einem Smartphone oder Tablet auf Ihre Cloud zuzugreifen:

1

Besorgen Sie sich einen My Cloud-Anmeldenamen. Dies wird üblicherweise beim Hinzufügen eines Benutzers zum Gerät ausgeführt. Siehe Einrichten des Cloud-Zugriffs.

2

Laden Sie die kostenlosen Apps WD Photos und My Cloud auf Ihr Mobilgerät herunter, und installieren Sie sie.
Diese Apps sind im Apple App Store und bei Google Play erhältlich.

3

Führen Sie ein einmaliges Setup für Secure Access durch. Siehe Durchführen eines einmaligen Setups für Secure Access.

4

Greifen Sie mit Ihrem Mobilgerät auf Ihren Inhalt zu. Siehe Zugreifen auf Inhalt von einem Smartphone oder Tablet aus.

Übersicht über den Zugriff über einen Windows- oder Mac-Desktop-PC

Falls Sie sich außer Haus aufhalten (z. B. in einem Café), können Sie von Ihrem Laptop aus problemlos auf Ihr Gerät zugreifen. Für den Remotezugriff sind die My Cloud-Desktopanwendung, ein Laptop mit aktivierter WLAN-Funktion und eine WLAN-Verbindung erforderlich.

Führen Sie folgende Schritte aus, um über die My Cloud-Desktopanwendung auf Ihre Cloud zuzugreifen:

1

Besorgen Sie sich einen My Cloud-Anmeldenamen. Dies wird üblicherweise beim Hinzufügen eines Benutzers zum Gerät ausgeführt. Siehe Einrichten des Cloud-Zugriffs.

2

Laden Sie die Desktop-Anwendung My Cloud aus dem WD-Schulungszentrum http://wd.com/setup/wdmycloudmirror herunter, und installieren Sie diese auf Ihrem Computer.

3

Führen Sie ein einmaliges Setup für Secure Access durch. Siehe Durchführen eines einmaligen Setups für Secure Access.

4

Zugreifen auf Ihren Inhalt von einem Laptop aus Siehe Zugreifen auf Inhalt von Ihrem Laptop aus.
