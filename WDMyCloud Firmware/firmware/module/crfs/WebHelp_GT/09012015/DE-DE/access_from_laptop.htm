Zugreifen auf Inhalt von Ihrem Laptop aus

Die Desktop-Anwendung „My Cloud“ bietet Funktionen für den Zugriff auf Inhalt Ihres My Cloud Mirror-Geräts über das Laptop. Für diese Methode sind eine Internetverbindung und ein My Cloud-Anmeldename erforderlich (siehe Einrichten von Remotezugriff).

Führen Sie folgende Schritte aus, um von einem Laptop aus auf Inhalte zuzugreifen:

    1. Laden Sie die Desktopanwendung „My Cloud“ aus dem WD-Schulungszentrum (http:/www./wd.com/setup/wdmycloudmirror) herunter und installieren Sie diese auf Ihrem Mac oder PC.
    2. Erstellen Sie eine Verbindung zum Netzwerk (kabelgebunden oder drahtlos).
    3. Rufen Sie auf Ihrem Computer die My Cloud-Desktopanwendung auf.
    4. Geben Sie Ihre Anmeldedaten ein.
    5. Doppelklicken Sie auf das My Cloud Mirror-Gerät.
    6. Erweitern Sie im linken Fenster die entsprechenden Ordner, um den freizugebenden Inhalt anzuzeigen.
    7. Doppelklicken Sie im rechten Bereich auf ein Element, um es anzuzeigen oder wiederzugeben.
      Wichtig: Die Unterstützung der Wiedergabe oder Anzeige von Dateien richtet sich nach der Konfiguration Ihres Computers und Browsers und danach, ob die notwendigen Medienplayer installiert sind.
