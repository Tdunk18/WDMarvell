Entfernen eines Geräts

Gehen Sie folgendermaßen vor, um ein Gerät aus der Cloud zu entfernen:

    1. Klicken Sie auf Cloud-Zugriff.
    2. Klicken Sie im linken Fenster auf den Benutzer, dessen Gerät Sie entfernen möchten.
    3. Suchen Sie im Abschnitt „Cloud-Geräte“ das Gerät.
      Die Statusinformationen geben eine Verbindung (Verbunden) für WD Photos und/oder My Cloud an.
    4. Führen Sie einen der folgenden Schritte aus:
    • Klicken Sie für die entsprechende WD Photos-Zeile auf das Symbol für Löschen , um den Zugriff eines Geräts auf die Cloud über die mobile App WD Photos zu entfernen.
    • Klicken Sie für die entsprechende My Cloud-Zeile auf das Symbol für Löschen , um den Zugriff eines Geräts auf die Cloud über die My Cloud-App zu entfernen.
    • Klicken Sie für die Zeile des Mobilgeräts auf das Symbol für Löschen , um das Gerät aus Ihrer My Cloud Mirror-Umgebung zu entfernen.
    1. Klicken Sie im Bestätigungsfeld auf Ja.
