Sperren/Entsperren des Zugriffs für ein DLNA-Gerät

Sie können als Administrator den Zugriff eines DLNA-Geräts auf Ihre Cloud sperren beziehungsweise entsperren. Weitere Einzelheiten zu DLNA finden Sie unter Info zu My Cloud Mirror und DLNA.

Führen Sie die folgenden Schritte aus, um den Zugriff auf ein DLNA-Gerät zu sperren/entsperren:

    1. Klicken Sie auf Einstellungen.
    2. Klicken Sie im linken Fenster auf Medien.
    3. Klicken Sie im Feld „Medienstreaming“ auf Medienplayer anzeigen.
    4. Suchen Sie im Dialogfeld „Optionen für Medienstreaming“ das bzw. die Geräte, dessen Zugriff geändert werden soll.
    5. Klicken Sie entsprechend auf den Umschalter:
    • Ein: Zugriff gestatten
    • Aus: Zugriff verweigern
    1. Klicken Sie auf Schließen.
