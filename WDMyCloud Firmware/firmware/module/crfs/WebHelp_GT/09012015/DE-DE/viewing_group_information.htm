Anzeigen von Gruppeninformationen

Gehen Sie folgendermaßen vor, um die Details einer Gruppe anzuzeigen:

    1. Klicken Sie auf Benutzer.
    2. Klicken Sie auf das Symbol Gruppen.
    3. Wählen Sie in der Spalte links die Gruppe aus, die Sie anzeigen möchten. Im Gruppenprofilbereich werden die Informationen zur ausgewählten Gruppe angezeigt.
