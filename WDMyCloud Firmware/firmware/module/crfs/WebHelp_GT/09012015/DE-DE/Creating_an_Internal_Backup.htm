Erstellen einer internen Sicherung

Mit einer internen Sicherung können Sie vorhandene Inhalte an einem anderen Speicherort des My Cloud Mirror-Geräts sichern.

Gehen Sie folgendermaßen vor, um eine interne Sicherung zu erstellen:

    1. Klicken Sie auf Sicherungen.
    2. Klicken Sie auf Interne Sicherungen.
    3. Klicken Sie auf Auftrag erstellen.
    4. Geben Sie im Feld „Auftragsname“ einen Namen für den Sicherungsauftrag ein.
    5. Klicken Sie im Feld „Quellordner“ auf Durchsuchen.
    6. Wählen Sie die zu sichernden Daten aus und klicken Sie auf OK.
    7. Klicken Sie im Feld „Zielordner“ auf Durchsuchen.
    8. Wählen Sie den Zielort für die Sicherung der Daten aus und klicken Sie auf OK.
    9. Wählen Sie im Feld „Wiederholte Sicherung“ aus, um diese Funktion einzuschalten.Geben Sie dann folgende Informationen ein:
    • Wählen Sie die Sicherungsfrequenz aus: täglich, wöchentlich, monatlich
    • Wählen Sie aus dem Pulldownmenü eine Uhrzeit (Stunden und Minuten) aus. 
    1. Wenn Sie das Feld „Wiederholte Sicherung“ nicht aktiviert haben, geben Sie folgende Informationen ein:
    • Klicken Sie auf das Datumsfeld und wählen Sie das Datum aus, an dem die Sicherung erstellt werden soll.
    • Wählen Sie aus dem Pulldownmenü eine Uhrzeit (Stunden und Minuten) aus.
    1. Aktivieren Sie das Kontrollkästchen „Inkrementelle Sicherung“, wenn Sie eine inkrementelle Sicherung erstellen möchten.
    2. Klicken Sie auf Erstellen. Der Auftrag wird im Bereich der Warteschlange der internen Sicherungen angezeigt und zur festgelegten Zeit ausgeführt.
