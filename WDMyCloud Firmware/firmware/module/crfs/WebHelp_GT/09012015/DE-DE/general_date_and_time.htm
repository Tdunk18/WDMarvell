Einstellen von Datum und Uhrzeit

Führen Sie die nachfolgend genannten Schritte aus, um das Datum und die Uhrzeit auf dem My Cloud Mirror-Gerät einzustellen.

Festlegen der Zeitzone

    1. Klicken Sie auf Einstellungen.
    2. Klicken Sie im linken Fenster auf Allgemein.
    3. Wählen Sie im Feld „Zeitzone“ eine Zeitzone aus dem Dropdownmenü aus.
    4. Klicken Sie auf Speichern.

Synchronisieren von Datum und Uhrzeit mit NTP

    1. Klicken Sie auf Einstellungen.
    2. Klicken Sie im linken Fenster auf Allgemein.
    3. Klicken Sie auf , um den NTP-Dienst einzuschalten.
    4. Legen Sie fest, welche Server vom NTP-Dienst verwendet werden sollen. Wie gehe ich dazu vor?    

       

      So konfigurieren Sie den NTP-Dienst:

      1. Klicken Sie auf „Konfigurieren“ neben dem Feld „Primärer Server“.
      2. Klicken Sie im Dialogfeld „NTP-Einstellungen“ auf Benutzer-NTP hinzufügen.
      3. Geben Sie die Serverinformationen ein, und klicken Sie dann auf Speichern.
      4. Wiederholen Sie gegebenenfalls die Schritte 1 und 2. Sie können maximal drei NTP-Server angeben.
