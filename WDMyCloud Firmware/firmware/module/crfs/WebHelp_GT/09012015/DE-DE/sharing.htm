Freigeben von Inhalt für andere außerhalb Ihres Netzwerks

Mit der Desktopanwendung „My Cloud“ auf Ihrem Computer können Sie Inhalt für andere außerhalb Ihres Netzwerks freigeben.

Gehen Sie folgendermaßen vor, um Inhalte freizugeben:

    1. Laden Sie gegebenenfalls die Desktop-Anwendung „My Cloud“ aus dem WD-Schulungszentrum (http://www.wd.com/setup/wdmycloudmirror) herunter und installieren Sie diese auf Ihrem Mac oder PC.
    2. Rufen Sie auf Ihrem Computer die My Cloud-Desktopanwendung auf.
    3. Doppelklicken Sie auf das My Cloud Mirror-Gerät.
    4. Erweitern Sie im linken Fenster die entsprechenden Ordner, um den freizugebenden Inhalt anzuzeigen.
    5. Klicken Sie im rechten Fenster auf ein Element (oder halten Sie beim Klicken die Umschalttaste gedrückt, um mehrere Elemente auszuwählen).
    6. Klicken Sie mit der rechten Maustaste auf Ihre Auswahl, und wählen Sie dann „E-Mail“.
    7. Füllen Sie das E-Mail-Fenster wie gewünscht aus und senden Sie die Nachricht.
