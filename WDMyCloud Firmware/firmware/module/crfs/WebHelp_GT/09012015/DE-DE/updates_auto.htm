Automatisches Aktualisieren des My Cloud Mirror-Geräts

Sie können das My Cloud Mirror-Gerät so einrichten, dass bequem nach verfügbaren Firmware-Aktualisierungen gesucht wird und diese automatisch installiert werden. Das ist die einfachste Möglichkeit um sicherzustellen, dass die My Cloud Mirror-Firmware aktuell ist.

HINWEIS: Nach der automatischen Installation einer Softwareaktualisierung wird das My Cloud Mirror-Gerät neu gestartet. Da sich ein Neustart auf die Verfügbarkeit der Cloud für die Benutzer auswirkt, sollten Sie die Aktualisierungen für bestimmte Zeiten planen, wenn möglichst wenige Benutzer auf die Cloud zugreifen.

Führen Sie die nachfolgend genannten Schritte aus, um dasMy Cloud Mirror-Gerät automatisch zu aktualisieren:

    1. Klicken Sie auf Einstellungen.
    2. Klicken Sie im linken Fenster auf Firmware-Aktualisierung.
    3. Klicken Sie im Bereich „automatische Aktualisierung“ auf , um die automatischen Aktualisierungen zu aktivieren.
    4. Legen Sie im Feld „Aktualisierungszeitplan“ den Tag und die Uhrzeit für die Aktualisierung fest:

    • Klicken Sie auf den ersten Dropdownpfeil und wählen Sie den Wochentag aus.

    • Klicken Sie auf den zweiten Dropdownpfeil und wählen Sie eine Uhrzeit aus.

    • Klicken Sie auf den dritten Dropdownpfeil und wählen Sie AM oder PMaus.
