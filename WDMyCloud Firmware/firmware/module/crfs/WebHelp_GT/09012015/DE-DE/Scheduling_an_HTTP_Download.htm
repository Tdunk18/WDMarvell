Planen eines HTTP-Downloads

Mit der HTTP-Download-App können Sie einen HTTP-Download zum My Cloud Mirror-Gerät planen. Gehen Sie folgendermaßen vor, um einen HTTP-Download zu planen:

    1. Klicken Sie auf Apps.
    2. Klicken Sie auf HTTP-Downloads.
    3. Geben Sie im URL-Feld die URL der Datei ein, die heruntergeladen werden soll.
      Beispiel: http://192.168.0.32/test.txt
    4. Geben Sie im Feld „Speichern unter“ den gewünschten Speicherort für die heruntergeladene Datei ein oder klicken Sie auf Durchsuchen, um den gewünschten Speicherort auszuwählen.
      Beispiel: Volume_1/Öffentlich/Freigegebene Videos 
    5. Geben Sie je nach Wunsch im Feld „Umbenennen“ einen neuen Namen für die Datei ein.
    6. Klicken Sie auf , wenn es sich um eine wiederholte Sicherung handelt.
    • Wählen Sie die Sicherungsfrequenz aus (täglich, wöchentlich, monatlich).
    • Geben Sie je nach ausgewählter Frequenz die Uhrzeit, den Tag bzw. das Datum für die Sicherung ein.
    1. Geben Sie das Datum ein, an dem der Download ausgeführt werden soll (MM/TT/JJJJ), oder klicken Sie auf , um ein Datum auszuwählen.
    2. Wählen Sie aus den Pulldownmenüs eine Uhrzeit für den Download aus.
    3. Klicken Sie auf Erstellen. Der Download wird erstellt und zur Warteschlange der HTTP-Downloads hinzugefügt.
