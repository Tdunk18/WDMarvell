Verbinden des My Cloud Mirror-Geräts mit dem Netzwerk

Die nachstehenden Schritte erläutern, wie das My Cloud Mirror-Laufwerk über ein Ethernet-Kabel mit dem Heimnetzwerk verbunden werden kann. Nachdem diese anfängliche Verbindung erstellt wurde, können Sie das Laufwerk zur Drahtlosverbindung mit dem Netzwerk konfigurieren.

Führen Sie die folgenden Schritte aus, um eine Verbindung mittels Ethernet-Kabel herzustellen:

    1. Schließen Sie ein Kabelende an den Ethernet-Port an der Rückseite des My Cloud Mirror-Geräts an.
    1. Schließen Sie das andere Ende des Kabels direkt an einen Router oder Port eines Netzwerk-Switches an.
    2. Schließen Sie das Ende des Adapters an den Netzanschluss an der Rückseite des Laufwerks an.
    3. Schließen Sie den Netzstecker an eine Netzsteckdose an. Das My Cloud Mirror-Gerät wird automatisch gestartet.

Wichtig: Warten Sie, bis das My Cloud Mirror-Gerät fertig hochgefahren wurde (circa drei Minuten), bevor Sie die Verbindung konfigurieren.

Nachdem Sie das My Cloud Mirror-Gerät über ein Ethernet-Kabel mit dem Netzwerk verbunden haben, können Sie über die Weboberfläche eine Drahtlosverbindung einrichten. Siehe Konfigurieren von Verbindungsoptionen.
