Einrichten des Cloud-Zugriffs

Für den Remotezugriff auf Ihre Cloud von einem Smartphone, Tablet oder Computer aus ist ein My Cloud-Anmeldename erforderlich. (Grundsätzlich wird der Remotezugriff von einem Gerät aus über den My Cloud-Zentralserver ermöglicht, der Zugriff auf die mit Ihrem Heimnetzwerk verbundenen NAS-Geräte gewährt.) Die Art und Weise des Remotezugriffs von Mobilgeräten (Smartphones und Tablets) und Computern unterscheidet sich jedoch geringfügig. Jeder Benutzer muss über einen My Cloud-Anmeldenamen verfügen.

Gehen Sie folgendermaßen vor, um den Remotezugriff einzurichten:

    1. Stellen Sie sicher, dass der Cloud-Zugriff auf dem Gerät aktiviert ist.   Wie geht das?

       

      So aktivieren Sie den Cloud-Zugriff:

      1. Klicken Sie auf Einstellungen.
      2. Klicken Sie im linken Fenster auf Allgemein.
      3. Klicken Sie im Bereich „Cloud-Zugriff“ auf , um den Cloud-Dienst zu aktivieren.

       

    2. Klicken Sie auf Cloud-Zugriff.
    3. Klicken Sie im linken Bereich auf den Benutzer, dem Sie Cloud-Zugriff gewähren möchten.
    4. Geben Sie im Abschnitt „My Cloud-Anmeldename für <Benutzername>“ eine E-Mail-Adresse ein, wenn nicht bereits eine enthalten ist, und klicken Sie dann auf . Daraufhin wird dem Benutzer eine E-Mail mit Anweisungen gesendet, wie er sich bei My Cloud anmelden kann.
    1. Führen Sie einen der folgenden Schritte aus:
    • Setzen Sie den Registrierungsprozess selbst fort. Wie geht das?

       

      So setzen Sie den Registrierungsprozess fort:

      1. Prüfen Sie das in Schritt 4 angegebene E-Mail-Konto und lassen Sie die E-Mail-Benachrichtigung anzeigen.
      2. Klicken Sie in der E-Mail auf den angegebenen Link und befolgen Sie die Anweisungen zum Abschließen der Registrierung.

       

    • Benachrichtigen Sie den Benutzer, dass Sie ein Webkonto eingerichtet haben und lassen Sie ihn mit dem Registrierungsprozess fortfahren. Wie geht das?
      HINWEIS: Es gibt eine weitere Möglichkeit, für ein Mobilgerät Zugriff auf das My Cloud Mirror-Laufwerk einzurichten. Siehe „Gewähren von Remotezugriff für ein Mobilgerät (alternative Methode)“.

       

      Wenden Sie sich an den Benutzer, und teilen Sie ihm mit, dass Sie Remotezugriff-Inhalt für ihn eingerichtet haben und dass er den Registrierungsprozess mit den folgenden Schritten fortsetzen muss:

      1. Prüfen Sie das E-Mail-Konto (die in Schritt 4 angegebene E-Mail-Adresse).
      2. Öffnen Sie die E-Mail-Benachrichtigung von My Cloud.
      3. Klicken Sie in der E-Mail auf den Link und befolgen Sie die angegebenen Anweisungen zum Abschließen der Registrierung.
