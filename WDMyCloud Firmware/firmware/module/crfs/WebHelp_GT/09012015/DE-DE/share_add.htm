Hinzufügen einer Freigabe

Gehen Sie folgendermaßen vor, um eine Freigabe zu erstellen:

    1. Klicken Sie auf Freigaben.
    2. Klicken Sie auf das Symbol „Freigabe hinzufügen“.
    3. Wählen Sie im Pulldownmenü das Volume aus, in dem die Freigabe erstellt werden soll.
    4. Geben Sie einen Namen für die Freigabe ein.
    5. Geben Sie eine Beschreibung für die Freigabe ein. Dies ist ein optionales Feld.
    6. Klicken Sie auf , um den Papierkorb zu aktivieren. Das Feld bleibt in der Position „Ein“.
    • Wenn Sie den Papierkorb nicht aktivieren möchten, stellen Sie sicher, dass das Feld auf Aus festgelegt ist.
    1. Klicken Sie auf , um das Medien-Serving zu aktivieren. Wenn diese Option aktiviert ist, können Sie die in der Freigabe gespeicherten Medien zu anderen Geräten streamen. Dies ist ein optionales Feld.
    • Wenn Sie Medien-Serving nicht aktivieren möchten, stellen Sie sicher, dass das Feld auf Aus festgelegt ist.
    1. Klicken Sie auf , um den FTP-Zugriff zu aktivieren. Dies ermöglicht den FTP-Zugriff auf diese Freigabe. Dies ist ein optionales Feld.
    • Wenn Sie den FTP-Zugriff nicht aktivieren möchten, stellen Sie sicher, dass das Feld auf Ausfestgelegt ist.
    1. Klicken Sie auf , um den WebDAV-Zugriff zu aktivieren. Dies ermöglicht den WebDAV-Zugriff auf diese Freigabe. Dies ist ein optionales Feld.
    • Wenn Sie den WebDAV-Zugriff nicht aktivieren möchten, stellen Sie sicher, dass das Feld auf Aus festgelegt ist.
    1. Klicken Sie auf , um den NSF-Zugriff zu aktivieren. Dies ermöglicht den Network File System (NFS)-Zugriff auf die Freigabe. Dies ist ein optionales Feld.
    • Wenn Sie den NFS-Zugriff nicht aktivieren möchten, stellen Sie sicher, dass das Feld auf Aus festgelegt ist.
    1. Die neue Freigabe wird hinzugefügt und im linken Bereich angezeigt.
