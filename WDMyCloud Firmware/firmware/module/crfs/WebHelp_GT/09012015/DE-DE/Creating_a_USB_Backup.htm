Erstellen einer USB-Sicherung

Gehen Sie folgendermaßen vor, um Daten auf dem My Cloud Mirror-Gerät auf bzw. von einem USB-Gerät zu sichern.

    1. Klicken Sie auf Backups.
    2. Klicken Sie auf USB-Sicherungen.
    3. Klicken Sie auf Auftrag erstellen.
    4. Geben Sie einen Auftragsnamen für die Sicherung ein.
    5. Legen Sie fest, in welche Richtung die Sicherung erfolgen soll. Folgende Optionen stehen zur Verfügung:
    • USB zu NAS: Daten auf dem USB-Gerät werden auf dem My Cloud Mirror-Gerät gesichert.
    • NAS zu USB: Daten auf dem My Cloud Mirror-Gerät werden auf einem USB-Gerät gesichert.
    1. Wählen Sie den Quellordner für die Sicherung aus:
    • Klicken Sie auf Durchsuchen und wählen Sie den Ordner aus, den Sie vom Zielgerät sichern möchten.
    1. Wählen Sie den Zielordner für die Sicherung aus.
    • Klicken Sie auf Durchsuchen und wählen Sie den Ordner aus, in dem Sie das Quellgerät sichern möchten.
    1. Geben Sie über das Dropdownmenü einen Sicherungstyp an.
    2. Um beim Verbinden automatisch eine USB-Sicherung zu starten, klicken Sie auf das Umschaltsymbol .
    3. Klicken Sie auf Erstellen. Dies erstellt den Sicherungsauftrag für die angeforderte Sicherung.
    4. Klicken Sie im Bereich „USB-Sicherungsaufträge“ auf Details, um den Fortschritt des Sicherungsauftrags zu überprüfen.
