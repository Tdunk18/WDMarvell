Prüfen des Remotezugriffstatus

Folgendes können Sie jederzeit prüfen:

    • Das My Cloud Mirror-Gerät ist für Remotezugriff konfiguriert.
    • Der Remotezugriff auf das My Cloud Mirror-Gerät wird über eine direkte oder eine indirekte Verbindung aktiviert.
    • Ist ein bestimmtes Mobilgerät für einen bestimmten Benutzer aktuell für den Remotezugriff auf Ihre Cloud aktiviert?

Gehen Sie folgendermaßen vor, um den Status des Mobilzugriffs zu überprüfen:

    1. Klicken Sie auf „Cloud-Zugriff“.
    2. Klicken Sie im linken Fenster auf einen Benutzernamen.
      In der nachstehenden Tabelle werden die Felder für den Mobilzugriff und die möglichen Werte aufgeführt:

Abschnitt

Feld-/Optionsname

Werte und Beschreibungen

Cloud-Zugriffsprofil

Verbindungsstatus

  • Verbunden (direkt) Mobiler Zugriff ist über eine Direktverbindung aktiviert.

  • Verbunden (Relais): Mobiler Zugriff ist über eine Relaisverbindung aktiviert.

  • Fehlgeschlagen: My Cloud Mirror kann keine Remotezugriffsverbindung herstellen.

Cloud-Geräte für Benutzername

Geräte- oder App-Name

  • Verbunden: Gerät verfügt über Remotezugriff auf Ihre Cloud.

  • Wartet: Wartet auf Geräteaktivierung.

  • Code abgelaufen: Aktivierungscode ist abgelaufen.

  • Deaktiviert: Die Cloud-Dienstoption ist deaktiviert. Siehe Ein-/Ausschalten des Cloud-Zugriffs.

  • Verbindung fehlgeschlagen: Remoteverbindung ist fehlgeschlagen. Sie können versuchen, das Problem durch eine Fehlersuche für Ihre Netzwerkverbindung und durch die Installation der aktuellsten Firmware zu beheben.

 

HINWEIS: Sollen Einstellungen für den mobilen Zugriff eines anderen Benutzers angezeigt werden, klicken Sie in der linken Spalte auf den entsprechenden Benutzernamen. Die Ansicht wird mit den Cloud-Geäteinformationen dieses Benutzers aktualisiert.
