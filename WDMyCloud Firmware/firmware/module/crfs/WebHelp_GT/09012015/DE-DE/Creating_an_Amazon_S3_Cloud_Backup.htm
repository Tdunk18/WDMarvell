Erstellen einer Amazon S3-Cloud-Sicherung

Hinweis: Bevor Sie eine Cloud-Sicherung erstellen, registrieren sie sich bei http://aws.amazon.com/s3 für den Amazon S3-Dienst.

Nachdem Sie ein Amazon S3-Konto eingerichtet haben, führen Sie die nachfolgend genannten Schritte aus, um eine Cloud-Sicherung zu erstellen.

    1. Klicken Sie auf Backups.
    2. Klicken Sie auf Cloud-Sicherung.
    3. Klicken Sie oben am Bildschirm auf die Amazon S3-Schaltfläche.
    4. Klicken Sie auf Erstellen.
    5. Geben Sie einen Auftragsnamen für die Cloud-Sicherung ein.
    6. Klicken Sie auf Weiter.
    7. Geben Sie die folgenden von Amazon S3 bereitgestellten Informationen ein:
    • Region: Wählen Sie im Dropdownmenü die für die Cloud verwendete Region aus.
    • Zugriffsschlüssel: Geben Sie den Zugriffsschlüssel ein, der Ihnen von Amazon S3 bereitgestellt wurde.
    • Privater Schlüssel: Geben Sie den privaten Schlüssel ein, der Ihnen von Amazon S3 bereitgestellt wurde.
    • Remotepfad: Geben Sie den Remotepfad für die Cloud ein. Hierbei handelt es sich üblicherweise um den Bucketnamen.
    1. Klicken Sie auf Weiter.
    2. Wählen Sie im Feld „Typ“ eine der folgenden Optionen aus:
    • Heraufladen: Mit dieser Option werden Ihre My Cloud Mirror-Daten im Amazon S3-Bucket gespeichert.
    • Herunterladen: Mit dieser Option werden die Daten Ihres Amazon S3-Bucket auf dem My Cloud Mirror-Server gespeichert.
    1. Wählen Sie im Feld „Sicherungstyp“ im Dropdownmenü die Art der Sicherung aus, die Sie ausführen möchten. Folgende Optionen stehen zur Verfügung:
    • Vorhandene Dateien überschreiben: Mit dieser Option werden Dateien im Zielordner, die den gleichen Namen tragen wie die Quelldatei, überschrieben.
    • Vollständige Sicherung: Mit dieser Option wird bei jedem Ausführen der Sicherung ein separater Ordner erstellt, der alle Sicherungsdaten enthält.
    • Inkrementelle Sicherung: Mit dieser Option werden Dateien überschreiben, wenn die entsprechenden Quelldateien neuer sind als die Zieldateien.
    1. Klicken Sie auf Weiter.
    2. Geben Sie im Feld „Lokaler Pfad“ einen Pfad für die Sicherung auf dem My Cloud Mirror-Gerät ein.
    • Klicken Sie auf Durchsuchen, um auf dem Gerät nach einem Speicherort für die Sicherung zu suchen.
      (Beispiel: Volume_1/Sicherung)
    1. Klicken Sie auf , um die Funktion der automatischen Aktualisierung zu aktivieren. Mit dieser Option wird die Sicherung automatisch auf Grundlage eines Zeitplans aktualisiert.
    • Wenn Sie die Funktion der automatischen Aktualisierung aktivieren:

      • Wählen Sie den Zeitplan für die automatische Aktualisierung aus: täglich, wöchentlich oder monatlich

      • Wählen Sie aus dem Dropdownmenü die Uhrzeit für die automatische Aktualisierung aus (bei Auswahl der täglichen Aktualisierung).

      • Wählen Sie im Dropdownmenü den Tag und die Uhrzeit für die automatische Aktualisierung aus (wöchentliche oder monatliche Sicherung).

    1. Klicken Sie auf Weiter.
    2. Überprüfen Sie die Einstellungen und klicken Sie auf Fertig stellen. Die Amazon S3-Sicherung wird erstellt.

HINWEIS: Greifen Sie auf den Sicherungs-Bucket Ihrer Amazon S3-Cloud zu, um die Gerätesicherungen anzuzeigen.
