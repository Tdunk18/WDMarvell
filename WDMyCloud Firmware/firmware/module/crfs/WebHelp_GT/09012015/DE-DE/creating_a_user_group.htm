Erstellen einer Gruppe

Gruppen sind Sammlungen von Benutzerkonten, die als verwaltbare Einheiten zusammengefasst werden. Gehen Sie folgendermaßen vor, um eine Gruppe zu erstellen:

    1. Klicken Sie auf Benutzer.
    2. Klicken Sie auf Gruppen.
    3. Klicken Sie auf das Symbol Gruppen hinzufügen.
    4. Geben Sie einen Gruppennamen für die neue Gruppe ein. Der Name darf bis zu 16 Zeichen enthalten, jedoch keine Leerzeichen.
    5. Wählen Sie die Benutzer aus, die Sie zur Gruppe hinzufügen möchten.
    6. Klicken Sie auf Übernehmen. Die neue Gruppe wird erstellt und in der linken Spalte am Bildschirm angezeigt.
