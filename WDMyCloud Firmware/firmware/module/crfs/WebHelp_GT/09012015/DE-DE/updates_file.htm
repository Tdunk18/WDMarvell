Aktualisieren des My Cloud Mirror-Geräts über eine Datei

In einigen Fällen müssen Sie möglicherweise zur Aktualisierung des My Cloud Mirror-Geräts zunächst eine Datei von der Western Digital-Support-Site herunterladen und diese anschließend installieren.

HINWEIS: Für dieses Verfahren ist ein Internetzugang erforderlich.

Führen Sie die nachfolgend genannten Schritte aus, um das My Cloud Mirror-Gerät manuell aus einer Datei zu aktualisieren:

    1. Öffnen Sie eine neue Registerkarte oder ein neues Fenster in Ihrem Browser.
    2. Verweisen Sie auf der neuen Registerkarte/im neuen Fenster Ihren Browser auf die technische Support-Site von Western Digital:
      http://support.wdc.com
    • Ist eine neue Firmware-Aktualisierung verfügbar, speichern Sie diese an einem Speicherort auf Ihrem Gerät.
    1. Kehren Sie zur My Cloud Mirror-Weboberfläche zurück.
    2. Klicken Sie auf Einstellungen.
    3. Klicken Sie im linken Fenster auf Firmware-Aktualisierung.
    4. Klicken Sie im Abschnitt „Manuelles Update“ auf Aus Datei aktualisieren.
    5. Navigieren Sie zu dem Speicherort, an dem Sie die Aktualisierung in Schritt 3 gespeichert haben.
    6. Wählen Sie die Aktualisierungsdatei aus und klicken Sie auf Öffnen.
    7. Klicken Sie auf OK, um die Installation zu bestätigen. Nach der Installation der Aktualisierung wird dasMy Cloud Mirror-Gerät neu gestartet.
      HINWEIS: Während des My Cloud Mirror-Neustarts wird der Dienst für alle Benutzer, die aktuell auf die Cloud zugreifen, unterbrochen.
