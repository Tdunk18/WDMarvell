Erstellen eines P2P-Downloads

Mit der Peer-to-Peer (P2P)-Download-App können Sie einen P2P-Download zum My Cloud Mirror-Gerät planen. Gehen Sie folgendermaßen vor, um einen P2P-Download zu planen:

    1. Klicken Sie auf Apps.
    2. Klicken Sie auf P2P-Downloads.
    3. Klicken Sie auf , um einen P2P-Download zu aktivieren.
    • Klicken Sie auf Konfigurieren, um Änderungen an der Konfiguration des P2P-Download vorzunehmen.
    1. Geben Sie die URL oder die Torrent-Datei des für den Download ein:
    • Wenn Sie über eine URL verfügen:

      • Geben Sie im Feld „Torrent aus URL hinzufügen“ die URL der Torrent-Datei ein, die heruntergeladen werden soll.

      • Klicken Sie auf Torrent hinzufügen. Der P2P-Download wird erstellt und zur Warteschlange der P2P-Downloads hinzugefügt.

    • Wenn Sie über eine Torrent-Datei verfügen:

      • Wählen Sie Klicken, um eine Datei heraufzuladen aus.

      • Wählen Sie eine heraufzuladende Torrent-Datei aus und klicken Sie auf Öffnen. Der P2P-Download wird erstellt und zur Warteschlange der P2P-Downloads hinzugefügt.
