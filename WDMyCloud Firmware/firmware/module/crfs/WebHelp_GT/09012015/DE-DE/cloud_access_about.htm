Info zum Cloud-Zugriff

Cloud-Zugriff ist im Wesentlichen der Zugriff auf Ihre persönliche Cloud (das heißt, das My Cloud Mirror-Gerät) über ein Gerät (z. B. Smartphone, Tablet oder Computer) aus. Eine Übersicht über die Methoden für den Remotezugriff auf Ihren Inhalt finden Sie unter Grundlagen zum Remotezugriff.

Als Administrator können Sie die folgenden Aufgaben für den Cloud-Zugriff ausführen:

    • Cloud-Zugriff einrichten  
    • Remotezugriff ein-/ausschalten
    • Festlegen, wie Geräte auf Ihr Laufwerk zugreifen (Einrichten von statischen, DHCP- oder manuellen Verbindungen)  
    • Remotezugriffstatus prüfen 
    • Leistung durch Portweiterleitung steigern
