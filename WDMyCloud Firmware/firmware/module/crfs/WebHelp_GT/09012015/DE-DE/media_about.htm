Info zu Medieneinstellungen

Aus der Medienansicht können Sie Aufgaben im Zusammenhang mit dem DLNA-Medienserver und dem iTunes-Medienplayer ausführen.

Was ist DLNA?

 DLNA (Digital Living Network Alliance) ist ein von Herstellern von Unterhaltungselektronikgeräten verwendeter Standard, über den Unterhaltungsgeräte Inhalte über ein Heimnetzwerk miteinander austauschen können. Sie können beispielsweise My Cloud Mirror so konfigurieren, dass Sie Dateien auf Ihrem Großbild-TV wiedergeben/anzeigen können, vorausgesetzt, bei Ihrem Fernseher handelt es sich um ein DLNA-zertifiziertes Gerät.
