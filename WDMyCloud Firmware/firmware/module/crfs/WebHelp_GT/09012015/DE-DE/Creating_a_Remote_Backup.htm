Erstellen einer Remotesicherung

Mit einer Remotesicherung können Sie die das My Cloud Mirror-Gerät auf einem anderen Server innerhalb oder außerhalb des Netzwerks sichern.  Führen Sie die nachfolgend genannten Schritte aus, um eine Remotesicherung für den My Cloud Mirror-Server zu erstellen.

    1. Klicken Sie auf Backups.
    2. Klicken Sie auf Remotesicherung.
    3. Klicken Sie auf Auftrag erstellen.
    4. Geben Sie einen Auftragsnamen ein.
    5. Wählen Sie einen Server aus der Remoteserver-Dropdownliste aus.
    6.  Geben Sie im Feld „Remote-IP“ die IP-Adresse des Remotesicherungsgeräts ein.
      (Beispiel: 192.168.1.16)
    7. Geben Sie im Feld „Passwort“ das Passwort für das Remotesicherungsgerät ein.
    8. Geben Sie den SSH-Benutzernamen ein.
    9. Geben Sie im Feld „SSH-Passwort“ das SSH-Passwort ein.
    10. Klicken Sie auf Durchsuchen, um einen Quellordner auszuwählen.
    11. Markieren Sie in der Liste der Remotesicherungsordner ein oder mehrere zu schützende Freigabeverzeichnisse aus.
    12. Klicken Sie auf OK.
    13. Klicken Sie auf „Durchsuchen“, um ein Remotesicherungsverzeichnis auszuwählen.
    14. Wählen Sie ein Verzeichnis aus, und klicken Sie auf „OK“.
    15. Geben Sie über das Dropdownmenü einen Sicherungstyp an.
    16. Um einen Sicherungszeitplan festzulegen, klicken Sie neben „Serie“ auf das Umschaltsymbol .
    17. Um eine Übertragungsverschlüsselung festzulegen, klicken Sie auf das Umschaltsymbol .
    18. Klicken Sie auf Erstellen.
