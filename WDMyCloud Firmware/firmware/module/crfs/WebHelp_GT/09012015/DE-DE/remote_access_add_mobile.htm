Hinzufügen eines Mobilgeräts

Mit den Apps MyCloud-Alben und My Cloud können Sie von einem Mobilgerät (beispielsweise einem Smartphone oder Tablet) aus auf die Inhalte in der Cloud zugreifen. Laden Sie diese kostenlosen Apps auf Ihr Mobilgerät herunter, bevor Sie die hier beschriebene Prozedur beginnen.

So fügen Sie ein Mobilgerät hinzu:

  1. Sofern noch nicht erfolgt, aktivieren Sie Remotezugriff auf Ihre Cloud für alle Benutzer.
  2. Klicken Sie auf das Symbol „Cloud-Zugriff“.
  1. Klicken Sie im linken Fenster auf den Namen des Benutzers, dessen Mobilgerät Sie Zugriff auf die Cloud gewähren möchten.
  1. Klicken Sie auf Code abrufen. My Cloud Mirror zeigt einen Code an, den Sie innerhalb von 24 Stunden in Ihr Mobilgerät eingeben müssen. Notieren Sie diesen Code, um ihn jederzeit präsent zu haben.
  1. Klicken Sie auf „Fertig“.
  2. Geben Sie an Ihrem Mobilgerät den Aktivierungscode ein.
    Nach der Aktivierung Ihres Geräts können Sie über die My Cloud Apps auf die Cloud-Inhalte zugreifen.
