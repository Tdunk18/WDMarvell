Info zu Setup und Konfiguration

Als Eigentümer und Administrator des My Cloud Mirror-Geräts können Sie sofort auf Ihre Medien zugreifen. Sie sollten jedoch im Rahmen des ersten Setups Ihrer Verbindung einige Schritte ausführen. Unter Erste Schritte finden Sie eine Übersicht über die empfohlenen Setup-Schritte.

Während Sie mit Ihrer Familie Ihre persönliche Cloud genießen, müssen Sie möglicherweise von Zeit zu Zeit Änderungen an dieser persönlichen Cloud vornehmen. Diese Art Änderungen können Sie mit einem Klick auf „Einstellungen“ aufrufen. Die Einstellungen entfallen auf die folgenden Kategorien:

    • Allgemein
    • Netzwerk
    • Medien
    • Hilfsprogramm
    • Benachrichtigungen
    • Firmware-Aktualisierung

Unter den einzelnen Kategorien werden verfügbare Konfigurationsschritte aufgeführt.
