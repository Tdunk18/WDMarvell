Zugreifen auf Inhalt von einem Smartphone oder Tablet aus

Sie können mit Ihrem Smartphone oder Tablet auf den Inhalt des My Cloud Mirror-Geräts zugreifen. Dafür müssen jedoch zuvor die folgenden Aufgaben ausgeführt werden: Einrichten von Cloud-Zugriff, Herunterladen von WD-Mobile-Apps und Durchführen eines einmaligen Setups für sicheren Zugriff von einem Mobilgerät aus.

Führen Sie folgende Schritte aus, um von einem Smartphone oder Tablet auf Ihre Inhalte zuzugreifen:

    1. Vergewissern Sie sich, dass Ihr Mobilgerät über eine WLAN- oder 3G-/4G-Verbindung verfügt.
    2. Rufen Sie eine WD-Mobil-App auf.
    3. Wenn Sie dazu aufgefordert werden, tippen Sie auf "Jetzt verbinden".
    4. Wählen Sie Ihr My Cloud Mirror-Laufwerk aus.
    5. Wählen Sie die Freigabe aus, auf die Sie zugreifen möchten, und navigieren Sie dann zum gewünschten Inhalt.
