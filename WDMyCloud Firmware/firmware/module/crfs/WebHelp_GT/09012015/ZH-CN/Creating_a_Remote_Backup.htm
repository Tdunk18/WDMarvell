创建远程备份

远程备份允许将 My Cloud Mirror 设备备份到网络以内或以外的另一台服务器上。使用以下步骤创建 My Cloud Mirror 服务器的远程备份。

    1. 单击“备份”。
    2. 单击远程备份。
    3. 单击创建作业按钮。
    4. 请输入作业名。
    5. 从“远程服务器”下拉菜单中选择服务器。
    6.  在“远程 IP”字段中，输入远程备份设备的 IP 地址。
      （例如 192.168.1.16）
    7. 在“密码”字段中，输入远程备份设备的密码。
    8. 请输入 SSH 用户名。
    9. 在“ SSH 密码”字段中，输入 SSH 密码。
    10. 单击浏览以选择源文件夹。
    11. 从远程备份文件夹列表中检查一个或多个共享目录以提供保护。
    12. 单击“确定”。
    13. 如需选择远程备份目录，单击“浏览”。
    14. 选择目录，然后单击“确定”。
    15. 从下拉菜单中选择“备份类型”。
    16. 如需设置备份计划，单击“循环”旁边的切换图标。
    17. 如需设置“传输加密”，单击切换图标。
    18. 单击“创建”。
