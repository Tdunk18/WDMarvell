执行一次性设置以获得安全访问权限

本过程介绍如何在智能手机或平板电脑与 My Cloud Mirror 设备之间建立安全连接。请参考以下有关特定移动设备的说明：

    • Android 智能手机或平板电脑  

    • iPhone 或 iPad

    • Windows Phone 或平板电脑

    • Windows 或 Mac 桌面计算机

获得 Android 智能手机或平板电脑的安全访问权限

    1. 执行以下步骤连接智能手机或平板电脑：

      1. 在移动设备上点击“设置”。

      2. 点击“无线和网络”。

      3. 点击“Wi-Fi 设置”。

      4. 选择您的家庭网络。

      5. 输入网络密码，然后点击连接。

    1. 启动 WD 应用。
    2. 点击“立即连接”。
    3. 点击 My Cloud Mirror 设备的名称。
    4. 点击下拉箭头，然后选择您的用户名。
    5. 执行以下某一操作：

    • 如果为该用户设置了密码，请输入密码，然后点击登录。

    • 如果没有为该用户设置密码，请点击登录。

    1. （可选）如果您想要访问您的任何内容，请点击某个共享名称，然后导航到所需的文件。

获得 iPhone 或 iPad 的安全访问权限

    1. 执行以下步骤连接智能手机或平板电脑：

    1. 在移动设备上点击“设置”。

    2. 点击“Wi-Fi”。

    3. 点击“Wi-Fi 设置”。

    4. 选择您的家庭网络。

    5. 输入您的网络密码，然后点击加入。

    1. 启动 WD 应用。
    2. 点击“立即连接”。
    3. 点击 My Cloud Mirror 设备名称旁边的加号 (+)。
    4. 点击所需的用户名。
      如果为该用户设置了密码，请输入密码。
    5. （可选）如果您想要访问您的任何内容，请点击某个共享名称，然后导航到所需的文件。

获得 Windows 智能手机或平板电脑的安全访问权限

    1. 执行以下步骤连接智能手机或平板电脑：

    1. 在您的计算设备上点击设置。

    1. 点击“Wi-Fi”。

    2. 选择您的家庭网络。

    3. 输入您的网络密码，然后点击完成。

    1. 启动 WD 应用。
    2. 点击“立即连接”。
    3. 点击 My Cloud Mirror 驱动器的名称。
    4. 点击下拉箭头，然后选择一个用户名。
      如果为该用户设置了密码，请输入密码。
    5. （可选）如果您想要访问您的任何内容，请执行以下操作：

    1. 在“设备”屏幕上，点击 My Cloud Mirror 驱动器的名称。

    2. 点击共享，然后浏览到所需的内容。
