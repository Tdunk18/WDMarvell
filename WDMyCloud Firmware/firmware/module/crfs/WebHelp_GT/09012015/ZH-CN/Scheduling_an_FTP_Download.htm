计划 FTP 下载

FTP 下载应用可用来安排 My Cloud Mirror 的 FTP 下载。使用以下步骤安排 FTP 下载。

    1. 单击应用。
    2. 单击 FTP 下载。
    3. 单击创建作业。
    4. 在“作业名称”字段中输入 FTP 备份作业的名称。
    5. 在“主机”字段中，输入主机值。
    6. 在“登录方式”字段中，选择要用来下载 FTP 的方法：
    • 账户：输入该账户的用户名和密码。
    • 匿名
    1. 在“类型”字段中，选择要下载的信息的类型：
    • 文件
    • 文件夹 
    1. 在“源文件夹”字段中，单击浏览以选择源目录。 
    2. 在“目标文件夹”字段中，单击浏览以选择目标目录。
    3. 在“语言”字段中，从下拉菜单选择 FTP 客户端支持的语言。
    4. 如果这是定期备份，单击 。
    • 选择备份频率（每天、每周、每月）
    • 根据所选频率，输入备份时间、备份日和备份日期。
    1. 输入要执行下载的日期 (mm/dd/yyyy) 或单击 选择日期。
    2. 从下拉菜单中选择下载时间。
    3. 单击创建。您的下载随即创建并添加到 FTP 下载队列。
