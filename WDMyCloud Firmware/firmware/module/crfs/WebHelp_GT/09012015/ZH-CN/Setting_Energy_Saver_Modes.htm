设置节电器模式

使用以下步骤降低 My Cloud Mirror 设备的能耗。

    1. 单击设置。
    2. 单击常规。
    3. 在“节电器”区域，选择以下选项设置设备的能耗。

模式

描述

驱动器睡眠

在设备上启用/禁用睡眠模式。选择“开”时，驱动器睡眠模式启用。

  • 单击 启用/禁用驱动器睡眠。

LED

在设备上启用/禁用 LED。选择“开”时，LED 禁用。

  • 单击 启用/禁用 LED。

电源恢复

在因为电源故障而发生意外关机后自动重新启动设备。选择“开”时，电源恢复启用。

  • 单击 启用/禁用电源恢复。

电源计划

用来设置指示设备何时关闭的计划。

  • 单击 启用/禁用电源计划。
  • 单击配置为设备设置计划。

Web 访问超时

设置设备自动将您从设备上注销的默认时间。

  • 从下拉菜单中选择 Web 访问超时时间。
