添加用户帐户

按照下面的步骤将用户添加到 My Cloud Mirror 设备：

    1. 单击“用户”。
    2. 单击添加用户图标：
    3. 输入用户名。此名称用来登录到系统。
    4. 输入名字。这是用户的名字。
    5. 输入姓氏。这是用户的姓氏。
    6. 输入电子邮件地址。这是用户的电子邮件地址。此地址也用于创建 My Cloud 账户。
    7. 输入用户账户的密码。
    8. 重新输入密码进行确认。
    9. 单击应用。
    10. 单击 。这将把密码字段更改为“开”状态，并启用密码字段。
    • 如果不想指定密码，请确保密码字段为“关”状态，然后单击下一步。
