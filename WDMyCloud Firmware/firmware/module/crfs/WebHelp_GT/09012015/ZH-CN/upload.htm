将内容上载到 My Cloud Mirror 设备

要将照片、视频、音乐和文件上载到 My Cloud Mirror 设备，可以使用以下过程：

    • Mac
    • Windows PC
    • USB

从 Mac 上载内容

使用以下步骤将 Mac 中的文件上载到 My Cloud Mirror 设备：

    1. 在 Mac 上，从菜单栏中选择转到 > 网络。
    2. 执行以下某一操作：
    • 在 Finder 的左窗格中，双击 My Cloud Mirror 设备的名称。
    • 在右窗格中，找到 My Cloud Mirror 设备的名称并双击。
    1. 双击特定共享的名称以打开文件夹并查看所有现有的子文件夹。（Public 是自动创建的共享。）
    2. 将计算机中的文件拖放到所需的文件夹中。
      请不要将文件放入 Time Machine 和 WD SmartWare Backups 文件夹中。
      注：如果将计算机中的文件复制到 My Cloud Mirror 设备，这些文件将会存放到两个位置：原始位置和您的设备。

从 PC 上载内容

使用以下步骤将 Windows PC 中的文件上载到 My Cloud Mirror 设备：

    1. 在计算机上，同时按 Windows 和 R 键以打开“运行”对话框。
    2. 在“打开”字段中输入 \\，后跟 My Cloud Mirror 设备的名称。（默认名称为 mycloudmirror，不包含空格。）示例：\\mycloudmirror
    3. 单击“确定”。
      此时将打开 Windows 资源管理器或文件资源管理器，可用的共享列在右窗格中。
    4. 双击特定共享的名称以打开文件夹并查看所有子文件夹。（Public 是自动创建的共享。）
    5. 将计算机中的文件拖放到此共享中。
      注：如果将计算机中的文件复制到 My Cloud Mirror 设备，这些文件将会存放到两个位置：原始位置和您的设备。

     

    从 USB 上载内容

    使用以下步骤将 USB 中的文件上载到 My Cloud Mirror 设备：

    1. 将 USB 驱动器连接到计算机以获得最佳性能。

    2. 将计算机和 My Cloud Mirror 设备通过有线以太网连接直接连接到路由器。

    3. 将数据从外部 USB 驱动器复制到 My Cloud Mirror 设备

注：如果将计算机中的文件复制到 My Cloud Mirror 设备，这些文件将会存放到两个位置：原始位置和您的设备。
