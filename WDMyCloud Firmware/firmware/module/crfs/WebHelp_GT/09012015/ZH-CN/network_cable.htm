将 My Cloud Mirror 设备连接到网络

本过程介绍如何使用以太网电缆将 My Cloud Mirror 驱动器连接到家庭网络。建立此初始连接后，就可配置驱动器以无线方式连接到网络。

使用以下步骤用以太网电缆连接。

    1. 将电缆的一端连接到 My Cloud Mirror 设备背面的以太网端口中。
    1. 将电缆的另一端直接连接到路由器或网络交换机端口。
    2. 将适配器的一端连接到驱动器背面的电源插座中。
    3. 将电源适配器插入电源插座。My Cloud Mirror 设备将会自动开机。

重要说明：在配置连接器之前，请等待 My Cloud Mirror 设备完成通电（大概需要三分钟）。

在通过以太网电缆将 My Cloud Mirror 设备连接到网络后，即可使用 Web 界面设置无线连接。请参阅配置连接选项。
