使用 Apple Time Machine 备份您的数据

您可以将 My Cloud Mirror 设备中的文件备份到 Mac 计算机上的 Apple Time Machine。

使用以下步骤将数据备份到 Apple Time Machine。

    1. 单击“设置”。
    2. 在左窗格中，单击“常规”。
    3. 在“MAC 备份”区域，单击 打开 Time Machine 备份选项。
    4. 单击“配置”。
    5. 在“选择共享”下拉列表中，选择备份文件将要存放到的共享。
    6. 单击“保存”。
