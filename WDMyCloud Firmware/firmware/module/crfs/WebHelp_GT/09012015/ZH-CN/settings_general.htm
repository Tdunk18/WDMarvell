关于常规设置

您可以通过“设置”选项来设置和配置 My Cloud Mirror 设备环境。

“设置”类别包括“常规”视图。下面列出了可在此视图中完成的任务。

  • 设备配置文件

    • 更改设备名称  
    • 更改设备描述

  • 语言和时钟

    • 设置 My Cloud Mirror 界面中显示的语言  
    • 设置日期和时间
    • 设置时区  

  • 云访问

    • 打开/关闭远程访问  
    • 配置连接选项
    • 启用端口转发  
    • 检查远程访问状态  

  • 节电器

    • 打开/关闭设备正面的 LED  

    • 确定设备何时进入睡眠状态  

  • Mac 备份

    • 配置向 Apple Time Machine 备份  

    • 关闭 Time Machine 备份
