欢迎使用

您的 My Cloud Mirror 设备系统是一种高性能的系统，在连接到无线路由器时，它能让您从任何计算机、平板电脑或智能手机上安全地无线访问您的媒体和文件。您的 My Cloud Mirror 设备是个人云的中心组件，提供简单而可靠的界面，以便用户轻松进行设置、共享和维护。

个人云中的用户角色

访问个人云的每个人都必须有一个 My Cloud Mirror 驱动器用户账户。作为 My Cloud Mirror 的所有者，您有一个特殊的用户账户 (admin)，该账户具有管理员权限。使用这些权限，您可以根据具体需求设置和配置设备，以及将其他用户添加到您的个人云。您有权确定究竟哪些人可访问您的设备。

注：默认情况下，admin 用户名没有密码。您随时可以添加密码。

重要说明：您可以在运行 My Cloud Mirror 设置软件或完成“入门”屏幕中的操作时，更改管理员名称并设置可选的密码。请确保记住该名称和密码。如果没有这些凭据，您将无法显示 My Cloud Mirror Web 界面。

显示 My Cloud Mirror 界面

您可以通过 My Cloud Mirror 设备用户界面设置和管理个人云。该界面显示在 Mac 或 PC 上运行的 Web 浏览器中。 

要显示 Web 界面，请执行以下操作：

    1. 打开一个浏览器窗口。
    2. 在地址栏中，输入 http://
      ；示例：http://wdmycloudmirror
      注：如果输入该 URL 后未能显示 Web 界面，请尝试输入以下 URL：http://wdmycloudmirror.local
