从文件更新 My Cloud Mirror 设备

在某些情况下，您可能需要通过首先从 Western Digital 支持站点下载并安装某个文件来更新 My Cloud Mirror 设备。

注：此过程需要访问 Internet。

使用以下步骤手动从文件更新 My Cloud Mirror 设备：

    1. 打开新的浏览器选项卡或窗口。
    2. 在新的选项卡/窗口中，将浏览器指向 Western Digital 技术支持站点：
      http://support.wdc.com
    • 如果有新固件更新可用，请将它保存到计算设备上的某个位置。
    1. 返回到 My Cloud Mirror 网页界面。
    2. 单击设置。
    3. 在左窗格中，单击固件更新。
    4. 在“手动更新”区域中，单击从文件更新。
    5. 导航到您在执行步骤 3 时保存更新的位置。
    6. 选择该更新文件，然后单击打开。
    7. 单击确定确认安装。在安装更新后，My Cloud Mirror 设备将重新启动。
      注：当 My Cloud Mirror 重新启动时，当前正在访问云的所有用户的服务将中断。
