什么是云备份？

云备份允许创建可通过 Internet 以及各种移动设备访问的远程备份。My Cloud Mirror 设备使用下列云服务来创建远程备份：

    • ElephantDrive：ElephantDrive 是一种云备份服务，可远程提供安全、自动的文件备份。
    • Amazon S3：Amazon Simple Storage Service (S3) 是一种在线文件存储 Web 服务，可用于从 Web 上的任何位置随时存取任意数量的数据。
