Adding a User Account

Follow the steps below to  add a user to your My Cloud Mirror device:

    1. Click Users.
    2. Click the Add User icon:
    3. Enter a User Name. This name is used to log into the system.
    4. Enter a First Name. This is the first name of the user.
    5. Enter a Last Name. This is the  last name of the user.
    6. Enter an Email Address. This is the user's email address. This address is also used to create a My Cloud account.
    7. Enter a password for the user account.
    8. Reenter your password to confirm it.
    9. Click Apply.
    10. Click . This changes the password field to ON and enables the password fields.
    • If you don't want to assign a password, ensure that the password field is Off and click Next.
