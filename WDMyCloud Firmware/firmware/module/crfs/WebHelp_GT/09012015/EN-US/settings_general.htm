About General Settings

You set up and configure your My Cloud Mirror device environment through the Settings options.

The Settings category includes the General view. Tasks you can complete from this view are listed here.

  • Device Profile

    • Change the device name  
    • Change the device description

  • Language & Clock

    • Set the language displayed in the My Cloud Mirror interface  
    • Set the date and time
    • Set the time zone  

  • Cloud Access

    • Turn on/off remote access  
    • Configure connection options
    • Enable port forwarding  
    • Check remote access status  

  • Energy Saver

    • Turn on/off LED in front of the device  

    • Determining when your device goes to sleep  

  • Mac Backups

    • Configure backups to Apple Time Machine  

    • Turn off Time Machine backups
