Creating a Group

Groups are a collection of user accounts gathered into manageable units. Use the following steps to create a Group.

    1. Click Users.
    2. Click Groups.
    3. Click the Add Groups icon.
    4. Enter a Group Name for your new group. The name can be up to 16 characters and cannot contain spaces.
    5. Select the users you'd like to add in the new group.
    6. Click Apply. Your new group is created and displays in the left column of the screen.
