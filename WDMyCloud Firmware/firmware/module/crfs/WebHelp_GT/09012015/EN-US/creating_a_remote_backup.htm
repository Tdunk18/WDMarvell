Creating a Remote Backup

Remote backup allows you to back up your My Cloud Mirror device to another server on or outside of your network.  Use the following steps to create a remote backup for your My Cloud Mirror server.

    1. Click Backups.
    2. Click Remote Backups.
    3. Click the Create Job button.
    4. Enter a Job Name.
    5. Select a server from the Remote Server dropdown.
    6.  In the Remote IP field, enter the IP address of the remote backup device.
      (Example 192.168.1.16)
    7. In the Password field, enter the password for the remote backup device.
    8. Enter a SSH User Name.
    9. In the SSH Password field, enter the password SSH password.
    10. Click Browse to select a source folder.
    11. From the list of Remote Backup folders, check one or more share directories to protect.
    12. Click OK.
    13. To select a Remote Backup directory, click Browse.
    14. Choose a directory and click OK.
    15. Choose the Backup Type from the dropdown menu.
    16. To set a backup schedule, next to Recurrence, click the toggle icon .
    17. To set Transfer Encryption, click the toggle icon .
    18. Click Create.
