Scheduling an HTTP Download

The HTTP Download app allows you to schedule an http download to your My Cloud Mirror device. Use the following steps to schedule an HTTP download.

    1. Click Apps.
    2. Click HTTP Downloads.
    3. In the URL field, enter the URL of the file you’d like to download.
      Example: http://192.168.0.32/test.txt
    4. In the Save To field, enter the location where you’d like to save the download or click Browse to select the location where you’d like to save the download.
      Example: Volume_1/Public/Shared Videos 
    5. In the Rename field, enter a new name for the file, if you’d like to change the file name.
    6. Click  if this is a recurring backup.
    • Select the frequency of the backup (Daily, Weekly, Monthly)
    • Enter the Time, Day, and Date of the backup, depending on the frequency selected.
    1. Enter the date you’d like to perform the download (mm/dd/yyyy) or select click  to select a date.
    2. Enter a time for your download from the pull-down menus.
    3. Click Create. Your download is created and added to the HTTP Downloads Queue.
