Setting Up Other Users

After you set up your owner account and configure remote access, you're ready to add additional users to your device.

The key task is to add new users and set them up with remote access. See Adding a User Account.

As cloud administrator, you can also perform the following tasks:

    • Add a password to a user account. See Adding/Changing a Password

    • Change a user's access to a share. See Changing Access to a Share

    • Delete a user account. See Deleting a User Account.

    • Change a user name and password. See Adding/Changing a Password
