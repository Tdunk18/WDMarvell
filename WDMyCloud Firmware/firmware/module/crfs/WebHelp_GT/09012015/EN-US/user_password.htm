Adding/Changing a Password

Use the following steps to add or change the password for a user account.

    1. Click Users.
    2. In the left pane, select a user name.
    3. In the Password field, click to activate the Password option, if not already activated.
    4. Click Configure.
    5. Enter a password in the Password field.
    6. Enter the same password in the Confirm password field.
    7. Click Save.
