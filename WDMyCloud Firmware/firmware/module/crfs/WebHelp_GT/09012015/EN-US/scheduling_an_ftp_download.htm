Scheduling an FTP Download

The FTP Download app allows you to schedule an FTP download to your My Cloud Mirror. Use the following steps to schedule an FTP download.

    1. Click Apps.
    2. Click FTP Downloads.
    3. Click Create Job.
    4. In the Job Name field, enter a name for your FTP backup job.
    5. In the Host field, enter a host value.
    6. In the Login Method field, select the method you’d like to use to download your FTP:
    • Account: Enter the User Name and Password for the account.
    • Anonymous
    1. In the Type field, select the type of information you’d like to download:
    • File
    • Folder 
    1. In the Source Folder field, click Browse to select source directory. 
    2. In the Destination Folder  field, click Browse to select a destination directory.
    3. In the Language field, select the language supported by the FTP client from the pull-down menu.
    4. Click  if this is a recurring backup.
    • Select the frequency of the backup (Daily, Weekly, Monthly)
    • Enter the Time, Day, and Date of the backup, depending on the frequency selected.
    1. Enter the date you’d like to perform the download (mm/dd/yyyy) or select click  to select a date.
    2. Enter a time for your download from the pull-down menus.
    3. Click Create. Your download is created and added to the FTP Downloads Queue.
