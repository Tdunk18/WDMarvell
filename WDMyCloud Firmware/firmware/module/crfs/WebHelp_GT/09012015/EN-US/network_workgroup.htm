Specifying a Workgroup

You can enter the name of a workgroup In the My Cloud Mirror device.

What is a workgroup?

A workgroup is a name you associate with multiple computing devices within a network. Workgroups:

    • Facilitate the easy exchange of information among computing devices

    • Do not require a central server

    • Apply to Microsoft Windows environments only

 

Use the following steps to specify a workgroup:

    1. Click Settings.
    2. In the left pane, click Network.
    3. In the Network Workgroup area, enter the name of the Workgroup you'd like to join.
