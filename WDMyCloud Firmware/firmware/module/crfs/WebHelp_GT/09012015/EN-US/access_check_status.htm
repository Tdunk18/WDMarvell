Checking Remote Access Status

At any time, you can check whether:

    • The My Cloud Mirror device is configured for remote access
    • Remote access to the My Cloud Mirror device is enabled through a direct or relayed connection
    • A particular mobile device for a certain user is currently enabled to access your cloud remotely

Use the following steps to check your mobile access status:

    1. Click Cloud Access.
    2. In the left pane, click a user name.
      The following table lists mobile access fields and their possible values:

Section

Field/Option Name

Values and Descriptions

Cloud Access Profile

Connection Status

  • Connected (Direct): Mobile access is enabled through a direct connection.

  • Connected (Relay): Mobile access is enabled through a relayed connection.

  • Failed: My Cloud Mirror cannot establish a remote access connection.

Cloud Devices for user name

Device or app name

  • Connected: Device has remote access to your cloud.

  • Waiting: Awaiting device activation.

  • Code Expired: Activation code expired.

  • Disabled: The Cloud Service option is turned off. See Turning Cloud Access On/Off.

  • Connection Failed: Remote connection failed. You can try to fix the problem by troubleshooting your network connection and ensuring that the latest firmware is installed.

 

NOTE: If you want to view mobile access settings for a different user, click the person's user name in the left column. The view refreshes with that user's cloud device information.
