Turning Cloud Access On/Off

If you are an administrator, you can turn cloud access or off  for all users (that is, you can control whether computing devices can access the content on the My Cloud Mirror device).

Use the following steps to activate/deactivate Cloud Access.

    1. Click Settings.
    2. In the left pane, click General.
    3. Under Cloud Access in the Cloud Service field, click to turn the cloud service on or off, as appropriate:
    • On: Enables remote access for all users.
    • Off: Disables remote access for all users.

For details on turning off access for a single device, see Removing a Device.

NOTE: Info about the Connection Options area is available in Configuring Connection Options, which explains the different firewall options and includes an explanation of how to improve performance. By default, My Cloud Mirror automatically establishes a direct connection between your mobile devices and router.

If mobile access is enabled through a relay connection (that is, the Connection Status says “Connected (Relay),” you can improve performance by enabling port forwarding on your WD Venue connector (see Enabling Port Forwarding).
