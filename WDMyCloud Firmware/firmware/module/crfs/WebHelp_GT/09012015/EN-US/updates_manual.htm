Checking for Updates and Installing Manually

If the My Cloud Mirror device is not set up to update its firmware automatically (see Updating My Cloud Mirror Automatically), you can manually check for an update and install it.

Use the following steps to check for an update and install it:

    1. Click Settings.
    2. In the left pane, click Firmware Update.
    3. In the Available Updates section, click Check for Update.
      At this point, the My Cloud Mirror device checks to see whether updates are available. If the returned status message is "No new updates," you have completed this procedure.
      If the returned status lists an available update, proceed to Step 4.
    4. Click Release Notes to view details about an available update in a new browser window. When you are finished reviewing the release notes, close its browser window.
    5. Click Install Update.
    6. Click OK to confirm the reboot of the My Cloud Mirror device.
      NOTE: Keep in mind that service for any user currently accessing the cloud will be interrupted.
