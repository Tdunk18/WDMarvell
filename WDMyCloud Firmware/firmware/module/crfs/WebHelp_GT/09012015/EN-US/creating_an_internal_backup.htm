Creating an Internal Backup

An internal backup allows you to take existing content and back it up to another storage location on your My Cloud Mirror device.

Use the following steps to create an internal backup.

    1. Click Backups.
    2. Click Internal Backups.
    3. Click Create Job.
    4. In the Job Name field, enter a name for your backup job.
    5. In the Source Folder field, click Browse.
    6. Select the data you’d like to back up, then click OK.
    7. In the Destination Folder field, click Browse.
    8. Select the destination where you’d like to backup your data, then click OK.
    9. In the Recurring Backup field, select  to turn on this function, then enter the following information:
    • Select the frequency of the backup: Daily, Weekly, Monthly
    • Select a time (hour and minutes) from the pull-down menu. 
    1. If you did not enable the Recurring backup field, enter the following information:
    • Click the Date field and select the date you’d like the backup to occur.
    • Select a time (hour and minutes) from the pull-down menu
    1. Click the Incremental Backup checkbox if you want this to be an incremental backup.
    2. Click Create. Your job displays in the Internal Backup Queue area and will begin backing up at the indicated time.
