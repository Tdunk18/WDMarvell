Adding a Share

Use the following steps to create a share:

    1. Click Shares.
    2. Click the Add Share icon
    3. Select the volume where you'd like to create your share from the pull-down menu.
    4. Enter a Share Name.
    5. Enter a Share Description. This is an optional field..
    6. Click to enable the Recycle Bin. The field says On.
    • If you do not want to enable the Recycle Bin, ensure that the field is set to Off.
    1. Click  to enable Media Serving. This allows you to stream media stored in this share to other devices. This is an optional field.
    • If you do not want to enable the Media Serving, ensure that the field is set to Off.
    1. Click  to enable FTP Access. This allows FTP access on this share. This is an optional field.
    • If you do not want to enable the FTP Access, ensure that the field is set to Off.
    1. Click  to enable WebDAV Access. This allows WebDAV access on this share. This is an optional field.
    • If you do not want to enable the WebDAV Access, ensure that the field is set to Off.
    1. Click  to enable NSF Access. This allows Network File System (NSF) access on this share. This is an optional field.
    • If you do not want to enable the NSF Access, ensure that the field is set to Off.
    1. Your new Share is added and displays in the left pane.
