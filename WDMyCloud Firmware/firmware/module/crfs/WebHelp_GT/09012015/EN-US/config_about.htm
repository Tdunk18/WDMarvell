About Setup and Configuration

As the owner and administrator of the My Cloud Mirror device, you can access your media immediately. However, there are some tasks you should complete as part of the initial setup of your connector. See Getting Started for an overview of recommended setup tasks.

As you and your family enjoy your personal cloud, you might have to make changes to your personal cloud from time to time. These types of changes are initiated by clicking Settings. Settings comprise the following categories:

    • General
    • Network
    • Media
    • Utilities
    • Notifications
    • Firmware Update

Each category lists configuration tasks available.
