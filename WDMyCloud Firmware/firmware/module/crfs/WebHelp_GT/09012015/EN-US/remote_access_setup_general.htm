Setting Up Cloud Access

Accessing your cloud remotely from a smartphone, tablet, or computer requires a My Cloud login. (Essentially remote access from a computing device is enabled through the My Cloud central server that gives you access to the NAS devices connected to your home network.) However, the manner in which you access the cloud remotely from mobile devices (smartphones and tablets) and computers differ slightly. Regardless, each user must have a My Cloud login.

Use the following steps to set up remote access.

    1. Ensure that Cloud access is enabled on your device.   How do I do that?

       

      To enable Cloud access:

      1. Click Settings.
      2. In the left pane, click General.
      3. In the Cloud Access area, click to enable the Cloud Service.

       

    2. Click Cloud Access.
    3. In the left pane, click the user to whom you'd like to allow cloud access.
    4. In the My Cloud login for <user name> section, enter an email address if one is not already included, then click .This sends an email to the user with instructions on how to sign up for My Cloud.
    1. Do one of the following:
    • Continue the sign-up process for yourself. How do I do that?

       

      To continue the sign-up process:

      1. Check the email account your provided in Step 4 and view the email notification.
      2. Within the email, click the link provided and follow the instructions to complete sign-up.

       

    • Notify the user that you have set up a web account and have them continue with the sign-up process. How do I do that?
      NOTE: There is another way of giving a mobile device remote access to the My Cloud Mirror drive. See Granting Remote Access to a Mobile Device (Alternate Method).

       

      Contact the user and tell them that you set up a remote access content for them and they need to continue the sign-up process by doing the following:

      1. Check their email account (this is the email address you provided in Step 4).
      2. Open the email notification from My Cloud.
      3. Click the link in the email and follow the instructions provided  to complete sign-up.
