Exécution de tests de diagnostic

Vous pouvez vérifier l'état général de votre dispositif en exécutant l'un des tests suivants :

    1. Rapide : recherche les problèmes de performance majeurs sur votre dispositif et génère une évaluation de réussite ou d'échec. Ce test se réalise en quelques minutes.

    • Complet : teste chaque secteur de votre dispositif et fournit des commentaires sur la condition du dispositif My Cloud Mirror.  Ce test peut prendre plusieurs heures, selon la quantité de données stockées.

Suivez ces étapes pour effectuer un test de diagnostic :

    1. Cliquez sur Paramètres.
    2. Dans le volet gauche, cliquez sur Outils.
    3. Dans la zone Diagnostic système, cliquez sur Test rapide ou Test complet.
      Un indicateur affiche l'état du test et une boîte de dialogue présente les résultats.
