Création d'un groupe

Un groupe représente plusieurs comptes d'utilisateurs rassemblés dans une unité administrable. Suivez ces étapes pour créer un groupe.

    1. Cliquez sur Utilisateurs.
    2. Cliquez sur Groupes.
    3. Cliquez sur l’icône Ajouter des Groupes.
    4. Saisissez un Nom de groupe pour votre nouveau groupe. Le nom peut contenir 16 caractères, mais pas d'espace.
    5. Sélectionnez les utilisateurs que vous souhaitez ajouter au groupe.
    6. Cliquez sur Appliquer. Votre nouveau groupe est créé et s'affiche dans la colonne gauche de l'écran.
