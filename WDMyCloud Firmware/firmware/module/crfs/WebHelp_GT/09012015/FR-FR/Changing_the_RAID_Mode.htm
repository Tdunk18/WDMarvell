Modification du mode RAID

Suivez les étapes suivantes pour modifier le mode RAID de votre dispositif My Cloud Mirror.

Avertissement : Toute modification du mode RAID supprimera toutes vos données.

    1. Cliquez sur Stockage.
    2. Cliquez sur RAID.
    3. Cliquez sur Modifier le mode RAID en bas de l'écran.
    4. Sélectionnez le mode RAID que vous souhaitez appliquer au serveur My Cloud Mirror. Les options incluent :

Mode

Description

JBOD

L'utilisation d'un ou de plusieurs lecteurs n'étant pas en configuration RAID, mais gérés en tant que volumes logiques séparés.

Fractionnement

Combinaison de lecteurs en ligne pour créer un grand volume logique.

RAID 0

Les données sont agrégées par bande sur plusieurs disques durs, permettant une lecture et un enregistrement accélérés des données en combinant le fonctionnement d'au moins deux lecteurs pour augmenter les performances.

RAID 1

Deux copies instantanées des données sont enregistrées sur des disques durs séparés. En cas de défaillance de l'un d'eux, les autres continuent à lire et enregistrer les données jusqu'à ce que le disque dur défectueux soit remplacé.

    1. Cochez la case Basculer vers <RAID Mode>.
    2. Cliquez sur Suivant.
    3. Vérifiez l'état de vos disques durs, puis cliquez sur Suivant.
    4. Cliquez sur Terminer.
    5. À l'aide du curseur, sélectionnez l'espace que vous souhaitez dédier au mode RAID sélectionné.

      REMARQUE : Si vous choisissez de ne pas utiliser la totalité du volume pour votre sélection RAID, vous pouvez fractionner l'espace restant du disque en cochant la case Fractionner l'espace disque restant.
    1. Cliquez sur Suivant.
    2. Sélectionnez pour activer l'option Auto-restauration de votre configuration RAID, puis cliquez sur Suivant.

      REMARQUE : Cet écran ne s'affiche pas pour les modes JBOD, fractionnement et RAID 0.
    1. Vérifiez les modifications de la configuration RAID en vous assurant que les modifications réalisées sont correctes, puis cliquez sur Suivant.
    2. Révisez les écrans d'avertissement.
      Avertissement : Le changement de mode supprimera toutes vos données.
    1. Cliquez sur Terminer. L'application des modifications demandées relatives au disque dur commence.
    2. Cliquez sur Terminer. Vos disques durs sont désormais reconfigurés selon le mode sélectionné.
