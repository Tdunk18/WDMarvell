Activation de la redirection de port

Si l'accès distant à votre cloud est activé via une connexion relais, vous pouvez améliorer les performances réseau en activant la redirection de port.
Comment savoir si ma connexion d'accès mobile est une connexion relais ?

Si votre connexion est une connexion relais, le champ État de la connexion affiche « Connexion (Relais) ». Pour plus de détails concernant la vérification de cet état, reportez-vous à Vérification de l'état de l'accès distant.

 

Important : Avant de commencer cette procédure, vous devez connaître la plage valide de numéros de ports que vous pouvez utiliser pour votre routeur et sélectionner les ports appropriés pour les protocoles HTTP et HTTPS. La plage de numéros de port est 1024 - 66535. En règle générale, le port 80 est affecté au protocole HTTP et le port 443 au protocole HTTPS.

Suivez ces étapes pour activer la redirection de port.

    1. Cliquez sur Paramètres.
    2. Dans le volet gauche, cliquez sur Général.
    3. Dans la section Accès au cloud, cliquez sur pour activer l'accès au cloud, le cas échéant.
    4. Cliquez sur Configurer.
    5. Dans le champ Connectivité, cliquez sur Manuel.
    6. Dans le champ Port externe 1, entrez le numéro du port qui sera utilisé pour les sessions HTTP.
    7. Dans le champ Port externe 2, entrez le numéro du port qui sera utilisé pour les sessions HTTPS.
    8. Cliquez sur Appliquer.
    9. Configurez votre routeur de façon à utiliser les ports indiqués aux étapes 6 et 7.
