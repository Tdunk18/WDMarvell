Bienvenue

Votre dispositif My Cloud Mirror est un système très performant qui offre un accès sans fil sécurisé à vos contenus multimédias et à vos fichiers depuis n’importe quel ordinateur, tablette ou smartphone, lorsqu’il est connecté à un routeur sans fil. Votre dispositif My Cloud Mirror est un composant central de votre cloud personnel et propose une interface simple et robuste pour une configuration, un partage et une maintenance faciles.

Ce que vous devez savoir

Pour tirer le meilleur parti de votre cloud personnel, vous devez connaître quelques points essentiels concernant votre dispositif My Cloud Mirror :

    • Rôles de l'utilisateur

En outre, vous pouvez utiliser les tâches suivantes afin de profiter au mieux des fonctionnalités principales du dispositif :

    • Accès depuis un smartphone ou une tablette

    • Accès depuis un ordinateur portable  

    • Affichage de l’interface Web de My Cloud Mirror

    • Partage de contenu avec d'autres personnes en dehors de votre réseau

    • Chargement de fichiers

    • Mise en route (configuration de base)

Rôles de l'utilisateur dans votre cloud personnel

Toute personne accédant à votre cloud personnel doit posséder un compte utilisateur pour le lecteur My Cloud Mirror. En tant que propriétaire My Cloud Mirror, vous possédez un compte d’utilisateur spécial (admin) qui vous octroie des privilèges d’administration. Grâce à ces privilèges, vous pouvez configurer le dispositif selon vos besoins spécifiques et ajouter des utilisateurs supplémentaires à votre cloud personnel. Vous pouvez déterminer exactement qui peut accéder au dispositif.

REMARQUE : Par défaut, le nom d'utilisateur admin n'a pas de mot de passe. Vous pouvez en créer un à tout moment.

Important : Vous avez la possibilité de modifier le nom de l’administrateur et de définir un mot de passe facultatif lors de l’exécution du logiciel de configuration My Cloud Mirror ou des écrans Mise en route. Veillez à conserver ces nom et mot de passe. Il vous sera impossible d’accéder à l’interface Web My Cloud Mirror sans ces informations d’identification.

Accès au contenu depuis un smartphone ou une tablette

Deux applications mobiles WD vous permettent d’accéder au contenu de votre dispositif My Cloud depuis votre smartphone ou tablette :

    • Albums My Cloud : visualisez et gérez vos photos.

    • My Cloud : visualisez des photos, reproduisez des vidéos, de la musique et lisez des fichiers.

Obtenez ces applications gratuites depuis le centre d'apprentissage WD (http://www.wd.com/setup/wdmycloudmirror), ainsi que sur les sites d'applications tels que l'App Store d'Apple, Google Play, Windows Store et Amazon.com.

Votre parcours d'accès depuis un dispositif mobile (administrateur)

Flux de travail de base en tant que propriétaire/administrateur My Cloud Mirror :

1

Configurez le dispositif My Cloud Mirror pour l’accès distant et inscrivez-vous pour obtenir un identifiant My Cloud. Procédure

2

Téléchargez les applications mobiles WD. Procédure

3

Lancez l'application WD et effectuez une seule et unique configuration pour sécuriser l'accès. Procédure

4

Voilà ! Lisez et affichez votre contenu.

Parcours d'accès depuis un dispositif mobile pour les autres utilisateurs

REMARQUE : Lorsque vous ajoutez un utilisateur à votre dispositif My Cloud Mirror, vous communiquez également son adresse électronique à My Cloud pour lui créer un identifiant My Cloud. Une fois créé, un e-mail est envoyé à l’adresse électronique du nouvel utilisateur contenant un lien vers My Cloud lui permettant de poursuivre le processus d’inscription.

Parcours de base pour les utilisateurs non administrateurs :

1

Cliquez sur le lien figurant dans le message provenant de My Cloud et poursuivez le processus d’inscription.

2

Téléchargez les applications mobiles WD. Procédure

3

Lancez l'application WD et effectuez une seule et unique configuration pour sécuriser l'accès. Procédure

4

Lisez et affichez le contenu.

Accès au contenu depuis un ordinateur portable

Utilisez l’application bureautique My Cloud pour accéder au contenu stocké sur le dispositif My Cloud Mirror depuis votre Mac ou votre PC. My Cloud vous permet d’accéder à distance aux photos, aux vidéos, aux musiques et aux fichiers stockés sur vos dispositifs WD NAS, y compris sur les lecteurs My Cloud Mirror et My Book Live connectés à votre réseau domestique. Comme pour votre application mobile WD, vous avez besoin d’un identifiant My Cloud pour accéder au contenu.

Téléchargez l’application bureautique My Cloud sur le site http://www.wd.com/setup/wdmycloudmirror.

Pour obtenir des instructions détaillées, consultez Accès au contenu depuis un ordinateur portable.

Affichage de l’interface My Cloud Mirror

L’interface utilisateur du dispositif My Cloud Mirror vous permet de configurer et de gérer votre cloud personnel. L'interface s'affiche dans un navigateur Web s'exécutant sur Mac ou PC. 

Affichage de l'interface Web :

    1. Ouvrez une fenêtre de navigateur.
    2. Dans la barre d'adresse, tapez http://<wdmycloudmirror>;
      Exemple : http://wdmycloudmirror
      REMARQUE : si cette URL ne permet pas d'afficher l'interface Web, essayez de saisir l'URL suivante : http://wdmycloudmirror.local

Partage de contenu avec d'autres personnes en dehors de votre réseau

Pour partager du contenu stocké sur le dispositif My Cloud Mirror, utilisez votre ordinateur et l’application bureautique My Cloud. Pour obtenir des instructions détaillées, consultez Partage de contenu avec d'autres personnes en dehors de votre réseau.
