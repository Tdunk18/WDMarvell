À propos des utilisateurs

Toute personne accédant à votre cloud personnel doit disposer d’un compte d’utilisateur pour utiliser le dispositif My Cloud Mirror. En tant que propriétaire du dispositif, vous possédez un compte d'utilisateur spécial (admin) qui vous octroie des privilèges d'administration. Grâce à ces privilèges, vous pouvez définir et configurer le dispositif en fonction de vos besoins précis et ajouter d'autres personnes à votre cloud personnel. Vous pouvez déterminer exactement qui utilise votre dispositif.

Par défaut, le nom d'utilisateur admin n'a pas de mot de passe. Vous pouvez en créer un à tout moment.

Important : Lors de l’installation initiale de votre dispositif My Cloud Mirror et de l’exécution de son logiciel de configuration ou de l’assistant de mise en route, vous avez la possibilité de modifier le nom d’administrateur et de définir un mot de passe optionnel. Assurez-vous de conserver ce nom et ce mot de passe. Il vous sera impossible d’accéder à l’interface Web My Cloud Mirror sans ces informations d’identification.

En tant qu’administrateur du dispositif My Cloud Mirror, vous pouvez effectuer les opérations suivantes :

    • Ajouter des utilisateurs

    • Ajouter ou modifier des mots de passe

    • Visualiser les informations relatives aux utilisateurs

    • Modifier l'accès d'un utilisateur à un dossier partagé  

    • Supprimer des utilisateurs.

    • Modifier le nom d'utilisateur et le mot de passe d'une personne
