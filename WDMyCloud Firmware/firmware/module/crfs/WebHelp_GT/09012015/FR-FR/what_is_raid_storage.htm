Qu'est-ce que le stockage RAID

La technologie RAID (Regroupement redondant de disques indépendants) vous permet de stocker les mêmes données à des endroits différents, sur plusieurs disques durs, en offrant suffisamment de redondance, de meilleures performances et l'intégrité des données. Il existe différents niveaux d'architecture RAID, chacun proposant une méthode différente de partage et de distribution des données parmi les lecteurs. Votre dispositif My Cloud Mirror vous permet de sélectionner l’un des modes de stockage suivants :

    • JBOD :  l'utilisation d'un ou plusieurs lecteurs n'étant pas en configuration RAID, mais gérés en tant que volumes logiques séparés.
    • Fractionnement : combinaison de lecteurs en ligne pour créer un grand volume logique.
    • RAID 0 : Le mode RAID 0 permet l'entrelacement entre tous les lecteurs du groupe RAID. RAID 0 ne permet pas la redondance de données mais offre les meilleures performances par rapport aux autres niveaux RAID. RAID 0 divise les données en de plus petits segments et entrelace les segments de données sur tous les lecteurs du groupe.
    • RAID 1 : En mode RAID 1, le contrôleur RAID duplique toutes les données d'un lecteur vers un second lecteur du groupe. RAID 1 offre une redondance complète des données, mais double la capacité de stockage requise.
