Accès au contenu depuis un smartphone ou une tablette

Vous pouvez utiliser votre smartphone ou votre tablette pour accéder au contenu stocké sur votre dispositif My Cloud Mirror. Avant cela, vous devez avoir effectué les tâches suivantes : Configuration de l’accès au cloud, Téléchargement des applications mobiles WD et Exécution d’une configuration unique pour obtenir un accès sécurisé.

Suivez ces étapes pour accéder au contenu depuis un smartphone ou une tablette :

    1. Assurez-vous que votre dispositif mobile dispose d'une connexion Wi-Fi ou 3G/4G.
    2. Lancez une application mobile WD.
    3. Si vous y êtes invité(e), appuyez sur Se connecter.
    4. Sélectionnez votre lecteur My Cloud Mirror.
    5. Sélectionnez la ressource partagée à ouvrir, puis accédez au contenu de votre choix.
