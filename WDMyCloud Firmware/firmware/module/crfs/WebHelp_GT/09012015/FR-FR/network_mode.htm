Modification du mode réseau

Par défaut, le mode réseau est DHCP, ce qui signifie que le dispositif My Cloud Mirror obtient automatiquement l’adresse IP et d’autres paramètres à partir de votre réseau. Si vous souhaitez définir une adresse IP manuellement, modifiez le mode à l'aide de cette procédure.

Suivez ces étapes pour changer le mode réseau (de DHCP à statique) :

    1. Cliquez sur Paramètres.
    2. Dans le volet de gauche, cliquez sur Réseau.
    3. Dans la section Mode réseau IPv4 de la zone Service réseau, cliquez sur Statique.
    4. Suivez les instructions de l'assistant de configuration du réseau local.
