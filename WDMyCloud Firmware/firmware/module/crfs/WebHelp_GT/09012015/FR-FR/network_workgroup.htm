Spécification d'un groupe de travail

Vous pouvez saisir le nom d’un groupe de travail dans le dispositif My Cloud Mirror.

Qu'est-ce qu'un groupe de travail ?

Un groupe de travail est un nom que vous associez à plusieurs dispositifs informatiques au sein d'un réseau. Les groupes de travail :

    • facilitent l'échange d'informations entre dispositifs informatiques,

    • n'ont pas besoin d'un serveur central,

    • s'appliquent uniquement aux environnements Microsoft Windows.

 

Suivez ces étapes pour spécifier un groupe de travail :

    1. Cliquez sur Paramètres.
    2. Dans le volet de gauche, cliquez sur Réseau.
    3. Dans la zone Groupe de travail réseau, saisissez le nom du groupe de travail auquel vous souhaitez accéder.
