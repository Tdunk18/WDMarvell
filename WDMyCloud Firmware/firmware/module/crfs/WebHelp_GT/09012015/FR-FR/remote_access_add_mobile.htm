Ajout d'un dispositif mobile

Vous pouvez accéder aux contenus du cloud depuis un dispositif mobile (tel qu’un smartphone ou une tablette) grâce aux Albums MyCloud et aux applications My Cloud. Avant de commencer la procédure décrite ici, téléchargez ces applications gratuites sur votre dispositif mobile.

Pour ajouter un dispositif mobile, procédez comme suit :

  1. Activez l'accès distant à votre cloud pour tous les utilisateurs, si vous ne l'avez pas déjà fait.
  2. Cliquez sur l'icône Accès à distance.
  1. Dans le volet gauche, cliquez sur le nom de l'utilisateur qui possède le dispositif mobile auquel vous souhaitez accorder l'accès au cloud.
  1. Cliquez sur Obtenir un code. My Cloud Mirror affiche un code que vous devez saisir sur votre dispositif mobile dans un délai de 24 heures. Notez ce code pour ne pas l'oublier.
  1. Cliquez sur Terminer.
  2. Saisissez le code d'activation sur votre dispositif mobile.
    Une fois le dispositif activé, accédez au contenu de votre cloud via les applications My Cloud.
