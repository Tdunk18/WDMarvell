Ajout d'un compte utilisateur

Suivez ces étapes pour ajouter un utilisateur à votre dispositif My Cloud Mirror :

    1. Cliquez sur Utilisateurs.
    2. Cliquez sur l'icône Ajouter un utilisateur :
    3. Saisissez un Nom d'utilisateur. Ce nom sert à se connecter au système.
    4. Saisissez un Prénom. Il s'agit du prénom de l'utilisateur.
    5. Saisissez un Nom. Il s'agit du nom de l'utilisateur.
    6. Saisissez une adresse électronique. Il s'agit de l'adresse e-mail de l'utilisateur. Cette adresse sert également à la création du compte My Cloud.
    7. Saisissez un mot de passe pour le compte utilisateur.
    8. Saisissez à nouveau le mot de passe pour le confirmer.
    9. Cliquez sur Appliquer.
    10. Cliquez sur . Ceci active le champ Mot de passe.
    • Si vous ne souhaitez pas attribuer de mot de passe, assurez-vous que le champ Mot de passe est désactivé, puis cliquez sur Suivant.
