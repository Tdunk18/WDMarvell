Création d'une sauvegarde à distance

La sauvegarde à distance permet d’effectuer une sauvegarde de votre dispositif My Cloud Mirror sur un autre serveur, sur ou hors de votre réseau.  Suivez ces étapes pour créer une sauvegarde à distance de votre serveur My Cloud Mirror.

    1. Cliquez sur Sauvegarde.
    2. Cliquez sur Sauvegarde à distance.
    3. Cliquez sur le bouton Créer une tâche.
    4. Saisissez un nom de tâche.
    5. Sélectionnez un serveur dans le menu déroulant Serveur distant.
    6.  Dans le champ Adresse IP distante, saisissez l'adresse IP du dispositif de sauvegarde à distance.
      (192.168.1.16, par exemple)
    7. Dans le champ Mot de passe, saisissez le mot de passe du dispositif de sauvegarde à distance.
    8. Saisissez un Nom d’utilisateur SSH.
    9. Dans le champ Mot de passe SSH, saisissez le mot de passe SSH.
    10. Cliquez sur Parcourir pour sélectionner un dossier source.
    11. Dans la liste des dossiers Sauvegarde à distance, cochez un ou plusieurs répertoires de partage à protéger.
    12. Cliquez sur OK.
    13. Pour sélectionner un répertoire Sauvegarde à distance, cliquez sur Parcourir.
    14. Choisissez un répertoire et cliquez sur OK.
    15. Choisissez le type de sauvegarde dans le menu déroulant.
    16. Pour définir une programmation de sauvegarde, cliquez sur l’icône bascule située en regard de Sauvegarde périodique.
    17. Pour définir le chiffrement du transfert, cliquez sur l’icône bascule .
    18. Cliquez sur Créer.
