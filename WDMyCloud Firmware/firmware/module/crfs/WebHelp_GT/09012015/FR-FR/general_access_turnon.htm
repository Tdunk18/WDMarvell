Activation/désactivation de l'accès au cloud

Si vous êtes administrateur, vous pouvez activer ou désactiver l’accès au cloud de tous les utilisateurs (autrement dit, vous pouvez déterminer les dispositifs informatiques autorisés à accéder au contenu présent sur le dispositif My Cloud Mirror).

Suivez ces étapes pour activer/désactiver l'accès au cloud.

    1. Cliquez sur Paramètres.
    2. Dans le volet gauche, cliquez sur Général.
    3. Dans le champ Service du cloud sous Accès au cloud, cliquez sur pour activer ou désactiver le service du cloud :
    • Activé : active l'accès distant pour tous les utilisateurs.
    • Désactivé : désactive l'accès distant pour tous les utilisateurs.

Pour plus d'informations sur la désactivation de l'accès pour un seul dispositif, consultez Suppression d'un dispositif.

REMARQUE : Des informations concernant la zone Options de connexion sont disponibles dans la rubrique Configuration des options de connexion, qui décrit les différentes options de pare-feu et explique comment améliorer les performances. Par défaut, My Cloud Mirror établit automatiquement une connexion directe entre vos dispositifs mobiles et le routeur.

Si l'accès mobile est activé via une connexion relais, (c'est-à-dire, si le champ État de la connexion affiche « Connecté (Relais) »), vous pouvez améliorer les performances en activant la redirection de port sur votre connecteur WD Venue (reportez-vous à Activation de la redirection de port).
