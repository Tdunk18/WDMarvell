Partage de contenu avec d'autres personnes en dehors de votre réseau

Pour partager des contenus avec des utilisateurs extérieurs au réseau, exécutez l’application bureautique My Cloud sur votre ordinateur.

Suivez ces étapes pour partager des contenus :

    1. Si vous ne l’avez pas encore fait, téléchargez l’application bureautique My Cloud depuis le centre d’apprentissage WD (http://www.wd.com/setup/wdmycloudmirror), puis installez-la sur votre Mac ou votre PC.
    2. Sur votre ordinateur, lancez l'application bureautique My Cloud.
    3. Cliquez deux fois sur le dispositif My Cloud Mirror.
    4. Dans le volet gauche, développez les dossiers si nécessaire pour afficher le contenu que vous souhaitez partager.
    5. Dans le volet droit, cliquez sur un élément (ou appuyez sur Maj et cliquez pour sélectionner plusieurs éléments).
    6. Effectuez un clic droit sur l’élément sélectionné, puis cliquez sur E-mail.
    7. Complétez dument la fenêtre E-mail et envoyez le message.
