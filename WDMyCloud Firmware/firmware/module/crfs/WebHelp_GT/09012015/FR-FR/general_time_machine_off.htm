Désactivation des sauvegardes Time Machine

Si vous avez déjà configuré une sauvegarde du dispositif My Cloud Mirror à l’aide d’Apple Time Machine, vous pouvez interrompre ces sauvegardes à tout moment.

Suivez ces étapes pour désactiver les sauvegardes Time Machine.

    1. Cliquez sur Paramètres.
    2. Dans le volet gauche, cliquez sur Général.
    3. Dans la zone Sauvegardes Mac, cliquez sur pour désactiver les sauvegardes Time Machine.
