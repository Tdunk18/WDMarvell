Qu'est-ce qu'une sauvegarde sur le cloud ?

Une sauvegarde sur le cloud vous permet de créer des sauvegardes distantes accessibles par Internet et via divers appareils mobiles. Le dispositif My Cloud Mirror utilise les services du cloud suivants pour créer des sauvegardes à distance :

    • ElephantDrive : ElephantDrive est un service de sauvegarde sur le cloud qui fournit à distance des sauvegardes sûres et automatiques de vos fichiers.
    • Amazon S3 : Amazon Simple Storage Service (S3) est un service Web de stockage de fichiers en ligne qui peut être utilisé pour stocker et récupérer toute quantité de données, à tout moment, depuis n'importe où sur le Web.
