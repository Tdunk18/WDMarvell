À propos de la configuration

En tant que propriétaire et administrateur du dispositif My Cloud Mirror, vous pouvez accéder à votre contenu multimédia immédiatement. Néanmoins, vous devez effectuer certaines tâches lors de la configuration initiale de votre connecteur. Pour connaître les tâches de configuration recommandées, consultez Mise en route.

Il se peut que vous ayez de temps à autre à apporter des modifications à votre cloud personnel, lors de son utilisation. Pour effectuer ces modifications, cliquez sur Paramètres. Le menu Paramètres comprend les catégories suivantes :

    • Général
    • Réseau
    • Multimédia
    • Outils
    • Notifications
    • Mise à jour du micrologiciel

Chaque catégorie répertorie les tâches de configuration disponibles.
