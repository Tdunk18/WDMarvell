Configuration des modes d'économie d'énergie

Réduisez la consommation d’énergie de votre dispositif My Cloud Mirror en suivant ces étapes.

    1. Cliquez sur Paramètres.
    2. Cliquez sur Général.
    3. Dans la zone Économiseur d'énergie, sélectionnez les options suivantes pour définir la consommation d'énergie de votre dispositif.

Mode

Description

Mise en veille du lecteur

Active/désactive le mode veille de votre dispositif. Lorsqu'il est activé, le mode Mise en veille du lecteur l'est aussi.

  • Cliquez sur pour activer/désactiver la mise en veille du lecteur.

LED

Active/désactive les LED de votre dispositif. Lorsque l'option est activée, les LED sont désactivées.

  • Cliquez sur pour activer/désactiver les LED.

Rétablissement de l'alimentation

Redémarre automatiquement votre dispositif après un arrêt inattendu dû à une panne de courant. Lorsque l'option est activée, le rétablissement de l'alimentation l'est aussi.

  • Cliquez sur pour activer/désactiver le rétablissement de l'alimentation.

Calendrier d'arrêt

Vous permet de définir un calendrier indiquant quand le dispositif doit s'arrêter.

  • Cliquez sur pour activer/désactiver le calendrier d'arrêt.
  • Cliquez sur Configurer pour définir le calendrier du dispositif.

Délai d'expiration de l'accès Web dépassé

Choisissez l'heure par défaut à laquelle le dispositif se déconnecte automatiquement.

  • Sélectionnez l'heure d'expiration du délai de l'accès Web dans le menu déroulant.
