Blocage/Déblocage de l'accès pour un dispositif DLNA

En tant qu'administrateur, vous pouvez bloquer ou débloquer l'accès d'un dispositif DLNA à votre cloud. Pour plus d’informations sur la norme DLNA, consultez À propos de My Cloud Mirror et de DLNA.

Suivez ces étapes pour bloquer/débloquer l'accès au dispositif DLNA.

    1. Cliquez sur Paramètres.
    2. Dans le volet de gauche, cliquez sur Multimédia.
    3. Dans le champ Streaming multimédia, cliquez sur Afficher les lecteurs multimédia.
    4. Dans la boîte de dialogue Options de streaming multimédia, localisez les dispositifs dont vous souhaitez modifier l'accès.
    5. Cliquez sur le commutateur :
    • Activé : autorise l'accès.
    • Désactivé : refuse l'accès.
    1. Cliquez sur Fermer.
