Vérification de l'état de l'accès distant

À tout moment, vous pouvez vérifier si :

    • le dispositif My Cloud Mirror est configuré pour un accès distant
    • l’accès distant au dispositif My Cloud Mirror est activé via une connexion directe ou relais
    • Le dispositif mobile d'un utilisateur en particulier est actuellement activé pour accéder à votre cloud à distance

Suivez ces étapes pour vérifier l'état de votre accès mobile :

    1. Cliquez sur Accès au cloud.
    2. Dans le volet gauche, cliquez sur un nom d'utilisateur.
      Le tableau suivant répertorie les champs liés à l'accès mobile et leurs valeurs possibles :

Section

Champ/Nom d'option

Valeurs et description

Profil de l'accès au cloud

État de la connexion

  • Connecté (Direct) : l'accès mobile est activé via une connexion directe.

  • Connecté (Relais) : l'accès mobile est activé via une connexion relais.

  • Échec : My Cloud Mirror ne parvient pas à établir de connexion par accès distant.

Dispositifs du cloud pour nom d'utilisateur

Nom du dispositif ou de l'application

  • Connecté : le dispositif dispose d'un accès distant à votre cloud.

  • En attente : en attente de l'activation du dispositif.

  • Code expiré : code d'activation expiré.

  • Désactivé : l'option Service cloud est désactivée. Consultez Activation/désactivation de l'accès au cloud.

  • Échec de la connexion : échec de la connexion à distance. Vous pouvez tenter de résoudre le problème en vérifiant votre connexion réseau et en vous assurant que la dernière version du micrologiciel est installée.

 

REMARQUE : Pour afficher les paramètres d'accès mobile d'un autre utilisateur, cliquez sur son nom d'utilisateur dans la colonne de gauche. L'affichage est actualisé, et les informations relatives au dispositif du cloud de cet utilisateur apparaissent.
