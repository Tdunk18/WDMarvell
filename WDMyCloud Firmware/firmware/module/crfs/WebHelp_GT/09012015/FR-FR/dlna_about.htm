À propos du dispositif My Cloud Mirror et de DLNA

Vous pouvez configurer le dispositif My Cloud Mirror de manière à pouvoir lire/visionner des fichiers sur votre téléviseur grand écran, à condition de disposer d’un téléviseur certifié DLNA. DLNA (Digital Living Network Alliance) est une norme utilisée par les fabricants de matériel électronique grand public, qui permet aux dispositifs de divertissement de partager du contenu entre eux sur un réseau domestique. Les PC et les téléphones mobiles sont également des dispositifs DLNA.

Avec le dispositif My Cloud Mirror, il vous suffit d’activer la fonctionnalité Serveur DLNA pour assurer la connexion avec les dispositifs DLNA.  Procédure

Reportez-vous à la documentation relative à votre dispositif DLNA afin de le configurer l'accès au réseau.

Si vous constatez une incohérence quant à la quantité de contenu accessible via votre dispositif DLNA, pensez à restaurer la base de données du serveur DLNA. Cette tâche et les informations essentielles concernant son utilisation sont décrites dans la rubrique Nouvelle analyse/Restauration de la base de données DLNA.
