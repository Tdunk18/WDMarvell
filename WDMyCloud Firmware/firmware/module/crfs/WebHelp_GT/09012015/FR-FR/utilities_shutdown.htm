Arrêt de My Cloud Mirror

Important : L’arrêt du dispositif My Cloud Mirror interrompt le service pour tout utilisateur accédant à votre cloud à ce moment-là.

Suivez ces étapes pour arrêter le dispositif My Cloud Mirror :

    1. Cliquez sur Paramètres.
    2. Dans le volet gauche, cliquez sur Outils.
    3. Dans la zone Maintenance du dispositif, cliquez sur Arrêt.
