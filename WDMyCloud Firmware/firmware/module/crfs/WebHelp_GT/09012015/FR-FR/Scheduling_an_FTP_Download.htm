Planification d'un téléchargement FTP

L’application de téléchargement FTP vous permet de planifier un téléchargement FTP sur votre dispositif My Cloud Mirror. Suivez ces étapes pour planifier un téléchargement FTP.

    1. Cliquez sur Applications.
    2. Cliquez sur Téléchargements FTP.
    3. Cliquez sur Créer une tâche.
    4. Dans le champ Nom de tâche, saisissez un nom de tâche de sauvegarde FTP.
    5. Dans le champ Hôte, saisissez la valeur de l’hôte.
    6. Dans le champ Méthode de connexion, sélectionnez la méthode que vous souhaitez utiliser pour effectuer un téléchargement FTP :
    • Compte : Saisissez le nom d'utilisateur et le mot de passe du compte.
    • Anonyme
    1. Dans le champ Type, sélectionnez le type d'informations que vous souhaitez télécharger.
    • Fichier
    • Dossier 
    1. Dans le champ Dossier source, cliquez sur Parcourir pour sélectionner le répertoire source. 
    2. Dans le champ Dossier de destination, cliquez sur Parcourir pour sélectionner le répertoire de destination.
    3. Dans le champ Langue, sélectionnez la langue prise en charge par le client FTP dans le menu déroulant.
    4. Cliquez sur s'il s'agit d'une sauvegarde périodique.
    • Sélectionnez la fréquence de sauvegarde (Quotidienne, Hebdomadaire, Mensuelle).
    • Saisissez l'heure, le jour et la date de la sauvegarde, selon la fréquence sélectionnée.
    1. Saisissez la date à laquelle vous souhaitez effectuer le téléchargement (mm/jj/aaaa), ou cliquez sur pour sélectionner une date.
    2. Saisissez l'heure du téléchargement dans les menus déroulants.
    3. Cliquez sur Créer. Votre téléchargement est créé et ajouté à la file d'attente de téléchargements FTP.
