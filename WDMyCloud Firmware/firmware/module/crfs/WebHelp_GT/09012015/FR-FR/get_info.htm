Obtention d'informations sur le produit

My Cloud Mirror à tout moment (y compris les informations de contact).

Suivez ces étapes pour obtenir les informations sur le produit :

    1. Cliquez sur dans l'angle supérieur droit.
    2. Dans le menu déroulant, cliquez sur Assistance.
    3. Dans la section Ressources d'assistance, sélectionnez un lien vers l'information que vous souhaitez obtenir.
