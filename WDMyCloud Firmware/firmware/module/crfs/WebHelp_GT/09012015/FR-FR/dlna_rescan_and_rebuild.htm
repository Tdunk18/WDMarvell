Nouvelle analyse/Restauration de la base de données DLNA

Si vous observez un décalage entre le contenu My Cloud Mirror accessible ou visible via un dispositif DLNA, envisagez les options suivantes concernant la base de données du serveur DLNA :

    • Relancer l’analyse : My Cloud Mirror recherche les modifications apportées à vos bibliothèques multimédias. Essayez d'abord ceci.
    • Restaurer : My Cloud Mirror restaure intégralement la base de données.
      REMARQUE : Soyez prudent lors de l'exécution de cette procédure. La restauration de la base de données du serveur DLNA peut prendre un certain temps, selon le nombre et la taille des fichiers concernés.

Suivez ces étapes pour effectuer une nouvelle analyse ou restaurer la base de données DLNA :

    1. Cliquez sur Paramètres.
    2. Dans le volet de gauche, cliquez sur Multimédia.
    3. Dans le champ Base de données DLNA, cliquez soit sur Nouvelle analyse, soit sur Restaurer.
