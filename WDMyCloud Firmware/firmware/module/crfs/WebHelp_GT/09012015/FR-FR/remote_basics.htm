Concepts de base de l'accès distant

Vous pouvez accéder au contenu stocké sur le dispositif My Cloud Mirror n’importe où et de trois manières différentes :

    • Depuis un dispositif mobile, tel qu'un smartphone ou une tablette.  
    • Depuis votre ordinateur (plus probablement votre ordinateur portable).  

Présentation de l'accès depuis un dispositif mobile

Suivez ces étapes pour accéder à votre cloud depuis un smartphone ou une tablette :

1

Obtenez un identifiant My Cloud. Cela se produit généralement lorsque vous ajoutez un utilisateur à votre dispositif. Consultez la rubrique Configuration de l'accès au cloud.

2

Téléchargez et installez les applications gratuites WD Photos et My Cloud sur votre dispositif mobile.
Ces applications sont disponibles depuis l'App Store d'Apple et Google Play.

3

Exécutez une configuration unique pour obtenir un accès sécurisé. Consultez la rubrique Exécution d'une configuration unique pour obtenir un accès sécurisé

4

Accédez à votre contenu depuis le dispositif mobile. Consultez la rubrique Accès au contenu depuis un smartphone ou une tablette

Présentation de l'accès depuis un poste de travail Windows ou Mac

Lorsque vous n'êtes pas chez vous (par exemple, si vous êtes dans un café), vous pouvez facilement accéder à votre dispositif depuis votre ordinateur portable. L’application bureautique My Cloud, un ordinateur portable compatible Wi-Fi et une connexion Wi-Fi sont nécessaires pour établir un accès distant.

Suivez ces étapes pour accéder à votre cloud grâce à l’application bureautique My Cloud :

1

Obtenez un identifiant My Cloud. Cela se produit généralement lorsque vous ajoutez un utilisateur à votre dispositif. Consultez la rubrique Configuration de l'accès au cloud.

2

Téléchargez l’application bureautique My Cloud depuis le centre d’apprentissage WD http://wd.com/setup/wdmycloudmirror, puis installez-la sur votre ordinateur.

3

Exécutez une configuration unique pour obtenir un accès sécurisé. Consultez la rubrique Exécution d'une configuration unique pour obtenir un accès sécurisé

4

Accédez à votre contenu depuis votre ordinateur portable. Consultez Accès au contenu depuis un ordinateur portable
