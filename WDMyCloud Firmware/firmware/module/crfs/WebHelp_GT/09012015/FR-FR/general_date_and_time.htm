Configuration de la date et de l'heure

Suivez ces étapes pour définir la date et l’heure de votre dispositif My Cloud Mirror.

Définition du fuseau horaire

    1. Cliquez sur Paramètres.
    2. Dans le volet gauche, cliquez sur Général.
    3. Dans le champ Fuseau horaire, sélectionnez un fuseau horaire dans le menu déroulant.
    4. Cliquez sur Enregistrer.

Synchronisation de la date et de l'heure avec le protocole NTP

    1. Cliquez sur Paramètres
    2. Dans le volet gauche, cliquez sur Général.
    3. Cliquez sur pour activer le service NTP.
    4. Identifiez les serveurs qui seront utilisés par le service NTP. Procédure    

       

      Pour configurer le service NTP, procédez comme suit :

      1. Cliquez sur le lien Configurer se trouvant à côté du champ Serveur principal.
      2. Dans la boîte de dialogue Paramètres NTP, cliquez sur Ajouter un utilisateur NTP.
      3. Indiquez les détails relatifs au serveur, puis cliquez sur Enregistrer.
      4. Répétez les étapes 1 et 2 autant de fois que nécessaire. Vous pouvez indiquer trois serveurs NTP au maximum.
