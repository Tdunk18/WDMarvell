Configuration des notifications

Le dispositif My Cloud Mirror transmet des notifications concernant divers événements liés à votre dispositif, y compris pour indiquer qu’une mise à jour logicielle est disponible, que l’installation d’un micrologiciel est terminée, que la capacité d’un dispositif particulier a été dépassée, etc. Les notifications s'affichent dans un panneau contextuel et sont disponibles par e-mail, selon la configuration de votre dispositif. Elles sont classées par ordre d'importance :

    • Critique : ces types de problèmes doivent être traités immédiatement.
    • Avertissement : Problèmes auxquels vous devez faire attention, mais qui ne sont pas aussi graves que les problèmes critiques.
    • Info : informations détaillées.

En tant qu'administrateur, vous pouvez configurer les éléments suivants :

    • Catégories d'événements pour lesquelles vous souhaitez être notifié par e-mail.
    • Catégories d'événements que vous souhaitez afficher dans l'interface Web.

Suivez ces étapes pour configurer les notifications :

    1. Cliquez sur Paramètres.
    2. Dans le volet gauche, cliquez sur Notifications.
    3. Sélectionnez l'une des options suivantes, ou les deux, et suivez la procédure appropriée :
    • Envoyer des notifications par e-mail.
    • Afficher les notifications dans l'interface utilisateur Web.
