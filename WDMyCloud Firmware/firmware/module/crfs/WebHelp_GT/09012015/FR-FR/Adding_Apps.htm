Ajout d'applications

Le dispositif My Cloud Mirror offre une sélection d’applications que vous pouvez ajouter pour le rendre encore plus productif.

Suivez les étapes suivantes pour ajouter des applications à votre dispositif My Cloud Mirror.

    1. Cliquez sur Applications.
    2. Cliquez sur l’icône Parcourir les applications.
    3. Cochez l'application que vous souhaitez ajouter à votre dispositif, puis cliquez sur Installer. Les applications sélectionnées s'ajoutent à votre dispositif.
