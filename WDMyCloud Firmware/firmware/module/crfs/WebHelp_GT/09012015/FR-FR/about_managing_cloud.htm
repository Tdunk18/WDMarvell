À propos de la gestion de votre cloud personnel

La gestion de votre cloud personnel englobe plusieurs tâches, dont la plupart ne doivent être effectuées qu'occasionnellement. En règle générale, les tâches de gestion de votre cloud seront limitées une fois les tâches décrites à la rubrique Mise en route effectuées.

Cette rubrique répertorie les tâches de gestion de cloud par catégorie :

    • Utilisateurs
    • Partages
    • Accès au cloud

Utilisateurs

Sélectionnez un des liens ci-dessous pour afficher les instructions relatives à l'exécution de tâches utilisateur.

    • Ajouter un compte d’utilisateur 
    • Ajouter ou modifier un mot de passe utilisateur  
    • Afficher les informations de l'utilisateur  
    • Modifier l'accès utilisateur à un dossier partagé  
    • Supprimer un compte d'utilisateur

Accès au cloud

Pour qu’un smartphone ou une tablette puisse accéder au contenu de votre dispositif, vous devez l’enregistrer dans l’écosystème de votre cloud personnel. Générez un code qui permettra d’associer votre dispositif mobile au dispositif My Cloud Mirror. Sélectionnez un des liens ci-dessous pour afficher les instructions relatives à la gestion de votre accès au cloud.

    • Supprimer un dispositif informatique de votre cloud personnel  
    • Vérifier l'état de l'accès distant  
    • Améliorer les performances grâce à la redirection de port
