Affichage des informations de l'utilisateur

Suivez ces étapes pour visualiser les informations relatives au compte utilisateur sélectionné. Elles incluent les privilèges d'accès qui ont été accordés.

    1. Cliquez sur Utilisateurs.
    2. Dans le volet gauche, sélectionnez un utilisateur. Le volet droit affiche les informations suivantes :

Section

Valeurs et description

Profil de l'utilisateur

Cette zone affiche les informations élémentaires de l'utilisateur sélectionné, notamment :

  • Nom d'utilisateur : le nom utilisé pour se connecter au dispositif.

  • Prénom : le prénom de l'utilisateur.

  • Nom : le nom de l'utilisateur.

  • Adresse électronique : l'adresse électronique de l'utilisateur. Elle sert également à accéder à My Cloud.

  • Mot de passe : vous permet d'activer/désactiver le mot de passe de l'utilisateur. Consultez Ajout/modification du mot de passe d'un utilisateur.

  • Adhésion au groupe : indique les groupes auxquels l'utilisateur appartient. Vous pouvez modifier l'adhésion de l'utilisateur à un groupe ici. Consultez Modification de l'adhésion à un groupe.

  • Quotas : Visualisez et définissez les quotas ici. Consultez Définir les quotas d'utilisateur.

Accès au dossier partagé

Affiche les dossiers partagés disponibles, et indique le type d'accès correspondant dont dispose l'utilisateur. Modifiez l'accès de l'utilisateur à un dossier partagé en cliquant sur l'une de ces options :

  •  Accès en lecture/écriture : permet de visualiser et mettre à jour le dossier partagé.

  •  Accès en lecture seule : permet de visualiser le dossier partagé, mais pas de le mettre à jour.

  •  Refuser l'accès. l'utilisateur n'a pas accès à ce dossier partagé.
