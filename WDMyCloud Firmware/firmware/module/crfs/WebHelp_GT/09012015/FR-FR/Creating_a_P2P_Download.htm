Création d'un téléchargement P2P

L’application de téléchargement pair à pair (P2P) vous permet de planifier un téléchargement P2P sur votre dispositif My Cloud Mirror. Suivez ces étapes pour planifier un téléchargement P2P.

    1. Cliquez sur Applications.
    2. Cliquez sur Téléchargements P2P.
    3. Cliquez sur pour activer le téléchargement P2P.
    • Cliquez sur Configurer pour modifier la configuration de téléchargement P2P.
    1. Saisissez l'URL ou le fichier torrent du téléchargement :
    • Si vous disposez d'une URL :

      • Dans le champ Ajouter un torrent depuis une URL, saisissez l'URL du fichier torrent que vous souhaitez télécharger.

      • Cliquez sur Ajouter un torrent. Le téléchargement P2P est créé et ajouté à la file d'attente de téléchargements P2P.

    • Si vous disposez d'un fichier torrent :

      • Sélectionnez Cliquer pour charger un fichier.

      • Sélectionnez un fichier torrent à charger, puis cliquez sur Ouvrir. Le téléchargement P2P est créé et ajouté à la file d'attente de téléchargements P2P.
