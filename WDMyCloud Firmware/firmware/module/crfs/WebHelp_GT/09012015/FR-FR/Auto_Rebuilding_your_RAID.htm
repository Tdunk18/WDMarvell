Auto-restauration de la configuration RAID

L’option Auto-restauration permet de restaurer automatiquement la configuration RAID de votre dispositif My Cloud Mirror en cas de défaillance de disque. Suivez ces étapes pour activer l'option Auto-restauration.

    1. Cliquez sur Stockage.
    2. Cliquez sur RAID.
    3. Vérifiez le champ Auto-restauration pour déterminer si le dispositif My Cloud Mirror est actuellement configuré pour l’auto-restauration.
    4. L'option Auto-restauration est actuellement activée.
    5. L'option Auto-restauration est actuellement désactivée.
    1. Sélectionnez pour activer l'option Auto-restauration si elle est actuellement désactivée.
