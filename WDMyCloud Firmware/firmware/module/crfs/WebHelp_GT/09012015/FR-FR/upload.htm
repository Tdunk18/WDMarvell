Chargement de contenu vers le dispositif My Cloud Mirror

Pour charger des photos, des vidéos, de la musique et des fichiers sur votre dispositif My Cloud Mirror, suivez l’une des procédures ci-dessous :

    • Mac
    • PC Windows
    • USB

Chargement de contenu depuis un Mac

Suivez ces étapes pour charger des fichiers de votre Mac vers votre dispositif My Cloud Mirror :

    1. Sur votre Mac, sélectionnez Accéder > Réseau dans la barre de menus.
    2. Effectuez l'une des actions suivantes :
    • Dans le volet gauche du Finder, cliquez deux fois sur le nom du dispositif My Cloud Mirror.
    • Dans le volet droit, identifiez le nom du dispositif My Cloud Mirror et cliquez deux fois dessus.
    1. Double-cliquez sur le nom d'un dossier partagé pour ouvrir le dossier et afficher tous les sous-dossiers existants. (Le dossier Public est un dossier partagé créé automatiquement.)
    2. Faites glisser les fichiers de votre ordinateur vers le dossier de votre choix.
      Ne déposez pas de fichiers dans les dossiers de sauvegarde WD SmartWare et Time Machine.
      REMARQUE : Si vous copiez les fichiers de votre ordinateur vers le dispositif My Cloud Mirror, ces fichiers résideront à deux endroits : leur emplacement d’origine et votre dispositif.

Chargement de contenu depuis un PC

Suivez ces étapes pour charger des fichiers de votre PC Windows vers le dispositif My Cloud Mirror :

    1. Sur votre ordinateur, ouvrez une boîte de dialogue Exécuter en appuyant simultanément sur les touches Windows et R.
    2. Dans le champ Ouvrir, saisissez \\, puis le nom du dispositif My Cloud Mirror. (Le nom par défaut est mycloudmirror, sans espace.) Exemple : \\mycloudmirror
    3. Cliquez sur OK.
      L'Explorateur Windows ou l'explorateur de fichiers s'ouvre avec la liste des dossiers partagés disponibles dans le volet droit.
    4. Double-cliquez sur le nom d'un dossier partagé pour l'ouvrir et afficher tous ses sous-dossiers. (Le dossier Public est un dossier partagé créé automatiquement.)
    5. Faites glisser les fichiers de votre ordinateur vers ce dossier partagé.
      REMARQUE : Si vous copiez les fichiers de votre ordinateur vers le dispositif My Cloud Mirror, ces fichiers résideront à deux endroits : leur emplacement d’origine et votre dispositif.

     

    Chargement de contenu depuis un lecteur USB

    Suivez ces étapes pour charger des fichiers d’un lecteur USB vers le dispositif My Cloud Mirror :

    1. Connectez le lecteur USB à votre ordinateur pour optimiser les performances.

    2. Connectez votre ordinateur et le dispositif My Cloud Mirror directement à votre routeur via une connexion Ethernet filaire.

    3. Copiez les données de votre lecteur USB externe vers le dispositif My Cloud Mirror.

REMARQUE : Si vous copiez les fichiers de votre ordinateur vers le dispositif My Cloud Mirror, ces fichiers résideront à deux endroits : leur emplacement d’origine et votre dispositif.
