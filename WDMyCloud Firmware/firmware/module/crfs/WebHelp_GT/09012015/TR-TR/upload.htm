My Cloud Mirror Aygıtına İçerik Yükleme

My Cloud Mirror aygıtınıza fotoğraf, video, müzik ve dosyalar yüklemek için aşağıdaki işlemlerden birini kullanabilirsiniz:

    • Mac
    • Windows PC
    • USB

Mac bilgisayardan içerik yükleme

Mac'inizden My Cloud Mirror aygıtınıza dosya yüklemek için aşağıdaki adımları takip edin:

    1. Mac bilgisayarınızda, menü çubuğundan Git > Ağ butonunu seçin.
    2. Aşağıdakilerden birini yapın:
    • Finder'ın sol panelinde My Cloud Mirror aygıtının adına çift tıklayın.
    • Sağ panelde My Cloud Mirror aygıtının adını bulun ve üstüne çift tıklayın.
    1. Klasörü açmak ve mevcut alt klasörleri görüntülemek için belirli bir paylaşımın adına çift tıklayın. (Ortak otomatik olarak oluşturulan bir paylaşımdır.)
    2. Bilgisayarınızdan istediğiniz klasöre dosyaları sürükleyip bırakın.
      Dosyaları Time Machine ve WD SmartWare Yedekleme klasörlerine sürüklemeyin.
      NOT: Eğer bir bilgisayardan My Cloud Mirror aygıtınıza dosya kopyalıyorsanız bu dosyalar iki konumda yer alacaktır: Orijinal konumunda ve aygıtınızda.

PC'den İçerik Yükleme

Windows PC'nizden My Cloud Mirror aygıtına dosya yüklemek için aşağıdaki adımları uygulayın:

    1. Bilgisayarınızda, Windows ve R tuşlarına birlikte basarak Çalıştır iletişim kutusunu açın.
    2. Open (Açık) alanında önce \\ ardından My Cloud Mirror aygıt ismini girin. (Varsayılan ad boşluk olmadan mycloudmirror şeklindedir.) Örnek: \\mycloudmirror
    3. Tamam öğesine tıklayın.
      Windows Explorer veya File Explorer, sağdaki bölmede listelenen kullanılabilir paylaşımlarla açılır.
    4. Klasörü açmak ve mevcut alt klasörleri görüntülemek için belirli bir paylaşımın adına çift tıklayın. (Ortak otomatik olarak oluşturulan bir paylaşımdır.)
    5. Bilgisayarınızdan bu paylaşıma dosyaları sürükleyip bırakın.
      NOT: Eğer bir bilgisayardan My Cloud Mirror aygıtınıza dosya kopyalıyorsanız bu dosyalar iki konumda yer alacaktır: Orijinal konumunda ve aygıtınızda.

     

    USB'den İçerik Yükleme

    Bir USB'den My Cloud Mirror aygıtınıza dosya yüklemek için aşağıdaki adımları uygulayın:

    1. En iyi performans için USB sürücüsünü bilgisayarınıza bağlayın.

    2. Bilgisayarınızı ve My Cloud Mirror aygıtını kablolu bir Ethernet bağlantısı ile doğrudan yönlendiricinize bağlayın.

    3. Harici USB sürücünüzdeki verileri My Cloud Mirror aygıtınıza kopyalayın

NOT: Eğer bir bilgisayardan My Cloud Mirror aygıtınıza dosya kopyalıyorsanız bu dosyalar iki konumda yer alacaktır: Orijinal konumunda ve aygıtınızda.
