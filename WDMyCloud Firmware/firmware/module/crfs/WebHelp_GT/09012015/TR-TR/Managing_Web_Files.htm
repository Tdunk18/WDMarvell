Web Dosyalarını Yönetme

Web Dosya Görüntüleyicisi, My Cloud Mirror aygıtınızdaki dosyaları görüntüleyip yönetmenize imkan tanıyan bir uygulamadır.

Web dosyalarınızı yönetmek için aşağıdaki adımları uygulayın.

    1. Uygulamalar butonuna tıklayın.
    2. Web Dosya Görüntüleyicisi butonuna tıklayın. Aygıtınızdaki dosya birimleri sağdaki bölmede gösterilir.

Web Dosyalarını Görüntüleme

Birimleriniz, klasörleriniz ve dosyalarınız Web Dosya Görüntüleyicisinin sağ bölmesinde gösterilir. Herhangi bir birimin veya klasörün içeriğini görüntülemek için ilgili klasöre çift tıklayın.

Web Dosyalarını Yönetme

Dosyalarınız ve klasörleriniz üzerinde gerçekleştirebileceğiniz eylemler Web Dosya Görüntüleyicisinin sağ üst kısmında gösterilir. Web dosyalarınızda gerçekleştirmek üzere aşağıdaki eylemlerden birini seçin:

Simge

İşlev

Açıklama

Kopyala

Bu eylem seçili bir dosyayı başka bir konuma kopyalamanıza imkan tanır. Bir dosyayı yeni bir konuma kopyalamak için aşağıdaki adımları uygulayın:

    1. Kopyalamak istediğiniz dosyayı seçin.
    2. butonuna tıklayın.
    3. Dosya kopyanızın hedefini seçin.
    4. Tamam butonuna tıklayın. Dosyanız seçili hedefe kopyalanır.

Taşı

Bu eylem bir dosyayı bir konumdan diğerine taşımanıza imkan tanır. Bir dosyayı yeni bir konuma taşımak için aşağıdaki adımları uygulayın:

    1. Taşımak istediğiniz dosyayı seçin.
    2. butonuna tıklayın.
    3. Dosyanızı taşımak istediğiniz hedefi seçin.
    4. Tamam butonuna tıklayın. Dosyanız seçili hedefe taşınır.

Karşıya Yükle

Bu eylem bilgisayarınızdaki veya başka aygıttaki bir dosyayı My Cloud Mirror sunucunuza yüklemenize imkan tanır.
Aygıtınıza bir dosya yüklemek için aşağıdaki adımları uygulayın:

    1. Web Dosya Görüntüleyicisinin sağ bölmesinde karşıya yükleyeceğiniz dosya için hedef seçin.
    2. butonuna tıklayın.
    3. Karşıya Yükle ekranında Dosya Seç simgesine tıklayın
    4. Aygıtınıza yüklemek istediğiniz dosyaya gidin ve dosyayı seçin.
    5. Aç butonuna tıklayın.
    6. Karşıya Yükle butonuna tıklayın.
    • Başka dosya yüklemek isterseniz tüm dosyalarınız karşıya yüklenene kadar adım 3 ve 4'ü uygulayın.
    1. Kapat butonuna tıklayın. Karşıya yüklediğiniz dosyalar hedef klasörde gösterilir.

İndir

Bu eylem My Cloud Mirror aygıtınızdan bilgisayarınıza veya diğer aygıtlara bir dosya indirmenize imkan tanır.
Bilgisayarınıza veya diğer aygıtlara bir dosya indirmek için aşağıdaki adımları uygulayın:

    1. Web Dosya Görüntüleyicisinin sağ bölmesinde indirmek istediğiniz dosyayı seçin.
    2. butonuna tıklayın.
    3. Seçilen dosya bilgisayarınıza veya diğer aygıtlara indirilir.

Yeniden Adlandır

Bu eylem My Cloud Mirror aygıtınızda seçtiğiniz bir dosyayı yeniden adlandırmanıza imkan tanır. Bir dosyayı yeniden adlandırmak için aşağıdaki adımları uygulayın.

    1. Web Dosya Görüntüleyicisinin sağ bölmesinde yeniden adlandırmak istediğiniz dosyayı seçin.
    2. butonuna tıklayın.
    3. Yeniden Adlandır alanında dosyanızı yeniden adlandırın.
    4. Kaydet butonuna tıklayın. Dosyanız yeniden adlandırılır.

Sil

Bu eylem seçilen bir dosyayı My Cloud Mirror aygıtınızdan silmenize imkan tanır. Bir dosyayı silmek için aşağıdaki adımları uygulayın.

    1. Web Dosya Görüntüleyicisinin sağ bölmesinde silmek istediğiniz dosyayı seçin.
    2. butonuna tıklayın.
    3. Seçili dosyayı silmek istediğinizi doğrulamak için Tamam butonuna tıklayın. Seçili dosya, aygıtınızdan silinir.

Yenile

Bu eylem son yenilemeden sonra veya dosya listesine son erişmenizden beri eklenen yeni dosyaları gösterecek şekilde dosya listesini yeniler.

  • butonuna tıklayın. Dosyalarınız yenilenir.
