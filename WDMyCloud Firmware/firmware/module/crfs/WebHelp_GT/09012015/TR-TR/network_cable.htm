My Cloud Mirror Aygıtını Ağınıza Bağlama

Bu prosedür My Cloud Mirror sürücünüzü Ethernet kablosu kullanarak ev ağınıza nasıl bağlayabileceğinizi açıklar. Bu ilk bağlantıyı yaptıktan sonra sürücüyü ağınıza kablosuz olarak bağlanacak şekilde yapılandırabilirsiniz.

Bir Ethernet kablosuyla bağlamak için aşağıdaki adımları uygulayın.

    1. Kablonun bir ucunu My Cloud Mirror aygıtının arkasındaki Ethernet portuna bağlayın.
    1. Kablonun diğer ucunu doğrudan yönlendiriciye veya ağ anahtarı bağlantı noktasına bağlayın.
    2. Adaptörün ucunu sürücünün arkasındaki güç kaynağı yuvasına bağlayın.
    3. Güç adaptörünü güç çıkışına takın. My Cloud Mirror aygıtı otomatik olarak güç almaya başlar.

Önemli: Bağlayıcıyı yapılandırmadan önce My Cloud Mirror aygıtının güç almayı tamamlamasını (yaklaşık 3 dakika) bekleyin.

My Cloud Mirror aygıtını Ethernet kablosu ile ağınıza bağladıktan sonra kablosuz bağlantı kurmak için web arayüzünü kullanabilirsiniz. Bkz. Bağlantı Seçeneklerini Yapılandırma.
