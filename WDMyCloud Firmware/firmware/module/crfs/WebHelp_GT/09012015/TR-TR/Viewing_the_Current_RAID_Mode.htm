Geçerli RAID Modunu Görüntüleme

Aygıtınızda o anda kullanılan RAID modunu görüntülemek için aşağıdaki adımları uygulayın.

    1. Depolama butonuna tıklayın.
    2. RAID butonuna tıklayın.
    3. Sağdaki bölmede aşağıdaki bilgiler gösterilir:
    • RAID Durumu
    • Otomatik Yenileme durumu
    • RAID Birimi
