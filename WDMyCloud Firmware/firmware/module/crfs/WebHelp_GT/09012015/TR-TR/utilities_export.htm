Sistem Ayarlarını Dışa Aktarma

Yönetici olarak, yapılandırma dosyasına mevcut My Cloud Mirror ayarlarınızı kaydedebilirsiniz. Bu işlem, genellikle My Cloud Mirror aygıtının varsayılan orijinal ayarlarını geri yüklemeden önce yapılır (Nasıl Yapabilirim?). My Cloud Mirror aygıtı fabrika ayarlarına döndürüldükten sonra kaydedilen ayarları içe aktarabilirsiniz (Nasıl Yapabilirim?)

Sistem ayarlarınızı dışa aktarmak için aşağıdaki adımları uygulayın:

    1. Ayarlar butonuna tıklayın.
    2. Soldaki bölmeden Yardımcı Programlar butonuna tıklayın.
    3. Sistem Yapılandırma alanında Yapılandırma Dosyasını Kaydet butonuna tıklayın.
