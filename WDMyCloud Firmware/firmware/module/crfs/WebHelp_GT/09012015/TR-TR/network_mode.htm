Ağ Modunu Değiştirme

Varsayılan olarak ağ modu DHCP'dir ve bu, My Cloud Mirror aygıtının ağınızdan IP adresinizi ve diğer ayarları otomatik olarak alacağı anlamına gelir. IP adresini manuel olarak ayarlamak istiyorsanız bu prosedürü tamamlayarak modu değiştirin.

Ağ modunu değiştirmek için (DHCP'den Sabit'e) aşağıdaki adımları uygulayın:

    1. Ayarlar butonuna tıklayın.
    2. Soldaki bölmeden Ağ butonuna tıklayın.
    3. Ağ Hizmeti bölümündeki IPv4 Ağ Modu alanında Sabit butonuna tıklayın.
    4. LAN Kurulum Sihirbazındaki adımları izleyin.
