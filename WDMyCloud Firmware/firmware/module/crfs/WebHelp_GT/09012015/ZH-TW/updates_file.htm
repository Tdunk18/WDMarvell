從檔案更新 My Cloud Mirror 裝置

在某些狀況下，您可能需要先從 Western Digital 支援網站下載檔案再進行安裝，來更新 My Cloud Mirror 裝置。

附註：此程序需要存取網際網路。

使用下列步驟，可從檔案手動更新 My Cloud Mirror 裝置：

    1. 開啟新瀏覽器分頁或視窗。
    2. 在新分頁/視窗中，將您的瀏覽器指向 Western Digital 技術支援網站：
      http://support.wdc.com
    • 如果有可用的新韌體更新，請將它儲存在您電腦裝置的位置上。
    1. 回到 My Cloud Mirror 網頁介面。
    2. 按一下 [設定]。
    3. 按一下左窗格中的 [韌體更新]。
    4. 在 [手動更新] 區域中按一下 [從檔案更新]。
    5. 瀏覽至您在步驟 3 中儲存更新的位置。
    6. 選擇更新檔案，再按一下 [開啟]。
    7. 按一下 [確定] 確認安裝。在安裝更新後，My Cloud Mirror 裝置會重新啟動。
      附註：當 My Cloud Mirror 重新開啟時，任何目前存取雲端的使用者將會被中斷。
