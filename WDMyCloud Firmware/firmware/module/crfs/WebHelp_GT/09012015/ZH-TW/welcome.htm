歡迎使用

您的 My Cloud Mirror 裝置系統是高效能系統，可對連線至無線路由器的任何電腦、平板電腦或智慧型手機，提供對您媒體和檔案的安全無線存取。您的 My Cloud Mirror 裝置是您個人雲端的中央元件，提供簡單、穩固的介面，讓您能輕鬆地進行設定、共享與維護。

您必須知道

為了能充分運用您的個人雲端，您必須知道一些有關 My Cloud Mirror 裝置的重要事項：

    • 使用者角色

此外您應檢閱這些工作，讓您可以利用裝置的核心功能：

    • 智慧型手機或平板電腦存取

    • 筆記型電腦存取

    • 顯示 My Cloud Mirror 網頁介面

    • 與在您網路外部的人共享內容

    • 上傳檔案

    • 開始使用 (基本設定)

個人雲端中的使用者角色

存取您個人雲端的每一人都必須具備 My Cloud Mirror 磁碟機的使用者帳戶。身為 My Cloud Mirror 的擁有者，您具備特殊的使用者帳戶 (admin)，讓您擁有系統管理員權限。有了這些權限，您便可設定裝置以符合您特定的需求，並將其他使用者新增到您的個人雲端。您有能力可確切決定哪些人可以存取裝置。

附註：根據預設，admin 使用者名稱沒有密碼。您可以隨時新增密碼。

重要：在執行 My Cloud Mirror 設定軟體或完成 [開始使用] 畫面時，有機會變更系統管理員名稱並設定選擇性密碼。請務必記下名稱與密碼。若沒有這些認證，您將無法顯示 My Cloud Mirror 網頁介面。

使用智慧型手機或平板電腦存取內容

有兩個 WD 行動應用程式可讓您從智慧型手機或平板電腦存取 My Cloud 裝置的內容：

    • My Cloud 相簿：檢視與管理相片。

    • My Cloud：檢視與播放相片、影片、音樂與檔案。

可從 WD 學習中心 (http://www.wd.com/setup/wdmycloudmirror) 或其他應用程式網站 (如 Apple App Store、Google Play、Windows Store 及 Amazon.com) 下載這些免費應用程式。

您 (系統管理員) 適用的行動裝置存取工作流程

身為 My Cloud Mirror 擁有者/系統管理員的您所適用的基本工作流程：

1

設定 My Cloud Mirror 裝置可供遠端存取並申請 My Cloud 帳號。做法

2

下載 WD 行動應用程式。做法

3

啟動 WD 應用程式並執行安全存取的一次性設定。做法

4

這樣就完成了！播放並檢視您的內容。

其他人的行動裝置存取工作流程

附註：當您新增使用者至 My Cloud Mirror 時，同時也提供使用者的電子郵件地址給 My Cloud，以便建立 My Cloud 帳號。建立好後，會寄送一封電子郵件至新使用者的電子郵件地址，其中包含 My Cloud 的連結，讓他們繼續進行註冊程序。

非系統管理使用者的基本工作流程：

1

按一下來自 My Cloud 之郵件中的連結，並繼續申請程序。

2

下載 WD 行動應用程式。做法

3

啟動 WD 應用程式並執行安全存取的一次性設定。做法

4

播放並檢視內容。

使用筆記型電腦存取您的內容

您可以使用 My Cloud 桌面應用程式，從 Mac 或 PC 存取 My Cloud Mirror 裝置的內容。My Cloud 可讓您遠端存取 WD NAS 裝置上儲存的相片、影片、音樂與檔案，WD NAS 裝置包括與您家用網路連線的 My Cloud Mirror 與 My Book Live 磁碟機。如同 WD 行動應用程式，您需要極重要的 My Cloud 帳號才能存取您的內容。

您可以從 http://www.wd.com/setup/wdmycloudmirror 下載 My Cloud 桌面應用程式。

如需分解步驟指示，請參閱使用筆記型電腦存取內容。

顯示 My Cloud Mirror 介面

My Cloud Mirror 裝置使用者介面可讓您設定與管理您的個人雲端。介面會在您 Mac 或 PC 上執行的網頁瀏覽器中顯示。 

若要顯示網頁介面：

    1. 開啟瀏覽器視窗。
    2. 在網址列中輸入 http://
      ；範例：http://wdmycloudmirror
      附註：如果 URL 無法顯示網頁介面，請嘗試輸入此 URL：http://wdmycloudmirror.local

與在您網路外部的其他人共享內容
