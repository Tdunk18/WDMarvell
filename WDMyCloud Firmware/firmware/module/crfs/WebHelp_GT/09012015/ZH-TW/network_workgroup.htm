指定工作群組

您可在 My Cloud Mirror 裝置中輸入工作群組的名稱。

什麼是工作群組？

工作群組是一個名稱，您將此名稱與網路內的多個電腦裝置產生關聯。工作群組：

    • 有助於在電腦裝置之間輕鬆地交換資訊

    • 不需要中央伺服器

    • 僅套用至 Microsoft Windows 環境

 

使用下列步驟，可指定工作群組：

    1. 按一下 [設定]。
    2. 按一下左窗格中的 [網路]。
    3. 在 [網路工作群組] 區域中，輸入您要加入之工作群組的名稱。
