新增使用者帳戶

遵循下列步驟，將使用者新增至 My Cloud Mirror 裝置：

    1. 按一下 [使用者]。
    2. 按一下 [新增使用者] 圖示：
    3. 輸入使用者名稱。這是用來登入系統的名稱。
    4. 輸入名字。這是使用者的名字。
    5. 輸入姓氏。這是使用者的姓氏。
    6. 輸入電子郵件地址。這是使用者的電子郵件地址。此地址也將用來建立 My Cloud 帳戶。
    7. 輸入使用者帳戶的密碼。
    8. 再次輸入密碼，用以確認。
    9. 按一下套用。
    10. 按一下 。這會將密碼欄位改為 [開啟]，並啟用密碼欄位。
    • 如果不想指派密碼，請務必將密碼欄位設為 [關閉]，並按一下 [下一頁]。
