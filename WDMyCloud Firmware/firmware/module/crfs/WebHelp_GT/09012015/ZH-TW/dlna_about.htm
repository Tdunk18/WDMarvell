My Cloud Mirror 裝置與 DLNA 說明

您可以設定 My Cloud Mirror 裝置，並只需確定您的電視是 DLNA 認證的裝置，就能在大螢幕電視上播放/檢視檔案。DLNA (數位生活網路聯盟) 是消費性電子產品製造商使用的標準，讓整個家用網路上的娛樂裝置能互相共享內容。其他的 DLNA 裝置包括行動電話與 PC。

透過 My Cloud Mirror 裝置，您只要開啟 DLNA 伺服器功能，就能啟用與 DLNA 裝置的連線能力。做法

請參閱文件中有關特定 DLNA 裝置的說明，以設定該裝置存取您的網路。

如果您碰到可透過 DLNA 裝置存取的內容數量不符合時，您可以考慮重新組建 DLNA 伺服器資料庫。重新掃描/重新組建 DLNA 資料庫將說明此工作，包括重要的使用資訊。
