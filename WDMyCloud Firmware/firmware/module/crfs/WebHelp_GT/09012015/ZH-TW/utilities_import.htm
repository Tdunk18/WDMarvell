匯入系統設定

身為系統管理員，您可以匯入先前儲存的組態檔案。基本上，此工作會以先前儲存的組態版本取代目前的版本。

重要：在還原預設系統設定後匯入組態檔，並不會還原使用者與行動裝置資料。您必須再次新增此資訊。完成匯入後，My Cloud Mirror 裝置會重新開啟，這將中斷對目前存取您雲端的使用者所享有的服務。

使用下列步驟，可匯入系統設定。

    1. 按一下 [設定]。
    2. 按一下左窗格中的 [公用程式]。
    3. 在 [系統組態] 區域中按一下 [匯入檔案]。
    4. 選取組態檔的位置，然後按一下 [開啟]。您的組態就會還原。
