手動檢查更新並安裝

如果 My Cloud Mirror 裝置未設定為會自動更新其韌體 (請參閱自動更新 My Cloud Mirror)，您可以手動檢查是否有更新並進行安裝。

使用下列步驟，可檢查是否有更新並進行安裝：

    1. 按一下 [設定]。
    2. 按一下左窗格中的 [韌體更新]。
    3. 在 [可用更新] 區段中，按一下 [檢查更新]。
      此時，My Cloud Mirror 裝置會檢查是否有可用的更新。如果傳回的狀態訊息是「沒有新的更新」，則您已完成此程序。
      如果傳回的狀態列出可用的更新，請前往步驟 4。
    4. 按一下「發行備註」即可以新的瀏覽器視窗檢視可用更新的相關詳細資訊。當您檢閱完發行備註後，關閉其瀏覽器視窗。
    5. 按一下 [安裝更新]。
    6. 按一下 [確定] 確認重新啟動 My Cloud Mirror 裝置。
      附註：請記住，使用者目前存取的任何雲端服務都會被中斷。
