設定雲端存取

必須有 My Cloud 帳號，才能從智慧型手機、平板電腦或電腦遠端存取您的雲端(基本上從電腦裝置進行遠端存取是透過 My Cloud 中央伺服器進行的，它可讓您存取與您家用網路連線的 NAS 裝置)。然而，您從行動裝置 (智慧型手機與平板電腦) 遠端存取雲端的方式與從電腦的存取方式略有不同。無論使用何種裝置，每位使用者都需要有 My Cloud 帳號。

使用下列步驟，可設定遠端存取。

    1. 確認您的裝置已啟用雲端存取。   我要怎麼做？

       

      啟用雲端存取：

      1. 按一下 [設定]。
      2. 按一下左窗格中的 [一般]。
      3. 在 [雲端存取] 區域中，按一下 以啟用雲端服務。

       

    2. 按一下 [雲端存取]。
    3. 在左窗格中，按一下您要允許雲端存取的使用者。
    4. 在 [<使用者名稱> 的 My Cloud 帳號] 區段中，輸入尚未加入之使用者的電子郵件地址，然後按一下 。如此會傳送附帶如何申請 My Cloud 之指示的電子郵件給該使用者。
    1. 執行下列其中一項：
    • 繼續為您自己進行申請程序。我要怎麼做？

       

      若要繼續申請程序：

      1. 檢查您在步驟 4 中提供的電子郵件帳號，並檢視電子郵件通知。
      2. 按一下電子郵件內提供的連結，並依照指示完成申請。

       

    • 通知使用者您已申請網路帳號，並讓使用者繼續申請程序。我要怎麼做？
      附註：有另一個方法可提供 My Cloud Mirror 磁碟機的行動裝置遠端存取權。請參閱「將遠端存取權授與行動裝置 (替代方法)」。

       

      聯絡使用者並告知您已為他們設定遠端存取內容，他們必須執行以下步驟來繼續申請程序：

      1. 檢查其電子郵件帳戶 (這是您在步驟 4 中提供的電子郵件地址)。
      2. 開啟來自 My Cloud 的電子郵件通知。
      3. 按一下電子郵件中的連結，並依照提供的指示完成申請。
