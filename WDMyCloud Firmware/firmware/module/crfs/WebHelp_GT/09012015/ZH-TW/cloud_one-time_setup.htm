執行安全存取的一次性設定

此程序說明如何在您的智慧型手機或平板電腦與 My Cloud Mirror 裝置之間建立安全的連線。使用您特有行動裝置適用的指示：

    • Android 智慧型手機或平板電腦  

    • iPhone 或 iPad

    • Windows 手機或平板電腦

    • Windows 或 Mac 桌上型電腦

取得 Android 智慧型手機或平板電腦的安全存取

    1. 執行以下步驟，將您的智慧型手機或平板電腦連線：

      1. 點選您行動裝置上的 [設定]。

      2. 點選 [無線和網路]。

      3. 點選 [Wi-Fi 設定]。

      4. 選取您的家用網路。

      5. 輸入您的網路密碼，並點選 [連線]。

    1. 啟動 WD 應用程式。
    2. 點選 [現在連線]。
    3. 點選 My Cloud Mirror 裝置名稱。
    4. 點選下拉箭頭，然後選擇您的使用者名稱。
    5. 執行下列其中一項：

    • 如果已為使用者設定密碼，請輸入該密碼，再點選 [登入]。

    • 如果尚未為使用者設定密碼，請點選 [登入]。

    1. (選擇性) 如果您要存取您的任何內容，請點選共享區名稱，然後瀏覽至所需的檔案。

取得 iPhone 或 iPad 的安全存取

    1. 執行以下步驟，將您的智慧型手機或平板電腦連線：

    1. 點選您行動裝置上的 [設定]。

    2. 點選 [Wi-Fi]。

    3. 點選 [Wi-Fi 設定]。

    4. 選取您的家用網路。

    5. 輸入您的網路密碼並點選 [加入]。

    1. 啟動 WD 應用程式。
    2. 點選 [現在連線]。
    3. 點選 My Cloud Mirror 裝置名稱旁的加號 (+)。
    4. 點選所需的使用者名稱。
      如果已為使用者設定密碼，請輸入該密碼。
    5. (選擇性) 如果您要存取您的任何內容，請點選共享區名稱，然後瀏覽至所需的檔案。

取得 Windows 智慧型手機或平板電腦的安全存取

    1. 執行以下步驟，將您的智慧型手機或平板電腦連線：

    1. 點選您電腦裝置上的 [設定]。

    1. 點選 [Wi-Fi]。

    2. 選取您的家用網路。

    3. 輸入您的網路密碼並點選 [完成]。

    1. 啟動 WD 應用程式。
    2. 點選 [現在連線]。
    3. 點選 My Cloud Mirror 磁碟機名稱。
    4. 點選下拉箭頭，然後選取使用者名稱。
      如果已為使用者設定密碼，請輸入該密碼。
    5. (選擇性) 如果您要存取您的任何內容，請執行以下步驟：

    1. 在 [裝置] 畫面上點選 My Cloud Mirror 磁碟機名稱。

    2. 觸碰一個共用，然後瀏覽到所要的內容。
