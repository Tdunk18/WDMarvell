變更 RAID 模式

使用下列步驟變更您 My Cloud Mirror 裝置的目前 RAID 模式。

警告：對 RAID 模式進行變更會刪除所有資料。

    1. 按一下 [儲存裝置]。
    2. 按一下 [RAID]。
    3. 按一下畫面底部的 [變更 RAID 模式]。
    4. 選取您要用於 My Cloud Mirror 伺服器的 RAID 模式。選項包括：

模式

描述

JBOD

使用一個或多個不是 RAID 設定的磁碟機，而是以個別邏輯磁碟區來管理。

跨越

以線性方式結合磁碟機，建立一個大型邏輯磁碟區。

RAID 0

資料等量存放於多個硬碟上，由於結合運用兩個以上磁碟區，因此可以加速讀取和記錄資料，提高效能。

RAID 1

由於會在兩個不同的硬碟上記錄兩個即時資料副本，所以如果一個副本失敗，另一個仍可繼續讀/寫資料，直到故障硬碟獲得更換。

    1. 選取 [切換至 <RAID 模式>] 核取方塊。
    2. 按一下 [下一步]。
    3. 檢閱硬碟狀態，然後按 [下一頁]。
    4. 按一下 [完成]。
    5. 使用 [磁碟區] 滑桿，選取您要專用於選定 RAID 模式的空間量。

      附註：如果選擇不將整個磁碟區用於 RAID 選項，則可選取 [將剩餘磁碟空間設定為跨越] 核取方塊，將剩餘磁碟空間設定為「跨越」。
    1. 按一下 [下一步]。
    2. 選取 開啟 RAID 組態的 [自動重新組建] 選項，然後按一下 [下一步]。

      附註：此畫面不會顯示 JBOD、跨越和 RAID 0 模式
    1. 檢閱對 RAID 進行的變更，確認所做變更都正確，然後按一下 [下一步]。
    2. 檢閱警告畫面。
      警告：當您變更模式時，所有資料都會遭到清除。
    1. 按一下 [完成]。隨即開始進行要求的硬碟變更。
    2. 按一下 [完成]。您的硬碟現已重新設定為選定模式。
