還原預設設定

系統還原可讓系統管理員對 My Cloud Mirror 裝置執行完整出廠還原。此選項可將您的裝置還原為出廠預設設定，且不遺失資料。

執行還原前，您可以先將您的目前組態儲存並匯出 (做法)，然後於日後匯入 (做法)。

重要：進行還原會中斷對目前存取您雲端的使用者所提供的服務。還原正在進行時，請勿拔出 My Cloud Mirror 裝置。

使用下列步驟，可還原 My Cloud Mirror 裝置：

    1. 按一下 [設定]。
    2. 按一下左窗格中的 [公用程式]。
    3. 在 [系統還原] 區域中按一下 [系統還原]。
    4. 檢閱組態訊息，然後按一下 [確定]。裝置會重新啟動。在重新啟動期間，請勿拔除您的裝置。重新啟動完成後，請啟動儀表板。
