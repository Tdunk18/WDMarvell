啟用 ElephantDrive 雲端備份

ElephantDrive 是一種雲端備份服務，可對您的檔案提供安全的遠端自動備份。啟用後，ElephantDrive 會自動與您的設備同步。

使用下列步驟啟用使用 ElephantDrive 的雲端備份。

    1. 按一下 [備份]。
    2. 按一下 [雲端備份]。
    3. 按一下畫面頂端的 [Elephant Drive] 按鈕。
    4. 按一下 。您的 ElephantDrive 雲端備份現已開啟。
    5. 按一下 [註冊]。
    6. 輸入下列資訊，再按一下 [註冊]：
    • 電子郵件地址：輸入要用來接收 ElephantDrive 資訊的電子郵件地址。
    • 密碼：輸入新帳戶的密碼
    • 驗證密碼：再次輸入密碼。

您現在已啟用 ElephantDrive 雲端備份。
