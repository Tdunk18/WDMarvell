一般設定說明

您可以透過 [設定] 選項設定 My Cloud Mirror 裝置環境。

「設定」類別包括「一般」檢視。此處會列出從您可從此檢視中完成的工作。

  • 裝置設定檔

    • 變更裝置名稱  
    • 變更裝置描述

  • 語言與時間

    • 設定 My Cloud Mirror 介面顯示的語言  
    • 設定日期與時間
    • 設定時區  

  • 雲端存取

    • 開啟/關閉遠端存取  
    • 設定連線選項
    • 啟用連接埠轉送  
    • 檢查遠端存取狀態  

  • 能源節省

    • 開啟/關閉裝置正面的 LED 燈  

    • 決定您的裝置何時進入睡眠狀況  

  • Mac 備份

    • 設定 Apple Time Machine 的備份  

    • 關閉 Time Machine 備份
