自動更新 My Cloud Mirror 裝置

您可以設定 My Cloud Mirror 裝置便利地檢查出是否有可用的韌體更新並自動安裝。這是確定 My Cloud Mirror 韌體為最新版的最簡單方法。

附註：自動安裝軟體更新後，My Cloud Mirror 裝置會重新啟動。重新啟動會影響到使用者存取雲端的能力，因此您應排程更新會在使用者存取雲端的可能性最小的時間進行。

使用下列步驟，可自動更新 My Cloud Mirror 裝置。

    1. 按一下 [設定]。
    2. 按一下左窗格中的 [韌體更新]。
    3. 在 [自動更新] 區域中按一下 ，啟用自動更新。
    4. 在 [更新排程] 欄位中，排程更新的日期和時間：

    • 按一下第一個下拉箭頭，選擇週中的日。

    • 按一下第二個下拉箭頭，選擇時間。

    • 按一下第三個下拉箭頭，選取上午或下午。
