Criando um Backup via USB

Use as etapas a seguir para fazer o backup de seus dados no My Cloud Mirror para ou a partir de um dispositivo USB.

    1. Clique em Backups.
    2. Clique em Backups via USB.
    3. Clique em Criar trabalho.
    4. Insira um nome de trabalho para seu backup.
    5. Indique a direção do seu backup a partir do menu suspenso. As opções incluem:
    • USB para NAS: Faz backup dos dados no seu dispositivo USB para o dispositivo My Cloud Mirror.
    • NAS para USB: Faz backup dos dados no seu dispositivo My Cloud Mirror para um dispositivo USB.
    1. Selecione a pasta de origem para o backup:
    • Clique em Procurar e selecione a pasta que você gostaria de fazer o backup do seu dispositivo de destino.
    1. Selecione a pasta de destino para o backup.
    • Clique em Procurar e selecione a pasta na qual você deseja fazer o backup do seu dispositivo de origem.
    1. Indique o Tipo de Backup a partir do menu suspenso.
    2. Para iniciar um backup de USB automaticamente ao se conectar, clique no ícone de alternância .
    3. Clique em Criar. Isso cria o trabalho de backup para o backup solicitado.
    4. Clique em Detalhes na área Trabalhos de Backup via USB para verificar o andamento de seu backup.
