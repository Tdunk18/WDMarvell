Verificando o status do acesso remoto

A qualquer momento, você pode verificar:

    • O dispositivo My Cloud Mirror está configurado para acesso remoto
    • O acesso remoto para o dispositivo My Cloud Mirror é habilitado por uma conexão direta ou restabelecida
    • Um dispositivo móvel específico de um determinado usuário está atualmente ativado para acessar sua nuvem remotamente

Use as etapas a seguir para verificar o status do seu acesso móvel:

    1. Clique em Acesso à nuvem.
    2. No painel esquerdo, clique no nome do usuário.
      A tabela a seguir lista os campos de acesso móvel e seus possíveis valores:

Seção

Nome do campo/opção

Valores e descrições

Perfil de acesso à nuvem

Status da conexão

  • Conectado (direto): O acesso móvel está ativado por meio de uma conexão direta.

  • Conectado (retransmitido): O acesso móvel está ativado por meio de uma conexão com retransmissão.

  • Falhou: My Cloud Mirror não pode estabelecer uma conexão de acesso remoto.

Dispositivos na nuvem para nome do usuário

Nome do dispositivo ou do aplicativo

  • Conectado: O dispositivo possui acesso remoto à sua nuvem.

  • Aguardando: Aguardando a ativação do dispositivo.

  • Código expirado: Código de ativação expirado.

  • Desativado: A opção Serviço de nuvem está desativada. Consulte como Ativar/Desativar o acesso à nuvem.

  • Falha de conexão: A conexão remota falhou. Você pode tentar solucionar o problema resolvendo o problema da sua conexão de rede e garantindo que o firmware mais recente está instalado.

 

OBSERVAÇÃO: Se você quiser visualizar as configurações de acesso móvel para um usuário diferente, clique no nome de usuário da pessoa na coluna esquerda. A exibição é atualizada com as informações do dispositivo da nuvem do usuário.
