Carregando conteúdo para o dispositivo do My Cloud Mirror

Para fazer o upload de fotos, vídeos, música e arquivos para seu dispositivo My Cloud Mirror, use um dos seguintes procedimentos:

    • Mac
    • Windows PC
    • USB

Carregando conteúdo por um Mac

Use as seguintes etapas para carregar arquivos do seu Mac para seu dispositivo My Cloud Mirror:

    1. No seu Mac, escolha Ir > Rede na barra do menu.
    2. Faça uma das seguintes opções:
    • No painel à esquerda do Localizador, clique duas vezes no nome do dispositivo My Cloud Mirror.
    • No painel à direita, localize o nome do dispositivo My Cloud Mirror e clique duas vezes.
    1. Clique duas vezes no nome de um compartilhamento específico para abrir a pasta e exibir todas as subpastas existentes. (Público é um compartilhamento criado automaticamente.)
    2. Arraste e solte arquivo de seu computador para a pasta desejada.
      Não solte arquivos nas pastas Backups do Time Machine e do WD SmartWare.
      OBSERVAÇÃO: Se for copiar arquivos de um computador para o dispositivo My Cloud Mirror, esses arquivos ficarão em dois lugares: no local de origem e no seu dispositivo.

Carregando conteúdo por um PC

Use as seguintes etapas para carregar arquivos do seu PC Windows para o dispositivo My Cloud Mirror:

    1. No seu computador, abra a caixa de diálogo Executar ao pressionar as teclas Windows e R ao mesmo tempo.
    2. No campo Abrir, digite \\ seguido do nome do dispositivo My Cloud Mirror. (O nome padrão é mycloudmirror, sem os espaços.) Exemplo: \\mycloudmirror
    3. Clique em OK.
      O Windows Explorer ou o Explorador de Arquivos é aberto com os compartilhamentos disponíveis listados no painel direito.
    4. Clique duas vezes no nome de um compartilhamento específico para abrir a pasta e exibir todas as subpastas. (Público é um compartilhamento criado automaticamente.)
    5. Arraste e solte arquivo de seu computador para esse compartilhamento.
      OBSERVAÇÃO: Se for copiar arquivos do computador para o dispositivo My Cloud Mirror, esses arquivos ficarão em dois lugares: no local de origem e no seu dispositivo.

     

    Carregando conteúdo de uma unidade USB

    Use as etapas a seguir para fazer o upload dos arquivos de um USB para o dispositivo My Cloud Mirror:

    1. Conecte a unidade USB ao seu computador para melhor desempenho.

    2. Conecte seu computador e o dispositivo My Cloud Mirror diretamente ao roteador com uma conexão de Ethernet instalada.

    3. Copie os dados de sua unidade USB externa para o dispositivo do My Cloud Mirror

OBSERVAÇÃO: Se for copiar arquivos do computador para o dispositivo My Cloud Mirror, esses arquivos ficarão em dois lugares: no local de origem e no seu dispositivo.
