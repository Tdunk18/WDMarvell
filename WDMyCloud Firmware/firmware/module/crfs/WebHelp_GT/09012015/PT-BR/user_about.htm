Sobre Usuários

Todos que acessam a sua nuvem pessoal devem ter uma conta de usuário para usar o dispositivo do My Cloud Mirror. Como o proprietário do dispositivo, você tem uma conta de usuário especial (admin) que lhe concede privilégios de administrador. Com esses privilégios, você pode instalar e configurar o dispositivo de acordo com as suas necessidades específicas, bem como adicionar outras pessoas à sua nuvem pessoal. Você tem o poder de determinar exatamente quem usa seu dispositivo.

Como padrão, o nome de usuário admin não tem uma senha. Você pode adicionar uma a qualquer momento.

Importante: Ao configurar inicialmente seu dispositivo, ao executar o software de instalação do My Cloud Mirror ou o assistente Iniciar, existe a oportunidade de mudar o nome do administrador e definir uma senha opcional. Não se esqueça de guardar este nome e esta senha. Você não será capaz de exibir a interface da Web do My Cloud Mirror sem essas credenciais.

Como administrador do dispositivo My Cloud Mirror, é possível:

    • Adicionar usuários

    • Adicionar ou editar senhas de usuário

    • Exibir informações do usuário

    • Alterar o acesso de um usuário a um compartilhamento  

    • Remover usuários.

    • Alterar o nome de usuário e a senha de alguém
