Conectando o dispositivo My Cloud Mirror à sua rede

Este procedimento explica como conectar a unidade do My Cloud Mirror à sua rede local usando um cabo Ethernet. Depois de fazer a conexão inicial, é possível configurar a unidade para conectar sua rede sem fio.

Use as etapas a seguir para conectar-se com um cabo de Ethernet.

    1. Conecte uma extremidade do cabo na porta Ethernet na parte de trás do dispositivo My Cloud Mirror.
    1. Conecte a outra extremidade do cabo diretamente à porta do comutador de rede ou do roteador.
    2. Conecte uma extremidade do adaptador à fonte de alimentação na parte traseira da unidade.
    3. Conecte o adaptador de alimentação a uma tomada. O dispositivo My Cloud Mirror liga automaticamente.

Importante: Espere que o dispositivo My Cloud Mirror termine de ligar (aproximadamente três minutos) antes de configurar o conector.

Depois de conectar o dispositivo My Cloud Mirror à sua rede por um cabo de Ethernet, é possível usar a interface Web para configurar uma conexão sem fio. Consulte Configurando opções de conexão.
