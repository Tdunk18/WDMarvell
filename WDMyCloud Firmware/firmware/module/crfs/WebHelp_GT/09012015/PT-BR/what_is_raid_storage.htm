O que é um armazenamento de RAID

RAID (Redundant Array of Independent Disks, ou discos independentes de matriz redundante) permite armazenar os mesmos dados em locais diferentes em diversos discos rígidos, oferecendo a redundância necessária, melhor desempenho e integridade de dados. Há vários níveis diferentes de RAID, e cada um oferece um método diferente de compartilhamento ou distribuição de dados entre as unidades. Seu dispositivo My Cloud Mirror permite selecionar a partir dos seguintes modos de armazenamento:

    • JBOD:  O uso de uma ou mais unidades que não deve ser feito em uma configuração de RAID; ele deve ser gerenciado como volumes lógicos individuais.
    • Spanning (Transposição): Combinação de unidades de forma linear para criar um grande volume lógico.
    • RAID 0: O modo RAID 0 oferece registro em disco por todas as unidades no grupo de unidades de RAID. O RAID 0 não oferece redundância de dados, mas oferece o melhor desempenho dos níveis de RAID. O RAID 0 quebra os dados em segmentos menores e registra os segmentos de dados por cada unidade no grupo de unidades.
    • RAID 1: No modo de RAID 1, o controlador de RAID duplica todos os dados de uma unidade para uma segunda unidade no grupo de unidades. O RAID 1 oferece redundância de dados completa, mas dobra a capacidade de armazenamento de dados necessária.
