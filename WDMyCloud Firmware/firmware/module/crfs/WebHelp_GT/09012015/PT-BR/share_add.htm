Adicionando um compartilhamento

Use as etapas a seguir para criar um compartilhamento:

    1. Clique em Compartilhamentos.
    2. Clique no ícone Adicionar compartilhamento
    3. Selecione o volume onde você gostaria de criar um compartilhamento a partir do menu suspenso.
    4. Insira um Nome de Compartilhamento.
    5. Insira uma descrição do compartilhamento. Este é um campo opcional.
    6. Clique em para ativar a Lixeira. O campo diz Ativado.
    • Se você não deseja ativar a lixeira, verifique se o campo esta definido como Desativado.
    1. Clique em para ativar o Serviço de mídia. Isso permite que você faça um streaming de mídia armazenada em outros dispositivos. Este é um campo opcional.
    • Se você não deseja ativar o serviço de mídia, certifique-se que o campo esta definido como Desativado.
    1. Clique em  para ativar o acesso via FTP. Isso permite o acesso via FTP neste compartilhamento. Este é um campo opcional.
    • Se você não deseja ativar o acesso via FTP, certifique-se que o campo esta definido como Desativado.
    1. Clique em para ativar o acesso via WebDAV. Isso permite o acesso via WebDAV neste compartilhamento. Este é um campo opcional.
    • Se você não deseja ativar o acesso via WebDAV, certifique-se que o campo esta definido como Desativado.
    1. Clique em para ativar o acesso via NSF. Isso permite o acesso via Sistema de Arquivo de Rede (NSF) neste compartilhamento. Este é um campo opcional.
    • Se você não deseja ativar o acesso via NSF, certifique-se que o campo está definido como Desativado.
    1. O novo compartilhamento é criado e exibido no painel à esquerda.
