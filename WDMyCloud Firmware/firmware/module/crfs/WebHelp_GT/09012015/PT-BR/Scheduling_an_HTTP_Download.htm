Programando um download via HTTP

O aplicativo de Download via HTTP permite programar um download via http para o dispositivo My Cloud Mirror. Use as etapas a seguir para agendar um download via HTTP.

    1. Clique em Aplicativos.
    2. Clique em Downloads via HTTP.
    3. No campo de URL, informe o URL do arquivo que você gostaria de baixar.
      Exemplo: http://192.168.0.32/test.txt
    4. No campo Salvar para, informe o local para o qual você gostaria de salvar o download ou clique em Procurar para selecionar o local onde você gostaria de salvar o download.
      Exemplo: Volume_1/Public/Shared Videos 
    5. No campo Renomear, digite um novo nome para o arquivo, se quiser alterá-lo.
    6. Clique em se este for um backup recorrente.
    • Selecione a frequência do backup (diariamente, semanalmente ou mensalmente)
    • Insira o horário, dia e data do backup, dependendo da frequência selecionada.
    1. Digite a data em que deseja realizar o download (mm/dd/aaaa) ou selecione  para selecionar uma data.
    2. Insira um horário para seu download dos menus suspensos.
    3. Clique em Criar. Seu download é criado e adicionado à fila de downloads via HTTP.
