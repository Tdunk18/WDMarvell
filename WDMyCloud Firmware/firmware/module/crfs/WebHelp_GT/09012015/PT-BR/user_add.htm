Adicionando uma conta de usuário

Siga as etapas abaixo para adicionar um usuário a seu dispositivo My Cloud Mirror:

    1. Clique em Usuários.
    2. Clique no ícone Adicionar usuário:
    3. Digite um Nome de usuário. Este nome é usado para entrar no sistema.
    4. Informe um Primeiro nome. Este é o primeiro nome do usuário.
    5. Informe um Sobrenome. Este é o sobrenome do usuário.
    6. Informe um endereço de e-mail. Este é o endereço de e-mail do usuário. Este endereço também é usado para criar uma conta do My Cloud.
    7. Informe uma senha para a conta de usuário.
    8. Informe a senha novamente para confirmá-la.
    9. Clique em Aplicar.
    10. Clique em . Isso altera o campo de senha para Ativado e ativa os campos de senha.
    • Se você não deseja atribuir uma senha, garanta que o campo senha esteja desativado e clique em Próximo.
