Sobre o dispositivo My Cloud Mirror e DLNA

É possível configurar o dispositivo My Cloud Mirror para que possa executar/visualizar os arquivos na tela da TV desde que seja um dispositivo certificado por DLNA. O DLNA (Digital Living Network Alliance) é um padrão usado por fabricantes de eletrônicos de consumo para permitir que os dispositivos de entretenimento compartilhem conteúdo com outros por meio de uma rede local. Outros dispositivos DLNA incluem celulares e PCs.

Com o dispositivo My Cloud Mirror, é possível habilitar a conectividade com os dispositivos DLNA simplesmente ao sintonizar o recurso do servidor DLNA.  Como fazer

Veja a documentação para seu dispositivo DLNA particular para configurar para acessar sua rede.

Se você encontrar uma incompatibilidade na quantidade de conteúdo acessível pelo seu dispositivo DLNA, é possível restaurar o banco de dados do servidor DLNA. Esse trabalho, incluindo as principais informações sobre o uso dela, é abordado em Verificando novamente/restaurando o banco de dados DLNA.
