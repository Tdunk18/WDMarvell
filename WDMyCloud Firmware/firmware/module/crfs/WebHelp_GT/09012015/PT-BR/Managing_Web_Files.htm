Gerenciando arquivos da Web

O Web File View é um aplicativo que permite visualizar e gerenciar os arquivos no seu dispositivo My Cloud Mirror.

Use as etapas a seguir para gerenciar seus arquivos da web.

    1. Clique em Aplicativos.
    2. Clique em Visualizador de arquivo da Web. Os volumes de arquivos disponíveis no seu dispositivo são exibidos no painel à direita.

Visualizando arquivos da Web

Seus volumes, pastas e arquivos são exibidos no painel à direita do Web File Viewer. Para visualizar os conteúdos de qualquer volume ou pasta, clique duas vezes na pasta.

Gerenciando arquivos da Web

As ações disponíveis para serem realizadas nos arquivos e pastas exibidos no canto superior direito do Web File Viewer. Selecione uma das ações a seguir para realizar nos seus arquivos da web:

Ícone

Função

Descrição

Copiar

Esta ação permite copiar um arquivo selecionado para outro local. Use as etapas a seguir para copiar um arquivo para um novo local:

    1. Selecione o arquivo que você gostaria de copiar.
    2. Clique em .
    3. Selecione o destino da cópia do seu arquivo.
    4. Clique em OK. Seu arquivo é copiado para o destino selecionado.

Mover

Esta ação permite a movimentação de um arquivo de um local para outro. Use as etapas a seguir para mover um arquivo para um novo local:

    1. Selecione o arquivo que você gostaria de mover.
    2. Clique em .
    3. Selecione o destino para onde você gostaria de mover seu arquivo.
    4. Clique em OK. Seu arquivo é movido para o destino selecionado.

Upload

Esta ação permite fazer o upload de um arquivo do seu computador ou outro dispositivo para o servidor do My Cloud Mirror.
Use as etapas a seguir para carregar um arquivo para o dispositivo:

    1. No painel à direita do Web File Viewer, seus volumes, pastas e arquivos são exibidos.
    2. Clique em .
    3. Na tela de Upload, clique no ícone Selecionar arquivo
    4. Procure e selecione o arquivo que você deseja carregar no dispositivo.
    5. Clique em Abrir
    6. Clique em Upload.
    • Se você deseja carregar mais arquivos, realize as etapas 3 e 4 até que todos os arquivos tenham sido carregados.
    1. Clique em Fechar. Seus arquivos carregados são exibidos na pasta de destino.

Fazer o download

Esta ação permite que você faça o download de um arquivo do dispositivo do My Cloud Mirror para seu computador ou outros dispositivos.
Use as etapas a seguir para fazer o download de um arquivo para o seu computador ou outros dispositivos:

    1. No painel à direita do Web File Viewer, selecione o arquivo que você deseja baixar.
    2. Clique em .
    3. O arquivo selecionado é baixado para seu computador ou outros dispositivos.

Renomear

Esta ação permite que você renomeie um arquivo selecionado no dispositivo do My Cloud Mirror. Use as etapas a seguir para renomear um arquivo.

    1. No painel à direita do Web File Viewer, selecione o arquivo que você deseja renomear.
    2. Clique em .
    3. No campo Renomear, renomeie seu arquivo.
    4. Clique em Salvar. Seu arquivo foi renomeado.

Excluir

Esta ação permite que você exclua um arquivo selecionado do dispositivo do My Cloud Mirror. Use as etapas a seguir para excluir um arquivo.

    1. No painel à direita do Web File Viewer, selecione o arquivo que você deseja excluir.
    2. Clique em .
    3. Clique em OK para confirmar que você deseja excluir o arquivo selecionado. O arquivo selecionado é excluído do dispositivo.

Atualizar

Esta ação atualizou a lista de arquivos, mostrando qualquer arquivo novo que foi incluído desde a última atualização ou desde o último acesso à lista de arquivos.

  • Clique em  . Seus arquivos foram atualizados.
