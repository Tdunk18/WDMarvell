Excluindo um trabalho de backup via USB

Use as etapas a seguir para excluir um trabalho de backup USB.

    1. Clique em Backups.
    2. Clique em Backups de USB.
    3. No área de Trabalhos de backup USB, selecione o trabalho de backup que você deseja excluir.
    4. Clique no ícone Excluir trabalhos.
    5. Clique em OK. Sua trabalho de backup foi excluído.
