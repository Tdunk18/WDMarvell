Atualizando o banco de dados do servidor do iTunes

Se encontrar um erro na quantidade de conteúdo do My Cloud Mirror acessível ou que pode ser visualizado no iTunes, atualize o banco de dados do servidor do iTunes.

OBSERVAÇÃO: A atualização do banco de dados do servidor do iTunes pode levar um tempo considerável, dependendo do número e tamanho dos arquivos de música e vídeo envolvidos.

Use as etapas a seguir para verificar novamente o banco de dados do servidor do iTunes:

    1. Clique em Configurações.
    2. No painel na esquerda, clique em Mídia.
    3. Na área do iTunes, campo Banco de dados do iTunes, clique em Atualizar.
