Alterando o modo de RAID

Use as seguintes etapas para alterar o modo RAID atual do seu dispositivo My Cloud Mirror .

Aviso: As mudanças feitas para o Modo de RAID excluirão todos os seus dados.

    1. Clique em Armazenar.
    2. Clique em RAID.
    3. Clique em Alterar o modo de RAID na parte inferior da tela.
    4. Selecione o modo de RAID que você deseja usar para seu servidor do My Cloud Mirror. As opções incluem:

Modo

Descrição

JBOD

O uso de uma ou mais unidades que não deve ser feito em uma configuração de RAID; ele deve ser gerenciado como volumes lógicos individuais.

Expansão

Combinação de unidades de forma linear para criar um grande volume lógico.

RAID 0

Os dados são distribuídos em vários discos rígidos, permitindo a leitura acelerada e gravação de dados pela combinação do trabalho de duas ou mais unidades para aumentar o desempenho.

RAID 1

Duas cópias instantâneas dos dados são registradas em unidades de disco diferentes para que, se uma falhar, a outra continua a ler/gravar os dados até que a unidade de disco com defeito seja substituída.

    1. Marque a caixa de seleção Alternar para <RAID Mode> .
    2. Clique em Próximo.
    3. Analise o status dos discos rígidos e clique em Próximo.
    4. Clique em Concluir.
    5. Usando o slider de Volume, selecione a quantidade de espaço que você gostaria de atribuir ao modo de RAID selecionado.

      OBSERVAÇÃO: Se você optar por não usar todo o volume para sua seleção de RAID, você pode configurar o espaço de disco restante como Expansão ao marcar a caixa de seleção Configurar o espaço em disco restante como de expansão.
    1. Clique em Próximo.
    2. Selecione para ativar a opção de Restauração automática para a configuração do RAID, depois clique em Próximo.

      OBSERVAÇÃO: Esta tela não exibe os modos JBOD, Expansão e RAID 0
    1. Analise as mudanças do RAID e verifique se estão corretas. Em seguida, clique em Próximo.
    2. Analise a tela de aviso.
      Aviso: Quando você alterar os modos, todos os dados serão apagados.
    1. Clique em Concluir. A mudança de disco rígido solicitada inicia.
    2. Clique em Concluir. Os discos rígidos estão reconfigurados para o modo selecionado.
