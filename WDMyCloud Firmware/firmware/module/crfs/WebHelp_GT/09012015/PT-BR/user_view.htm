Visualizando informações sobre usuários

Use as etapas a seguir para visualizar informações sobre a conta de usuário selecionada. Os detalhes incluem privilégios de acesso que foram concedidos.

    1. Clique em Usuários.
    2. No painel esquerdo, selecione um usuário. O painel à direita mostra as seguintes informações:

Seção

Valores e descrições

Perfil do usuário

Esta área exibe informações básicas de usuário sobre o usuário selecionado, incluindo:

  • Nome de usuário: o nome usado para acessar o dispositivo.

  • Primeiro nome: o primeiro nome do usuário.

  • Sobrenome: o sobrenome do usuário.

  • Endereço de e-mail: o endereço de e-mail para o usuário. Também é usado para acessar o My Cloud.

  • Senha: Permite ativar/desativar a senha do usuário. Consulte Adicionando/alterando uma senha de usuário

  • Inscrição no grupo: Identifica a quais grupos o usuário pertence. Você pode modificar a inscrição em grupos de um usuário aqui. Consulte Modificando a inscrição no grupo.

  • Cotas: Visualize e defina cotas de usuário aqui. Consulte Definindo cotas de usuário.

Acesso ao compartilhamento

Exibe os compartilhamentos disponíveis e indica o tipo de acesso que o usuário tem aos compartilhamentos. Alterar o acesso de um usuário a um compartilhamento ao clicar em uma das opções a seguir:

  •  Acesso de leitura/gravação: pode visualizar e atualizar o compartilhamento.

  •  Acesso somente para leitura: pode visualizar o compartilhamento, mas não pode atualizá-lo.

  •  Negar acesso. O usuário não tem acesso a este compartilhamento.
