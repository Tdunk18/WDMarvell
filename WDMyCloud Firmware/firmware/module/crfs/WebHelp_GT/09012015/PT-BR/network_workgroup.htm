Especificando um grupo de trabalho

É possível inserir o nome de um grupo de trabalho no dispositivo My Cloud Mirror.

O que é um grupo de trabalho?

Um grupo de trabalho é um nome que você associa a vários dispositivos de computação dentro de uma rede. Grupos de trabalho:

    • Facilitam a troca fácil de informações entre os dispositivos de computação

    • Não necessitam de um servidor central

    • Aplicam-se apenas aos ambientes do Microsoft Windows

 

Use as etapas a seguir a seguir para especificar um grupo de trabalho:

    1. Clique em Configurações.
    2. No painel esquerdo, clique em Rede.
    3. Na área de Grupo de trabalho da rede, clique no nome do grupo de trabalho do qual você deseja participar.
