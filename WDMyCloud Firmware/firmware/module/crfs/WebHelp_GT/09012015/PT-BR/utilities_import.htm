Importando as configurações de sistema

Como administrador, você pode importar um arquivo de configuração salvo anteriormente. Essencialmente, esse trabalho substitui a configuração atual pela versão salva anteriormente.

Importante: Importar um arquivo de configuração depois de restaurar as configurações padrão do sistema não restaura os dados do usuário e do dispositivo móvel. Você precisará adicionar essas informações novamente. Quando a importação estiver completa, o dispositivo My Cloud Mirror reinicia, o que irá interromper o serviço para os usuários que estão acessando sua nuvem no momento.

Use as etapas a seguir para importar configurações do sistema.

    1. Clique em Configurações.
    2. No painel esquerdo, clique em Utilitários.
    3. Na área Configuração de sistema, clique em Importar arquivo.
    4. Selecione o local do arquivo de configuração e clique em Abrir. Sua configuração foi restaurada.
