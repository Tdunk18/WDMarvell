Verificação de atualizações e instalação manual

Se o dispositivo My Cloud Mirror não estiver configurado para atualizar seu firmware automaticamente (consulte Atualizando o My Cloud Mirror automaticamente), você poderá verificar manualmente se há uma atualização e instalá-la.

Use as etapas a seguir para verificar uma atualização e instalá-la:

    1. Clique em Configurações.
    2. No painel esquerdo, clique em Atualização de firmware.
    3. Na seção Atualizações disponíveis, clique em Verificar se há novas atualizações.
      Neste ponto, o dispositivo My Cloud Mirror verifica se há atualizações disponíveis. Se a mensagem de status retornada for "Nenhuma nova atualização", você terá concluído este procedimento.
      Se o status retornado listar uma atualização disponível, continue com a Etapa 4.
    4. Clique em Notas da versão para exibir detalhes sobre uma atualização disponível em uma nova janela do navegador. Quando terminar de revisar as notas da versão, feche a janela do navegador.
    5. Clique em Instalar atualização.
    6. Clique em OK para confirmar a reinicialização do dispositivo My Cloud Mirror.
      OBSERVAÇÃO: Lembre-se que o serviço de qualquer usuário que está acessando atualmente a nuvem será interrompido.
