Ligar/desligar o acesso à nuvem

Se você for um administrador, é possível ligar o acesso à nuvem ou desligar para todos os usuários (ou seja, é possível controlar se os dispositivos de computação podem acessar o conteúdo do dispositivo My Cloud Mirror).

Use as etapas a seguir para ativar/desativar o Acesso à nuvem.

    1. Clique em Configurações.
    2. No painel esquerdo, clique em Geral.
    3. Em Acesso à Nuvem no campo Serviço de Nuvem, clique em para ativar ou desativar o serviço de nuvem conforme apropriado:
    • On: Ativa o acesso remoto para todos os usuários.
    • Off: Desativa o acesso remoto para todos os usuários.

Para obter detalhes sobre a desativação do acesso para um único dispositivo, consulte Removendo um dispositivo.

OBSERVAÇÃO: As informações sobre a área das Opções de conexão estão disponíveis em Configurando opções de conexão, que explica as diferentes opções de firewall e inclui uma explicação de como melhorar o desempenho. Por padrão, o My Cloud Mirror estabelece automaticamente uma conexão direta entre seus dispositivos móveis e o roteador.

Se o acesso móvel estiver ativado por meio de uma conexão com retransmissão, isto é, o Status da conexão for "Conectado (retransmitido)", você pode melhorar o desempenho ativando o encaminhamento de porta no seu conector local da WD (consulte Ativando o encaminhamento de porta).
