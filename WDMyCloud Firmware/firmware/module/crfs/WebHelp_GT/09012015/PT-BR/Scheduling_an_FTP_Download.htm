Programando um download via FTP

O aplicativo de Download via FTP permite programar um download via FTP para seu My Cloud Mirror. Use as etapas a seguir para agendar um download via FTP.

    1. Clique em Aplicativos.
    2. Clique em Downloads via FTP.
    3. Clique em Criar Trabalho.
    4. No campo Nome do Trabalho, insira um nome para seu backup via FTP.
    5. No campo Host, insira um valor de host.
    6. No campo Método de login, selecione o método que gostaria de usar para o download do seu FTP:
    • Conta: Informe o nome de usuário e senha para a conta.
    • Anônimo
    1. No campo Tipo, selecione o tipo de informação que você gostaria de baixar:
    • Arquivo
    • Pasta 
    1. No campo Pasta de Origem, clique em Procurar para selecionar o diretório de origem. 
    2. No campo Pasta de Destino, clique em Procurar para selecionar um diretório de destino.
    3. No campo Idioma, selecione o idioma aceito pelo cliente de FTP para o menu suspenso.
    4. Clique em se este for um backup recorrente.
    • Selecione a frequência do backup (diariamente, semanalmente ou mensalmente)
    • Insira o horário, dia e data do backup, dependendo da frequência selecionada.
    1. Digite a data em que deseja realizar o download (mm/dd/aaaa) ou selecione  para selecionar uma data.
    2. Insira um horário para seu download dos menus suspensos.
    3. Clique em Criar. Seu download é criado e adicionado à fila de downloads via FTP.
