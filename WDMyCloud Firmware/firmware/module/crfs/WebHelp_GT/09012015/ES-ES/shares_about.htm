Acerca de los archivos compartidos

Un archivo compartido es un área del dispositivo My Cloud Mirror para almacenar archivos (algo parecido a una carpeta o un directorio).

Los archivos compartidos pueden ser públicos, a cuyo contenido pueden acceder todos los usuarios, o privados, que limitan el acceso a usuarios seleccionados. Para iniciar tareas relacionadas con archivos compartidos haga clic en Archivos compartidos.

Puede:

    • Crear archivos compartidos.  

    • Cambiar los privilegios de acceso de un usuario a un archivo compartido.  

    • Eliminar archivos compartido.
