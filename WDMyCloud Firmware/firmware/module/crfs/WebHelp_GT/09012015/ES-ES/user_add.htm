Agregar una cuenta de usuario

Siga estos pasos para agregar un usuario a su dispositivo My Cloud Mirror:

    1. Haga clic en Usuarios.
    2. Haga clic en el icono Agregar usuario:
    3. Introduzca un nombre de usuario. Este nombre se utiliza para iniciar sesión en el sistema.
    4. Introduzca un nombre. Este es el nombre del usuario.
    5. Introduzca sus apellidos. Estos son los apellidos del usuario.
    6. Introduzca una dirección de correo electrónico. Esta es la dirección de correo electrónico del usuario. Esta dirección también se utiliza para crear una cuenta de My Cloud.
    7. Introduzca una contraseña para la cuenta de usuario.
    8. Vuelva a introducir la contraseña para confirmarla.
    9. Haga clic en Aplicar.
    10. Haga clic en . Esta activa el campo de contraseña y lo habilita.
    • Si no desea asignar una contraseña, asegúrese de que el campo de contraseña está desactivado y haga clic en Siguiente.
