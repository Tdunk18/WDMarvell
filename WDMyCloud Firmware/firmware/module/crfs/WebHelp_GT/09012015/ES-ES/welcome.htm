Bienvenido

Su sistema de almacenamiento My Cloud Mirror es un sistema de alto rendimiento que proporciona un acceso inalámbrico seguro a sus elementos multimedia y archivos desde cualquier ordenador, tableta o smartphone conectado a un enrutador inalámbrico. El dispositivo My Cloud Mirror es el elemento central de su nube personal y proporciona una interfaz sencilla y robusta para una configuración, uso compartido y mantenimiento sencillos.

Tipos de usuario en su nube personal

Todo el que acceda a su nube personal debe disponer de una cuenta de usuario para la unidad My Cloud Mirror. Como propietario de My Cloud Mirror, dispone de una cuenta de usuario especial (admin) que le otorga privilegios de administrador. Con dichos privilegios, puede determinar y configurar el dispositivo de acuerdo con sus necesidades específicas, y agregar a otros usuarios a su nube personal. Puede determinar exactamente quién puede tener acceso al dispositivo.

NOTA: De manera predeterminada, el nombre de usuario admin no tiene contraseña. Puede añadir una en cualquier momento.

Importante: Tiene la posibilidad de cambiar el nombre del administrador y establecer una contraseña opcional al ejecutar el software de configuración de My Cloud Mirror o al completar las pantallas de introducción. Asegúrese de que guarda el nombre y la contraseña. No podrá visualizar la interfaz de la web de My Cloud Mirror sin dichas credenciales.

Acceso al contenido desde un smartphone o una tableta

Dos aplicaciones móviles de WD le permiten acceder a los contenidos de su dispositivo My Cloud desde su smartphone o tableta:

    • Álbumes de My Cloud: Vea y gestione sus fotos.

    • My Cloud: Vea y reproduzca fotos, vídeos, música y archivos.

Obtenga estas aplicaciones gratuitas en el Centro de aprendizaje de WD (http://www.wd.com/setup/wdmycloudmirror) o en otros sitios de aplicaciones como el App Store de Apple, Google Play, la Tienda Windows y Amazon.com.

Acceso al contenido desde un ordenador portátil

Utilice la aplicación de escritorio My Cloud para poder acceder al contenido del dispositivo My Cloud Mirror desde su Mac o PC. My Cloud le permite acceder remotamente a las fotos, los vídeos y los archivos almacenados en sus dispositivos WD NAS, lo que incluye las unidades My Cloud Mirror y My Book Live conectadas a su red doméstica. Como con la aplicación móvil de WD, es absolutamente necesario un inicio de sesión de My Cloud para poder acceder a su contenido.

Descargue la aplicación de escritorio My Cloud en http://www.wd.com/setup/wdmycloudmirror.

Para ver instrucciones paso a paso, consulte Acceso al contenido desde el ordenador portátil.

Visualización de la interfaz de My Cloud Mirror

La interfaz de usuario del dispositivo My Cloud Mirror le permite configurar y gestionar su nube personal. La interfaz se visualiza en el navegador web que ejecute en su Mac o PC. 

Para visualizar la interfaz de la web:

    1. Abra la ventana del navegador.
    2. En la barra de direcciones, escriba http://;
      Por ejemplo: http://wdmycloudmirror
      NOTA: Si la URL no puede mostrar la interfaz de la web, introduzca esta URL: http://wdmycloudmirror.local

Compartir contenido con otros fuera de su red

Para compartir contenido almacenado en My Cloud Mirror, utilice el ordenador y la aplicación de escritorio My Cloud. La aplicación está disponible para su descarga de manera gratuita en http://www.mycloud.com. Para ver instrucciones paso a paso, consulte Compartir contenido con otros fuera de su red.
