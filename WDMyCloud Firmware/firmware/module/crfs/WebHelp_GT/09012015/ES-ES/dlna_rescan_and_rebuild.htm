Volver a explorar/reconstruir la base de datos DLNA

Si encuentra una falta de coincidencia en la cantidad de contenido de la unidad My Cloud Mirror accesible o visible a través de un dispositivo DLNA, puede realizar una de estas dos operaciones en la base de datos del servidor DLNA.

    • Volver a explorar: My Cloud Mirror busca los cambios realizados en sus bibliotecas multimedia. Pruebe esto primero.
    • Reconstruir: My Cloud Mirror reconstruye la base de datos desde cero.
      NOTA: Tenga cuidado al completar este procedimiento. Reconstruir la base de datos DLNA del servidor puede tardar bastante tiempo, en función del número y del tamaño de los archivos relacionados.

Siga estos pasos para volver a explorar o reconstruir la base de datos DLNA:

    1. Haga clic en Configuración.
    2. En el panel izquierdo, haga clic Multimedia.
    3. En el campo Base de datos DLNA, haga clic en Volver a explorar o Reconstruir.
