Acerca de los usuarios

Cualquiera que acceda a su nube personal debe disponer de una cuenta de usuario para usar el dispositivo My Cloud Mirror. Como propietario del dispositivo, dispone de una cuenta de usuario especial (admin) que le otorga privilegios de administrador. Con dichos privilegios, puede determinar y configurar el dispositivo de acuerdo con sus necesidades específicas, y agregar a otras personas a su nube personal. Puede determinar exactamente quién utiliza el dispositivo.

De manera predeterminada, el nombre de usuario admin no tiene contraseña. Puede añadir una en cualquier momento.

Importante: Al configurar el dispositivo por primera vez, mientras se ejecuta el software de configuración del dispositivo My Cloud Mirror o el asistente para comenzar, tiene la posibilidad de cambiar el nombre del administrador y establecer una contraseña opcional (asegúrese de anotar este nombre y contraseña). No podrá visualizar la interfaz de la web de My Cloud Mirror sin dichas credenciales.

Como administrador del dispositivo My Cloud Mirror, puede:

    • Agregar usuarios

    • Agregar o editar contraseñas de usuario

    • Ver la información de usuario

    • Cambiar el acceso de un usuario a un archivo compartido  

    • Eliminar usuarios.

    • Cambiar el nombre y la contraseña de usuario de una persona
