Acerca de la configuración general

El entorno de My Cloud Mirror se instala y se configura a través de la opción Configuración.

La categoría Configuración incluye la vista General. A continuación se enumeran las tareas que puede completar desde esta vista.

  • Perfil del dispositivo

    • Cambiar el nombre del dispositivo  
    • Cambiar la descripción del dispositivo

  • Idioma y reloj

    • Definir el idioma que se mostrará en la interfaz de My Cloud Mirror  
    • Establecer la fecha y la hora
    • Establecer la zona horaria  

  • Acceso a la nube

    • Activar/desactivar el acceso remoto  
    • Configurar las opciones de conexión
    • Habilitar el enrutamiento de puertos  
    • Comprobar el estado del acceso remoto  

  • Ahorro de energía

    • Activar/desactivar el LED de la parte frontal del dispositivo  

    • Determinar cuándo entra en reposo el dispositivo  

  • Copias de seguridad de Mac

    • Configurar copias de seguridad en Apple Time Machine  

    • Desactivar las copias de seguridad de Time Machine
