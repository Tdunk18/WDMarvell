Agregar un archivo compartido

Siga estos pasos para crear un archivo compartido:

    1. Haga clic en Recursos compartidos.
    2. Haga clic en el icono Agregar archivo compartido.
    3. En el menú desplegable, seleccione el volumen en el que desea crear el archivo compartido.
    4. Especifique un nombre para el recurso compartido.
    5. Introduzca una descripción para el archivo compartido. Este es un campo opcional.
    6. Haga clic en para habilitar la Papelera de reciclaje. El campo está activado.
    • Si no quiere habilitar la Papelera de reciclaje, asegúrese de que el campo está desactivado.
    1. Haga clic en para habilitar el envío multimedia. Esto le permite transmitir archivos multimedia almacenados en este archivo compartido a otros dispositivos. Este es un campo opcional.
    • Si no quiere habilitar el envío multimedia, asegúrese de que el campo está desactivado.
    1. Haga clic en  para habilitar el acceso FTP. Esto permite el acceso FTP en este archivo compartido. Este es un campo opcional.
    • Si no quiere habilitar el acceso FTP, asegúrese de que el campo está desactivado.
    1. Haga clic en para habilitar el acceso WebDAV. Esto permite el acceso WebDAV en este archivo compartido. Este es un campo opcional.
    • Si no quiere habilitar el acceso WebDAV, asegúrese de que el campo está desactivado.
    1. Haga clic en  para habilitar el acceso NSF. Esto permite el acceso al sistema de archivos de red (NSF) en este archivo compartido. Este es un campo opcional.
    • Si no quiere habilitar el acceso NSF, asegúrese de que el campo está desactivado.
    1. Se ha agregado el nuevo archivo compartido, que se visualiza en el panel izquierdo.
