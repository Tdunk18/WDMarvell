Informazioni sulla gestione del proprio cloud personale

La gestione del proprio cloud personale comprende molte attività, gran parte delle quali risultano necessarie solo occasionalmente. Nella maggior parte dei casi, le operazioni di gestione del cloud si riducono al minimo dopo aver completato le operazioni illustrate nella Guida introduttiva.

Il presente argomento elenca le attività di gestione del cloud per categorie:

    • Utenti
    • Condivisioni
    • Accesso cloud

Utenti

Selezionare uno dei collegamenti in basso per visualizzare le istruzioni sulla modalità di esecuzione delle attività utente.

    • Aggiunta di un account utente. 
    • Aggiungere o cambiare la password di un utente  
    • Visualizzazione delle informazioni dell'utente.  
    • Modificare l'accesso di un utente a una condivisione  
    • Rimuovere un account utente.

Accesso cloud

Affinché uno smartphone o un tablet possa accedere al contenuto del dispositivo, è necessario registrarlo come parte dell'ecosistema del cloud personale. Viene generato un codice che abbinerà il dispositivo mobile al dispositivo My Cloud Mirror. Selezionare uno dei collegamenti in basso per visualizzare le istruzioni sulla modalità di gestione dell'accesso cloud.

    • Impostare Accesso cloud   
    • Eseguire la configurazione una tantum per ottenere l'accesso sicuro da un dispositivo mobile  
    • Rimuovere un dispositivo di elaborazione dal proprio cloud personale  
    • Controllare lo stato dell'accesso remoto  
    • Migliorare le prestazioni mediante port forwarding
