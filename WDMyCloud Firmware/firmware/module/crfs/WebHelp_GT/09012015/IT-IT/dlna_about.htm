Informazioni sul dispositivo My Cloud Mirror e DLNA

È possibile configurare il dispositivo My Cloud Mirror per riprodurre/visualizzare file sulla propria TV a grande schermo, purché sia un dispositivo certificato DLNA. DLNA (Digital Living Network Alliance) è uno standard utilizzato dai produttori di apparecchi elettronici di consumo per consentire agli apparecchi per l'intrattenimento di condividere contenuti tra loro all'interno della rete domestica. Tra gli altri dispositivi DLNA figurano i telefoni cellulari e i PC.

Con il dispositivo My Cloud Mirror, la connettività con i dispositivi DLNA si abilita semplicemente attivando la funzione Server DLNA.    Procedura

Consultare la documentazione del proprio dispositivo DLNA specifico per configurarne l'accesso alla rete.

Se si riscontra una discrepanza nella quantità di contenuti accessibili da un dispositivo DLNA, potrebbe essere opportuno ripristinare il database del server DLNA. Questa operazione, comprese le informazioni essenziali sull'utilizzo, è illustrata in Nuova scansione/ripristino del Database DLNA
