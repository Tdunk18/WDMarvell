Creazione di un backup USB

Attenersi ai passaggi seguenti per eseguire il backup dei dati nel proprio My Cloud Mirror verso o da un dispositivo USB.

    1. Fare clic su Backup.
    2. Fare clic su Backup USB.
    3. Fare clic su Crea processo.
    4. Immettere un nome di processo per il backup.
    5. Indicare la direzione del backup dal menu a discesa. Le opzioni includono:
    • da USB a NAS: Esegue il backup dei dati nel dispositivo USB verso il dispositivo My Cloud Mirror.
    • da NAS a USB: Esegue il backup dei dati nel proprio dispositivo My Cloud Mirror verso un dispositivo USB.
    1. Selezionare la cartella di origine per il backup:
    • Fare clic su Sfoglia e selezionare la cartella di cui eseguire il backup dal dispositivo di destinazione.
    1. Selezionare la cartella di destinazione per il backup.
    • Fare clic su Sfoglia e selezionare la cartella in cui eseguire il backup del dispositivo di origine.
    1. Indicare il tipo di backup, selezionandolo dal menu a discesa.
    2. Per avviare automaticamente un backup USB, fare clic sull'icona attiva/disattiva .
    3. Fare clic su Crea. In questo modo viene creato il processo di backup per il backup richiesto.
    4. Fare clic su Dettagli nell'area Processi backup USB per controllare l'avanzamento del processo di backup.
