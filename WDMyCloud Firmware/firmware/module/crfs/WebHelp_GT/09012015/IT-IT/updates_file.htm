Aggiornamento del dispositivo My Cloud Mirror da un file

In alcuni casi, potrebbe essere necessario aggiornare il dispositivo My Cloud Mirror effettuando dapprima il download di un file dal sito del supporto Western Digital e quindi installandolo.

NOTA: questa procedura richiede l'accesso a Internet.

Per aggiornare manualmente da file il dispositivo My Cloud Mirror, attenersi ai passaggi seguenti:

    1. Aprire una nuova scheda del browser o una nuova finestra.
    2. Nella nuova scheda/finestra, puntare al sito del supporto tecnico Western Digital:
      http://support.wdc.com
    • Se è disponibile un nuovo aggiornamento firmware, salvarlo in una posizione del dispositivo di calcolo.
    1. Tornare all'interfaccia Web di My Cloud Mirror.
    2. Fare clic su Impostazioni.
    3. Nel riquadro sinistro, fare clic Aggiornamento firmware.
    4. Nell'area Aggiornamento manuale, fare clic su Aggiorna da file.
    5. Navigare alla posizione in cui è stato salvato l'aggiornamento nel Passaggio 3.
    6. Selezionare il file di aggiornamento e fare clic su Apri.
    7. Fare clic su OK per confermare l'installazione. Dopo l'installazione dell'aggiornamento, il dispositivo My Cloud Mirror si riavvia.
      NOTA: Quando My Cloud Mirror si riavvia, il servizio per qualunque utente attualmente connesso al cloud viene interrotto.
