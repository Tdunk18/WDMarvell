Eseguire la configurazione una sola volta per l'accesso sicuro

Questa procedura illustra come stabilire una connessione sicura tra il proprio smartphone o tablet e il dispositivo My Cloud Mirror. Attenersi alle istruzioni del proprio dispositivo mobile specifico.

    • Smartphone Android o tablet  

    • iPhone o iPad

    • Telefono o tablet Windows

    • Desktop Windows o Mac

Ottenere accesso sicuro per uno smartphone o un tablet android

    1. Connettere il proprio smartphone o tablet procedendo come segue:

      1. Sul proprio dispositivo mobile, toccare Impostazioni.

      2. Toccare Wireless e reti.

      3. Toccare Impostazioni Wi-Fi.

      4. Seleziona la rete domestica.

      5. Inserire la password di rete e toccare Connetti.

    1. Lanciare un'app WD.
    2. Toccare Connetti ora.
    3. Toccare il nome del dispositivo My Cloud Mirror.
    4. Tocca la freccia di selezione e seleziona il nome utente.
    5. Esegui una delle azioni seguenti:

    • Se è stata impostata una password per l'utente, inserirla e toccare Login.

    • Se non è stata impostata una password per l'utente, toccare Login.

    1. (Opzionale) Se si desidera accedere a qualunque contenuto, toccare il nome di una condivisione, quindi navigare fino al file desiderato.

Ottenere accesso sicuro per un iPhone o iPad

    1. Connettere il proprio smartphone o tablet procedendo come segue:

    1. Sul proprio dispositivo mobile, toccare Impostazioni.

    2. Toccare Wi-Fi.

    3. Toccare Impostazioni Wi-Fi.

    4. Seleziona la rete domestica.

    5. Inserire la password di rete e toccare Unisci.

    1. Lanciare un'app WD.
    2. Toccare Connetti ora.
    3. Toccare il segno più (+) accanto al nome del dispositivo My Cloud Mirror.
    4. Toccare il nome utente desiderato.
      Se è stata impostata una password per l'utente, inserirla.
    5. (Opzionale) Se si desidera accedere a qualunque contenuto, toccare il nome di una condivisione, quindi navigare fino al file desiderato.

Ottenere accesso sicuro per uno smartphone o un tablet Windows

    1. Connettere il proprio smartphone o tablet procedendo come segue:

    1. Sul proprio dispositivo di elaborazione, toccare Impostazioni.

    1. Toccare Wi-Fi.

    2. Seleziona la rete domestica.

    3. Inserire la password di rete e toccare Fatto.

    1. Lanciare un'app WD.
    2. Toccare Connetti ora.
    3. Toccare il nome dell’unità My Cloud Mirror.
    4. Tocca la freccia di selezione e seleziona un nome utente.
      Se è stata impostata una password per l'utente, inserirla.
    5. (Opzionale) Se si desidera accedere a qualunque contenuto, procedere come segue:

    1. Nella schermata Dispositivi, toccare il nome della propria unità My Cloud Mirror.

    2. Tocca una condivisione, quindi portati sul contenuto desiderato.
