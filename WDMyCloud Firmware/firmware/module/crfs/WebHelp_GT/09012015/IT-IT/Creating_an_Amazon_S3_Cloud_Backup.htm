Creazione di un cloud backup Amazon S3

Nota: Prima di creare il cloud backup, accedere per registrarsi al servizio Amazon S3 da http://aws.amazon.com/s3.

Una volta configurato l'account Amazon S3, attenersi ai passaggi seguenti per creare un cloud backup.

    1. Fare clic su Backup.
    2. Fare clic su Backup cloud.
    3. Fare clic sul pulsante Amazon S3 nella parte alta della schermata.
    4. Fare clic su Crea.
    5. Immettere un nome di processo per il cloud backup.
    6. Fare clic su Avanti.
    7. Immettere le informazioni seguenti fornite da Amazon S3:
    • Area: selezionare dal menu a discesa l'area utilizzata per il cloud.
    • Chiave di accesso: immettere la chiave di accesso fornita da Amazon S3.
    • Chiave privata: immettere la chiave privata fornita da Amazon S3.
    • Percorso Remoto: immettere il percorso remoto per il cloud. Si tratta in genere del nome bucket.
    1. Fare clic su Avanti.
    2. Nel campo Tipo selezionare una delle opzioni seguenti:
    • Carica: Esegue il backup dei dati My Cloud Mirror nel bucket Amazon S3.
    • Download: Esegue il backup del bucket Amazon S3 nel server My Cloud Mirror.
    1. Nel campo Tipo backup selezionare il tipo di backup che si desidera eseguire dal menu a discesa. Le opzioni includono:
    • Sovrascrivi file esistenti: sovrascrive i file nella cartella di destinazione con lo stesso nome del file di origine.
    • Backup Completo: crea una cartella separata contenente tutti i dati di backup ogni volta che il backup viene eseguito.
    • Backup Incrementale: sovrascrive i file di origine più recenti rispetto ai file di destinazione.
    1. Fare clic su Avanti.
    2. Nel campo Percorso locale immettere un percorso per il backup nel dispositivo My Cloud Mirror.
    • Fare clic su Sfoglia per accedere alla posizione per il backup nel dispositivo.
      (Esempio Volume_1/backup)
    1. Fare clic su  per attivare la funzionalità Aggiornamento automatico, che aggiorna automaticamente il backup in base alla pianificazione creata.
    • Per attivare la funzionalità Aggiornamento automatico:

      • selezionare la Pianificazione aggiornamento automatico: Ogni giorno, Ogni settimana o Ogni mese

      • Selezionare Ora aggiornamento automatico dal menu a discesa (opzione Ogni giorno).

      • Selezionare Data e ora aggiornamento automatico dai menu a discesa (opzione Ogni settimana o Ogni mese).

    1. Fare clic su Avanti.
    2. Esaminare le impostazioni, quindi fare clic su Fine. Il cloud backup Amazon S3 è stato creato.

NOTA: acceder al bucket di cloud backup Amazon S3 per visualizzare i backup del proprio dispositivo.
