Cos'è il DLNA?

 DLNA (Digital Living Network Alliance) è uno standard utilizzato dai produttori di apparecchi elettronici di consumo per consentire agli apparecchi domestici per l'intrattenimento di condividere contenuti tra loro all'interno della rete domestica. Per esempio, è possibile configurare My Cloud Mirror per riprodurre/visualizzare file sulla propria TV a grande schermo, purché sia un dispositivo certificato DLNA.
