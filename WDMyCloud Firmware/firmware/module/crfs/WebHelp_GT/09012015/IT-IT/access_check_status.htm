Controllo dello stato dell’accesso remoto

In qualunque momento, è possibile controllare se:

    • Il dispositivo My Cloud Mirror è configurato per l'accesso remoto
    • L'accesso remoto al dispositivo My Cloud Mirror è abilitato attraverso una connessione diretta o trasmessa
    • un dispositivo mobile particolare di un determinato utente è attualmente abilitato per l'accesso in remoto al proprio cloud

Attenersi ai passaggi seguenti per controllare lo stato dell'accesso mobile:

    1. Fare clic su Accesso cloud.
    2. Nel riquadro sinistro, fare clic su un nome utente.
      La tabella seguente elenca i campi di accesso mobile e i relativi possibili valori:

Sezione

Nome campo/opzione

Valori e descrizioni

Profilo accesso cloud

Stato di connessione

  • Connesso (diretto): L'accesso mobile è abilitato attraverso una connessione diretta.

  • Connesso (trasmissione) L'accesso mobile è abilitato attraverso una connessione trasmessa.

  • Operazione non riuscita: My Cloud Mirror non riesce a stabilire una connessione di accesso remoto.

Dispositivi cloud nome utente

Nome dispositivo o applicazione

  • Connesso; Il dispositivo dispone di accesso remoto al cloud.

  • In attesa: In attesa di attivazione del dispositivo.

  • Codice scaduto: Codice di attivazione scaduto.

  • Disabilitato: L'opzione Servizio cloud è disattivata. Vedere Attivazione/disattivazione di Accesso cloud.

  • Connessione non riuscita: Connessione remota non riuscita. È possibile tentare di risolvere il problema effettuando una ricerca degli errori nella connessione di rete e accertandosi che sia installato il firmware più recente.

 

NOTA: Se si desidera visualizzare le impostazioni di accesso mobile per un utente diverso, fare clic sul nome utente di quella persona nella colonna a sinistra. La vista viene aggiornata con le informazioni del dispositivo cloud dell'utente.
