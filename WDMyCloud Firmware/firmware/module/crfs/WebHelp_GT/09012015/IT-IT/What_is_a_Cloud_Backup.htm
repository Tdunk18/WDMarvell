Cos'è il cloud backup?

Un cloud backup consente di creare dei backup remoti accessibili tramite Internet e attraverso vari dispositivi mobili. Il dispositivo My Cloud Mirror si serve dei servizi cloud seguenti per creare backup remoti:

    • ElephantDrive: ElephantDrive è un servizio di cloud backup che fornisce backup sicuri e automatici in remoto ai file.
    • Amazon S3: Amazon Simple Storage Service (S3) è un servizio Web online di archiviazione file che può essere utilizzato per archiviare e recuperare la quantità di dati che si desidera in qualsiasi momento e da qualsiasi posizione del Web.
