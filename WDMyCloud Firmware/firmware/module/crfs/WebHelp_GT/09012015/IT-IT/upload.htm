Caricamento di contenuti sul dispositivo My Cloud Mirror

Per caricare foto, video, musica e file nel dispositivo My Cloud Mirror, è necessario attenersi a una delle procedure seguenti:

    • Mac
    • PC Windows
    • USB

Caricamento di contenuti da Mac

Per caricare file dal Mac al dispositivo My Cloud Mirror:

    1. Sul Mac, scegliere Vai > Rete dalla barra dei menu.
    2. Esegui una delle azioni seguenti:
    • Nel riquadro sinistro del Finder, fare doppio clic sul nome del dispositivo My Cloud Mirror.
    • Nel riquadro destro, individuare il nome del dispositivo My Cloud Mirror e fare doppio clic su di essa.
    1. Fare doppio clic sul nome di una particolare condivisione per aprire la cartella e visualizzare eventuali sottocartelle esistenti. (Pubblica è una condivisione creata automaticamente).
    2. Trascinare e rilasciare i file dal computer alla cartella desiderata.
      Non rilasciare i file nelle cartelle Time Machine e WD SmartWare Backups.
      NOTA: Se si copiano i file dal computer al dispositivo My Cloud Mirror, tali file risiederanno in due posizioni: nella posizione originale e sul dispositivo.

Caricamento di contenuti da PC

Per caricare file dal PC Windows al dispositivo My Cloud Mirror:

    1. Sul computer, aprire una finestra di dialogo Esegui premendo contemporaneamente i tasti Windows ed R.
    2. Nel campo Apri, inserire \\ seguito dal nome del dispositivo My Cloud Mirror. (Il nome predefinito è mycloudmirror, senza gli spazi). Esempio: \\mycloudmirror
    3. Fare clic su OK.
      Si apre Esplora risorse o File Explorer con le condivisioni disponibili elencate nel riquadro destro.
    4. Fare doppio clic sul nome di una particolare condivisione per aprire la cartella e visualizzare eventuali sottocartelle. (Pubblica è una condivisione creata automaticamente).
    5. Trascinare e rilasciare i file dal computer a questa condivisione.
      NOTA: Se si copiano i file dal computer al dispositivo My Cloud Mirror, tali file risiederanno in due posizioni: nella posizione originale e sul dispositivo.

     

    Caricamento di contenuti da USB

    Per caricare file dal USB al dispositivo My Cloud Mirror, attenersi alle procedure seguenti:

    1. per prestazioni ottimali, connettere l'unità USB nel computer.

    2. Connettere il computer e il dispositivo My Cloud Mirror direttamente al router con una connessione Ethernet cablata.

    3. Copiare i dati dall'unità USB esterna in un dispositivo My Cloud Mirror

NOTA: Se si copiano i file dal computer al dispositivo My Cloud Mirror, tali file risiederanno in due posizioni: nella posizione originale e sul dispositivo.
