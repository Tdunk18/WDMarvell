Creazione di un backup remoto

Il backup remoto consente di eseguire il backup del dispositivo My Cloud Mirror in un altro server o all'esterno della propria rete.    Attenersi ai passaggi seguenti per creare un backup remoto per il server My Cloud Mirror.

    1. Fare clic su Backup.
    2. Fare clic su Backup remoti.
    3. Fare clic sul pulsante Crea processo.
    4. Immettere un nome processo.
    5. Selezionare un server dal menu a discesa Server remoto.
    6.  Nel campo IP remoto, immettere l'indirizzo IP del dispositivo di backup remoto.
      (Esempio 192.168.1.16)
    7. Nel campo Password, immettere la password del dispositivo di backup remoto.
    8. Immettere un nome utente SSH.
    9. Nel campo password SSH, immetti di nuovo la password SSH.
    10. Fare clic su Sfoglia per selezionare una cartella di origine.
    11. Dall'elenco delle cartelle di backup remoto, selezionare una o più directory di condivisione da proteggere.
    12. Fare clic su OK.
    13. Per selezionare una directory di backup remoto, fare clic su Sfoglia.
    14. Scegliere una directory e fare clic su OK.
    15. Scegliere il tipo di backup dal menu a discesa.
    16. Per impostare una pianificazione di backup, accanto a Ricorrenza, fare clic sull'icona attiva/disattiva.
    17. Per impostare il Trasferimento crittografia, fare clic sull'icona attiva/disattiva.
    18. Fare clic su Crea.
