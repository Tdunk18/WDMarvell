Informazioni sulle impostazioni generali

L'ambiente del dispositivoMy Cloud Mirror viene impostato e configurato tramite le opzioni delle Impostazioni.

La categoria Impostazioni include la vista Generale. Di seguito sono elencate le attività che è possibile completare.

  • Profilo dispositivo

    • Modificare il nome del dispositivo  
    • Modificare la descrizione del dispositivo

  • Lingua e orologio

    • Per impostare la lingua visualizzata nell'interfaccia web di My Cloud Mirror  
    • Impostare data e ora
    • Impostare il fuso orario  

  • Accesso cloud

    • Attivare/disattivare l'accesso remoto  
    • Configurare le opzioni di connessione
    • Abilitare il port forwarding  
    • Controllare lo stato dell'accesso remoto  

  • Risparmio energetico

    • Accendere/spegnere il LED nella parte anteriore del dispositivo  

    • Determinare quando il dispositivo entra in modalità stand-by  

  • Backup Mac

    • Configurare i backup su Apple Time Machine  

    • Disattivare i backup di Time Machine
