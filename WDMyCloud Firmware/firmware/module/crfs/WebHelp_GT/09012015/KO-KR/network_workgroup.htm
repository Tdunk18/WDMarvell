작업 그룹 지정

My Cloud Mirror 장치에서 작업 그룹의 이름을 입력할 수 있습니다.

작업 그룹이란?

작업 그룹은 네트워크 내의 여러 컴퓨팅 장치와 연결하는 이름입니다. 작업 그룹의 특징은 다음과 같습니다.

    • 컴퓨팅 장치 간의 정보 교환을 용이하게 합니다.

    • 중앙 서버가 필요하지 않습니다.

    • Microsoft Windows 환경에만 적용됩니다.

 

다음 단계를 사용하여 작업 그룹을 지정합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 네트워크를 클릭합니다.
    3. 네트워크 작업 그룹 영역에서 연결할 작업 그룹의 이름을 입력합니다.
