가정에서 내용 액세스

가정에서 홈 네트워크에 연결된 컴퓨팅 장치를 통해 드라이브에 저장된 내용에 액세스하는 방법은 원격으로 액세스하는 방법과 약간 다릅니다. 홈 네트워크에 연결된 경우 다음을 수행할 수 있습니다.

    • 스마트폰 또는 태블릿에서 내용 액세스
    • 컴퓨터에서 내용 액세스

중요: 시작하기 전에 클라우드 액세스 설정에 설명된 단계를 완료해야 합니다.

홈 네트워크에 연결된 모바일 장치에서 내용 액세스

    1. 아직 하지 않은 경우 WD 모바일 앱을 다운로드합니다.
    2. 스마트폰 또는 태블릿에서 홈 네트워크에 연결합니다.
      방법을 모르는 경우 사용 중인 장치의 설명서를 참조하십시오.
    3. WD 모바일 앱(My Cloud 또는 WD Photos)을 실행합니다.
    4. 메시지가 표시되면 지금 연결을 누릅니다.
    5. My Cloud Mirror 드라이브를 선택합니다.
    6. 액세스하려는 공유를 선택한 다음 원하는 콘텐츠로 이동합니다.

맨 위로

홈 네트워크에 연결된 컴퓨터에서 내용 액세스

    1. 컴퓨터가 홈 네트워크에 연결되어 있고 인터넷에 액세스할 수 있는지 확인합니다.
    2. 아직 My Cloud 데스크톱 응용 프로그램을 다운로드하지 않은 경우 WD 교육 센터(http:.wd.com/setup/wdmycloudmirror)에서 다운로드하여 Mac 또는 PC에 설치합니다.
    3. 컴퓨터에서 My Cloud 데스크톱 응용 프로그램을 실행합니다.
    4. 로그인 자격 증명을 입력합니다.
    5. My Cloud Mirror 드라이브를 두 번 클릭합니다.
    6. 왼쪽 창에서 필요에 따라 폴더를 확장하여 원하는 내용으로 이동합니다.
