다른 사용자 설정

소유자 계정을 설정하고 원격 액세스를 구성하고 나면 장치에 다른 사용자를 추가할 수 있습니다.

주요 작업은 새 사용자를 추가하고 원격 액세스를 설정하는 것입니다. 사용자 계정 추가를 참조하십시오.

클라우드 관리자는 다음 작업도 수행할 수 있습니다.

    • 사용자 계정에 암호를 추가합니다. 암호 추가/변경을 참조하십시오.

    • 공유에 대한 사용자 액세스를 변경합니다. 공유 액세스 변경을 참조하십시오.

    • 사용자 계정을 삭제합니다. 사용자 계정 삭제를 참조하십시오.

    • 사용자 이름 및 암호 변경 암호 추가/변경을 참조하십시오.
