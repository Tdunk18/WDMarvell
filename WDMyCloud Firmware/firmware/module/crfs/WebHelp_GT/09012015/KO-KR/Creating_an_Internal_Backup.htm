내부 백업 만들기

내부 백업을 사용하면 기존 콘텐츠를 My Cloud Mirror 장치의 다른 스토리지 위치에 백업할 수 있습니다.

다음 단계를 사용하여 내부 백업을 만듭니다.

    1. 백업을 클릭합니다.
    2. 내부 백업을 클릭합니다.
    3. 작업 만들기를 클릭합니다.
    4. 작업 이름 필드에 백업 작업의 이름을 입력합니다.
    5. 소스 폴더 필드에서 찾아보기를 클릭합니다.
    6. 백업할 데이터를 선택한 후 확인을 클릭합니다.
    7. 대상 폴더 필드에서 찾아보기를 클릭합니다.
    8. 데이터를 백업할 대상을 선택한 후 확인을 클릭합니다.
    9. 반복 백업 필드에서 을 선택하여 이 기능을 켠 후 다음 정보를 입력합니다.
    • 백업 빈도를 선택합니다. '매일', '매주', '매월' 중에서 선택합니다.
    • 풀다운 메뉴에서 시간(시간과 분)을 선택합니다. 
    1. 반복 백업 필드를 활성화하지 않은 경우 다음 정보를 입력합니다.
    • 날짜 필드를 클릭하고 백업할 날짜를 선택합니다.
    • 풀다운 메뉴에서 시간(시간과 분)을 선택합니다.
    1. 증분 백업을 하려는 경우 증분 백업 확인란을 클릭합니다.
    2. 만들기를 클릭합니다. 작업이 내부 백업 대기열 영역에 표시되고, 지정된 시간에 백업이 시작됩니다.
