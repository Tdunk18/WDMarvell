진단 테스트 수행

다음 테스트 중 하나를 실행하여 장치의 상태를 확인할 수 있습니다.

    1. 빠른 테스트: 장치에서 주요 성능 문제를 확인하고 통과 또는 실패 평가를 생성합니다. 빠른 테스트를 완료하는 데 몇 분 정도 걸릴 수 있습니다.

    • 전체 테스트: 장치의 모든 섹터를 테스트하여 My Cloud Mirror 장치의 상태에 대한 피드백을 제공합니다.  저장된 데이터의 양에 따라 전체 테스트를 완료하는 데 몇 시간 정도 걸릴 수 있습니다.

다음 단계를 사용하여 진단 테스트를 수행합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 유틸리티를 클릭합니다.
    3. 시스템 진단 영역에서 빠른 테스트 또는 전체 테스트를 클릭합니다.
      표시기에 테스트 상태가 표시되고, 대화 상자에 결과가 표시됩니다.
