RAID 자동 복원

자동 복원 옵션을 사용하면 드라이브 손상 시 My Cloud Mirror 장치의 RAID를 자동으로 복원할 수 있습니다. 다음 단계를 사용하여 자동 복원 옵션을 활성화합니다.

    1. 스토리지를 클릭합니다.
    2. RAID를 클릭합니다.
    3. 자동 복원 필드를 검토하여 현재 My Cloud Mirror 장치의 자동 복원이 설정되어 있는지 여부를 확인합니다.
    4. 현재 자동 복원이 활성화되어 있습니다.
    5. 현재 자동 복원이 비활성화되어 있습니다.
    1. 현재 비활성화되어 있는 경우 자동 복원 옵션을 선택하여 활성화합니다.
