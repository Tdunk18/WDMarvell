자동으로 My Cloud Mirror 장치 업데이트

사용 가능한 펌웨어 업데이트를 편리하게 확인하여 자동으로 설치하도록 My Cloud Mirror 장치를 설정할 수 있습니다. 이는 My Cloud Mirror 펌웨어를 최신 상태로 유지하는 가장 간편한 방법입니다.

참고: 소프트웨어 업데이트가 자동으로 설치되면 My Cloud Mirror 장치가 다시 부팅됩니다. 다시 부팅 시 사용자들의 클라우드 액세스가 영향을 받으므로 클라우드에 액세스하는 사용자 수가 가장 적은 시간에 업데이트가 수행되도록 예약해야 합니다.

다음 단계를 사용하여 My Cloud Mirror 장치를 자동으로 업데이트할 수 있습니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 펌웨어 업데이트를 클릭합니다.
    3. 자동 업데이트 영역에서 을 클릭하여 자동 업데이트를 활성화합니다.
    4. 업데이트 일정에서 업데이트 날짜와 시간을 예약합니다.

    • 첫 번째 드롭다운 화살표를 클릭하고 요일을 선택합니다.

    • 두 번째 드롭다운 화살표를 클릭하고 시간을 선택합니다.

    • 세 번째 드롭다운 화살표를 클릭하고 AM 또는 PM을 선택합니다.
