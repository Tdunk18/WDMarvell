앱 활성화/비활성화

앱을 사용하려면 먼저 장치에서 사용할 수 있도록 활성화해야 합니다. 활성화되면 해당 앱이 앱 페이지에 표시됩니다. 반대로, 앱 페이지에서 앱을 제거하지만 삭제하지는 않으려는 경우 앱을 비활성화하여 앱 페이지에서 제거할 수 있습니다.

다음 단계를 사용하여 My Cloud Mirror 장치의 앱을 활성화/비활성화합니다.

    1. 설정을 클릭합니다.
    2. 앱을 클릭합니다.
    3. 설치된 앱 영역에서 활성화/비활성화할 앱을 선택합니다.
    4. 세부 정보를 클릭합니다.
    5. 활성화 또는 비활성화를 클릭합니다.
    6. 닫기를 클릭합니다. 이제 앱이 활성화/비활성화되었습니다.
