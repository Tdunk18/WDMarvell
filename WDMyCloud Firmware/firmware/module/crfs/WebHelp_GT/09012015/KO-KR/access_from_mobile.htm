스마트폰 또는 태블릿에서 내용 액세스

스마트폰 또는 태블릿을 사용하여 My Cloud Mirror 장치에 저장된 콘텐츠에 액세스할 수 있습니다. 그러려면 먼저 클라우드 액세스 설정, WD 모바일 앱 다운로드, 안전하게 액세스하기 위한 1회 설정 수행 작업을 완료해야 합니다.

다음 단계를 사용하여 스마트폰 또는 태블릿에서 콘텐츠에 액세스합니다.

    1. 모바일 장치가 Wi-Fi 또는 3G/4G에 연결되어 있는지 확인합니다.
    2. WD 모바일 앱을 실행합니다.
    3. 메시지가 표시되면 지금 연결을 누릅니다.
    4. My Cloud Mirror 드라이브를 선택합니다.
    5. 액세스하려는 공유를 선택한 다음 원하는 콘텐츠로 이동합니다.
