DLNA 데이터베이스 다시 검색/복원

DLNA 장치에서 액세스하거나 볼 수 있는 My Cloud Mirror 콘텐츠의 용량이 일치하지 않는 경우 DLNA 서버 데이터베이스에 대해 다음 작업 중 하나를 수행할 수 있습니다.

    • 다시 검색: My Cloud Mirror 장치에서 미디어 라이브러리의 변경 사항을 검색합니다. 이 작업을 먼저 수행하십시오.
    • 복원: My Cloud Mirror 장치에서 데이터베이스를 맨 처음부터 완전히 복원합니다.
      참고: 이 절차를 수행할 때는 주의해야 합니다. 들어 있는 파일의 개수와 크기에 따라 DLNA 서버 데이터베이스를 복원하는 데 상당한 시간이 걸릴 수 있습니다.

다음 단계를 사용하여 DLNA 데이터베이스를 다시 검색하거나 복원합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 미디어를 클릭합니다.
    3. DLNA 데이터베이스 필드에서 다시 검색 또는 복원을 클릭합니다.
