일반 설정 정보

설정 옵션을 통해 My Cloud Mirror 장치 환경을 설정하고 구성할 수 있습니다.

설정 범주에는 일반 보기가 포함됩니다. 이 보기에서 완료할 수 있는 작업은 다음과 같습니다.

  • 장치 프로파일

    • 장치 이름 변경  
    • 장치 설명 변경

  • 언어 및 시계

    • My Cloud Mirror 인터페이스에 표시되는 언어 설정  
    • 날짜 및 시간 설정
    • 시간대 설정  

  • 클라우드 액세스

    • 원격 액세스 켜기/끄기  
    • 연결 옵션 구성
    • 포트 전달 활성화  
    • 원격 액세스 상태 확인  

  • 절전

    • 장치 전면의 LED 켜기/끄기  

    • 장치를 절전 모드로 전환하는 경우 결정  

  • Mac 백업

    • Apple Time Machine으로 백업 구성  

    • Time Machine 백업 끄기
