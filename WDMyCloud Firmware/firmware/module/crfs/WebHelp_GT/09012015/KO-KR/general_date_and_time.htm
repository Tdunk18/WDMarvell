날짜 및 시간 설정

다음 단계를 사용하여 My Cloud Mirror 장치의 날짜와 시간을 설정합니다.

시간대 설정

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 일반을 클릭합니다.
    3. 시간대 필드의 드롭다운 메뉴에서 시간대를 선택합니다.
    4. 저장을 클릭합니다.

NTP에 날짜와 시간 동기화

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 일반을 클릭합니다.
    3. 을 클릭하여 NTP 서비스를 켭니다.
    4. NTP 서비스에서 사용할 서버를 결정합니다. 방법    

       

      NTP 서비스를 구성하려면

      1. 주 서버 필드 옆의 구성 링크를 클릭합니다.
      2. NTP 설정 대화 상자에서 사용자 NTP 추가를 클릭합니다.
      3. 서버 세부 정보를 입력한 후 저장을 클릭합니다.
      4. 필요에 따라 1~2단계를 반복합니다. 최대 3개의 NTP 서버를 입력할 수 있습니다.
