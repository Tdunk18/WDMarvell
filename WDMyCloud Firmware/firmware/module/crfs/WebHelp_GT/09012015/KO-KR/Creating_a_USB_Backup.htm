USB 백업 만들기

다음 단계를 사용하여 My Cloud Mirror의 데이터를 USB 장치에서 또는 USB 장치로 백업합니다.

    1. 백업을 클릭합니다.
    2. USB 백업을 클릭합니다.
    3. 작업 만들기를 클릭합니다.
    4. 백업의 작업 이름을 입력합니다.
    5. 풀다운 메뉴에서 백업 방향을 지정합니다. 옵션은 다음과 같습니다.
    • USB에서 NAS로: USB 장치의 데이터를 My Cloud Mirror 장치에 백업합니다.
    • NAS에서 USB로: My Cloud Mirror 장치의 데이터를 USB 장치에 백업합니다.
    1. 백업의 소스 폴더를 선택합니다.
    • 찾아보기를 클릭하고 대상 장치에서 백업할 폴더를 선택합니다.
    1. 백업의 대상 폴더를 선택합니다.
    • 찾아보기를 클릭하고 소스 장치를 백업할 폴더를 선택합니다.
    1. 풀다운 메뉴에서 백업 유형을 표시합니다.
    2. 연결 시 USB 백업을 자동으로 시작하려면 토글 아이콘 을 클릭합니다.
    3. 만들기를 클릭합니다. 그러면 요청된 백업에 대한 백업 작업이 만들어집니다.
    4. USB 백업 작업 영역에서 세부 정보를 클릭하여 백업 작업의 진행률을 확인합니다.
