연결 옵션 구성

모바일 장치와 컴퓨터에서 My Cloud Mirror 장치에 원격으로 액세스할 수 있는 방법을 결정하는 연결 옵션을 지정할 수 있습니다.

다음 단계를 사용하여 연결 옵션을 구성합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 일반을 클릭합니다.
    3. 클라우드 액세스 영역의 클라우드 서비스 필드에서 구성을 클릭합니다.
    4. 대화 상자에서 다음 설정 중 하나를 클릭합니다.
    • 자동: 모바일 장치와 라우터 간의 직접 연결을 설정합니다. 이 연결에서 uPnP(범용 플러그 앤 플레이)를 통해 포트를 열 수 없는 경우 릴레이 연결이 설정됩니다.
    • 수동: 선택한 두 포트를 통해 연결을 설정합니다. 이 옵션은 포트 전달을 통해 직접 연결을 설정합니다. 이 옵션을 사용하는 방법에 대한 단계별 지침은 포트 전달 사용을 참조하십시오. 이 옵션을 사용할 때는 라우터를 수동으로 적절히 구성해야 합니다.
    • Win XP: 포트 80~443을 통해 연결을 설정합니다. 이러한 포트를 사용할 수 없는 경우 릴레이 연결이 설정됩니다. 원격 장치 중 하나 이상에서 Windows XP 운영 체제를 실행하는 경우 이 옵션을 선택해야 합니다.
    1. 적용을 클릭합니다.
