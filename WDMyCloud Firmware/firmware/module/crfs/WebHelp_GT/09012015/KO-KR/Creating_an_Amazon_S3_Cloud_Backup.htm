Amazon S3 클라우드 백업 만들기

참고: 클라우드 백업을 만들기 전에 http://aws.amazon.com/s3에 접속하여 Amazon S3 서비스에 등록하십시오.

Amazon S3 계정이 설정되면 다음 단계를 사용하여 클라우드 백업을 만듭니다.

    1. 백업을 클릭합니다.
    2. 클라우드 백업을 클릭합니다.
    3. 화면 상단에 있는 Amazon S3 버튼을 클릭합니다.
    4. 만들기를 클릭합니다.
    5. 클라우드 백업의 작업 이름을 입력합니다.
    6. 다음을 클릭합니다.
    7. Amazon S3가 제공한 다음 정보를 입력합니다.
    • 지역: 드롭다운 메뉴에서 클라우드에 사용된 지역을 선택합니다.
    • 액세스 키: Amazon S3가 제공한 액세스 키를 입력합니다.
    • 개인 키: Amazon S3가 제공한 개인 키를 입력합니다.
    • 원격 경로: 클라우드의 원격 경로를 입력합니다. 이것은 보통 버킷 이름입니다.
    1. 다음을 클릭합니다.
    2. 유형 필드에서 다음 옵션 중 하나를 선택합니다.
    • 업로드: My Cloud Mirror 데이터를 Amazon S3 버킷에 백업합니다.
    • 다운로드: Amazon S3 버킷을 My Cloud Mirror 서버에 백업합니다.
    1. 백업 유형 필드의 드롭다운 메뉴에서 수행할 백업의 유형을 선택합니다. 옵션은 다음과 같습니다.
    • 기존 파일 덮어쓰기: 소스 파일과 이름이 동일한 대상 폴더의 파일을 덮어씁니다.
    • 전체 백업: 백업을 수행할 때마다 모든 백업 데이터가 들어 있는 별도의 폴더를 만듭니다.
    • 증분 백업: 대상 파일보다 새 버전인 소스 파일로 파일을 덮어씁니다.
    1. 다음을 클릭합니다.
    2. 로컬 경로 필드에 My Cloud Mirror 장치의 백업 경로를 입력합니다.
    • 찾아보기를 클릭하여 장치의 백업 위치를 찾습니다.
      (예: Volume_1/backup)
    1. 을(를) 클릭하여 자동 업데이트 기능을 활성화합니다. 그러면 만든 일정에 따라 백업이 자동으로 업데이트됩니다.
    • 자동 업데이트 기능을 활성화하려면:

      • 자동 업데이트 일정을 선택합니다. &apos;매일&apos;, &apos;매주&apos; 또는 &apos;매월&apos; 중에서 선택합니다.

      • 드롭다운 메뉴에서 자동 업데이트 시간을 선택합니다(매일 옵션).

      • 드롭다운 메뉴에서 자동 업데이트 날짜와 시간을 선택합니다(매주 또는 매월 옵션).

    1. 다음을 클릭합니다.
    2. 설정을 검토한 후 마침을 클릭합니다. Amazon S3 클라우드 백업이 만들어집니다.

참고: Amazon S3 클라우드 백업 버킷에 액세스하여 장치 백업을 봅니다.
