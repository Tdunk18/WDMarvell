원격 액세스 상태 확인

언제든지 다음 사항을 확인할 수 있습니다.

    • My Cloud Mirror 장치가 원격 액세스를 지원하도록 구성되어 있는지 여부
    • My Cloud Mirror 장치에 대한 원격 액세스가 직접 연결을 통해 활성화되는지 아니면 릴레이 연결을 통해 활성화되는지 여부
    • 현재 특정 사용자의 특정 모바일 장치가 클라우드에 원격으로 액세스하도록 설정되어 있는지 여부

다음 단계를 사용하여 모바일 액세스 상태를 확인합니다.

    1. 클라우드 액세스를 클릭합니다.
    2. 왼쪽 창에서 사용자 이름을 클릭합니다.
      다음 표에는 모바일 액세스 필드와 가능한 필드 값이 나와 있습니다.

섹션

필드/옵션 이름

값 및 설명

클라우드 액세스 프로파일

연결 상태

  • 연결됨(직접): 모바일 액세스가 직접 연결을 통해 설정되어 있습니다.

  • 연결됨(릴레이): 모바일 액세스가 릴레이 연결을 통해 설정되어 있습니다.

  • 실패함: My Cloud Mirror에서 모바일 액세스 연결을 설정할 수 없습니다.

사용자 이름의 클라우드 장치

장치 또는 앱 이름

  • 연결됨: 장치에서 클라우드에 원격으로 액세스할 수 있습니다.

  • 대기 중: 장치 활성화를 기다리는 중입니다.

  • 코드 만료: 활성화 코드가 만료되었습니다.

  • 비활성화됨: 클라우드 서비스 옵션이 꺼져 있습니다. 클라우드 액세스 켜기/끄기를 참조하십시오.

  • 연결 실패: 원격 연결에 실패했습니다. 네트워크 연결 문제를 해결하고 최신 펌웨어가 설치되어 있는지 확인하여 문제를 해결할 수 있습니다.

 

참고: 다른 사용자에 대한 모바일 액세스 설정을 보려면 왼쪽 열에서 해당 사용자의 사용자 이름을 클릭하십시오. 보기가 해당 사용자의 클라우드 장치 정보로 새로 고쳐집니다.
