DLNA란?

 DLNA(Digital Living Network Alliance)는 가정의 엔터테인먼트 장치 간에 홈 네트워크를 통해 내용을 공유할 수 있도록 하기 위해 가전 제품 제조업체에서 사용하는 표준입니다. 예를 들어 TV가 DLNA 인증 장치인 경우 대형 화면의 TV에서 파일을 보고 재생할 수 있도록 My Cloud Mirror를 구성할 수 있습니다.
