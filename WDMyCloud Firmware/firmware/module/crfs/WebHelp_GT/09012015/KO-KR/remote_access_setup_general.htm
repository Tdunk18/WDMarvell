클라우드 액세스 설정

스마트폰, 태블릿 또는 컴퓨터에서 원격으로 클라우드에 액세스하려면 My Cloud에 로그인해야 합니다. (기본적으로 컴퓨팅 장치의 원격 액세스는 홈 네트워크에 연결된 NAS 장치에 대한 액세스를 제공하는 My Cloud 중앙 서버를 통해 활성화됩니다.) 클라우드에 원격으로 액세스하는 방법은 모바일 장치(스마트폰 및 태블릿)와 컴퓨터 간에 약간 다릅니다. 그러나 어떤 경우든 각 사용자는 My Cloud에 로그인해야 합니다.

다음 단계를 사용하여 원격 액세스를 설정합니다.

    1. 장치에서 클라우드 액세스가 활성화되어 있는지 확인합니다.   어떻게 해야 합니까?

       

      클라우드 액세스를 활성화하려면:

      1. 설정을 클릭합니다.
      2. 왼쪽 창에서 일반을 클릭합니다.
      3. 클라우드 액세스 영역에서 을 클릭하여 클라우드 서비스를 활성화합니다.

       

    2. 클라우드 액세스를 클릭합니다.
    3. 왼쪽 창에서 클라우드 액세스를 허용할 사용자를 클릭합니다.
    4. <user name>의 My Cloud 로그인 섹션에 입력된 이메일 주소가 없는 경우 이메일 주소를 입력한 후 을 클릭합니다. 그러면 사용자에게 My Cloud 등록 방법이 들어 있는 이메일이 전송됩니다.
    1. 다음 작업 중 하나를 수행합니다.
    • 관리자 자신에 대한 등록 프로세스를 계속합니다. 어떻게 해야 합니까?

       

      등록 프로세스를 계속하려면

      1. 4단계에서 제공한 이메일 계정을 확인하고 이메일 알림을 봅니다.
      2. 이메일 내에서 제공된 링크를 클릭하고 지침에 따라 등록을 완료합니다.

       

    • 사용자에게 웹 계정이 설정되었으니 등록 프로세스를 계속할 수 있음을 알립니다. 어떻게 해야 합니까?
      참고: My Cloud Mirror 드라이브에 대한 원격 액세스 권한을 모바일 장치에 부여하는 또 다른 방법이 있습니다. 모바일 장치에 원격 액세스 권한 부여(대체 방법)를 참조하십시오.

       

      사용자에게 연락하여 원격 액세스 내용을 설정했으니 다음을 수행하여 등록 프로세스를 계속해야 함을 알립니다.

      1. 사용자의 이메일 계정(4단계에서 제공한 이메일 주소)을 확인합니다.
      2. My Cloud에서 이메일 알림을 엽니다.
      3. 이메일에 포함된 링크를 클릭하고 제공된 지침에 따라 등록을 완료합니다.
