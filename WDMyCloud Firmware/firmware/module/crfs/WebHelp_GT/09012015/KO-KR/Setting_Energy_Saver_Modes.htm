절전 모드 설정

다음 단계를 사용하여 My Cloud Mirror 장치의 에너지 소비를 줄일 수 있습니다.

    1. 설정을 클릭합니다.
    2. 일반을 클릭합니다.
    3. 절전 영역에서 다음 옵션을 선택하여 장치의 에너지 소비를 설정합니다.

모드

설명

드라이브 대기

장치의 절전 모드를 활성화/비활성화합니다. 켜기가 선택된 경우 드라이브 절전 모드가 활성화됩니다.

  • 을 클릭하여 드라이브 절전을 활성화/비활성화합니다.

LED

장치의 LED를 활성화/비활성화합니다. 켜기가 선택된 경우 LED가 비활성화됩니다.

  • 을 클릭하여 LED를 활성화/비활성화합니다.

전원 복구

정전 때문에 갑자기 종료된 경우 장치를 자동으로 다시 시작합니다. 켜기가 선택된 경우 전원 복구가 활성화됩니다.

  • 을 클릭하여 전원 복구를 활성화/비활성화합니다.

전원 일정

장치가 종료되는 시점을 알려주는 일정을 설정할 수 있습니다.

  • 을 클릭하여 전원 일정을 활성화/비활성화합니다.
  • 구성을 클릭하여 장치의 일정을 설정합니다.

웹 액세스 제한 시간

장치에서 자동으로 로그아웃할 기본 시간을 설정합니다.

  • 드롭다운 메뉴에서 웹 액세스 제한 시간을 선택합니다.
