수동으로 업데이트 확인 및 설치

My Cloud Mirror 장치가 펌웨어를 자동으로 업데이트하도록 설정되지 않은 경우(자동으로 My Cloud Mirror 업데이트 참조) 수동으로 업데이트를 확인하여 설치할 수 있습니다.

다음 단계를 사용하여 업데이트를 확인하고 설치합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 펌웨어 업데이트를 클릭합니다.
    3. 사용 가능한 업데이트 섹션에서 업데이트 확인을 클릭합니다.
      그러면 My Cloud Mirror 장치에서 업데이트를 사용할 수 있는지 여부를 확인합니다. "새 업데이트 없음"이라는 메시지가 반환되면 이 절차를 완료한 것입니다.
      반환된 상태에 사용 가능한 업데이트가 나열되어 있으면 4단계를 진행합니다.
    4. 출시 정보를 클릭하여 새 브라우저 창에서 사용 가능한 업데이트에 대한 세부 정보를 확인합니다. 출시 정보 검토를 마치면 해당 브라우저 창을 닫습니다.
    5. 업데이트 설치를 클릭합니다.
    6. 확인을 클릭하여 My Cloud Mirror 장치의 재부팅을 확인합니다.
      참고: 클라우드에 현재 액세스 중인 모든 사용자에 대한 서비스가 중단됩니다.
