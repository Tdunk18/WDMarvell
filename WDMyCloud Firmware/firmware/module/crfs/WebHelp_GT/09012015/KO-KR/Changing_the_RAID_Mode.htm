RAID 모드 변경

다음 단계를 사용하여 My Cloud Mirror 장치에서 현재 RAID 모드를 변경합니다.

경고: RAID 모드를 변경하면 모든 데이터가 삭제됩니다.

    1. 스토리지를 클릭합니다.
    2. RAID를 클릭합니다.
    3. 화면 하단에 있는 RAID 모드 변경을 클릭합니다.
    4. My Cloud Mirror 서버에 사용할 RAID 모드를 선택합니다. 옵션은 다음과 같습니다.

모드

설명

JBOD

RAID 구성에 없지만 별도의 논리 볼륨으로 관리되는 하나 이상의 드라이브 사용.

스패닝

큰 논리 볼륨 하나를 만들기 위한 선형의 드라이브 조합.

RAID 0

성능이 개선되도록 드라이브 둘 이상의 동작을 결합하여, 데이터가 여러 하드 드라이브로 스트라이프되고, 데이터 읽기와 기록이 가속화됩니다.

RAID 1

두 동시 데이터 사본이 별도의 하드 드라이브에 기록되어, 이 중 하나가 실패할 경우 결함 있는 하드 드라이브를 교환할 때까지 다른 하나가 데이터 읽기/쓰기를 계속할 수 있습니다.

    1. <RAID 모드>로 전환 확인란을 선택합니다.
    2. 다음을 클릭합니다.
    3. 하드 드라이브의 상태를 검토한 후 다음을 클릭합니다.
    4. 마침을 클릭합니다.
    5. 볼륨 슬라이더를 사용하여 선택된 RAID 모드에 전용으로 사용할 공간 용량을 선택합니다.

      참고: RAID 선택에 전체 볼륨을 사용하지 않도록 선택하려는 경우 스패닝으로 남은 디스크 공간을 구성합니다. 확인란을 선택하여 남은 디스크 공간을 스패닝으로 구성할 수 있습니다.
    1. 다음을 클릭합니다.
    2. 을(를) 선택하여 RAID 구성에 대해 자동 복원 옵션을 켠 후 다음을 클릭합니다.

      참고: JBOD, 스패닝, RAID 0 모드의 경우에는 이 화면이 표시되지 않습니다.
    1. RAID의 변경 내용을 검토하여 변경 내용이 올바른지 확인한 후 다음을 클릭합니다.
    2. 경고 화면을 검토합니다.
      경고: 모드를 변경하면 모든 데이터가 지워집니다.
    1. 마침을 클릭합니다. 요청된 하드 드라이브 변경이 시작됩니다.
    2. 마침을 클릭합니다. 이제 하드 드라이브가 선택된 모드로 다시 구성되었습니다.
