공유 액세스 변경

사용자 화면 또는 공유 화면에서 사용자의 액세스 권한을 변경할 수 있습니다. 다음 단계를 사용하여 선택된 공유에 대한 사용자의 액세스 권한을 변경합니다.

사용자 화면에서 공유 액세스 변경

    1. 사용자를 클릭합니다.
    2. 왼쪽 창에서 사용자를 선택합니다.
    3. 공유 액세스 영역에서 공유에 해당하는 적절한 아이콘을 클릭하여 사용자의 액세스 권한을 변경합니다.
    • 읽기/쓰기 액세스
    • 읽기 전용 액세스
    • 액세스 거부

 

공유 화면에서 공유 액세스 변경

    1. 공유를 클릭합니다.
    2. 왼쪽 창에서 공유를 선택합니다.
    3. 공용 액세스 옵션이 꺼져 있는지 확인합니다. 공용 액세스를 사용하면 모든 사람이 해당 공유를 볼 수 있습니다.
    • 공용 액세스가 켜져 있는 경우 을 클릭합니다.
    1. 사용자 액세스 섹션에서 공유에 해당하는 적절한 아이콘을 클릭하여 사용자의 액세스 권한을 변경합니다.
    • 읽기/쓰기 액세스
    • 읽기 전용 액세스
    • 액세스 거부
