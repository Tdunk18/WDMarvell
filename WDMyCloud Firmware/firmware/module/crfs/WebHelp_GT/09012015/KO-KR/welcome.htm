시작

My Cloud Mirror 장치 시스템은 무선 라우터에 연결되어 있을 때 아무 컴퓨터, 태블릿 또는 스마트폰에서 미디어와 파일에 안전하게 무선으로 액세스할 수 있도록 해주는 고성능 시스템입니다. My Cloud Mirror 장치는 개인 클라우드의 중앙 구성 요소로서 설정, 공유, 유지 관리가 간편하도록 단순하면서도 강력한 인터페이스를 제공합니다.

알아야 할 사항

개인 클라우드의 활용을 극대화하기 위해 My Cloud Mirror 장치에 대해 알아두어야 할 몇 가지 중요한 사항이 있습니다.

    • 사용자 역할

또한 장치의 핵심 기능을 활용할 수 있도록 다음 작업을 검토해야 합니다.

    • 스마트폰 또는 태블릿 액세스

    • 랩톱 액세스  

    • My Cloud Mirror 웹 인터페이스 표시

    • 네트워크 외부 사용자와의 내용 공유

    • 파일 업로드

    • 시작(기본 설정)

개인 클라우드의 사용자 역할

개인 클라우드에 액세스하는 모든 사용자는 My Cloud Mirror 드라이브에 대한 사용자 계정을 가지고 있어야 합니다. My Cloud Mirror 소유자는 관리자 권한을 부여하는 특수 사용자 계정(admin)을 가지고 있습니다. 이러한 권한이 있으면 특정 필요에 맞게 장치를 설정 및 구성하고 개인 클라우드에 다른 사용자를 추가할 수 있습니다. 소유자는 장치에 액세스할 수 있는 사용자를 정확하게 지정할 수 있습니다.

참고: 기본적으로, admin 사용자 이름에는 암호가 없습니다. 그러나 언제든지 암호를 추가할 수 있습니다.

중요: My Cloud Mirror 설치 소프트웨어를 실행하거나 시작 화면을 완료하는 동안 관리자 이름을 변경하고 선택적 암호를 설정할 수 있습니다. 이름과 암호는 잘 보관해 두어야 합니다. 이러한 자격 증명 없이는 My Cloud Mirror 웹 인터페이스를 표시할 수 없습니다.

맨 위로

스마트폰 또는 태블릿에서 내용 액세스

다음 두 WD 모바일 앱을 사용하여 스마트폰이나 태블릿에서 My Cloud 장치의 콘텐츠에 액세스할 수 있습니다.

    • My Cloud 앨범: 사진을 보고 관리합니다.

    • My Cloud: 사진, 동영상, 음악, 파일을 보고 재생합니다.

이러한 무료 앱은 WD 교육 센터(http://www.wd.com/setup/wdmycloudmirror) 또는, Apple App Store, Google Play, Windows Store, Amazon.com 등의 앱 사이트에서 구할 수 있습니다.

관리자의 모바일 장치 액세스 워크플로

My Cloud Mirror 소유자/관리자의 기본 워크플로는 다음과 같습니다.

1

My Cloud Mirror 장치를 원격 액세스용으로 구성하고 My Cloud 로그인을 등록합니다. 방법

2

WD 모바일 앱을 다운로드합니다. 방법

3

WD 앱을 실행하고 안전하게 액세스하기 위한 1회 설정을 수행합니다. 방법

4

이제 내용을 보고 재생합니다.

기타 사용자의 모바일 장치 액세스 워크플로

참고: My Cloud Mirror에 사용자를 추가할 때 My Cloud에 사용자의 이메일 주소도 제공하여 My Cloud 로그인을 만들게 됩니다. 일단 만들어지면 등록 프로세스를 계속 진행할 수 있도록 My Cloud에 대한 링크가 들어 있는 이메일이 새 사용자의 이메일 주소로 전송됩니다.

관리자가 아닌 사용자의 기본 워크플로는 다음과 같습니다.

1

My Cloud에서 전송된 메시지 내의 해당 링크를 클릭하여 등록 프로세스를 계속합니다.

2

WD 모바일 앱을 다운로드합니다. 방법

3

WD 앱을 실행하고 안전하게 액세스하기 위한 1회 설정을 수행합니다. 방법

4

내용을 보고 재생합니다.

랩톱에서 내용 액세스

My Cloud 데스크톱 응용 프로그램을 사용하여 Mac 또는 PC에서 My Cloud Mirror 장치의 콘텐츠에 액세스할 수 있습니다. My Cloud를 사용하면 My Cloud Mirror, My Book Live 드라이브를 포함하여 홈 네트워크에 연결된 WD NAS 장치에 저장되어 있는 사진, 동영상, 음악, 파일에 원격으로 액세스할 수 있습니다. WD 모바일 앱에서와 마찬가지로 My Cloud 로그인이 있어야 내용에 액세스할 수 있습니다.

My Cloud 데스크톱 응용 프로그램을 http://www.wd.com/setup/wdmycloudmirror에서 다운로드하십시오.

단계별 지침은 랩톱에서 내용 액세스를 참조하십시오.

My Cloud Mirror 인터페이스 표시

My Cloud Mirror 장치 사용자 인터페이스를 사용하여 개인 클라우드를 설정하고 관리할 수 있습니다. 이 인터페이스는 Mac 또는 PC에서 실행되는 웹 브라우저에 표시됩니다. 

웹 인터페이스를 표시하려면

    1. 브라우저 창을 엽니다.
    2. 주소 표시줄에 http://
      (예: http://wdmycloudmirror
      )를 입력합니다. 참고: 이 URL로 웹 인터페이스가 표시되지 않으면 다음 URL을 입력해 보십시오. http://wdmycloudmirror.local
