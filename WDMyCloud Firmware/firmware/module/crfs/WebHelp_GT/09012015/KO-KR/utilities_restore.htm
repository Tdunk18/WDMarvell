기본 설정 복원

시스템 복원을 사용하면 관리자가 My Cloud Mirror 장치에서 전체 초기 설정 복원을 수행할 수 있습니다. 이 옵션은 데이터 손실 없이 장치를 초기 기본 설정으로 복원합니다.

복원을 수행하기 전에 현재 구성을 저장하고 내보낸(방법) 다음 나중에 가져올(방법) 수 있습니다.

중요: 복원하는 동안 클라우드에 현재 액세스 중인 사용자에 대한 서비스가 중단됩니다. 복원하는 동안 My Cloud Mirror 장치를 분리하지 마십시오.

다음 단계를 사용하여 My Cloud Mirror 장치를 복원합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 유틸리티를 클릭합니다.
    3. 시스템 복원 영역에서 시스템 복원을 클릭합니다.
    4. 구성 메시지를 검토한 후 확인을 클릭합니다. 장치가 다시 부팅됩니다. 이 다시 부팅 프로세스 중에 장치를 분리하지 마십시오. 다시 부팅이 완료되면 대시보드를 실행합니다.
