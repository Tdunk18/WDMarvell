클라우드 백업이란?

클라우드 백업을 사용하면 인터넷에서 그리고 다양한 모바일 장치를 통해 액세스할 수 있는 원격 백업을 만들 수 있습니다. My Cloud Mirror 장치는 다음 클라우드 서비스를 사용하여 원격 백업을 만듭니다.

    • ElephantDrive: ElephantDrive는 파일을 원격으로 안전하게 자동으로 백업해주는 클라우드 백업 서비스입니다.
    • Amazon S3: Amazon S3(Simple Storage Service)는 웹상의 어디에서든 언제든지 용량과 관계 없이 데이터를 저장하고 검색하는 데 사용할 수 있는 온라인 파일 스토리지 웹 서비스입니다.
