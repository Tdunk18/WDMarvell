SSH 활성화/비활성화

SSH(보안 셸 프로토콜)를 통해 명령줄 작업을 수행하고 클라우드에 안전하게 액세스할 수 있도록 My Cloud Mirror 장치를 구성할 수 있습니다.

다음 단계를 사용하여 SSH를 활성화/비활성화합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 네트워크를 클릭합니다.
    3. 네트워크 서비스 영역에서 을 클릭하여 SSH를 활성화 또는 비활성화합니다.

    참고: 시스템이 SSHD라는 사용자를 만듭니다.
