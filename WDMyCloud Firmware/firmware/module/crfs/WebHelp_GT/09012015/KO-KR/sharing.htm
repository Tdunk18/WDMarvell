네트워크 외부 사용자와 내용 공유

컴퓨터에서 실행 중인 My Cloud 데스크톱 응용 프로그램을 사용하여 네트워크 외부 사용자와 내용을 공유할 수 있습니다.

다음 단계를 사용하여 콘텐츠를 공유합니다.

    1. 아직 My Cloud 데스크톱 응용 프로그램을 다운로드하지 않은 경우 WD 교육 센터((http://www.wd.com/setup/wdmycloudmirror)에서 다운로드하여 Mac 또는 PC에 설치합니다.
    2. 컴퓨터에서 My Cloud 데스크톱 응용프로그램을 실행합니다.
    3. My Cloud Mirror 장치를 두 번 클릭합니다.
    4. 왼쪽 창에서 필요에 따라 폴더를 확장하여 공유할 내용을 표시합니다.
    5. 오른쪽 창에서 단일 항목을 클릭하거나, Shift 키를 누른 채 클릭하여 여러 항목을 선택합니다.
    6. 선택 항목을 마우스 오른쪽 버튼으로 클릭한 다음 이메일을 선택합니다.
    7. 필요한 대로 이메일 창을 완성하여 메시지를 보냅니다.
