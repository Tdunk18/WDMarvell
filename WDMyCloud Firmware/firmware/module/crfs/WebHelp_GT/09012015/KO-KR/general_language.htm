표시되는 언어 설정

My Cloud Mirror 웹 인터페이스에 표시되는 언어를 선택할 수 있습니다.

참고: 인터페이스를 제대로 표시하려면 컴퓨팅 장치에 적절한 글꼴이 설치되고 국가별 설정이 구성되어 있어야 합니다.

다음 단계를 사용하여 My Cloud Mirror 장치의 표시 언어를 설정합니다.

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 일반을 클릭합니다.
    3. 언어 필드의 드롭다운 메뉴에서 언어를 선택합니다.
    4. 저장을 클릭합니다.
