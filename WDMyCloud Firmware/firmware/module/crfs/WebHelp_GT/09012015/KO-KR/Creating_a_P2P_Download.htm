P2P 다운로드 만들기

P2P 다운로드 앱을 사용하면 My Cloud Mirror로 P2P 다운로드 일정을 예약할 수 있습니다. 다음 단계를 사용하여 P2P 다운로드 일정을 예약합니다.

    1. 앱을 클릭합니다.
    2. P2P 다운로드를 클릭합니다.
    3. 을 클릭하여 P2P 다운로드를 활성화합니다.
    • 구성을 클릭하여 P2P 다운로드 구성을 변경합니다.
    1. 다운로드에 대한 URL 또는 Torrent 파일을 입력합니다.
    • URL이 있는 경우:

      • URL에서 Torrent 추가 필드에 다운로드할 Torrent 파일의 URL을 입력합니다.

      • Torrent 추가를 클릭합니다. P2P 다운로드가 만들어져서 P2P 다운로드 대기열에 추가됩니다.

    • Torrent 파일이 있는 경우:

      • 클릭하여 파일 업로드를 선택합니다.

      • 업로드할 Torrent 파일을 선택한 후 열기를 클릭합니다. P2P 다운로드가 만들어져서 P2P 다운로드 대기열에 추가됩니다.
