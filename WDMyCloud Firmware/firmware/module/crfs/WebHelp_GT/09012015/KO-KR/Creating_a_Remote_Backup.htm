원격 백업 만들기

원격 백업을 사용하면 My Cloud Mirror 장치를 네트워크의 다른 서버나 네트워크 외부의 다른 서버에 백업할 수 있습니다.  다음 단계를 사용하여 My Cloud Mirror 서버의 원격 백업을 만듭니다.

    1. 백업을 클릭합니다.
    2. 원격 백업을 클릭합니다.
    3. 작업 만들기 버튼을 클릭합니다.
    4. 작업 이름을 입력합니다.
    5. 원격 서버 드롭다운에서 서버를 선택합니다.
    6.  원격 IP 필드에 원격 백업 장치의 IP 주소를 입력합니다.
      (예: 192.168.1.16)
    7. 암호 필드에 원격 백업 장치의 암호를 입력합니다.
    8. SSH 사용자 이름을 입력합니다.
    9. SSH 암호 확인 필드에 암호 SSH 암호를 입력합니다.
    10. 찾아보기를 클릭하여 소스 폴더를 선택합니다.
    11. 원격 백업 폴더의 목록에서 보호하려는 하나 이상의 공유 디렉터리를 확인합니다.
    12. 확인을 클릭합니다.
    13. 원격 백업 디렉터리를 선택하려면 찾아보기를 클릭합니다.
    14. 디렉터리를 선택하고 확인을 클릭합니다.
    15. 드롭다운 메뉴에서 백업 유형을 선택합니다.
    16. 백업 일정을 설정하려면 반복 옆에 있는 토글 아이콘 을 클릭합니다.
    17. 전송 암호화를 설정하려면 토글 아이콘 을 클릭합니다.
    18. 만들기를 클릭합니다.
