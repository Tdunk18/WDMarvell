네트워크 모드 변경

기본적으로 네트워크 모드는 DHCP입니다. 즉 My Cloud Mirror 장치가 네트워크에서 IP 주소와 기타 설정을 자동으로 가져옵니다. IP 주소를 수동으로 설정하려면 다음 절차를 완료하여 모드를 변경하십시오.

다음 단계를 사용하여 네트워크 모드를 변경합니다(DHCP에서 정적으로).

    1. 설정을 클릭합니다.
    2. 왼쪽 창에서 네트워크를 클릭합니다.
    3. 네트워크 서비스 영역의 IPv4 네트워크 모드에서 정적을 클릭합니다.
    4. LAN 설정 마법사의 단계를 따릅니다.
