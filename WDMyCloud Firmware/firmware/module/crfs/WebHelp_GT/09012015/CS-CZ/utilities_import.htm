Import systémových nastavení

Jako správce máte možnost naimportovat dříve uložený konfigurační soubor. V zásadě tato úloha nahradí aktuální konfiguraci dříve uloženou verzí.

Důležité: Import konfiguračního souboru po obnovení výchozího systémového nastavení neobnoví data uživatelů ani mobilních zařízení. Tyto informace budete muset zadat znovu. Po dokončení importu se zařízení My Cloud Mirror restartuje a přeruší se tak služby pro uživatele, kteří aktuálně přistupují ke cloudu.

Chcete-li importovat svá nastavení systému, postupujte následovně.

    1. Klikněte na položku Nastavení.
    2. V levém podokně klikněte na položku Nástroje.
    3. V oblasti Konfigurace systému klikněte na položku Importovat soubor.
    4. Vyberte umístění konfiguračního souboru a poté klikněte na Otevřít. Konfigurace bude obnovena.
