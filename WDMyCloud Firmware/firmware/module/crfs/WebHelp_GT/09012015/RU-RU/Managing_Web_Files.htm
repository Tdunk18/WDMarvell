Управление веб-файлами

Средство веб-просмотра файлов — это приложение, которое позволяет просматривать файлы на устройстве My Cloud Mirror и управлять ими.

Описанные далее действия позволяют управлять веб-файлами.

    1. Нажмите Приложения.
    2. Нажмите Средство веб-просмотра файлов. На панели справа отображаются тома файлов, доступные на устройстве.

Просмотр веб-файлов

Тома, папки и файлы отображаются на панели справа в окне средства веб-просмотра файлов. Для просмотра содержимого любого тома или папки дважды щелкните данный элемент.

Управление веб-файлами

Действия, которые можно выполнить с файлами и папками, показаны в правом верхнем углу средства веб-просмотра файлов. Выберите одно из описанных далее действий для выполнения с веб-файлами.

Значок

Функция

Описание

Копировать

Это действие предназначено для копирования выбранного файла в другое место. Описанные далее действия позволяют скопировать файл в новое место.

    1. Выберите файл, который нужно скопировать.
    2. Нажмите .
    3. Выберите место, в которое нужно поместить копию файла.
    4. Нажмите кнопку ОК. Файл копируется в выбранное место назначения.

Переместить

Это действие предназначено для перемещения файла из одного места в другое. Описанные далее действия позволяют переместить файл в новое место.

    1. Выберите файл, который нужно переместить.
    2. Нажмите .
    3. Выберите место, в которое нужно переместить файл.
    4. Нажмите кнопку ОК. Файл перемещается в выбранное место назначения.

Отправка

Это действие предназначено для выгрузки файла с компьютера или другого устройства на сервер My Cloud Mirror.
Описанные далее действия позволяют выгрузить файл на устройство.

    1. На правой панели средства веб-просмотра файлов выберите место назначения для выгружаемого файла.
    2. Нажмите .
    3. На экране «Выгрузка» нажмите значок выбора файла
    4. Найдите и выделите файл, который нужно выгрузить на устройство.
    5. Нажмите Открыть.
    6. Нажмите Выгрузка.
    • Если нужно выгрузить дополнительные файлы, выполняйте действия 3 и 4, пока все файлы не будут выгружены.
    1. Нажмите Закрыть. Выгруженные файлы появятся в папке назначения.

Загрузить

Это действие предназначено для загрузки файла с устройства My Cloud Mirror на компьютер или другие устройства.
Описанные далее действия позволяют загрузить файл на компьютер или другие устройства.

    1. На правой панели средства веб-просмотра файлов выберите файл, который нужно загрузить.
    2. Нажмите .
    3. Выбранный файл загружается на компьютер или другие устройства.

Переименовать

Это действие предназначено для изменения имени выбранного файла на устройстве My Cloud Mirror. Описанные далее действия позволяют переименовать файл.

    1. На правой панели средства веб-просмотра файлов выберите файл, который нужно переименовать.
    2. Нажмите .
    3. В поле «Переименовать» переименуйте файл.
    4. Нажмите Сохранить. Файл переименовывается.

Удалить

Это действие предназначено для удаления выбранного файла с устройства My Cloud Mirror. Описанные далее действия позволяют удалить файл.

    1. На правой панели средства веб-просмотра файлов выберите файл, который нужно удалить.
    2. Нажмите .
    3. Нажмите кнопку ОК, чтобы подтвердить свое намерение удалить выбранный файл. Выбранный файл удаляется с устройства.

Обновить

Данное действие предназначено для обновления списка файлов, чтобы показать все новые файлы, которые были добавлены с момента последнего обновления или открытия списка файлов.

  • Нажмите . Список файлов обновляется.
