Включение функции перенаправления портов

Если удаленный доступ к вашему облаку осуществляется посредством подключения с ретрансляцией, можно повысить производительность сети за счет включения функции перенаправления портов.
Как узнать, что для получения мобильного доступа используется подключение с ретрансляцией?

Состояние подключения «Подключено (ретрансляция)» свидетельствует о том, что используется подключение с ретрансляцией. Сведения о том, как узнать о текущем состоянии подключения, см. в разделе Проверка состояния удаленного доступа.

 

Внимание! Прежде чем приступить к выполнению этой процедуры, необходимо узнать действующий диапазон номеров портов, которые можно использовать для маршрутизатора, и выбрать соответствующие порты для HTTP и HTTPS. Допустимый диапазон номеров портов: 1024–66535. Обычно порты назначаются следующим образом: 80 для HTTP и 443 для HTTPS.

Описанные далее действия позволяют включить функцию перенаправления портов.

    1. Нажмите Настройки.
    2. На панели слева нажмите Общие.
    3. В разделе «Доступ к облаку» нажмите , чтобы включить доступ к облаку, если нужно.
    4. Нажмите Настроить.
    5. В поле «Возможность подключения» выберите значение Вручную.
    6. В поле «Внешний порт 1» введите номер порта, через который будут выполняться HTTP-сеансы.
    7. В поле «Внешний порт 2» введите номер порта, через который будут выполняться HTTPS-сеансы.
    8. Нажмите Применить.
    9. Настройте маршрутизатор для использования портов, указанных при выполнении шагов 6 и 7.
