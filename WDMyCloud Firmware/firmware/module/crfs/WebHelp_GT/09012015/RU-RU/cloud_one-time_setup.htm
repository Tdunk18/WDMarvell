Выполнение одноразовой настройки для получения безопасного доступа

С помощью этой процедуры объясняется способ установки безопасного подключения между смартфоном или планшетным компьютером и устройством My Cloud Mirror. Используйте соответствующую инструкцию для своего мобильного устройства:

    • Смартфон или планшетный компьютер под управлением ОС Android  

    • iPhone или iPad

    • телефон или планшетный компьютер с ОС Windows.

    • Компьютер под управлением ОС Windows или компьютер Mac

Получение безопасного доступа для смартфона или планшетного компьютера с ОС Android

    1. Подключите свой смартфон или планшетный компьютер, выполнив следующие действия.

      1. На своем мобильном устройстве нажмите Настройки.

      2. Нажмите Беспроводные и проводные сети.

      3. Нажмите Параметры Wi-Fi.

      4. Выберите домашнюю сеть.

      5. Введите пароль сети и нажмите Подключиться.

    1. Запустите приложение WD.
    2. Нажмите Подключиться.
    3. Нажмите на название устройства My Cloud Mirror.
    4. Нажмите стрелку раскрывающегося списка и выберите имя пользователя.
    5. Выполните одно из следующих действий:

    • если для этого пользователя задан пароль, введите его и нажмите Вход;

    • если пароль для этого пользователя не задан, нажмите Вход.

    1. (Необязательно) Чтобы получить доступ к какому-либо содержимому, нажмите название ресурса и перейдите к нужному файлу.

Получение безопасного доступа для устройств iPhone или iPad

    1. Подключите свой смартфон или планшетный компьютер, выполнив следующие действия.

    1. На своем мобильном устройстве нажмите Настройки.

    2. Нажмите Wi-Fi.

    3. Нажмите Параметры Wi-Fi.

    4. Выберите домашнюю сеть.

    5. Введите пароль сети и нажмите Присоединиться.

    1. Запустите приложение WD.
    2. Нажмите Подключиться.
    3. Нажмите знак «плюс» (+) рядом с именем устройства My Cloud Mirror.
    4. Нажмите необходимое имя пользователя.
      Если для пользователя задан пароль, введите его.
    5. (Необязательно) Чтобы получить доступ к какому-либо содержимому, нажмите название ресурса и перейдите к нужному файлу.

Получение безопасного доступа для смартфона или планшетного компьютера с ОС Windows

    1. Подключите свой смартфон или планшетный компьютер, выполнив следующие действия.

    1. На своем вычислительном устройстве нажмите Настройки.

    1. Нажмите Wi-Fi.

    2. Выберите домашнюю сеть.

    3. Введите пароль сети и нажмите кнопку Готово.

    1. Запустите приложение WD.
    2. Нажмите Подключиться.
    3. Нажмите на название диска My Cloud Mirror.
    4. Нажмите стрелку раскрывающегося списка и выберите имя пользователя.
      Если для пользователя задан пароль, введите его.
    5. (Необязательно) Чтобы получить доступ к какому-либо содержимому, выполните следующие действия.

    1. В окне «Устройства» нажмите имя диска My Cloud Mirror.

    2. Выберите ресурс и перейдите к нужному содержимому.
