Добавление учетной записи пользователя

Описанные далее действия позволяют добавить пользователя на устройствоMy Cloud Mirror.

    1. Нажмите Пользователи.
    2. Щелкните значок Добавить пользователя:
    3. Введите имя пользователя. Это имя используется для входа в систему.
    4. Введите имя. Это имя данного пользователя.
    5. Введите фамилию. Это фамилия данного пользователя.
    6. Введите адрес электронной почты. Это адрес электронной почты данного пользователя. Этот адрес также используется для создания учетной записи My Cloud.
    7. Введите пароль для учетной записи пользователя.
    8. Повторно введите пароль, чтобы подтвердить его.
    9. Нажмите Применить.
    10. Нажмите . При этом значение в поле пароля изменяется на «ВКЛ» и активируются поля пароля.
    • Если вы не хотите назначать пароль, убедитесь в том, что в поле пароля указано значение «ВЫКЛ», и нажмите кнопку Далее.
