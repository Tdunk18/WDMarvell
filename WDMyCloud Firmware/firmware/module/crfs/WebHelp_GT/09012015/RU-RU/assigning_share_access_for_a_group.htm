Назначение доступа к ресурсу для группы

После создания группы можно определить для ее членов уровень доступа. С помощью описанных далее действий можно назначить для группы доступ к ресурсу.

    1. Нажмите Пользователи.
    2. Нажмите Группы
    3. На панели слева выберите группу, для которой нужно назначить доступ. Информация о группе отображается в области «Профиль группы».
    4. В области «Доступ к ресурсу» выберите уровень доступа для каждого ресурса на устройстве.
    • доступ для чтения и записи. Выберите этот вариант, чтобы разрешить полный доступ к выбранной информации.
    • доступ только для чтения. Этот вариант позволяет члену группы считывать выбранную информацию, но не позволяет изменять ее.
    • запретить доступ. У члена группы нет доступа к этой информации.

Теперь ресурсы назначены выбранной группе.
