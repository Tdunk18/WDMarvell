Создание группы

Группы представляют собой коллекции учетных записей пользователей, которыми можно управлять. Описанные далее действия позволяют создать группу.

    1. Нажмите Пользователи.
    2. Нажмите Группы.
    3. Нажмите на значок Добавитьгруппы.
    4. Введите Название группы для новой группы. Длина названия должна составлять не более 16 символов, в названии нельзя использовать пробелы.
    5. Выберите пользователей, которых нужно добавить в новую группу.
    6. Нажмите Применить. Новая группа создается и отображается в левом столбце на экране.
