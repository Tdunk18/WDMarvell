Подключение устройства My Cloud Mirror к сети

В этом разделе объясняется, как подключить диск My Cloud Mirror к домашней сети с помощью кабеля Ethernet. После осуществления этого первоначального подключения можно настроить диск для беспроводного подключения к сети.

Описанные далее действия позволяют выполнить подключение с помощью кабеля Ethernet.

    1. Подключите один конец кабеля к порту Ethernet на задней стороне устройства My Cloud Mirror.
    1. Другой конец кабеля подключите непосредственно к маршрутизатору или порту сетевого коммутатора.
    2. Подключите конец адаптера к разъему питания на задней панели диска.
    3. Подключите адаптер питания к розетке. Устройство My Cloud Mirror включится автоматически.

Внимание! Прежде чем настраивать соединитель, подождите, пока устройство My Cloud Mirror окончательно включится (около трех минут).

После подключения My Cloud Mirror к сети с помощью кабеля Ethernet можно настроить беспроводное подключение через веб-интерфейс. См. Настройка параметров подключения.
