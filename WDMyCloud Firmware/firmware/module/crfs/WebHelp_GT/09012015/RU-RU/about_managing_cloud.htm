Управление персональным облаком

Управление личным облаком состоит из множества задач, большинство из которых нужно выполнять лишь время от времени. В большинстве случаев после выполнения задач, описанных в разделе Начало работы, задачи по управлению облаком будут сведены к минимуму.

В этом разделе задачи по управлению облаком упорядочены по следующим категориям.

    • Пользователи
    • Ресурсы
    • Доступ к облаку

Пользователи

Нажмите одну из ссылок ниже, чтобы узнать, как выполняются соответствующие связанные с пользователями задачи.

    • Добавление учетной записи пользователя. 
    • Добавление или изменение пароля пользователя  
    • Просмотр сведений о пользователях.  
    • Изменение доступа пользователя к ресурсу  
    • Удаление учетной записи пользователя.

    • Создание ресурса  
    • Удаление ресурса

Доступ к облаку

Прежде чем смартфон или планшетный компьютер сможет получать доступ к содержимому вашего устройства, его нужно зарегистрировать как часть экосистемы вашего персонального облака. Вы создаете код, который свяжет ваше мобильное устройство с устройством My Cloud Mirror. Нажмите одну из ссылок ниже, чтобы узнать, как управлять доступом к облаку.

    • Настройка доступа к облаку   
    • Одноразовая настройка для получения безопасного доступа с мобильного устройства  
    • Удаление вычислительного устройства из персонального облака  
    • Проверка состояния удаленного доступа  
    • Повышение производительности за счет перенаправления портов
