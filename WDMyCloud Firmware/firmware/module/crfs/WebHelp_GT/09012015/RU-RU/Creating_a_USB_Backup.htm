Создание резервного копирования по USB

Описанные далее действия позволяют выполнять резервное копирование данных, хранящихся на My Cloud Mirror, на USB-устройство или с него.

    1. Нажмите Резервные копии.
    2. Нажмите Резервные копии USB.
    3. Нажмите Создать задание.
    4. Введите название задания для резервного копирования.
    5. Выберите направление резервного в раскрывающемся меню. Доступны описанные далее варианты.
    • С USB в NAS. Выполняется резервное копирование данных, хранящихся на USB-устройстве, на устройство My Cloud Mirror.
    • С NAS в USB. Выполняется резервное копирование данных, хранящихся на устройстве My Cloud Mirror, на USB-устройство.
    1. Выберите исходную папку для резервного копирования, выполнив для этого следующие действия.
    • Нажмите Обзор и выберите папку на устройстве назначения, из которой нужно выполнять резервное копирование.
    1. Выберите папку назначения для резервного копирования.
    • Нажмите Обзор и выберите папку, в которую нужно выполнять резервное копирование исходного устройства.
    1. Укажите тип резервного копирования в выпадающем меню.
    2. Чтобы автоматически начинать резервное копирование по USB при подключении, нажмите значок переключения .
    3. Нажмите Создать. При этом создается задание для нужного процесса резервного копирования.
    4. Нажмите Сведения в области «Задания резервного копирования по USB», чтобы проверить ход выполнения задания.
