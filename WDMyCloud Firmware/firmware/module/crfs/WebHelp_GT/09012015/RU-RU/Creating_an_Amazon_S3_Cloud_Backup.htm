Создание облачного резервного копирования с помощью Amazon S3

Примечание. Прежде чем создавать резервную копию в облаке, выполните вход в службу Amazon S3 на веб-странице http://aws.amazon.com/s3.

После настройки учетной записи Amazon S3 выполните следующие действия для создания облачной резервной копии.

    1. Нажмите Резервные копии.
    2. Нажмите Облачное резервное копирование.
    3. Нажмите кнопку Amazon S3 в верхней части экрана.
    4. Нажмите Создать.
    5. Введите название задания для облачного резервного копирования.
    6. Нажмите Далее.
    7. Введите следующую информацию, предоставленную Amazon S3.
    • Регион. Выберите регион, используемый для вашего облака, из раскрывающегося меню.
    • Ключ доступа. Введите ключ доступа, предоставленный Amazon S3.
    • Секретный ключ. Введите секретный ключ, предоставленный Amazon S3.
    • Удаленный путь. Введите удаленный путь для вашего облака. Обычно это название вашего контейнера.
    1. Нажмите Далее.
    2. В поле «Тип» выберите один из следующих вариантов.
    • Выгрузка. Выполняется копирование данных, хранящихся на My Cloud Mirror, в контейнер Amazon S3.
    • Загрузка. Выполняется копирование данных, хранящихся в контейнере Amazon S3, на сервер My Cloud Mirror.
    1. В поле «Тип резервного копирования» выберите нужный тип из раскрывающегося меню. Доступны следующие варианты.
    • Перезаписать существующие файлы. Выполняется перезапись файлов в папке назначения, имена которых совпадают с именами исходных файлов.
    • Полное резервное копирование. При каждом выполнении резервного копирования создается отдельная папка, которая содержит все данные резервной копии.
    • Добавочное резервное копирование. Выполняется замена файлов исходными файлами, если они новее, чем файлы в папке назначения.
    1. Нажмите Далее.
    2. В поле «Локальный путь» введите путь для резервной копии на устройстве My Cloud Mirror.
    • Нажмите Обзор, чтобы перейти в местоположение резервной копии на устройстве.
      (Пример: Том_1/резервная_копия.)
    1. Нажмите , чтобы активировать функцию «Автоматическое обновление». При этом автоматически обновляется резервная копия по созданному расписанию.
    • Если вы активировали функцию «Автоматическое обновление», выполните следующие действия.

      • Выберите расписание автоматического обновления: ежедневно, еженедельно или ежемесячно.

      • Выберите время автоматического обновления из раскрывающегося меню (если выбран вариант «Ежедневно»).

      • Выберите дату и время автоматического обновления из раскрывающихся меню (если выбран вариант «Еженедельно» или «Ежемесячно»).

    1. Нажмите Далее.
    2. Проверьте параметры настройки и нажмите Готово. Создается облачная резервная копия с помощью Amazon S3.

ПРИМЕЧАНИЕ. Откройте контейнер облачного резервного копирования в Amazon S3, чтобы увидеть резервные копии своего устройства.
