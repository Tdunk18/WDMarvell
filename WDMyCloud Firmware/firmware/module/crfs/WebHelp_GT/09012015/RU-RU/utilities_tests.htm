Выполнение диагностической проверки

Можно проверить работоспособность устройства, запустив одну из следующих проверок.

    1. Быстрая: проверяет устройство на наличие основных проблем производительности и дает оценку прохождения или не прохождения проверки. Для выполнения быстрой проверки может понадобиться несколько минут.

    • Полная: проверяет каждый сектор устройства и предоставляет сведения о состоянии устройства My Cloud Mirror.  Для выполнения полной проверки может понадобиться несколько часов в зависимости от количества хранимых данных.

Описанные далее действия позволяют выполнить диагностическую проверку.

    1. Нажмите Настройки.
    2. На панели слева нажмите Служебные программы.
    3. В области «Диагностика системы» нажмите Быстрая проверка или Полная проверка.
      Индикатор показывает состояние проверки, а результаты отображаются в диалоговом окне.
