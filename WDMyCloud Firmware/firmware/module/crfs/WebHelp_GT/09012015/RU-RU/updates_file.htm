Обновление устройства My Cloud Mirror из файла

В некоторых случаях может потребоваться обновление устройства My Cloud Mirror по следующей схеме: сначала выполняется загрузка файла с сайта поддержки Western Digital, а затем его установка.

ПРИМЕЧАНИЕ. Для выполнения этой процедуры требуется доступ к Интернету.

Описанные далее действия позволяют обновить устройство My Cloud Mirror вручную из файла.

    1. Откройте новую вкладку или окно браузера.
    2. В новой вкладке или в новом окне откройте сайт технической поддержки Western Digital:
      http://support.wdc.com
    • Если доступно свежее обновление для микропрограммы, сохраните его в папке на вычислительном устройстве.
    1. Вернитесь в веб-интерфейс My Cloud Mirror.
    2. Нажмите Настройки.
    3. На панели слева нажмите Обновление микропрограммы.
    4. В разделе «Ручное обновление» нажмите Обновление из файла.
    5. Перейдите в папку с обновлением, сохраненным при выполнении шага 3.
    6. Выберите файл обновления и нажмите Открыть.
    7. Для подтверждения установки нажмите кнопку ОК. После установки обновления будет выполнена перезагрузка устройства My Cloud Mirror.
      ПРИМЕЧАНИЕ. При перезагрузке My Cloud Mirror предоставление услуг пользователям, которые используют доступ к облаку на момент перезагрузки, будет прервано.
