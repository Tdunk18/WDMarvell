Проверка состояния удаленного доступа

В любое время можно выполнить следующие проверки:

    • настроено ли устройство My Cloud Mirror для удаленного доступа;
    • как осуществляется удаленный доступ к устройству My Cloud Mirror: через прямое подключение или подключение с ретрансляцией;
    • может ли определенное мобильное устройство конкретного пользователя получить удаленный доступ к вашему облаку.

С помощью следующих действий можно проверить состояние мобильного доступа.

    1. Нажмите Доступ к облаку.
    2. Выберите имя пользователя на панели слева.
      Следующая таблица содержит поля мобильного доступа и их возможные значения.

Профиль доступа к облаку

Состояние подключения

  • Подключено (напрямую): мобильный доступ осуществляется посредством прямого подключения.

  • Подключено (ретрансляция): мобильный доступ осуществляется посредством подключения с ретрансляцией.

  • Сбой: My Cloud Mirror не может установить подключение для удаленного доступа.

Устройства облака для пользователя имя_пользователя

Название устройства или приложения

  • Подключено: устройство имеет удаленный доступ к вашему облаку.

  • Ожидание: ожидается активация устройства.

  • Истек срок действия кода: истек срок действия кода активации.

  • Отключено: параметр «Служба облака» отключен. См. Включение и отключение доступа к облаку.

  • Сбой подключения: не удалось установить удаленное подключение. Можно попробовать решить эту проблему, устранив неполадки сетевого подключения и убедившись в том, что установлена последняя версия микропрограммы.

 

ПРИМЕЧАНИЕ. Если нужно посмотреть настройки мобильного доступа для другого пользователя, выберите нужное имя пользователя в левом столбце. Представление обновится с учетом информации об устройствах облака этого пользователя.
