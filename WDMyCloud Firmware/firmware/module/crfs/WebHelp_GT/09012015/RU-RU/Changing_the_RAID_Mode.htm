Изменение режима RAID-массива

С помощью описанных далее действий можно изменить текущий режим RAID-массива на устройстве My Cloud Mirror.

Предупреждение! При изменении режима RAID-массива будут удалены все данные.

    1. Нажмите Хранилище.
    2. Нажмите RAID.
    3. Нажмите Изменить режим RAID в нижней части экрана.
    4. Выберите режим RAID, который следует использовать для сервера My Cloud Mirror. Доступны следующие варианты изменения режима:

Режим

Описание

JBOD

Использование одного или нескольких дисков не в конфигурации RAID-массива, а путем управления ими как отдельными логическими томами.

Объединение

Объединение дисков линейно для создания одного большого логического тома.

RAID 0

Данные записываются поочередно на несколько жестких дисков, позволяя быстрее считывать и записывать данные за счет объединения работы двух и большего количества дисков для повышения производительности.

RAID 1

Две одномоментные копии данных записываются на отдельных жестких дисках, чтобы в случае отказа одного из них другие продолжали бы чтение или запись данных до замены неисправного диска.

    1. Установите флажок «Перейти на <режим RAID>».
    2. Нажмите Далее.
    3. Проверьте состояние жестких дисков, после чего нажмите кнопку Далее.
    4. Нажмите кнопку «Готово».
    5. С помощью ползунка «Том» выберите объем пространства, который следует выделить выбранному режиму RAID-массива.

      ПРИМЕЧАНИЕ. Если вы решите не использовать весь доступный объем для выбранного режима RAID-массива, можно установить флажок Настроить оставшееся дисковое пространство для объединения и воспользоваться соответствующей возможностью.
    1. Нажмите Далее.
    2. Выберите , чтобы активировать параметр автоматического восстановления для конфигурации RAID-массива, и нажмите кнопку Далее.

      ПРИМЕЧАНИЕ. Этот экран не выводится для режимов JBOD, «Объединение» и RAID 0.
    1. Проверьте внесенные в RAID-массив изменения, убедитесь, что они верны, после чего нажмите Далее.
    2. Ознакомьтесь с экраном предупреждения.
      Предупреждение. При изменении режимов массива стираются все данные.
    1. Нажмите кнопку «Готово». Начинается внесение запрошенных изменений для жестких дисков.
    2. Нажмите кнопку «Готово». Теперь изменена конфигурация жестких дисков в соответствии с выбранным режимом.
