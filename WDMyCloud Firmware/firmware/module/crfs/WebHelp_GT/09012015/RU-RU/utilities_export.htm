Экспорт системных настроек

Как администратор вы можете сохранять текущие настройки My Cloud Mirror в файл конфигурации. Обычно это делается перед восстановлением заводских настроек по умолчанию для устройства My Cloud Mirror (подробнее). После сброса настроек устройства My Cloud Mirror в заводские значения по умолчанию можно импортировать сохраненные настройки (подробнее).

Описанные далее действия позволяют экспортировать системные настройки.

    1. Нажмите Настройки.
    2. На панели слева нажмите Служебные программы.
    3. В области «Конфигурация системы» нажмите Сохранить файл конфигурации.
