Настройки мультимедиа

В представлении «Мультимедиа» можно выполнять задачи, касающиеся сервера мультимедиа DLNA и проигрывателя iTunes.

Что такое DLNA?

 DLNA (Digital Living Network Alliance) — это стандарт, использующийся производителями бытовой электронной техники для обеспечения возможности обмена содержимым между бытовыми развлекательными устройствами в домашней сети. Например, устройство My Cloud Mirror можно настроить для воспроизведения и просмотра файлов на большом телевизоре при условии, что этот телевизор является DLNA-совместимым устройством.
