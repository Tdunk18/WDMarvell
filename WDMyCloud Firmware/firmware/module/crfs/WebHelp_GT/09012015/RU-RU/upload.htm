Выгрузка содержимого на устройство My Cloud Mirror

Для передачи на устройство My Cloud Mirror фотографий, видеороликов, музыки и других файлов следует использовать одну из перечисленных ниже последовательностей действий.

    • Mac
    • ПК под управлением ОС Windows
    • USB

Передача содержимого с компьютера Mac

Для того чтобы выгрузить файлы с компьютера Mac на устройство My Cloud Mirror, выполните следующие действия.

    1. На компьютере Mac в строке меню последовательно выберите Переход > Сеть.
    2. Выполните одно из следующих действий:
    • На левой панели окна Finder дважды нажмите на название устройства My Cloud Mirror.
    • На панели справа найдите название устройства My Cloud Mirror и дважды нажмите на него.
    1. Дважды щелкните название нужного ресурса, чтобы открыть папку и просмотреть существующие вложенные папки. (Public — это ресурс, созданный автоматически.)
    2. Перетащите файлы из своего компьютера в нужную папку.
      Не перетаскивайте файлы в папки резервного копирования Time Machine и WD SmartWare.
      ПРИМЕЧАНИЕ. При копировании с компьютера на устройство My Cloud Mirror файлы будут сохранены в двух местах: в исходной папке и на устройстве.

Передача содержимого с ПК

Описанные далее действия позволяют выгрузить файлы с компьютера под управлением ОС Windows на устройство My Cloud Mirror.

    1. На компьютере откройте диалоговое окно «Выполнить», воспользовавшись сочетанием клавиш Windows + R.
    2. В поле «Открыть» введите \\ и название устройства My Cloud Mirror. (По умолчанию используется название mycloudmirror без пробелов.) Пример. \\mycloudmirror
    3. Нажмите ОК.
      Отобразится окно Windows Explorer, в правой области которого будут перечислены доступные ресурсы.
    4. Дважды щелкните название нужного ресурса, чтобы открыть папку и просмотреть какие-либо вложенные папки. (Public — это ресурс, созданный автоматически.)
    5. Перетащите файлы из своего компьютера в этот ресурс.
      ПРИМЕЧАНИЕ. При копировании с компьютера на устройство My Cloud Mirror файлы будут сохранены в двух местах: в исходной папке и на устройстве.

     

    Выгрузка содержимого с USB-устройства

    Описанные далее действия позволяют выгрузить файлы с USB-устройства на устройство My Cloud Mirror.

    1. Подключите USB-накопитель к компьютеру для максимально эффективного выполнения операции.

    2. Подключите компьютер и устройство My Cloud Mirror непосредственно к маршрутизатору при использовании проводного подключения Ethernet.

    3. Скопируйте данные с внешнего USB-накопителя на устройство My Cloud Mirror.

ПРИМЕЧАНИЕ. При копировании с компьютера на устройство My Cloud Mirror файлы будут сохранены в двух местах: в исходной папке и на устройстве.
