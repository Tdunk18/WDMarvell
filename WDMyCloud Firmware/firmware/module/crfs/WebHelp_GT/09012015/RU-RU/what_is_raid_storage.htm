Понятие хранилища RAID

RAID-массив позволяет хранить одни и те же данные в разных местах на нескольких жестких дисках, благодаря чему обеспечиваются необходимая отказоустойчивость, более высокая производительность и целостность данных. Есть несколько разных уровней RAID-массивов, которые предполагают разные методы предоставления доступа к данным или их распределения по дискам. Устройство My Cloud Mirror позволяет выбрать нужный из описанных ниже режимов хранения.

    • JBOD.  Использование одного или нескольких дисков не в конфигурации RAID-массива, а путем управления ими как отдельными логическими томами.
    • Объединение. Объединение дисков линейным способом для создания одного большого логического тома.
    • RAID 0. В режиме RAID 0 обеспечивается чередование чтения и записи данных по всем дискам в группе RAID. RAID 0 не обеспечивает резервирование данных, но отличается максимальной производительностью по сравнению со всеми другими уровнями RAID. В режиме RAID 0 данные разбиваются на сегменты, которые поочередно записываются на все диски группы.
    • RAID 1. В режиме RAID 1 RAID-контроллер дублирует все данные с одного диска на второй в группе. RAID 1 обеспечивает полное резервирование данных, но при этом удваивает необходимый объем пространства для хранения данных.
