Добавление загрузки по HTTP в расписание

Приложение загрузок по HTTP позволяет добавить в расписание My Cloud Mirror загрузку по протоколу HTTP. Описанные далее действия позволяют добавить в расписание загрузку по HTTP.

    1. Нажмите Приложения.
    2. Нажмите Загрузки по HTTP.
    3. В поле «URL-адрес» введите URL-адрес файла, который нужно загрузить.
      Пример. http://192.168.0.32/test.txt
    4. В поле «Сохранить в» введите место, в котором нужно сохранить загрузку, или же нажмите Обзор, чтобы выбрать такое место.
      Пример. Том_1/Public/Shared Videos 
    5. В поле «Переименовать» введите новое имя файла, если его нужно изменить.
    6. Нажмите , если это операция повторяющегося резервного копирования.
    • Выберите частоту резервного копирования (ежедневно, еженедельно, ежемесячно).
    • Введите время, день и дату резервного копирования в зависимости от выбранной частоты.
    1. Введите дату, когда нужно выполнить загрузку (мм/дд/гггг), или нажмите , чтобы выбрать дату.
    2. Введите время загрузки в раскрывающемся меню.
    3. Нажмите Создать. Загрузка создается и добавляется в очередь загрузок HTTP.
