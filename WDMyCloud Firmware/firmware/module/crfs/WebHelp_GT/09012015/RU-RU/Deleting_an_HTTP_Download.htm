Удаление загрузки по HTTP

Описанные далее действия позволяют удалить запланированную загрузку по HTTP.

    1. Нажмите Приложения.
    2. Нажмите Загрузки по HTTP.
    3. В области «Очередь загрузок по HTTP» выберите загрузку, которую нужно удалить.
    4. Нажмите Удалить.
    5. Ознакомьтесь с экраном предупреждения и нажмите OK. Загрузка удаляется и исчезает из очереди загрузок по HTTP.
