Указание рабочей группы

Можно указать название рабочей группы на устройстве My Cloud Mirror.

Что такое рабочая группа?

Рабочая группа — это название, связанное с несколькими вычислительными устройствами в сети. Рабочие группы обеспечивают следующие возможности.

    • Упрощают процесс обмена информацией между вычислительными устройствами.

    • Нет требуют наличия центрального сервера.

    • Используются только в средах Microsoft Windows.

 

Описанные далее действия позволяют указать рабочую группу.

    1. Нажмите Настройки.
    2. На панели слева нажмите Сеть.
    3. В области «Сетевая рабочая группа» введите название рабочей группы, к которой нужно присоединиться.
