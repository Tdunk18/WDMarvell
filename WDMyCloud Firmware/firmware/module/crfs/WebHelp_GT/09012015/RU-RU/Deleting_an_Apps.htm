Удаление приложения

Описанные далее действия позволяют удалить приложение с устройства My Cloud Mirror.

    1. Нажмите Приложения.
    2. В области «Приложения» выберите приложение, которое нужно удалить.
    3. Нажмите значок «Удалить приложение».  
    4. Нажмите ОК. Приложение удаляется из области «Установленные приложения» и со страницы «Приложения».
