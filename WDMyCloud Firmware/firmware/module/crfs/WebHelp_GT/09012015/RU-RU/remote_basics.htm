Базовые сведения об удаленном доступе

Доступ к содержимому, хранящемуся в My Cloud Mirror, можно получить из любого места тремя способами::

    • с мобильного устройства, например смартфона или планшетного компьютера;  
    • с компьютера (вероятнее всего, с ноутбука);  

Общие сведения о доступе с мобильного устройства

Описанные далее действия позволяют получить доступ к облаку с помощью смартфона или планшетного компьютера.

1

Зарегистрируйтесь в My Cloud. Обычно это делается при добавлении пользователя устройства. См. Настройка доступа к облаку.

2

Загрузите и установите на мобильное устройство бесплатные приложения WD Photos и My Cloud.
Эти приложения доступны в магазинах Apple App Store и Google Play.

3

Выполните разовую настройку для безопасного доступа. См. Выполнение одноразовой настройки для получения безопасного доступа.

4

Получите доступ к содержимому с мобильного устройства. См. Доступ к содержимому с помощью смартфона или планшетного компьютера.

Общие сведения о доступе с компьютера с ОС Windows или компьютера Mac

Находясь вне дома (например, в кафе), вы можете без труда получить доступ к устройству с ноутбука. Для удаленного доступа нужны приложение My Cloud для компьютеров, ноутбук с модулем Wi-Fi и подключение к сети Wi-Fi.

Описанные далее действия позволяют получить доступ к облаку с помощью приложения My Cloud для настольных компьютеров.

1

Зарегистрируйтесь в My Cloud. Обычно это делается при добавлении пользователя устройства. См. Настройка доступа к облаку.

2

Из центра обучения WD (http://wd.com/setup/wdmycloudmirror) загрузите приложение My Cloud для настольных компьютеров и установите его на компьютер.

3

Выполните разовую настройку для безопасного доступа. См. Выполнение одноразовой настройки для получения безопасного доступа.

4

Получите доступ к содержимому с ноутбука. См. Доступ к содержимому с ноутбука.
