Добавление загрузки по FTP в расписание

Приложение загрузок по FTP позволяет добавить в расписание My Cloud Mirror загрузку по протоколу FTP. Описанные далее действия позволяют добавить в расписание загрузку по FTP.

    1. Нажмите Приложения.
    2. Нажмите Загрузки по FTP.
    3. Нажмите Создать задание.
    4. В поле «Название задания» введите название задания резервного копирования по FTP.
    5. В поле «Узел» укажите значение узла.
    6. В поле «Метод входа» выберите один из указанных далее методов, который нужно использовать для загрузки по FTP.
    • Учетная запись: введите имя пользователя и пароль для учетной записи.
    • Анонимное
    1. В поле «Тип» выберите тип информации, которую нужно загрузить.
    • Файл
    • Папка 
    1. В поле «Исходная папка» нажмите Обзор, чтобы выбрать исходную директорию. 
    2. В поле «Конечная папка» нажмите Обзор, чтобы выбрать конечную директорию.
    3. В поле «Язык» выберите в раскрывающемся меню язык, поддерживаемый FTP-клиентом.
    4. Нажмите , если это операция повторяющегося резервного копирования.
    • Выберите частоту резервного копирования (ежедневно, еженедельно, ежемесячно).
    • Введите время, день и дату резервного копирования в зависимости от выбранной частоты.
    1. Введите дату, когда нужно выполнить загрузку (мм/дд/гггг), или нажмите , чтобы выбрать дату.
    2. Введите время загрузки в раскрывающемся меню.
    3. Нажмите Создать. Загрузка создается и добавляется в очередь загрузок по FTP.
