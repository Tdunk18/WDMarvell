Просмотр сведений о пользователях

Описанные далее действия позволяют просмотреть информацию о выбранной учетной записи пользователя. В том числе указываются предоставленные права доступа.

    1. Нажмите Пользователи.
    2. На панели слева выберите пользователя. На панели справа отображается описанная далее информация.

Раздел

Значения и описания

Профиль пользователя

В этой области отображается базовая информация о выбранном пользователя, в том числе следующая.

  • Имя пользователя — это имя, используемое для входа в систему устройства.

  • Имя — имя пользователя.

  • Фамилия — фамилия пользователя.

  • Адрес электронной почты — адрес электронной почты пользователя. Он также используется для доступа к My Cloud.

  • Пароль: позволяет включить или отключить пароль для пользователя. См. Добавление и изменение пароля.

  • Членство в группе: показывает группы, к которым относится пользователь. Здесь можно изменить членство пользователя в группах. См. «Изменение членства в группах».

  • Квоты: здесь можно просмотреть и настроить квоты для пользователя. См. «Настройка квот для пользователей».

Доступ к ресурсу

Здесь отображаются доступные ресурсы и тип доступа пользователя к этим ресурсам. Изменить доступ пользователя к ресурсу можно, нажав один из указанных далее вариантов.

  •  Доступ для чтения и записи: возможность просматривать и обновлять ресурс.

  •  Доступ только для чтения: возможность просматривать ресурс, но не обновлять его.

  •  запретить доступ. у пользователя нет доступа к этому ресурсу.
