Проверка наличия обновлений и их установка вручную

Если для устройства My Cloud Mirror не настроено автоматическое обновление микропрограммы (см. раздел Автоматическое обновление My Cloud Mirror), можно выполнить проверку наличия обновлений и установить их вручную.

Описанные далее действия позволяют проверить наличие обновлений и установить их.

    1. Нажмите Настройки.
    2. На панели слева нажмите Обновление микропрограммы.
    3. В разделе «Доступные обновления» нажмите Проверить наличие обновлений.
      На данном этапе устройство My Cloud Mirror проверяет наличие доступных обновлений. Если отображается сообщение о состоянии «Новые обновления отсутствуют», данная процедура завершена.
      Если отображается список доступных обновлений, перейдите к шагу 4.
    4. Выберите пункт «Замечания о выпуске», чтобы просмотреть подробные сведения о доступном обновлении в новом окне браузера. Просмотрев замечания о выпуске, закройте окно браузера.
    5. Нажмите Установить обновление.
    6. Нажмите кнопку OK, чтобы подтвердить перезагрузку устройства My Cloud Mirror.
      ПРИМЕЧАНИЕ. Помните, что для любого пользователя, который в данным момент пытается получить доступ к облаку, служба будет недоступна.
