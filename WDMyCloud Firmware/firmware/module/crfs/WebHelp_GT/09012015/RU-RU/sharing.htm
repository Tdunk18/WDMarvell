Предоставление общего доступа к содержимому другим пользователям вне сети

Для предоставления общего доступа к содержимому другим пользователям вне сети запустите на своем компьютере приложение My Cloud для настольных компьютеров.

Описанные далее действия позволяют предоставить общий доступ к содержимому.

    1. Если это еще не сделано, из центра обучения WD (http://www.wd.com/setup/wdmycloudmirror) загрузите приложение My Cloud для компьютеров и установите его на компьютер Mac или ПК.
    2. На компьютере запустите приложение My Cloud для настольных компьютеров.
    3. Дважды нажмите устройство My Cloud Mirror.
    4. При необходимости в левой области можно разворачивать папки для отображения содержимого, к которому вы хотите предоставить общий доступ.
    5. В правой области щелкните один нужный элемент (или выберите мышью несколько элементов, удерживая клавишу SHIFT).
    6. Правой кнопкой мыши щелкните выбранные элементы и выберите пункт «Отправить по электронной почте».
    7. Заполните поля в открывшемся окне так, как нужно, и отправьте сообщение.
