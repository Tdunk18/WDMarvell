リモート バックアップの作成

リモート バックアップでは、My Cloud Mirror デバイスを別のサーバーまたはネットワーク外にバックアップできます。次の手順に従い、My Cloud Mirror サーバーのリモート バックアップを作成します。

    1. [バックアップ] をクリックします。
    2. [リモート バックアップ] をクリックします。
    3. [ジョブを作成] ボタンをクリックします。
    4. [ジョブ名] を入力します。
    5. [リモート サーバー] ドロップダウンからサーバーを選択します。
    6.  [リモート IP] フィールドで、リモート バックアップ デバイスの IP アドレスを入力します。
      (例 192.168.1.16)
    7. [パスワード] フィールドで、リモート バックアップ デバイスのパスワードを入力します。
    8. SSH ユーザー名を入力します。
    9. [SSH パスワード] フィールドに、SSH パスワードを入力します。
    10. [参照] をクリックして、ソース フォルダを選択してください。
    11. [リモート バックアップ] フォルダのリストから、保護したい共有ディレクトリを一つ以上チェックします。
    12. [OK] をクリックします。
    13. [リモート バックアップ] ディレクトリを選択するには、[参照] をクリックします。
    14. ディレクトリを選択し、[OK] をクリックします。
    15. ドロップダウン メニューを使って [バックアップ タイプ] を選択します。
    16. バックアップのスケジュールを設定するには、[繰り返し] の隣にある トグル アイコンをクリックします。
    17. [転送暗号化] を設定するには、 トグル アイコンをクリックします。
    18. [作成] をクリックします。
