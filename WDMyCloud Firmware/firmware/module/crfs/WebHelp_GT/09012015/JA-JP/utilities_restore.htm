デフォルト設定の復元

システム復元では、管理者が My Cloud Mirror で完全な初期状態への復元を実行できます。このオプションはデバイスを初期設定に復元します。データは失われません。

復元を実行する前に、現在の設定を保存してエクスポートし (操作手順)、後でインポートできます (操作手順)。

重要：復元を行うと、現在クラウドにアクセスしているユーザーに対するサービスは中断されます。復元の進行中は My Cloud Mirror デバイスを切断しないでください。

次の手順に従い、My Cloud Mirror デバイスを復元します。

    1. [設定] をクリックします。
    2. 左側のペインで、[ユーティリティ] をクリックします。
    3. [システム復元] 領域で、[システム復元] をクリックします。
    4. 設定メッセージを確認し、[OK] をクリックします。デバイスは再起動します。再起動処理中にデバイスを切断しないでください。再起動が完了したら、ダッシュボードを起動します。
