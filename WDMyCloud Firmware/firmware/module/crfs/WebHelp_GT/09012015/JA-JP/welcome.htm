My Cloud Mirror デバイスは高パフォーマンスのシステムであり、ワイヤレス ルーターに接続するときに、安全なワイヤレス アクセスによって、あらゆるコンピュータ、タブレット、またはスマートフォンからメディアおよびファイルにアクセスできます。My Cloud Mirror デバイスはパーソナル クラウドの中心となるコンポーネントであり、セットアップ、共有、メンテナンスが簡単にできる簡素で堅牢なインターフェイスを備えています。

知っておく必要のあること

パーソナル クラウドを最大限に活用するには、My Cloud Mirror デバイスについていくつかの重要なことを理解しておくことが必要です。

    • ユーザーの役割

さらに、デバイスの中核機能を活用できるように、以下のタスクを確認しておく必要があります。

    • スマートフォンまたはタブレット アクセス

    • ラップトップ アクセス

    • My Cloud Mirror の Web インターフェイスの表示

    • ネットワーク外部の他のユーザーとのコンテンツの共有

    • ファイルのアップロード

    • 使用開始 (基本セットアップ)

パーソナル クラウドのユーザーの役割

パーソナル クラウドにアクセスするには、My Cloud Mirror ドライブのユーザー アカウントが必要です。My Cloud Mirror 所有者には、管理者特権を持つ特別なユーザー アカウント (admin) が与えられます。管理者特権があると、特定のニーズに合わせてデバイスをセットアップおよび構成し、他のユーザーをパーソナル クラウドに追加できます。デバイスにアクセスできるユーザーを厳密に指定できます。

注:デフォルトでは、admin ユーザー名にはパスワードはありません。いつでもパスワードを追加できます。

重要：My Cloud Mirror セットアップ ソフトウェアまたは開始画面の実行時に、管理者名を変更し、オプションのパスワードを設定できます。名前とパスワードは常に把握していてください。これらの資格情報を入力しないと、My Cloud Mirror Web インターフェイスを表示できません。

スマートフォンまたはタブレットからのコンテンツへのアクセス

次の 2 つの WD モバイル アプリでは、スマートフォンまたはタブレットから My Cloud デバイスのコンテンツにアクセスできます。

    • My Cloud アルバム：写真を表示して管理します。

    • My Cloud：写真、ビデオ、音楽、およびファイルを表示および再生します。

これらの無料アプリは WD ラーニング センター (http://www.wd.com/setup/wdmycloudmirror) や、Apple App Store、Google Play、Windows ストア、および Amazon.com などの他のアプリ サイトから入手することができます。

3

WD アプリを起動し、安全なアクセス用に 1 回限りのセットアップを実行します。操作手順

4

Voila!コンテンツを再生および表示します。

モバイル デバイス アクセスのワークフロー (他のユーザー向け)

注 : ユーザーを My Cloud Mirror に追加するときには、そのユーザーのメール アドレスを My Cloud に入力し、My Cloud ログインを作成します。ログイン情報が作成されると、新しいユーザーのメール アドレスにメールが送信されます。このメールには、登録処理を続行するための My Cloud へのリンクが記載されています。

4

コンテンツを再生および表示します。

ラップトップからのコンテンツへのアクセス

Mac または PC から My Cloud Mirrorデバイスのコンテンツにアクセスするには、My Cloud デスクトップ アプリケーションを使用します。My Cloud を使用すると、ホーム ネットワークに接続された My Cloud Mirror や My Book Live ドライブなどの WD NAS デバイスに保存されている写真、ビデオ、音楽、ファイルにリモート アクセスできます。WD モバイル アプリと同様に、コンテンツにアクセスするにはすべての重要な My Cloud ログインが必要です。

My Cloud デスクトップ アプリケーションは、http://www.wd.com/setup/wdmycloudmirror からダウンロードしてください。

詳細な手順については、「ラップトップからのコンテンツへのアクセス」を参照してください。

My Cloud Mirror の Web インターフェイスの表示

My Cloud Mirror デバイスのユーザー インターフェイスでは、個人向けクラウドをセットアップして管理できます。このインターフェイスは、Mac または PC で実行する Web ブラウザに表示されます。 

Web インターフェイスを表示するには:

    1. ブラウザ ウィンドウを開きます。
    2. アドレス バーに、http:// と入力します。
      例:http://wdmycloudmirror
      注:この URL で Web インターフェイスが表示されない場合は、次の URL を入力してみてください。http://wdmycloudmirror.local
