メディアの設定について

[メディア] ビューからは、DLNA メディア サーバーと iTunes メディア プレイヤーに関するタスクを実行できます。

DLNA とは

 DLNA (Digital Living Network Alliance) は、家庭用電化製品メーカーによって使用される標準であり、家庭内のエンターテイメント デバイスがホーム ネットワーク全体で互いにコンテンツを共有できるようにします。例えば、お使いのテレビが DLNA 証明済みデバイスであれば、大画面テレビでファイルを再生や表示できるように My Cloud Mirror を設定できます。

「My Cloud Mirror と DLNA について」を参照してください。

 

 

DLNA メディア サーバー

  • DLNA メディア サーバーのオン/オフ  
  • DLNA デバイス用アクセスのブロック/ブロックの解除  
  • DLNA サーバー データベースの再スキャン/復元  

iTunes サーバー

  • iTunes サーバーのオン/オフ  
  • iTunes データベースの再スキャン
