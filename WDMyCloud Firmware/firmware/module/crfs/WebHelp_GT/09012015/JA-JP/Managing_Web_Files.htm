Web ファイルの管理

Web File View は、My Cloud Mirror デバイスでファイルを表示して管理するためのアプリです。

次の手順に従い、Web ファイルを管理します。

    1. [アプリ] をクリックします。
    2. [Web File Viewer] をクリックします。デバイスで使用可能なファイル ボリュームは、右側のペインに表示されます。

Web ファイルの表示

ボリューム、フォルダ、およびファイルが Web File Viewer の右側のペインに表示されます。任意のボリュームまたはフォルダのコンテンツを表示するには、フォルダをダブルクリックします。

Web ファイルの管理

ファイルとフォルダに対して実行可能な処理は、 Web File Viewer の右下に表示されます。次の中から、Web ファイルに対して実行する処理を選択します。

コピー

選択したファイルを別の場所にコピーできます。次の手順に従い、ファイルを新しい場所にコピーします。

    1. コピーするファイルを選択します。
    2. をクリックします。
    3. ファイルのコピー先を選択します。
    4. [OK] をクリックします。ファイルが選択した場所にコピーされます。

移動

ファイルをある場所から別の場所に移動できます。次の手順に従い、ファイルを新しい場所に移動します。

    1. 移動するファイルを選択します。
    2. をクリックします。
    3. ファイルの移動先を選択します。
    4. [OK] をクリックします。ファイルが選択した場所に移動されます。

アップロード

コンピュータまたは他のデバイスからファイルを My Cloud Mirror サーバーにアップロードできます。
次の手順に従い、ファイルをデバイスにアップロードします。

    1. Web File Viewer の右側のペインで、ファイルのアップロード先を選択します。
    2. をクリックします。
    3. [アップロード] 画面で、[ファイルの選択] アイコンをクリックします。
    4. デバイスにアップロードするファイルに移動して選択します。
    5. [開く] をクリックします。
    6. [アップロード] をクリックします。
    • アップロードするファイルを追加する場合は、すべてのファイルがアップロードされるまで、手順 3 と 4 を繰り返します。
    1. [閉じる] をクリックします。アップロードされたファイルがアップロード先のフォルダに表示されます。

ダウンロード

ファイルを My Cloud Mirror デバイスからコンピュータまたは他のデバイスにダウンロードできます。
次の手順に従い、ファイルをコンピュータまたは他のデバイスにダウンロードします。

    1. Web File Viewer の右側のペインで、ダウンロードするファイルを選択します。
    2. をクリックします。
    3. 選択したファイルがコンピュータまたは他のデバイスにダウンロードされます。

名前の変更

My Cloud Mirror デバイスの選択されたファイルの名前を変更できます。次の手順に従い、ファイル名を変更します。

    1. Web File Viewer の右側のペインで、名前を変更するファイルを選択します。
    2. をクリックします。
    3. [名前の変更] フィールドで、ファイル名を変更します。
    4. [保存] をクリックします。ファイル名が変更されます。

削除

選択されたファイルが My Cloud Mirror デバイスから削除されます。次の手順に従い、ファイルを削除します。

    1. Web File Viewer の右側のペインで、削除するファイルを選択します。
    2. をクリックします。
    3. [OK] をクリックし、選択したファイルを削除することを確認します。選択したファイルがデバイスから削除されます。
