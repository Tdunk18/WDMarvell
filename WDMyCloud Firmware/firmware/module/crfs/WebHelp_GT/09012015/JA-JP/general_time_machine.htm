Apple Time Machine によるデータのバックアップ

My Cloud Mirror デバイスのファイルを Mac コンピュータの Apple Time Machine にバックアップできます。

次の手順に従い、データを Apple Time Machine にバックアップします。

    1. [設定] をクリックします。
    2. 左側のペインで、[全般] をクリックします。
    3. [MAC バックアップ] 領域で をクリックし、Time Machine バックアップ オプションをオンにします。
    4. [設定] をクリックします。
    5. [共有を選択] ドロップダウン リストで、バックアップ ファイルの保存先の共有を選択します。
    6. [保存] をクリックします。
