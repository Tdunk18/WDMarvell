サポートの要求

My Cloud Mirror デバイスで問題が発生した場合、WD テクニカル サポートがトラブルシューティングを行って最善の解決策を決定するには、My Cloud Mirror デバイスについての情報が必要です。デバイスの情報を取得してサポートを要求するには、2 つの方法があります。

    • システム レポートを実行し、それを WD に自動的に送信します。

    • システム レポートを手動で作成し、それを WD に送信します。

システム レポートを実行して WD に自動的に送信するには:

    1. 右上隅の をクリックします。
    2. [サポート] をクリックします。
    3. [自動カスタマー サポート] 領域で、[プライバシー ポリシー] リンクをクリックして、WD のプライバシー ポリシーを確認します。
    4. [サポートを要求] をクリックします。
    5. 以前に行っていない場合は、WD Knowledge Base (http://support.wdc.com) にサインインします。
    6. 要求を作成して送信します。

システム レポートを手動で作成して WD に送信するには:

    1. 右上隅の をクリックします。
    2. [サポート] をクリックします。
    3. [システム レポートを作成して保存] 領域で、[作成して保存] をクリックします。
    4. レポートをコンピュータに保存します。
    5. レポートを電子メールで WD テクニカル サポートに送ります。
