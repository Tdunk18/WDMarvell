クラウド アクセスを設定中

スマートフォン、タブレット、コンピュータなどからクラウドにリモートでアクセスするには、My Cloud ログインが必要です。(基本的に、コンピュータ デバイスからのリモート アクセスは、ホーム ネットワークに接続された NAS デバイスにアクセスできるようにする My Cloud 中央サーバーによって有効になります)。ただし、モバイル デバイス (スマートフォンやタブレット) とコンピュータでは、クラウドにリモート アクセスする方法が若干異なります。いずれにしても、各ユーザーには My Cloud ログインが必要です。

次の手順に従い、リモート アクセスを設定します。

    1. クラウド アクセスがデバイスで有効なことを確認します。   手順についてはこちらをご覧ください。

       

      クラウド アクセスを有効にするには:

      1. [設定] をクリックします。
      2. 左側のペインで、[全般] をクリックします。
      3. [クラウド アクセス] 領域で をクリックし、クラウド サービスを有効にします。

       

    2. [クラウド アクセス] をクリックします。
    3. 左側のウィンドウで、クラウド アクセスを許可するユーザーをクリックします。
    4. <ユーザー名> セクションの My Cloud ログインでは、まだ追加されていない場合は電子メールアドレスを入力し、をクリックします。これによって、My Cloud の登録方法が記載された電子メールがユーザーに送信されます。
    1. 次のいずれかを行います。
    • 自分で登録処理を続けます。手順についてはこちらをご覧ください。

       

      登録処理を続けるには:

      1. 手順 4 で指定した電子メール アカウントを調べて、電子メール通知を確認します。
      2. 電子メールに記載されているリンクをクリックし、指示に従って登録を完了します。

       

    • セットアップした Web アカウントのユーザーに通知し、続きの登録処理はユーザーに実行してもらいます。手順についてはこちらをご覧ください。
      注:My Cloud Mirror ドライブにモバイル デバイスがリモート アクセスできるようにする方法は他にもあります。「モバイル デバイスへのリモート アクセスの許可 (代替方法)」を参照してください。

       

      ユーザーに連絡し、ユーザーのためにリモート アクセス コンテンツをセットアップしたので、次の手順に従って登録処理を続ける必要があることを伝えます。

      1. 電子メール アカウントを確認します (手順 4 で指定した電子メール アドレス)。
      2. My Cloud からの電子メール通知を開きます。
      3. 電子メールに記載されているリンクをクリックし、表示される指示に従って登録を完了します。
