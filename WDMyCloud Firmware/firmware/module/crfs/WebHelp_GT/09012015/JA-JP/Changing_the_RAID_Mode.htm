RAID モードの変更

次の手順に従い、My Cloud Mirror デバイスの現在の RAID モードを変更します。

警告：RAID モードの変更により、すべてのデータが削除されます。

    1. [ストレージ] をクリックします。
    2. [RAID] をクリックします。
    3. 画面の下部にある [RAID モードの変更] をクリックします。
    4. My Cloud Mirror サーバーで使用する RAID モードを選択します。オプション：

モード

説明

JBOD

RAID 設定ではない 1 台以上のドライブを使用しますが、個別の論理ボリュームとして管理されます。

スパニング

ドライブを直線的に組み合わせ、1 つの大規模な論理ボリュームを作成します。

RAID 0

複数のハード ドライブ間でデータをストライプ化します。2 台以上のドライブの処理を組み合わせ、パフォーマンスを高めることで、データの読み書きを高速化します。

RAID 1

データの 2 つの即時コピーが別のハード ドライブに記録されるため、いずれかのドライブが故障した場合、障害のあるハード ドライブが交換されるまで、他のドライブがデータの読み書きを継続できます。

    1. [<RAID モード> に切り替える] チェックボックスをオンにします。
    2. [次へ] をクリックします。
    3. ハード ドライブのステータスを確認し、[次へ] をクリックします。
    4. [完了] をクリックします。
    5. [ボリューム] スライダを使用し、選択した RAID モード専用にする領域のサイズを指定します。

      注:RAID でボリューム全体を使用するように選択しない場合は、[残りディスク領域をスパニングとして構成する] チェック ボックスをオンにすると、残りのディスク領域をスパニングとして構成できます。
    1. [次へ] をクリックします。
    2. を選択し、RAID 構成の [自動復元] オプションをオンにし、[次へ] をクリックします。

      注:JBOD、スパニング、および RAID 0 モードの場合は、この画面が表示されません。
    1. RAID の変更を確認し、変更が正しいことを検証してから、[次へ] をクリックします。
    2. 警告画面を確認します。
      警告：モードを変更すると、すべてのデータが消去されます。
    1. [完了] をクリックします。要求されたハード ドライブの変更が開始されます。
    2. [完了] をクリックします。ハード ドライブが選択したモードに再構成されます。
