HTTP ダウンロードのスケジュール

HTTP ダウンロード アプリでは、My Cloud Mirror デバイスへの HTTP ダウンロードをスケジュールできます。次の手順に従い、HTTP ダウンロードをスケジュールします。

    1. [アプリ] をクリックします。
    2. [HTTP ダウンロード] をクリックします。
    3. [URL] フィールドで、ダウンロードするファイルの URL を選択します。
      例: http://192.168.0.32/test.txt
    4. [保存先] フィールドでダウンロードを保存する場所を入力するか、[参照] をクリックしてダウンロードを保存する場所を選択します。
      例：Volume_1/Public/Shared Videos 
    5. ファイル名を変更する場合は、[名前の変更] フィールドで新しいファイル名を入力します。
    6. 繰り返しバックアップの場合は、 をクリックします。
    • バックアップの頻度を選択します (毎日、毎週、毎月)。
    • 選択した頻度に応じて、バックアップの時刻、曜日、日付を入力します。
    1. ダウンロードを実行する日付 (mm/dd/yyyy) を入力するか、 をクリックして日付を選択します。
    2. プルダウン メニューからダウンロード時刻を選択します。
    3. [作成] をクリックします。ダウンロードが作成され、[HTTP ダウンロード キュー] に追加されます。
