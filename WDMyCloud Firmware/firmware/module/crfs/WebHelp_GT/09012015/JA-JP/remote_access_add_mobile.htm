モバイル デバイスの追加

MyCloud Albums と My Cloud アプリを使用すると、スマートフォンやタブレットと同様に、モバイル デバイスからもクラウド コンテンツにアクセスできます。この手順を開始する前に、これらの無料アプリをモバイル デバイスにダウンロードしてください。

モバイル デバイスを追加するには:

  1. まだ行っていない場合は、クラウドですべてのユーザーのリモート アクセスを有効にします。
  2. [クラウド アクセス] アイコンをクリックします。
  1. 左側のペインで、クラウドへのアクセスを許可するモバイル デバイスのユーザー名をクリックします。
  1. [コードの取得] をクリックします。My Cloud Mirror に、24 時間以内にモバイル デバイスに入力しなければならないコードが表示されます。コードを忘れないように、書き留めます。
  1. [完了] をクリックします。
  2. モバイル デバイスで、認証コードを入力します。
    デバイスが認証されたら、My Cloud アプリを使用して、クラウド コンテンツにアクセスします。
