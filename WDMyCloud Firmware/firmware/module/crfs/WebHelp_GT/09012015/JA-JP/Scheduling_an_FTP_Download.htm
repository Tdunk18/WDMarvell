FTP ダウンロードのスケジュール

FTP ダウンロード アプリでは、My Cloud Mirror への FTP ダウンロードをスケジュールできます。次の手順に従い、FTP ダウンロードをスケジュールします。

    1. [アプリ] をクリックします。
    2. [FTP ダウンロード] をクリックします。
    3. [ジョブ作成] をクリックします。
    4. [ジョブ名] フィールドで、FTP バックアップ ジョブ名を入力します。
    5. [ホスト] フィールドのホストの値を入力します。
    6. [ログイン方法] フィールドで、FTP ダウンロードで使用する方法を選択します。
    • アカウント:アカウントのユーザー名とパスワードを入力します。
    • 匿名
    1. [タイプ] フィールドで、ダウンロードする情報のタイプを選択します。
    • ファイル
    • フォルダ 
    1. [ソース フォルダ] フィールドで [参照] をクリックしてソース ディレクトリを選択します。 
    2. [ダウンロード先フォルダ] フィールドで [参照] をクリックしてダウンロード先ディレクトリを選択します。
    3. [言語] フィールドのプルダウン メニューから、FTP クライアントでサポートされる言語を選択します。
    4. 繰り返しバックアップの場合は、 をクリックします。
    • バックアップの頻度を選択します (毎日、毎週、毎月)。
    • 選択した頻度に応じて、バックアップの時刻、曜日、日付を入力します。
    1. ダウンロードを実行する日付 (mm/dd/yyyy) を入力するか、 をクリックして日付を選択します。
    2. プルダウン メニューからダウンロード時刻を選択します。
    3. [作成] をクリックします。ダウンロードが作成され、[FTP ダウンロード キュー] に追加されます。
