手動での更新の確認とインストール

My Cloud Mirror がファームウェアを自動的に更新する (「My Cloud Mirror の自動更新」を参照) ように設定されていない場合、手動で更新を確認してインストールできます。

次の手順に従い、更新を確認してインストールします。

    1. [設定] をクリックします。
    2. 左側のペインで、[ファームウェア更新] をクリックします。
    3. [利用可能な更新] セクションで、[更新の確認] をクリックします。
      この時点で、My Cloud Mirror デバイスが、利用できる更新があるかどうかが確認します。"新しい更新はない" というステータス メッセージが返る場合、この手順は完了しています。
      返されたステータスに、利用可能な更新が表示される場合、手順 4 に進みます。
    4. 新しいブラウザ ウィンドウで利用可能な更新についての詳細を表示するには、[リリース ノート] をクリックします。リリース ノートを確認した後、ブラウザ ウィンドウを閉じます。
    5. [更新のインストール] をクリックします。
    6. [OK] をクリックして My Cloud Mirror デバイスの再起動を確認します。
      注:現在クラウドにアクセスしているユーザーに対するサービスが中断されることに注意してください。
