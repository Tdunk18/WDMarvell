My Cloud Mirror デバイスの自動更新

利用可能なファームウェア更新を適時チェックして自動的にインストールするように、My Cloud Mirror デバイスをセットアップできます。これは、My Cloud Mirror ファームウェアを最も簡単に最新の状態に維持できる方法です。

注:ソフトウェア更新が自動的にインストールされたら、My Cloud Mirror デバイスが再起動します。再起動はユーザーがクラウドにアクセスできるかどうかに影響するので、クラウドへのユーザー アクセスが最も少なくなるときに更新が行われるようにスケジュールする必要があります。

次の手順に従い、My Cloud Mirror デバイスを自動的に更新します。

    1. [設定] をクリックします。
    2. 左側のペインで、[ファームウェア更新] をクリックします。
    3. [自動更新] 領域で をクリックし、自動更新を有効にします。
    4. [更新スケジュール] フィールドで、更新の日時を指定します。

    • 1 番目のドロップダウン矢印をクリックし、曜日を選択します。

    • 2 番目のドロップダウン矢印をクリックし、時刻を選択します。

    • 3 番目のドロップダウン矢印をクリックし、[午前] または [午後] を選択します。
