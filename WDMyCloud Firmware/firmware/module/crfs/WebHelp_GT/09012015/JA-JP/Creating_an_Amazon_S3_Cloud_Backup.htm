Amazon S3 クラウド バックアップの作成

注記：クラウド バックアップを作成する前に、http://aws.amazon.com/s3 から Amazon S3 サービスに登録します。

Amazon S3 アカウントが設定されたら、次の手順に従い、クラウド バックアップを作成します。

    1. [バックアップ] をクリックします。
    2. [クラウド バックアップ] をクリックします。
    3. 画面の上部にある [Amazon S3] ボタンをクリックします。
    4. [作成] をクリックします。
    5. クラウド バックアップのジョブ名を入力します。
    6. [次へ] をクリックします。
    7. Amazon S3 から提供された次の情報を入力します。
    • 地域：ドロップダウン メニューからクラウドで使用される [地域] を選択します。
    • アクセス キー：Amazon S3 から提供されたアクセス キーを入力します。
    • プライベート キー：Amazon S3 から提供されたプライベート キーを入力します。
    • リモート パス：クラウドのリモート パスを入力します。通常はバケット名です。
    1. [次へ] をクリックします。
    2. [タイプ] フィールドで、次のオプションのいずれかを選択します。
    • アップロード：My Cloud Mirror データを Amazon S3 バケットにバックアップします。
    • ダウンロード：Amazon S3 バケットを My Cloud Mirror サーバーにバックアップします。
    1. [バックアップ タイプ] フィールドのドロップダウン メニューから、実行するバックアップのタイプを選択します。オプション：
    • 既存のファイルを上書きする：バックアップ元ファイルと同じ名前のファイルがバックアップ先フォルダにある場合は上書きします。
    • 完全バックアップ：バックアップを実行するたびに、すべてのバックアップ データが格納される個別のフォルダを作成します。
    • 増分バックアップ：バックアップ元ファイルがバックアップ先ファイルよりも新しい場合にファイルを上書きします。
    1. [次へ] をクリックします。
    2. [ローカル パス] フィールドで、My Cloud Mirror デバイスのバックアップ パスを入力します。
    • [参照] をクリックし、デバイスのバックアップ場所を参照します。
      (例 Volume_1/backup)
    1. をクリックし、自動更新機能を有効にします。作成したスケジュールに基づいて、自動的にバックアップが更新されます。
    • 自動更新機能を有効にする場合:

      • 次の中から自動更新スケジュールを選択します。毎日、毎週、毎月

      • ドロップダウン メニューから [自動更新時刻] を選択します ([毎日] オプション)。

      • ドロップダウン メニューから [自動更新日時] を選択します ([毎週] または [毎月] オプション)。

    1. [次へ] をクリックします。
    2. 設定を確認し、[完了] をクリックします。Amazon S3 クラウド バックアップが作成されます。

注:Amazon S3 クラウド バックアップ バケットにアクセスし、デバイス バックアップを確認します。
