安全なアクセスのために 1 回限りのセットアップを実行する

この手順ではスマートフォンまたはタブレットと My Cloud Mirror デバイスの間で安全な接続を確立する方法を説明します。以下の特定のモバイル デバイスに応じた手順を使用してください。

    • Android スマートフォンまたはタブレット  

    • iPhone または iPad

    • Windows Phone またはタブレット

    • Windows または Mac デスクトップ

Android スマートフォンまたはタブレット用に安全なアクセスを取得

    1. 以下を実行してスマートフォンまたはタブレットを接続します。

      1. モバイル デバイスで [設定] をタップします。

      2. [ワイヤレスとネットワーク] をタップします。

      3. [Wi-Fi 設定] をタップします。

      4. ホームネットワークを選択します。

      5. ネットワーク パスワードを入力して [接続] をタップします。

    1. WD アプリを起動します。
    2. [今すぐ接続] をタップします。
    3. My Cloud Mirror デバイスの名前をタップします。
    4. ドロップダウンの矢印をタップして、ユーザー名を選択します。
    5. 次のいずれかを行います。

    • パスワードがユーザー用に設定されている場合、そのパスワードを入力して [ログイン] をタップします。

    • パスワードがユーザー用に設定されていない場合、[ログイン] をタップします。

    1. (任意) ご自分のコンテンツにアクセスする場合、共有名をタップして目的のファイルに移動します。

iPhone または iPad 用に安全なアクセスを取得

    1. 以下を実行してスマートフォンまたはタブレットを接続します。

    1. モバイル デバイスで [設定] をタップします。

    2. [Wi-Fi] をタップします。

    3. [Wi-Fi 設定] をタップします。

    4. ホームネットワークを選択します。

    5. ネットワーク パスワードを入力して [参加] をタップします。

    1. WD アプリを起動します。
    2. [今すぐ接続] をタップします。
    3. My Cloud Mirror デバイスの名前の隣にあるプラス記号 (+) をタップします。
    4. 目的のユーザー名をタップします。
      パスワードがユーザー用に設定されている場合、そのパスワードを入力します。
    5. (任意) ご自分のコンテンツにアクセスする場合、共有名をタップして目的のファイルに移動します。

Windows スマートフォンまたはタブレット用に安全なアクセスを取得

    1. 以下を実行してスマートフォンまたはタブレットを接続します。

    1. コンピュータ デバイスで [設定] をタップします。

    1. [Wi-Fi] をタップします。

    2. ホームネットワークを選択します。

    3. ネットワーク パスワードを入力して [完了] をタップします。

    1. WD アプリを起動します。
    2. [今すぐ接続] をタップします。
    3. My Cloud Mirror ドライブの名前をタップします。
    4. ドロップダウンの矢印をタップして、ユーザー名を選択します。
      パスワードがユーザー用に設定されている場合、そのパスワードを入力します。
    5. (オプション) ご自分のコンテンツにアクセスする場合、以下を実行します。

    1. デバイス画面で My Cloud Mirror ドライブの名前をタップします。

    2. 共有をタップし、目的のコンテンツに移動します。
