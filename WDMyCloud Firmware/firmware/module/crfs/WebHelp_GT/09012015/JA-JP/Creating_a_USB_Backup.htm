USB バックアップの作成

次の手順に従い、My Cloud Mirror のデータを USB デバイスとの間でバックアップします。

    1. [バックアップ] をクリックします。
    2. [USB バックアップ] をクリックします。
    3. [ジョブを作成] をクリックします。
    4. バックアップのジョブ名を入力します。
    5. プルダウン メニューからバックアップのディレクトリを指定します。オプション:
    • USB から NAS:USB デバイスのデータを My Cloud Mirror デバイスにバックアップします。
    • NAS から USB:My Cloud Mirror デバイスのデータを USB デバイスにバックアップします。
    1. バックアップ元フォルダを選択します。
    • [参照] をクリックし、バックアップ先デバイスへバックアップするフォルダを選択します。
    1. バックアップ先フォルダを選択します。
    • [参照] をクリックし、デバイスのバックアップを格納するフォルダを選択します。
    1. プルダウン メニューを使って [バックアップ タイプ] を選択します。
    2. 接続したときに自動的に USB バックアップを開始するには、 トグル アイコンをクリックします。
    3. [作成] をクリックします。要求されたバックアップのバックアップ ジョブが作成されます。
    4. バックアップ ジョブの進行状況を確認するには、[USB バックアップ ジョブ] 領域の [詳細] をクリックします。
