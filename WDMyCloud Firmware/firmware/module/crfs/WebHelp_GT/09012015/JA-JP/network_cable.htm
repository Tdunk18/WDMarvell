ネットワークへの My Cloud Mirror デバイスの接続

この手順では、イーサネット ケーブルを使用して My Cloud Mirror ドライブをホーム ネットワークに接続する方法について説明します。この最初の接続を行った後、ネットワークにワイヤレスで接続するようにドライブを設定できます。

次の手順に従い、イーサネット ケーブルを使用して接続します。

    1. My Cloud Mirror デバイスの背面にあるイーサネット ポートにケーブルの一端を接続します。
    1. ケーブルのもう一端をルーターまたはネットワーク スイッチのポートに直接接続します。
    2. アダプタのコネクタをドライブ背面にある電源ソケットに接続します。
    3. 電源アダプタをコンセントに差し込みます。My Cloud Mirror デバイスの電源が自動的に入ります。

重要：My Cloud Mirror デバイスが立ち上がるまで待ってから (約 3 分)、コネクタを設定します。

イーサネット ケーブルを使用して My Cloud Mirror デバイスをネットワークに接続した後に、Web インターフェイスを使用して、ワイヤレス接続を設定できます。「接続オプションの設定」を参照してください。
