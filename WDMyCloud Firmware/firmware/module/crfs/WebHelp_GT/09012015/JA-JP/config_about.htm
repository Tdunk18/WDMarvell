セットアップと設定について

My Cloud Mirror デバイスの所有者および管理者は、すぐにメディアにアクセスできます。ただし、コネクタの初期セットアップとして完了する必要のあるタスクがあります。推奨されるセットアップ タスクの概要については、「使用開始」を参照してください。

家族と一緒にパーソナル クラウドをお使いになるには、パーソナル クラウドを適宜変更する必要が生じる場合があります。これらの種類の変更は [設定] をクリックして始めます。設定は以下のカテゴリによって構成されます。

    • 全般
    • ネットワーク
    • メディア
    • ユーティリティ
    • 通知
    • ファームウェア更新

各カテゴリで利用可能な設定タスクを示します。

全般

デバイス プロファイル

  • デバイス名の変更  
  • デバイスの説明の変更

言語と時刻

  • My Cloud Mirror インターフェイスの表示言語の設定  
  • 日時の設定
  • タイム ゾーンの設定  

クラウド アクセス

  • クラウド アクセスのオン/オフ  
  • 接続オプションの設定
  • ポート転送の有効化  
  • リモート アクセス ステータスの確認  

省電力

  • デバイス前面の LED のオン/オフ  

  • デバイスがスリープ状態になるタイミングの決定

  • Apple Time Machine へのバックアップの設定  

  • Time Machine によるバックアップのオフ
