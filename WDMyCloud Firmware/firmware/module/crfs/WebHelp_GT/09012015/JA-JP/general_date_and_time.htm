日時の設定

次の手順に従い、My Cloud Mirror デバイスの日時を設定します。

タイム ゾーンの設定

    1. [設定] をクリックします。
    2. 左側のペインで、[全般] をクリックします。
    3. [タイム ゾーン] フィールドで、ドロップダウン メニューからタイム ゾーンを選択します。
    4. [保存] をクリックします。

NTP との日時の同期

    1. [設定] をクリックします。
    2. 左側のペインで、[全般] をクリックします。
    3. をクリックし、NTP サービスをオンにします。
    4. NTP サービスが使用するサーバーを決定します。操作方法    

       

      NTP サービスを設定するには:

      1. [プライマリ サーバー] フィールドの横にある [設定] リンクをクリックします。
      2. [NTP 設定] ダイアログ ボックスで、[ユーザー NTP の追加] をクリックします。
      3. サーバーの詳細を入力し、[保存] をクリックします。
      4. 必要に応じて手順 1 と 2 を繰り返します。最大で 3 つの NTP サーバーを入力できます。
