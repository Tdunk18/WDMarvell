ユーザー アカウントの削除

管理者は、My Cloud Mirror システムからユーザー アカウントを削除できます。接続された特定の共有またはデバイスに対するアクセスを制限するだけの場合は、「共有へのユーザー アクセスの変更」を参照してください。次の手順に従い、ユーザー アカウントを削除します。

    1. [ユーザー] をクリックします。
    2. 左側のペインで、削除するユーザーの名前をクリックします。
    3. [ユーザーの削除] アイコンをクリックします。
    4. [OK] をクリックします。選択したユーザー アカウントが削除され、左側のペインに表示されなくなります。
