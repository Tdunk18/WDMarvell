省電力モードの設定

次の手順に従い、My Cloud Mirror デバイスの電力消費量を削減します。

    1. [設定] をクリックします。
    2. [全般] をクリックします。
    3. [省電力] 領域で、次のオプションを選択し、デバイスの電力消費量を設定します。

モード

説明

ドライブ スリープ

デバイスのスリープ モードを有効または無効にします。[オン] を選択すると、ドライブのスリープ モードが有効になります。

  • をクリックすると、ドライブ スリープが有効または無効になります。

LED

デバイスの LED を有効/無効にします。[オン] を選択すると、LED が無効になります。

  • をクリックすると、LED が有効または無効になります。

電源回復

電源故障により、予期しないシャットダウンが発生した後に、デバイスを自動的に再起動します。[オン] を選択すると、電源回復が有効になります。

  • をクリックすると、電源回復が有効または無効になります。

電源スケジュール

デバイスのシャットダウン日時を指定して、スケジュールを設定できます。

  • をクリックすると、電源スケジュールが有効または無効になります。
  • [設定] をクリックし、デバイスのスケジュールを設定します。

Web アクセス タイムアウト

デバイスから自動的にタイムアウトするまでのデフォルトの時間を設定します。

  • ドロップダウン メニューから Web アクセス タイムアウトを選択します。
