グループの作成

グループは、管理可能なユニットに集約されたユーザー アカウントの集合です。次の手順に従い、グループを作成します。

    1. [ユーザー] をクリックします。
    2. [グループ] をクリックします。
    3. [グループを追加] アイコンをクリックします。
    4. 新しいグループのグループ名を入力します。名前はスペースを含まない最大 16 文字で指定できます。
    5. 新しいグループに追加するユーザーを選択します。
    6. ［適用］をクリックします。新しいグループが作成され、画面の左側の列に表示されます。
