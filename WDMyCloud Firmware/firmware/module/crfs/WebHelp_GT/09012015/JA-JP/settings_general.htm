一般的な設定について

My Cloud Mirror デバイス環境のセットアップと構成には、[設定] のオプションを使用します。

[設定] カテゴリには、[全般] ビューが含まれます。このビューから実行できるタスクは以下のとおりです。

  • デバイス プロファイル

    • デバイス名の変更  
    • デバイスの説明の変更

  • 言語と時刻

    • My Cloud Mirror インターフェイスの表示言語の設定  
    • 日時の設定
    • タイム ゾーンの設定  

  • クラウド アクセス

    • リモート アクセスのオン/オフ  
    • 接続オプションの設定
    • ポート転送の有効化  
    • リモート アクセス ステータスの確認  

  • 省電力

    • デバイス前面の LED のオン/オフ  

    • デバイスがスリープ状態になるタイミングの決定  

  • Mac バックアップ

    • Apple Time Machine へのバックアップの設定  

    • Time Machine によるバックアップのオフ
