ネットワーク外部の他のユーザーとのコンテンツの共有

ネットワーク外部の他のユーザーとコンテンツを共有するには、コンピュータで動作する My Cloud デスクトップ アプリケーションを使用します。

次の手順に従い、コンテンツを共有します。

    1. まだ行っていない場合は、My Cloud デスクトップ アプリケーションを WD ラーニング センター (http://www.wd.com/setup/wdmycloudmirror) からダウンロードし、Mac または PC にインストールします。
    2. コンピュータで My Cloud デスクトップ アプリケーションを起動します。
    3. My Cloud Mirror デバイスをダブルクリックします。
    4. 左側のペインで必要なフォルダを展開し、共有するコンテンツを表示します。
    5. 右側のペインで、1 つの項目をクリックします (または、Shift キーを押しながらクリックして複数の項目を選択します)。
    6. 選択した項目を右クリックして、[電子メール アドレス] を選択します。
    7. 必要に応じて電子メール ウィンドウを入力し、メッセージを送信します。
