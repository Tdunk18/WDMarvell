Vise gjeldende RAID-modus

Bruk følgende fremgangsmåte for å vise gjeldende RAID-modus på enheten.

    1. Klikk på Lagring.
    2. Klikk på RAID
    3. I den høyre ruten vises følgende informasjon:
    • RAID-helse
    • Auto-gjenoppbygging status
    • RAID-volum
