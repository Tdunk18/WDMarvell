Wykonywanie jednorazowej konfiguracji w celu zapewnienia bezpiecznego dostępu

W poniższej procedurze przedstawiono sposób nawiązywania bezpiecznego połączenia między smartfonem lub tabletem i urządzeniem My Cloud Mirror. Należy skorzystać z instrukcji dotyczących określonego urządzenia przenośnego:

    • Tablet lub smartfon z systemem Android  

    • iPhone lub iPad

    • Telefon lub tablet z systemem Windows

    • Komputer z systemem Windows lub komputer Mac

Uzyskiwanie bezpiecznego dostępu w przypadku tabletu lub smartfonu z systemem Android

    1. Podłącz smartfon lub tablet, wykonując poniższe czynności:

      1. Na urządzeniu przenośnym stuknij pozycję Ustawienia.

      2. Stuknij pozycję Sieci i sieci bezprzewodowe.

      3. Stuknij pozycję Ustawienia Wi-Fi.

      4. Wybierz sieć domową.

      5. Wprowadź hasło sieciowe i stuknij pozycję Połącz.

    1. Uruchom aplikację WD.
    2. Stuknij pozycję Połącz teraz.
    3. Stuknij nazwę urządzenia My Cloud Mirror.
    4. Stuknij strzałkę listy rozwijanej i wybierz nazwę użytkownika.
    5. Wykonaj jedną z następujących czynności:

    • Jeśli dla użytkownika określono hasło, wprowadź je, a następnie stuknij pozycję Zaloguj.

    • Jeśli dla użytkownika nie określono hasła, stuknij pozycję Zaloguj.

    1. (Opcjonalnie) Aby uzyskać dostęp do zawartości, stuknij nazwę udziału, a następnie przejdź do żądanego pliku.

Uzyskiwanie bezpiecznego dostępu w przypadku urządzeń iPhone lub iPad

    1. Podłącz smartfon lub tablet, wykonując poniższe czynności:

    1. Na urządzeniu przenośnym stuknij pozycję Ustawienia.

    2. Stuknij pozycję Wi-Fi.

    3. Stuknij pozycję Ustawienia Wi-Fi.

    4. Wybierz sieć domową.

    5. Wprowadź hasło sieciowe i stuknij pozycję Dołącz.

    1. Uruchom aplikację WD.
    2. Stuknij pozycję Połącz teraz.
    3. Stuknij znak plus (+) obok nazwy urządzenia My Cloud Mirror.
    4. Stuknij żądaną nazwę użytkownika.
      Jeśli dla użytkownika określono hasło, wprowadź je.
    5. (Opcjonalnie) Aby uzyskać dostęp do zawartości, stuknij nazwę udziału, a następnie przejdź do żądanego pliku.

Uzyskanie bezpiecznego dostępu w przypadku tabletu lub smartfonu z systemem Windows

    1. Podłącz smartfon lub tablet, wykonując poniższe czynności:

    1. Na dowolnym urządzeniu komputerowym stuknij pozycję Ustawienia.

    1. Stuknij pozycję Wi-Fi.

    2. Wybierz sieć domową.

    3. Wprowadź hasło sieciowe i stuknij pozycję Gotowe.

    1. Uruchom aplikację WD.
    2. Stuknij pozycję Połącz teraz.
    3. Stuknij nazwę dysku My Cloud Mirror.
    4. Stuknij strzałkę listy rozwijanej i wybierz nazwę użytkownika.
      Jeśli dla użytkownika określono hasło, wprowadź je.
    5. (Opcjonalnie) Aby uzyskać dostęp do zawartości:

    1. Na ekranie Urządzenia stuknij nazwę dysku My Cloud Mirror.

    2. Stuknij udział, a następnie przejdź do żądanej zawartości.
