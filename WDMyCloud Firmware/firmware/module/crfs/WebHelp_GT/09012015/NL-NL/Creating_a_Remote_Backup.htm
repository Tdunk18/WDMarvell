Een externe back-up maken

Met externe back-ups kunt u een back-up maken van uw My Cloud Mirror-apparaat naar een andere server of buiten uw netwerk.  Gebruik de volgende stappen om een externe back-up te maken voor uw My Cloud Mirror-server.

    1. Klik op Back-ups.
    2. Klik op Externe back-ups.
    3. Klik op de knop Taak maken.
    4. Voer een taaknaam in.
    5. Selecteer een server in het keuzemenu Externe server.
    6.  In het veld Externe IP voert u het IP-adres in van het externe back-upapparaat.
      (bijvoorbeeld 192.168.1.16)
    7. In het veld Wachtwoord voert u het wachtwoord in van het externe back-upapparaat.
    8. Voer een SSH-gebruikersnaam in.
    9. Voer in het veld SSH-wachtwoord het SSH-wachtwoord in.
    10. Klik op Bladeren om een bronmap te selecteren.
    11. Selecteer in de lijst van Externe back-up-mappen een of meer sharemappen om te beschermen.
    12. Klik op OK.
    13. Klik op Bladeren om een Externe back-up-map te selecteren.
    14. Kies een map en klik op OK.
    15. Kies het Type back-up in het keuzemenu.
    16. Klik naast Terugkeerpatroon op het omschakelpictogram om een back-upschema in te stellen.
    17. Klik op het omschakelpictogram om Overdrachtsversleuteling in te stellen.
    18. Klik op Maken.
