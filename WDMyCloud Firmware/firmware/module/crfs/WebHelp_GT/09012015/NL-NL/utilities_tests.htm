Diagnostische tests uitvoeren

U kunt de conditie van uw apparaat bekijken door een van de volgende testen uit te voeren:

    1. Snel: Controleert uw apparaat op grote problemen met de prestaties; het resultaat is een beoordeling die goed of fout kan zijn. Een snelle test kan een paar minuten duren.

    • Volledig: Test elke sector van uw apparaat en geeft feedback over de conditie van het My Cloud Mirror- apparaat.  Een volledige test kan uren duren, afhankelijk van de hoeveelheid opgeslagen gegevens,

Gebruik de volgende stappen om een diagnostische test uit te voeren:

    1. Klik op Instellingen.
    2. Klik in het linkerdeelvenster op Hulpprogramma's.
    3. Klik in het vak Systeemdiagnose op Snelle test of Volledige test.
      Een indicator geeft de status van de test weer en een dialoogvenster geeft de resultaten weer.
